## 2. 项目一句话

《归墟长生》是凡人流底层修仙长篇。

坊市杂役赵长生得到残破归墟炉，从废丹、残器、枯草、破玉简里炼出生路；穿越者沈临风带着天命系统，以为自己拿了主角剧本，后来发现自己只是新仙界司命台培养和回收的棋子。

终局不是继承新仙界，也不是复辟旧天庭，而是重炼万界循环。

## 3. 最高硬规则

以下规则不可推翻：

1. 归墟不是邪恶反派，而是天地终末机制。
2. 归墟大潮不是杀死最终 Boss 就能解决，真正问题是循环失衡。
3. 旧天庭不是完美答案，它在第一次归墟大潮中失败过。
4. 新仙界不是单纯蠢坏，它有墟眼危机，但收割制度已经腐烂。
5. 赵长生不是天命之子，也不是高贵血脉，而是命数断页之人。
6. 赵长生能用归墟炉，是因为命数残缺，不是因为被旧天庭提前安排。
7. 归墟炉不是无限电池；归墟无穷但不可直接用，残炉只能把一线终末之力转成重炼能力。
8. 沈临风确实是穿越者，不是假记忆。
9. 沈临风是旧司命台残火引入的异世变数，但被新仙界改造系统套牢。
10. 飞升真相是筛选、登记、验源、分流和回收。
11. 终局必须是重炼循环，不是继承新仙界，也不是复辟旧天庭。

## 7. 第一卷执行边界

第一卷《青石坊市》的目标是让读者相信：

1. 赵长生够穷。
2. 修仙界够狠。
3. 归墟炉够爽但有限。
4. 赵长生够谨慎。
5. 沈临风系统有趣但不对劲。
6. 青石坊市待不下去了。

第一卷必须出现：

- 丹渣称重。
- 发霉灵米当工钱。
- 灵井水变淡。
- 残器涨价。
- 上宗收灵灰、废器、残玉简。
- 杂役契写“青槐灾民赵长生”。
- 胸口旧烙痕在归墟炉启动时发热。
- 系统显示赵长生为关键人物，但不显示气运。
- 卷尾系统报错四行冲突提示：“司命残录校验失败 / 目标命数接口缺页 / 建议接近目标 / 警告远离异常源”。

第一卷禁止解释：

- 旧天庭九器全貌。
- 第一次归墟大潮。
- 新仙界、墟眼、中天玄阙。
- 补墟司、补天司、补墟账本。
- 司命台真实来源。
- 沈临风穿越原因全貌。
- 赵长生命簿已死。
- 无名老修身份。
- 飞升完整流程。

## 8. 正文写作尺度

正文基调：

- 先小后大。
- 先资源后世界。
- 先生存后求道。
- 不提前解释旧天庭和仙界。
- 第一卷只需要把一粒废丹写好。

战斗尺度：

- 第一卷只写炼气低阶战斗。
- 赵长生不能正面越级乱杀。
- 第一次杀人靠地形、炉灰、残器、低阶符箓、沈临风一次提示和敌人轻视成立。
- 杀人后必须有伤势、物证、追查和心理代价。

感情和亲密戏尺度：

- 可以有正常的恩爱、亲密和情欲张力。
- 明确亲密戏少量出现，点到即止，以留白为主。
- 不写露骨过程，不把床戏当卖点。
- 女主不是奖励，也不是修炼资源或功能挂件。
- 感情线必须绑定道途、责任、代价和彼此选择。

制度线和“账”的尺度：

- “账”是制度压迫隐喻，不是主角一路查账的玩法。
- 第一章《账上无错》可以明写工钱账和废料账；第二章开始必须降频，转向灵井、契书、任务牌、族契、仙籍、配额和行动代价。
- 不让每卷都靠账册推进，不让所有反派都变成账房或管事。
- 终局写重炼循环、公开责任和规则重定，不写“对账成功”。
- 完整规则见 `02故事架构/04_伏笔暗线/02_制度压迫表现层级与账线降频规则.md`。
