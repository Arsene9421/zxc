# 第 1 章单章详纲：账上无错

> 重构说明：本章由旧版第 1-2 章压缩而来，负责在一章内完成丹铺压迫、灵米/丹渣/炉灰入账、青槐灾民身份和旧灰责任，并把主线钩子推到旧灰缸底异常。

## 1. 本章定位

| 项目 | 内容 |
|------|------|
| 章节功能 | 建立赵长生的生存压力，并把废料账直接推到旧灰缸 |
| 主要场景 | 回春丹铺前堂、后院、夜里旧灰缸 |
| 出场人物 | 赵长生、周槐安、小伙计、买丹散修 |
| 当前冲突 | 账面无错但实物全是霉米和废渣，赵长生被迫承担旧灰责任 |
| 信息增量 | 丹药涨价、废料入账、青槐灾民契、青云办事处查废料 |
| 章尾钩子 | 赵长生夜里翻旧灰缸，铁钩敲到非灰非缸底的硬物 |

## 2. 执行要点

1. 苦难只保留能推动主线的部分：霉米、丹渣、炉灰和旧灰责任。
2. 不展开长篇灵井日常，只用青槐灾民身份和旧契压住人物处境。
3. 废料账必须成为捡炉的直接原因。
4. 章尾只给硬物异常，不解释黑炉。

## 3. 禁写项

- 不写归墟炉来历。
- 不写系统、沈临风或后期真相。
- 不让赵长生主动寻宝。
