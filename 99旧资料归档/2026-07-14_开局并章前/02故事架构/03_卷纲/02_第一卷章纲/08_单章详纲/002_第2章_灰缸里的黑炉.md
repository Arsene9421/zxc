# 第 2 章单章详纲：灰缸里的黑炉

> 重构说明：本章由旧版第 3-4 章压缩而来。重点修正“主角凭什么试炉”的问题：先由废丹碎末误入炉口触发微弱异常，再逼出赵长生最低限度确认，而不是发现炉子后主动研究外挂。

## 1. 本章定位

| 项目 | 内容 |
|------|------|
| 章节功能 | 发现黑炉，并完成第一次最低限度验证 |
| 主要场景 | 回春丹铺后院、小柴房 |
| 出场人物 | 赵长生、小伙计短暂出现 |
| 当前冲突 | 黑炉不能交、不能丢、不能卖，但又出现无法忽视的小异常 |
| 信息增量 | 黑炉压在旧灰缸底，胸口旧烙痕轻热，废丹碎末误入后霉味退一圈 |
| 爽点边界 | 发霉碎丹只退一点霉，出现一丝药香，不成丹，不可食 |
| 章尾钩子 | 赵长生确认废物能回一点药性，但风险大于收益，必须藏炉和记录 |

## 2. 执行要点

1. 黑炉发现来自清旧灰责任，不是寻宝。
2. 第一次反应必须是意外触发，不是主动试验。
3. 赵长生的主动验证只能发生在异常之后，且用最低价值、最小剂量。
4. 结论必须写清：不成丹，不可食，不可示人。
5. 爽点立刻被废料账和暴露风险压住。

## 3. 禁写项

- 不写认主、炉灵、天地异象。
- 不解释归墟炉来源。
- 不让赵长生吞服碎丹或突破。
- 不写系统、沈临风或后期世界观。
