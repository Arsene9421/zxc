# 《归墟长生》世界观底层逻辑 v0.2

> 本文档为 2026-07-11 确认的背景定稿。
> 作用：提前锁定《归墟长生》的底层因果链，保证后续剧情又爽又合理。
> 原则：前期可以只写废丹、坊市、资源争夺；后台逻辑必须提前成立，后续不轻易推翻。

---

## 0. 一句话定稿

归墟大潮是太虚与归墟循环失衡后的宇宙级反噬。旧天庭上一次试图救尽万界，但在大潮中系统超载，并因内部保全派分裂而失败，归墟炉破碎，仙界核心被打出墟眼。保全派在废墟上建立新仙界，选择以墟眼危机为名收割下界，维持仙界火种。

无数年后，这套临时自救制度已经固化为压迫秩序。新仙界通过飞升通道、天命系统和监察网络持续收割万界本源，反而让太虚与归墟更加失衡，第二次归墟大潮逼近。

赵长生是幼年灾祸中从命簿断页里活下来的人，因命数残缺而被归墟炉承认；沈临风是旧司命台残余从异世拉来的变数，却被新仙界套上改造后的天命系统。两人一个重炼残缺，一个挣脱天命，最终不继承新仙界，也不复辟旧天庭，而是把仙界重炼成万界循环中枢。

## 1. 创作硬原则

本书的世界观必须服务两件事：

1. 前期爽点直接。
2. 后期逻辑站得住。

不能为了显得宏大，一开局就倒旧天庭、太虚、归墟海。读者前期只需要看到赵长生够穷、资源够紧、归墟炉能从废物里炼出价值。

高层设定的展开顺序：

| 阶段 | 读者看到的表象 | 背后的真相 |
|------|----------------|------------|
| 坊市 | 修仙界穷，废丹废器也有人争 | 下界长期被抽取本源 |
| 宗门 | 资源稀缺，外门竞争残酷 | 灵脉和道统都在衰败 |
| 金丹元婴 | 古战场残魂、天劫异常 | 仙门被改造，飞升通道变质 |
| 化神灵界 | 飞升者失踪，万年抽源 | 新仙界定期收割万界 |
| 仙界 | 飞升不是逍遥，而是登记、服役、回收 | 仙界靠下界填补墟眼 |
| 终局 | 仙帝不是唯一问题 | 整个修行循环坏了 |

四条硬规则：

- 归墟不是反派，而是天地终末机制。
- 旧天庭不是完美答案，它失败过。
- 新仙界不是单纯蠢坏，它有现实危机，但制度已经腐烂。
- 赵长生不是天命之子，他是天命之外的残缺者。

## 2. 万界结构

本书不做无限套娃。仙界之上不是新的升级地图，而是规则层。

| 层级 | 含义 | 剧情作用 |
|------|------|----------|
| 下界 | 赵长生开局所在人间修仙界 | 坊市、宗门、修仙国、凡人流生存 |
| 中层世界 | 灵界、妖界、魔界等 | 百族、真灵、万年抽源、仙界监察 |
| 仙界 | 目前统治万界的高层世界 | 飞升真相、新仙界秩序、旧天庭废墟 |
| 太虚 | 世界之外的可能之处 | 新道诞生、未成形规则、终局重铸 |
| 归墟海 | 万物残败后的终末之海 | 残败本源归处，归墟炉的真正炉料 |

正常循环应该是：

1. 太虚生出新道、新气、新世界可能。
2. 万界承载这些新道，孕育生灵、修士、文明、法宝、道统。
3. 万物运转太久之后，必然出现残败、破损、衰竭。
4. 残败本源流向归墟。
5. 归墟炉一类的循环器物将残败本源重炼。
6. 重炼后的本源重新反哺太虚和万界。

旧天庭的职责，是维护这个循环。

新仙界的问题，是不承认自己也应该进入循环。它害怕仙界归墟，不让仙界残败本源回到循环中，反而不断从下界抽血填补自己。结果循环被堵塞，太虚新生越来越弱，归墟沉积越来越重，第二次归墟大潮越来越近。

## 3. 归墟大潮

归墟大潮不是魔神，不是最终怪物，也不是一个有明确人格的反派。

它是太虚与归墟之间失衡后的宇宙级反噬。

万界正常运转时，归墟只是终末机制。丹药废了，药性散入归墟；法宝碎了，宝性散入归墟；修士死了，命数散入归墟；世界崩了，界源坠入归墟。

但如果残败本源长期无法重炼，太虚又得不到足够反哺，归墟端就会越来越重，太虚端越来越弱。等失衡超过极限，归墟海就会大规模翻涌，万界同时出现灵气枯竭、道则断裂、世界坠落、生灵命数归零。

这就是归墟大潮。

归墟大潮的叙事价值：

- 它让终局不只是打仙帝。
- 它证明新仙界的收割只能续命，不能解决问题。
- 它证明旧天庭当年不是弱，而是面对过真正的宇宙级难题。
- 它逼赵长生回答：如果旧天庭失败，新仙界腐烂，那第三条路是什么？

归墟大潮不应写成“杀死某个归墟之主就能结束”。真正要解决的，是循环失衡。

---

## 4. 第一次归墟大潮与旧天庭失败

旧天庭不是被某个邪恶仙帝简单灭掉的。它败在一次真正的大潮里，也败在内部路线分裂里。

第一次归墟大潮前，旧天庭已经发现万界同时衰败：

- 大量下界灵脉干枯。
- 中层世界界壁开裂。
- 亡魂无法正常入轮回。
- 太虚新道减少。
- 归墟海潮位上升。
- 登仙台接引的飞升者身上出现残败污染。

旧天庭启动全部循环体系：

| 机构或器物 | 当时职责 |
|------------|----------|
| 司命台 | 寻找可能补天、镇界、续道之人 |
| 登仙台 | 接引修士入仙，调配到各处危局 |
| 补天尺 | 修补世界裂缝 |
| 镇界印 | 稳定濒临崩溃的世界本源 |
| 轮回灯 | 接引大劫中的亡魂，防止魂魄散尽 |
| 万法碑 | 保存濒灭道统 |
| 太虚舟 | 进入世界之外寻找新道 |
| 归墟炉 | 回收残败本源，重炼为新的天地资粮 |
| 斩道剑 | 斩断已经腐朽、会拖累万界的旧道 |

关键分裂在于：救尽万界，还是保住仙界火种。

旧天庭内部可以暂分两派：

| 派别 | 主张 | 后果 |
|------|------|------|
| 循环派 | 万界都在循环内，能救一界是一界 | 归墟炉和补天体系被推到极限 |
| 保全派 | 救所有世界会一起死，必须保仙界火种 | 后来成为新仙界的根源 |

旧天庭失败的关键过程：

1. 循环派试图将大量衰败世界的本源导入归墟炉重炼。
2. 归墟炉承载的残败本源超过极限，炉火不稳。
3. 保全派在最关键时刻改写镇界印和登仙台阵枢，试图切断部分下界，把仙界本源单独护住。
4. 循环突然断裂，归墟之力倒灌仙界核心。
5. 仙界核心被打穿，形成墟眼。
6. 归墟炉破碎，司命台崩塌，登仙台断裂，九器散落。
7. 旧天庭诸神大半战死或失踪。
8. 保全派在废墟上建立新仙界，宣称自己保住了最后一座天。

这段历史必须保持双重性：

- 循环派的理想是对的，但他们确实没能救下所有世界。
- 保全派的选择不是毫无理由，但他们后来把临时自救变成了永久压迫。

赵长生后期面对旧天庭遗民时，不能简单说“你们全是对的”。他必须看到：旧天庭的旧路已经失败，新仙界的新路已经腐烂，自己只能重炼第三条路。

## 5. 新仙界与墟眼

墟眼是新仙界收割万界的实体原因。

墟眼位于仙界核心，是第一次归墟大潮时归墟之力倒灌留下的伤口。它不断吞噬仙界本源、仙脉、仙道法则，让仙界看似高高在上，实际一直在漏血。

新仙界早期确实面临现实危机：

- 不填补墟眼，仙界会崩。
- 仙界崩后，大量世界会失去旧秩序残余保护。
- 第二次归墟大潮若来，万界可能没有任何中枢抵挡。

所以新仙界最初的口号是：

> 先保仙界，再救万界。

但无数年后，这个口号变质为：

> 只保仙界，万界皆为燃料。

新仙界的三层逻辑：

| 层级 | 内容 |
|------|------|
| 表层 | 维持万界秩序，防止混乱 |
| 中层 | 抽取下界本源，填补墟眼 |
| 深层 | 仙界权贵借危机固化统治，把收割制度变成永恒利益 |

这样新仙界不是单纯贪婪，但它必须成为主角要推翻的腐朽制度。

仙界人物可以有苦衷，仙界制度必须有罪。

## 6. 飞升通道与收割体系

旧天庭时期，飞升是登仙台接引。

修士完成自身修行后，登仙台会接引其进入更大的天地循环。飞升者不必被剥夺本源，而是根据自身道途进入仙界、镇界、补天、护法、入太虚等职责。

新仙界时期，正常仙门已断。

现在下界残留的飞升通道，是新仙界改造过的收割通道。飞升不是奖励，而是筛选、登记和回收。

天劫也发生了变化：

| 旧天劫 | 新天劫 |
|--------|--------|
| 考验修士是否能承载更高道则 | 检测修士本源、气运、法宝、血脉 |
| 渡过后接引入登仙台 | 渡过后进入新仙界登记体系 |
| 失败者魂归轮回或散入归墟 | 失败者可能被仙界提前回收 |

飞升者的可能命运：

| 类型 | 命运 |
|------|------|
| 普通飞升者 | 登记、服役、挖仙矿、交仙税 |
| 战力强者 | 编入仙兵，替仙界镇压下界或归墟污染 |
| 血脉特殊者 | 被仙族、大族、仙府带走研究或炼血 |
| 法宝特殊者 | 法宝被夺，人被监控 |
| 气运强者 | 送入司命台系统阵营 |
| 天命系统持有者 | 成熟后回收系统成果 |
| 不服管束者 | 打入矿场、炼成仙器材料或被抹去命数 |

这条设定要在中后期逐步揭开。

前期只需要留下痕迹：

- 飞升传说越来越少。
- 古籍里的飞升记载和现实不符。
- 化神修士渡劫失败后消失得太干净。
- 天劫像是在扫描，而不只是考验。
- 归墟炉聚魂后能看到渡劫失败者被“收走”的残影。

## 7. 天命系统与司命台

沈临风的天命系统来自旧天庭司命台残骸，但被新仙界改造。

旧司命台的原始职责不是奴役众生，而是：

- 观测命数裂痕。
- 在大劫前寻找可能补天、镇界、续道的人。
- 给这些人有限机缘，让他们在危局中成长。
- 记录世界危机，调配资源。

新仙界夺取司命台主体后，将其改造成天命棋子培养器：

1. 在下界筛选气运强者或异数。
2. 绑定系统。
3. 发布任务。
4. 提供奖励。
5. 引导其收集机缘、搅动局势、快速成长。
6. 成熟后通过飞升通道回收。

系统必须有两层逻辑：

| 层级 | 来源 | 表现 |
|------|------|------|
| 底层残火 | 旧司命台残余 | 识别世界危机，寻找真正变量，提示沈临风接近赵长生 |
| 上层控制 | 新仙界司命台 | 培养可回收棋子，要求沈临风服从任务，后期命令他远离赵长生 |

因此系统前期可以真有用：

- 给功法。
- 给丹药。
- 给低阶护身符。
- 提示机缘。
- 显示部分气运。
- 发布任务奖励。

但系统奖励不是无代价的。每次领取奖励，都会增加沈临风和系统的因果绑定；每次完成任务，都会让新仙界更了解他的命数、道途、弱点和欲望。

中期关键爆点：

> 【主线任务：远离赵长生。】
> 【任务奖励：元婴机缘。】
> 【失败惩罚：天命值削减九成。】

沈临风拒绝这次任务，才真正从“系统主角”变成“有自主意志的人”。

## 8. 沈临风穿越定稿

沈临风确实是穿越者，不是系统植入假记忆。

他的穿越原因：

旧司命台残火发现本界命数大多已经被新仙界监控，普通天命之子很难逃出系统掌控。于是它从异世拉来一个不在本界命簿中的灵魂，试图制造变量。

但新仙界掌握了司命台大部分权限。沈临风降临此界时，被新仙界截获并绑定改造后的天命系统。

所以沈临风一开始就是矛盾体：

- 他是旧司命台想引入的变数。
- 他也是新仙界想培养的棋子。
- 他有系统便利。
- 他也被系统监控。
- 他自以为拿了主角剧本。
- 实际上他是两套天命逻辑争夺的对象。

沈临风成长线：

| 阶段 | 状态 |
|------|------|
| 前期 | 我有系统，我是主角，赵长生像隐藏 NPC |
| 中期 | 系统为什么总让我接近赵长生，又突然让我远离赵长生 |
| 爆点 | 我不接这个任务 |
| 后期 | 系统不是我的主人，天命也不是我的命 |
| 终局 | 夺回司命台残火，重定众生命数的规则 |

沈临风的前期调性：半轻松半谨慎。

他可以吐槽，可以有现代人视角，可以误判赵长生，但不能降智。他怕死，知道修仙界危险，只是没有赵长生那么稳。

## 9. 赵长生命数残缺定稿

赵长生不是天命之子，而是天命之外的人。

他的命数残缺来源：

赵长生幼年时，青石坊市附近某个小村发生过一场无人重视的灾祸。表面上看，是山洪、瘟疫、妖兽或散修斗法造成的村灭；真实原因，是第一次归墟大潮后散落下界的一缕旧天庭残器气息扰乱当地命数。

司命台残簿记录中，那一地的人都死了，赵长生也死了。

但他没有死透。

一名无名老修临死前，用一枚沾有归墟炉灰的残铜片替赵长生遮住将熄的命火。那残铜片后来或融入心脉，或在多年流浪中遗失，前期不需要明说。

从那以后：

- 命簿上赵长生已经死过一次。
- 天命系统无法完整读取他的未来。
- 天机推演只能看到断页。
- 仙界监察者初期会忽略他。
- 归墟炉却能感应到他身上的残缺气息。

归墟炉为什么认赵长生？

因为归墟炉不认完整天命，不认绝世天才，不认仙界册封。它认的是残缺之后仍未熄灭的一线本源。

赵长生的灵根是杂的，出身是低的，命数是断过的，修行路也是残的。他不是被安排好的救世主，而是一次失败事故里从命簿断页中爬出来的人。

核心设定句：

> 完整者入不得归墟。有缺者，方能重炼。

这句话可以作为后期重要揭示。

## 10. 归墟炉机制

归墟炉不是许愿机，也不是普通聚宝盆。

它的基本规则：

> 有缺之物，入炉返本。
> 残败之物，归墟重炼。
> 越有来历、越有本源、越有残余价值的东西，越能被归墟炉重新养出价值。

归墟炉运转需要三要素：

| 要素 | 含义 |
|------|------|
| 残余本源 | 废丹残药性、残器宝气、枯草生机、破界界源等 |
| 归墟炉火 | 炉子自身的重炼能力，随碎片和赵长生理解逐步恢复 |
| 承载代价 | 时间、灵气、材料、宝气、魂力、气运、界源等 |

限制必须明确：

1. 不能凭空造宝。
2. 没有残余本源的东西不能重炼。
3. 重炼需要时间，越高级越慢。
4. 重炼需要代价，越高级越贵。
5. 归墟炉残缺，能力分阶段解锁。
6. 重炼高阶物品会留下同源波纹，可能吸引旧天庭残片、新仙界监察者和归墟污染者。
7. 前期不能超过物品原本上限太多，只能恢复、提纯、补足；后期加入其他本源后，才能真正重炼升阶。

能力递进：

| 阶段 | 能力 | 剧情作用 |
|------|------|----------|
| 坊市 | 废丹、灵米、枯草、残器恢复一点价值 | 活下去，积累第一桶资源 |
| 宗门 | 补功法、修阵盘、养灵田 | 低调发育 |
| 筑基金丹 | 聚宝气、炼本命法宝、补丹方 | 形成战力体系 |
| 元婴化神 | 聚魂、聚运，窥见天劫残影 | 接入世界真相 |
| 灵界 | 聚界，修复洞天、小世界 | 从修物升级到修世界 |
| 仙界 | 聚道，补全残缺大道 | 触及仙界秩序 |
| 终局 | 重炼仙界 | 完成全书命题 |

前期爽点要写得具体：

- 一枚发霉碎丹恢复药香。
- 一把裂开的低阶飞剑重新聚出半寸寒光。
- 一株将死灵草重新抽芽。
- 一片破玉简补出三个关键字。
- 一小撮废炉灰炼出能稳定丹火的残渣。

这些小爽点，后期都要能对应大逻辑：

废丹可以返本，仙界也可以重炼。

## 11. 旧天庭九器的规则含义

旧天庭九器不是简单收集品。它们分别对应万界循环的一条规则。

| 器名 | 旧职能 | 新仙界扭曲 | 终局重炼方向 |
|------|--------|------------|--------------|
| 归墟炉 | 回收残败本源，重炼新生 | 被打碎，残片散落 | 重新成为循环炉心 |
| 司命盘 | 观测命数裂痕，扶持补天之人 | 改造成天命系统，培养可回收棋子 | 不再安排命运，只记录危机和自愿选择 |
| 登仙台 | 接引完成修行者进入更大循环 | 改造成飞升收割通道 | 恢复真正飞升，取消强制回收 |
| 补天尺 | 修补世界裂缝 | 被封存或只用于仙界本土 | 重新用于万界补天 |
| 万法碑 | 保存万界道统 | 道统被仙界垄断 | 开放道统传承，防止大道断绝 |
| 轮回灯 | 接引残魂，维持轮回 | 亡魂被炼成资源或封入仙器 | 让生死重新进入循环 |
| 镇界印 | 稳定一界本源 | 被用于锁死下界和抽源 | 稳定万界，而非压榨万界 |
| 太虚舟 | 横渡世界之外，接引新道 | 被仙界控制，阻断外部变量 | 重新连接太虚可能 |
| 斩道剑 | 斩断腐朽大道 | 被新仙界用来镇压异端 | 斩断坏死秩序，防止新循环僵化 |

九器线的写法：

- 每一器不是为了加战力，而是为了揭示一条世界规则。
- 每一器都可以在不同卷出现碎片、残痕、传承或伪造品。
- 赵长生不应只是集齐九器开大，而是逐渐理解每一器的职责，最后把旧器职能重炼成新秩序。

## 12. 三条道路的理念冲突

终局不是“好人打坏人”那么简单，而是三条道路竞争。

| 道路 | 代表 | 主张 | 问题 |
|------|------|------|------|
| 旧天庭 | 循环派、旧天庭遗民 | 万界皆在循环内，残缺可补，众生可渡 | 当年失败过，救尽万界导致系统超载 |
| 新仙界 | 保全派后裔、仙界权贵 | 资源有限，牺牲下界，保住仙界火种 | 临时自救变成永久收割，制度腐烂 |
| 赵长生 | 归墟炉持有者、命数断页之人 | 旧制已败，新制已腐，必须承认残缺并重炼循环 | 代价巨大，旧仙界会崩，既得利益者必然反扑 |

赵长生不能是空喊救世的圣人。

他的道路来自底层经验：

- 废丹不是扔掉就完。
- 残器不是补补就行。
- 有些东西坏得太深，不能原样修。
- 真正的重炼，必须承认它已经坏了。

所以他面对新仙界，不是“我要坐上仙帝位，做一个好仙帝”。

他面对旧天庭，也不是“你们的制度很好，我来复辟”。

他的答案是：

> 坏掉的炉，不能只换主人。
> 碎掉的炉，也不能照旧拼回去。
> 要入归墟，重新炼一遍。

## 13. 终局重炼后的新循环

重炼仙界不能只是一句口号。它必须变成一套新规则。

重炼后的仙界不再是万界皇城，而是万界循环中枢。

它仍然高于下界，但不再是统治关系，而是职责关系。飞升者不再被强制登记、奴役、回收，而是进入更大的修行与循环体系。

新循环暂定规则：

| 机制 | 重炼后作用 |
|------|------------|
| 新归墟炉 | 回收不可避免的残败本源，重炼新生 |
| 新司命盘 | 记录命数裂痕和世界危机，不强行安排众生命运 |
| 新登仙台 | 开放真正飞升通道，飞升者可选择道职 |
| 新补天尺 | 修补濒危世界，而不是只护仙界 |
| 新万法碑 | 保存并开放道统，防止传承被高层垄断 |
| 新轮回灯 | 护持魂魄循环，不让亡魂被炼成资源 |
| 新镇界印 | 稳定各界本源，禁止强制抽源 |
| 新太虚舟 | 接引新道、新界、新可能 |
| 新斩道剑 | 斩断腐朽大道，防止新秩序再次僵死 |

飞升者的新选择：

- 继续修行。
- 镇守一界。
- 参与补天。
- 入太虚寻道。
- 守归墟炉火。
- 护轮回灯。
- 传法于万界。
- 在大限前自愿归墟重炼。

新循环不是乌托邦。

修仙仍然艰难，资源仍然有限，争斗仍然存在，道途仍有生死。但万界不再被制度性收割，飞升不再是回收通道，天命不再被仙界垄断。

赵长生终局不坐仙帝位。

他可以成为归墟道祖，但这个称号不是皇位，而是职责。他点燃归墟炉，让仙界从统治万界的高层，变成维护万界循环的炉心。

## 14. 卷际揭示节奏

大背景不能提前灌给读者。每一卷只揭开足够推动本卷冲突的一层。

| 卷 | 表层剧情 | 背景揭示 |
|----|----------|----------|
| 第一卷 青石坊市 | 废丹、杂役、归墟炉初试、沈临风登场 | 修仙界正在变穷；卷尾出现“万物归墟，仙门已断” |
| 第二卷 外门苟修 | 宗门废料、残卷、灵田、外门竞争 | 资源稀缺不是局部现象；系统频繁关注赵长生 |
| 第三卷 筑基风波 | 黑市、拍卖、丹炉、筑基 | 宝气概念出现；旧万宝殿残痕初显 |
| 第四卷 洞府与灵田 | 废山变灵山，洞府经营 | 灵脉可被修补，也可被抽干 |
| 第五卷 金丹战场 | 正魔大战、古战场、残器 | 古战场残魂看到仙界清洗痕迹 |
| 第六卷 海外妖族 | 海市、百族、真灵遗迹 | 万界并非单一人族修仙界，真灵血脉也被仙界觊觎 |
| 第七卷 元婴化神 | 气运争夺、系统异常、天劫 | 天劫不是自然考验，而是仙界筛选机制 |
| 第八卷 灵界真相 | 灵界百族、万年抽源、监察者 | 新仙界定期收割万界本源 |
| 第九卷 仙界底层 | 飞升者矿场、仙税、司命台后台 | 墟眼、新仙界制度、旧天庭遗民全部浮出水面 |
| 第十卷 重炼仙界 | 三道路选择、归墟大潮再临 | 仙界不是终点，而是坏掉的循环中枢 |

第一卷尤其要克制。

第一卷可以种下这些隐线：

- 青石坊市底层修士异常贫困。
- 掌柜克扣废丹废渣也很认真，说明资源紧到废物都有价值。
- 老散修偶尔提到几十年前飞升传说断绝。
- 沈临风系统一开始就提示赵长生，却不给原因。
- 赵长生幼年经历可以只露一点，比如他没有族谱、没有完整来历，或对某场旧灾避而不谈。
- 归墟炉卷尾浮现残字，不解释完整含义。

不要在第一卷解释旧天庭。

## 15. 主角调性定稿

赵长生：

- 谨慎，不轻信机缘。
- 能忍，但不是懦弱。
- 不主动害无辜。
- 对确认的敌人不留后患。
- 不做空口善事，但有自己的底线。
- 他的道义来自亲眼见过底层怎么活，而不是高高在上的救世宣言。

赵长生最适合的底色：

> 谨慎但有底线，外冷内有一线火。

沈临风：

- 半轻松半谨慎。
- 有系统流吐槽感，但不能降智。
- 前期以为自己是主角。
- 中期开始怕系统。
- 后期敢关掉系统面板。
- 终局不是赵长生附庸，而是司命台线的真正承担者。

沈临风最适合的底色：

> 嘴上轻松，心里怕死；起初信系统，最后信自己。

新仙界反派调性：

- 制度必须坏。
- 权贵必须该杀。
- 底层仙人、飞升者、旧天庭遗民可以各有立场。
- 少数高层可以有苦衷，但苦衷不能洗白制度性收割。

## 16. 不得轻易推翻的硬规则

后续写作中，以下规则不要轻易改变：

1. 归墟不是邪恶反派。
2. 归墟大潮不是杀一个最终 Boss 就能解决的灾难。
3. 赵长生不是被旧天庭提前安排好的救世主。
4. 赵长生能用归墟炉，是因为命数残缺，不是因为血统高贵。
5. 沈临风确实是穿越者，不是假记忆。
6. 沈临风的系统有旧司命台残火和新仙界控制两层逻辑。
7. 飞升通道已经被新仙界改造，飞升真相是筛选和回收。
8. 旧天庭有理想，但失败过，不能原样复辟。
9. 新仙界有危机，但收割制度已经腐烂，不能只换好皇帝。
10. 终局必须是重炼循环，不是继承仙界，也不是恢复旧天庭。

仍可后续细化但不能推翻大方向的内容：

- 赵长生幼年旧灾的具体地点、人物、表层事件。
- 无名老修的身份是否需要揭示。
- 墟眼的正式名称和仙界补墟机构名称。
- 旧天庭循环派、保全派的代表人物。
- 九器每一器的残片出现在哪一卷。
- 赵长生最终是否保留肉身、是否长期守炉、是否远离新仙界中枢。

## 17. 第一卷反推要求

因为底层逻辑已经定稿，第一卷设计时必须服务这些后台逻辑，但不能提前说破。

第一卷必须让读者感到：

- 这个修仙界很穷。
- 废物也有价格。
- 底层修士的命不值钱。
- 赵长生不是天才，但足够谨慎。
- 归墟炉的能力爽，但有限制。
- 沈临风有系统很爽，但系统有一点不对劲。
- 赵长生和沈临风不是天然兄弟，而是互相试探后才合作。
- 第一卷结尾的逃亡不是小打小闹，而是把赵长生从坊市推向更大的修仙世界。

第一卷不该写：

- 旧天庭大段说明。
- 仙界阴谋全貌。
- 归墟大潮解释。
- 赵长生幼年真相全揭。
- 沈临风系统来源全揭。

第一卷只需要把一粒废丹写好。

因为这本书的核心写法是：

> 先让读者相信废丹可以重炼。
> 再让读者相信功法、灵脉、洞天、命数可以重炼。
> 最后让读者相信仙界本身也可以重炼。
