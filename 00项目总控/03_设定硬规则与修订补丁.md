# 设定硬规则与修订补丁

> 本文档是全项目最高优先级设定约束之一。
> 如果其他旧文档和本文冲突，以本文为准。

## 1. 不得推翻的硬规则

1. 归墟不是邪恶反派，而是天地终末机制。
2. 归墟大潮不是杀死最终 Boss 就能解决的灾难，真正问题是循环失衡。
3. 旧天庭不是完美答案，它在第一次归墟大潮中失败过。
4. 新仙界不是单纯蠢坏，它有墟眼危机，但收割制度已经腐烂。
5. 赵长生不是天命之子，也不是高贵血脉，而是命数断页之人。
6. 赵长生能用归墟炉，是因为命数残缺，不是因为被旧天庭提前安排。
7. 沈临风确实是穿越者，不是假记忆。
8. 沈临风是旧司命台残火引入的异世变数，但被新仙界改造系统套牢。
9. 飞升通道已经被新仙界改造，飞升真相是筛选、登记和回收。
10. 终局必须是重炼循环，不是继承新仙界，也不是复辟旧天庭。

## 2. 已修订的旧设定

### 2.1 旧天庭失败

旧说法：

> 旧天庭被新仙界篡夺，新仙界打碎归墟炉。

修订后：

> 旧天庭在第一次归墟大潮中体系超载，内部保全派在关键时刻改写阵枢，导致归墟之力倒灌仙界核心，归墟炉破碎，墟眼形成。保全派在废墟上建立新仙界。

### 2.2 天命系统来源

旧说法：

> 天命系统是仙界投放到下界的棋子培养器。

修订后：

> 天命系统表层是新仙界控制的棋子培养器，底层残留旧司命台逻辑。沈临风本是旧司命台残火引入的异世变数，降临时被新仙界截获并绑定系统。

### 2.3 归墟炉身份

旧说法：

> 归墟炉是旧天庭万宝母器核心。

修订后：

> 归墟炉是旧天庭循环九器之一，负责回收残败本源、重炼新生。万宝殿偏宝物孕育和修复，归墟炉偏本源回收与重炼。

### 2.4 旧仙界用词

后续避免使用“旧仙界”。

统一称呼：

| 名称 | 含义 |
|------|------|
| 旧天庭 | 上古循环秩序 |
| 新仙界 | 保全派建立的收割秩序 |
| 仙界 | 当前高层世界本身 |
| 新循环仙界 | 终局重炼后的万界循环中枢 |

## 3. 废料回收漏洞

为了解释赵长生能捡漏，设定如下：

1. 新仙界只直接控制高阶资源，低阶废料靠宗门、坊市、丹铺层层上缴。
2. 层层盘剥导致大量废料在本地被贪污、掉包、低价倒卖。
3. 坊市掌柜、外门管事不知道废料真正用途，只知道“上面要收”。
4. 真正有价值的旧天庭残片常常无灵光、无气息、无法鉴定，会被当成废铁混在低阶废料里。
5. 归墟炉和相关残片只有在命数残缺者靠近时才会触发同源感应。

这解释：

- 赵长生为什么能捡到归墟炉。
- 底层废料市场为什么存在。
- 新仙界为什么没有早早发现归墟炉。

## 4. 仙界监察未立刻发现赵长生的原因

四层遮蔽：

| 遮蔽 | 内容 |
|------|------|
| 命数遮蔽 | 赵长生在司命残簿中是已死之人，系统无法锁定完整坐标 |
| 层级遮蔽 | 下界系统只是残缺终端，很多异常只能缓存，不能实时上传 |
| 权限遮蔽 | 旧司命台残火会吞掉或篡改一部分异常报告 |
| 阈值遮蔽 | 归墟炉前期只恢复废丹、灵米、残器，波动太低，不足以触发仙界监察 |

后续后果：

- 第一卷只是系统短暂卡顿。
- 第二卷系统频繁刷新和赵长生有关的任务。
- 第三卷开始出现下界监察暗线。
- 第八卷系统正式要求沈临风远离赵长生。
- 第十一卷仙界司命台后台查到完整异常。

## 5. 归墟炉代价硬规则

1. 本源守恒：只能恢复残余本源，不能凭空造出不存在的药性、宝气、道则。
2. 上限限制：前期只能恢复到原物接近完好的程度，不能把低阶废丹炼成高阶神丹。
3. 升阶要求：真正升阶必须加入额外本源。
4. 炉火消耗：每次重炼都消耗归墟炉火，炉火需要灵气、宝气、炉灰、残片缓慢滋养。
5. 残灰副作用：使用过度会产生归墟残灰。少量残灰可用，大量残灰会腐蚀灵草、法器、经脉。
6. 波纹风险：重炼高阶物品会留下同源波纹，吸引旧天庭残片、新仙界监察者和归墟污染者。

## 6. 天命系统奖励硬规则

| 奖励类型 | 来源 | 风险 |
|----------|------|------|
| 小额实物奖励 | 系统残存储备或新仙界预置资源 | 增加绑定因果 |
| 机缘引导 | 系统把沈临风引到已有机缘处 | 推动他成为可回收棋子 |
| 权限型奖励 | 司命盘残余权限，如提示、经验、气运加持 | 后台记录他的命数、道途、弱点 |

系统不能凭空无限发放高阶宝物。

沈临风拿过的每一次奖励，都是新仙界读取他命数的一次机会。

## 7. 魂魄、命数、气运、本源

| 概念 | 含义 | 去向 |
|------|------|------|
| 魂魄 | 个体意识与记忆的承载 | 正常入轮回灯或轮回体系 |
| 命数 | 一生因果轨迹与寿限记录 | 由司命盘记录，死后归档或散落 |
| 气运 | 外部机缘、时代势能、众生命势 | 可被宗门、王朝、系统聚散 |
| 本源 | 生命、法宝、世界最底层的存在资粮 | 残败后流向归墟，可被重炼 |

限制：

- 赵长生聚魂看到的是残魂影像，不是随便复活死人。
- 归墟炉重炼的是本源，不是直接改写所有人生死。
- 系统操作的是命数和气运，不是绝对命运。

## 8. 归墟污染者

归墟污染者不是归墟信徒，也不是纯魔物。

他们是被未重炼的归墟残败本源侵蚀的修士、仙人、器物或世界残民。

特征：

- 寿元流失异常。
- 法宝灵性腐烂。
- 记忆断裂。
- 道心执念被放大。
- 本能靠近残界、旧器、墟眼。
- 有些已经迷失，有些仍保留理智。

主题作用：

> 归墟本身不是邪恶，但没有循环重炼的归墟残败，会变成污染。

## 9. 九器与九种本源对齐

| 本源 | 对应九器或机构 | 说明 |
|------|----------------|------|
| 灵 | 补天尺、镇界印 | 灵气、灵脉、天地灵机 |
| 宝 | 归墟炉、万宝殿 | 宝气、器脉、灵宝本源 |
| 魂 | 轮回灯 | 残魂、轮回、魂力 |
| 血 | 太虚舟、百族真灵线 | 真灵血脉、族群源血 |
| 运 | 司命盘 | 气运、命势、时代潮流 |
| 法 | 万法碑 | 功法、法术、道统传承 |
| 界 | 镇界印 | 世界本源、界壁、地脉 |
| 时 | 太虚舟 | 时痕、世界流速、古今残影，只做规则线，不做前期战斗外挂 |
| 道 | 斩道剑 | 大道规则、腐朽道统、断道重开 |

时间能力风险极高。前中期禁止把“时”写成随意逆转时间。

## 10. 新循环防腐机制

终局重炼后的仙界必须避免再次腐烂。

三道防线：

1. 新司命盘只记录危机和命数裂痕，不强制安排命运。
2. 新万法碑开放道统，高层不能垄断知识和功法。
3. 新斩道剑成为制度自毁机制，当某条大道、某个机构、某个仙职固化成压迫秩序时，可以斩断其权柄。

## 11. 重炼仙界时的底层保护

赵长生不是无差别毁灭仙界。

终局处理原则：

- 先打开新登仙台和归墟炉心，给底层飞升者、被压迫仙人、旧天庭遗民选择。
- 愿意入新循环者，可转入道职、回归万界或守炉补天。
- 深度绑定收割制度、靠墟眼续命的权贵，随旧秩序崩解而亡。
- 部分仙界区域会坠入归墟重炼，但不是无差别屠灭。
