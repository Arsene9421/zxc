## 1. 卷目标

第一卷不解释大世界观。

目标是让读者相信：

- 赵长生够穷。
- 修仙界够狠。
- 归墟炉够爽。
- 赵长生够谨慎。
- 沈临风的系统有趣但不对劲。
- 青石坊市待不下去了。

## 2. 表层主线

赵长生在青石坊市丹铺做杂役，被克扣工钱，只能靠发霉灵米和废丹渣过活。

他清理废炉灰时捡到残破归墟炉。

他开始谨慎试验归墟炉能力。

沈临风穿越并绑定天命系统，系统新手任务指向青石坊市，并提示他接近赵长生。

两人互相试探。

赵长生因出售异常资源被盯上。

沈临风系统任务把他卷入危局。

两人合作，赵长生第一次杀人，带着归墟炉逃离青石坊市。

## 3. 后台暗线

第一卷埋四条线：

1. 世界正在变穷。
2. 飞升和天劫不正常。
3. 赵长生命数有缺。
4. 沈临风系统不是纯善外挂。

## 4. 必须出现的细节

世界变穷：

- 废丹渣称重。
- 发霉灵米当工钱。
- 灵井水变淡。
- 残器涨价。
- 上宗收灵灰、废器、残玉简。

赵长生旧事：

- 杂役契写“青槐灾民赵长生”。
- 掌柜骂他是死人堆里捡回来的命。
- 黑雨或苦井水触发短暂失神。
- 胸口旧烙痕在归墟炉启动时发热。

系统异常：

- 系统显示赵长生为“关键人物”。
- 不显示赵长生气运。
- 问赵长生身份，回复“权限不足”。
- 卷尾报错：“司命残录校验失败。”

## 6. 第一卷反派

| 反派 | 作用 |
|------|------|
| 丹铺掌柜周槐安 | 压榨赵长生，引出废丹和废炉灰 |
| 散修 | 盯上异常丹药或灵草 |
| 上宗外门管事蒋怀礼 | 废料上缴体系下沉 |
| 黑市牙人裴九 | 残器和消息渠道 |
| 小帮派头目 | 给第一次杀人制造压力 |

## 6.1 第一卷主要动线

赵长生日常：

> 破屋 → 灵井 → 回春丹铺后院 → 废料场 → 破屋

沈临风入场：

> 东门 → 青石主街 → 系统提示 → 回春丹铺 → 后院

卷尾逃亡：

> 回春丹铺后巷 → 黑市巷 → 废料场 → 废料车道 → 北小门 → 枯石岭外坡 → 乱石药径 → 断溪谷外围 → 废灵脉支沟

详细空间规则见：

`01设定库/02_地理势力/02_地图地理/03_第一卷逃亡路线.md`

## 6.2 第一卷修为边界

| 人物或敌人 | 修为建议 |
|------------|----------|
| 赵长生开局 | 炼气一层巅峰或炼气二层初 |
| 赵长生卷尾 | 炼气二层稳定，触及炼气三层门槛 |
| 沈临风开局 | 炼气二层，系统奖励后有炼气三层保命能力 |
| 丹铺掌柜周槐安 | 炼气三层 |
| 小帮派头目 | 炼气四层 |
| 外门管事蒋怀礼 | 炼气五层 |

赵长生第一次杀人不能靠境界碾压，必须靠布置、地形、残器、沈临风信息差和敌人轻视。

## 7. 卷尾

卷尾达成：

1. 赵长生第一次杀人。
2. 赵长生和沈临风从互相试探变成临时生死合作者。
3. 归墟炉吸收残炉碎片。
4. 炉壁浮现：

> 万物归墟，仙门已断。

5. 沈临风系统短暂报错：

> 【司命残录校验失败。】  
> 【发现未知残源。】  
> 【权限不足。】

随后系统强行恢复：

> 【支线任务完成：逃离青石坊市。】

## 8. 第一卷不能写

- 旧天庭九器全貌。
- 第一次归墟大潮。
- 新仙界墟眼。
- 补墟司。
- 司命台真实来源。
- 沈临风穿越原因。
- 赵长生命簿已死。
- 无名老修身份。
