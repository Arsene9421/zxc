# 段纲：第 11-20 章 市场风险与系统入场

> 本段目标：接住第 8-10 章旧货街和裴九入场后的风险，把“异常残物”从小买卖压成尾随、查账和消息扩散；同时让沈临风以“爽文系统视角”入场，再让系统在赵长生身上失灵。

---

## 第 11 章：枯种与死虫

| 项目 | 内容 |
|------|------|
| 章节功能 | 第八次归墟炉试验，明确不能复活已死生命 |
| 主要场景 | 破屋、棚区、灵井边 |
| 出场人物 | 赵长生、棚区少年、老散修 |
| 当前冲突 | 赵长生想知道归墟炉是否能处理生命残缺，但不敢试人和活物 |
| 信息增量 | 枯种能发芽，死虫无效；归墟炉恢复的是残余生机，不是复活 |
| 爽点/压迫 | 爽点是枯种冒芽；压迫是边界明确，赵长生不能把它当起死回生神器 |
| 代价 | 枯芽很弱，离土即枯；死虫入炉只留下冷灰，炉火还消耗一分 |
| 章尾钩子 | 棚区少年问他屋里是不是种了灵草，赵长生第一次考虑搬走或加固屋门 |

执行细节：

1. 枯种可以来自灵米铺扫出的坏谷，死虫来自棚区墙角。
2. 赵长生先试枯种，再试死虫，顺序体现谨慎。
3. 死虫无效时要写赵长生松一口气又有点发寒：这东西不是万能。
4. 棚区少年闻到土腥和淡灵气，形成生活层面的暴露风险。
5. 赵长生把“不可复死”写进小册，作为后续不乱用归墟炉的性格证据。

禁写提醒：

- 不复活生命。
- 不写归墟炉吞魂。

## 第 12 章：半真半假的买卖

| 项目 | 内容 |
|------|------|
| 章节功能 | 小额交易产生第一次明确尾随，爽点转风险 |
| 主要场景 | 旧货街、黑市巷、棚区回路 |
| 出场人物 | 赵长生、罗三缺、裴九、坊市散修 |
| 当前冲突 | 赵长生想出手修复后的低阶小物，却被买家和牙人同时盯上 |
| 信息增量 | 异常低阶资源一旦入市，底层散修会因几块灵石动杀心 |
| 爽点/压迫 | 爽点是赵长生终于换到一点逃亡资本；压迫是他被尾随到棚区 |
| 代价 | 为甩尾，他多绕一圈，耗费体力，还丢掉一件可用残物作假线索 |
| 章尾钩子 | 尾随散修没有追进门，却在巷口留下一个记号 |

执行细节：

1. 交易物可以是恢复一点灵性的裂符笔或低效丹，不要太值钱。
2. 赵长生故意说东西是从废料场捡来的，不承认自己能修。
3. 罗三缺半信半疑，裴九则更关心消息能卖给谁。
4. 尾随散修的目标是抢货源，不是替任何势力办事。
5. 赵长生第一次把废料场地形和棚区小路联系起来，为后续准备退路。

禁写提醒：

- 不让赵长生当场杀尾随者。
- 不让裴九直接揭穿归墟炉。

## 第 13 章：东门来的新人

| 项目 | 内容 |
|------|------|
| 章节功能 | 沈临风入场，用系统爽文视角对照赵长生的底层视角 |
| 主要场景 | 东门、青石主街、青云办事处外 |
| 出场人物 | 沈临风、巡市修士、青云办事处杂役、路人散修 |
| 当前冲突 | 沈临风带着系统任务进入青石坊市，却发现这里不是游戏新手村 |
| 信息增量 | 系统发布新手任务：抵达青石坊市；可显示普通散修气运和低阶任务提示 |
| 爽点/压迫 | 爽点是系统奖励小额资源；压迫是入城、通行牌、摊税都要钱 |
| 代价 | 沈临风花钱比赵长生粗，刚入城就被收通行费和带路费 |
| 章尾钩子 | 系统刷新支线：【接近关键人物：赵长生。】 |

执行细节：

1. 沈临风视角可以有轻微现代人误判：以为有系统就能顺利升级。
2. 系统给的小额奖励不能太强，最多低阶护身符、小额灵石或基础小术熟练。
3. 东门只用于入场，不能成为卷尾逃亡路线。
4. 让沈临风看到青石主街和内圈秩序，和赵长生外圈生活形成反差。
5. 系统第一次提赵长生只给“关键人物”，不要给身份解释。

禁写提醒：

- 不解释沈临风穿越原因。
- 不出现新司命台后台字样。

## 第 14 章：关键人物

| 项目 | 内容 |
|------|------|
| 章节功能 | 赵沈第一次远距离交汇，系统读取赵长生失败 |
| 主要场景 | 回春丹铺后院、青石主街转角 |
| 出场人物 | 赵长生、沈临风、周槐安、丹铺小伙计 |
| 当前冲突 | 沈临风想按系统提示接近赵长生，却看到赵长生只是被骂的杂役 |
| 信息增量 | 系统能读普通修士气运，却对赵长生显示“气运评级：读取失败” |
| 爽点/压迫 | 爽点是系统第一次失灵，制造悬念；压迫是赵长生此时仍被周槐安压榨 |
| 代价 | 沈临风为了观察赵长生，在丹铺买了一枚不划算的低阶丹 |
| 章尾钩子 | 沈临风问系统“赵长生是谁”，系统回复：【权限不足。】 |

执行细节：

1. 赵长生在沈临风眼里不能显得神秘潇洒，而是袖口沾灰、脸色疲惫。
2. 系统读周槐安可以显示普通气运或“灰色小利”，读赵长生失败。
3. 沈临风不要立刻把赵长生当兄弟，要先把他当“任务目标”。
4. 赵长生注意到沈临风看自己的眼神太直，开始警惕。
5. 章尾系统权限不足，给沈临风第一次不安。

禁写提醒：

- 不显示“命簿已死之人”。
- 不让系统识别归墟炉。

## 第 15 章：他不像机缘

| 项目 | 内容 |
|------|------|
| 章节功能 | 赵沈第一次正式试探，确立两人不可能立刻互信 |
| 主要场景 | 灵井、旧货街边缘 |
| 出场人物 | 赵长生、沈临风、老散修、罗三缺 |
| 当前冲突 | 沈临风试图自然搭话，赵长生判断他不是普通散修 |
| 信息增量 | 沈临风对坊市底层规则不熟；赵长生对危险视线敏感 |
| 爽点/压迫 | 爽点是双主角智识碰撞；压迫是旁人也注意到二人接触 |
| 代价 | 沈临风说漏一些不合本地口风的话，赵长生更不信他 |
| 章尾钩子 | 系统提示：【目标周边存在异常干扰。】 |

执行细节：

1. 灵井处可让两人第二次靠近，沈临风用“顺手帮忙”失败，因为赵长生不接受来路不明善意。
2. 沈临风问得太像任务，赵长生反问他的通行牌和住处。
3. 老散修可以打断谈话，提一句“井水又苦了”，继续压世界变穷。
4. 罗三缺远远看见两人接触，把沈临风也纳入“可卖消息”。
5. 赵长生离开后换三次路线，沈临风第一次意识到目标比系统说的难接近。

禁写提醒：

- 不让二人互报底牌。
- 不让沈临风解释自己穿越。

## 第 16 章：权限不足

| 项目 | 内容 |
|------|------|
| 章节功能 | 深化系统异常，让沈临风从自信转为疑惑 |
| 主要场景 | 黑市巷、灵井旁小摊、沈临风临时住处 |
| 出场人物 | 沈临风、赵长生、裴九、两个黑市摊主 |
| 当前冲突 | 系统强推沈临风继续接近赵长生，但不给理由 |
| 信息增量 | 系统可以发布任务和危险提示，却无法回答赵长生身份 |
| 爽点/压迫 | 爽点是系统外挂出现裂缝；压迫是沈临风发现任务可能不是单纯机缘 |
| 代价 | 沈临风为了完成接触，花掉系统奖励的一部分灵石买消息 |
| 章尾钩子 | 裴九把沈临风也记进消息册：一个外来的、有钱但不熟规矩的新人 |

执行细节：

1. 系统可显示：【支线任务：接近赵长生。奖励：低阶护身符。】
2. 沈临风追问任务理由，系统只回：【权限不足。】
3. 裴九卖给沈临风的消息半真半假：赵长生是丹铺杂役、青槐灾民、常去废料场。
4. 赵长生在黑市巷远处发现沈临风问自己消息，警惕值上升。
5. 沈临风开始怀疑：赵长生不像机缘，更像一处系统也看不懂的坑。

禁写提醒：

- 不出现司命残录校验失败，留到卷尾。
- 不让沈临风获得完整地图或完整人物档案。

## 第 17 章：两个不对劲的人

| 项目 | 内容 |
|------|------|
| 章节功能 | 赵沈形成观察关系，外部查账压力启动 |
| 主要场景 | 旧货街、棚区、回春丹铺后院 |
| 出场人物 | 赵长生、沈临风、周槐安、丹铺小伙计、裴九远线 |
| 当前冲突 | 赵长生被多方视线盯上，沈临风也因打听赵长生而进入黑市消息网 |
| 信息增量 | 周槐安发现废料账缺口；裴九已经能把赵沈二人的消息串起来卖 |
| 爽点/压迫 | 爽点是两条主角线正式交叉；压迫是交叉本身引来更大风险 |
| 代价 | 赵长生被要求补交一批说不清的废料，沈临风被黑市当成可宰新人 |
| 章尾钩子 | 周槐安冷声问赵长生：“最近废灰少了三斤，你给我说说，少到哪里去了？” |

执行细节：

1. 让赵长生和沈临风各自意识到对方不正常，但都没有证据。
2. 赵长生可以误判沈临风是某个势力派来试探的人。
3. 沈临风可以误判赵长生是“隐藏任务 NPC”或大机缘守门人，但这种想法要被现实打脸。
4. 周槐安的查账要从小缺口开始，不是直接发现归墟炉。
5. 本章作为中段门槛，结尾把故事从“捡漏试验”推入“追索升级”。

禁写提醒：

- 不让两人结盟。
- 不让周槐安知道归墟炉。
- 不解释废料上缴最终流向。

## 第 18 章：废料账

| 项目 | 内容 |
|------|------|
| 章节功能 | 周槐安正式核废料数，丹铺压迫升级 |
| 主要场景 | 回春丹铺后院、账房、废灰缸 |
| 出场人物 | 赵长生、周槐安、丹铺小伙计、废料搬运散修 |
| 当前冲突 | 周槐安发现废灰和丹渣数量对不上，准备把缺口推给赵长生 |
| 信息增量 | 废料不是垃圾，上宗每月要收；掌柜私吞也怕上宗查账 |
| 爽点/压迫 | 压迫来自“主角捡漏空间正在关闭”；爽点是赵长生看出周槐安账本也不干净 |
| 代价 | 赵长生被罚夜里补清废料，还被扣下半月工钱 |
| 章尾钩子 | 周槐安让他签补赔字据，若赔不出就交给青云办事处 |

执行细节：

1. 周槐安查账不是为了正义，而是怕蒋怀礼或青云办事处追查。
2. 赵长生不能直接指出掌柜私吞，只能记住账册里前后不合的数字。
3. 小伙计趁机搜赵长生小柴房，但搜不到归墟炉。
4. 废料搬运散修提到“青云门这月收得更急”，把正式压力前置。
5. 章尾让赵长生意识到留在丹铺会被合法栽赃。

禁写提醒：

- 不解释废料最终接补墟体系。
- 不让周槐安发现归墟炉。

## 第 19 章：消息卖两遍

| 项目 | 内容 |
|------|------|
| 章节功能 | 裴九推动黑市消息扩散，让追索不再只在丹铺内部 |
| 主要场景 | 黑市巷、旧货街后墙、小帮派棚屋 |
| 出场人物 | 裴九、坊市散修、小帮派副手、小帮派头目远线、罗三缺 |
| 当前冲突 | 裴九把赵长生异常交易拆开卖给不同买家，各方动机开始合流 |
| 信息增量 | 黑市消息不是一次性传递，而是分层加价、半真半假地扩散 |
| 爽点/压迫 | 压迫来自主角不在场时敌人也行动；爽点是敌人动机具体，不靠大阴谋 |
| 代价 | 赵长生的住处、常走路线、旧货街交易习惯被人拼出半张图 |
| 章尾钩子 | 小帮派决定先不报周槐安，自己去摸赵长生的货源 |

执行细节：

1. 裴九可以把同一条消息卖两遍：卖给散修的是“杂役有异常残器”，卖给小帮派的是“废料场可能有私货”。
2. 罗三缺补充赵长生挑货习惯，但不要让他完全恶毒，只是贪小利。
3. 小帮派动机是抢资源和控制棚区，不是替上宗办案。
4. 这一章可以短暂离开赵长生视角，让读者知道危险正在靠近。
5. 结尾要让小帮派头目判断赵长生“胆小、好欺、没人撑腰”。

禁写提醒：

- 不写幕后仙界势力。
- 不让裴九掌握完整真相。

## 第 20 章：身后脚步

| 项目 | 内容 |
|------|------|
| 章节功能 | 第一次明确尾随危机，展示赵长生的反跟踪能力 |
| 主要场景 | 旧货街、棚区、灵井、废料场外围 |
| 出场人物 | 赵长生、坊市散修、沈临风远处观察、棚区少年 |
| 当前冲突 | 散修跟踪赵长生，想确认他的残物来源 |
| 信息增量 | 底层散修为几块灵石可以动杀心；棚区不是安全住处 |
| 爽点/压迫 | 爽点是赵长生靠熟路甩开尾随；压迫是他必须消耗一次低阶手段 |
| 代价 | 他用掉一张旧扰目符或一撮炉灰，且丢掉一个假包裹作诱饵 |
| 章尾钩子 | 沈临风看见赵长生不是普通杂役，系统却仍只能提示“异常干扰” |

执行细节：

1. 赵长生通过水坑倒影、摊贩反应、脚步节奏判断被跟。
2. 甩尾路线用旧货街 -> 灵井 -> 棚区 -> 废料场外围。
3. 尾随散修不能太蠢，他只是轻敌，认为杂役不敢反抗。
4. 沈临风远处看见赵长生绕路，不插手，先观察。
5. 赵长生回破屋后加门闩、藏米、换藏炉位置，增强危机感。

禁写提醒：

- 不让赵长生杀散修。
- 不让沈临风系统给完整敌人名单。
