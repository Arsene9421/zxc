# 段纲：第 01-10 章 开局压迫与归墟炉初试

> 本段目标：先让读者相信赵长生活得艰难，再让归墟炉以“小、慢、贵、危险”的方式带来第一层爽点。不要急着展开大世界观。

---

## 内容审计补丁：第 9-10 章追读节奏

第 1-8 章已经完成底层冷感、归墟炉边界和旧货街消息风险，但情绪兑现仍偏少。第 9-10 章不能只继续证明“更穷、更危险”，必须在不破坏凡人流压力的前提下给一次微兑现。

执行原则：

1. 第 9 章的兑现不是买到聚气丹、突破或发财，而是赵长生靠谨慎、丹铺经验和市场观察，避开一次坑价或假丹，并拿到一件后续可用的小机会。
2. 第 9 章尽量减少夜间试炉和记录感，重点写公开场景里的价格、假货、试探和被记脸。
3. 第 10 章的裴九不是黑市规则讲解员，而是第一个真正测试赵长生信息边界的人。
4. 罗三缺可以继续聪明，但只能怀疑“土法、货源、屋里有好灶或懂得养残器”，不能逼近黑炉、归墟炉、炉灰、破玉简残字或真实机制。
5. 第 9-10 章的读者收获应是“赵长生没有赢大局，但确实赢下一手判断”，让第 11 章再回到归墟炉生命边界试验时不显得连续流水账。

## 第 1 章：账上无错

| 项目 | 内容 |
|------|------|
| 章节功能 | 合并完成丹铺压迫、废料责任、黑炉发现和第一次最低限度验证 |
| 主要场景 | 回春丹铺前堂、后院、旧灰缸、小柴房 |
| 出场人物 | 赵长生、周槐安、丹铺小伙计、买丹散修 |
| 当前冲突 | 账面无错但到手全是霉米和丹渣；废灰责任逼赵长生夜里翻旧灰缸 |
| 信息增量 | 发霉灵米抵工钱、青槐灾民契、废料入账、黑炉与胸口旧烙痕轻微反应 |
| 爽点/压迫 | 小爽点来自废丹碎末误入后霉味退一圈；压迫来自“不成丹、不可食、不可示人” |
| 代价 | 袖口被旧灰燎破，黑炉不能交、不能卖、不能丢，气味外泄和废料账都有风险 |
| 章尾钩子 | 赵长生记录黑炉结果，确认废物能回一点药性，但不能立刻改变处境 |

执行细节：

1. 不用空泛句概括苦难，所有压迫落到账册、霉米、废料木牌、旧契和灰袋责任上。
2. 黑炉发现来自清旧灰责任，不是主动寻宝。
3. 第一次异常来自废丹筛潮碎末误入炉口，不是主角主动科研。
4. 赵长生后续验证只用最小碎丹角，结论必须是“不成丹，不可食，不可示人”。
5. 不写炉灵、认主、天地异象或突破。

## 第 7 章：炉灰有热

| 项目 | 内容 |
|------|------|
| 章节功能 | 第七次试验炉灰，建立炉灰价值和腐蚀风险，并让废料账危机成形 |
| 主要场景 | 废料场、破屋、回春丹铺后院 |
| 出场人物 | 赵长生、周槐安、丹铺小伙计、废料场搬运散修 |
| 当前冲突 | 赵长生想试炉灰，但废料场开始重称废料，拿灰也变危险 |
| 信息增量 | 废料流向：丹铺/器铺/符铺 -> 废料场 -> 青云办事处 -> 青云门库房 |
| 爽点/压迫 | 炉灰炼出温热黑砂，但核心功能是一次性遮气、灼手或腐蚀绳结；压迫是黑砂灼伤手背、腐蚀布袋、留下灰痕，废料账也开始对不上 |
| 代价 | 手背被黑砂灼出红痕，藏灰布袋被烧穿一角 |
| 章尾钩子 | 周槐安盯着废灰缸空缺，第一次追问清灰责任 |

执行细节：

1. 废料场要第一次正式出现：灰坑、残器铁片、破木车棚、旧灰井。
2. 赵长生熟悉地形的细节从这里开始埋，为后面第一次杀人服务。
3. 黑砂只能作为后续遮气、灼手、腐蚀绳结或扰乱的小工具，不是毒杀神器、稳定隐身粉或清场手段。
4. 废料搬运散修提到青云办事处要查废料，埋蒋怀礼后续入场。
5. 本章末把第一阶段爽点转成风险：归墟炉越用，账越危险。

禁写提醒：

- 不解释炉灰与旧天庭残炉关系。
- 不让黑砂直接成为强力法宝。
- 不解释登仙台、无尸骨或飞升真相。

## 第 9 章：不够买一枚聚气丹

| 项目 | 内容 |
|------|------|
| 章节功能 | 用钱细节压实贫穷，同时给一次微兑现：赵长生买不起聚气丹，却靠丹铺经验和谨慎避开假废丹或坑价，拿到一件后续可用的小机会 |
| 主要场景 | 破屋、回春丹铺前堂或门外、灵米铺、旧货街边缘、灵井回路 |
| 出场人物 | 赵长生、灵米铺老板、丹铺散修、兜售废丹的低阶散修、旁观散修 |
| 当前冲突 | 赵长生想确认自己离正经修炼资源还有多远，但市场给穷人的“便宜丹药”多半是坑 |
| 信息增量 | 完整聚气丹仍是一块下品灵石起；废聚气丹也会被炒到十五到二十碎灵；灵米铺扫出的坏谷和丹药残包都可能成为后续低阶试验材料 |
| 爽点/压迫 | 爽点是赵长生避开一次假废丹或坑价，保住碎灵并看清一个市场规则；压迫是他仍不够买聚气丹，且因为懂行被散修记脸 |
| 代价 | 他不能公开揭穿假货，只能低头避开；懂得太多反而让人记住“回春丹铺小杂役会看丹” |
| 章尾钩子 | 旁观散修记住赵长生的脸：这个穷杂役买不起丹，却看得出一点丹药真假 |

执行细节：

1. 本章不安排新的归墟炉试验，避免连续“白天受压、夜里试炉、写记录”的固定骨架。
2. 赵长生可以看见完整聚气丹价格，也可以听见废聚气丹涨价，但不能买到完整聚气丹。
3. 微兑现必须具体：避开假丹、识破短秤、用极少碎灵换到坏谷或低风险残包，三者选一到两项即可，不要堆成好运章。
4. 坏谷若出现，只作为第 11 章枯种试验的前置材料，不在本章入炉。
5. 旁观散修记住他，是因为他“会看丹、买东西太谨慎、穷却不容易骗”，不是因为看出归墟炉。
6. 章尾压力落在市场盯梢上，不落在周槐安查账上，避免连续账压同质化。

## 第 10 章：裴九的竹签

| 项目 | 内容 |
|------|------|
| 章节功能 | 裴九入场，黑市规则和消息价格出现；通过一次当面试探，把旧货风险升级成消息风险 |
| 主要场景 | 黑市巷入口 |
| 出场人物 | 赵长生、裴九、散修消息贩子 |
| 当前冲突 | 裴九想确认赵长生是否有稳定残物来源，赵长生想知道消息网怎么记人但不能露底 |
| 信息增量 | 黑市竹签不是福利入口，而是记账、传话和定价的工具；消息可以比货更贵 |
| 爽点/压迫 | 爽点是赵长生看出裴九在用竹签试探价码，故意只留下低价值、可误读的信息；压迫是裴九仍判断他值得继续盯 |
| 代价 | 赵长生不能完全拒绝黑市线，否则会显得心虚；他只能用贫穷、丹铺杂役身份和残物货源假象遮住真实机制 |
| 章尾钩子 | 裴九判断赵长生可能有稳定残物来源，但误判方向在“丹铺边角料、废料场私货或土法养残器” |

执行细节：

1. 裴九可以聪明、会压价、会问错开口，但不能识破归墟炉。
2. 赵长生与裴九的交锋要有人对人张力，不写成黑市说明书。
3. 罗三缺传来的“好灶”口风必须被限定为市井猜测：烘货、养残器、丹铺偷料门路，不升级为炉类真相。
4. 赵长生可以主动丢出一个低价值假象，例如“废料场边角料”或“丹铺旧笔泡墨法”，让消息线朝错误方向走。
5. 章尾不是赵长生获得黑市靠山，而是黑市正式把他列入可持续观察对象。
