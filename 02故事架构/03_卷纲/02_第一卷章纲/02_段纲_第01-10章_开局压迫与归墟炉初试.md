# 段纲：第 01-10 章 开局压迫与归墟炉初试

> 本段目标：先让读者相信赵长生活得艰难，再让归墟炉以“小、慢、贵、危险”的方式带来第一层爽点。不要急着展开大世界观。

---

## 第 1 章：账上无错

> 章名已定为《账上无错》。本章核心物件仍是发霉灵米、丹渣和罚扣账。

| 项目 | 内容 |
|------|------|
| 章节功能 | 开局压迫，建立赵长生是回春丹铺底层杂役，不是隐藏少爷 |
| 主要场景 | 回春丹铺前堂、后院、废丹称重处 |
| 出场人物 | 赵长生、周槐安、丹铺小伙计、买丹散修 |
| 当前冲突 | 周槐安用发霉灵米和丹渣抵工钱，还反扣赵长生炉灰损耗 |
| 信息增量 | 青石坊市底层用碎灵计价；丹渣、废炉灰都要称重入账 |
| 爽点/压迫 | 压迫点来自“连废料都不是自己的”；爽点暂不释放，只埋赵长生会算账、会忍 |
| 代价 | 赵长生实际到手不足一块下品灵石价值，且大半是霉米和碎灵 |
| 章尾钩子 | 周槐安嫌废灰缸太满，让赵长生夜里去后院清废炉灰 |

执行细节：

1. 开篇不要解释世界观，用一杆秤、一袋霉灵米、一张罚扣账把贫穷写实。
2. 周槐安不是大反派式张狂，而是精明、吝啬、会用账册压人。
3. 赵长生不顶嘴，但会在心里把每一笔罚扣记清楚。
4. 买丹散修随口嫌聚气丹涨价，埋“底层资源变贵”。
5. 丹铺小伙计可嘲笑赵长生“青槐灾民”，但不要展开青槐真相。

禁写提醒：

- 不写归墟炉。
- 不写旧天庭、新仙界、补墟司。
- 不让赵长生当场反杀或热血爆发。

## 第 2 章：青槐灾民

| 项目 | 内容 |
|------|------|
| 章节功能 | 投放赵长生旧事回声，展示灵井水变淡和坊市底层日常 |
| 主要场景 | 坊市灵井、杂役契、小柴房 |
| 出场人物 | 赵长生、老散修、丹铺小伙计、周槐安 |
| 当前冲突 | 赵长生取水排队被挤，被嘲“灾民也配抢灵井水” |
| 信息增量 | 灵井水近年变涩、变淡；杂役契上写“青槐灾民赵长生” |
| 爽点/压迫 | 压迫点来自身份和资源双重低位；赵长生用沉默保住半桶井水 |
| 代价 | 井水洒了一半，赵长生回去还被罚晚到 |
| 章尾钩子 | 周槐安翻出杂役契，逼他签下“废灰清理无损耗”字据 |

执行细节：

1. 老散修抱怨“十年前井水还能养活灵草，现在一缸水连符纸都润不开”，埋世界变穷。
2. 赵长生看到井水短暂发黑或发苦，出现黑雨、苦井水、青槐树影的碎片失神。
3. 失神必须短，不解释原因。
4. 周槐安用契书压他：杂役契不是自由雇佣，而是能合法扣押底层人的工具。
5. 小柴房里赵长生检查自己身上只有碎灵和霉米，把经济底线定住。

禁写提醒：

- 不说赵长生命簿已死。
- 不写青槐村旧址。
- 不让无名老修正式出场。

## 第 3 章：废灰缸里的黑炉

| 项目 | 内容 |
|------|------|
| 章节功能 | 归墟炉入手，但保持废物感和危险感 |
| 主要场景 | 回春丹铺后院、废灰缸、小柴房 |
| 出场人物 | 赵长生、丹铺小伙计、夜间巡市修士远影 |
| 当前冲突 | 赵长生被迫独自清废灰，废灰缸里有腐蚀性热灰 |
| 信息增量 | 废料要上缴；周槐安私下掉包一部分废料，形成捡漏空间 |
| 爽点/压迫 | 爽点是赵长生在废物堆里发现异常；压迫是他不敢声张 |
| 代价 | 手掌被热灰烫伤，袖口被腐蚀出洞 |
| 章尾钩子 | 巴掌大小的三足缺一黑炉被灰埋着，赵长生胸口旧烙痕忽然发热 |

执行细节：

1. 黑炉第一眼不像法宝，像一块无灵光的废铁。
2. 赵长生不是“认主成功”，而是觉得这东西不对劲，先藏进灰袋。
3. 废灰缸旁可出现一块残铜或旧炉片，但不解释来历。
4. 巡市修士远处经过，让赵长生明白废料也不是随便拿的。
5. 小伙计偷懒把最脏的活推给赵长生，形成合理独处机会。

禁写提醒：

- 不写“旧天庭循环九器之一”。
- 不让归墟炉说话。
- 不出现认主光柱、天地异象。

## 第 4 章：第一缕药香

| 项目 | 内容 |
|------|------|
| 章节功能 | 完成归墟炉第一次试验，确认它能重炼残缺药性 |
| 主要场景 | 赵长生破屋 |
| 出场人物 | 赵长生 |
| 当前冲突 | 赵长生不敢确定黑炉是机缘还是祸根，只能用最低价值的发霉碎丹试 |
| 信息增量 | 归墟炉能让发霉碎丹退去霉斑，出现一丝药香，但不是完整丹药 |
| 爽点/压迫 | 小爽点：废物变得可用；压迫：效果太小，不足以改变处境 |
| 代价 | 炉火微弱闪动后变暗，屋内留下难闻灰味，赵长生必须清理痕迹 |
| 章尾钩子 | 赵长生把结果、时间、投入物和风险记入破纸册，决定再试一次 |

执行细节：

1. 赵长生先检查门闩、窗缝、墙根，再试炉，突出谨慎。
2. 发霉碎丹只能恢复一丝药香，不能成为完整聚气丹。
3. 他不立刻吃，先用针尖刮下一点试药性。
4. 旧烙痕在炉火亮起时微热，赵长生以为是烫伤牵动旧伤。
5. 这章的爽点要“克制”，让读者知道外挂好用但不爆炸。

禁写提醒：

- 不写炉灵。
- 不写立刻突破。
- 不写丹药无毒无害。

## 第 5 章：一斤能吃的灵米

| 项目 | 内容 |
|------|------|
| 章节功能 | 第二次试验发霉灵米，证明归墟炉能解决一点生存问题 |
| 主要场景 | 破屋、灵米铺外、棚区 |
| 出场人物 | 赵长生、灵米铺老板、棚区少年 |
| 当前冲突 | 赵长生想吃一顿不霉的灵米，但又怕消耗炉火和暴露异常 |
| 信息增量 | 普通灵米 5-8 碎灵一斤，好灵米更贵；底层修士常吃发霉灵米 |
| 爽点/压迫 | 爽点是霉米变得可食用；压迫是只够吃，不够卖，不够修炼 |
| 代价 | 一斤霉米处理完只剩七八成，灵气很弱，炉火又暗一分 |
| 章尾钩子 | 棚区少年闻到米香，赵长生意识到连一顿饭都可能引来视线 |

执行细节：

1. 灵米铺老板可以顺口说“青云办事处又要涨摊费”，让物价上涨有现实压力。
2. 赵长生用碎灵买不起好米，只能问价后离开。
3. 霉米重炼后不是灵食大餐，只是终于不刮嗓子。
4. 棚区少年出现时，赵长生没有大方分米，只是把锅盖压紧，体现底层真实。
5. 章尾让赵长生开始把试验安排在更深夜。

禁写提醒：

- 不让赵长生靠灵米暴富。
- 不写温情施舍抢主线。

## 第 6 章：枯叶回青

| 项目 | 内容 |
|------|------|
| 章节功能 | 第三次试验枯黄灵草叶，明确归墟炉只能恢复残余生机 |
| 主要场景 | 丹铺后院、破屋 |
| 出场人物 | 赵长生、周槐安、丹铺小伙计 |
| 当前冲突 | 周槐安盘点药渣，怀疑有人偷拿边角料 |
| 信息增量 | 灵草年份越来越低，丹铺把枯叶也算入废料账 |
| 爽点/压迫 | 爽点是枯叶重新泛青；压迫是周槐安开始盯废料数量 |
| 代价 | 泛青只维持一段时间，药性弱，不能当成完整灵草卖 |
| 章尾钩子 | 周槐安让小伙计重称废料，赵长生发现账册对自己越来越危险 |

执行细节：

1. 枯叶来源要合理：清药筛时混进袖中，或小伙计丢在废料桶。
2. 归墟炉让叶脉回青，但根茎已死，不能复活整株。
3. 赵长生尝试把枯叶分成两半，确认投入越少收益越小。
4. 周槐安在同章开始计较废料缺斤，形成“试验越多越危险”。
5. 赵长生把可卖和不可卖分开记，体现不莽。

禁写提醒：

- 不写高年份灵草。
- 不写归墟炉能无条件复活植物。

## 第 7 章：半粒废丹

| 项目 | 内容 |
|------|------|
| 章节功能 | 第四次试验丹渣，给第一层修炼爽点，同时加入丹毒风险 |
| 主要场景 | 小柴房、破屋、灵井边 |
| 出场人物 | 赵长生、老散修、丹铺小伙计 |
| 当前冲突 | 赵长生想用低效丹丸修炼，但怕药性不稳害死自己 |
| 信息增量 | 完整聚气丹 1-2 下品灵石，废聚气丹也值 10-20 碎灵 |
| 爽点/压迫 | 爽点是丹渣凝出半粒低效丹丸；压迫是这点资源仍要冒毒性风险 |
| 代价 | 服用后经脉刺痛，修炼只推进一小截，第二天脸色发青 |
| 章尾钩子 | 老散修看出他像服过劣丹，提醒“便宜丹药吃多了，迟早堵死经脉” |

执行细节：

1. 半粒丹丸要有杂色、裂纹和药渣味，不像精品。
2. 赵长生先刮一点，再含一小块，不是一口吞。
3. 修炼时只写炼气一二层的小周天，不写大突破。
4. 丹毒让他第二天手指发麻，提醒读者资源有代价。
5. 老散修的提醒可顺带提一句“从前飞升的人吃丹都谨慎”，但不要展开飞升。

禁写提醒：

- 不让赵长生突破到炼气三层。
- 不让废丹变完美丹。

## 第 8 章：裂符笔

| 项目 | 内容 |
|------|------|
| 章节功能 | 第五次试验裂符笔，扩展归墟炉适用范围到器物 |
| 主要场景 | 旧货街边缘、罗三缺摊位、破屋 |
| 出场人物 | 赵长生、旧货摊主罗三缺、路过散修 |
| 当前冲突 | 赵长生想买最便宜残物试炉，但旧货街处处是骗局和视线 |
| 信息增量 | 裂纹符笔 20-40 碎灵，残器也涨价；旧货摊常用“古修遗物”骗散修 |
| 爽点/压迫 | 爽点是符笔笔锋恢复一点灵性；压迫是罗三缺记住他专挑残缺物 |
| 代价 | 花掉一笔碎灵，修好的符笔仍只能低价卖或自用一次 |
| 章尾钩子 | 罗三缺盯着他离开，嘀咕“这小杂役挑废物倒挺准” |

执行细节：

1. 旧货街不要一次展开全貌，只写摊位、叫卖、破玉简和残器堆。
2. 赵长生故意买几件更破的东西掩饰真正目标。
3. 符笔恢复后不可写成符器，只是笔锋能承一丝灵墨。
4. 他尝试画最简单的记号符，效果不稳。
5. 旧货摊主罗三缺不是朋友，是嘴碎的信息风险。

禁写提醒：

- 不让赵长生突然会制符。
- 不写旧货摊主看出归墟炉。

## 第 9 章：破玉简残字

| 项目 | 内容 |
|------|------|
| 章节功能 | 第六次试验破玉简，投放飞升异常和登仙台残字 |
| 主要场景 | 旧货街、破屋 |
| 出场人物 | 赵长生、罗三缺、老散修短暂路过 |
| 当前冲突 | 赵长生想确认归墟炉是否能恢复信息，但破玉简可能引来更大麻烦 |
| 信息增量 | 破玉简可能有残功、地图、旧闻；低阶修士常为残字被骗 |
| 爽点/压迫 | 爽点是残字多显出几行；压迫是内容看不全，反而更危险 |
| 代价 | 玉简裂纹加深，赵长生神识刺痛，归墟炉火短暂紊乱 |
| 章尾钩子 | 残字里出现“登仙台”“无尸骨”“断”几个碎词 |

执行细节：

1. 赵长生买破玉简时被罗三缺嘲笑“这种东西只能垫桌脚”。
2. 归墟炉不能补出完整功法，只让残字多显几行。
3. 赵长生读不懂“登仙台”真正含义，只以为是某个古地名。
4. 老散修可在旧货街醉骂“哪有什么飞升，去了也没人回来”，只作怪谈。
5. 章尾以赵长生合上玉简、后背发冷收束。

禁写提醒：

- 不解释飞升回收。
- 不写完整登仙台地图。

## 第 10 章：炉灰有热

| 项目 | 内容 |
|------|------|
| 章节功能 | 第七次试验炉灰，建立炉灰价值和腐蚀风险，并让废料账危机成形 |
| 主要场景 | 废料场、破屋、回春丹铺后院 |
| 出场人物 | 赵长生、周槐安、丹铺小伙计、废料场搬运散修 |
| 当前冲突 | 赵长生想试炉灰，但废料场开始重称废料，拿灰也变危险 |
| 信息增量 | 废料流向：丹铺/器铺/符铺 -> 废料场 -> 青云办事处 -> 青云门库房 |
| 爽点/压迫 | 爽点是炉灰炼出温热黑砂；压迫是黑砂腐蚀布袋，废料账也开始对不上 |
| 代价 | 赵长生手背被黑砂灼出红痕，藏灰布袋被烧穿一角 |
| 章尾钩子 | 周槐安盯着废灰缸空缺，第一次问：“昨夜是谁清的灰？” |

执行细节：

1. 废料场要第一次正式出现：灰坑、残器铁片、破木车棚、旧灰井。
2. 赵长生熟悉地形的细节从这里开始埋，为后面第一次杀人服务。
3. 黑砂只能作为后续遮眼、腐蚀、扰乱的小工具，不是毒杀神器。
4. 废料搬运散修提到青云办事处要查废料，埋蒋怀礼后续入场。
5. 本章末把第一阶段爽点转成风险：归墟炉越用，账越危险。

禁写提醒：

- 不解释炉灰与旧天庭残炉关系。
- 不让黑砂直接成为强力法宝。
