# 段纲：第 21-30 章 正式追索与逃亡倒计时

> 本段目标：把前二十章的“异常资源”和“系统任务”压成具体危局。蒋怀礼正式介入废料账，周槐安栽赃成形，裴九继续卖消息，小帮派从试探转向约局，赵长生和沈临风不是因为信任合作，而是被同一张网逼到一起。

---

## 第 21 章：蒋怀礼入坊

| 项目 | 内容 |
|------|------|
| 章节功能 | 上宗外门管事蒋怀礼登场，私人纠纷升级为正式废料账压力 |
| 主要场景 | 青云办事处、回春丹铺前堂、账房 |
| 出场人物 | 蒋怀礼、周槐安、赵长生、巡市修士 |
| 当前冲突 | 蒋怀礼查废料上缴，周槐安想把账目缺口推给赵长生 |
| 信息增量 | 蒋怀礼炼气五层，正面不可敌；他有权查账、封货、传令门房 |
| 爽点/压迫 | 压迫来自制度层级；爽点是赵长生发现周槐安比自己更怕查账 |
| 代价 | 赵长生被列为“待问杂役”，行动受到限制 |
| 章尾钩子 | 蒋怀礼淡淡说：“今晚之前，把缺的废料补齐，或者把人交出来。” |

执行细节：

1. 蒋怀礼表面讲规矩，不需要凶狠；他的身份本身就是压力。
2. 周槐安在蒋怀礼面前低声下气，转头压赵长生。
3. 蒋怀礼不知道归墟炉，只关心废料、账册、回扣和责任。
4. 赵长生不能对蒋怀礼起杀心，只能判断“这个人正面碰不得”。
5. 本章让北小门、青云办事处和废料车道进入后续封锁逻辑。

禁写提醒：

- 不让蒋怀礼识破归墟炉。
- 不让赵长生正面挑衅炼气五层。

## 第 22 章：黑市巷支线

| 项目 | 内容 |
|------|------|
| 章节功能 | 系统强推沈临风介入赵长生危机，赵沈被同一危局捆住 |
| 主要场景 | 黑市巷、旧货街后墙、灵米铺后巷 |
| 出场人物 | 赵长生、沈临风、裴九、坊市散修 |
| 当前冲突 | 赵长生想通过黑市查谁在卖自己消息，沈临风也被系统任务推到同一地点 |
| 信息增量 | 系统可以提示危险方向，但不能给完整答案；裴九同时卖两人的消息 |
| 爽点/压迫 | 爽点是赵沈信息互补；压迫是互补本身暴露他们可能有关联 |
| 代价 | 沈临风为了救场用掉一次低阶护身符或系统小奖励 |
| 章尾钩子 | 小帮派看到两人同路，判断赵长生背后可能真有外来同伙 |

执行细节：

1. 赵长生来黑市巷不是求助裴九，而是确认消息流向。
2. 沈临风系统提示“目标周边存在杀意”，但不说谁要杀。
3. 一次小冲突中，沈临风用护身符挡下试探，不是击败敌人。
4. 赵长生看出沈临风有外挂式保命手段，但不知道系统。
5. 两人第一次短暂配合后立刻分开，关系仍是互相防备。

禁写提醒：

- 不让沈临风一键解决黑市麻烦。
- 不让赵长生承认归墟炉。

## 第 23 章：棚区勒索

| 项目 | 内容 |
|------|------|
| 章节功能 | 小帮派第一次正面施压，赵长生住处不再安全 |
| 主要场景 | 棚区、赵长生破屋、排水沟旁 |
| 出场人物 | 赵长生、小帮派副手、棚区少年、沈临风远线 |
| 当前冲突 | 小帮派上门收“棚区保护费”，实际搜赵长生私货 |
| 信息增量 | 外圈棚区由小帮派控制，死个散修未必有人查 |
| 爽点/压迫 | 压迫是主角最后的破屋安全感被打碎；爽点是赵长生提前藏走关键物 |
| 代价 | 破屋被翻乱，藏粮被踩，赵长生丢失一部分碎灵和旧衣 |
| 章尾钩子 | 小帮派副手留话：明夜废料场，把货源交出来 |

执行细节：

1. 小帮派不说“归墟炉”，只说“你从废料场摸了什么好东西”。
2. 赵长生在破屋布的小陷阱只能报警或拖延，不能伤杀敌人。
3. 棚区少年看到一切，不敢帮，只能躲，体现底层现实。
4. 沈临风从系统任务得知赵长生危险，但赶到时只看到残局。
5. 赵长生决定不能再把破屋当长期据点。

禁写提醒：

- 不让赵长生在棚区大打出手。
- 不写帮派知道残炉。

## 第 24 章：旧灰井

| 项目 | 内容 |
|------|------|
| 章节功能 | 赵长生准备废料场退路，为第一次杀人铺地形 |
| 主要场景 | 废料场、废灰坑、破木车棚、旧灰井 |
| 出场人物 | 赵长生、废料搬运散修、沈临风暗中跟随 |
| 当前冲突 | 赵长生必须去废料场补工，同时评估可能被堵截的路线 |
| 信息增量 | 废料场有灰坑、残器铁片、破木车、旧灰井、废料车道 |
| 爽点/压迫 | 爽点是赵长生把熟悉劳动地形转化为保命准备；压迫是敌人也可能在这里堵他 |
| 代价 | 他冒险移动一些残器和灰袋，留下被发现的可能 |
| 章尾钩子 | 沈临风看出赵长生不像在干活，更像在给自己留逃命路线 |

执行细节：

1. 赵长生不布大型陷阱，只做低阶、合理、劳动痕迹能解释的小调整。
2. 例如记住哪处灰坑表面结壳、哪片残器锋利、哪辆破木车轮轴松。
3. 沈临风暗中观察，不完全理解，但系统提示“环境危险度上升”。
4. 废料搬运散修提到北小门夜里常放废料车，给逃亡路线合理入口。
5. 赵长生把一包炉灰和残器铁片藏到车棚角落。

禁写提醒：

- 不布置成熟阵法。
- 不让赵长生像职业杀手一样完美预案。

## 第 25 章：危险方向

| 项目 | 内容 |
|------|------|
| 章节功能 | 沈临风提供一次有效但有限的系统提示，促成有限合作 |
| 主要场景 | 黑市巷、废料场边缘、灵井小路 |
| 出场人物 | 赵长生、沈临风、小帮派探子、裴九远线 |
| 当前冲突 | 小帮派探子试探赵长生路线，沈临风系统提示危险方向 |
| 信息增量 | 系统能感知杀意或危险方向，但无法确认完整敌人计划 |
| 爽点/压迫 | 爽点是赵沈第一次信息互补成功；压迫是系统越介入，沈临风越被卷入 |
| 代价 | 沈临风暴露自己能提前察觉危险，赵长生对他更警惕 |
| 章尾钩子 | 赵长生承认：“今晚以前，我们最好别分开太远。” |

执行细节：

1. 系统提示可以是：【危险提示：西北方向存在恶意锁定。】
2. 沈临风只知道方向，不知道小帮派人数和伏击地点。
3. 赵长生根据方向、地形和裴九消息判断废料场会出事。
4. 二人争论：沈临风想先报巡市，赵长生指出巡市会先查他们。
5. 形成临时约定：若夜里出事，在废料场边缘汇合。

禁写提醒：

- 不让系统给完整路线。
- 不让两人互相信任过快。

## 第 26 章：残剑寒光

| 项目 | 内容 |
|------|------|
| 章节功能 | 第九次归墟炉试验，准备一次性保命锋芒 |
| 主要场景 | 破屋残址、废料场小棚、夜色中的旧货街边 |
| 出场人物 | 赵长生、沈临风短暂在场或外守 |
| 当前冲突 | 赵长生需要一件能在关键时刻破局的残器，但归墟炉炉火已弱 |
| 信息增量 | 残剑碎片有宝气残余，重炼消耗明显高于废丹和灵米 |
| 爽点/压迫 | 爽点是残剑碎片生出一缕寒光；压迫是它只能用一次，且炉火几乎见底 |
| 代价 | 赵长生法力被抽空一截，旧烙痕发痛，归墟炉短时间不能继续频繁试验 |
| 章尾钩子 | 残剑寒光照出炉壁上更深的裂纹，赵长生意识到下一次可能会出大事 |

执行细节：

1. 残剑碎片来源可以是之前低价买下，也可以是废料场偷藏的边角残片。
2. 归墟炉只让碎片锋芒短暂恢复，不能变完整飞剑。
3. 沈临风如果在场，只能看到赵长生处理残器异常，不能看清归墟炉全貌。
4. 赵长生把残剑锋芒包在灰布里，作为最后保命手段。
5. 本章要明确：归墟炉不是无代价续航，连续使用后明显亏空。

禁写提醒：

- 不让残剑成为长期飞剑。
- 不让沈临风完全看懂归墟炉。

## 第 27 章：北小门将封

| 项目 | 内容 |
|------|------|
| 章节功能 | 正式封锁压力抬头，北小门时间差进入倒计时 |
| 主要场景 | 青云办事处、北小门口风、废料车道入口 |
| 出场人物 | 蒋怀礼、门房、巡市修士、赵长生、沈临风远线 |
| 当前冲突 | 蒋怀礼要求核查废料车和北小门出入，赵长生的逃亡窗口收窄 |
| 信息增量 | 北小门封锁来得慢，但一旦传令到位，赵沈就很难走 |
| 爽点/压迫 | 压迫是时间窗口收窄；爽点是赵长生从账、门、车道推演出唯一生路 |
| 代价 | 赵长生必须放弃继续观察，提前准备当夜离开 |
| 章尾钩子 | 门房收到口风：今晚后，废料车也要逐车核人 |

执行细节：

1. 蒋怀礼不瞬移追杀，他通过青云办事处、门房和巡市修士形成制度压力。
2. 门房不是绝对忠诚，但会记人、收钱、怕担责。
3. 赵长生判断北小门不是安全门，而是最后一个还没关死的窄缝。
4. 沈临风系统只能提示“北侧风险升高”，不能直接给路线。
5. 本章让卷尾逃亡路线的时间差成立。

禁写提醒：

- 不让蒋怀礼亲自守在每个节点。
- 不让系统给完整逃亡地图。

## 第 28 章：补赔字据

| 项目 | 内容 |
|------|------|
| 章节功能 | 周槐安栽赃成形，赵长生确认丹铺已无退路 |
| 主要场景 | 回春丹铺账房、后院、小柴房外 |
| 出场人物 | 赵长生、周槐安、丹铺小伙计、账房帮工 |
| 当前冲突 | 周槐安逼赵长生签废料缺口补赔字据，想把私吞和缺口都压到他身上 |
| 信息增量 | 字据一签，赵长生就会从“穷杂役”变成可合法押送的欠账人 |
| 爽点/压迫 | 爽点是赵长生看出字据漏洞和周槐安私账；压迫是他不能公开反咬 |
| 代价 | 赵长生拒签后被盯死，后巷和小柴房都有人守 |
| 章尾钩子 | 周槐安低声吩咐伙计：“今夜别让他出后院。” |

执行细节：

1. 字据内容要具体：废灰、丹渣、残器边角、补赔灵石数额。
2. 赵长生不能靠嘴赢，只能拖延、装傻、记住印章和账册漏洞。
3. 周槐安仍不知道归墟炉，只想甩锅和自保。
4. 小伙计可以搜身或堵门，但不能搜到核心物。
5. 本章把“当夜必须走”的压力落到法律和身份上。

禁写提醒：

- 不写查账辩论赢掌柜。
- 不让赵长生当场翻脸杀人。

## 第 29 章：废料场约局

| 项目 | 内容 |
|------|------|
| 章节功能 | 小帮派和消息线合流，废料场伏击方向明确 |
| 主要场景 | 棚区、黑市巷、废料场外围 |
| 出场人物 | 赵长生、沈临风、裴九、小帮派眼线、小帮派副手远线 |
| 当前冲突 | 小帮派传话逼赵长生夜里去废料场交货源，裴九继续倒卖半真半假消息 |
| 信息增量 | 废料场不只是劳动地点，也会成为敌人判断赵长生货源的地点 |
| 爽点/压迫 | 爽点是赵长生提前知道伏击方向；压迫是他明知危险也必须利用那条路 |
| 代价 | 为确认消息，赵沈又花掉碎灵和一次系统提示，且暴露两人可能有关联 |
| 章尾钩子 | 沈临风系统提示废料场方向杀意上升，但仍不给人数和具体布置 |

执行细节：

1. 赵长生不能相信裴九，只能利用他卖消息的习惯反推敌人位置。
2. 小帮派副手传话粗暴，但目标是抢货源，不是查大真相。
3. 沈临风想绕开废料场，赵长生指出北小门和废料车道必须经过这一带。
4. 两人第一次把各自掌握的信息合在一起，但仍不交底。
5. 章尾让读者明确：下一夜不是偶然遇袭，而是带着误判和反误导的必经局。

禁写提醒：

- 不让赵沈完全相信彼此。
- 不把裴九写成帮忙送情报。

## 第 30 章：今夜就走

| 项目 | 内容 |
|------|------|
| 章节功能 | 逃亡倒计时收束，正式进入夜逃和反杀前状态 |
| 主要场景 | 破屋、废料场边、回春丹铺后巷口 |
| 出场人物 | 赵长生、沈临风、周槐安远线、小帮派眼线、门房远线 |
| 当前冲突 | 赵长生必须在丹铺栽赃、帮派约局和北小门封锁同时压来前离开 |
| 信息增量 | 赵长生能带走的东西极少：碎灵、低效丹、残剑碎片、炉灰包、干粮、灰布 |
| 爽点/压迫 | 爽点是底层谨慎终于转成逃亡准备；压迫是他失去破屋、丹铺和坊市落脚点 |
| 代价 | 放弃大部分积攒物，带伤、缺钱、缺法力进入夜路 |
| 章尾钩子 | 赵长生告诉沈临风：“今晚走。再晚一步，门就不是门，是笼口。” |

执行细节：

1. 赵长生不能从容打包全部家当，必须做取舍。
2. 他把炉灰包、残剑寒光、低效丹、灰布和少量碎灵按逃亡用途分配。
3. 沈临风系统刷新支线任务：协助目标脱离青石坊市，但奖励含糊或异常高。
4. 两人约定只是临时同路，不是兄弟结盟。
5. 章尾必须明确：下一段进入后巷堵截、废料场伏击和第一次杀人。

禁写提醒：

- 不让两人安全离坊。
- 不让沈临风系统给传送或隐身能力。
