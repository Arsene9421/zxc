# 第 1 章单章详纲：账上无错

> 并章重构说明：本章由此前拆分的《账上无错》和《灰缸里的黑炉》合并而来。目标是在一章内完成“丹铺压迫 -> 废料责任 -> 旧灰缸发现黑炉 -> 废丹碎末误入触发异常 -> 最低限度确认”，不再把苦难铺陈拆成多章。

## 1. 本章定位

| 项目 | 内容 |
|------|------|
| 章节功能 | 建立赵长生的生存压力，并让归墟炉在第一章进入主线且完成第一次最低限度验证 |
| 主要场景 | 回春丹铺前堂、后院、旧灰缸、小柴房 |
| 出场人物 | 赵长生、周槐安、小伙计、买丹散修 |
| 当前冲突 | 账面无错但实物全是霉米和废渣，赵长生被迫承担旧灰责任，又发现一个不能交、不能丢、不能卖的黑炉 |
| 信息增量 | 丹药涨价、废料入账、青槐灾民契、青云办事处查废料、黑炉能让废丹碎末退一点霉 |
| 章尾钩子 | 赵长生记录“不成丹，不可食，不可示人”，确认废物能回一点药性，但风险大于收益 |

## 2. 执行要点

1. 苦难只保留能推动主线的部分：霉米、丹渣、炉灰、旧灰责任和青槐灾民契。
2. 废料账必须成为捡炉的直接原因，赵长生不是主动寻宝。
3. 黑炉不能交、不能卖、不能随手丢的理由要写清楚，避免“主角知道外挂所以私藏”。
4. 第一次异常必须由废丹筛潮碎末误入炉口触发，不是赵长生主动科研。
5. 赵长生后续验证只能用最低价值、最小剂量的发霉碎丹角，结论必须写清：不成丹，不可食，不可示人。
6. 章末保留一点小爽点，但立刻被废料账、气味外泄和不可示人的风险压住。

## 3. 禁写项

- 不写归墟炉来历。
- 不写认主、炉灵、天地异象。
- 不写系统、沈临风或后期真相。
- 不让赵长生吞服碎丹或突破。
- 不用空泛作者腔概括苦难，必须落到物件、账目、动作和风险上。
