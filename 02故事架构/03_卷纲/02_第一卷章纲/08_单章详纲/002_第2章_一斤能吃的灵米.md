# 第 2 章单章详纲：一斤能吃的灵米

> 章名状态：沿用段纲暂定为《一斤能吃的灵米》。本章是归墟炉第二次试验章，从“废丹药性”推进到“发霉灵米恢复成普通旧灵米”，只解决一顿饭的生存问题，不能写成灵食、暴富或修为大进。

## 2. 本章定位

| 项目 | 内容 |
|------|------|
| 章节功能 | 第二次试验发霉灵米，证明归墟炉能把少量霉米恢复到普通旧灵米水准 |
| 主要场景 | 灵米铺外、棚区破屋、破屋门外短暂动静 |
| 出场人物 | 赵长生、灵米铺老板、棚区少年；其他散修只作背景 |
| 当前冲突 | 赵长生想吃一顿不霉的灵米，但怕投料过多毁米、留下米香和暴露黑炉 |
| 信息增量 | 普通灵米约 5-8 碎灵一斤，好灵米更贵；发霉旧米便宜但底层常吃；青云办事处摊费上涨会继续推高物价 |
| 压迫点 | 他买不起普通灵米，只能处理手里的发霉灵米；处理后数量减少、灵气很弱，连一顿饭都可能招来视线 |
| 爽点处理 | 爽点是霉米终于不刮嗓子，可真正收获只是一小锅能吃的米，不够卖、不够修炼、不够改变身份 |
| 章尾钩子 | 棚区少年闻到米香，赵长生压紧锅盖，意识到以后试验必须更深夜、更少量、更隐蔽 |

## 3. 开场要求

开场承接第 4 章，不跳成“已经掌握炉法”：

1. 赵长生白天仍要照常做杂役，黑炉和试过的碎丹都必须藏好。
2. 第 4 章那一丝药香没有让他立刻敢吃丹，只让他确认归墟炉可能恢复残缺物。
3. 他先去灵米铺外看价，不是为了购物爽点，而是确认自己买不起能吃的米。
4. 开场要让读者看到：一次小试有效后，赵长生第一反应仍是算风险和生活账，不是膨胀。

## 4. 场景顺序

### 4.1 灵米铺外问价

- 赵长生借送灰、取水或下工路过灵米铺，先看门口新牌。
- 普通灵米标价七碎灵一斤，发霉旧米二碎灵一斤，好灵米挂在柜内，价格更高。
- 灵米铺老板顺口抱怨青云办事处又要涨摊费，米价还要往上走。
- 赵长生问价后没有买普通灵米，只摸清价格，带着手里那点发霉灵米离开。
- 这里不长篇算账，用一两笔让读者知道他买不起。

### 4.2 回棚区破屋

- 赵长生从灵米铺绕回棚区破屋，路上经过漏雨棚、咳嗽散修、争水声或骂摊费的声音。
- 破屋比小柴房更适合试米：离丹铺远，不容易被周槐安和小伙计发现，但棚区人杂，味道更容易引来邻近视线。
- 他先检查门闩、窗缝、灶口、墙根和屋外脚步，再把黑炉从灰袋夹层取出。
- 本章要把“可试验空间”从丹铺小柴房过渡到破屋，为后续更深夜试验打基础。

### 4.3 选米和留对照

- 赵长生把霉米摊在破瓦或旧布上，挑出霉斑最重的一小撮。
- 他不处理整斤，只先处理一小把；“一斤能吃的灵米”是本章愿望和压力，不是实际一次产量。
- 留一撮未处理霉米作对照，再称或估量投入前后的数量。
- 霉米潮、结团、刮嗓子的细节要写清，避免爽点悬空。

### 4.4 归墟炉第二次试验

- 赵长生把一小撮霉米分批放入黑炉，不一把倒满。
- 炉底仍只是暗红微光；霉米潮、散、杂质多，处理方式和碎丹不同，不写成炉子两次就虚。
- 屋内先有霉味和灰味，随后出现极淡米香，但香气不浓，不能写成灵食满屋。
- 炉身变冷或炉底余灰更暗，提示代价。
- 胸口旧烙痕本章可不再重点写热，避免伏笔过密；若写，只是一瞬牵动旧伤。

### 4.5 米变得能吃但缩水

- 处理后的米霉斑退尽，坏米和黑点要挑掉，剩下部分恢复成普通旧灵米水准，灵气很薄。
- 数量明显减少，只剩七八成；碎米和黑点要被挑掉。
- 赵长生先煮很小一锅，用剩水控制火，不铺张。
- 米入口有正常旧灵米味道，但不是香甜灵饭，也不能支撑修炼。
- 不写修为增长，只写胃里暖了一点、喉咙不再被霉味顶住。

### 4.6 棚区少年闻香

- 棚区少年从门外或墙缝附近停步，可能只是问“你煮的什么米”或盯着门缝闻。
- 赵长生没有大方分米，也不冷酷羞辱，只把锅盖压紧，说是丹铺抵的霉米，味道重。
- 少年没有变成同伴，只作为“连一小锅不霉的米都能引来视线”的风险提醒。
- 赵长生等脚步走远后，把剩米分成两份：今晚吃一点，明早带一点，不能让香味再散。

### 4.7 记录和下一步判断

- 赵长生在破纸册中短记：霉米数量、处理后剩量、米香、炉底暗灰、成旧灵米但灵气薄、气味风险。
- 明确判断：不可卖，不可常白日煮，不可让棚区人知道。
- 章尾落在他把黑炉藏得更深，并决定以后试验必须在更深夜、小份量、先封门缝后动炉。
- 下一章自然接周槐安盘点药渣/灵草废料，压住试验连续爽感。

## 6. 人物执行

### 6.1 赵长生

- 仍然冷、稳、慢、谨慎。
- 去灵米铺不是消费升级，而是确认市价和自己的贫穷。
- 试验只用少量霉米，留下对照，记录缩水和气味。
- 吃到能咽的米可以有一小点满足，但不能写成狂喜或境界提升。
- 面对棚区少年时不能大方施舍，也不能暴露异常，只压锅盖、收味道。

### 6.2 灵米铺老板

- 功能是投放物价上涨和摊费压力。
- 可以嘴碎、嫌穷客挡门，但不变成主线反派。
- 不知道归墟炉，不发现赵长生的试验。

### 6.3 棚区少年

- 功能是暴露风险，不是温情线。
- 可以瘦、饿、闻到米香停步，但不要写成需要赵长生立刻收养或长期照顾。
- 他让赵长生意识到：底层不是没人注意，只是大家都穷到盯着每一点味道。

## 7. 语言执行提醒

1. 延续第 2-4 章修订后的口径：少用孤立短句制造情绪，动作和判断按因果连续写。
2. 开段主语尽量明确，关键动作优先写“赵长生”。
3. 灵米铺和棚区细节要服务冲突，不写大段地图说明。
4. 米的变化要具体：霉斑退尽、结团散开、坏米挑掉、数量缩水、入口像正常旧米、灵气薄。
5. 爽点必须小：能吃一顿饭已经是改善，但不是卖货、修炼和翻身。

## 8. 禁写项

1. 不让霉米变成高等灵米或灵食。
2. 不让赵长生靠灵米卖钱暴富。
3. 不写修为突破、经脉大开或灵气暴涨。
4. 不写归墟炉来源、旧天庭、新仙界、补墟司、司命台、墟眼。
5. 不写沈临风、系统或后期人物。
6. 不让灵米铺老板、棚区少年或任何旁人发现黑炉。
7. 不写温情施舍抢主线；棚区少年只承担风险提醒。

## 9. 正文目标

目标字数：3000-3500 字为理想区间，4000 字以内可接受。

正文效果：

1. 读者看到归墟炉能把少量废物恢复成真正可用的生存资源。
2. 赵长生第一次吃到“不刮嗓子”的米，但只是一小锅，且引来风险。
3. 灵米价格、摊费上涨和棚区视线把小爽点压回现实。
4. 章尾把试验规则推进为“更深夜、更少量、更隐蔽”，为第 6 章周槐安盯废料做压力铺垫。
