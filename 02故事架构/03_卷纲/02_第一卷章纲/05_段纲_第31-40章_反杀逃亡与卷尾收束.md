# 段纲：第 31-40 章 反杀逃亡与卷尾收束

> 本段目标：完成第一卷从“青石坊市待不下去”到“带伤逃出青石坊市”的转折。第一次杀人必须靠地形、残器、炉灰、低阶符箓、沈临风一次提示和敌人轻视成立，不能靠归墟炉公开秒人。

---

## 第 31 章：后巷堵截

| 项目 | 内容 |
|------|------|
| 章节功能 | 逃亡启动，周槐安的私人压迫变成栽赃和堵截 |
| 主要场景 | 回春丹铺后巷、药渣房、小柴房外 |
| 出场人物 | 赵长生、沈临风、周槐安、丹铺伙计、小帮派眼线 |
| 当前冲突 | 赵长生准备离开丹铺时，被周槐安派人堵在后巷 |
| 信息增量 | 周槐安想把废料缺口和私吞漏洞都压到赵长生身上 |
| 爽点/压迫 | 爽点是赵长生利用熟悉后勤动线脱身；压迫是他已经没有回头路 |
| 代价 | 赵长生暴露自己早有防备，丹铺伙计认出他带走灰袋和破包 |
| 章尾钩子 | 沈临风赶到，系统提示后巷外也有杀意，两人被迫同路 |

执行细节：

1. 周槐安不亲自追太远，他更适合在后巷指挥、栽赃、喊巡市。
2. 赵长生利用松动青砖、药渣桶、灰袋味道遮掩方向。
3. 沈临风不是英雄登场救人，而是被系统任务和危险提示拖进现场。
4. 赵长生不感谢沈临风，只问他“你怎么知道我在这里”。
5. 本章结尾要让后巷已不能回，黑市巷成为唯一拖延空间。

禁写提醒：

- 不让周槐安知道归墟炉。
- 不让沈临风一招打退所有人。

## 第 32 章：假线

| 项目 | 内容 |
|------|------|
| 章节功能 | 黑市巷误导追兵，裴九卖消息的双刃剑彻底显现 |
| 主要场景 | 黑市巷、旧货摊后墙、南棚区岔口 |
| 出场人物 | 赵长生、沈临风、裴九、坊市散修、小帮派探子 |
| 当前冲突 | 赵沈要穿过黑市巷，但裴九已经把他们的动向卖给小帮派 |
| 信息增量 | 裴九可交易、可背叛、可误导；他不站队，只看价码 |
| 爽点/压迫 | 爽点是赵长生用假残器和假路线反卖一次信息；压迫是裴九仍把真线索卖出去 |
| 代价 | 赵长生花掉一部分碎灵和一件可用残物，只换来短暂时间差 |
| 章尾钩子 | 小帮派判断赵长生会去废料场，因为那是他最熟的地方 |

执行细节：

1. 赵长生故意让裴九以为自己要走南棚区暗沟线。
2. 裴九收钱后把“南线”卖给一拨散修，又把“他可能反走废料场”卖给小帮派。
3. 沈临风看不惯裴九，赵长生提醒他：这种人骂不动，只能利用。
4. 黑市巷的摊布、残器架、破储物袋摊可以用来遮挡视线。
5. 章尾让读者知道：赵长生赢了半步，也输掉了最终路线。

禁写提醒：

- 不把裴九写成热血帮忙。
- 不让误导完全成功。

## 第 33 章：废料场伏击

| 项目 | 内容 |
|------|------|
| 章节功能 | 小帮派直接杀机爆发，第一次杀人前的绝境建立 |
| 主要场景 | 废料场、破木车棚、废灰坑 |
| 出场人物 | 赵长生、沈临风、小帮派头目、小帮派副手、两名帮众 |
| 当前冲突 | 小帮派在废料场堵住赵沈，要求交出异常资源和货源 |
| 信息增量 | 小帮派头目炼气四层压阵，副手炼气三层巅峰下手，正面压制赵长生 |
| 爽点/压迫 | 压迫是境界差和人数差；爽点是赵长生对地形更熟 |
| 代价 | 赵长生先挨一击，护身手段损耗，沈临风也被逼出系统奖励防护 |
| 章尾钩子 | 赵长生被逼到灰坑边，残剑碎片还没找到出手机会 |

执行细节：

1. 第一次杀人对象锁定为炼气三层巅峰副手；小帮派头目亲自压阵，提供炼气四层压力，但不作为本次被杀对象。
2. 敌人轻视赵长生，认为丹铺杂役不敢杀人，只想逼问货源。
3. 沈临风能挡一次，但不能代替赵长生破局。
4. 废料场要用前面埋过的灰坑、残器区、破木车轮轴、旧灰井。
5. 敌人目标是抢资源，不是查归墟炉本质。

禁写提醒：

- 不让赵长生正面赢炼气四层。
- 不让归墟炉公开吞敌。

## 第 34 章：灰坑反杀

| 项目 | 内容 |
|------|------|
| 章节功能 | 完成赵长生第一次杀人，确立他的狠、怕、稳和代价 |
| 主要场景 | 废灰坑、残器铁片区、破木车棚 |
| 出场人物 | 赵长生、沈临风、小帮派副手、帮众、小帮派头目远压或后撤 |
| 当前冲突 | 赵长生必须在被带走或被搜身前制造唯一破绽 |
| 信息增量 | 低阶越级必须靠地形、信息差、一次性资源和敌人轻视 |
| 爽点/压迫 | 爽点是底层经验撬开强敌破绽；压迫是杀人过程不潇洒、不干净 |
| 代价 | 赵长生受伤、法力耗尽、手抖、恶心，残剑锋芒损毁 |
| 章尾钩子 | 死人倒进灰坑，远处巡市铃声响起，赵沈必须立刻处理后果 |

执行细节：

1. 沈临风系统只提示一次，例如：【左侧三息后有杀意爆发。】不能给完整打法。
2. 赵长生把炉灰扬向敌人眼口，制造一息迟滞。
3. 敌人踩上表面结壳的灰坑或松动木车轴，重心失衡。
4. 赵长生用低阶符箓或旧扰目符压住另一名帮众，不能群杀。
5. 残剑碎片只在补刀或破护体时起作用，用后寒光散尽。
6. 补刀必须艰难：对方还会抓他手腕、挣扎、吐血或反击。
7. 杀后赵长生不能冷酷台词，要出现呼吸错乱、胃里翻涌、短暂失神。
8. 沈临风也被震住，第一次意识到这里不是任务副本。
9. 小帮派头目不被赵长生正面击杀，可因追兵、巡市铃声、废料场混乱或自身谨慎暂退，留下后续追压空间。

禁写提醒：

- 不写潇洒秒杀。
- 不让敌人弱智站着等死。
- 不让杀人没有血迹和物证。

## 第 35 章：血灰与储物袋

| 项目 | 内容 |
|------|------|
| 章节功能 | 处理第一次杀人后果，获得残炉碎片但留下追查隐患 |
| 主要场景 | 废料场、废灰坑、破木车棚 |
| 出场人物 | 赵长生、沈临风、远处巡市修士声音、未死帮众或逃走眼线 |
| 当前冲突 | 赵沈必须在追兵到来前处理尸体、血迹、储物袋和物证 |
| 信息增量 | 杀人后果包括尸体、血迹、灵力残留、储物袋禁制、目击者 |
| 爽点/压迫 | 爽点是赵长生夺得逃亡资本；压迫是处理粗糙，后患明确 |
| 代价 | 赵长生不能完全清理现场，沈临风为了拖时间消耗系统护身或小术 |
| 章尾钩子 | 敌人储物袋里有一块无灵光残炉碎片，归墟炉第一次明显震动 |

执行细节：

1. 储物袋不能轻松打开，可写禁制粗糙但赵沈没时间细破。
2. 他们只拿走最容易带的东西：少量碎灵、低阶药粉、灰布、残炉碎片。
3. 血迹用炉灰和药渣遮掩，但不能完全消失。
4. 若有帮众逃走或半死喊人，会让追捕更合理。
5. 赵长生看到残炉碎片时不懂来历，只知道归墟炉烫得厉害。
6. 沈临风系统不能识别残炉碎片，只提示“未知残源干扰”也要留到卷尾或临界，不要现在解释。

禁写提醒：

- 不写完美毁尸灭迹。
- 不让储物袋成为巨额财富。
- 不解释残炉碎片来历。

## 第 36 章：废料车道

| 项目 | 内容 |
|------|------|
| 章节功能 | 从杀人现场转入逃亡路线，法力、伤势和时间压力同时收紧 |
| 主要场景 | 废料车道、破棚、废料车旁 |
| 出场人物 | 赵长生、沈临风、废料车夫、门房远线、追兵影子 |
| 当前冲突 | 赵沈要借废料车道接近北小门，但身上有血、灰和灵力残留 |
| 信息增量 | 废料车道能遮掩脚印和气味，但也是被查账的窄道 |
| 爽点/压迫 | 爽点是用底层劳动路线逃命；压迫是封锁口令正在传来 |
| 代价 | 赵长生伤口被药渣和灰尘刺激，沈临风法力也见底 |
| 章尾钩子 | 北小门方向传来门房盘问废料车的声音 |

执行细节：

1. 赵长生用灰布、药渣味和废料车辙遮掩血腥。
2. 沈临风想用系统奖励快速恢复，发现只能缓一口气，不能满血。
3. 废料车夫可以收钱装作没看见，但不会冒死帮他们。
4. 两人第一次因路线选择争吵：沈临风想走宽路，赵长生坚持废料车道。
5. 章尾要让北小门半封未封，时间差成立。

禁写提醒：

- 不让伤势消失。
- 不让废料车夫变忠义帮手。

## 第 37 章：半封的北小门

| 项目 | 内容 |
|------|------|
| 章节功能 | 抢在正式封锁前逃出青石坊市，完成空间转场 |
| 主要场景 | 北小门、门房、废料车队尾 |
| 出场人物 | 赵长生、沈临风、北小门门房、巡市修士、小帮派追兵远影 |
| 当前冲突 | 北小门开始核查通行，赵沈身份和伤势都经不起细查 |
| 信息增量 | 北小门不是无人把守，只是封锁来得慢、记录粗糙、可被底层缝隙利用 |
| 爽点/压迫 | 爽点是赵沈抢出窄门；压迫是门内开始形成通缉口风 |
| 代价 | 赵长生花掉所剩不多的碎灵，沈临风用一次系统危险提示排除死路 |
| 章尾钩子 | 门后有人喊：“丹铺那个青槐杂役跑了！” |

执行细节：

1. 赵长生可伪装成随废料车出门的搬灰杂役，但伪装很粗糙。
2. 门房收钱不代表安全，他会记住灰布、血味和沈临风的外来口音。
3. 沈临风系统提示只能是“左侧门道风险高”，不能直接告诉最佳路线。
4. 赵长生不走东门，坚定走北小门，符合前置地图。
5. 出门后不能松口气，立刻进入枯石岭外坡和荒野压力。

禁写提醒：

- 不写门房完全无脑。
- 不让系统隐身或传送。

## 第 38 章：枯石岭夜路

| 项目 | 内容 |
|------|------|
| 章节功能 | 将压迫从坊市规则切换为荒野规则 |
| 主要场景 | 枯石岭外坡、乱石药径、枯草灌木 |
| 出场人物 | 赵长生、沈临风、远处追兵、低阶毒虫或夜兽动静 |
| 当前冲突 | 赵沈带伤夜行，既要甩追兵，又要避免走错药径 |
| 信息增量 | 枯石岭灵气薄、碎石多、夜间难行；旧采药路不在正式地图上 |
| 爽点/压迫 | 爽点是赵长生底层听来的采药路派上用场；压迫是他也不熟全路 |
| 代价 | 赵长生摔伤或伤口裂开，沈临风脚步拖慢，二人互相不满 |
| 章尾钩子 | 系统提示一条岔路有危险，赵长生却发现另一条路也有新脚印 |

执行细节：

1. 赵长生知道大方向，不知道每个细节，避免写成完美地图。
2. 沈临风系统排除一条死路，证明有用；但剩下选择仍要赵长生判断。
3. 两人的冲突要保留：沈临风觉得赵长生太保守，赵长生觉得沈临风太信系统。
4. 远处追兵不必大队深入，更多是火光、喊声、符光短闪。
5. 毒虫或夜兽只是拖延，不展开成新副本。

禁写提醒：

- 不写野外逃亡像郊游。
- 不让系统提供完整地图。

## 第 39 章：断溪谷外围

| 项目 | 内容 |
|------|------|
| 章节功能 | 追兵止步，主角付出逃亡代价，残炉碎片触发卷尾前兆 |
| 主要场景 | 断溪谷外围、药瘴浅坡、废灵脉支沟入口 |
| 出场人物 | 赵长生、沈临风、少量追兵远影 |
| 当前冲突 | 断溪谷外围地气混乱，追兵不愿深追，但赵沈也被药瘴和伤势拖住 |
| 信息增量 | 断溪谷和废灵脉支沟能干扰追踪，但不是安全根据地 |
| 爽点/压迫 | 爽点是终于拉开距离；压迫是两人都已经耗到临界 |
| 代价 | 疗伤粉用掉，低效丹不敢再吃，沈临风系统短暂卡顿 |
| 章尾钩子 | 残炉碎片被归墟炉吸入，炉壁裂纹亮起看不懂的残纹 |

执行细节：

1. 追兵止步要有理由：药瘴、地气乱、天快亮、没有足够好处。
2. 赵长生不能恢复太快，只能临时压血。
3. 沈临风系统在断溪谷附近开始出现短暂延迟或乱码，但不解释。
4. 残炉碎片吸收要低调：无大光柱，只是炉身发烫、裂纹亮起。
5. 赵长生和沈临风都看见异常，但理解不同：赵长生更怕，沈临风更怀疑系统。

禁写提醒：

- 不解释断溪谷深层源头。
- 不去青槐村旧址。

## 第 40 章：万物归墟，仙门已断

| 项目 | 内容 |
|------|------|
| 章节功能 | 第一卷收束，完成逃离、临时同盟、卷尾残字和系统报错 |
| 主要场景 | 废灵脉支沟、半塌旧采药棚、石壁残纹 |
| 出场人物 | 赵长生、沈临风 |
| 当前冲突 | 两人暂时脱身，但伤势、身份污名、系统异常和归墟炉残字同时压来 |
| 信息增量 | 归墟炉吸收残炉碎片后浮现“万物归墟，仙门已断”；系统报错但不给解释 |
| 爽点/压迫 | 爽点是逃出青石坊市并拿到第一条大钩子；压迫是他们并不安全，反而被更大异常盯上 |
| 代价 | 赵长生失去青石坊市落脚点，背上偷废料和杀人的风险；沈临风系统也不再完全可信 |
| 章尾钩子 | 系统强行恢复：【支线任务完成：逃离青石坊市。】但沈临风看见前一瞬报错没有消失 |

执行细节：

1. 归墟炉残字写作：

   > 万物归墟，仙门已断。

2. 沈临风系统短暂报错写作：

   ```text
   【司命残录校验失败。】
   【发现未知残源。】
   【权限不足。】
   ```

3. 随后系统强行恢复：

   ```text
   【支线任务完成：逃离青石坊市。】
   ```

4. 赵长生不理解“仙门已断”，只把它当作危险线索收起。
5. 沈临风不解释系统，只第一次认真问：“你到底是什么人？”
6. 赵长生不回答，只说：“现在回去，我们两个都死。”
7. 结尾落在两人临时同盟上：不是兄弟，不是完全信任，而是眼下必须一起活下去。
8. 第二卷入口只露方向：青云门、荒野杂务、散修队伍或旧采药路，不直接写入宗门受欢迎。

禁写提醒：

- 不解释旧天庭、司命台、新仙界。
- 不让赵长生得知命簿已死。
- 不让两人安全闭关。
- 不让卷尾变成大团圆。
