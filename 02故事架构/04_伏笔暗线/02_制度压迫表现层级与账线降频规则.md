# 制度压迫表现层级与账线降频规则

> 本文件用于约束“账、账册、账本、登记、配额、名单、契书、仙籍”等制度压迫元素的使用频率和表现层级。核心目标：保留贯穿全书的制度压迫暗线，但避免把小说写成“到处对账”。

---

## 1. 总判断

“账”不是本书题材，也不是主角一路推进剧情的唯一动作。

本书核心仍是：

1. 底层求活。
2. 废料重炼。
3. 谨慎积累。
4. 同伴合作。
5. 追查飞升和世界衰败真相。
6. 终局重炼循环。

“账”只是一种制度外壳，用来把资源分配、身份控制、命数安排和本源回收具象化。

正确用法：

> 第一章的小账册，是后期仙界制度的最小影子。

错误用法：

> 赵长生一路拿着账本查到仙界，所有反派都靠账册推进，所有爽点都变成对账成功。

## 2. 核心定义

| 概念 | 正确理解 | 禁止理解 |
|------|----------|----------|
| 账 | 资源和责任被制度记录、折算、转嫁的外壳 | 小说主玩法 |
| 查账 | 某些阶段的证据获取方式 | 每卷固定主线动作 |
| 清账 | 承认真实代价、公开责任、改变规则 | 主角接管账本继续统治 |
| 补墟账本 | 仙界制度腐烂的证据系统 | 万能真相说明书 |
| 命数记录 | 司命台对变量的控制接口 | 简单游戏面板 |
| 族契、仙籍、功簿 | 不同阶段的制度外壳 | 全都换皮成账册 |

一句话规则：

> 账是隐喻和证据，不是重复剧情动作。

## 3. 分阶段表现层级

| 阶段 | 表层表现 | 背后本质 | “账”字面频率 |
|------|----------|----------|--------------|
| 第一卷：青石坊市 | 工钱账、废料入账、杂役契、霉米抵工钱 | 底层劳动、废料和身份都被规矩压住 | 可以较高，但集中在关键章 |
| 第二卷：青云外门 | 任务牌、功劳簿、库房配额、丹房记名 | 宗门资源分配不公 | 降低“账册”，改写任务和配额 |
| 第三到五卷：下界中后段 | 税契、战功、丹方垄断、灵田归属、洞府地契 | 宗门和修仙国垄断突破资源 | 只在制度节点出现 |
| 第六到九卷：灵界篇 | 族契、血样、猎血许可、名单、战损记录、抽源额度 | 血脉、族运、界源被制度化折算 | 少写“账本”，多写契、名单、许可、额度 |
| 第十到十三卷：仙界篇 | 仙籍、仙税、役期、道职、补墟明账内账黑账 | 飞升者和万界本源被合法化接管 | 可以重新抬高，但必须落到事件 |
| 第十四卷：终局 | 公开责任、归还本源、重定司命盘、重炼墟眼循环 | 改掉收割规则，建立新循环 | 不写“对账胜利”，写规则重定 |

## 4. 卷级使用规则

### 4.1 第一卷

第一卷可以明写账，因为开局压迫就是丹铺工钱和废料规则。

允许重点出现：

1. 第 1 章《账上无错》：工钱账吃人。
2. 第 6-10 章：废料称重压力逐步形成。
3. 第 21 章：周槐安核废料数，准备甩锅。

需要降频：

1. 第 2 章转为灵井、青槐灾民、杂役契。
2. 第 3 章转为夜里清灰和残炉入手。
3. 第 4-10 章重点是归墟炉试验、谨慎记录和暴露风险，不要每章都写大段扣账。

第一卷读者感受应是：

> 这个世界连废物都有规矩，不是作者在反复讲账。

### 4.2 第二卷到第五卷

下界中段要把“账册压迫”换成宗门和修仙国的资源秩序。

优先写：

1. 功劳不够，不能换丹。
2. 任务牌记名，出事有人背锅。
3. 丹房废料清单只作为证据之一，不做主玩法。
4. 筑基资源被家族、宗门和黑市垄断。
5. 洞府、灵田、战功、税契把资源绑定到身份。

禁止：

1. 每卷都安排一次查账高潮。
2. 所有敌人都变成账房或管事。
3. 所有证据都来自账册。

### 4.3 第六卷到第九卷

灵界篇不应写成“灵界账本篇”。

灵界表层应当是：

1. 族契。
2. 血样。
3. 猎血许可。
4. 货栈暗号。
5. 战损名册。
6. 抽源额度。
7. 验源记录。

这些东西背后可以接入补墟外围账口，但正文表层要像灵界社会自己的制度，而不是仙界账房提前下场。

灵界读者感受应是：

> 百族被契约、血样和战争规则困住，而不是大家都在翻账本。

### 4.4 第十卷到第十三卷

仙界篇可以重新抬高“账本”概念，因为这时真相进入制度中枢。

但仍要遵守：

1. 仙籍优先于账本。
2. 仙税和役期优先于抽象黑账。
3. 登仙监、洗尘池、分流台先以场景压迫出现。
4. 补墟账本必须通过事件打开，不能当设定说明书。
5. 纪晚凝不是“查账工具人”，她的转向来自证据、责任和制度良知。

仙界读者感受应是：

> 飞升者进入的是身份、税役和道职网络，账本只是揭开腐烂的证据。

### 4.5 第十四卷

终局不能写成主角掌握总账本后清算所有人。

终局要写：

1. 公开被掩盖的真实伤害。
2. 归还被强行抽走的本源。
3. 司命盘从安排命运改为记录危机。
4. 飞升从回收通道改为循环接口。
5. 墟眼从吞噬伤口重炼为循环炉口。

终局关键词应是：

> 重炼循环。

不是：

> 对账成功。

## 6. 证据来源配比

每卷关键证据不宜只来自文书。

建议比例：

| 证据来源 | 建议作用 |
|----------|----------|
| 生活细节 | 开卷和底层压迫，优先使用 |
| 空间痕迹 | 地脉、残界、古战场、废料场、旧天庭废墟 |
| 人物选择 | 同伴背叛、反派甩锅、制度内人物转向 |
| 实物残片 | 残器、破玉简、账符、旧令、阵盘 |
| 文书记录 | 只在需要坐实制度链时使用 |
| 系统异常 | 沈临风线专用，不替代主角判断 |
| 战斗和逃亡代价 | 证明证据不是白捡来的 |

硬规则：

> 每卷最多一到两次关键文书证据高潮，其余证据必须通过行动、空间和人物代价获得。

## 7. 人物分工

| 人物 | 可以承载 | 禁止写成 |
|------|----------|----------|
| 赵长生 | 记数、谨慎、反查矛盾、重炼残物 | 拿算盘修仙的查账主角 |
| 沈临风 | 系统任务和命数记录异常 | 系统客服或数据分析工具 |
| 纪晚凝 | 仙界制度内的责任和证据转向 | 因恋爱倒戈的账房女主 |
| 白倩霜 | 族契、源血和百族责任 | 血账证据容器 |
| 陆维舟 | 阵法、空间和界源痕迹 | 只为查账开门的工具人 |
| 谢未明 | 命数污染和天机错算 | 单纯解释司命台的人形旁白 |
| 俞简、严慎等制度人物 | 把伤害制度化的人 | 杀掉就解决问题的小反派 |

## 8. 正文阶段检查清单

写每卷或每段纲前检查：

1. 本阶段表层压迫是不是和上一阶段重复。
2. 是否又把“账册、账本、查账”当成主要动作。
3. 本章能不能改用契书、名单、任务牌、仙籍、许可、配额、空间痕迹或人物代价。
4. 文书证据是否真的必须出现。
5. 主角组拿到证据是否付出了行动成本。
6. 读者看到的是事件，还是设定说明。
7. 终局是否仍指向重炼循环，而不是主角接管制度。

## 9. 对第一章的当前判断

第 1 章《账上无错》可以保留“账”的显性表达。

原因：

1. 它是全书第一个制度压迫缩影。
2. 周槐安的压迫方式就是账面无错、人被吃掉。
3. 本章标题已经绑定这个讽刺点。

但从第 2 章起，应立即降频：

1. 第 2 章转向灵井、青槐灾民、杂役契。
2. 第 3 章转向清灰、后院独处、黑炉入手。
3. 第 4-10 章转向归墟炉试验和废料暴露风险。

第一卷整体目标不是让赵长生查账，而是让读者相信：

> 修仙界连废物都不属于底层，赵长生只能从这些废物里炼出生路。
