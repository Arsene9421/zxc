# 全书总纲

## 1. 核心故事

青石坊市丹铺杂役赵长生，五行杂灵根，无家族、无师承、无背景。

他在废炉灰中捡到一尊残破归墟炉。此炉能让有缺之物返本归元。

赵长生靠废丹、残器、枯草一步步积累资源，从底层杂役走上长生路。

同一时间，穿越者沈临风绑定天命修仙系统，以为自己拿了主角剧本。系统却不断提示他接近赵长生。

两人从互相试探，到利益合作，到生死搭档，再到共同破局。

全书采用十四卷结构：

- 下界篇 5 卷：从青石坊市底层，到化神断仙门。
- 灵界篇 4 卷：从灵界底层，到百族、万年抽源、登仙台残路。
- 仙界篇 5 卷：从仙界底层，到司命台、补墟司、旧天庭废墟、归墟重炼。

后期他们发现：

- 修仙界贫穷不是自然现象。
- 飞升通道早已被改造。
- 天命系统是新仙界的棋子培养器。
- 旧天庭曾维持万界循环，但已经失败。
- 新仙界靠收割下界填补墟眼。
- 归墟大潮正在逼近。

赵长生最终不继承新仙界，也不复辟旧天庭，而是重炼仙界。

## 2. 赵长生主线

| 阶段 | 目标 |
|------|------|
| 前期 | 活下去，靠归墟炉积累资源 |
| 中期 | 长生，掌握自己的命 |
| 后期 | 反抗仙界收割下界 |
| 终局 | 重炼万界循环 |

## 3. 沈临风主线

| 阶段 | 目标 |
|------|------|
| 前期 | 依靠系统变强，以为自己是主角 |
| 中期 | 怀疑系统，发现系统总围绕赵长生 |
| 爆点 | 拒绝远离赵长生的主线任务 |
| 后期 | 夺回司命台残火，摆脱天命安排 |
| 终局 | 重定众生命数规则 |

## 4. 终局三选择

| 选择 | 问题 |
|------|------|
| 继承新仙界 | 只是换好人管理坏系统 |
| 恢复旧天庭 | 旧天庭当年已经失败 |
| 归墟重炼 | 代价巨大，但能建立新循环 |

赵长生选择第三条。

## 5. 主题句

> 真正的长生，不是接受天命奖励，而是把残缺命数重新炼一遍。

> 仙路已断，便重炼一条。
