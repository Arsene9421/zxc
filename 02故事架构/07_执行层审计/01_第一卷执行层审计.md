# 第一卷执行层审计

> 本文档用于把第一卷《青石坊市》从卷纲草案推进到章纲前的可执行边界。本文不写正文，不拆 30 到 50 章章纲，不新增大设定。

---

## 1. 审计结论

结论：条件通过。

第一卷已经具备进入章纲准备的条件：

1. 卷目标清楚：穷、狠、爽、谨慎、系统不对劲、坊市待不下去。
2. 场景舞台清楚：回春丹铺、破屋、灵井、旧货街、黑市巷、废料场、北小门、枯石岭、断溪谷外围。
3. 反派压力清楚：周槐安压榨、裴九卖消息、小帮派抢资源、蒋怀礼查废料账。
4. 能力边界清楚：归墟炉只处理低阶残缺物，系统只表现为新手外挂和异常卡顿。
5. 战力边界清楚：赵长生不能正面越级乱杀，第一次杀人必须靠布置、地形、残器、符箓、信息差和敌人轻视。
6. 禁泄露边界清楚：第一卷不解释旧天庭、新仙界、补墟司、司命台真实来源、墟眼、赵长生命簿已死和沈临风穿越原因全貌。

当前可以进入第一卷 30 到 50 章章纲准备，但不建议直接写正文。

## 7. 反派行动线

第一卷压力应从生存压迫逐步升级为逃亡压力。

| 阶段 | 反派动作 | 造成的压力 | 证据或爽点 |
|------|----------|------------|------------|
| 1 | 周槐安用发霉灵米、废丹渣抵工钱 | 赵长生必须从废料中找活路 | 废丹、霉米、丹铺账 |
| 2 | 赵长生清废炉灰，捡到残炉 | 归墟炉初试开始 | 残炉、炉灰、旧烙痕 |
| 3 | 周槐安发现废丹渣或炉灰账不对 | 赵长生不能长期在丹铺安全试验 | 废料称重和账册压力 |
| 4 | 赵长生去旧货街或灵米铺试探交易 | 裴九或旧货摊主注意异常 | 残器、破玉简、小额碎灵 |
| 5 | 散修或小帮派盯上异常资源 | 资源爽点转成杀机 | 黑市巷、棚区、废料场盯梢 |
| 6 | 蒋怀礼介入废料账，周槐安想甩锅 | 私人纠纷升级为上宗废料问题 | 上宗收废料口风 |
| 7 | 沈临风系统强推接近赵长生 | 两人被同一场危局捆住 | 系统关键人物、气运读取失败 |
| 8 | 小帮派头目在废料场或车道堵截 | 赵长生第一次杀人并逃离 | 地形反杀、残器、炉灰、系统提示 |
| 9 | 北小门封锁开始形成 | 赵沈必须抢时间出坊市 | 逃亡爽点和后续追索 |

执行要求：

1. 周槐安、裴九、蒋怀礼、小帮派不能一开始就联合。
2. 每个人都只按自身利益行动。
3. 没有人知道归墟炉本质。
4. 每次阶段胜利都必须带来更高压力。

## 8. 归墟炉前十次试验边界

| 次数 | 投入物 | 第一卷可写效果 | 代价和风险 | 禁止写法 |
|------|--------|----------------|------------|----------|
| 1 | 发霉碎丹 | 霉斑退去，有一丝药香 | 炉火微弱消耗，赵长生开始记录 | 变成完整聚气丹 |
| 2 | 发霉灵米 | 可食用，灵气很弱 | 只能救急，不能大量改善生活 | 变成高等灵米 |
| 3 | 枯黄灵草叶 | 重新泛青 | 只能恢复残余生机 | 变成高年份灵草 |
| 4 | 废丹渣 | 凝出半粒低效丹丸 | 有丹毒和药性不稳风险 | 无代价连刷丹药 |
| 5 | 裂符笔 | 笔锋恢复一点灵性 | 只能卖低价或自用一次 | 变成完整符器 |
| 6 | 破玉简碎片 | 多显出几个残字 | 字句残缺，需要猜 | 补出完整神功 |
| 7 | 炉灰 | 炼出温热黑砂 | 暗示炉灰价值，但可腐蚀 | 直接解释旧天庭炉灰 |
| 8 | 枯种或死虫 | 枯种发芽，死虫无效 | 确认不能复活死物 | 复活生命 |
| 9 | 残剑碎片 | 一缕寒光，消耗大 | 可用于卷尾保命或反杀，但会损毁 | 获得完整飞剑 |
| 10 | 残炉碎片 | 炉壁浮现残字 | 卷尾钩子，引来后续追索 | 直接解锁九器真相 |

归墟炉执行原则：

1. 每次试验都要有记录、等待、消耗或失败边界。
2. 赵长生必须越来越谨慎，而不是越来越莽。
3. 异常资源一旦进入市场，就必须引来风险。
4. 归墟炉不能临场公开吞人、吞血、吞运、吞魂。

## 10. 低阶战斗与第一次杀人审计

第一卷只写炼气低阶战斗。

| 人物 | 修为尺度 | 执行边界 |
|------|----------|----------|
| 赵长生开局 | 炼气一层巅峰或炼气二层初 | 法力少，小术少，靠谨慎和地形 |
| 赵长生卷尾 | 炼气二层稳定，触及三层门槛 | 仍不能正面碾压炼气四层 |
| 沈临风开局 | 炼气二层，系统奖励后有炼气三层保命能力 | 看起来轻松，实战不稳 |
| 周槐安 | 炼气三层 | 可压赵长生，但怕账被查 |
| 小帮派头目 | 炼气四层 | 可作为第一次杀人压力 |
| 蒋怀礼 | 炼气五层 | 正面不能杀，只能骗、拖、逃 |

第一次杀人首选地点：

```text
废料场边缘 / 废灰坑 / 通往北小门的破木车棚附近
```

合理打法：

1. 赵长生提前熟悉灰坑、残器铁片、破木车和旧灰井。
2. 小帮派头目轻视赵长生，只想抢异常资源。
3. 沈临风给一次危险方向或敌人动作提示。
4. 赵长生用炉灰遮眼、残器断路、符箓扰乱或地形失衡制造机会。
5. 补刀必须艰难，不能冷酷潇洒。
6. 杀后必须受伤、法力消耗、恶心、手抖、物证处理粗糙。
7. 最大结果不是庆祝，而是“现在不走就死”。

禁止写法：

1. 归墟炉直接吞人。
2. 赵长生正面秒杀炼气四层。
3. 沈临风系统一键救场。
4. 杀人没有尸体、血迹、储物袋、目击者、巡市和追查问题。
5. 逃亡后立刻安全闭关。

## 11. 资源与物价边界

第一卷赵长生资源状态：

| 项目 | 边界 |
|------|------|
| 开局身家 | 不足 2 块下品灵石，且大半是碎灵 |
| 日常粮食 | 发霉灵米为主，普通灵米也要算账 |
| 丹药 | 一枚完整聚气丹都是正经资源 |
| 法器 | 买不起完整低阶飞剑，只能用残器 |
| 符箓 | 最多有一两张低阶保命或扰乱符箓 |
| 储物袋 | 正常小储物袋买不起，破损储物袋也很贵 |
| 卷尾资本 | 少量碎灵、低效丹、修复残器、干粮、破布和可能夺来的风险物 |

第一卷不能让赵长生暴富。

可接受的提升：

1. 有普通灵米吃。
2. 偶尔能用低效聚气丹。
3. 修复一两件低阶残器保命。
4. 有少量碎灵和下品灵石作为逃亡资本。

## 12. 空间动线审计

### 12.1 日常动线

```text
破屋 -> 灵井 -> 回春丹铺后院 -> 废料场 -> 破屋
```

用途：

1. 展示赵长生底层位置。
2. 投放灵井水变淡和青槐旧灾回声。
3. 让废料场成为熟悉地形，为卷尾反杀服务。

### 12.2 试探交易动线

```text
破屋 -> 旧货街边缘 -> 灵米铺 -> 绕路回棚区
```

用途：

1. 让归墟炉小爽点转成市场风险。
2. 让裴九、旧货摊主或散修注意到异常。
3. 展示碎灵和下品灵石的真实压力。

### 12.3 沈临风入场动线

```text
东门 -> 青石主街 -> 系统提示 -> 回春丹铺 -> 后院
```

用途：

1. 让沈临风以系统视角进入青石坊市。
2. 形成“他看见赵长生，但看不懂赵长生”的误判。
3. 让两人第一次接触保持试探，而不是立刻信任。

### 12.4 被盯梢动线

```text
旧货街 -> 黑市巷 -> 棚区 -> 废料场
```

用途：

1. 让黑市消息、散修贪念和小帮派压力逐步出现。
2. 给赵长生做反跟踪、绕路和记账的空间。
3. 为废料场伏击做准备。

### 12.5 卷尾逃亡动线

```text
回春丹铺后巷
-> 黑市巷
-> 废料场
-> 废料车道
-> 北小门
-> 枯石岭外坡
-> 乱石药径
-> 断溪谷外围
-> 废灵脉支沟
```

执行要求：

1. 北小门不是没人管，而是封锁来得慢。
2. 东门不能作为逃亡路线。
3. 逃出坊市不是安全，只是换成荒野压力。
4. 卷尾不能去青槐村旧址。
