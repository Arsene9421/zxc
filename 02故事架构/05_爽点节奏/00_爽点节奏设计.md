# 爽点节奏设计

## 1. 核心爽点循环

前期基础循环：

1. 赵长生发现废物中还有残余价值。
2. 谨慎试验归墟炉。
3. 得到小资源提升。
4. 不暴露，继续积累。
5. 敌人因资源盯上他。
6. 赵长生提前留退路。
7. 反杀或脱身。
8. 得到更高级残物。

## 2. 第一卷爽点

| 爽点 | 内容 |
|------|------|
| 废丹恢复 | 发霉碎丹恢复药香 |
| 灵米续命 | 发霉灵米变得能吃 |
| 枯草泛青 | 确认归墟炉能养出生机 |
| 残器生光 | 裂器重新有灵性 |
| 破玉简补字 | 残缺功法有了价值 |
| 系统错判 | 沈临风以为赵长生是隐藏 NPC |
| 谨慎避险 | 赵长生没有立刻卖丹，而是加门闩、做记录 |
| 第一次反杀 | 敌人以为他是杂役，结果被反杀 |

## 3. 中期爽点

- 赵长生靠废料补全功法。
- 别人抢完整法宝，赵长生收残器碎片炼本命法宝。
- 沈临风拒绝系统高额任务。
- 天劫真相揭开，赵长生提前布置避开回收。

## 4. 后期爽点

- 飞升仙界后，赵长生仍从废仙石、残仙器起家。
- 系统后台被沈临风反向夺权。
- 新仙界权贵发现自己看不起的废料路线，才是重炼仙界的关键。
- 墟眼从吞噬伤口变成循环炉口。

## 5. 节奏禁忌

不要：

- 连续闭关太久没有外部事件。
- 连续讲设定没有资源爽点。
- 归墟炉一次解决所有问题。
- 系统奖励替沈临风解决所有危机。
- 赵长生轻易相信沈临风。
