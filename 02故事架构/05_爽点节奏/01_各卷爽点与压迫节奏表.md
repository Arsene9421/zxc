# 各卷爽点与压迫节奏表

> 本文档用于把十四卷的压迫、试验、反转、收获、代价和卷尾节奏压成故事架构层执行工具。本文承接 `00_爽点节奏设计.md`、`04_伏笔暗线/01_卷际证据链推进表.md` 和 `06_架构审计/01_各卷核心冲突与反派行动表.md`，不写正文，不拆章纲。

---

## 1. 设计目的

全书爽点不能只靠升级、捡宝和打脸。

本书的核心爽点应来自：

1. 在残缺废物里发现可用价值。
2. 在制度压迫里找到漏洞。
3. 用谨慎记录、低成本试错和提前布局赢过更强对手。
4. 把敌人以为无用、低贱、可折算的东西重炼成破局关键。
5. 让沈临风从系统奖励的爽感，逐步转向自主选择的爽感。
6. 让中后期的证据、制度和理念冲突，都必须转化成行动回报。

基本节奏循环：

```text
压迫
  -> 低成本试验
  -> 小收获
  -> 敌人误判
  -> 主角提前布局
  -> 反转脱身或夺证
  -> 获得资源、证据或关系推进
  -> 付出代价
  -> 卷尾胜利转化为下一卷更高压力
```

## 3. 阶段节奏曲线

### 3.1 下界篇节奏

下界篇的爽点主轴：

> 废料捡漏 -> 低阶试错 -> 资源反杀 -> 建立节点 -> 认知升级。

| 卷 | 爽点重心 | 压迫重心 | 节奏要求 |
|----|----------|----------|----------|
| 第一卷 | 废料恢复和第一次反杀 | 坊市底层生存 | 小而密，必须让每一粒废丹都有重量 |
| 第二卷 | 宗门废料和同伴专业性 | 外门任务和资源分配 | 用任务压力带出叶疏彤、陆维舟、顾长庚 |
| 第三卷 | 绕开正统筑基 | 家族和市场垄断 | 拍卖、黑市、追杀、突破要互相咬合 |
| 第四卷 | 洞府经营和本命法宝胚 | 地盘、契约、维护成本 | 洞府爽点必须带契约和安全成本 |
| 第五卷 | 天劫真相和断仙门 | 战争、古战场、飞升异常 | 爽点从资源变成认知，但仍要有战场行动 |

下界篇禁忌：

1. 不要连续闭关解决升级。
2. 不要让归墟炉变成无限资源机。
3. 不要让赵长生正面越大境界乱杀。
4. 不要提前解释补墟司、新司命台和墟眼。

### 3.2 灵界篇节奏

灵界篇的爽点主轴：

> 重新底层求生 -> 击穿猎血商路 -> 尊重族群选择 -> 抽源站夺证 -> 破飞升荣耀。

| 卷 | 爽点重心 | 压迫重心 | 节奏要求 |
|----|----------|----------|----------|
| 第六卷 | 从灵界底层重新活下去，夺猎血许可 | 猎血商队和血名册 | 追逃、货栈、血样、伪装要具体 |
| 第七卷 | 白倩霜主动选择，破族契和水门 | 大族代执和族契压力 | 感情线必须绑定族群责任 |
| 第八卷 | 冲抽源站，沈临风拒绝系统 | 万年抽源和天命惩罚 | 证据爽点要转成救残界、夺刻度、断任务 |
| 第九卷 | 破验源口，非荣耀飞升 | 登仙台筛选和飞升回收 | 飞升真相要通过行动和残魂证言落地 |

灵界篇禁忌：

1. 不要把猎血写成单人追女主。
2. 不要让白倩霜变成被救血包。
3. 不要让抽源站变成单纯资料库。
4. 不要让沈临风拒绝系统后毫无损失。

### 3.3 仙界篇节奏

仙界篇的爽点主轴：

> 仙界底层重开 -> 反查系统后台 -> 拒绝旧天庭复辟 -> 击穿保全道路 -> 重炼循环规则。

| 卷 | 爽点重心 | 压迫重心 | 节奏要求 |
|----|----------|----------|----------|
| 第十卷 | 废仙石、残仙器、黑籍缝隙里重开局 | 仙籍、仙税、仙矿、登仙监 | 仙界篇必须保持凡人流质感 |
| 第十一卷 | 沈临风反向夺权，纪晚凝核验证据动摇 | 系统后台和补墟封存 | 系统线和补墟证据线要互证，不要各写各的 |
| 第十二卷 | 拒绝复辟旧天庭 | 旧天庭遗民和新仙界追索 | 爽点是有资格拒绝旧权力，不是拿王座 |
| 第十三卷 | 多线证据汇合，击穿裴玄阙道路 | 中天玄阙、暗册、军令、墟眼 | 理念辩论必须靠证据和行动支撑 |
| 第十四卷 | 重炼墟眼为循环炉口 | 旧秩序反扑和大潮压力 | 终局爽点是规则改变和职责重定 |

仙界篇禁忌：

1. 不要把仙界写成更高级宗门。
2. 不要让纪晚凝靠爱情转向。
3. 不要让沈临风拿到更高级系统权限就解决问题。
4. 不要让裴玄阙危机论据全是谎言。
5. 不要让终局变成权力分封。

## 6. 卷尾节奏规则

每卷卷尾不能只“解决本卷问题”，还要把胜利转成新压力。

| 卷尾类型 | 例子 | 作用 |
|----------|------|------|
| 逃离但被追索 | 第一卷逃离青石坊市 | 胜利带来更大范围追查 |
| 拿到机缘但被看见 | 第二卷镇炉足线索或旧炉阵脚条件项曝光 | 资源爽点转成宗门压力 |
| 突破但暴露路线 | 第三卷筑基成功 | 个人升级转成资源链冲突 |
| 建立据点但招来觊觎 | 第四卷洞府显旧痕 | 经营爽点转成地盘压力 |
