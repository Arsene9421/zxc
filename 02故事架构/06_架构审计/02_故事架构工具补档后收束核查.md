## 3. 三张工具的总分工

当前三张工具不是重复文件，而是三个层面的执行约束。

```text
证据链：读者和主角组本卷能知道什么
  -> 冲突链：谁逼他们行动，证据如何被夺出来
  -> 节奏链：行动如何产生压迫、反转、收获和代价
```

这组分工成立。

后续卷纲不能只看其中一张表。每卷拆纲时至少要同时回答三组问题：

1. 本卷要拿到什么证据，必须压住什么真相。
2. 哪个反派、机构或制度主动制造压力，让证据必须通过行动获得。
3. 本卷读者获得的爽点是什么，主角组为此付出什么代价，卷尾如何把胜利转成下一卷压力。

## 5. 主证据链闭环核查

### 5.1 废料上缴链

结论：闭环成立。

推进路径清楚：

```text
第一卷废丹和上宗收废料
  -> 第二到三卷宗门、黑市和上宗回收路径
  -> 第十到十一卷登仙监外账和补墟司回执
  -> 第十三卷补墟总账差额和九源投入曲线
```

冲突链和节奏链能支撑这条线。第一到三卷的反派动机仍是底层资源、宗门考核和家族垄断，不会提前暴露补墟司；第十到十三卷才把废料链接入仙界制度和墟眼死循环。

执行层要求：第一卷只能写“有人收废料”和“废料有异常价值”，不能说出废料最终补墟。

### 5.2 飞升天劫链

结论：闭环成立。

推进路径清楚：

```text
第一卷渡劫无尸骨传闻和残字
  -> 第五卷天劫扫描、古战场残魂和断仙门
  -> 第九卷验源记录、聚魂残影和飞升失败者证言
  -> 第十卷登仙监编号、洗尘池和仙籍分流
  -> 第十四卷新登仙台规则
```

这条线的问题不在证据不足，而在第五卷和第九卷的揭示尺度。第五卷只能确认飞升异常和仙门已断，第九卷才能确认飞升执行口径是筛选、登记、验源、分流、回收。

执行层要求：第一卷传闻只能作为底层怪谈，不做设定解释。

### 5.3 司命系统与命数断页链

结论：闭环成立。

推进路径清楚：

```text
第一卷系统不显示赵长生气运、司命残录校验失败
  -> 第二到五卷系统任务频繁围绕赵长生、奖励出现代价
  -> 第七到九卷司命残录与新司命台冲突、远离赵长生任务
  -> 第十到十一卷新司命台分殿、任务模板、宿主回收记录
  -> 第十三到十四卷命数黑账和司命盘重定
```

沈临风线的节奏也成立：前期拿系统便利，中期发现奖励代价，第八卷拒绝高额任务，第十一卷反追踪系统，终局重定司命盘规则。

执行层要求：第一卷系统必须像爽文外挂，但要有不对劲的卡顿和权限不足；不能写成仙界后台机构展示。

### 5.4 灵界血账与界源抽源链

结论：闭环成立。

推进路径清楚：

```text
第六卷血名册、猎血许可和货栈账册
  -> 第七卷族契裁定、大族代执和水门回执
  -> 第八卷血税名单、战损草表、界源刻度和抽源额度
  -> 第九到十一卷验源口和补墟司外账接收
  -> 第十三卷血魂界运黑账和九源阵槽
  -> 第十四卷百族契约和万界节点规则重订
```

冲突链避免了“猎血是单个反派追女主”的风险。阮青鳞、明厌、玄烬、俞简各自承担商队、监察、族契、入账环节，白倩霜也有主动选择位置。

执行层要求：第六到八卷只能夺外围账和执行文书，不能直接拿到补墟黑账原册。

### 5.5 补墟账本、旧天庭旧账与墟眼真相链

结论：闭环成立，但第十三卷信息密度最高。

推进路径清楚：

```text
第八到九卷仙界账符和验源前置记录
  -> 第十卷纪晚凝外账房、登仙监和仙籍分流
  -> 第十一卷明账内账差额、封账痕迹和宿主回收记录
  -> 第十二卷旧天庭旧账和九器旧职对照
  -> 第十三卷黑账深库、保全法旨、墟眼观测实录
  -> 第十四卷公开清账、司命重定、九器重组和墟眼炉口
```

这条链同时承接纪晚凝、秦照古、洛闻霜、裴玄阙、沈临风和赵长生的后期功能。整体成立，但第十三卷必须在卷纲阶段拆分空间层、人物层和证据层，否则容易变成黑账、军令、司命接口、九源大阵和墟眼解释同时涌入。

执行层要求：第十二卷只证明旧天庭有公共职责且失败过，不让旧天庭变成终局答案；第十三卷证明裴玄阙道路错误，但不能把墟眼危机写成骗局。
