# 突破瓶颈与修行风险

> 各境界具体晋升流程见：`20_境界晋升与突破细则.md`。
> 道心、心魔、道誓和因果细则见：`38_道心心魔道誓与因果.md`。

## 1. 为什么需要风险

如果修行只要资源够就能一路突破，凡人流压力会消失。

突破必须有风险：

- 资源不够会失败。
- 功法残缺会失败。
- 心性不稳会失败。
- 外部干扰会失败。
- 强行吞丹会留下隐患。

## 2. 突破构成

每次大突破至少涉及五项：

| 要素 | 含义 |
|------|------|
| 法力积累 | 境界基础 |
| 功法路线 | 冲关路径 |
| 肉身经脉 | 承受冲击 |
| 神魂心性 | 避免失控 |
| 外部护持 | 丹药、阵法、护法、闭关地 |

## 3. 突破失败后果

| 失败程度 | 后果 |
|----------|------|
| 轻微失败 | 法力反噬，修养数日到数月 |
| 中等失败 | 经脉受损，境界停滞 |
| 严重失败 | 根基受创，寿元折损 |
| 极端失败 | 走火入魔、肉身崩坏、神魂受损 |

赵长生靠归墟炉可以修补部分损伤，但不能无代价清除所有后果。

## 4. 下界关键瓶颈

| 突破 | 瓶颈 |
|------|------|
| 炼气小层级 | 法力积累、丹毒、功法效率 |
| 炼气到筑基 | 筑基丹、闭关地、道基稳定 |
| 筑基到金丹 | 结丹机缘、心魔、本命法宝胚 |
| 金丹到元婴 | 神魂强度、功法完整度、婴火 |
| 元婴到化神 | 天地感应、一界气运、天劫异常 |

## 5. 灵界和仙界瓶颈

灵界：

- 炼虚需要空间灵机。
- 合体需要法身材料和血脉/道统承载。
- 大乘需要一界气运和道果雏形。
- 渡劫会触发仙界筛选。

仙界：

- 真仙需要稳定仙躯和仙籍。
- 玄仙需要仙法和仙器。
- 金仙需要道果稳定。
- 太乙需要大道分支权柄。
- 大罗需要万界因果承载。

## 6. 心魔

心魔不是随便冒出来的幻觉。

心魔来源：

- 道心矛盾。
- 杀戮因果。
- 功法冲突。
- 系统任务诱导。
- 归墟残灰污染。
- 命数残缺回响。

赵长生的心魔重点不应是情爱，而是：

- 活下去和救人的冲突。
- 谨慎和担当的冲突。
- 重炼仙界要付出代价的冲突。

沈临风的心魔重点：

- 自己到底是主角还是棋子。
- 离开系统后还剩什么。

## 7. 归墟残灰风险

归墟炉使用过度会产生归墟残灰。

少量残灰：

- 可入药。
- 可淬器。
- 可遮蔽气息。

大量残灰：

- 腐蚀灵草。
- 损伤法器。
- 侵蚀经脉。
- 引来归墟污染者。

这用于限制归墟炉无限使用。
