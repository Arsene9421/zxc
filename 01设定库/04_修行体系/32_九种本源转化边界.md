# 九种本源转化边界

> 本文档补全灵、宝、魂、血、运、法、界、时、道九种本源的使用和转化边界。核心目标：防止“本源万能转换”，让归墟炉、补墟司和旧天庭九器各有边界。

---

## 1. 九种本源总表

| 本源 | 主要含义 | 典型载体 | 主要对应 |
|------|----------|----------|----------|
| 灵 | 灵气、灵脉、天地灵机 | 灵石、灵泉、灵脉、灵草 | 补天尺、镇界印 |
| 宝 | 宝气、器脉、法宝灵性 | 法器、法宝、灵宝、仙器 | 归墟炉、万宝殿 |
| 魂 | 魂魄、魂力、轮回残光 | 残魂、魂灯、轮回灯火 | 轮回灯 |
| 血 | 血脉、真灵源血、族群生机 | 真灵血、族契、祖血池 | 太虚舟、百族线 |
| 运 | 气运、命势、时代潮流 | 宗门气数、王朝国运、系统加权 | 司命盘 |
| 法 | 功法、法术、道统传承 | 玉简、碑文、传承印、万法碑 | 万法碑 |
| 界 | 世界本源、界壁、地脉 | 界核、洞天、残界碎片 | 镇界印 |
| 时 | 时痕、古今残影、流速差 | 古战场、太虚航迹、时砂 | 太虚舟 |
| 道 | 大道规则、道果、道伤、断道 | 道果、道痕、斩道剑痕 | 斩道剑 |

九种本源不是九种蓝条。

它们是万界循环中不同层面的底层资粮。

## 2. 三条基本原则

### 2.1 同源优先

修补什么，优先用同类本源。

| 要修补 | 最合理材料 |
|--------|------------|
| 灵脉衰竭 | 灵源、界源辅助 |
| 法宝破损 | 宝气、同类材料 |
| 神魂裂痕 | 魂力、养魂资源 |
| 血脉枯竭 | 血源、族契、生机 |
| 功法残缺 | 法源、传承碎片 |
| 洞天破裂 | 界源、地脉、镇界之物 |

跨源修补可以救急，但长期会留下偏差。

### 2.2 邻源可导

部分本源存在自然相邻关系，可以低损耗导引。

| 相邻关系 | 说明 |
|----------|------|
| 灵生宝 | 炼器以灵材养出宝气 |
| 宝藏法 | 法宝可承载法术、器纹、传承 |
| 法近道 | 功法修到极处，才可能触及大道 |
| 界生灵 | 世界本源稳定，才能孕育灵机 |
| 血承魂 | 血脉可承载族群记忆和魂印 |
| 运牵命 | 气运会影响命数分支，但不等于命数 |
| 时留影 | 时间本源留下历史残影，不等于逆转历史 |

邻源导引需要材料、阵法、修为、时间和损耗。

### 2.3 跨源有代价

不相邻本源强行转化，必然代价巨大。

比如：

- 用魂力补界源，会产生怨界。
- 用血源补墟眼，会形成血脉债。
- 用气运养法宝，会让法宝绑定命数。
- 用道果补仙界，会让高阶修士道途衰败。
- 用时源改命，会造成更大的命数塌陷。

新仙界补墟体系的短期有效，正是建立在大量跨源强转和代价外包上。

## 3. 归墟炉的本源边界

归墟炉不是万能转换器。

它的本质能力是：

> 回收残败本源，分拣、净化、重炼，让本该腐烂的东西重新进入循环。

归墟炉能做：

- 从废丹里分出残余药性和灵源。
- 从残器里分出宝气和器脉。
- 从古战场里看见魂影和道痕。
- 从残界碎片里聚拢界源。
- 从旧器碎片里识别对应本源。
- 终局让不同本源重新回到各自循环位置。

归墟炉不能做：

- 把普通石头炼成仙器。
- 把魂力直接炼成灵石。
- 把别人气运无代价转给赵长生。
- 把死者残魂炼成完整活人。
- 把时间倒回去重选一遍。
- 把道源当经验条喂主角升级。

## 4. 各本源可转化边界

### 4.1 灵源

灵源是最常用、最底层的修行资源。

可转化：

- 灵源可以养灵草。
- 灵源可以参与炼丹。
- 灵源可以温养法器。
- 灵源可以支持阵法。
- 灵源可以维持洞天低级运转。

不可转化：

- 不能直接变成寿元。
- 不能直接修复神魂。
- 不能无代价生出界源。
- 不能让残缺功法自动补完。

### 4.2 宝源

宝源来自器物灵性、材料承载和器纹结构。

可转化：

- 残器宝气可以修复同类器物。
- 高阶宝源可做阵眼。
- 宝源可承载法源，让法术固化成器纹。

不可转化：

- 不能直接提升境界。
- 不能替代修士悟道。
- 不能无代价修魂。
- 不能被无限拆解再重炼。

### 4.3 魂源

魂源涉及生死，最容易写崩。

可转化：

- 可安魂、养魂、修复轻微神魂裂痕。
- 可作为轮回灯接引残魂的灯油。
- 可提供残影线索。

不可转化：

- 不能给主角当修炼燃料。
- 不能随意炼成法力。
- 不能批量复活亡者。
- 不能绕过轮回灯安排生死。

### 4.4 血源

血源涉及族群、真灵、体质和传承。

可转化：

- 可强化血脉法术。
- 可维系族契和祖地。
- 可稳定某些血脉洞天或真灵阵基。
- 可作为白倩霜线的冲突核心。

不可转化：

- 不能让赵长生吞血升级。
- 不能把真灵血脉写成女主被猎取的福利。
- 不能脱离族群责任随意移植。
- 不能无代价补墟眼。

### 4.5 运源

运源是最容易变成玄学万能钥匙的本源，必须克制。

可转化：

- 可牵引机缘。
- 可护持宗门、王朝或族群度过一劫。
- 可被司命盘记录和调度。
- 可作为系统奖励的底层加权。

不可转化：

- 不能直接换修为。
- 不能保证不死。
- 不能把配角气运献给主角。
- 不能用来解释所有巧合。

### 4.6 法源

法源来自功法、道统、法术结构和传承。

可转化：

- 可补残卷。
- 可推演法术变体。
- 可让器物承载法术。
- 可让宗门道统延续。

不可转化：

- 不能让主角瞬间学会所有功法。
- 不能跳过资质、神魂和实践。
- 不能无限推演无代价神通。
- 不能代替道源。

### 4.7 界源

界源是世界根基。

可转化：

- 可稳定界壁。
- 可孕育灵脉。
- 可支持洞天和小世界。
- 可暂时填补墟眼边缘。

不可转化：

- 不能当普通灵石花。
- 不能无限抽取不伤世界。
- 不能随身携带过量而不引发界压。
- 不能用低阶阵法完整复制。

### 4.8 时源

时源只用于痕迹、流速和太虚定位，不用于随意改历史。

可转化：

- 可看到古今残影。
- 可解释遗迹回溯。
- 可造成洞天内外流速差。
- 可用于太虚航行定位。

不可转化：

- 不能随便回到过去。
- 不能重开失败战斗。
- 不能让人物无限重生刷结果。
- 不能让修炼时间倍率无上限。

### 4.9 道源

道源是最上层的规则本源。

可转化：

- 可修补大道裂痕。
- 可斩断腐朽道统。
- 可支撑终局重炼。
- 可让道君、道祖级存在承担循环职责。

不可转化：

- 不能被低阶主角提前掌控。
- 不能当经验值。
- 不能被普通法宝承载。
- 不能让一个人的意志凌驾万界循环。

## 5. 新仙界的强行转化

新仙界补墟体系的问题不是完全无效，而是有效但错误。

| 强行转化 | 短期效果 | 长期代价 |
|----------|----------|----------|
| 界源填墟眼 | 稳定仙界裂口 | 下界灵脉枯竭、灾劫增多 |
| 魂力填裂缝 | 缓解深层腐蚀 | 轮回堵塞、怨魂滋生 |
| 血源补阵基 | 强化封禁 | 百族被猎取、族运崩坏 |
| 气运续仙界 | 延缓道则衰败 | 天命之子被回收 |
| 宝气补仙器 | 维持封禁器脉 | 飞升者本命法宝被拆 |
| 道果稳玄阙 | 高层续命稳道 | 仙界越发僵化 |

所以终局不是“找到更多材料补墟眼”，而是结束强行转化，让归墟重新成为循环。

## 6. 旧天庭九器分工

九器不是九个外挂，而是九种本源的公共设施。

| 九器 | 正常职责 | 越界风险 |
|------|----------|----------|
| 归墟炉 | 重炼残败本源 | 被误用为万能造物炉 |
| 司命盘 | 记录危机和命数裂痕 | 被改造成安排命运的系统 |
| 登仙台 | 接引修士进入更大循环 | 被改造成飞升回收通道 |
| 补天尺 | 修补世界裂缝 | 只服务仙界本土 |
| 万法碑 | 保存开放道统 | 垄断功法和传承 |
| 轮回灯 | 维持亡魂循环 | 截留亡魂炼资源 |
| 镇界印 | 稳定世界本源 | 锁死下界、抽取界源 |
| 太虚舟 | 横渡太虚、接引新道 | 阻断外部变量 |
| 斩道剑 | 斩断腐朽大道 | 镇压异端、维护僵化秩序 |

## 7. 写作硬规则

1. 本源不能万能互换。
2. 同源修补成本最低，跨源转化代价最高。
3. 魂、血、运、时、道五类本源尤其不能廉价化。
4. 归墟炉能分拣和重炼残败本源，不是凭空造物。
5. 新仙界补墟体系的罪恶在于把跨源代价外包给下界、百族和飞升者。
6. 终局重炼不是赵长生获得所有本源，而是让九类本源回到正确循环。
7. 每次使用高阶本源，都必须留下消耗、污染、反噬、账本或因果后果。
