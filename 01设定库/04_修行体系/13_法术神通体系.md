# 法术与神通体系

## 1. 区分

| 类型 | 含义 |
|------|------|
| 法术 | 法力的具体运用，低中阶常见 |
| 秘术 | 有特殊代价或门槛的法术 |
| 神通 | 与血脉、道途、大道规则相关的高阶能力 |
| 道法 | 接近规则层的运用，后期出现 |

第一卷只写低阶法术。

## 2. 法术分类

| 分类 | 例子 | 用途 |
|------|------|------|
| 攻击 | 火球术、金针术、水箭术 | 正面战斗 |
| 防御 | 土墙术、灵罩术 | 保命 |
| 身法 | 轻身术、清风步 | 逃跑 |
| 隐匿 | 敛息术、遮气术 | 赵长生常用 |
| 生活 | 清尘术、引火术、凝水术 | 日常 |
| 种植 | 小灵雨术、催芽术 | 灵草线 |
| 辅助 | 探灵术、辨药术 | 捡漏和试验 |

## 3. 第一卷可用法术

赵长生适合：

- 清尘术。
- 敛息术。
- 小灵雨术。
- 引火术。
- 简单探灵术。

沈临风适合：

- 火球术。
- 轻身术。
- 护身术。
- 系统辅助下的基础法术熟练度。

敌人适合：

- 火球术。
- 金针术。
- 土墙术。
- 低阶毒雾。
- 简易束缚术。

## 4. 法术限制

- 施法需要法力。
- 低阶修士连续施法会空蓝。
- 法术需要手诀、咒诀或熟练度。
- 法术会留下灵力波动。
- 坊市内斗法容易引来巡市修士。

赵长生不适合频繁正面斗法。

他的战斗方式应是：

- 提前布置。
- 敛息绕路。
- 用符箓和残器补短板。
- 在敌人轻视时反杀。

## 5. 神通

神通后期才出现。

来源：

- 血脉。
- 本命法宝。
- 元婴神魂。
- 旧天庭器物。
- 大道感悟。

禁止前期随便给赵长生强大神通。

归墟炉前期是工具，不是让赵长生立刻获得大神通。
