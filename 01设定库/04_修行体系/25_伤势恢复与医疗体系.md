# 伤势恢复与医疗体系

> 本文档补全战斗后果：受伤怎么分级，丹药能救到什么程度，恢复需要多久。

---

## 1. 伤势分类

修士受伤分六类：

| 类型 | 说明 | 典型来源 |
|------|------|----------|
| 外伤 | 皮肉、筋骨、脏腑损伤 | 法器、妖兽、近身搏杀 |
| 内伤 | 法力震荡、脏腑受冲击 | 法术、护罩破裂 |
| 经脉伤 | 经脉裂伤、堵塞、逆行 | 强行突破、法力反噬 |
| 神魂伤 | 意识、魂魄、神识受损 | 幻术、摄魂、元婴压制 |
| 本源伤 | 命火、寿元、根基受损 | 禁术、归墟污染、重伤濒死 |
| 道伤 | 大道、权柄、因果层损伤 | 太乙以上、旧天庭器、墟眼 |

前期主要写外伤、内伤、经脉伤。

神魂伤中期出现，本源伤后期加重，道伤仙界篇后才常用。

## 2. 恢复手段

| 手段 | 能处理 | 限制 |
|------|--------|------|
| 普通止血包扎 | 外伤 | 对法力伤效果弱 |
| 疗伤丹 | 外伤、轻中内伤 | 丹毒，不能接骨续魂 |
| 续脉丹 | 经脉伤 | 昂贵，恢复慢 |
| 养魂丹 | 神魂伤 | 稀缺，见效慢 |
| 灵泉/灵池 | 肉身和经脉调养 | 被宗门垄断 |
| 医修治疗 | 综合处理 | 费用高，需信任 |
| 闭关静养 | 大多数伤势 | 时间成本 |
| 归墟炉辅助 | 修补部分残缺本源 | 消耗炉火，不能无代价 |

## 3. 恢复时间

| 伤势 | 恢复时间 | 说明 |
|------|----------|------|
| 轻外伤 | 数日 | 丹药可缩短 |
| 重外伤 | 十数日到数月 | 筋骨脏腑需调养 |
| 轻内伤 | 数日到半月 | 法力震荡需平复 |
| 重内伤 | 数月 | 斗法能力下降 |
| 经脉轻伤 | 数月 | 影响修炼速度 |
| 经脉重伤 | 数年或终身 | 可能断道 |
| 神魂轻伤 | 数月到数年 | 易失眠、幻听、心魔 |
| 神魂重伤 | 长期 | 可能失忆、痴迷、夺舍风险 |
| 本源伤 | 很难恢复 | 需要高阶本源或归墟炉后期能力 |

## 4. 丹药不能解决什么

丹药不能：

- 让断肢瞬间完好无损。
- 让重伤者马上满状态战斗。
- 无代价修复神魂。
- 直接补回大量寿元。
- 清除所有归墟污染。
- 修复大道裂痕。

丹药可以救急，但不应抹掉战斗后果。

## 5. 归墟炉与疗伤

归墟炉能处理“残缺”，所以后期可以参与疗伤。

但限制很严：

| 阶段 | 能力 |
|------|------|
| 前期 | 只能辅助处理废丹、药性、少量经脉滞涩 |
| 中期 | 可修补部分法器反噬和轻微本源裂痕 |
| 后期 | 可处理残魂影像、命火裂痕、世界本源伤 |
| 终局 | 处理循环层面的残败 |

不能前期就用归墟炉随便治好重伤。

## 6. 医修体系

医修不是单纯奶妈。

医修能力包括：

- 辨伤。
- 正骨。
- 导引法力。
- 排丹毒。
- 续经脉。
- 安神魂。
- 解毒。
- 调理灵根和道基。

医修也需要资源。

底层修士常常不是治不好，而是治不起。

## 7. 写作规则

1. 战斗后必须有伤势或资源损耗。
2. 重伤恢复要花时间，不能下一章满血。
3. 经脉伤和神魂伤要影响后续突破。
4. 医修、丹药、灵泉都要有成本。
5. 赵长生的谨慎性要体现在避免受伤，而不是受伤后随便修好。
