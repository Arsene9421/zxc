# 寿元与长生体系

> 本文档回答：修仙为什么要突破，长生到底指什么，寿命如何限制人物选择。

---

## 1. 核心定义

本书统一使用“寿元”作为修士寿命体系的核心名词。

相关概念：

| 名词 | 含义 | 用法 |
|------|------|------|
| 寿元 | 一个生命当前境界能承载的自然寿命上限 | 修士最关心的硬指标 |
| 寿限 | 寿元即将耗尽的临界状态 | 老修士闭死关、抢资源的动机 |
| 命火 | 生命仍在燃烧的根本状态 | 与青槐旧灾、轮回灯、归墟炉相关 |
| 命数 | 一生因果轨迹和寿限记录 | 司命盘、天命系统操作的对象 |
| 寿劫 | 寿元将尽或强行延寿引发的衰败关口 | 高境界后期压力 |
| 道衰 | 仙界高阶修士大道腐朽、权柄反噬 | 仙界篇替代单纯老死的压力 |

寿元不是简单数字，而是“肉身、神魂、命数、本源”共同承载的结果。

## 2. 长生不是不死

修仙者追求长生，但本书不能把“成仙”写成绝对不死。

本书采用三层长生观：

| 层级 | 含义 | 问题 |
|------|------|------|
| 延寿 | 通过境界、丹药、灵物延长寿命 | 资源昂贵，越往后越难 |
| 久视 | 高境界后寿元极长，能见王朝兴灭、世界变迁 | 仍有寿劫、道衰、归墟污染 |
| 真正长生 | 不再只延长个人寿命，而是让自身道路进入循环 | 只有道君/道祖层面才触及 |

所以书名《归墟长生》的真正含义不是“赵长生活得最久”，而是：

> 把必然走向残败的生命、世界和仙界，重新放回可循环的新生之中。

## 3. 标准寿元表

以下是正常情况下的平均上限，不是绝对值。资质、功法、伤势、丹毒、归墟污染、命数残缺都会影响实际寿元。

| 阶段 | 境界 | 标准寿元 | 说明 |
|------|------|----------|------|
| 凡人 | 普通人 | 60-90 年 | 富贵、医药、灾荒会影响 |
| 下界 | 炼气 | 100-150 年 | 法力滋养肉身，但仍会衰老 |
| 下界 | 筑基 | 220-300 年 | 道基稳固，肉身明显延寿 |
| 下界 | 金丹 | 500-800 年 | 金丹反哺肉身，开始脱离凡寿 |
| 下界 | 元婴 | 1200-1800 年 | 元婴成形，神魂延续能力增强 |
| 下界 | 化神 | 2500-3500 年 | 元神感应天地，下界寿元顶点 |
| 灵界 | 炼虚 | 5000-7000 年 | 接触虚空灵机，寿元受界源影响 |
| 灵界 | 合体 | 9000-12000 年 | 法身与元神合一，寿元大增 |
| 灵界 | 大乘 | 18000-30000 年 | 一界气运和道果雏形支撑 |
| 灵界 | 渡劫 | 不稳定 | 不是安稳境界，过劫则入仙界，败则被回收或陨落 |
| 仙界 | 飞升过渡期 | 视原境界而定 | 未稳仙躯，不算真正仙寿 |
| 仙界 | 真仙 | 5-10 万年 | 仙躯稳定，但有仙役、仙税、仙伤 |
| 仙界 | 玄仙 | 15-30 万年 | 仙法体系成形，寿元更长 |
| 仙界 | 金仙 | 50-100 万年 | 道果初稳，常被称为长生仙 |
| 仙界 | 太乙 | 200-500 万年 | 触及大道分支，但有道衰 |
| 仙界 | 大罗 | 近乎与一方大道同寿 | 仍受万界因果、墟眼和归墟大潮影响 |
| 仙界 | 道君/道祖 | 不以普通寿元计算 | 与所掌规则和循环绑定 |

数字写正文时不需要频繁报表，但必须作为后台硬规则。

## 4. 寿元如何消耗

寿元不是只按年自然流逝。

会加速消耗寿元的因素：

| 因素 | 影响 |
|------|------|
| 重伤 | 肉身本源受损，寿元折损 |
| 强行突破 | 根基不稳，寿元换境界 |
| 丹毒 | 长期吞丹会侵蚀经脉和命火 |
| 禁术 | 短时间爆发，直接燃烧寿元 |
| 心魔 | 神魂受损，寿元上限降低 |
| 归墟残灰 | 本源被残败气息侵蚀 |
| 天命系统奖励 | 可能透支命数或增加因果绑定 |
| 补墟抽源 | 下界、灵界整体寿元生态被压低 |

这解释底层修士为什么急：

- 炼气修士看似能活一百多年，但前几十年若没有筑基希望，后面气血衰败就很难再冲关。
- 筑基丹不仅是突破资源，也是抢在肉身走下坡路之前改命。
- 老散修晚年会冒险进秘境、黑市、古战场，因为不拼也会死。

## 5. 延寿资源

延寿资源必须稀缺，且有上限。

| 类型 | 作用 | 限制 |
|------|------|------|
| 延寿丹 | 增加少量寿元 | 重复服用效果递减，丹毒明显 |
| 寿元果 | 天然延寿灵果 | 生长周期长，被大势力垄断 |
| 龟息秘术 | 减缓寿元流逝 | 不能修炼或行动，等于沉睡 |
| 血脉换寿术 | 夺取他人生命本源 | 邪术，容易引归墟污染和天机反噬 |
| 本源补寿 | 用高阶本源修补命火 | 成本极高，可能被仙界盯上 |
| 仙界寿籍 | 新仙界登记后的仙寿制度 | 看似延寿，实则纳入仙籍和补墟账本 |

赵长生前期不能靠归墟炉随便炼出延寿丹。

归墟炉只能做到：

1. 修复“本来就有药性残余”的废丹。
2. 恢复少量被损耗但未死尽的生机。
3. 后期修补命火裂痕。

它不能凭空创造寿元。

## 6. 寿元与境界压力

每个大境界都必须有寿元压力。

| 境界 | 寿元压力 |
|------|----------|
| 炼气 | 年轻时不筑基，气血衰败后几乎无望 |
| 筑基 | 筑基后有时间积累，但金丹资源被宗门和家族垄断 |
| 金丹 | 若本命法宝和丹道根基不成，后期战力和突破都弱 |
| 元婴 | 神魂延寿明显，但天机、命数、心魔风险上升 |
| 化神 | 下界寿元已长，但正常仙门断裂，继续留在下界会被困死 |
| 炼虚 | 灵界寿元更长，但万年抽源会让一代代强者被清算 |
| 合体 | 法身越强，越容易被仙界监察记录 |
| 大乘 | 接近一界顶点，成为抽源和飞升筛选重点 |
| 真仙 | 仙寿长，但仙税、仙役、仙籍锁住自由 |
| 太乙/大罗 | 不再怕普通老死，但怕道衰、墟眼、归墟大潮 |

## 7. 寿劫与道衰

低中境界主要怕寿元耗尽。

高境界主要怕寿劫、道衰和本源腐败。

| 类型 | 发生阶段 | 表现 |
|------|----------|------|
| 寿劫 | 炼气到大乘都可能出现 | 命火衰弱、肉身枯败、神魂暗淡 |
| 天人五衰 | 仙界底层到中层 | 仙躯衰败、仙元浑浊、仙籍反噬 |
| 道衰 | 太乙、大罗 | 所修大道腐朽，权柄反噬，因果缠身 |
| 归墟侵蚀 | 接触残界、墟眼、归墟残灰者 | 记忆断裂、本源腐烂、寿元异常流失 |

新仙界补墟制度的可怕之处在于：

> 它表面上延缓仙界道衰，实质上把下界和灵界的寿元、本源和气运拿来替仙界续命。

## 8. 人物应用

| 人物 | 寿元线作用 |
|------|------------|
| 赵长生 | 从底层寿元焦虑出发，最终理解长生不是单纯活久 |
| 沈临风 | 系统奖励可能暗中绑定命数，甚至用寿元作隐藏代价 |
| 叶疏彤 | 生机线，可以修复灵植和地脉，但不能廉价复活人 |
| 白倩霜 | 真灵血脉被猎取，本质是强族夺取弱族血源和寿元潜力 |
| 纪晚凝 | 补墟账本里必须记录“抽源延寿”的真实代价 |
| 洛闻霜 | 轮回灯线，负责强调寿元尽头不等于可随意复活 |
| 裴玄阙 | 用万界寿元延缓仙界道衰的理念反派 |

## 9. 写作原则

1. 寿元要成为人物选择的压力，不要只当背景数字。
2. 老修士争资源不是贪婪标签，而是寿限逼近。
3. 延寿资源越有效，代价越大。
4. 长生不能只写成个人爽点，后期要上升到循环问题。
5. 赵长生终局解决的不是“我一个人怎么不死”，而是“万界为什么不断被迫用别人寿元续命”。
