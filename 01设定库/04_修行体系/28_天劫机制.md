# 天劫机制

> 本文档补全天劫的来源、层级、变化和新仙界篡改后的筛选机制。

---

## 1. 天劫定义

旧天庭时期，天劫是：

> 世界循环对修士境界跃迁的承载测试。

它不只是雷劈。

天劫会检测：

- 肉身能否承载更高法力。
- 神魂能否承载更高道则。
- 道途是否稳定。
- 是否对一界造成失衡。
- 是否适合进入更大循环。

## 2. 天劫类型

| 类型 | 作用 |
|------|------|
| 雷劫 | 锤炼肉身、法力、法宝 |
| 心魔劫 | 检测道心和执念 |
| 风火劫 | 焚炼杂质，考验根基 |
| 神魂劫 | 检测神魂承载 |
| 因果劫 | 高阶修士面对旧因果 |
| 道劫 | 太乙、大罗后面对大道反噬 |
| 归墟劫 | 接近归墟、墟眼、重炼循环时出现 |

低阶不需要频繁出现天劫。大境界越高，劫越明显。

## 3. 旧天劫和新天劫

| 项目 | 旧天劫 | 新天劫 |
|------|--------|--------|
| 本质 | 承载测试 | 筛选和验源工具 |
| 目的 | 判断能否进入更大循环 | 判断是否有回收价值 |
| 失败者 | 魂归轮回或散入归墟 | 可能被提前回收 |
| 渡过者 | 登仙台接引 | 登记、分流、服役、回收 |
| 记录者 | 旧司命盘只记录裂痕 | 新司命台记录命数和资源价值 |

新仙界没有创造天劫，但篡改了天劫后的接引和记录机制，并在部分劫中加入验源功能。

## 4. 各境界天劫压力

| 阶段 | 劫的表现 |
|------|----------|
| 炼气到筑基 | 通常无正式天劫，更多是心性和道基关 |
| 筑基到金丹 | 小心魔、丹成异象、法力反冲 |
| 金丹到元婴 | 心魔和神魂劫明显 |
| 元婴到化神 | 天地感应，开始出现真正天劫和验源痕迹 |
| 大乘到渡劫 | 天劫成为飞升筛选核心 |
| 真仙以后 | 仙劫、道职反噬、天人五衰 |
| 太乙到大罗 | 道劫、因果劫 |
| 终局 | 归墟劫、墟眼反噬、循环重炼劫 |

## 5. 验源机制

新仙界将天劫改造成验源工具。

验的不是单纯善恶，而是价值：

| 验源对象 | 新仙界用途 |
|----------|------------|
| 法宝 | 可夺取、入库、拆解 |
| 血脉 | 可炼血、研究、收归仙族 |
| 气运 | 可引入司命台系统 |
| 神魂 | 可用于轮回灯扭曲体系或仙器 |
| 本源 | 可补墟 |
| 道统 | 可登记、封锁、回收 |

这解释为什么飞升失败者消失得太干净。

## 6. 赵长生与天劫

赵长生的天劫特殊，但不能写成天道偏爱。

特殊原因：

- 命数断页，司命台记录不完整。
- 归墟炉可感应残败本源。
- 他修的路线涉及补残、重炼、旧器。
- 天劫验源时无法完整识别他。

风险：

- 天劫异常会引来新司命台注意。
- 归墟炉波纹可能暴露。
- 命数断页会让劫象紊乱，不一定更轻松。

## 7. 沈临风与天劫

沈临风的天劫会被系统影响。

系统可能：

- 提前提示劫象。
- 提供避劫奖励。
- 引导他走可回收路线。
- 记录他的弱点。

他越依赖系统渡劫，后期越难摆脱司命台。

## 8. 避劫与渡劫资源

| 资源 | 作用 | 限制 |
|------|------|------|
| 避劫丹 | 降低部分劫力 | 不能免劫 |
| 护劫阵 | 分担雷火风劫 | 消耗巨大 |
| 护魂符 | 抵御心魔和神魂劫 | 高阶难得 |
| 本命法宝 | 承接部分劫力 | 可能损坏 |
| 气运护持 | 稳定劫象 | 会牵扯宗门或一界 |
| 登仙台残路 | 绕开部分新仙界机制 | 风险极高 |

## 9. 写作规则

1. 天劫不是随机雷电特效。
2. 天劫要检测修士能否承载更高层规则。
3. 新仙界篡改的是接引、记录、验源和回收，不是凭空制造所有劫。
4. 渡劫成功不等于自由，可能意味着被登记。
5. 赵长生不能靠归墟炉无脑吞劫。
