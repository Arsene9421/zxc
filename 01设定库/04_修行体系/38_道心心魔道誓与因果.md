# 道心、心魔、道誓与因果

> 本文档补全修士为什么会卡心性、为什么不能随便发誓、杀戮和背叛为什么会在突破时反噬。核心目标：让高境界不是资源够了就能升。

---

## 1. 基本定义

| 概念 | 含义 | 用法 |
|------|------|------|
| 道心 | 修士对自身道路、选择和代价的稳定认知 | 决定突破时能不能承受质变 |
| 心魔 | 道心裂缝、恐惧、执念、罪责、污染和命数反噬的显化 | 突破、重伤、幻术、天劫中爆发 |
| 道誓 | 以自身道途、命火、神魂或命数作担保的誓约 | 约束高于口头承诺 |
| 因果 | 行为留下的长期牵连和责任回响 | 不等于宿命，但会在关键时刻追上来 |
| 业债 | 主动伤害、欺骗、背叛、夺取后形成的沉重因果 | 影响心魔、天劫和命数推演 |

道心不是善良值。

一个冷酷修士也可以有稳定道心；一个满口仁义的人也可能道心虚浮。

## 2. 道心如何形成

道心来自长期选择，不来自一句口号。

道心由五层构成：

| 层级 | 内容 |
|------|------|
| 求生认知 | 我为什么要活下去 |
| 修行目标 | 我为什么要修仙、求长生 |
| 行事底线 | 我什么可以做，什么不做 |
| 代价承认 | 我做了选择，就承认后果 |
| 道路承担 | 我的道如果影响别人，我是否承担责任 |

低阶修士可以只靠求生和欲望修炼。

高阶突破时，必须回答更深的问题。

## 3. 心魔来源

心魔不是随机幻觉。

| 来源 | 表现 |
|------|------|
| 道心矛盾 | 平时压下的选择在突破时爆发 |
| 杀戮因果 | 被杀者、幸存者、后果形成回响 |
| 背誓 | 违背道誓，命火和神魂不稳 |
| 功法冲突 | 修行路线互相否定 |
| 丹毒和伤势 | 身体痛苦放大神魂裂缝 |
| 系统诱导 | 沈临风把系统任务误当自己的路 |
| 归墟污染 | 残败记忆和执念侵蚀道心 |
| 命数断页 | 赵长生的旧灾和已死记录形成回响 |
| 族契撕裂 | 白倩霜线中个人和族群责任冲突 |
| 账本罪责 | 纪晚凝线中制度总账压过具体死账 |

心魔可以被幻术引出，但幻术不是心魔本身。

## 4. 心魔等级

| 等级 | 状态 | 后果 |
|------|------|------|
| 0 | 念头杂乱 | 分心、修炼效率下降 |
| 1 | 执念成影 | 梦魇、幻听、打坐不稳 |
| 2 | 心关 | 小境界卡住，法力运转不畅 |
| 3 | 心魔劫 | 大突破时出现幻境、旧人、旧选择 |
| 4 | 道心裂 | 突破失败，神魂或道基受损 |
| 5 | 入魔/失道 | 道路被执念吞掉，可能转邪、归墟污染加重 |

心魔不一定让人发疯。

更高级的心魔，往往表现为“理由越来越正确，但人越来越不能回头”。

裴玄阙的保全派道路就是一种高阶制度心魔：危机真实，方法腐烂，却不肯承认。

## 5. 道誓规则

道誓不是普通发誓。

| 誓约 | 担保对象 | 反噬 |
|------|----------|------|
| 口头誓 | 名誉和关系 | 信誉受损 |
| 心誓 | 道心 | 打坐不稳，心魔加重 |
| 命火誓 | 命火和寿元 | 寿元折损，命火摇晃 |
| 神魂誓 | 神魂 | 违誓后神识刺痛、记忆混乱 |
| 道途誓 | 自身道路 | 突破时形成重关 |
| 族契誓 | 族运和血源 | 牵连族契、祖地 |
| 仙籍誓 | 仙籍、道职、账本 | 被仙界制度追责 |

低阶修士一般不敢随便立重誓。

重誓不是增强情绪的台词，而是剧情债务。

## 6. 违誓后果

违誓不一定立刻天打雷劈。

更合理的后果是：

- 修炼时法力岔路。
- 神识外放时刺痛。
- 突破时心魔重现。
- 天劫验出命数裂痕。
- 司命台或仙籍记录违誓。
- 道侣、族契、宗门关系断裂。
- 关键时刻功法无法圆满运转。

越高阶的誓，反噬越慢但越深。

## 7. 因果规则

因果不是宿命。

因果是行为留下的联系。

| 行为 | 因果 |
|------|------|
| 救人 | 不一定有好报，但可能形成信任、人情、命数交点 |
| 杀人 | 不一定立刻遭报，但会留下追查、亲友、心魔、业债 |
| 夺宝 | 得到资源，也承接物主因果和敌人追索 |
| 背叛 | 短期获利，长期破坏道心和信誉 |
| 立约 | 获得合作稳定，也承担违约代价 |
| 公开真相 | 推动大势，也会引来制度反扑 |

因果不能写成机械报应。

它应当像账：可能暂时不收，但不会凭空消失。

## 8. 杀戮与道心

本书不是不准主角杀人。

但杀人要区分：

| 类型 | 道心影响 |
|------|----------|
| 自保反杀 | 影响较小，但第一次会冲击心性 |
| 杀敌灭口 | 有必要性，但会积累冷硬倾向 |
| 杀无辜 | 业债重，极伤道心 |
| 为资源滥杀 | 容易转邪，归墟污染风险上升 |
| 战争杀戮 | 个体道心和制度因果叠加 |
| 处置墟潮载体 | 痛苦但必要，需承认代价 |

赵长生可以狠，但不能把杀人写成无心理、无追查、无因果的纯爽点。

他的危险不是软弱，而是容易把一切都算成账。

## 9. 主要人物心魔方向

| 人物 | 心魔核心 |
|------|----------|
| 赵长生 | 活下去和承担责任的冲突；把人和事都算成账的冷硬倾向 |
| 沈临风 | 自己到底是主角、棋子，还是一个要自己负责的人 |
| 叶疏彤 | 只救一株草、一块田，能不能面对更大废土 |
| 陆维舟 | 技术中立是否也是逃避责任 |
| 顾长庚 | 正统规则若护不住人，还要不要守旧规矩 |
| 谢未明 | 看见命数是否就有权安排别人 |
| 白倩霜 | 为族群牺牲个人，还是为个人放弃族群 |
| 纪晚凝 | 总账正确能不能抹掉每一笔死账 |
| 洛闻霜 | 守轮回秩序是否会变成见死不救 |
| 秦照古 | 尊重旧天庭牺牲是否等于必须复辟 |
| 裴玄阙 | 危机真实是否证明收割永远正当 |

## 10. 心魔与突破

| 境界 | 心魔表现 |
|------|----------|
| 炼气 | 恐惧、贪念、急躁影响小周天 |
| 筑基 | 道基是否稳定，怕死和贪快会放大 |
| 金丹 | 必须回答“我修的是什么道” |
| 元婴 | 元婴像另一个自己，逃避的问题会具象化 |
| 化神 | 元神感应天地，个人道心接触世界问题 |
| 炼虚/合体 | 与界源、法身、血脉结合，道心不稳会撕裂承载 |
| 大乘/渡劫 | 一界气运、天劫、飞升验源会放大道心裂痕 |
| 真仙以上 | 仙籍、道职、账本和道衰会形成制度心魔 |
| 太乙/大罗 | 自身道路若腐烂，会变成大道层心魔 |

大境界突破不能只写资源。

至少要有一次“心性问题被逼出来”的过程。

## 11. 道侣、族契与心魔

道侣关系可以帮助稳定道心，但不能替代道心。

合理：

- 道侣指出盲点。
- 护法时帮助抵御外部干扰。
- 共同承担选择后的后果。
- 让修士看清自己逃避的问题。

不合理：

- 表白后心魔自动消失。
- 双修解决道心裂痕。
- 女主替赵长生承担业债。
- 族契强迫白倩霜消除个人选择。

## 12. 归墟炉与道心

归墟炉会放大“残缺可炼”的道路。

但它也会带来特殊心魔：

- 赵长生是否会觉得所有残缺都该由他重炼。
- 赵长生是否会把无法救的人视为失败材料。
- 赵长生是否会为了终局牺牲具体的人。
- 归墟炉能处理残败本源，是否意味着他有权处理所有人的生死。

洛闻霜、叶疏彤、白倩霜、纪晚凝都应在不同方向限制赵长生走偏。

## 13. 写作硬规则

1. 道心不是善良值，而是道路稳定性。
2. 心魔来自真实矛盾，不是随机幻觉。
3. 道誓不能随便发，发了就要进账。
4. 因果不是机械报应，但行为后果不会凭空消失。
5. 杀人可以爽，但必须有追查、心理或因果后续。
6. 大境界突破必须至少触及一次心性问题。
7. 道侣、系统、归墟炉都不能替角色跳过道心关。
8. 终局重炼循环之前，赵长生必须回答“有能力重炼，是否就有权重炼”的问题。
