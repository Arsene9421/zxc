# 修炼环境、灵脉与洞府效率

> 本文档补全不同修炼环境的效率差异、灵脉等级、洞府价值和高界灵机承载规则。核心目标：解释宗门、洞府、灵脉、仙籍为什么能决定修炼速度。

---

## 1. 基本原则

修炼不是哪里都一样。

外部环境影响：

- 灵气浓度。
- 灵气纯度。
- 灵脉稳定。
- 是否安全。
- 是否有阵法聚灵。
- 是否有地火、灵泉、药圃、炼器室等配套。
- 是否被宗门、家族、仙界制度监控。

底层修士慢，不只是灵根差，还因为他们连稳定修炼环境都没有。

## 2. 修炼环境效率表

以“普通灵屋”为 1 倍基准。

| 环境 | 效率 | 说明 |
|------|------|------|
| 凡俗无灵地 | 0.1 到 0.3 倍 | 只能勉强恢复，长期修炼很慢 |
| 荒野弱灵地 | 0.3 到 0.6 倍 | 安全差，妖兽和散修威胁多 |
| 坊市棚区 | 0.4 到 0.8 倍 | 灵气杂、人多、噪声、争斗多 |
| 普通灵屋 | 1 倍 | 散修可租，价格不低 |
| 坊市中等灵室 | 1.5 到 2.5 倍 | 按时辰收费 |
| 宗门外门灵室 | 2 到 4 倍 | 需贡献点和排队 |
| 宗门内门灵室 | 4 到 8 倍 | 身份门槛高 |
| 小型洞府 | 3 到 8 倍 | 稳定、安全、可长期经营 |
| 中型洞府 | 8 到 15 倍 | 有灵泉、药圃、阵法 |
| 大宗祖脉节点 | 20 倍以上 | 高阶修士和核心弟子占据 |
| 秘境灵地 | 不稳定，可能极高 | 周期开启，风险大 |
| 残界灵地 | 波动极大 | 有界源，也有污染 |

效率不是唯一指标。

一个安全的 3 倍洞府，往往比一个危险的 10 倍秘境更适合长期修炼。

## 3. 灵脉等级

| 灵脉等级 | 适用范围 | 作用 |
|----------|----------|------|
| 微型灵脉 | 小灵田、小铺、棚区边缘 | 维持低阶灵草和日常恢复 |
| 一阶灵脉 | 炼气修士 | 坊市、外门区域 |
| 二阶灵脉 | 筑基修士 | 小宗门核心、筑基洞府 |
| 三阶灵脉 | 金丹修士 | 大宗门山峰、修仙家族祖地 |
| 四阶灵脉 | 元婴修士 | 修仙国核心、秘境 |
| 五阶灵脉 | 化神修士 | 下界稀有祖脉、古战场深处 |
| 灵界灵脉 | 炼虚以上 | 质量高，但受族群和仙界监察限制 |
| 仙脉 | 真仙以上 | 产仙灵气，受仙界制度管理 |

低阶修士不能长期占用高阶灵脉核心。

原因：

- 灵气太浓，经脉承受不了。
- 高阶灵机杂入后，低阶功法炼化不动。
- 宗门不会让低阶修士浪费核心资源。
- 高阶修士也需要稳定闭关。

## 4. 灵气浓度过高的风险

灵气不是越浓越好。

| 低阶修士强入高浓度环境 | 后果 |
|------------------------|------|
| 经脉胀痛 | 灵气涌入太快 |
| 法力驳杂 | 炼化不及，杂性沉积 |
| 丹田震荡 | 法力压缩不稳 |
| 神魂烦躁 | 灵机冲击感知 |
| 走火入魔 | 功法无法导引高阶灵气 |
| 被高阶存在注意 | 灵脉核心常有阵法和看守 |

所以宗门灵室要分等级。

不是越有钱越能直接进最高灵室。

## 5. 洞府价值

洞府不是房子。

一个合格洞府至少包括：

| 配置 | 作用 |
|------|------|
| 灵脉支点 | 修炼核心 |
| 聚灵阵 | 稳定灵气浓度 |
| 护山阵 | 防敌、防窥、防兽 |
| 静室 | 闭关突破 |
| 丹房 | 炼丹、处理药性 |
| 器室 | 炼器、修残器 |
| 药圃 | 灵草和丹材来源 |
| 水源或灵泉 | 调养肉身、灵植 |
| 退路 | 逃亡通道或隐蔽阵 |

赵长生经营洞府的爽点，不是买房，而是建立自己的资源循环。

叶疏彤、陆维舟在洞府阶段必须有不可替代性。

## 6. 洞府维护成本

洞府不是一次买下永久享受。

| 成本 | 说明 |
|------|------|
| 阵法维护 | 阵旗、灵石、阵盘定期更换 |
| 灵田养护 | 灵土、灵水、除虫、轮作 |
| 安全巡查 | 防散修、妖兽、宗门窥探 |
| 地契和税 | 宗门或修仙国可能收取 |
| 灵脉衰减 | 过度修炼会消耗灵脉 |
| 污染清理 | 归墟残灰、丹毒废水、器炉废渣 |

洞府经营越真实，凡人流滚雪球越可信。

## 7. 宗门为什么垄断修炼速度

宗门控制四类东西：

1. 灵脉。
2. 灵室。
3. 功法。
4. 破关资源。

因此宗门弟子不只是“有人教”，而是处在更高效率环境里。

| 身份 | 可用环境 |
|------|----------|
| 杂役 | 灵气边角，常被杂务压榨 |
| 外门 | 低级灵室、任务换贡献 |
| 内门 | 稳定灵室、丹药、师长指点 |
| 真传 | 灵脉节点、秘境名额、护法 |
| 长老 | 洞府山峰、宗门库存 |

这解释顾长庚的正统道路为什么强，也解释赵长生为什么需要从残缺中补路。

## 8. 下界环境衰败表现

下界被长期抽源后，不是立刻毁灭，而是环境慢慢变差。

| 表现 | 影响 |
|------|------|
| 灵井水变淡 | 坊市底层恢复变慢 |
| 灵田减产 | 丹药和灵米价格上升 |
| 灵脉降阶 | 宗门小辈修炼变慢 |
| 秘境周期变长 | 破关资源更少 |
| 妖兽躁动 | 野外风险提高 |
| 天劫异常 | 高阶突破更危险 |

这些应成为长期伏笔。

## 9. 灵界环境

灵界灵机质量更高，但规则更重。

| 区域 | 效率 | 限制 |
|------|------|------|
| 灵界荒野 | 2 到 4 倍下界普通环境 | 妖兽、百族、空间裂缝 |
| 人族城邦 | 稳定但收费高 | 身份和税 |
| 百族祖地 | 5 到 20 倍 | 族契限制 |
| 残界碎片 | 不稳定 | 污染和界源反噬 |
| 大族核心地 | 极高 | 外族难入 |
| 万年抽源战场 | 波动剧烈 | 修炼和突破都危险 |

赵长生刚入灵界时不能立刻享受高效率。

他要先适应界压、找到功法、建立据点。

## 10. 仙界环境

仙界仙灵气浓度高，但飞升者未必能用。

| 区域 | 资源状态 | 压迫 |
|------|----------|------|
| 飞升台外营 | 有仙灵气但管控严 | 登记、验源、分流 |
| 洗尘池 | 能稳定仙躯 | 检测和剥离本源 |
| 仙矿区 | 仙矿充足 | 劳役、仙税、监控 |
| 飞升者坊市 | 资源流通 | 价格高、黑市多 |
| 残仙器坊 | 捡漏机会 | 污染和追查 |
| 仙府核心 | 高效率 | 派系和仙籍限制 |
| 中天玄阙 | 极高层仙机 | 墟眼污染和制度核心 |

仙界篇仍可保持凡人流：

赵长生不是缺仙灵气，而是缺合法身份、稳定仙籍、干净资源和不被登记的修炼空间。

## 11. 阵法对环境的改变

阵法能改变环境，但不能凭空造资源。

| 阵法 | 作用 | 代价 |
|------|------|------|
| 聚灵阵 | 提高局部灵气浓度 | 消耗灵石，依赖灵脉 |
| 隔绝阵 | 防窥、防干扰 | 维护成本 |
| 净灵阵 | 排杂质、降丹毒环境 | 材料昂贵 |
| 稳脉阵 | 防灵脉暴动 | 需要阵修维护 |
| 护山阵 | 防御洞府 | 消耗巨大 |
| 界阵 | 稳定洞天和残界 | 灵界以后才成熟 |

陆维舟的重要性就在于：他能让资源环境真正变得可用。

## 12. 赵长生环境路线

| 阶段 | 环境目标 |
|------|----------|
| 第一卷 | 从棚区和丹铺边角挤出修炼时间 |
| 第二卷 | 借外门任务接触灵田、丹房废料、低级灵室 |
| 第三卷 | 争取筑基闭关地和资源 |
| 第四卷 | 建立洞府和灵田循环 |
| 第五卷 | 借古战场和断仙门接触高阶灵机 |
| 灵界篇 | 修残界、建据点、处理界压 |
| 仙界篇 | 从废仙石、残仙器和黑籍空间重新起家 |
| 终局 | 让万界重新拥有可循环的修炼环境 |

## 13. 写作硬规则

1. 修炼环境要直接影响速度、恢复和突破安全。
2. 灵气浓度过高也有承载风险。
3. 洞府必须有维护成本，不是永久安全屋。
4. 宗门强大来自灵脉、灵室、功法和破关资源垄断。
5. 灵界和仙界资源更高，但制度门槛更重。
6. 阵法能聚集和调度资源，不能无中生有。
7. 赵长生每次换地图都要重新解决修炼环境问题。
