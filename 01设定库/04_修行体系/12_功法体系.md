# 功法体系

## 1. 功法定义

功法是修士修炼根基。

它决定：

- 灵气吸收效率。
- 法力属性。
- 经脉运行路线。
- 突破上限。
- 战斗倾向。
- 与法术、法宝的适配。

功法不是法术。

功法是“怎么修”，法术是“怎么用”。

## 2. 功法等级

| 等级 | 对应阶段 | 说明 |
|------|----------|------|
| 基础功法 | 炼气 | 大路货，效率低 |
| 宗门功法 | 炼气到筑基 | 有完整路线 |
| 金丹功法 | 筑基到金丹 | 可支撑结丹 |
| 元婴功法 | 金丹到元婴 | 涉及神魂 |
| 化神功法 | 元婴到化神 | 接触天地感应 |
| 灵界道统 | 炼虚到渡劫 | 涉及界源、法身 |
| 仙法 | 仙界 | 涉及仙道法则 |
| 旧天庭经卷 | 规则级 | 不轻易完整出现 |

## 3. 完整度

| 状态 | 含义 |
|------|------|
| 完整 | 可稳定修炼到标注境界 |
| 残缺 | 缺关键层次，突破风险大 |
| 伪改 | 被人乱改，可能走火入魔 |
| 古本 | 用词古老，需要解读 |
| 被锁 | 需要宗门权限或血脉、誓约 |

归墟炉可补残字，但不能绕过理解和试验。

## 4. 赵长生前期功法

开局功法：

> 小五行吐纳法残本。

特点：

- 大路货。
- 五行皆可吸一点。
- 效率低。
- 适合杂灵根但上限不高。

赵长生前期需要：

- 补全残缺字句。
- 找到更适合五行杂灵根的过渡功法。
- 通过归墟炉从破玉简中补出关键运行路线。

## 4.1 赵长生功法路线是否特殊

赵长生的功法有特殊之处，但不是开局就得到无上神功。

正确路线：

> 大路货残本起步 → 归墟炉补残 → 多门低阶五行功法互证 → 形成适合自己的五行归元路线 → 金丹后走万宝归元/归墟重炼之道。

也就是说，赵长生功法的特殊性来自“不断补残、互证、重炼”，不是来自“第一章捡到仙帝功法”。

赵长生开局功法必须弱：

- 小五行吐纳法残本效率低。
- 经脉运行路线缺几处关键转折。
- 普通宗门修士看不上。
- 散修拿来勉强入门。

归墟炉只能补出部分缺字和残句，不能一次生成完整仙经。

## 4.3 功法核心理念

赵长生功法最终不是追求“最纯”，而是追求“能承载、能归元、能重炼”。

核心逻辑：

- 单灵根修士追求纯。
- 剑修追求锐。
- 魔修追求极端。
- 赵长生追求把残缺本源纳入炉中，重新归元。

所以他的功法会越来越像一座人体内的炉：

- 五行法力作炉壁。
- 宝气、药性、生机、魂影、界源作炉料。
- 归墟炉火作重炼核心。
- 命数残缺作入炉资格。

第一卷不要明说这些，只通过他记录试验和补残功法埋下。

## 6. 功法经济

功法很贵。

原因：

- 宗门垄断。
- 完整功法决定突破上限。
- 散修常拿残本修炼，风险高。
- 破玉简市场真假混杂。

第一卷赵长生买不起完整好功法，只能接触残本和破玉简。
