# 全书升级速度表

> 本文档控制全书修为节奏，防止前期太慢、后期太赶，也防止主角无代价飞升。

---

## 1. 总原则

升级速度必须服务三个目标：

1. 凡人流真实感：前期每一小步都要有资源和风险。
2. 长篇节奏：下界 5 卷，灵界 4 卷，仙界 5 卷，不能后期草率。
3. 逻辑闭环：每次大境界突破都对应资源、功法、心性、世界真相推进。

赵长生不能每卷固定升一级，也不能长期原地踏步。

## 3. 节奏风险

| 风险 | 表现 | 修正 |
|------|------|------|
| 前期太快 | 第一二卷就筑基或金丹 | 降低突破速度，强化资源积累 |
| 下界拖太久 | 第七卷还在下界 | 保持第五卷断仙门 |
| 灵界太草率 | 一两卷直接飞升 | 保留百族、抽源、登仙台三大矛盾 |
| 仙界太短 | 只写底层和终局 | 保留司命台、补墟司、旧天庭、墟眼 |
| 升级像流水账 | 每卷只为升境界 | 每次突破绑定世界真相和人物选择 |
| 战力膨胀 | 突破后无压力 | 换地图后重置社会地位和资源压力 |

## 4. 沈临风升级节奏

沈临风有系统，升级可以比普通修士显得顺，但必须有代价。

| 阶段 | 状态 |
|------|------|
| 前期 | 系统奖励让他修炼速度快于赵长生，但实战稳定性不足 |
| 中期 | 系统任务越来越围绕赵长生，奖励开始带有代价 |
| 灵界篇 | 拒绝“远离赵长生”任务，天命值被削，升级速度受挫 |
| 仙界篇 | 不再靠系统升级，而是反向夺取司命台权限 |
| 终局 | 不以战力压过赵长生，而以司命盘规则线完成自身主线 |

沈临风不能一直靠系统无代价领先。

## 5. 女主和长线人物节奏

| 人物 | 升级原则 |
|------|----------|
| 叶疏彤 | 修为不必永远跟赵长生同速，但生机和灵植专业必须跟上主线 |
| 白倩霜 | 血脉和族群责任推动成长，不靠赵长生喂资源 |
| 纪晚凝 | 初登场本就较高，成长主要是制度认知和道职转向 |
| 陆维舟 | 阵法水平跟主线提升，战力不必顶尖 |
| 顾长庚 | 作为正道对照，部分阶段可领先或并行 |
| 谢未明 | 修为服务命数线，不做正面战力核心 |

长线人物不必每个人都跟主角同境界，但必须有能参与主线的专业价值。

## 7. 写作规则

1. 突破不能只为涨战力。
2. 每次大突破都要对应一次认知升级。
3. 换地图后可以提升力量层级，但社会地位和资源压力要重置。
4. 后期不能靠连续突破糊弄剧情。
5. 赵长生最终强在重炼循环，不是单挑所有大罗。
