# 气运、天机与命数推演

> 本文档补全气运、命数、天机、推演、系统提示和命数断页规则。核心目标：让“天命系统”和“天机推演”有用但不万能，让赵长生的断页身份有边界。

---

## 1. 三个概念

| 概念 | 定义 | 主要载体 | 能否被操控 |
|------|------|----------|------------|
| 气运 | 外部机缘、时代势能、群体命势对个人的推力 | 个人、宗门、王朝、族群、世界 | 可聚散、可消耗、可污染 |
| 命数 | 个体一生因果轨迹和寿限记录 | 司命盘、命册、天劫痕、仙籍 | 可记录、遮蔽、误判，不能绝对决定 |
| 天机 | 世界运行中尚未显露的高阶信息 | 星象、因果线、界源波动、旧器残痕 | 可推演片段，代价很高 |

简化判断：

- 气运像“顺风和势”。
- 命数像“路径和账”。
- 天机像“尚未公开的世界信息”。

三者相关，但不能混成一个万能数值。

## 2. 气运不是无敌光环

气运高的人更容易：

- 遇到关键人物。
- 误入机缘现场。
- 在同等条件下多一线生机。
- 被时代浪潮推到关键位置。
- 获得宗门、族群、王朝的资源倾斜。

但气运不能：

- 替代修为。
- 替代功法。
- 替代法力。
- 让人不死。
- 保证每次赌命都赢。
- 让错误选择变成正确选择。

气运强者也会死。气运越强，卷入大局越深，反而更容易被司命台、新仙界、天劫和敌人盯上。

## 3. 气运来源

| 来源 | 说明 | 风险 |
|------|------|------|
| 个体积累 | 破局、修行、救人、立道、夺机缘 | 会被消耗，也会被嫉恨 |
| 宗门供养 | 宗门声望、灵脉、传承加持 | 宗门衰败会反噬 |
| 王朝民望 | 众生生计、香火、国运 | 易被献祭和篡夺 |
| 族群族运 | 血脉延续、祖地、族契 | 白倩霜线核心 |
| 世界界运 | 一界未衰时的整体势能 | 被抽源后会下降 |
| 旧器牵引 | 司命盘、归墟炉等九器造成的局势牵引 | 容易被误判为天命 |
| 系统灌注 | 新司命台临时加权 | 绑定因果，后期会回收 |

气运不是凭空出现。所谓“天降机缘”，往往是已有资源、旧因果、世界危机和个人选择碰到一起。

## 4. 气运可视化

正常修士看不见气运。

能看气运的手段包括：

| 手段 | 可见程度 | 限制 |
|------|----------|------|
| 望气术 | 只能看大概旺衰 | 易被伪装、误导 |
| 天机推演 | 能看趋势和风险 | 消耗神魂、寿元或气运 |
| 司命盘残权 | 能看命数和气运交点 | 会记录观察者 |
| 天命系统 | 用评级、任务、奖励包装结果 | 结果带新司命台偏见 |
| 道君级感应 | 可感应时代大势 | 不能看清所有个人细节 |

沈临风系统显示的“气运值”“机缘评级”不是客观真理，而是新司命台界面化后的筛选结果。

系统不给赵长生评级，不代表赵长生没有气运，而是赵长生命数断页，司命盘读不到完整路径。

## 5. 命数规则

命数记录的是一个人的因果轨迹，不是绝对剧本。

命数包括：

- 出身因果。
- 寿限范围。
- 重大劫点。
- 与他人的因果线。
- 道途倾向。
- 死后归档去向。

命数能被影响：

| 影响方式 | 结果 |
|----------|------|
| 改变重大选择 | 命数分支偏移 |
| 借助旧器遮蔽 | 短时间不被推演 |
| 被司命台篡改 | 任务和奖励引导路径 |
| 被归墟残败侵蚀 | 命数断裂、模糊、空白 |
| 飞升登记 | 命数被仙籍接管 |

命数不能被轻易彻底重写。

真正重写命数，必然牵涉司命盘、轮回灯、天劫、仙籍或本源层代价。

## 6. 赵长生的命数断页

赵长生是命数断页之人。

断页含义：

> 他的命数记录在青槐旧灾中被旧器碎屑、炉灰残铜和归墟残败共同撕开，司命盘无法连续读取。

这带来三种效果：

| 效果 | 表现 | 边界 |
|------|------|------|
| 难以推演 | 天机阁、新司命台看不清他的完整路径 | 不等于隐身 |
| 容易造成误判 | 围绕他的推演结果出现空白、错位 | 仍会留下行为痕迹 |
| 可接触归墟炉 | 不完整命数能靠近残缺循环器 | 不等于被归墟选中 |

断页不是天命加冕。

赵长生仍会受伤、失败、被追踪、被围杀。敌人可以通过交易记录、法力痕迹、人际关系和物证追他，不必靠命数推演。

## 7. 天机推演流程

一次合理推演应包含五步：

1. 确定问题：问得越具体，代价越高。
2. 找到锚点：人、物、地、命牌、血脉、旧物、因果残痕。
3. 付出代价：法力、神魂、寿元、气运、灵材或旧器权限。
4. 得到结果：趋势、警兆、片段、概率、象辞。
5. 承担反噬：误判、神魂伤、命数污染、被反向定位。

天机推演不能直接给“完整剧透”。

适合给：

- 吉凶大势。
- 一个模糊方向。
- 一句象辞。
- 一段残影。
- 某个关键物的异常。
- 某条路代价更大。

不适合给：

- 具体章节答案。
- 敌人完整计划。
- 所有宝物位置。
- 谁一定会死、谁一定会赢。

## 8. 推演代价

| 推演对象 | 代价 |
----------|------|
| 普通低阶修士 | 少量法力和神魂疲惫 |
| 同境修士 | 明显神魂消耗，有误差 |
| 高境修士 | 可能反噬，易被察觉 |
| 命数断页者 | 结果残缺，推演者受污染 |
| 旧天庭器 | 可能看到旧灾残影，神魂受伤 |
| 墟眼、归墟大潮 | 道伤、寿元折损、命数污染 |
| 道君、道祖 | 几乎不可推演，只能看大势 |

谢未明这类人物不能靠推演解决所有剧情。

他的价值应是发现“推演被污染了”，而不是永远算得准。

## 9. 新司命台污染

新司命台已经把很多天机通道改造成筛选和回收工具。

污染表现：

- 推演结果总把人引向飞升通道。
- 气运高者被标记为可培养棋子。
- 天命系统把危险包装成主线任务。
- 天机阁以为自己在看天机，实际读到的是新司命台筛过的信息。
- 宗门和王朝的气运被引导到补墟体系。

这解释了为什么许多“天命之子”越成长，越接近被回收。

## 10. 沈临风系统规则

系统功能不是凭空开挂，而是司命台权限的界面化。

| 系统表现 | 底层逻辑 | 风险 |
|----------|----------|------|
| 任务发布 | 新司命台引导命数分支 | 越做越被绑定 |
| 奖励发放 | 预置资源、机缘牵引、权限兑换 | 奖励有来源和账 |
| 气运评级 | 司命盘筛选结果 | 会误判断页者 |
| 危险提示 | 命数交点预警 | 可能故意延迟或误导 |
| 惩罚机制 | 削减系统加权和气运外壳 | 让沈临风脱离舒适区 |

沈临风拒绝系统任务时，失去的不是“真实气运”，而是新司命台套在他身上的加权、保护和奖励通道。

他会更危险，但也更自由。

## 11. 气运消耗和转移

气运可以被消耗，但不能像灵石一样随意交易。

可行方式：

| 方式 | 说明 |
|------|------|
| 护宗大阵 | 用宗门气运挡灾 |
| 王朝祭天 | 借民望和国运改易大势 |
| 系统任务 | 消耗气运加权换奖励 |
| 血脉族契 | 族运保护核心成员 |
| 旧器调度 | 司命盘调配一线生机 |

禁用方式：

- 主角随手夺别人气运永久变强。
- 气运直接换修为。
- 女主或配角把气运献给主人公。
- 系统无限刷新气运奖励。
- 气运高就不需要逻辑。

## 12. 写作硬规则

1. 气运只能增加机会和牵引，不能替代实力。
2. 命数不是宿命，选择会改变分支，但改变有代价。
3. 天机推演给方向，不给完整答案。
4. 赵长生断页是不被完整记录，不是免疫因果。
5. 沈临风系统显示是司命台接口，不是世界真相。
6. 谢未明的强点应是识别污染、校正误差、付代价看见关键片段。
7. 新仙界的可怕不在于“算得准”，而在于它能污染别人以为真实的天机。
8. 终局新司命盘只能记录危机和自愿选择，不能再安排命运。
