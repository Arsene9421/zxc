# 仙籍、道职与补墟账本

> 本文档补全仙界篇的制度规则：飞升者如何登记，仙籍如何束缚，道职如何成为权力工具，补墟账本如何证明新仙界制度腐败。核心目标：让纪晚凝线和仙界制度线有可写的硬逻辑。

---

## 1. 三个概念

| 概念 | 表层含义 | 深层含义 |
|------|----------|----------|
| 仙籍 | 飞升者进入仙界后的身份登记 | 命数、道途、本源和税役的接管 |
| 道职 | 仙界或旧天庭赋予的职责和权限 | 借公共循环之力履行某种功能 |
| 补墟账本 | 记录补墟资源来源和去向的账册 | 新仙界收割万界的证据链 |

仙界篇不能只写“更高级宗门”。

它的核心压迫是制度化、合法化、账本化。

## 2. 飞升者流程

新仙界飞升流程：

1. 登仙台残道接引。
2. 登仙监初步拘束。
3. 洗尘池清洗和检测。
4. 验源：检查灵、宝、魂、血、运、法、界、时、道痕迹。
5. 录入仙籍。
6. 分流：仙矿、镇界军、仙府、司命台、补墟司监控。
7. 征收仙税和役期。
8. 根据价值决定培养、压榨、拆解或回收。

飞升者以为自己进入仙界，实际是进入登记体系。

## 3. 仙籍等级

| 仙籍 | 对象 | 权利 | 束缚 |
|------|------|------|------|
| 临籍 | 新飞升者临时身份 | 可在飞升台外营活动 | 必须接受验源和分流 |
| 役籍 | 仙矿、劳役、底层飞升者 | 有最低生存配给 | 役期、仙税、禁迁 |
| 军籍 | 镇界军、仙兵 | 战功换资源 | 随时征调，命数受军印绑定 |
| 职籍 | 低中层仙官 | 可执法、调资源 | 道职考核，失职反噬 |
| 府籍 | 仙府、仙族、宗门内籍 | 有庇护和传承 | 绑定派系 |
| 罪籍 | 叛逃、欠税、拒役者 | 几乎无权 | 可被追捕、炼役、抹籍 |
| 黑籍 | 未登记或伪籍者 | 自由度高 | 一旦暴露后果极重 |
| 旧籍 | 旧天庭遗民或旧道职残留 | 可接触旧权限 | 被新仙界视为高危 |

赵长生进入仙界后不应直接成为逍遥真仙。

他要面对临籍、验源、洗尘池、仙税、黑市伪籍等问题。

## 4. 仙籍记录什么

仙籍不是户口本，而是命数和资源账本的接口。

记录内容：

- 姓名或道号。
- 出身世界。
- 飞升通道。
- 渡劫痕迹。
- 灵根和道基。
- 本命法宝。
- 气运评级。
- 血脉异常。
- 系统痕迹。
- 道统来源。
- 欠税、役期、功过。
- 是否有旧天庭器痕迹。

仙籍越完整，越容易获得资源，也越容易被控制。

黑籍越自由，越难在仙界合法行动。

## 5. 仙税

仙税不是单纯交钱。

可征收内容：

| 仙税类型 | 内容 |
------------|------|
| 仙石税 | 常规货币和资源 |
| 役期税 | 去仙矿、镇界军、补天工程服役 |
| 宝税 | 上缴法宝、材料、器纹 |
| 法税 | 登记功法、传承、秘术 |
| 运税 | 气运强者接受司命台任务 |
| 血税 | 特殊血脉被抽样、登记、征调 |
| 界税 | 出身世界被编入抽源名单 |
| 道税 | 高阶修士道果、道痕被征用 |

底层飞升者真正可怕的不是贫穷，而是连“未来能修什么、去哪里、欠多少”都被写进账里。

## 6. 道职的旧义与新义

旧天庭时期，道职是公共职责。

| 旧道职 | 正常职责 |
|--------|----------|
| 镇界使 | 稳定一界，不让世界崩坏 |
| 补天官 | 修补天裂和地脉 |
| 司命使 | 记录命数危机，不安排命运 |
| 守灯人 | 护持轮回灯，引亡魂入循环 |
| 登仙引使 | 接引飞升者进入更大循环 |
| 万法守碑人 | 保存并开放道统 |
| 太虚舟师 | 横渡太虚，寻找新道 |

新仙界时期，道职变成权力和征收资格。

| 新道职 | 表层说法 | 实际功能 |
|--------|----------|----------|
| 镇界仙官 | 维护下界稳定 | 锁界、抽源、镇压 |
| 补天仙官 | 填补天阙 | 分拣本源投入墟眼 |
| 司命仙官 | 扶持天命 | 筛选、培养、回收棋子 |
| 登仙监官 | 管理飞升 | 登记、验源、分流 |
| 洗尘使 | 洗去下界尘缘 | 检测、剥离、标记 |
| 万宝库使 | 管理仙宝 | 收缴、拆解飞升者法宝 |

这给纪晚凝提供复杂性：

她不是天然恶人。她的道职确实在维持一个快漏穿的仙界，只是这套维持方式已经把万界当账面耗材。

## 7. 道职权限和代价

道职能给仙官力量，但不是免费。

| 权限 | 作用 | 代价 |
|------|------|------|
| 法旨 | 借仙界制度压人 | 绑定制度因果 |
| 印信 | 调动阵法、仙兵、账册 | 失印会反噬 |
| 职火 | 执行道职时的法力加成 | 越职使用会烧命火 |
| 道契 | 与仙界大阵相连 | 仙界受损时共同承担 |
| 账权 | 查看、修改、封存账本 | 伪账会形成道职污点 |

道职不是境界。

低境仙官持有道职时，可以压制无籍飞升者；高境黑籍修士若没有道职，也会被仙界制度限制行动。

## 8. 补墟账本结构

补墟账本分四层：

| 层级 | 名称 | 内容 | 谁能看 |
|------|------|------|--------|
| 第一层 | 明账 | 仙税、役期、飞升者分流 | 普通仙官 |
| 第二层 | 内账 | 本源分类、补墟额度、损耗 | 补墟司核心 |
| 第三层 | 黑账 | 强制抽源、系统回收、炼魂、血税 | 极少数高层 |
| 第四层 | 旧账 | 旧天庭时期正常循环账和新仙界篡改痕迹 | 旧天庭遗迹、纪晚凝后期接触 |

纪晚凝的“账本关”应当来自这四层账逐步对不上。

她一开始相信明账和内账，后来发现黑账，最后拿旧账对照，确认制度已经背叛了它声称要保护的万界。

## 9. 账本记录的本源流向

| 账目 | 来源 | 去向 |
|------|------|------|
| 灵账 | 灵脉、灵石、灵田产出 | 仙矿、阵基、墟眼边缘 |
| 宝账 | 飞升者法宝、古宝、残仙器 | 万宝库、封禁器脉 |
| 魂账 | 渡劫失败者、亡魂、轮回残片 | 墟眼深层裂缝 |
| 血账 | 真灵血脉、特殊体质 | 血阵、封禁阵基 |
| 运账 | 天命之子、宗门国运 | 新司命台、仙界道则续命 |
| 法账 | 功法、传承、万法碑碎片 | 仙界垄断道统 |
| 界账 | 下界、中层世界界源 | 稳定中天玄阙 |
| 时账 | 遗迹时痕、太虚航迹 | 封锁外部变量 |
| 道账 | 道果、道痕、高阶修士 | 高层续命、斩断异端 |

补墟账本的可怕之处：

每一笔看起来都合理，但每一笔背后都有具体的人、族群、世界在失血。

## 10. 伪账和坏账

新仙界不是所有人都知道真相。

常见伪账：

| 伪账名义 | 真相 |
----------|------|
| 自然灾损 | 被抽源后的下界衰败 |
| 飞升失败 | 渡劫者被提前回收 |
| 妖族内乱 | 仙界挑动百族战争 |
| 叛修处理 | 拒绝仙籍和仙税的飞升者 |
| 补天消耗 | 投入墟眼的跨源强转 |
| 道统保护 | 万法碑传承被封锁 |

坏账表现：

- 投入资源越来越多，墟眼稳定时间越来越短。
- 下界可抽资源越来越少。
- 飞升者质量下降。
- 归墟污染者变多。
- 仙界高层要求扩大抽源范围。

这形成制度死循环。

## 11. 赵长生、沈临风、纪晚凝的关系

### 11.1 赵长生

赵长生的问题：

- 命数断页，仙籍录入会异常。
- 归墟炉会扰动验源。
- 他的法宝、命火、旧灾痕迹都不适合被洗尘池检测。
- 他可能需要伪籍、黑市身份或纪晚凝暂时压账。

赵长生对账本的态度应当从“躲账”变成“清账”。

终局不是烧掉所有账，而是让万界知道账从哪里来、欠给谁、谁该还。

### 11.2 沈临风

沈临风是系统持有者。

他的系统奖励、任务、气运加权都能在新司命台后台找到对应账目。

他后期反抗系统，不只是砸系统界面，而是要夺回司命盘规则：

> 司命盘只记录危机，不安排命运。

### 11.3 纪晚凝

纪晚凝是补墟司仙官。

她的成长线：

1. 相信制度必要。
2. 发现明账和内账有误差。
3. 查到黑账。
4. 拿旧账对照。
5. 确认补墟体系已经从临时自救变成永久收割。
6. 不因爱情倒戈，而因证据和责任改道。

她不能只是给赵长生递情报。

她要懂制度、懂账、懂代价，也要在终局承担清账责任。

## 12. 终局新规则

终局不能变成赵长生掌握新仙界账本后继续统治。

新循环的方向：

- 仙籍改为通行记录，不再接管命数。
- 道职恢复公共职责，不再变成收割权力。
- 司命盘记录危机和自愿选择，不发布强制天命。
- 补墟账本公开旧账，确认责任和赔付。
- 下界、中层世界、仙界都能查看自身被抽源记录。
- 墟眼从吞噬账目变成循环炉口，残败本源自愿入循环。

修仙仍有竞争、危险、债务和代价，但不再把制度性收割伪装成飞升奖励。

## 13. 写作硬规则

1. 仙界压迫要制度化，不只靠坏人作恶。
2. 仙籍给身份和资源，也接管命数、税役和行动边界。
3. 道职是职责和权限的结合，不是普通官衔。
4. 补墟账本必须能串起飞升、抽源、系统、百族和墟眼。
5. 纪晚凝转变必须靠账本证据和责任，不靠恋爱冲动。
6. 赵长生终局不能私有账本，必须清账和改规则。
7. 新循环不是无政府，而是取消命数垄断和本源收割。
