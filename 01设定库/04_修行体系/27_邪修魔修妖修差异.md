# 邪修、魔修、妖修差异

> 本文档补全不同修行路线，避免所有敌人都只是换皮人族修士。

---

## 1. 总原则

本书主流修士修法力。

不同修行路线的差异，不是完全换一套能量，而是：

- 获取资源的方式不同。
- 功法路线不同。
- 心性代价不同。
- 身体和神魂承载方式不同。
- 与世界秩序的关系不同。

## 2. 正道修士

正道不是绝对善良，而是相对稳定、可传承、可被宗门制度接纳。

特点：

- 重视功法完整。
- 重视道基稳定。
- 重视宗门、师承、资源秩序。
- 排斥明显采补、血祭、魂炼。

问题：

- 容易垄断资源。
- 容易把底层排除在规则之外。
- 大宗门也会腐败。

## 3. 散修

散修不是一个修行路线，而是身份状态。

特点：

- 功法残缺。
- 资源不稳。
- 战斗实用。
- 更重视保命、交易、捡漏。

赵长生前期气质接近底层散修，但他不是无底线散修。

## 4. 邪修

邪修定义：

> 以夺取他人寿元、精血、魂魄、命火、本源为主要修炼手段的修士。

常见手段：

- 采补。
- 血祭。
- 炼魂。
- 尸傀。
- 毒修。
- 人丹。
- 借命延寿。

邪修强在短期提升快，弱在后果极重。

代价：

- 法力驳杂。
- 心魔重。
- 天机污浊。
- 神魂不稳。
- 容易被归墟残败污染。

## 5. 魔修

魔修不等于邪修。

魔修定义：

> 以极端情绪、执念、杀伐、欲望或逆道法门磨炼法力和道心的修士。

魔修可以有自己的道，不一定都采补炼魂。

特点：

- 突破快。
- 战斗强。
- 心魔风险高。
- 容易走极端。
- 不被正道宗门接纳。

魔修和邪修区别：

| 类型 | 核心 |
|------|------|
| 魔修 | 用自身执念和极端法门修行 |
| 邪修 | 以掠夺他人生命、魂魄、本源为主要资源 |

魔修可以成为对手、盟友或悲剧人物；邪修大多是资源掠夺者。

## 6. 妖修

妖修定义：

> 妖族、兽类、草木、异类生灵开启灵智后修炼法力和血脉的路线。

妖修不只是人形修士换皮。

特点：

- 肉身强。
- 血脉影响大。
- 化形是重要关口。
- 族群传承往往比功法文字更重要。
- 真灵血脉可承载高阶本源。

妖修修炼重点：

| 阶段 | 重点 |
|------|------|
| 开灵 | 开启灵智，吸纳灵气 |
| 炼骨/炼血 | 强化妖身和血脉 |
| 化形 | 能以人形修行更复杂功法 |
| 真灵返祖 | 激发族群源血 |
| 族运承载 | 高阶妖修与族群命运绑定 |

白倩霜线要体现：真灵血脉不是美貌标签，而是会被制度追猎的高价值资源。

## 7. 鬼修与魂修

鬼修、魂修以后可出现。

定义：

- 鬼修：肉身已失或半亡状态，以阴魂修行。
- 魂修：活人专修神魂法门。

限制：

- 怕阳火、雷法、轮回灯规则。
- 神魂资源稀缺。
- 容易被邪修炼成材料。

轮回灯线会限制魂修不能随意逃出生死循环。

## 8. 与归墟污染者的区别

归墟污染者不是邪修、魔修、妖修。

他们是：

> 被未重炼的归墟残败本源侵蚀的修士、仙人、器物或世界残民。

区别：

| 类型 | 是否主动选择 | 核心问题 |
|------|--------------|----------|
| 邪修 | 多数主动 | 掠夺他人修行 |
| 魔修 | 主动或半主动 | 执念和极端道途 |
| 妖修 | 种族路线 | 血脉与族群 |
| 归墟污染者 | 多数被侵蚀 | 循环堵塞后的残败污染 |

## 9. 写作规则

1. 不要把魔修和邪修混成一类。
2. 妖修要有血脉、族群、化形、源血逻辑。
3. 邪修可以很爽地被杀，但也要有资源诱惑逻辑。
4. 魔修可以有理念，不必全是低级坏人。
5. 归墟污染者要服务主题：归墟本身不是恶，未被循环重炼的残败才会污染。
