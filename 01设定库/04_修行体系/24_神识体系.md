# 神识体系

> 本文档补全探查、锁定、储物袋禁制、神魂压制等规则。

---

## 1. 神识定义

神识是修士神魂外放后的感知和操控能力。

它不是单纯视力，也不是精神力万能外挂。

神识主要用于：

- 探查周围。
- 锁定敌人。
- 操控法器。
- 检查禁制。
- 打开储物袋。
- 识别部分灵力波动。
- 高阶时进行神魂攻击或防御。

## 2. 神识与境界

| 境界 | 神识状态 |
|------|----------|
| 炼气初期 | 只能模糊感知近处灵力波动 |
| 炼气中期 | 可短暂外放，辅助操控法器 |
| 炼气后期 | 可探查小范围动静，识别简单禁制 |
| 筑基 | 神识稳定外放，斗法中能锁定法器 |
| 金丹 | 神识覆盖范围明显扩大，可精细操控本命法宝 |
| 元婴 | 神识质变，可形成压制和元婴感应 |
| 化神 | 元神感应天地，可触及天机和界面异常 |
| 灵界高阶 | 神识接触空间、血脉、界源变化 |
| 仙界 | 仙识，可记录仙籍、道职、仙禁 |

第一卷不要大量写“神识一扫百丈”。

## 2.1 神识数值基准

神识必须有数值边界，但正文不建议频繁写成面板。

以下数值是“无强阵法遮蔽、无神魂伤、环境普通、目标没有刻意隐匿”的基准。

换算：

- 1 丈约 3 米。
- 1 里约 500 米。

| 境界 | 稳定精查半径 | 粗略警戒半径 | 极限一瞬半径 | 持续时间 |
|------|--------------|--------------|--------------|----------|
| 炼气初期 | 1 到 3 丈 | 3 到 5 丈 | 8 丈以内 | 10 到 30 息 |
| 炼气中期 | 3 到 8 丈 | 10 到 15 丈 | 20 丈以内 | 半盏茶 |
| 炼气后期 | 10 到 30 丈 | 40 到 60 丈 | 80 丈以内 | 一盏茶 |
| 筑基初期 | 50 到 100 丈 | 150 到 200 丈 | 300 丈以内 | 一刻钟 |
| 筑基后期 | 150 到 300 丈 | 400 到 600 丈 | 1 里以内 | 半个时辰 |
| 金丹初期 | 1 到 3 里 | 5 到 8 里 | 10 里以内 | 半个时辰到一时辰 |
| 金丹后期 | 5 到 10 里 | 20 到 30 里 | 50 里以内 | 数个时辰 |
| 元婴初期 | 20 到 50 里 | 80 到 150 里 | 200 里以内 | 半日以内 |
| 元婴后期 | 80 到 150 里 | 300 到 500 里 | 800 里以内 | 一日以内 |
| 化神 | 300 到 800 里 | 千里到三千里 | 五千里以内 | 大范围感应不可久持 |
| 炼虚 | 一州局部 | 一州大势 | 跨州一瞬 | 主要感应空间和界源，不做精密扫描 |
| 合体 | 数州局部 | 一界局部大势 | 跨域一瞬 | 以法身和天地共鸣为主 |
| 大乘/渡劫 | 一界关键节点 | 一界大势 | 接近界壁层面 | 会触动天劫和仙界监察 |
| 真仙以上 | 仙识按仙籍、道职、仙禁加权 | 可借仙界制度扩大 | 越权会留账 | 受仙籍和道职约束 |

### 2.1.1 如何理解这些数值

稳定精查：

- 能分辨人、物、禁制、法力波动。
- 能辅助开储物袋、查符箓、控法器。
- 消耗最大。

粗略警戒：

- 只能知道有动静、有灵力、有危险。
- 看不清细节。
- 适合巡逻和防偷袭。

极限一瞬：

- 瞬间扫过。
- 消耗大，容易暴露。
- 对强者、旧器、污染物可能反噬。

化神以上的“大范围”多是天地感应，不是低阶意义上的精密扫描。不能写成化神一念看清千里内每个人的钱袋。

## 2.2 战斗锁定距离

战斗中可稳定锁定敌人的距离，远小于平时探查范围。

| 境界 | 稳定锁定距离 | 可同时精控目标 |
|------|--------------|----------------|
| 炼气初期 | 1 到 2 丈 | 1 件低阶法器都吃力 |
| 炼气中期 | 3 到 5 丈 | 1 件法器或 1 张符箓 |
| 炼气后期 | 8 到 15 丈 | 1 到 2 件低阶法器 |
| 筑基 | 30 到 80 丈 | 2 到 4 件法器或阵旗 |
| 金丹 | 200 丈到 1 里 | 本命法宝精控，兼顾数件法器 |
| 元婴 | 3 到 10 里 | 可神识压制低境修士 |
| 化神 | 30 到 100 里 | 可借天地感应锁定界面异常 |
| 炼虚以上 | 看环境和界源共鸣 | 不宜频繁写成远程点杀 |

低阶斗法仍以视线、地形、法术射程和法器速度为主。

神识锁定只是提升命中和操控，不是必中。

## 2.3 环境削弱

神识范围会被环境大幅压缩。

| 环境 | 削弱 |
|------|------|
| 坊市人群、灵力杂乱 | 精度下降三成到五成 |
| 雨夜、山林、矿洞 | 范围下降二成到六成 |
| 阵法遮蔽 | 视阵法等级，可下降五成到完全隔绝 |
| 符箓遮掩 | 对低阶神识有效，高阶可破 |
| 灵脉紊乱 | 容易误判灵力波动 |
| 古战场 | 残魂和杀气干扰，神魂反噬风险上升 |
| 归墟污染区 | 不建议直接探查，可能形成神魂伤 |
| 高阶修士附近 | 探查者可能被反向锁定 |

第一卷青石坊市属于灵力杂乱、阵法零碎、人群密集环境。

赵长生炼气初中期即使能外放神识，也只能短距离确认异常，不能扫完整个坊市。

## 2.4 储物袋与神识数值

储物袋禁制按神识强度和技巧处理。

| 储物袋等级 | 原主状态 | 低阶修士处理时间 | 风险 |
|------------|----------|------------------|------|
| 低阶无主袋 | 原主死去数日 | 炼气后期数个时辰，筑基半个时辰 | 损坏袋内脆弱物 |
| 低阶有主袋 | 原主活着 | 同境很难强开 | 被原主察觉 |
| 筑基级储物袋 | 原主死去不久 | 筑基数个时辰，炼气基本无力 | 禁制反噬 |
| 金丹级储物袋 | 原主死亡 | 金丹以下不宜强开 | 自毁禁制 |
| 特殊储物器 | 有阵法或血契 | 需专业阵法、神识或钥匙 | 可能毁物伤魂 |

杀人夺宝必须有后续处理成本。

## 2.5 神识攻击数值边界

神识攻击不是低阶常规手段。

| 境界差 | 效果 |
|--------|------|
| 同小境界 | 只能短暂干扰，失败会头痛、眩晕 |
| 高一小境界 | 可打断施法或控器一瞬 |
| 高一大境界 | 可形成明显压制，但仍需防反噬和护魂法器 |
| 高两大境界 | 可碾压低境神魂，但容易被天劫、因果或护魂物记录 |
| 低境攻击高境 | 多数反噬，除非有特殊魂器、阵法、污染或偷袭 |

元婴以前，神识攻击只适合写成干扰、眩晕、错判。

元婴以后，神识压制才成为真正的战斗压迫。

## 3. 神识限制

| 限制 | 说明 |
|------|------|
| 消耗神魂 | 连续外放会疲惫 |
| 容易暴露 | 神识探查会被高阶修士察觉 |
| 被禁制阻挡 | 阵法、符箓、禁制可屏蔽 |
| 容易反噬 | 探查强者、旧器、污染物可能受伤 |
| 精度有限 | 低阶神识不能替代所有观察 |

## 4. 储物袋禁制

储物袋不是杀人后随手打开。

规则：

- 低阶储物袋多有简单神识烙印。
- 主人死后烙印会减弱，但不会立刻消失。
- 强行打开可能损坏储物袋。
- 储物袋内若有禁制，可能毁物或反噬。
- 抹除烙印需要时间、神识和技巧。

这让杀人夺宝有后续成本。

## 5. 神识战斗

低阶阶段，神识主要辅助战斗。

高阶阶段，神识可以参与攻击：

| 类型 | 作用 |
|------|------|
| 神识锁定 | 提升法器和法术命中 |
| 神识压制 | 高境界压低境界时常用 |
| 神魂刺 | 短暂扰乱敌人意识 |
| 幻术 | 影响感知和判断 |
| 护魂 | 防止被摄魂、夺舍、心魔侵入 |

神识攻击必须有代价。失败后可能神魂受损。

## 6. 赵长生与神识

赵长生前期神识不强。

他的优势不是神识范围大，而是：

- 观察细。
- 记账准。
- 对残缺气息敏感。
- 归墟炉能感应部分残败本源。

归墟炉感应不能替代神识。

它只能对特定残缺、本源、旧器产生反应，不是万能雷达。

## 7. 沈临风与神识

沈临风前期可能靠系统提示弥补神识不足。

但系统提示不是神识：

- 提示可能延迟。
- 提示可能被司命台操控。
- 提示可能只给任务相关信息。
- 系统越帮他，他越被记录。

## 8. 写作规则

1. 神识不能成为万能侦察。
2. 低阶神识必须范围小、消耗大、精度有限。
3. 探查强者和旧器有风险。
4. 储物袋、禁制、阵法都应和神识规则挂钩。
5. 元婴以后神识才成为真正强压迫手段。
