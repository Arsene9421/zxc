# 符箓体系

## 1. 符箓作用

符箓是低阶修士最重要的外力之一。

用途：

- 攻击。
- 防御。
- 遁逃。
- 传讯。
- 隐匿。
- 治疗。
- 辅助生活。

符箓适合第一卷，因为它一次性、昂贵、能制造资源压力。

## 2. 符箓构成

| 材料 | 作用 |
|------|------|
| 符纸 | 承载符力 |
| 符墨 | 储存灵性 |
| 符笔 | 引导灵力 |
| 符胆 | 符箓核心结构 |
| 封灵粉 | 防止灵力散失 |

符笔和符纸都可有残缺，适合归墟炉试验。

## 3. 符箓等级

| 等级 | 对应境界 |
|------|----------|
| 一阶 | 炼气 |
| 二阶 | 筑基 |
| 三阶 | 金丹 |
| 四阶 | 元婴 |
| 五阶 | 化神 |
| 灵符 | 灵界 |
| 仙符 | 仙界 |

第一卷只出现一阶符。

## 4. 第一卷常用符

| 符箓 | 用途 |
|------|------|
| 护身符 | 抵挡一次攻击 |
| 火弹符 | 低阶攻击 |
| 清风符 | 短距加速 |
| 隐气符 | 短时遮掩气息 |
| 传音符 | 短距传讯 |
| 净尘符 | 清洁 |
| 小疗符 | 临时止血 |

沈临风新手奖励可以是低阶护身符。

## 5. 限制

- 符箓多为一次性。
- 激发需要法力。
- 旧符可能失效。
- 高阶符箓价格昂贵。
- 低阶修士不能随意激发高阶符。

## 6. 归墟炉和符箓

归墟炉前期可处理：

- 裂纹符笔恢复一点灵性。
- 旧符纸恢复部分承载力。
- 废符显出残缺符纹。

不能：

- 把废符变成高阶仙符。
- 凭空生成完整符道传承。
