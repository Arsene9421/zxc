# 下界境界

## 1. 总览

下界境界：

> 炼气 → 筑基 → 金丹 → 元婴 → 化神

下界气质是资源贫瘠、功法残缺、飞升传说断裂。

## 2. 炼气期

炼气期分九层，另有大圆满。

| 小阶段 | 状态 | 常见人物 | 资源需求 |
|--------|------|----------|----------|
| 炼气一层 | 能感应灵气，施展少量小术 | 新入门散修、杂役中较强者 | 引气诀、灵米 |
| 炼气二层 | 法力可稳定运行小周天 | 普通底层散修 | 聚气丹、灵石 |
| 炼气三层 | 可稳定使用低阶法器和符箓 | 坊市打手 | 法器、符箓、恢复丹 |
| 炼气四层 | 初入炼气中期，战力明显高于底层 | 小帮派头目 | 更好功法、闭关资源 |
| 炼气五层 | 可担任低阶管事 | 上宗外门管事 | 稳定灵石来源 |
| 炼气六层 | 炼气中期圆满 | 资深散修 | 丹药、法器 |
| 炼气七层 | 炼气后期 | 小家族修士、宗门外门精英 | 筑基准备 |
| 炼气八层 | 法力深厚 | 宗门重点外门 | 筑基丹消息 |
| 炼气九层 | 炼气圆满前 | 宗门精英、散修头目 | 筑基丹主药 |
| 炼气大圆满 | 冲击筑基前状态 | 少数精英 | 筑基丹、护法、灵脉 |

第一卷修为边界：

| 人物 | 修为建议 |
|------|----------|
| 赵长生开局 | 炼气一层巅峰或炼气二层初 |
| 赵长生卷尾 | 炼气二层稳定，触及炼气三层门槛 |
| 沈临风开局 | 炼气二层，系统奖励后有炼气三层保命能力 |
| 丹铺掌柜周槐安 | 炼气三层 |
| 小帮派头目 | 炼气四层 |
| 外门管事蒋怀礼 | 炼气五层 |

## 3. 筑基期

筑基分初期、中期、后期、大圆满。

核心变化：

- 法力由气态转为液态。
- 丹田成道基。
- 寿元增加。
- 可以长期驱使更强法器。

核心瓶颈：

- 筑基丹。
- 安全闭关地。
- 护法。
- 灵脉支持。

筑基丹必须稀缺。它是宗门控制底层修士的关键资源。

## 4. 金丹期

金丹分初期、中期、后期、大圆满。

核心变化：

- 法力凝成金丹。
- 本命法宝开始成为核心战力。
- 寿元显著增加。
- 可参与修仙国大势。

核心瓶颈：

- 结丹机缘。
- 本命法宝材料。
- 高阶灵药。
- 古丹炉或丹方。

赵长生金丹阶段应形成“万宝金丹”或类似道路，但必须从归墟炉长期重炼残器和宝气积累而来。

## 5. 元婴期

元婴分初期、中期、后期、大圆满。

核心变化：

- 元婴成形。
- 神魂离体能力增强。
- 能接触气运、天机、界面裂缝。

核心瓶颈：

- 神魂材料。
- 完整功法。
- 宗门或一地气运。
- 避免被天机和仙界监察捕捉。

沈临风系统“远离赵长生”的爆点，可以安排在元婴到化神前后。

## 6. 化神期

化神分初期、中期、后期、大圆满。

核心变化：

- 元神与天地相感。
- 能触及界面之力。
- 能感知飞升通道异常。
- 开始被新仙界筛选机制注意。

核心瓶颈：

- 天劫。
- 一界气运。
- 道则感应。
- 飞升通道。

化神圆满不是全书顶点，而是下界顶点和灵界/仙界真相的门槛。

## 7. 下界战力原则

不要频繁越大境界杀敌。

可允许：

- 赵长生靠布置、残器、阵法、毒、信息差越小层级反杀。
- 归墟炉帮助他补足资源劣势。
- 沈临风系统提供一次性保命或任务提示。

禁止：

- 炼气二层正面碾压炼气五层。
- 筑基随手杀金丹。
- 归墟炉直接让赵长生跨大境界无代价杀敌。
