# 修行职业与技艺成长

## 1. 为什么需要职业体系

修仙长篇不能只有境界。

丹、器、符、阵、灵植、驭兽、傀儡、卜算等技艺，是资源循环和剧情爽点来源。

赵长生不需要样样顶尖，但他的归墟炉路线天然适合接触多种技艺。

职业晋升细则见：`35_修行职业晋升细则.md`。

## 2. 职业总表

| 技艺 | 作用 | 赵长生关联 |
|------|------|----------|
| 炼丹 | 丹药、废丹、丹方 | 开局最先接触 |
| 炼器 | 法器、残器、宝气 | 中期本命法宝 |
| 制符 | 符箓、符笔、符纸 | 第一卷保命资源 |
| 阵法 | 阵盘、阵旗、洞府 | 谨慎和地利 |
| 灵植 | 灵草、灵田 | 洞府经营 |
| 驭兽 | 灵兽、妖族、真灵血脉 | 海外妖族卷 |
| 傀儡 | 残器、机关 | 可作为副线 |
| 卜算 | 天机、命数 | 系统和司命台对照 |
| 炼魂 | 残魂、轮回灯 | 后期谨慎使用 |

## 3. 技艺等级

统一粗分：

| 等级 | 对应 |
|------|------|
| 学徒 | 炼气 |
| 一阶师 | 炼气后期到筑基 |
| 二阶师 | 筑基 |
| 三阶师 | 金丹 |
| 四阶师 | 元婴 |
| 五阶师 | 化神 |
| 宗师 | 灵界 |
| 仙匠/仙师 | 仙界 |

不要一开始给赵长生多个职业高等级。

## 4. 赵长生技艺路线

赵长生优先路线：

1. 炼丹废料识别。
2. 灵草生机判断。
3. 残器宝气判断。
4. 低阶阵法修补。
5. 洞府灵田经营。
6. 本命法宝重炼。
7. 洞天和界源修补。

赵长生的强不是“全职业天才”，而是归墟炉让他能从废料反推技艺本质。

## 5. 沈临风技艺路线

沈临风不应和赵长生完全重合。

适合：

- 系统识别机缘。
- 任务提示。
- 风属性身法。
- 基础符箓或阵法使用。
- 司命台相关命数理解。

后期重点是司命台和命数，不要让他也变成全能炼丹炼器大师。

## 6. 技艺和爽点

技艺爽点必须具体：

- 废丹为什么废。
- 残器哪一段器脉没死。
- 符笔裂在哪里还能修。
- 阵盘缺哪个阵脚。
- 灵草根须还有没有生机。

归墟炉不是黑箱越少，读者越能相信赵长生的成长。
