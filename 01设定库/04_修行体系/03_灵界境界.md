# 灵界境界

> 灵界境界服务第六卷及之后。
> 第一卷不需要出现这些名词，只需要后台先成立。

## 1. 总览

灵界境界暂定：

> 炼虚 → 合体 → 大乘 → 渡劫

灵界不是简单更高灵气版下界，而是开始接触：

- 界源。
- 百族道统。
- 空间裂缝。
- 仙界监察。
- 万年抽源。

## 2. 炼虚

核心变化：

- 元神开始感应虚空灵机。
- 法力能够短暂触及空间裂缝。
- 可修炼更完整的上界道统。

资源需求：

- 空灵石。
- 虚灵草。
- 空间稳定阵。
- 灵界功法。

剧情作用：

- 让赵长生认识万界不是单一下界。
- 引出太虚概念的早期影子。

## 3. 合体

核心变化：

- 法身与元神合一。
- 可承载族群血脉、法相、洞天雏形。
- 对界源变化更敏感。

资源需求：

- 法身材料。
- 血脉本源。
- 洞天碎片。
- 高阶护法阵。

剧情作用：

- 接入妖族、真灵、百族血脉线。
- 让新仙界收割血脉和界源变得具体。

## 4. 大乘

核心变化：

- 道途接近一界顶点。
- 可凝聚道果雏形。
- 能影响一界气运。

资源需求：

- 一界气运。
- 道果材料。
- 完整道统。
- 避开仙界监察。

剧情作用：

- 万年抽源的主要目标。
- 仙界重点监控对象。

## 5. 渡劫

渡劫不是安稳长期境界，而是大乘圆满后的劫关阶段。

旧天庭时期：

- 渡劫是进入更大循环的考验。

新仙界时期：

- 渡劫是筛选、验源和回收。

剧情作用：

- 揭示飞升真相。
- 让赵长生看到“飞升不是奖励”。

## 6. 灵界资源

| 资源 | 用途 |
|------|------|
| 灵晶 | 灵界通用高阶灵气货币 |
| 界砂 | 稳定空间和洞天 |
| 真灵血 | 妖族和百族血脉核心资源 |
| 洞天碎片 | 赵长生聚界能力的重要材料 |
| 道统残碑 | 万法碑线索 |
| 监察令 | 仙界监察者控制灵界势力的凭证 |

## 7. 灵界写法原则

灵界不能只是更大的宗门地图。

它要体现：

- 百族竞争。
- 世界本源被抽走。
- 仙界监察无处不在。
- 赵长生的归墟炉从修物升级到修界。
