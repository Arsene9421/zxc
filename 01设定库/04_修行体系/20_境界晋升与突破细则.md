# 境界晋升与突破细则

> 本文档回答：每个境界具体修什么，如何晋升，如何突破，失败会怎样。

---

## 1. 突破总公式

任何大境界突破都不是“经验条满了就升级”。

突破至少需要六项同时过关：

| 要素 | 内容 | 失败后果 |
|------|------|----------|
| 法力积累 | 当前境界法力达到上限 | 法力反噬、冲关失败 |
| 功法路线 | 有完整冲关法门 | 走火入魔、经脉错乱 |
| 肉身经脉 | 能承受法力质变 | 经脉裂伤、寿元折损 |
| 神魂心性 | 能承受境界变化 | 心魔、神魂受损 |
| 外部资源 | 丹药、灵脉、阵法、护法 | 法力中断、突破停滞 |
| 环境安全 | 闭关地、护法、避开仇敌 | 被袭、失败、根基受损 |

赵长生的优势不是不需要这些，而是能用归墟炉降低部分资源成本，并通过反复试验减少走错路的风险。

## 2. 炼气期

### 2.1 修什么

炼气修的是：

- 感应灵气。
- 引气入体。
- 打通基础经脉。
- 把灵气炼成自身法力。
- 学会最基础的小术和法器激发。

炼气九层不是九个花哨小境界，而是法力逐步稳定、经脉逐步打开的过程。

### 2.2 晋升方式

| 小阶段 | 晋升核心 |
|--------|----------|
| 一到三层 | 能稳定引气，完成小周天 |
| 四到六层 | 法力可连续运转，能稳定激发低阶法器 |
| 七到九层 | 法力充盈丹田，开始准备筑基 |
| 大圆满 | 法力、经脉、丹田都达到炼气上限 |

### 2.3 突破资源

- 引气诀或基础功法。
- 灵米、灵石、低阶聚气丹。
- 安静住处。
- 少量疗伤丹。

### 2.4 失败风险

- 法力岔气。
- 经脉刺痛。
- 丹毒堆积。
- 修炼效率下降。

第一卷赵长生只能在这一层次打转，不能暴涨。

## 3. 炼气到筑基

### 3.1 筑基本质

筑基不是单纯法力增加，而是：

> 把散乱法力压入丹田，凝成稳定道基，让修士真正踏入长期修行。

核心变化：

- 法力由气态转为液态。
- 丹田形成道基。
- 寿元大幅增长。
- 护体灵光更稳定。
- 可以长时间驱使法器。

### 3.2 突破条件

| 条件 | 说明 |
|------|------|
| 炼气大圆满 | 法力必须足够 |
| 筑基丹 | 稳定法力压缩，降低失败率 |
| 灵脉或灵室 | 防止突破中灵气断供 |
| 护法 | 防止外敌和走火 |
| 完整筑基法门 | 散修最缺 |
| 心性稳定 | 筑基会放大恐惧和执念 |

### 3.3 失败后果

| 程度 | 后果 |
|------|------|
| 轻微失败 | 法力散乱，修养数月 |
| 中等失败 | 丹田受损，几年内难再冲关 |
| 严重失败 | 道基裂痕，终生筑基无望 |
| 极端失败 | 经脉爆裂，身死或残废 |

筑基丹稀缺的意义就在这里：它不是爽文药丸，而是底层修士拿命换来的改命机会。

## 4. 筑基到金丹

### 4.1 金丹本质

金丹是：

> 将道基、法力、心性和道路凝成一颗可反哺肉身与神魂的丹。

它不是普通内丹，而是修士道途的第一枚核心。

### 4.2 修什么

筑基期主要修：

- 液态法力不断凝练。
- 道基稳定。
- 选择未来道路。
- 本命法宝胚。
- 丹火或本命真火雏形。

赵长生金丹不能走“纯灵根天才路线”，应走“万宝金丹/归元金丹”雏形：

- 五行法力作基础。
- 残器宝气作引。
- 归墟炉试验作校正。
- 本命法宝胚参与结丹。

### 4.3 突破条件

| 条件 | 说明 |
|------|------|
| 筑基大圆满 | 法力和道基饱满 |
| 结丹灵物 | 稳定金丹凝形 |
| 本命法宝胚 | 赵长生路线尤其需要 |
| 心魔关 | 金丹会问“你修的是什么道” |
| 高阶闭关地 | 灵气必须稳定 |
| 护法和阵法 | 结丹时间长，不能被打断 |

### 4.4 失败后果

- 假丹：有金丹气象但不完整，战力提升有限，前途受限。
- 碎丹：金丹未成反噬道基。
- 心魔反噬：道心不稳，金丹染杂。
- 本命法宝胚碎裂：赵长生会损失长期积累。

## 5. 金丹到元婴

### 5.1 元婴本质

元婴是：

> 金丹孕育神魂法身，形成可承载意识和法力的第二生命核心。

核心变化：

- 神魂强度大幅提升。
- 元婴可短暂离体。
- 神识范围扩大。
- 对命数、天机和界面裂缝开始敏感。

### 5.2 修什么

- 金丹孕婴。
- 神魂养炼。
- 婴火。
- 更完整的功法。
- 本命法宝与神魂适配。

### 5.3 突破条件

| 条件 | 说明 |
|------|------|
| 金丹圆满 | 金丹不能有明显裂痕 |
| 养魂灵物 | 防止神魂崩裂 |
| 元婴法门 | 散修极难获得完整版本 |
| 心魔关 | 元婴是“另一个自己”，心魔更强 |
| 安全闭关地 | 破丹成婴极其脆弱 |

### 5.4 失败后果

- 金丹裂而婴不成。
- 神魂受创。
- 元婴畸形，战力和寿元都受损。
- 被天机或系统捕捉异常。

沈临风系统“远离赵长生”的压力可在元婴到化神前后逐步爆发。

## 6. 元婴到化神

### 6.1 化神本质

化神是：

> 元婴与元神合一，开始感应天地与界面。

核心变化：

- 元神成形。
- 能触及天地感应。
- 能感知飞升通道异常。
- 开始被仙界筛选机制注意。

### 6.2 修什么

- 元神稳固。
- 天地感应。
- 一界气运。
- 道则影子。
- 对天劫和飞升通道的识别。

### 6.3 突破条件

| 条件 | 说明 |
|------|------|
| 元婴圆满 | 元婴不能残破 |
| 天地感应地 | 古战场、灵脉核心、界裂附近 |
| 一界气运 | 修士和世界建立更深联系 |
| 化神法门 | 宗门和天机阁垄断 |
| 天劫准备 | 本书天劫带筛选性质 |

### 6.4 失败后果

- 元神不稳。
- 天劫重伤。
- 被新仙界筛选机制记录。
- 飞升通道异常诱导。
- 寿元和命数被污染。

第五卷“断仙门”必须在这一阶段展开，因为化神正好能看见下界之上的问题。

## 7. 化神到炼虚

### 7.1 炼虚本质

炼虚不是单纯更强化神。

它是：

> 元神开始接触虚空灵机，法力能够感知空间裂缝和界面边缘。

赵长生进入灵界后不能立刻成大人物。化神到炼虚需要重新适应灵界灵机。

### 7.2 突破条件

- 灵界功法。
- 空间稳定阵。
- 洞天碎片或界砂。
- 虚灵草、空灵石。
- 避开仙界监察。

### 7.3 失败后果

- 元神被虚空灵机撕裂。
- 法力紊乱。
- 空间裂缝反噬。
- 被灵界大族或监察使发现。

## 8. 炼虚到合体

### 8.1 合体本质

合体是：

> 元神、法身、血脉或道统承载合一。

不同种族的合体方式不同：

| 类型 | 合体方式 |
|------|----------|
| 人族修士 | 元神与法身合一 |
| 真灵血脉 | 血脉源相与法身合一 |
| 妖族 | 妖身、血源、法相合一 |
| 赵长生路线 | 五行法力、残缺本源、洞天雏形互相承载 |

### 8.2 突破条件

- 法身材料。
- 血脉或道统承载。
- 洞天碎片。
- 高阶护法阵。
- 稳定族运或据点。

### 8.3 失败后果

- 法身崩裂。
- 血脉反噬。
- 洞天碎片失控。
- 被仙界监察记录为高价值目标。

## 9. 合体到大乘

### 9.1 大乘本质

大乘是：

> 道途接近一界顶点，开始凝聚道果雏形，并影响一界气运。

它不是安静闭关就能成的境界，需要和世界、族群、道统发生联系。

### 9.2 突破条件

- 一界气运。
- 道果材料。
- 完整道统。
- 大型护法阵。
- 避开仙界监察和万年抽源。

### 9.3 失败后果

- 道果雏形破碎。
- 一界气运反噬。
- 被万年抽源提前盯上。
- 宗族或据点受牵连。

## 10. 大乘到渡劫

### 10.1 渡劫本质

渡劫不是长期境界，而是劫关。

旧天庭时期，渡劫是进入更大循环的考验。

新仙界时期，渡劫被改造成：

- 筛选。
- 验源。
- 登记。
- 回收。

### 10.2 突破条件

- 大乘圆满。
- 道果雏形稳定。
- 天劫准备。
- 登仙台残路线索。
- 识破或绕过新仙界验源机制。

### 10.3 失败后果

- 被天劫重创。
- 被飞升通道回收。
- 命数被司命台登记。
- 法宝、血脉、气运被分流。

第九卷必须把渡劫写成飞升真相的爆点，而不是普通升级关卡。

## 11. 飞升过渡期到真仙

### 11.1 真仙本质

真仙不是“飞上去就自动逍遥”。

飞升者要经历：

- 登记。
- 洗尘。
- 验源。
- 分流。
- 服役或缴税。

真仙本质是：

> 仙躯稳定，法力转化为仙元，能适应仙界法则。

### 11.2 突破条件

- 稳定仙躯。
- 仙灵气洗炼。
- 仙籍或伪造仙籍。
- 仙元丹或废仙石资源。
- 避开登仙监和新司命台。

### 11.3 失败后果

- 洗尘池伤及根基。
- 仙元不稳。
- 被分入仙矿服役。
- 法宝被夺、命数被登记。

## 12. 真仙到玄仙

玄仙本质：

> 仙法成体系，仙元稳定，能承接低阶道职。

突破条件：

- 仙法传承。
- 仙器。
- 道职功勋。
- 稳定仙府或据点。

失败后果：

- 仙元浑浊。
- 道职反噬。
- 仙籍债务加重。

## 13. 玄仙到金仙

金仙本质：

> 道果初稳，寿元和权柄显著提升。

突破条件：

- 道果材料。
- 仙府资源。
- 宝气和气运。
- 一条稳定的仙道分支。

失败后果：

- 道果裂痕。
- 仙府资源反噬。
- 补墟账本因果缠身。

## 14. 金仙到太乙

太乙本质：

> 触及大道分支，能掌部分天地权柄。

突破条件：

- 大道残片。
- 旧天庭器物痕迹。
- 万界因果承载。
- 道心能承受权柄。

失败后果：

- 道衰提前出现。
- 权柄反噬。
- 被旧天庭遗迹或新仙界制度锁死。

## 15. 太乙到大罗

大罗本质：

> 大道完整度极高，与万界因果相连。

突破条件：

- 完整大道权柄。
- 万界气运。
- 墟眼附近的道则资源。
- 处理自身道路与万界因果的关系。

失败后果：

- 道路崩塌。
- 万界因果反噬。
- 被墟眼污染。
- 成为归墟污染者或新仙界制度傀儡。

## 16. 大罗到道君/道祖

道君/道祖不是普通升级。

它代表：

- 掌握一条大道。
- 重定一类规则。
- 承担一类天地职责。

赵长生成为归墟道祖的条件不是“经验够了”，而是：

1. 归墟炉重组。
2. 九器职责重新对齐。
3. 司命盘不再安排命运。
4. 登仙台从回收通道变回接引通道。
5. 墟眼从伤口重炼成循环炉口。
6. 赵长生愿意承担归墟循环，而不是坐仙帝位。

失败后果：

- 仙界崩坏。
- 墟眼扩大。
- 归墟大潮吞没万界。
- 赵长生自身被归墟炉反噬。

## 17. 写作应用

第一卷只需要用到：

- 炼气一到三层的法力积累。
- 低阶小术。
- 废丹、灵米、聚气丹。
- 法力耗尽后的虚弱。
- 小层级压制。

不要在第一卷解释完整晋升表。

但后台必须清楚：赵长生每一步升级都要满足法力、功法、肉身、神魂、资源和环境六项，不是捡到归墟炉就自动突破。
