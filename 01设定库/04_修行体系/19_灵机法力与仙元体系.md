# 灵机、法力与仙元体系

> 本文档回答：这部小说修炼的到底是什么，以及为什么不能乱用太多力量名词。
> 法力消耗、恢复和斗法续航细则见：`37_法力消耗恢复与斗法续航.md`。

---

## 1. 统一答案

本书修炼的核心力量统一叫：

> 法力。

正文叙述中，低中阶修士主要使用“法力”这个词。

完整逻辑是：

> 天地灵机孕育灵气，修士吸纳灵气，经功法炼化成自身法力。法力养身、养魂、驱器、施法、突破境界。飞升仙界后，法力经仙灵气和仙躯蜕变，称为仙元。再往后接触道则和本源，但道则、本源不是普通能量条。

最简口径：

| 层级 | 主词 | 是否常用 |
|------|------|----------|
| 环境 | 灵机、灵气 | 灵气常用，灵机少用 |
| 修士体内 | 法力 | 全书主词 |
| 法力凝练状态 | 真元 | 筑基以后偶尔用 |
| 仙界法力 | 仙元 | 仙界篇常用 |
| 规则和根基 | 道则、本源 | 后期概念，不当普通能量写 |

## 2. 为什么之前会显得叫法多

因为这些词不是同一层级。

问题在于如果正文里频繁混用，读者会以为它们是多套并行力量系统。

现在收束为一条链：

> 灵机/灵气是外部来源，法力是体内主力，真元是法力凝练说法，仙元是仙界形态，道则和本源是规则层与根基层。

也就是说：

- 炼气修士不是修“灵力体系”，他仍然炼法力。
- 筑基修士不是改修“真元体系”，只是法力更凝练。
- 灵界修士不是改修“界元体系”，而是在法力中接触界源。
- 仙界修士才出现明显形态变化，法力蜕变为仙元。
- 道则、本源不是拿来像法力一样消耗的蓝条。

## 3. 名词保留和禁用规则

| 名词 | 保留方式 | 规则 |
|------|----------|------|
| 灵机 | 世界观底层概念 | 少用，多用于灵脉衰败、补墟抽源 |
| 灵气 | 外部可吸收能量 | 常用，第一卷主要环境词 |
| 灵力 | 底层口语 | 可以用作低阶俗称，但叙述主词仍是法力 |
| 法力 | 体内修行力量 | 全书主词 |
| 真元 | 凝练法力 | 筑基到元婴可用，但不要替代法力 |
| 元神之力 | 神魂层力量 | 元婴、化神后使用 |
| 界源之力 | 世界本源外溢 | 灵界篇使用，不再把“界元”设成常规能量名 |
| 仙元 | 仙界法力形态 | 真仙后主词 |
| 道则 | 规则痕迹和大道运转 | 太乙、大罗后使用 |
| 本源 | 存在根本资粮 | 归墟炉、补墟体系、终局使用 |

明确删弱：

- 不把“界元”作为常规力量主词。
- 不写“灵界法力”这种含混说法。
- 不把“本源”写成随手消耗的技能能量。

## 4. 正文用词口径

| 阶段 | 推荐用词 | 避免 |
|------|----------|------|
| 第一卷 | 灵气、法力 | 不解释灵机、本源、道则 |
| 炼气期 | 法力、少量灵力口语 | 不把灵力当独立体系 |
| 筑基到金丹 | 法力、真元 | 不频繁堆新名词 |
| 元婴到化神 | 法力、真元、元神 | 不提前讲仙元 |
| 灵界篇 | 法力、界源、族运、血源 | 不把界元当主词 |
| 仙界篇 | 仙灵气、仙元、仙法 | 不再用普通法力当主词 |
| 终局 | 道则、本源、循环 | 不写成简单能量互轰 |

第一卷最稳写法：

> 赵长生运转小五行吐纳法，引一缕灵气入体，沿残缺经脉炼成微薄法力。

不用写：

> 赵长生吸纳灵机，转化灵力，又凝成法力。

后者太绕。

## 5. 法力如何形成

修炼过程：

1. 感应外界灵气。
2. 引灵气入体。
3. 按功法路线运行经脉。
4. 去除杂性。
5. 转化为带有个人属性的法力。
6. 法力反过来滋养肉身、丹田、神魂。

不同功法炼出的法力不同：

| 功法倾向 | 法力特点 |
|----------|----------|
| 火系功法 | 爆烈，攻击强，恢复慢 |
| 木系功法 | 生机强，适合灵植和疗伤 |
| 金系功法 | 锐利，适合剑器 |
| 水系功法 | 绵长，恢复和控制强 |
| 土系功法 | 厚重，护体和阵法稳定 |
| 五行功法 | 平衡，能承载多类本源，但前期效率低 |

赵长生五行杂灵根前期弱，是因为吸收慢、转化慢、法力不够纯。

他后期强，是因为五行法力能成为“炉壁”，承载多种残缺本源，不容易被单一属性限制。

## 6. 三界力量变化

| 阶段 | 外部来源 | 体内主词 | 战斗表现 |
|------|----------|----------|----------|
| 炼气 | 灵气 | 法力 | 小术、低阶法器、符箓 |
| 筑基 | 灵气、灵脉 | 法力/真元 | 护体、御器、较长斗法 |
| 金丹 | 高阶灵气、宝气 | 金丹真元 | 本命法宝、丹火、法力外放 |
| 元婴 | 灵气、神魂资源 | 真元、元婴之力 | 元婴遁、神魂压制 |
| 化神 | 天地感应 | 元神法力 | 感知界面、触碰道则影子 |
| 炼虚 | 虚空灵机、界源外溢 | 法力接触界源 | 空间裂缝、洞天碎片 |
| 合体 | 族运、血脉、界源 | 法身之力 | 法相、族群道统 |
| 大乘 | 一界气运、道果雏形 | 高阶法力接近仙元 | 影响一界局势 |
| 真仙以后 | 仙灵气 | 仙元 | 仙法、仙器、仙躯 |
| 太乙以上 | 道则、本源、权柄 | 仙元承载道则 | 大道压制、规则冲突 |

## 7. 法力、法术、法宝的关系

| 内容 | 作用 |
|------|------|
| 功法 | 决定法力如何形成 |
| 法力 | 驱动一切手段的燃料 |
| 法术 | 法力的具体使用方法 |
| 法器/法宝 | 放大和改变法力输出 |
| 符箓 | 提前封存法术结构，消耗少但昂贵 |
| 阵法 | 借地势和灵石放大法力 |

所以战斗不是只看谁法力多。

同样法力总量下：

- 功法更完整，法力质量更高。
- 法器更合适，输出更稳定。
- 法术更熟练，破绽更少。
- 情报更充分，更容易打中要害。

## 8. 赵长生的特殊性

赵长生修炼的仍然是法力，不是另开一套“归墟能量”。

特殊在于：

1. 五行杂灵根让他前期法力驳杂、修炼慢。
2. 归墟炉让他能用残缺资源补足法力积累。
3. 归墟炉火能把废丹药性、残器宝气、灵草生机重新归元。
4. 赵长生后期把五行法力、宝气、生机、魂影、界源等纳入“归元重炼”路线。
5. 到终局，他不是换一种能量，而是把法力修炼推到本源和循环层面。

一句话：

> 赵长生不是不修法力，而是把法力炼成一座能承载残缺本源的炉。

## 9. 沈临风系统的力量

沈临风也修法力。

系统不能凭空创造无限能量。

系统主要做三件事：

| 能力 | 本质 |
|------|------|
| 奖励资源 | 调用预置资源或引导他获得已有机缘 |
| 提示任务 | 读取命数和气运波动 |
| 提升熟练度 | 用司命台权限强行记录和模拟经验 |

风险：

- 系统每次帮他转化资源，都会记录他的命数。
- 系统奖励越多，他越像可回收棋子。
- 后期系统可用“天命值”影响他的法力运转和机缘判断。

## 10. 九种本源与法力

九种本源不是九种普通能量条。

它们是法力进一步接触世界根基后的方向。

| 本源 | 与法力关系 |
|------|------------|
| 灵 | 法力来源的环境基础 |
| 宝 | 法宝、器脉与法力放大 |
| 魂 | 神魂与法力结合 |
| 血 | 血脉承载法力和族群源力 |
| 运 | 气运影响法力修行路径和机缘 |
| 法 | 功法、法术、道统决定法力结构 |
| 界 | 世界本源支撑法力上限 |
| 时 | 法力触碰时间痕迹，只做残影和定位 |
| 道 | 法力最终接近规则层 |
