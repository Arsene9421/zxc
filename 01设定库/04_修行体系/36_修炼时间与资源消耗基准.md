# 修炼时间与资源消耗基准

> 本文档提供后台基准：不同资质、资源、功法和环境下，修士大致需要多久提升境界，消耗多少资源。它不是第一卷修炼日程，也不是主角升级时间表。

---

## 1. 使用原则

修炼时间不能写成固定经验条。

任何修炼速度都由七项共同决定：

| 因素 | 影响 |
|------|------|
| 灵根资质 | 决定吸收和转化灵气的基础效率 |
| 功法完整度 | 决定法力运行是否顺畅，能否突破上限 |
| 修炼环境 | 灵气浓度、灵脉稳定、是否安静安全 |
| 丹药资源 | 加快积累，但会带来丹毒和抗药性 |
| 神识和心性 | 决定控法精度、闭关稳定、心魔风险 |
| 伤势和丹毒 | 经脉、神魂、命火受损都会拖慢修炼 |
| 外部压力 | 逃亡、战斗、杂役、债务会大量压缩修炼时间 |

同样是炼气三层，有人三年可成，有人三十年仍卡住。差距不是作者随口说，而是由这些因素叠加。

## 2. 资质修炼效率系数

以“普通三灵根 + 完整基础功法 + 普通灵屋”为 1 倍基准。

| 资质 | 修炼效率 | 说明 |
|------|----------|------|
| 天灵根 | 3 到 5 倍 | 吸收快、转化纯，但仍需功法和心性 |
| 地灵根 | 2 到 3 倍 | 大宗门重点培养对象 |
| 单灵根 | 1.8 到 3 倍 | 属性纯，容易走专精路线 |
| 双灵根 | 1.3 到 2 倍 | 修炼较稳，资源适配较容易 |
| 三灵根 | 1 倍 | 普通修士基准 |
| 四灵根 | 0.5 到 0.7 倍 | 修炼慢，常被宗门边缘化 |
| 五行杂灵根 | 0.25 到 0.5 倍 | 转化慢、法力杂、突破慢 |
| 残灵根 | 0.1 到 0.5 倍 | 看残缺部位和修补手段 |
| 异灵根 | 1.5 到 4 倍 | 有合适功法时很快，无功法时可能被耽误 |

赵长生按正常五行杂灵根计算，修炼速度应很慢。

他的速度能被归墟炉、废丹、补残功法和高风险资源拉起来，但不能写成天灵根速度。

## 3. 功法完整度系数

| 功法状态 | 效率修正 | 风险 |
|----------|----------|------|
| 大路货完整功法 | 1 倍 | 上限低 |
| 残缺功法 | 0.3 到 0.8 倍 | 经脉岔路、突破风险高 |
| 宗门完整功法 | 1.2 到 2 倍 | 受宗门控制 |
| 高阶适配功法 | 2 到 4 倍 | 获取难，牵涉传承 |
| 伪改功法 | 表面 1 到 2 倍 | 暗藏走火风险 |
| 古本功法 | 不稳定 | 需要解读，可能极强也可能误修 |
| 赵长生补残互证路线 | 前期 0.8 到 1.2 倍，中后期提升 | 需要大量试验和失败记录 |

归墟炉补字不是自动理解。

补出残句后，赵长生仍要用废料、小周天、低风险运转去验证。

## 4. 丹药修炼加速

丹药能加速，但不能无限叠加。

| 丹药使用 | 修炼效率 | 代价 |
|----------|----------|------|
| 不用丹药 | 1 倍 | 稳，但慢 |
| 偶尔服用低阶补气丹 | 1.2 到 1.5 倍 | 少量丹毒 |
| 稳定服用成丹 | 1.5 到 2.5 倍 | 丹毒和费用上升 |
| 大量吞服 | 短期 2 到 4 倍 | 法力虚浮，突破风险显著上升 |
| 废丹处理后服用 | 1.1 到 1.8 倍 | 需试验剂量，仍有丹毒 |
| 高品质丹药 | 2 到 5 倍 | 昂贵，容易被势力垄断 |
| 破关丹药 | 不算日常效率 | 只降低突破失败率，不替代条件 |

同类丹药连续服用会产生抗药性。

低阶修士靠丹药加速修炼时，真正限制不是“吃不吃”，而是“买不买得起、排不排得掉丹毒、经脉能不能承受”。

## 5. 每日修炼时间

低阶修士不是一天十二个时辰都能修炼。

| 状态 | 每日稳定修炼时间 | 后果 |
|------|------------------|------|
| 杂役、散修求生 | 1 到 3 个时辰 | 修炼慢，但真实 |
| 坊市小铺伙计 | 2 到 4 个时辰 | 易被杂务打断 |
| 外门弟子 | 3 到 6 个时辰 | 有任务、考核、争斗 |
| 内门弟子 | 5 到 8 个时辰 | 有灵室和丹药 |
| 闭关状态 | 8 到 10 个时辰 | 需护法和资源 |
| 强行苦修 | 10 个时辰以上 | 神魂疲惫，经脉暗伤 |

炼气期闭关超过数日到十数日，就需要饮食、睡眠、排浊、调息。

筑基后可闭关更久，但仍需处理丹毒、心魔、法力杂质。

## 6. 下界修炼时间基准

以下按“无主角机缘、无重大事件干扰”的普通修士估算。

| 境界阶段 | 天灵根/大宗培养 | 双灵根/宗门培养 | 三灵根普通 | 四五灵根散修 |
|----------|------------------|------------------|------------|--------------|
| 引气入门 | 数日到数月 | 数月 | 半年到一年 | 一到三年 |
| 炼气一到三层 | 半年到两年 | 两到四年 | 三到六年 | 五到十年 |
| 炼气四到六层 | 一到三年 | 三到六年 | 六到十二年 | 十到二十年 |
| 炼气七到九层 | 二到五年 | 五到十年 | 十到二十年 | 二十年以上 |
| 炼气大圆满 | 五到十年 | 十到二十年 | 二十到三十年 | 三十到六十年 |
| 筑基准备 | 一到三年 | 三到八年 | 八到十五年 | 可能终生无望 |

这解释为什么筑基丹能改变命运。

很多散修不是不努力，而是在炼气后期已经年纪大、经脉老、丹毒重，筑基机会越来越小。

## 7. 中高境界时间基准

越往后，时间不只是积累问题，更是资源、传承和劫关问题。

| 阶段 | 普通顺利者 | 资源充足者 | 卡关者 |
|------|------------|------------|--------|
| 筑基初期到后期 | 30 到 80 年 | 15 到 40 年 | 百年以上 |
| 筑基到金丹 | 50 到 150 年 | 30 到 80 年 | 终生假丹或无望 |
| 金丹到元婴 | 200 到 500 年 | 100 到 300 年 | 寿元耗尽 |
| 元婴到化神 | 400 到 1000 年 | 200 到 600 年 | 被下界资源和仙门异常卡死 |
| 化神到炼虚 | 进入灵界后 100 到 500 年 | 50 到 200 年 | 界压、功法、监察卡住 |
| 炼虚到合体 | 500 到 1500 年 | 200 到 800 年 | 法身材料不足 |
| 合体到大乘 | 1000 到 4000 年 | 500 到 2000 年 | 族运、界源、道统不足 |
| 大乘到渡劫 | 无固定年限 | 看天劫和登仙台条件 | 可能主动压境等时机 |

主角组速度可以明显快于普通人，但必须由事件、资源、风险和代价支撑。

十四卷结构里的升级速度属于“高风险高收益路线”，不是普通修士平均速度。

## 8. 灵界与仙界修炼速度

### 8.1 灵界

灵界灵机质量更高，但竞争和垄断也更强。

| 条件 | 效率 |
|------|------|
| 刚入灵界，未适应界压 | 0.3 到 0.7 倍 |
| 适应灵界灵机后 | 1.5 到 3 倍于下界同等环境 |
| 大族祖地 | 5 到 20 倍 |
| 残界污染地 | 不稳定，可能快也可能伤根基 |
| 万年抽源期间 | 效率下降，天劫和界源紊乱 |

灵界不是人人修得快。

底层修士可能站在高浓度灵机中，却因为功法、族籍、资源税和监察限制而无法有效利用。

### 8.2 仙界

仙灵气不是飞升者随便吸的灵气升级版。

| 状态 | 效率 |
|------|------|
| 飞升过渡期 | 仙灵气难以炼化，效率低且伤根基 |
| 仙躯稳定后 | 仙元修炼进入正轨 |
| 有仙籍和资源配给 | 稳定但受税役约束 |
| 黑籍飞升者 | 自由但资源渠道极差 |
| 仙矿劳役 | 接触仙矿，却大多被压榨 |
| 道职加持 | 修炼和施法效率提升，但因果绑定加重 |

仙界篇的压迫不是没有仙灵气，而是有仙灵气也不能自由修。

## 9. 资源消耗基准

以下只定相对尺度，不提前锁死具体价格。

| 阶段 | 日常资源 | 破关资源 | 资源压力 |
|------|----------|----------|----------|
| 炼气初期 | 灵米、碎灵、低阶聚气丹 | 无明显破关丹 | 活下去就是压力 |
| 炼气中后期 | 灵石、聚气丹、灵屋租金 | 小破障丹、功法补全 | 坊市和宗门压榨 |
| 筑基 | 筑基丹、灵脉闭关地 | 筑基丹、护法阵 | 底层改命资源 |
| 金丹 | 结丹灵物、本命法宝胚 | 结金丹、丹火资源 | 宗门和家族垄断 |
| 元婴 | 养魂灵物、婴火资源 | 破婴丹、护魂阵 | 大势力核心资源 |
| 化神 | 天地感应地、一界气运 | 化神法门、天劫准备 | 下界顶点资源 |
| 炼虚以上 | 界砂、洞天碎片、族运、界源 | 空间灵物、法身材料 | 灵界制度垄断 |
| 真仙以上 | 仙石、仙元丹、仙器、道职 | 仙籍权限、道果材料 | 仙界账本垄断 |

## 10. 赵长生速度规则

赵长生可以快于普通五行杂灵根，但每次变快都要回答三件事：

1. 资源从哪里来。
2. 风险谁承担。
3. 后患什么时候体现。

合理加速来源：

- 废丹重炼。
- 残卷补字。
- 残器宝气。
- 低成本试错。
- 洞府经营。
- 灵界残界修复。
- 仙界废仙石、残仙器捡漏。

不合理加速：

- 归墟炉直接灌修为。
- 无视丹毒和抗药性。
- 闭关几日连破大境界。
- 没有功法却突破上限。
- 女主或配角献本源给他升级。

## 11. 写作硬规则

1. 修炼速度必须能从资质、功法、资源、环境、风险中解释。
2. 赵长生速度可以快，但不能像天灵根天命之子。
3. 丹药只加速积累，不替代突破条件。
4. 灵界和仙界不是资源天堂，越高层资源越制度化。
5. 低阶修士日常修炼时间必须受杂务、生存和伤势影响。
6. 每次大幅提速都要留下丹毒、伤势、债务、人情、暴露或因果。
