# 法力消耗、恢复与斗法续航

> 本文档补全法力消耗、恢复速度、斗法续航、爆发透支和法力耗尽后果。核心目标：防止低阶战斗变成无限放技能。

---

## 1. 基本原则

法力是修士体内驱动法术、法器、符箓、阵法、护体和突破的主力。

斗法的真实压力来自三件事：

1. 法力总量有限。
2. 法力恢复需要环境和时间。
3. 法力耗尽后会直接影响逃命、护体、神识和判断。

低阶修士最怕的不是招式少，而是：

> 一次法器激发失误，法力就见底；一次护罩破裂，后面连逃都逃不了。

## 2. 法力单位口径

正文不使用数值面板，但后台可用“成”来估算。

- 十成法力：满状态。
- 五成法力：还能斗法，但开始谨慎。
- 三成法力：只适合撤退或保命。
- 一成法力：激发低阶符箓、逃几步、撑一息护罩。
- 法力见底：不能再正常斗法。

这只是写作后台，不在正文频繁报数。

## 3. 低阶法力消耗基准

以炼气期为基准。

| 行为 | 炼气初期消耗 | 炼气中期消耗 | 炼气后期消耗 |
|------|--------------|--------------|--------------|
| 小火球、小水箭等小术 | 1 到 2 成 | 半成到 1 成 | 少于半成 |
| 敛息、清洁、灵雨等低消耗小术 | 半成到 1 成 | 少量 | 极少 |
| 激发低阶残器 | 2 到 4 成 | 1 到 2 成 | 半成到 1 成 |
| 激发完整低阶法器 | 3 到 5 成 | 1 到 3 成 | 1 成左右 |
| 维持护身小罩 | 每十息 1 到 2 成 | 每二三十息 1 成 | 可维持更久 |
| 短距离轻身逃跑 | 1 到 3 成 | 1 到 2 成 | 1 成以内 |
| 强行破禁或开储物袋 | 消耗大且伤神识 | 消耗明显 | 需要时间和技巧 |

第一卷炼气初中期修士，连续施展数个小术就会虚。

赵长生如果靠残器反杀，必须考虑一次激发后还剩多少法力逃跑和清理现场。

## 4. 各境界斗法续航

| 境界 | 正常斗法续航 | 高强度爆发 | 说明 |
|------|--------------|------------|------|
| 炼气初期 | 数十息 | 几息到十几息 | 法力少，怕拖 |
| 炼气中期 | 半盏茶以内 | 数十息 | 能用法器，但续航差 |
| 炼气后期 | 一盏茶左右 | 半盏茶 | 可连续使用法器和符箓 |
| 筑基 | 半个时辰以内 | 一刻钟 | 护体灵光和御器稳定 |
| 金丹 | 数个时辰 | 半个时辰到一时辰 | 本命法宝消耗大但效率高 |
| 元婴 | 半日以内 | 一到两个时辰 | 神识压制和元婴法术消耗明显 |
| 化神 | 可长时间周旋 | 爆发会触动天地感应 | 下界大范围斗法代价高 |
| 炼虚以上 | 视空间、界源、法身而定 | 高爆发常伤法身或界源 | 不宜写成无限大招 |
| 真仙以上 | 仙元厚，但受仙籍、仙禁、道职限制 | 越权施法会留账或反噬 | 仙界制度会参与消耗 |

战斗越高阶，消耗越不只是法力，还包括神魂、法宝、法身、界源、道职和因果。

## 5. 法力恢复速度

恢复速度取决于环境。

| 环境 | 炼气恢复满状态 | 筑基恢复满状态 | 说明 |
|------|----------------|----------------|------|
| 凡俗无灵地 | 数日甚至更久 | 很慢 | 只能靠丹药和灵石 |
| 坊市棚区 | 一日到数日 | 半日到一日 | 灵气杂乱，恢复不稳 |
| 普通灵屋 | 半日到一日 | 数个时辰 | 基准环境 |
| 宗门灵室 | 数个时辰 | 一到两个时辰 | 成本高 |
| 灵脉节点 | 一两个时辰 | 半个时辰到一时辰 | 需身份和阵法 |
| 洞府核心 | 视灵脉等级 | 稳定高效 | 维护成本高 |
| 灵界普通环境 | 低阶需适应 | 适应后更快 | 界压会影响初期 |
| 仙界仙灵气 | 飞升者未稳前难炼化 | 不适用普通法力 | 需仙躯和仙法 |

恢复不是只坐下就满。

受伤、丹毒、神魂疲惫、归墟污染都会降低恢复效率。

## 6. 灵石和丹药恢复

| 手段 | 效果 | 限制 |
|------|------|------|
| 下品灵石 | 稳定补充灵气 | 转化慢，战斗中不适合大量吸收 |
| 聚气丹 | 加快恢复和积累 | 有丹毒，起效需要时间 |
| 回气丹 | 战斗前后恢复较快 | 昂贵，低阶常买不起 |
| 灵米灵食 | 缓慢恢复体力和少量法力 | 日常用，不适合急救 |
| 灵泉浸体 | 恢复经脉和法力 | 被宗门或洞府垄断 |
| 归墟炉处理废丹 | 可低成本恢复部分药性 | 剂量要试验，仍有风险 |

战斗中吞丹不是立刻满法力。

合理写法：

- 丹药入口后需要数息到数十息化开。
- 法力回升时经脉发热或刺痛。
- 若同时施法，药性可能乱冲。
- 频繁使用会让战后恢复更慢。

## 7. 法器和法宝消耗

| 器物 | 消耗特点 |
|------|----------|
| 残器 | 消耗不稳定，可能突然抽空法力 |
| 低阶法器 | 激发成本高，维持成本中等 |
| 符笔、阵盘 | 单次消耗小，但长期消耗神识 |
| 本命法宝胚 | 与修士契合后效率高，但受损反噬大 |
| 法宝 | 输出强，金丹以下难以承受 |
| 仙器 | 需仙元或特殊权限，低阶强行用会伤本源 |

归墟炉修复残器后，也不能把残器变成零消耗神器。

修复只能让器脉更顺、风险更可控，不取消法力成本。

## 8. 护体与防御消耗

防御不是免费常驻。

| 防御方式 | 消耗 |
|----------|------|
| 炼气护身小罩 | 持续耗法力，受击额外消耗 |
| 筑基护体灵光 | 更稳定，但被破后反震明显 |
| 金丹法宝护身 | 消耗真元和法宝灵性 |
| 元婴神魂护罩 | 消耗神魂和真元 |
| 阵法防御 | 消耗灵石、阵旗、地脉 |
| 符箓防御 | 一次性，贵但省自身法力 |

低阶战斗中，逼敌人连续开护罩，就是有效消耗战。

## 9. 法力耗尽后果

| 程度 | 后果 |
|------|------|
| 剩三成以下 | 施法变慢，控器不稳，判断趋于保守 |
| 剩一成以下 | 护体几乎维持不住，只能逃或赌一次 |
| 法力见底 | 四肢虚软，丹田刺痛，神识迟钝 |
| 强行再用 | 经脉裂伤，法力逆行，昏厥 |
| 透支命火 | 寿元折损，后续突破风险上升 |

法力耗尽后，不能下一段又满状态。

至少要体现：

- 喘息。
- 手抖。
- 丹田空痛。
- 控器迟滞。
- 清理现场变困难。
- 必须服丹、用灵石或找地方调息。

## 10. 爆发秘术

爆发秘术可以存在，但必须有代价。

| 类型 | 效果 | 代价 |
|------|------|------|
| 燃血 | 短时增强肉身和法力 | 气血亏损，寿元轻折 |
| 燃魂 | 神识和法术瞬间增强 | 神魂伤、心魔风险 |
| 燃丹田 | 强行榨取法力 | 经脉裂伤，境界倒退 |
| 燃法宝 | 法宝爆发或自毁 | 长期积累损失 |
| 系统强行加持 | 沈临风短时超常发挥 | 命数记录加深，后期回收风险 |
| 归墟残灰激发 | 遮蔽或腐蚀敌方手段 | 污染自身经脉或器物 |

赵长生可以用狠招，但不能无代价爆种。

他更适合提前准备，而不是临场热血透支。

## 11. 赵长生低阶续航规则

第一卷和前期赵长生的续航应遵守：

1. 炼气初中期最多连续使用几个小术。
2. 残器只能作为关键一击或保命，不适合长时间正面斗法。
3. 符箓省法力，但贵，不能随便撒。
4. 废丹恢复有延迟和丹毒。
5. 法力低于三成时，赵长生会优先撤退或设法结束战斗。
6. 杀人后的清理工作也要消耗法力和神识。

## 12. 沈临风续航规则

沈临风前期看起来更快、更亮眼，但续航不能无限。

| 优势 | 代价 |
|------|------|
| 系统提示节省试探成本 | 容易依赖提示 |
| 风属性身法省命 | 消耗法力快 |
| 系统奖励小术熟练度 | 实战判断仍不足 |
| 任务奖励丹药 | 增加系统因果记录 |

沈临风适合打出短促漂亮的回合，不适合长期消耗战。

## 13. 写作硬规则

1. 低阶斗法必须体现法力消耗。
2. 战斗中吞丹不能瞬间满状态。
3. 法器越强，对低阶修士越可能抽空法力。
4. 护体和逃跑都要消耗法力。
5. 法力耗尽后必须影响清场、逃亡和后续恢复。
6. 爆发秘术必须有经脉、神魂、寿元、法宝或因果代价。
7. 归墟炉可以改善资源利用率，不能取消消耗和恢复逻辑。
