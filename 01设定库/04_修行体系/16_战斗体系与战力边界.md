# 战斗体系与战力边界

> 本文档回答：战力怎么算，战斗怎么写，为什么不能乱越级。
> 法力消耗、恢复和续航细则见：`37_法力消耗恢复与斗法续航.md`。

---

## 1. 战斗总原则

战斗不是境界数字互撞。

胜负由四层共同决定：

| 层级 | 内容 | 作用 |
|------|------|------|
| 根基层 | 境界、法力总量、法力质量、神识、肉身 | 决定基础上限 |
| 手段层 | 功法、法术、法器、符箓、丹药、阵法 | 决定实际输出 |
| 信息层 | 情报、底牌、经验、心理、是否轻敌 | 决定能否打中要害 |
| 环境层 | 地形、灵气浓度、禁制、坊市规则、天气 | 决定能否发挥完整战力 |

凡人流战斗的爽点不在于主角正面碾压，而在于：

> 用准备、情报、地形、残器和狠劲，撬开强敌的一瞬破绽。

## 2. 战力构成

| 因素 | 具体影响 |
|------|----------|
| 境界 | 法力总量、神识范围、反应速度、护体能力 |
| 法力质量 | 同样一成法力，高阶功法更凝练、穿透更强 |
| 功法适配 | 决定续航、属性、经脉稳定和爆发方式 |
| 法术熟练度 | 决定施法速度、破绽、命中和消耗 |
| 法器/法宝 | 放大输出，提供防御、速度、控制 |
| 符箓 | 一次性爆发，适合低阶修士越小层级 |
| 阵法 | 准备越充分，越能拉平境界差 |
| 丹药 | 续航、爆发、疗伤，但有丹毒和起效时间 |
| 神识 | 探查、锁定、预判、操控法器 |
| 肉身 | 承受反噬、近身爆发、抗毒抗伤 |
| 心性 | 是否慌乱，是否敢杀，是否能及时撤 |

## 3. 境界压制为什么存在

大境界压制不只是“数值大”。

| 压制来源 | 表现 |
|----------|------|
| 法力质量 | 高境界法力更凝练，低阶护罩容易被穿透 |
| 法力总量 | 高境界能打更久，低境界耗空就死 |
| 神识范围 | 高境界先发现、先锁定、先预判 |
| 护体能力 | 筑基后护体灵光明显强于炼气 |
| 施法速度 | 高境界法术成形更快 |
| 法器等级 | 低阶法器难破高阶防御 |
| 经验差距 | 高境界通常见过更多手段 |

所以不能频繁写低境界正面碾压高境界。

## 4. 境界战力边界

| 境界 | 战斗能力 | 典型压迫感 | 弱点 |
|------|----------|------------|------|
| 炼气初期 | 小术、低阶符、短时间激发残器 | 对凡人和杂役有压倒性 | 法力少，怕围杀、怕偷袭 |
| 炼气中期 | 可稳定使用法器，能斗法数十息 | 坊市打手、小帮派头目 | 续航仍差，神识弱 |
| 炼气后期 | 法力较厚，能连用法器符箓 | 外门精英、散修头目 | 怕阵法、毒、多人伏击 |
| 筑基 | 道基稳固，护体灵光，真元雏形 | 对炼气有明显压制 | 怕提前布置的阵法和高阶符 |
| 金丹 | 金丹真元，本命法宝，能影响一片战场 | 小宗门支柱、修仙国强者 | 怕元婴神识压制和法宝克制 |
| 元婴 | 元婴神魂，神识压制，元婴遁 | 可决定一宗一国局势 | 肉身和元婴都可能被针对 |
| 化神 | 元神感应天地，触界面之力 | 下界顶点，接近飞升门槛 | 容易被天劫和仙界筛选机制锁定 |
| 炼虚 | 接触虚空灵机和空间裂缝 | 灵界强者入门层 | 依赖空间稳定和灵界道统 |
| 合体 | 法身、法相、族群血脉或洞天雏形 | 百族中坚强者 | 法身受损代价巨大 |
| 大乘 | 道果雏形、一界气运 | 灵界顶点之一 | 是万年抽源重点目标 |
| 渡劫 | 劫关阶段，不稳定 | 飞升前最后门槛 | 被验源、筛选、回收 |
| 真仙 | 仙躯、仙元、仙法 | 对下界修士几乎是层级压制 | 受仙籍、仙税、仙职束缚 |
| 玄仙 | 仙法体系、低中阶仙器 | 仙界基层掌权者 | 权限和资源决定上限 |
| 金仙 | 道果初稳，掌府掌矿 | 补墟体系中层压迫 | 账本和因果反噬开始明显 |
| 太乙 | 大道分支权柄 | 仙界高层和旧天庭强者 | 道衰和旧因果 |
| 大罗 | 大道完整度高，牵动万界因果 | 终局级压迫 | 墟眼、归墟大潮、道路错误 |

## 5. 越级规则

### 5.1 可以越小层级

小层级差距可以通过准备弥补。

合理例子：

- 炼气二层靠陷阱、符箓、残器，反杀轻敌的炼气三层。
- 炼气三层提前布阵，拖住炼气四层。
- 炼气五层靠地形和毒，击杀受伤的炼气六层。

### 5.2 大境界只能短暂抗衡

跨大境界必须满足多个条件。

至少满足三项才合理：

| 条件 | 说明 |
|------|------|
| 对方受伤 | 法力、肉身、神魂至少一项不完整 |
| 对方轻敌 | 没有提前开护体或探查 |
| 主角提前布置 | 阵法、毒、陷阱、退路 |
| 有一次性资源 | 高阶符箓、残器爆发、丹药爆发 |
| 地形有利 | 狭巷、废料场、禁制区、灵气紊乱 |
| 情报优势 | 知道对方功法和弱点 |
| 后果严重 | 反杀后要受伤、损耗资源、暴露风险 |

跨大境界的结果一般是：

- 逃走。
- 伤到对方。
- 拖延时间。
- 借外力杀人。

不应频繁写成正面击杀。

### 5.3 禁止的越级

- 炼气二层正面杀炼气五层。
- 炼气期靠归墟炉直接杀筑基。
- 筑基初期正面碾压金丹。
- 金丹随手灭元婴。
- 赵长生靠残器无限爆发且无代价。

## 6. 战斗流程

一场完整战斗通常分六步：

| 步骤 | 内容 |
|------|------|
| 探 | 观察修为、法器、地形、是否有同伴 |
| 试 | 用小术、符箓或虚招试探底牌 |
| 压 | 境界高者压制，境界低者找环境和节奏 |
| 破 | 抓对方施法、换气、轻敌、贪心的一瞬 |
| 杀/逃 | 低阶战斗要么迅速补刀，要么迅速撤 |
| 清 | 处理尸体、储物袋、灵力残留和目击者 |

赵长生的战斗重点在“探、破、清”。

沈临风的战斗重点在“试、逃、关键时刻不退”。

## 7. 第一卷战斗边界

第一卷只写低阶战斗。

赵长生适合：

- 躲。
- 绕。
- 诱敌。
- 断后路。
- 用低阶符箓制造瞬间机会。
- 用修复残器补短板。
- 利用废料场、黑市巷、棚区地形。
- 杀完立刻处理痕迹。

赵长生不适合：

- 正面擂台。
- 长时间斗法。
- 大范围法术。
- 公开暴露归墟炉。
- 热血喊话后爆种。

第一卷合理战斗例子：

| 对手 | 合理打法 |
|------|----------|
| 丹铺掌柜 | 借账本、废丹、炉房地形反制 |
| 小帮派头目 | 诱入废料场，用残器和符箓断退路 |
| 坊市散修 | 假交易、提前布置毒烟或禁制 |
| 外门管事 | 正面不能杀，只能骗、拖、逃或借他人之手 |

## 8. 沈临风战斗方式

沈临风前期适合：

- 系统提示危险。
- 低阶护身符保命。
- 风属性身法拉扯。
- 用系统奖励快速掌握一两个小术。
- 在赵长生过于保守时制造机会。

弱点：

- 缺真实战斗经验。
- 容易相信系统判断。
- 容易低估赵长生对风险的敏感。
- 容易把敌人当任务怪。

第一卷沈临风不能太强。

他可以比赵长生看起来光鲜，但实战稳定性不如赵长生。

## 9. 法力消耗和续航

低阶修士法力很少。

第一卷规则：

- 炼气一二层连续施展数个小术就会法力见底。
- 激发法器比小术更耗法力。
- 符箓消耗少但贵。
- 丹药可恢复法力，但有丹毒和起效时间。
- 斗法后必须有疲惫、伤势或资源损耗。

这样可以避免战斗变成无限放技能。

## 10. 伤害类型

| 类型 | 表现 | 常见应对 |
|------|------|----------|
| 法术伤 | 火球、水箭、金针、土刺 | 护罩、闪避、属性克制 |
| 法器伤 | 飞刀、短剑、针、印 | 法器对撞、护体灵光 |
| 符箓伤 | 一次性爆发 | 提前避开、符箓对符箓 |
| 阵法伤 | 困、杀、迷、压制 | 找阵眼、破阵符、外力破阵 |
| 毒伤 | 毒烟、毒针、药瘴 | 解毒丹、避毒符、提前识别 |
| 神魂伤 | 幻术、摄魂、神识冲击 | 护魂符、神魂功法 |
| 本源伤 | 归墟残灰、旧器污染 | 后期才常见，难恢复 |

第一卷以法术、法器、符箓、毒和地形为主，不提前大量写神魂和本源伤。

## 11. 杀人后果

杀人不是只有战斗结果。

还要处理：

- 尸体。
- 血迹。
- 储物袋禁制。
- 目击者。
- 巡市修士。
- 对方同伴。
- 灵力残留。
- 黑市追查。
- 宗门账目。

赵长生第一次杀人后，必须毁尸灭迹、处理储物袋、迅速离开现场。

如果杀人没有后果，底层修仙社会就会失真。

## 12. 战斗写法禁忌

- 禁止每场战斗都靠爆种。
- 禁止归墟炉临场无代价救命。
- 禁止敌人明明境界高却不探查、不护体、不补刀。
- 禁止斗法没有消耗。
- 禁止杀人没有后续追查。
- 禁止越级后毫发无损。
- 禁止把所有战斗写成远程火球互轰。

## 13. 赵长生战力成长路线

| 阶段 | 战斗核心 |
|------|----------|
| 第一卷 | 地形、残器、符箓、谨慎反杀 |
| 第二卷 | 外门小术、阵法配合、丹房资源 |
| 第三卷 | 筑基后真元稳定，能正面打同阶 |
| 第四卷 | 洞府阵法、本命法宝胚、灵田资源 |
| 第五卷 | 金丹和本命法宝成型，参与大战 |
| 灵界篇 | 归墟炉从修物到修界，战斗加入界源和洞天 |
| 仙界篇 | 废仙器、仙元、仙籍限制和补墟制度压迫 |
| 终局 | 不靠单挑所有人，而是重炼旧秩序的规则基础 |
