# 修行职业晋升细则

> 本文档补全丹、器、符、阵、医、魂、天机、灵植等职业的晋升规则。核心目标：让技艺成长有门槛、有考核、有失败代价，不让赵长生变成无代价全职业天才。

---

## 1. 职业体系定位

修行职业不是生活技能点。

它们决定资源如何被生产、修复、保存、流通和垄断。

长篇中，职业体系有四个作用：

1. 提供具体爽点。
2. 支撑资源经济。
3. 制造门槛和垄断。
4. 让同伴有不可替代的专业性。

赵长生能接触多种职业，是因为归墟炉让他能从废料反推本质；但他不能每一门都成为第一人。

## 2. 通用等级

| 等级 | 对应境界 | 能力边界 | 社会地位 |
|------|----------|----------|----------|
| 学徒 | 炼气初中期 | 只能打杂、识材、处理废料 | 低层劳力 |
| 一阶师 | 炼气后期 | 可独立制作低阶成品 | 坊市可雇佣 |
| 二阶师 | 筑基 | 可稳定供应筑基级资源 | 小宗门重视 |
| 三阶师 | 金丹 | 能设计配方、器纹、阵盘 | 区域名师 |
| 四阶师 | 元婴 | 能处理神魂、法宝、护山阵等复杂问题 | 大宗门核心 |
| 五阶师 | 化神 | 接触本源雏形、天劫、界面异常 | 下界顶尖 |
| 宗师 | 炼虚到合体 | 能处理灵界材料、血源、界源 | 灵界大势力争夺 |
| 大宗师 | 大乘到渡劫 | 能参与登仙台、残界、百族盟契 | 一界级人物 |
| 仙师/仙匠 | 真仙到金仙 | 能处理仙元、仙器、仙籍、仙禁 | 仙界制度内资源 |
| 道匠 | 太乙以上 | 能接触道痕、旧天庭器、循环节点 | 极少数 |

职业等级不能高于承载能力太多。

炼气修士可以偶然修复一件低阶残器，但不能稳定炼制金丹级法宝。

## 3. 晋升通用条件

每次职业晋升至少需要六项：

| 条件 | 说明 |
|------|------|
| 材料经验 | 见过、摸过、处理过足够材料 |
| 理论传承 | 丹方、器图、符谱、阵书、医典等 |
| 法力控制 | 能稳定输出细微法力 |
| 神识精度 | 能辨别药性、器纹、符胆、阵眼 |
| 失败记录 | 知道什么会失败，为什么失败 |
| 独立成品 | 做出可被他人验证的成果 |

归墟炉可以降低试错成本，但不能替代理解。

## 4. 炼丹

### 4.1 成长路线

| 等级 | 能力 | 失败代价 |
|------|------|----------|
| 学徒 | 分拣药材、清炉、识别废丹 | 中毒、烫伤、误判药性 |
| 一阶师 | 炼制聚气、疗伤、辟谷等低阶丹 | 丹毒偏高、成丹率低 |
| 二阶师 | 炼制筑基辅助丹、续脉丹雏形 | 药性冲突、经脉反噬 |
| 三阶师 | 设计丹方变体，处理金丹级主药 | 丹火失控、药性暴走 |
| 四阶师 | 养魂、破婴、稳定心魔类丹药 | 神魂药性反噬 |
| 五阶师 | 化神、天劫、寿元边界丹药 | 折寿、命火污染 |
| 宗师以上 | 灵界血源、生机、界源类丹药 | 涉及族契和界源代价 |

### 4.2 赵长生关联

赵长生前期能靠废丹识别药性，但不能立刻成为炼丹大师。

他的优势：

- 看出废丹中残余药性。
- 反推失败原因。
- 用归墟炉降低材料损耗。
- 记账和试验习惯极强。

他的限制：

- 缺完整丹方。
- 缺稳定丹火。
- 缺高阶药材。
- 神识前期不够精细。
- 丹毒和抗药性仍然存在。

## 5. 炼器

| 等级 | 能力 | 失败代价 |
|------|------|----------|
| 学徒 | 识矿、打磨、修补粗胚 | 材料报废 |
| 一阶师 | 修低阶法器、做符笔、短刃、护符 | 器纹不稳 |
| 二阶师 | 炼筑基法器、阵盘部件 | 器脉断裂 |
| 三阶师 | 养法宝胚、聚宝气 | 宝气冲突 |
| 四阶师 | 本命法宝、器灵雏形 | 反噬主人 |
| 五阶师 | 处理古宝、灵宝残片 | 旧器污染 |
| 宗师以上 | 洞天法宝、界阵器、仙器残件 | 界源和道痕反噬 |

赵长生的本命法宝路线必须慢。

残器越多，器性越杂；归墟炉能分拣宝气，但不能替他省掉调和过程。

## 6. 制符

| 等级 | 能力 | 关键门槛 |
|------|------|----------|
| 学徒 | 裁符纸、研符墨、保养符笔 | 手稳 |
| 一阶师 | 火球、清风、敛息、护身等低阶符 | 符胆不断 |
| 二阶师 | 筑基级攻击、防御、遁符 | 法力封存 |
| 三阶师 | 金丹级符宝雏形 | 神识刻印 |
| 四阶师 | 元婴级禁符、护魂符 | 神魂稳定 |
| 五阶师 | 天劫、界面、镇封类符 | 承载本源 |
| 仙符师 | 仙禁、仙籍、道职符诏 | 仙界制度权限 |

制符适合低阶越级，但限制很硬：

- 成本高。
- 失败率高。
- 存放有损耗。
- 激发需要时机。
- 高阶符箓会反噬低阶使用者。

## 7. 阵法

| 等级 | 能力 | 典型成果 |
|------|------|----------|
| 学徒 | 认识阵纹、摆基础阵脚 | 聚灵小阵 |
| 一阶师 | 低阶困阵、警戒阵、遮掩阵 | 坊市防护 |
| 二阶师 | 筑基洞府阵、阵盘组合 | 小队战斗 |
| 三阶师 | 护山阵局部、地形阵 | 洞府经营 |
| 四阶师 | 大型护宗阵、传送阵雏形 | 宗门级工程 |
| 五阶师 | 界裂稳定、天劫辅助阵 | 下界顶尖 |
| 宗师 | 残界、洞天、百族战场大阵 | 灵界战略 |
| 仙阵师 | 登仙台、仙籍、补墟封禁 | 仙界制度核心 |

陆维舟必须有不可替代性。

赵长生可以懂阵法原理和修补残阵，但不能抢走陆维舟的主专业。

## 8. 医修

医修不是奶妈。

| 等级 | 能力 | 不能解决 |
|------|------|----------|
| 学徒 | 包扎、止血、识毒 | 法力伤 |
| 一阶医修 | 外伤、轻内伤、排低阶丹毒 | 经脉重伤 |
| 二阶医修 | 经脉轻伤、法力紊乱 | 神魂伤 |
| 三阶医修 | 金丹伤、毒伤、续脉 | 本源伤 |
| 四阶医修 | 神魂轻伤、元婴反噬 | 轮回问题 |
| 五阶医修 | 命火裂痕、寿元边界 | 无代价延寿 |
| 宗师医修 | 血源、族契、界源污染 | 整界衰败 |
| 仙医 | 道伤、仙籍反噬、道职灼伤 | 墟眼根本问题 |

医修能制造恢复节奏，但不能抹掉战斗后果。

## 9. 魂修与安魂职业

魂修必须区分正邪。

| 方向 | 正向能力 | 越界后 |
|------|----------|--------|
| 护魂 | 防幻术、防夺舍、防心魔 | 魂禁控制他人 |
| 安魂 | 稳定残魂、送魂入轮回 | 拘魂不放 |
| 梦术 | 治疗梦魇、读取残影 | 篡改记忆 |
| 摄魂 | 逼供、短暂压制 | 炼魂邪术 |
| 守灯 | 轮回灯相关传承 | 私用轮回权柄 |

洛闻霜线要把“安魂”和“炼魂”分清。

赵长生即使能聚魂，也不能绕过轮回灯秩序。

## 10. 天机与命数职业

| 等级 | 能力 | 代价 |
|------|------|------|
| 学徒 | 看星象、卜小吉凶 | 易错 |
| 一阶卜师 | 望气、寻物、避凶 | 法力消耗 |
| 二阶卜师 | 推演人物短期趋势 | 神魂疲惫 |
| 三阶卜师 | 推演宗门、战局、秘境 | 寿元或气运损耗 |
| 四阶卜师 | 接触命数裂痕 | 被反向定位 |
| 五阶卜师 | 发现飞升、天劫异常 | 命数污染 |
| 宗师 | 校正司命台污染 | 可能折寿 |
| 仙阶司命师 | 接触司命盘规则 | 容易成为制度工具 |

谢未明的价值不是剧透，而是识别“天机被污染”。

## 11. 灵植

| 等级 | 能力 | 典型成果 |
|------|------|----------|
| 学徒 | 浇灌、除虫、辨土 | 活下低阶灵草 |
| 一阶灵植师 | 低阶灵田管理 | 稳定产出 |
| 二阶灵植师 | 筑基药材培育 | 宗门灵田 |
| 三阶灵植师 | 药圃设计、灵泉调理 | 洞府资源根基 |
| 四阶灵植师 | 生机修复、灵脉复苏 | 战后恢复 |
| 五阶灵植师 | 洞天灵圃 | 跨界承载丹材 |
| 宗师 | 残界生机重建 | 灵界据点 |

叶疏彤要在这条线上持续成长。

她不是治疗挂件，而是生机和资源循环的专业核心。

## 12. 职业协作

长线人物分工建议：

| 人物 | 主专业 | 辅助专业 |
|------|--------|----------|
| 赵长生 | 重炼、炼丹识药、残器识别 | 阵法和灵植只懂应用 |
| 叶疏彤 | 灵植、生机、丹材 | 医修部分 |
| 陆维舟 | 阵法、界阵、阵器 | 低阶炼器 |
| 顾长庚 | 剑道、法源、战斗判断 | 万法碑线 |
| 谢未明 | 天机、命数、司命污染 | 望气 |
| 白倩霜 | 血源、族契、百族法 | 真灵战法 |
| 纪晚凝 | 仙籍、账本、道职规则 | 补墟制度 |
| 洛闻霜 | 安魂、轮回灯 | 魂修防线 |

这能防止赵长生包办一切。

## 13. 写作硬规则

1. 职业晋升必须有材料、理论、神识、失败记录和成品验证。
2. 归墟炉降低试错成本，但不能代替理解。
3. 赵长生可以跨职业识别残缺，但不能全职业登顶。
4. 同伴专业必须保留不可替代性。
5. 职业成果要具体到药性、器纹、符胆、阵眼、伤势、命数误差。
6. 高阶职业必然接触本源、界源、仙籍、道职或轮回，不能只是低阶职业放大版。
7. 职业失败要有材料损耗、伤势、污染、信誉和时间成本。
