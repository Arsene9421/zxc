# 器物与符箓实战尺度

> 本文档补全法器、残器、符箓在战斗中的通用效果和成本，不设计具体章节战斗。

---

## 1. 器物和符箓的定位

低中阶修士的战斗，不只靠法术。

| 类型 | 定位 |
|------|------|
| 法器 | 可重复使用的战力放大器 |
| 残器 | 有缺陷但仍有部分功能，适合捡漏和风险战斗 |
| 符箓 | 一次性封存法术，贵但能救命 |
| 阵盘 | 提前布置的地利工具 |

赵长生前中期的爽点，主要来自残器和废料重炼；沈临风前期的爽点，主要来自系统给出的符箓、提示和小术熟练度。

## 2. 法器等级和实战效果

| 等级 | 使用者 | 效果 |
|------|--------|------|
| 下品法器 | 炼气初中期 | 飞刀、小盾、短剑、符笔，威力有限 |
| 中品法器 | 炼气中后期 | 可明显压制低阶法器 |
| 上品法器 | 炼气后期、筑基初期 | 对低阶修士有强压迫 |
| 极品法器 | 炼气圆满、筑基 | 少见，价格昂贵 |
| 灵器 | 筑基、金丹 | 具备更强灵性，消耗更大 |
| 法宝 | 金丹以后 | 可炼成本命法宝，绑定道途 |

法器不是拿起来就能发挥满威力。

影响法器威力的因素：

- 修士法力总量。
- 法力属性是否适配。
- 神识操控能力。
- 法器完整度。
- 是否熟悉器纹。

## 3. 残器规则

残器有价值，但必须有缺陷。

| 残缺类型 | 效果 | 风险 |
|----------|------|------|
| 器身裂纹 | 威力下降 | 激发过猛会碎 |
| 器纹缺失 | 功能不稳定 | 可能反噬法力 |
| 灵性衰退 | 反应慢 | 斗法中失灵 |
| 材料腐蚀 | 结构脆 | 被同阶法器击碎 |
| 旧道痕残留 | 可能有特殊效果 | 可能引来污染或监察 |

归墟炉可以修复部分残器，但不能完全消除所有风险。

合理收益：

- 恢复一部分锋锐。
- 修补器纹缺口。
- 聚出少量宝气。
- 让原本不能用的残器勉强可用。

不合理收益：

- 低阶残器直接变成灵宝。
- 残器无限自我修复。
- 法器没有法力消耗。

## 4. 符箓等级和威力

| 等级 | 对应境界 | 效果 |
|------|----------|------|
| 一阶符 | 炼气 | 火弹、护身、清风、隐气、传音、小疗 |
| 二阶符 | 筑基 | 破盾、困敌、遁逃、护体 |
| 三阶符 | 金丹 | 大范围攻击、强力防护、短距挪移 |
| 四阶符 | 元婴 | 神魂防护、强禁制、远距传讯 |
| 五阶符 | 化神 | 界面感应、天劫辅助、强遁 |
| 灵符 | 灵界 | 空间、血脉、界源相关 |
| 仙符 | 仙界 | 仙法封存、仙禁、道职权限 |

低阶修士不能随意激发高阶符。

原因：

- 法力不足。
- 神识无法承载。
- 符力反冲。
- 高阶符会留下明显波动。

## 5. 符箓成本逻辑

符箓贵，是因为它提前封存了法术结构。

成本来自：

| 成本 | 说明 |
|------|------|
| 符纸 | 需要能承载灵性 |
| 符墨 | 需要灵兽血、灵草汁、矿粉 |
| 符笔 | 要稳定导引法力 |
| 制符师 | 失败率高，熟练度值钱 |
| 封灵材料 | 防止符力散失 |

符箓价格必须高到让底层修士心疼。

如果符箓太便宜，低阶战斗会失去压力。

## 6. 符箓限制

- 多数符箓一次性消耗。
- 旧符可能失效。
- 激发需要少量法力。
- 连续激发会冲击经脉。
- 高阶符会留下明显气息。
- 低阶符对高境界只能制造机会，不能稳定击杀。

## 7. 器符战斗原则

| 场景 | 合理用法 |
|------|----------|
| 弱者对强者 | 用符箓制造一瞬破绽，用残器补刀或逃跑 |
| 同阶斗法 | 法器决定压制力，符箓决定变数 |
| 伏击 | 阵盘、符箓、毒、地形联动 |
| 逃亡 | 清风符、隐气符、烟符、破禁符 |
| 防守 | 小盾、护身符、简易阵盘 |

凡人流战斗里，器符的核心作用是“创造机会”，不是替主角无代价赢。

## 8. 写作规则

1. 每件残器都要有缺陷。
2. 每张符都要有成本。
3. 高阶符不能频繁出现。
4. 低阶符不能越大境界稳定杀敌。
5. 赵长生修复残器必须消耗炉火、时间和试验成本。
