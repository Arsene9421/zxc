# 神魂、夺舍与轮回细则

> 本文档补全神魂、元婴、残魂、夺舍、轮回灯和复活边界。核心目标：防止“聚魂等于复活”“夺舍随便换号”“轮回可以任意调度”这些逻辑漏洞。

---

## 1. 基础定义

修士不是只有肉身和法力。一个完整个体至少由六层构成：

| 层级 | 含义 | 损伤后果 |
|------|------|----------|
| 肉身 | 血肉、筋骨、脏腑、经脉 | 外伤、内伤、经脉伤 |
| 命火 | 个体生命仍在燃烧的标志 | 寿元衰竭、濒死、本源伤 |
| 神魂 | 意识、记忆、感知、情绪和自我认知 | 失忆、疯癫、幻听、昏迷 |
| 神识 | 神魂外放后的感知和操控能力 | 疲惫、反噬、探查受阻 |
| 命数 | 一生因果轨迹和寿限记录 | 被遮蔽、断页、篡改、归档 |
| 本源 | 存在的底层资粮 | 根基损毁、归墟污染、难以恢复 |

神识不是神魂本身，只是神魂外放后的工具。

一个人神识强，不代表神魂一定完整；一个人神魂受伤，也不一定立刻失去全部神识。

## 2. 神魂随境界的变化

| 境界 | 神魂状态 | 写作尺度 |
|------|----------|----------|
| 凡人 | 神魂弱，依赖肉身和命火 | 死后难以留下清晰残魂 |
| 炼气 | 神魂被灵气滋养，神识初显 | 可短暂感知灵力，但不能离体 |
| 筑基 | 神魂更稳，能承载稳定神识 | 可抵抗低阶幻术、摄魂 |
| 金丹 | 神魂与金丹道基相连 | 神魂伤会影响金丹运转 |
| 元婴 | 元婴承载一部分神魂和法力核心 | 可短暂离体，但风险极高 |
| 化神 | 元神成形，神魂能感应天地和界面异常 | 可触及天机，但易被天机反噬 |
| 炼虚以上 | 神魂接触空间、血脉、界源 | 可被界源污染或牵引 |
| 真仙以上 | 仙魂受仙籍、道职、道则影响 | 道职可护魂，也可束魂 |
| 太乙以上 | 神魂与自身道途深度绑定 | 道伤会伤魂，魂伤会伤道 |

元婴不是第二条命。

元婴离体后，若没有肉身、洞天、法宝、阵法或同伴护持，很快会被天地灵机、敌人神识、归墟残灰或司命台记录捕捉。

## 3. 死亡后的正常去向

正常死亡分四步：

1. 肉身崩坏。
2. 命火熄灭。
3. 魂魄脱离肉身，进入轮回牵引。
4. 命数归档，本源残余流向归墟。

这四步并非瞬间完成。

所以战场、古遗迹、渡劫失败地常有残魂、残念、魂影和命数碎屑。

| 死亡类型 | 可能残留 | 说明 |
|----------|----------|------|
| 寿尽 | 命火平稳熄灭，残留较少 | 容易入轮回 |
| 横死 | 怨念、残魂、命数裂痕 | 容易被鬼修、邪修利用 |
| 渡劫死 | 天劫痕、验源痕、魂力残影 | 新仙界可能提前回收 |
| 被归墟污染吞噬 | 残败本源、记忆断片 | 不等于魂飞魄散，但极难恢复 |
| 被炼魂 | 魂力被抽取，命数被污染 | 轮回不完整，后患极重 |

## 4. 残魂、魂影、真魂

必须区分三种东西：

| 类型 | 含义 | 是否本人 |
|------|------|----------|
| 残魂 | 神魂残片，可能有部分意识 | 不完整，不能直接当本人 |
| 魂影 | 残魂、本源、命数留下的影像 | 不是本人，只是残留记录 |
| 真魂 | 相对完整的魂魄核心 | 才有进入轮回或被救回的可能 |

硬规则：

> 聚魂看到残魂影像，不等于复活。

魂影可以提供线索、记忆片段、历史真相和情感冲击，但不能当作“人还活着”。

如果后期要救一个濒死者，必须同时满足：

1. 命火未彻底熄灭，或真魂未完全散失。
2. 肉身、替身、轮回灯、洞天或特殊法宝有承载处。
3. 命数没有完全归档或被司命台抹除。
4. 本源残缺可被修补。
5. 救回后有明确代价。

## 5. 轮回体系

轮回不是仓库，也不是复活池。

旧天庭轮回灯的职责是：

- 接引残魂。
- 净化怨秽。
- 断开前世无法承载的因果。
- 让魂魄重新进入生死循环。
- 防止大量亡魂滞留，腐化成归墟污染。

轮回灯不能随便做到：

- 指定谁带记忆转世。
- 把死者完整拉回原身体。
- 批量复活战死者。
- 让修士用轮回逃避代价。
- 替新仙界处理被恶意炼魂后的烂账。

### 5.1 转世记忆

正常转世会洗去大部分前世记忆。

少数异常记忆可能来自：

| 来源 | 表现 | 风险 |
|------|------|------|
| 命数未归档 | 梦见前世片段 | 容易被司命台定位 |
| 轮回灯残缺 | 记忆洗炼不全 | 人格混乱 |
| 归墟污染 | 被残败记忆缠绕 | 道心腐蚀 |
| 旧天庭秘术 | 保留使命片段 | 代价沉重 |
| 异世穿越 | 命数接驳异常 | 被天命系统利用 |

## 6. 沈临风的穿越边界

沈临风是真穿越者，但不是普通夺舍者。

建议定为：

> 旧司命台残火从异世拉来一个变量魂火，新司命台在降临瞬间截获，并为其补上一层本界命数外壳。

这意味着：

- 他不是凭空生成。
- 他不是旧天庭安排好的救世主。
- 他身上有异世魂火、新司命台命数外壳、系统因果三层痕迹。
- 系统越给奖励，外壳越厚，绑定越深。
- 他摆脱系统，不是摆脱“穿越身份”，而是摆脱新司命台套在他身上的命数外壳。

沈临风不能靠“穿越者身份”免疫天劫、夺舍、命数推演和魂伤。

他的优势是视角差和系统前期便利，弱点是被司命台记录得更深。

## 7. 夺舍规则

夺舍是高风险禁忌手段，不是常规换身体。

夺舍至少需要六个条件：

| 条件 | 说明 |
|------|------|
| 夺舍者神魂足够强 | 通常元婴以上才有可操作性 |
| 目标命火未彻底熄灭或肉身可承载 | 死透的尸体承载力很差 |
| 肉身资质与功法有一定兼容 | 灵根、经脉、道基不合会反噬 |
| 原主残魂处理干净 | 残魂怨念会形成心魔 |
| 有阵法、丹药、魂器护持 | 否则魂魄暴露极易受伤 |
| 命数能接驳 | 命数断裂会被天劫、司命台或轮回灯追索 |

夺舍失败后果：

- 神魂撕裂。
- 记忆混乱。
- 肉身崩坏。
- 命数破洞。
- 被轮回灯或司命台记录。
- 成为半疯半醒的归墟污染者。

### 7.1 夺舍后的代价

夺舍成功也不是赚到。

| 代价 | 说明 |
|------|------|
| 寿元折损 | 新旧命火接驳不稳 |
| 修为倒退 | 肉身承载不了原本法力 |
| 道途偏移 | 原功法和新肉身不合 |
| 心魔增加 | 原主残念、夺舍罪业、命数裂痕叠加 |
| 天劫加重 | 天劫会验出命数不自然拼接 |
| 仙籍异常 | 仙界登记会发现魂身不符 |

所以夺舍适合写成反派、邪修、绝境修士的选择，不适合写成爽文常规便利。

## 8. 鬼修与魂修

鬼修不等于邪修。

| 类型 | 说明 | 风险 |
|------|------|------|
| 正统魂修 | 修神魂、护魂、安魂、超度 | 资源稀缺，容易被误解 |
| 鬼修 | 肉身死后以魂体继续修行 | 难承天劫，惧阳火和轮回牵引 |
| 邪魂修 | 炼魂、拘魂、吞魂 | 战力快，罪业重，反噬重 |
| 轮回守灯人 | 旧天庭轮回灯相关传承 | 容易被新仙界追杀 |

洛闻霜线应当强调：

轮回不是“谁有本事谁拿走亡魂”，而是生死循环的公共秩序。

## 9. 归墟炉与神魂

归墟炉能处理残败本源，因此后期会碰到魂魄问题。

但边界必须明确：

| 阶段 | 可做 | 不可做 |
|------|------|--------|
| 前期 | 感应残缺气息，偶尔看到残念影像 | 复活死人、炼魂提升 |
| 中期 | 修补轻微神魂裂痕，稳定濒死命火 | 批量招魂、无代价续命 |
| 灵界 | 聚集残界中的魂影，辨认万年抽源真相 | 把一族亡魂直接复活 |
| 仙界 | 配合轮回灯清理被截留亡魂 | 越过轮回灯私自安排生死 |
| 终局 | 重炼生死循环中的堵塞处 | 让赵长生想救谁就救谁 |

归墟炉负责重炼残败本源。

轮回灯负责亡魂归处。

二者可以配合，但不能互相替代。

## 10. 写作硬规则

1. 残魂影像只能提供线索和冲击，不能等同复活。
2. 元婴离体有时间、环境和资源限制。
3. 夺舍必须是高风险选择，成功也要付长期代价。
4. 轮回不能被赵长生、沈临风或旧天庭任意私用。
5. 洛闻霜存在的意义，是限制“归墟炉万能复活”的冲动。
6. 沈临风穿越要解释为司命台命数接驳异常，不写成普通魂穿换壳。
7. 神魂伤要影响睡眠、判断、突破、心魔和天劫，不是吃一粒丹就好。
8. 生死边界越往后越重要，但越不能写得廉价。
