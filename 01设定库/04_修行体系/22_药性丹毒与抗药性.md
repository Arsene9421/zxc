# 药性、丹毒与抗药性

> 本文档补全丹药体系的底层限制：丹药为什么有用，为什么不能无限吞丹，废丹为什么危险。

---

## 1. 丹药为什么有效

丹药不是“经验药水”。

丹药的本质是：

> 炼丹师把灵草、妖材、矿粉、血肉、灵液中的药性和灵气，按照丹方结构封存成可被修士吸收的稳定药团。

丹药有效，靠三件事：

| 要素 | 说明 |
|------|------|
| 药性 | 灵草、妖材、矿物中的特殊效用 |
| 灵气 | 能补充法力或滋养肉身的基础力量 |
| 丹方结构 | 让药性按顺序释放，避免互相冲突 |

所以丹药不是越多越好。药性太猛、释放太快、杂质太重，都会伤人。

## 2. 药性分类

| 药性 | 作用 | 常见丹药 |
|------|------|----------|
| 补气 | 增加法力积累和恢复 | 聚气丹、养气丹 |
| 固元 | 稳定根基和道基 | 固元丹、稳脉丹 |
| 疗伤 | 修补肉身、经脉、脏腑 | 疗伤丹、续脉丹 |
| 解毒 | 化解瘴毒、药毒、妖毒 | 解毒散、清瘴丹 |
| 养魂 | 滋养神魂，减少心魔风险 | 养魂丹、安神丹 |
| 破关 | 辅助大境界突破 | 筑基丹、结金丹 |
| 延寿 | 补命火，延寿元 | 延寿丹、寿元果丹 |
| 洗髓 | 清除杂质，改善经脉 | 洗髓丹、净脉丹 |

## 3. 丹毒来源

丹毒不是单一毒素，而是多种残留和冲突的总称。

来源：

| 来源 | 表现 |
|------|------|
| 药材年份不足 | 药性薄，杂质多 |
| 药材药性相冲 | 法力躁动、经脉刺痛 |
| 炉火不足 | 药性未化开，沉积体内 |
| 炉火过猛 | 药性焦败，伤经脉 |
| 丹方残缺 | 药性释放顺序错乱 |
| 炼丹师水平差 | 杂质封入丹中 |
| 废丹霉败 | 药性腐坏，形成药瘴 |
| 短期大量服用 | 经脉和丹田来不及吸收 |

丹毒主要沉积在：

- 经脉。
- 丹田。
- 血肉。
- 神魂边缘。
- 道基裂隙。

## 4. 丹毒后果

| 程度 | 后果 |
|------|------|
| 轻微 | 法力运转滞涩，修炼效率下降 |
| 中等 | 经脉刺痛，斗法时法力断续 |
| 严重 | 突破失败率上升，道基或金丹染杂 |
| 极端 | 丹毒入魂，心魔加重，寿元折损 |

丹毒最可怕的地方不是马上死人，而是它会让修士越修越窄。

## 5. 抗药性

抗药性指：

> 同类丹药长期服用后，身体和经脉对该药性逐渐迟钝，需要更多丹药才能得到同样效果。

抗药性来自三点：

1. 经脉习惯了药性刺激。
2. 丹田吸收效率下降。
3. 同类药渣沉积，阻碍新药性进入。

规则：

| 服用方式 | 结果 |
|----------|------|
| 偶尔服用 | 效果正常 |
| 连续服用 | 效果下降，丹毒累积 |
| 大量吞服 | 法力虚浮，突破风险上升 |
| 换不同药性 | 可缓解，但不能彻底解决 |
| 洗髓排毒 | 可降低抗药性，但成本高 |

这用于限制“吞丹无限升级”。

## 6. 丹药品阶与药性纯度

| 品质 | 药性 | 丹毒 |
|------|------|------|
| 废丹 | 药性散乱或腐败 | 极重 |
| 劣丹 | 药性可用但粗糙 | 重 |
| 成丹 | 药性稳定 | 正常 |
| 良丹 | 药性顺滑 | 轻 |
| 上品丹 | 药性纯净 | 很轻 |
| 极品丹 | 近乎无杂质 | 极少，但稀有 |

归墟炉前期能做的是：

- 把废丹中未死尽的药性重新聚拢。
- 降低部分腐败杂质。
- 让废丹变成低效劣丹或接近成丹。

不能做到：

- 把低阶废丹炼成高阶神丹。
- 完全消除丹毒。
- 凭空补足不存在的药性。

## 7. 修士如何处理丹毒

| 方法 | 成本 | 说明 |
|------|------|------|
| 停药静养 | 时间成本高 | 底层修士常用 |
| 排毒功法 | 功法难得 | 宗门垄断 |
| 洗髓丹 | 价格高 | 散修很难买 |
| 灵泉浸体 | 资源稀缺 | 大宗门、洞府才有 |
| 医修调理 | 需要长期治疗 | 费用高 |
| 归墟炉辅助 | 消耗炉火 | 赵长生可降低部分风险，但不能全免 |

## 8. 写作规则

1. 丹药能加快修炼，但不能替代功法、心性和突破条件。
2. 丹药越便宜，丹毒越明显。
3. 废丹爽点必须伴随试验和风险。
4. 赵长生用归墟炉处理废丹，也要记录药性、剂量、反应。
5. 沈临风系统奖励丹药也要有来源和因果，不是无限凭空发。
