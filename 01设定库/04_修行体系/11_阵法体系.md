# 阵法体系

## 1. 阵法作用

阵法是弱者对抗强者的重要工具。

赵长生谨慎性格适合学习低阶阵法。

阵法用途：

- 防御。
- 隐匿。
- 聚灵。
- 困敌。
- 杀敌。
- 传送。
- 封禁。

## 2. 阵法构成

| 部件 | 作用 |
|------|------|
| 阵盘 | 控制核心 |
| 阵旗 | 布置节点 |
| 阵基 | 固定地脉或方位 |
| 灵石 | 提供能量 |
| 阵纹 | 规则路线 |
| 阵眼 | 阵法关键点 |

## 3. 阵法等级

| 等级 | 对应境界 |
|------|----------|
| 一阶阵 | 炼气 |
| 二阶阵 | 筑基 |
| 三阶阵 | 金丹 |
| 四阶阵 | 元婴 |
| 五阶阵 | 化神 |
| 界阵 | 灵界 |
| 仙阵 | 仙界 |
| 规则大阵 | 旧天庭级 |

## 4. 第一卷可用阵法

| 阵法 | 用途 |
|------|------|
| 小隔音阵 | 遮掩试验动静 |
| 简易警戒阵 | 防人靠近破屋 |
| 小迷踪阵 | 巷道逃跑 |
| 聚灵小阵 | 提高一点修炼效率 |
| 禁制门闩 | 和物理门闩结合 |

第一卷阵法不能太强。

赵长生可以修补残阵盘，但不能靠阵法直接杀死高阶敌人。

## 5. 阵法和地形

赵长生第一次杀人适合结合：

- 废料场狭窄路线。
- 黑市巷死角。
- 残器触发。
- 简易警戒阵。
- 沈临风系统提示。

这样能保持凡人流可信度。

## 6. 归墟炉和阵法

归墟炉可修复：

- 裂阵盘。
- 残阵旗。
- 破损阵纹。

限制：

- 必须有原阵残余。
- 修复后未必完整。
- 需要灵石驱动。
