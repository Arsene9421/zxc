# 洞天、小世界与界源规则

> 本文档补全洞天、小世界、残界、界源、界压、万年抽源和叶疏彤洞天灵圃规则。核心目标：让“修界”成为赵长生中后期能力升级，但不让洞天变成随身无敌基地。

---

## 1. 空间层级

| 层级 | 定义 | 能否生灵长期生存 |
|------|------|------------------|
| 储物空间 | 只存死物的空间法器 | 不能 |
| 临时秘境 | 依附大世界的封闭区域 | 可短期 |
| 灵圃洞天 | 有灵土、灵水、简易循环的小型空间 | 可少量、有限 |
| 洞天 | 有界核雏形、地脉、灵机循环和出入口 | 可以 |
| 小世界 | 有稳定界壁、时序、生灵循环和独立界源 | 可以 |
| 下界 | 完整世界，能承载凡人和修士文明 | 可以 |
| 中层世界 | 更高承载力世界，如灵界、妖界、魔界 | 可以 |
| 仙界 | 高层世界，承载仙元、道职和墟眼封禁 | 可以，但被新仙界制度扭曲 |

储物袋不能装活人。

能装活人的空间，必然需要空气、灵机、温度、地脉、出入口、界壁和承载代价。

## 2. 洞天的六个组件

一个稳定洞天至少有六个组件：

| 组件 | 作用 | 缺失后果 |
|------|------|----------|
| 界核 | 洞天的核心锚点 | 空间崩塌或无法成长 |
| 界壁 | 内外边界 | 易被撕裂、入侵、泄露 |
| 地脉 | 承载灵土、山水、阵法 | 灵田衰败，建筑不稳 |
| 灵机循环 | 产生和流转灵气 | 只能消耗，不能自养 |
| 时序 | 日夜、季节、流速 | 灵植混乱，修士神魂不适 |
| 出入口 | 与外界交换物资和因果 | 被封死会困死里面的人 |

低阶洞天往往只具备其中两三项，所以不能当完整小世界。

## 3. 界源定义

界源不是灵气，也不是灵石。

界源是一个世界维持自身存在的根基，能孕育灵脉、稳定界壁、承载生灵因果。

关系如下：

> 界源稳定，地脉才能稳。  
> 地脉稳定，灵脉才能生。  
> 灵脉存在，灵气才能流。  
> 灵气流动，修士才有资源。

所以抽界源不是“拿走一批高级灵石”，而是挖走世界根基。

## 4. 抽取界源的后果

| 抽取程度 | 下界表现 |
|----------|----------|
| 轻度 | 灵田减产、妖兽躁动、灵泉枯竭 |
| 中度 | 宗门灵脉降阶、凡俗天灾变多、秘境崩塌 |
| 重度 | 修仙国衰落、天劫异常、地脉断裂 |
| 极重 | 世界位格下降，飞升通道畸变，归墟污染增多 |

界源损伤会延迟显现。

这可以解释为什么新仙界抽源多年，下界不是立刻毁灭，而是越来越穷、越来越乱、越来越难突破。

## 5. 洞天形成方式

| 方式 | 说明 | 风险 |
|------|------|------|
| 天然秘境成长 | 大世界自然孕育 | 容易被宗门垄断 |
| 古战场残片 | 大战打碎的空间残余 | 魂影、污染、道痕多 |
| 旧天庭遗留 | 九器或旧机构残留的小界 | 容易被新仙界追查 |
| 修士开辟 | 高阶修士以法力、阵法、界源开辟 | 消耗巨大 |
| 法宝承载 | 洞天法宝承载空间 | 法宝受损则洞天受损 |
| 归墟炉修复 | 重炼残界碎片，补成可用洞天 | 必须有残余界源 |

赵长生不能凭空造洞天。

他的优势在于能看出残界还有没有可重炼的界源，并用归墟炉修补原本会废掉的残片。

## 6. 洞天成长阶段

| 阶段 | 特征 | 可用于 |
|------|------|--------|
| 灵圃 | 一块灵土、一口灵泉、简易阵法 | 种植灵草、保存丹材 |
| 洞府 | 有山水阵法和修炼室 | 闭关、炼丹、炼器、防御 |
| 洞天雏形 | 有界核碎片和灵机循环 | 少量人员长期驻留 |
| 稳定洞天 | 界壁稳定，能承载传承和族群小支 | 建据点、避灾、养灵脉 |
| 残界修复 | 修复一个破碎小世界 | 灵界篇核心能力 |
| 循环节点 | 洞天成为万界循环的一部分 | 终局新循环节点 |

洞天越大，越难隐藏。

洞天一旦和世界界源相连，就会留下界源波纹，可能被镇界印、新仙界监察和高阶修士发现。

## 7. 叶疏彤的洞天灵圃

叶疏彤后期能跟上主线，关键不是被赵长生带着走，而是她有自己的生机路线。

洞天灵圃建议路线：

| 阶段 | 位置 | 功能 |
|------|------|------|
| 下界外门 | 贫瘠灵田复苏 | 展示灵植能力 |
| 洞府金丹 | 药圃和灵泉整合 | 形成主角组资源根基 |
| 断仙门 | 灵圃承载部分丹材和伤员 | 付出界压代价进入灵界 |
| 灵界底层 | 重新适应灵界灵机 | 修复受损灵根和灵圃 |
| 万年抽源 | 灵圃接入残界碎片 | 生机线参与补界 |
| 仙界篇 | 灵圃成为新循环生机节点 | 不靠赵长生名声立足 |

边界：

- 灵圃前期不能装大批活人。
- 灵圃不能无限产出高阶灵草。
- 灵圃跨界会受界压，灵草和主人都会受损。
- 叶疏彤必须为灵圃付出时间、资源、修为和风险。

## 8. 界压规则

不同世界位格不同。

低界修士进入高界，会遇到：

- 法力被压。
- 肉身沉重。
- 神识范围缩小。
- 低阶法器失灵或降效。
- 灵根需要适应更高层灵机。
- 洞天和储物法器可能不稳。

高界修士进入低界，也有代价：

- 被界壁排斥。
- 容易压坏地脉。
- 天劫和镇界规则盯上。
- 法力无法完全展开。
- 新仙界需要借道职、仙籍、镇界法旨压制反噬。

这解释了为什么仙界不能随便派大罗下界碾压一切。

## 9. 灵界万年抽源

灵界资源比下界丰富，但不是没有压迫。

灵界被抽源的方式：

1. 仙界挑动百族战争。
2. 大族为自保献出弱族祖地、血源和界源。
3. 监察使记录“自然战损”。
4. 补墟司把战损界源记入万年大补。
5. 灵界整体不崩，但底层族群不断衰败。

这样灵界不是简单资源天堂，而是更高等级的资源垄断世界。

## 10. 洞天战斗边界

洞天可以用于战斗，但不能乱用。

可用：

- 设伏。
- 防守。
- 储备资源。
- 躲避短期追杀。
- 承载阵法。
- 保护非战斗人员。

不可用：

- 随时把敌人收进去秒杀。
- 战斗中无代价闪进闪出。
- 无限携带军队。
- 被打坏后一章修好。
- 随身洞天不被任何人发现。

洞天受损后果：

- 灵田枯萎。
- 界壁裂开。
- 居住者神魂震荡。
- 灵脉降阶。
- 界核污染。
- 归墟残灰沉积。

## 11. 赵长生的“修界”路线

赵长生能力递进：

| 阶段 | 重炼对象 | 认知升级 |
|------|----------|----------|
| 前期 | 废丹、残器、枯草 | 残缺仍有价值 |
| 中期 | 洞府、阵法、灵脉 | 残缺可以经营 |
| 灵界 | 残界、界源、族地 | 残缺世界也能被救 |
| 仙界 | 仙籍、道职、墟眼账本 | 制度残缺需要清账 |
| 终局 | 墟眼和万界循环 | 不是补洞，而是重炼循环 |

“修界”不是战力数值提升，而是世界观认知升级。

## 12. 写作硬规则

1. 洞天不是储物袋升级版，能装活人就必须有界源和循环。
2. 界源不是灵气，抽界源会伤世界根基。
3. 叶疏彤的灵圃要有成长、代价和限制。
4. 赵长生只能修复有残余界源的残界，不能凭空造世界。
5. 跨界必须有界压，不能无痛换地图。
6. 灵界资源丰富但被垄断，不能写成人人富裕。
7. 洞天战斗要有暴露、损坏和维护成本。
8. 终局新循环要让洞天、小世界、下界、灵界、仙界重新接入循环，而不是让赵长生私有化万界。
