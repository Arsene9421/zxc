## 3. 资源生态总原则

资源不是全世界一起贫瘠。

正确逻辑是：

1. 总体循环在衰败，新增高阶资源越来越慢。
2. 资源分布极不平均，底层散修最缺，高层势力仍有存量和产地。
3. 宗门、家族、大族、仙界机构通过制度垄断资源。
4. 高阶修士不是没资源修炼，而是必须进入更危险、更昂贵、更高层的资源场。
5. 赵长生的爽点不是凭空刷资源，而是把别人看不上的残缺资源重新变成可用资源。

如果把世界写成到处穷死，高境界修士就无法成立；如果写成资源遍地，底层压迫又不成立。

本书采用“分层富集、底层匮乏”的资源生态。

## 4. 分层富集与底层匮乏

| 层级 | 资源状态 | 典型资源 | 叙事作用 |
|------|----------|----------|----------|
| 底层散修 | 极度匮乏 | 发霉灵米、废丹、碎灵、破符 | 凡人流压迫感 |
| 坊市小铺 | 有流通但利润薄 | 下品灵草、低阶丹药、残器 | 赵长生捡漏和交易的第一层 |
| 小宗门外门 | 资源被内门和长老优先分配 | 灵田任务、丹房废料、残卷 | 第二卷宗门压迫 |
| 小宗门内门 | 有稳定但不宽裕的资源 | 聚气丹、筑基丹名额、低阶法器 | 顾长庚等正统修士的竞争场 |
| 大宗门/修仙国 | 有祖脉、秘境、库存 | 筑基主药、金丹灵物、古宝线索 | 第三到第五卷大战和拍卖会成立 |
| 海外险地/古战场 | 资源高但风险高 | 古兽材料、残魂、旧器碎片、界痕 | 下界高境界修士继续修炼的来源 |
| 灵界普通区域 | 比下界富，但规则更重 | 灵晶、界砂、洞天矿 | 换地图后不重复下界贫穷 |
| 灵界大族核心地 | 资源富集且垄断 | 真灵药、血源池、族运阵 | 百族压迫和猎血制度成立 |
| 仙界底层 | 仙灵气足，但资源被税制和仙籍锁住 | 废仙石、矿渣、低阶仙食 | 仙界篇继续凡人流 |
| 仙界权贵 | 资源充足但来源腐烂 | 道果园、仙矿、补墟本源、下界气运 | 证明新仙界靠收割维持繁荣 |

因此，资源缺口主要压在“没有身份、没有渠道、没有靠山”的人身上。

## 5. 高境界如何继续修炼

### 5.1 下界高境界

下界不是没有化神资源，而是化神资源只存在于少数地方：

| 来源 | 说明 | 代价 |
|------|------|------|
| 大宗门祖脉 | 宗门多年积累的高阶灵脉 | 必须绑定宗门和派系 |
| 修仙国国库 | 王朝、宗门和家族共同控制 | 战争时才会大量释放 |
| 海外险地 | 古兽、海眼、残岛、风暴灵脉 | 死亡率高，情报昂贵 |
| 古战场 | 残魂、旧器、地脉裂痕 | 容易被污染或卷入旧局 |
| 秘境洞天 | 周期性开启 | 名额被垄断，散修难入 |
| 飞升残路 | 化神以上寻找更高层灵机 | 可能被新仙界筛选和回收 |

所以第五卷从元婴到化神不能靠普通闭关堆资源，必须放进修仙国大战、古战场和断仙门真相里写。

### 5.2 灵界高境界

灵界资源比下界丰富，但问题更严重：

- 百族占据不同资源生态。
- 大族垄断真灵药、血源池和界源矿。
- 仙界监察体系定期抽源。
- 弱族和散修有资源，但缺进入核心产地的资格。

灵界高境界修炼依靠：

| 境界 | 主要资源 |
|------|----------|
| 炼虚 | 洞天碎片、界砂、灵晶 |
| 合体 | 族运、真灵药、界源结晶 |
| 大乘 | 一界本源认可、大型族阵、万年灵药 |
| 渡劫 | 登仙台残路、天劫验源、避开仙界回收 |

灵界篇的矛盾不是“没资源”，而是“资源背后绑着族群、血脉和仙界抽源”。

### 5.3 仙界高境界

仙界资源更充足，但更不干净。

仙界高境界修炼依靠：

| 来源 | 表现 |
|------|------|
| 仙矿 | 飞升者和底层仙人服役开采 |
| 道果园 | 仙界权贵控制的高阶药园 |
| 补墟本源 | 从下界、灵界、残界抽取后提纯 |
| 旧天庭废墟 | 旧器碎片、旧道痕、九器残辉 |
| 司命台权限 | 气运和命数加持，但增加束缚 |
| 墟眼边缘 | 风险极高，靠近归墟污染 |

仙界篇要让读者看到：仙界不是穷，而是富得有罪。

## 6. 下界稀缺逻辑

下界资源稀缺的原因：

1. 灵脉自然衰败。
2. 宗门垄断高阶资源。
3. 新仙界补墟体系长期抽取本源。
4. 废料也被上缴，导致底层连残余价值都难保住。

注意：这里的“稀缺”主要指底层可获得资源稀缺，不是整个下界完全没有高阶资源。

## 7. 废料回收链

废料流向：

> 丹铺/器铺/符铺 → 坊市废料场 → 青云办事处 → 青云门库房 → 更高宗门或神秘渠道

底层不知道最终用途。

漏洞：

- 掌柜私吞。
- 搬运帮派偷换。
- 外门管事拿回扣。
- 真正旧天庭残片无灵光，常被当废铁。

赵长生捡漏成立的基础就在这里。

## 8. 资源和归墟炉

归墟炉对资源的处理：

| 资源状态 | 能否重炼 |
|----------|----------|
| 完好资源 | 没必要重炼，收益低 |
| 轻度破损 | 可恢复部分价值 |
| 重度残缺但本源未灭 | 最适合归墟炉 |
| 完全死尽 | 前期不能重炼 |
| 带旧天庭残痕 | 可能触发高阶能力 |

归墟炉的优势不在于无中生有，而在于它能重新利用现有资源体系丢弃的部分。

这让赵长生能从底层撬开一条路，但不能让他跳过所有境界、渠道和代价。

## 9. 第一卷资源目标

赵长生第一卷的资源提升只到：

- 有普通灵米吃。
- 偶尔能用低效聚气丹。
- 修复一两件低阶残器保命。
- 有少量碎灵和下品灵石作为逃亡资本。

不能第一卷暴富。
