# 器物体系

## 1. 器物等级

| 等级 | 常见使用者 | 说明 |
|------|------------|------|
| 凡器 | 凡人、低阶武者 | 无灵性 |
| 法器 | 炼气、筑基 | 下界前期主流 |
| 灵器 | 筑基、金丹 | 灵性更强 |
| 法宝 | 金丹、元婴 | 可炼成本命法宝 |
| 灵宝 | 元婴、化神 | 有较强灵性 |
| 通天灵宝 | 灵界高阶 | 可影响空间和界源 |
| 仙器 | 仙界 | 承载仙道法则 |
| 道器/旧天庭器 | 规则级 | 不作为普通装备体系 |

## 2. 品质状态

| 状态 | 含义 | 归墟炉处理 |
|------|------|------------|
| 完好 | 能正常使用 | 前期无明显收益 |
| 裂损 | 有裂纹，威力下降 | 可恢复部分灵性 |
| 残片 | 只剩部分器身 | 可聚宝气 |
| 死器 | 灵性彻底消散 | 前期无法重炼 |
| 古器残痕 | 带旧道纹或旧天庭痕迹 | 可能触发高阶线索 |

## 3. 法器类型

| 类型 | 用途 |
|------|------|
| 飞剑 | 攻击 |

## 4. 本命法宝

金丹阶段核心。

本命法宝与修士金丹、神魂、道途绑定。

赵长生本命法宝路线：

> 别人找完整灵宝胚胎，赵长生收残器废料，用归墟炉长期聚宝气，炼出自己的本命法宝。

这条线要服务“废物炼成至宝”的中期爽点。

## 5. 储物袋规则

储物袋对底层修士很珍贵。

限制：

- 空间小。
- 不能装活物。
- 破损储物袋可能吞物或崩坏。
- 打开别人储物袋需要抹去简单禁制。

第一卷杀人夺宝时，储物袋本身就是重大收益。

## 6. 旧天庭器物

旧天庭九器不进入普通法宝等级。

它们是规则器。

归墟炉前期像残破小炉，是因为本体破碎、炉火熄弱。

不能把九器写成普通装备排行榜。
