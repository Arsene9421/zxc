# 货币与物价体系

## 1. 下界货币

| 货币 | 换算 | 使用场景 |
|------|------|----------|
| 碎灵石（简称碎灵） | 100 枚碎灵石 = 1 下品灵石 | 底层杂役、小摊、灵米交易 |
| 下品灵石 | 基础货币 | 坊市常规交易 |
| 中品灵石 | 100 下品灵石 = 1 中品灵石 | 筑基修士和大额交易 |
| 上品灵石 | 100 中品灵石 = 1 上品灵石 | 大宗门、金丹以上 |
| 极品灵石 | 稀有 | 后期阵法、古传送阵 |

第一卷原则：

- 正文首次出现时写清“碎灵石”，随后坊市底层口语可简称“碎灵”。
- 赵长生多数时候用碎灵。
- 下品灵石对赵长生已经是正经钱。
- 中品灵石只作传闻或压迫感。

## 2. 灵界货币

暂定：

| 货币 | 用途 |
|------|------|
| 灵晶 | 灵界基础高阶货币 |
| 界砂 | 稳定洞天和空间裂缝 |
| 族印功勋 | 百族内部资源兑换 |
| 仙令残额 | 仙界监察体系发放的配额凭证 |

灵界货币重点不是现在细写，而是后续体现仙界对灵界资源的间接控制。

## 3. 仙界货币

暂定：

| 货币或凭证 | 用途 |
|------------|------|
| 仙石 | 仙界基础灵气货币 |
| 仙晶 | 高阶仙石 |
| 道功 | 仙界机构功勋 |
| 仙籍额度 | 决定飞升者合法身份 |
| 补墟配额 | 补墟司内部资源凭证 |

仙界篇要体现：

- 飞升者不是有仙气就自由。
- 仙籍和仙税本身就是压迫。
- 赵长生飞升后仍要从废仙石、残仙器起家。

赵长生开局身家：

> 不足 2 块下品灵石，且大半是碎灵。

## 7. 物价服务剧情

赵长生第一卷的经济状态：

- 买不起完整飞剑。
- 舍不得长期吃普通灵米。
- 一枚完整聚气丹都要精打细算。
- 破损储物袋也是巨款。
- 能把废聚气丹恢复到低效可用，就是重大改善。

沈临风第一卷的经济状态：

- 系统给的小额资源让他比赵长生宽裕。
- 但他不是富家少爷。
- 他花钱方式比赵长生粗，制造两人反差。
