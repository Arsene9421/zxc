# 丹药体系

## 1. 丹药作用

丹药解决四类问题：

| 类型 | 用途 |
|------|------|
| 修炼丹 | 增加灵气吸收和法力积累 |
| 突破丹 | 辅助跨境界突破 |
| 疗伤丹 | 治疗外伤、内伤、经脉损伤 |
| 特殊丹 | 解毒、避瘴、养魂、洗髓、延寿 |

## 2. 丹药品质

| 品质 | 状态 | 效果 |
|------|------|------|
| 废丹 | 药性散乱、有毒或霉败 | 正常不能服用 |
| 劣丹 | 可服但杂质重 | 丹毒明显 |
| 成丹 | 标准品质 | 正常效果 |
| 良丹 | 药性更稳 | 副作用少 |
| 上品丹 | 药性纯净 | 价格高 |
| 极品丹 | 接近无杂质 | 稀有，不宜前期频繁出现 |

归墟炉前期最多把废丹恢复成“低效劣丹”或接近成丹，不能直接炼出极品丹。

## 3. 丹毒

丹毒是限制丹药堆修为的重要规则。

来源：

- 药材年份不足。
- 丹炉火候差。
- 炼丹师水平低。
- 废丹强行服用。
- 短期大量吞服。

影响：

- 经脉滞涩。
- 法力虚浮。
- 突破失败率上升。
- 长期需要洗髓或排毒。

赵长生使用归墟炉恢复废丹，也不能完全无视丹毒。归墟炉能减少一部分杂质，但会消耗炉火。

## 4. 下界常用丹药

| 丹药 | 境界 | 用途 | 第一卷用途 |
|------|------|------|------------|
| 辟谷丹 | 炼气 | 减少饮食需求 | 废丹恢复试验 |
| 聚气丹 | 炼气 | 增加法力积累 | 赵长生第一桶修炼资源 |
| 养气丹 | 炼气中后期 | 稳定法力 | 后续宗门期 |
| 疗伤丹 | 炼气筑基 | 治疗伤势 | 逃亡和反杀后 |
| 解毒散 | 炼气 | 解瘴毒、药毒 | 青槐旧灾隐线 |
| 筑基丹 | 筑基 | 辅助筑基 | 第二三卷核心资源 |
| 固元丹 | 筑基 | 稳固道基 | 筑基后 |
| 结金丹 | 金丹 | 辅助结丹 | 金丹卷资源 |
| 养魂丹 | 元婴 | 滋养神魂 | 元婴卷 |

## 5. 灵界丹药

灵界丹药重点：

- 炼虚所需虚灵丹。
- 合体所需法身丹。
- 大乘所需道果雏丹。
- 渡劫所需避劫丹。

这些后期再细化。

## 6. 仙界丹药

仙界丹药重点：

- 仙元丹。
- 洗尘丹。
- 稳仙丹。
- 道果丹。

仙界丹药不应成为赵长生飞升后马上大量获取的资源。仙界底层仍要穷。

## 7. 丹方

丹方比丹药更珍贵。

丹方分：

- 完整丹方。
- 残缺丹方。
- 伪丹方。
- 古丹方。

归墟炉可补残方，但需要：

- 足够残字。
- 相关药材残性。
- 多次试验。
- 失败代价。

## 8. 丹炉和炉灰

丹炉影响成丹率。

炉灰有特殊价值：

- 普通炉灰残留微弱药性。
- 古炉灰可能残留炉火。
- 归墟炉可从特定炉灰中炼出温热黑砂。

炉灰线是第一卷特色资源。
