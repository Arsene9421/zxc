# 归墟炉

## 1. 定位

赵长生核心神器。

旧天庭循环九器之一。

职责：

> 接引残败本源，借归墟终末之力拆解归源，再以炉火重炼新生。

前期表现为底层废料捡漏和残物重炼外挂，后期揭示为旧天庭万界循环炉心。

归墟炉不是废品修复器，也不是无限能源外挂。它是旧循环破碎后残留在赵长生手里的归墟接口。

## 2. 外观与当前状态

赵长生开局得到的是残炉主体，不是完整归墟炉。

外观：

- 巴掌大小。
- 炉身漆黑。
- 三足缺一。
- 炉口有裂。
- 炉壁残留古老纹路。
- 隐约刻着两个残字：归墟。

当前状态：

- 炉身可承载低阶残物。
- 炉火只剩一线旧火。
- 炉口裂，封火与藏气能力很弱。
- 炉壁残铭不全，只能显示极少残字。
- 无法直接接引真正归墟潮力。
- 只能通过低阶残物试探出返本、去杂和重炼的边界。

## 3. 与归墟的关系

归墟是万物残败后的终末之海，不是干净灵力池。

归墟炉和归墟的关系：

> 归墟无穷，但不可直接用。归墟炉能用归墟，是因为它能把终末之力炼成重生之火。

归墟炉火源于归墟，但不是无限灵力。

归墟提供的是：

- 终末之力。
- 返本权柄。
- 残败本源的归处。
- 拆解、归源、去杂的规则方向。

归墟炉负责的是：

- 把危险的归墟终末之力关在炉内。
- 将残败之物拆回可处理状态。
- 洗去霉败、污浊、暗伤、残灰和腐朽结构。
- 把尚未死透的残余价值重新排布。
- 在材料、炉火和承载允许时重炼成新的可用形态。

因此，归墟炉不是从归墟取电，而是把归墟的终末力量转成重炼残缺的能力。

## 4. 运转要素

| 要素 | 含义 |
|------|------|
| 残败本源 | 废丹残药性、残器宝气、枯草生机、破界界源、残纹、承灵性等可炼之物 |
| 外部运转力 | 赵长生法力、灵石、地火、丹炉火、灵脉环境等推动重炼的现实消耗 |
| 归墟炉火 | 源于归墟的旧火种，负责拆解、归源、去杂和重炼；残炉阶段只开一线 |
| 承载结构 | 炉身、炉足、炉口、炉盖、炉耳、炉心、炉底归纹等决定能承受多少归墟之力 |
| 操控者 | 赵长生的修为、神识、经验和判断决定能不能压住过程 |

一句话：

> 材料提供可炼之物，外力推动炉火，归墟炉火负责返本重炼，赵长生负责控制方向和代价。

## 5. 核心重炼动作

正文不需要把这些术语当规则说明书写出来，但作者后台按此判断。

### 5.1 逼残值

把残物里还没散尽的价值逼出来。

例子：

- 丹渣逼出药露、丹砂、小药珠或一次引气药引。
- 枯叶逼回一缕生机，但不能复活整株灵草。
- 破玉简逼出残字，但不能补成完整传承。
- 断符笔逼出一点笔锋承灵性，但不能直接变成完整符器。

### 5.2 洗杂质

完整物品也能入炉，而且比残物更稳定。

例子：

- 下品丹洗去部分丹毒和浊气。
- 普通灵米洗掉霉气，变成能吃的旧灵米。
- 劣质法器洗掉暗伤或杂质。
- 符纸、符笔洗去潮气、裂纹暗伤或灵性堵塞。

### 5.3 合同源

同源残料可以合炼，但要大量损耗杂质。

例子：

- 多份同源丹渣合成小药珠或低效丹丸。
- 坏符纸边角合出少量粗劣符纸。
- 几支坏符笔拆出可用笔锋和承灵性。
- 同类残器合出器砂、器胚或一次性残器。

### 5.4 改用途

归墟炉不一定把东西修回原状，也可以牺牲原用途，改炼成赵长生眼下最需要的一次性活路。

例子：

- 丹渣不够成丹，可以改成一次引气药雾。
- 破符笔不够修成符器，可以改成一次性破禁笔锋。
- 烂法器不够修复，可以炼出一撮能腐蚀锁扣的器砂。
- 炉灰不能当宝物，但可以炼成灼手、遮气或腐蚀绳结的黑砂。

这是归墟炉前期爽点的重要来源：赵长生不是每次都修好东西，而是把废物改成当前关卡能用的活路。

### 5.5 燃代价

越想多榨，越接近归灰、腐蚀、暴露和反噬。

例子：

- 多榨丹渣，丹毒变重。
- 多炼炉灰，手背灼伤。
- 多补玉简，神识刺痛。
- 多洗法器，炉火短暂变暗。
- 多冲品质，材料归灰，气味、灵光或同源波纹外泄。

### 5.6 残灰触感与第一卷尺度

归墟残灰不是普通毒粉。

它的恐怖感来自一件事：让东西忘记自己原本的结构。

第一卷只写最低层残灰接触，不正式展开归墟污染者。

可写表现：

- 摸起来像细砂，但会让指腹先发干、发麻，再从冷变烫。
- 闻起来不是臭，而是旧铁、冷雨、熄灭炉火后的苦味。
- 能遮气味，因为它会吞掉一小截低阶灵气波动。
- 能腐蚀绳结，因为它会让草绳、麻线、旧皮扣的连结变脆。
- 能灼手，不是明火烫伤，而像皮肤被磨掉一层生机。
- 用后会留下灰黑痕，可能让追踪者发现这不是普通炉灰。

写作边界：

1. 第一卷残灰只能作一次性危险工具，不能当稳定毒药、隐身粉或清场手段。
2. 残灰可以遮气、遮血腥、腐蚀绳结、灼伤皮肤，但不能直接毒死修士。
3. 残灰每次使用都要带来新痕迹、新伤口或新暴露风险。
4. 正式归墟污染者中后期再出场，第一卷最多通过触感和后果埋恐怖感。

## 6. 硬限制

1. 归墟不是无限电池，归墟炉不能从归墟无代价抽取干净灵力。
2. 不能凭空造宝，缺失的药性、宝气、灵气、丹体、主材和道则不能无中生有。
3. 没有残余价值、同源锚点或外加材料的东西不能重炼。
4. 前期只能处理低阶残物和完整低阶物品，不能随意升大阶。
5. 高阶要主材。低阶聚气丹不能靠硬洗变筑基丹。
6. 完整物品可洗、可调、可去暗伤，但不能无代价突破品阶上限。
7. 重炼会牵动炉火、材料、法力、时间、灵气、宝气、魂力、气运或界源等承载代价。
8. 越省料、越冲品质、越改用途，失败和反噬越重。
9. 使用过度产生归墟残灰，可能腐蚀灵草、法器、经脉、神识和地脉。
10. 高阶重炼会留下同源波纹，引来旧天庭残片、新仙界监察者和归墟污染者。
11. 归墟炉能处理残败本源，但不能取代轮回灯复活亡魂，也不能包办血源、魂账、界账和命数账的全部解决。
12. 第一卷不解释归墟炉完整来源、旧天庭九器全貌、新仙界补墟体系和复杂本源理论。

## 7. 残炉与主残片

全书口径：

> 残炉主体在手，六件主残片补炉，终局以墟眼开炉。

归墟炉破碎后有大量炉屑、炉灰、断纹和旧铜残渣，但真正带来阶段性能力恢复的主残片只有六件。

小炉屑用途：

- 补一口炉火。
- 引发一次同源感应。
- 作为短期炉料。
- 提供旧天庭残器证据。
- 暴露赵长生。
- 引来旧器、监察者或污染者。

小炉屑不构成主升级。

六件主残片：

| 3 | 炉口环 / 封火残盖 | 合炼、封气、藏波纹、长时间炼制 | 第四卷或第五卷 | 支撑洞府经营、本命法宝胚和断仙门前的古战场残物 |

### 7.1 主残片获取原则

六件主残片不是地图上的六块拼图。

每一件主残片都必须对应一个旧循环职责被当代制度误用、遮掩或污染后的剧情节点。赵长生获得残片，不是单纯升级，而是打穿一层制度、危机或误用。

| 主残片 | 建议获取方式 | 剧情意义 | 风险 |

| 镇炉足 | 第二卷或第三卷，在青云门废料谷、小秘境或丹房旧炉阵脚中发现 | 让炉火稳定，支撑宗门废料、丹房、残卷和阵盘发育 | 宗门开始怀疑赵长生总能从废物里捡到价值 |
| 炉口环 / 封火残盖 | 第四卷或第五卷，在古战场、断仙门遗址或旧封禁阵中夺得 | 让归墟炉能藏波纹、封火、长时间炼制 | 下界监察线真正抬头，旧器波纹不再完全遮住 |

| 炉心旧火 | 第十一卷左右，在补墟司、新司命台相关旧火库或黑账深层中夺回 | 恢复真正归墟炉火的一部分，能处理仙界级残败本源 | 新司命台后台确认赵长生不是普通异常 |
| 炉底归纹 / 旧炉座 | 第十二卷或第十三卷，在旧天庭废墟、旧炉位或复辟派守护地取得 | 接触归墟炉旧职责和深层权限，为终局开炉做准备 | 旧天庭复辟派要求赵长生继承旧职，赵长生必须拒绝 |

执行要求：

1. 每件主残片都必须先制造麻烦，再恢复能力。
2. 主残片恢复的是结构和权限，不是给归墟炉充电。
3. 小炉屑可以补火、引路、作证据、制造风险，但不能替代主残片。
4. 主残片获取不能都靠捡漏，必须混合交易、争夺、误判、反杀、证据链和理念选择。
5. 每次补炉后，赵长生能处理的层级上升，暴露层级也必须同步上升。

## 9. 第一卷执行尺度

第一卷只展示残炉主体的最低层玩法，不解释完整归墟炉。

第一卷试炉原则：

1. 每次试炉必须解决一个现实困境，不只是展示功能。
2. 每次收益必须制造新的风险。
3. 每次结果都要有限，不能把残炉写成稳定生产机器。
4. 炉灰、丹毒、气味、神识刺痛、材料缺斤和旧货街记忆都可以作为代价。
5. 第一卷卷尾吸收的残炉碎片按“炉壁残铭”处理，主要提供残字和感应，不带来战力暴涨。

## 11. 终局开炉机制

“以墟眼为炉口”不是诗性口号，也不是赵长生用归墟炉吞掉墟眼。

墟眼不是普通洞口，而是仙界核心被归墟打穿后的规则伤口。新仙界长期往伤口里填灵、宝、魂、血、运、法、界、时、道等九源，只能短暂止血，不能恢复循环。

终局开炉分五步：

1. 公开旧账和黑账，剥夺玄阙对“只有我们能补墟”的解释权。
2. 切断永久收割式九源投入，不再把众生、亡魂、百族和下界当填料。
3. 六件主残片归位，让归墟炉具备承载墟眼伤口的结构。
4. 九器恢复职责但不复辟旧天庭，各自接回命、魂、界、法、道、登仙等接口。
5. 赵长生以归墟炉把墟眼边缘重炼成循环炉口：残败本源可以归墟，但必须经炉火、九器和万界节点重炼后回流，而不是无底吞噬。

六件主残片在终局中的作用：

| 主残片 | 终局作用 |
|--------|----------|
| 炉壁残铭 | 给墟眼重新命名，从吞噬伤口定为循环炉口 |
| 镇炉足 | 稳住墟眼边缘，防止重炼时仙界核心崩塌 |
| 炉口环 / 封火残盖 | 给伤口立边界，防止归墟潮外泄 |
| 引源耳 | 接入万界节点，让本源回流不再由仙界垄断 |
| 炉心旧火 | 点燃真正重炼循环的炉火 |
| 炉底归纹 / 旧炉座 | 承接旧循环职责，但不恢复旧天庭权力结构 |

终局代价：
