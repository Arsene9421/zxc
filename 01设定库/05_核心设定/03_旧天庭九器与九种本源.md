# 旧天庭九器与九种本源

## 1. 九器

旧天庭九器不是简单收集品。

它们分别对应万界循环的一条规则。

| 器名 | 旧职能 | 新仙界扭曲 | 终局重炼方向 |
|------|--------|------------|--------------|
| 归墟炉 | 回收残败本源，重炼新生 | 残炉主体流落下界，六件主残片和大量小炉屑散落 | 以墟眼为炉口，重新成为万界循环炉心 |
| 司命盘 | 观测命数裂痕，扶持补天之人 | 改造成天命系统 | 不再安排命运，只记录危机和自愿选择 |
| 登仙台 | 接引完成修行者进入更大循环 | 改造成飞升收割通道 | 恢复真正飞升 |
| 补天尺 | 修补世界裂缝 | 被封存或只用于仙界本土 | 重新用于万界补天 |
| 万法碑 | 保存万界道统 | 道统被仙界垄断 | 开放道统传承 |
| 轮回灯 | 接引残魂，维持轮回 | 亡魂被炼成资源或封入仙器 | 让生死重新进入循环 |
| 镇界印 | 稳定一界本源 | 被用于锁死下界和抽源 | 稳定万界，而非压榨万界 |
| 太虚舟 | 横渡世界之外，接引新道 | 被仙界控制，阻断外部变量 | 重新连接太虚可能 |
| 斩道剑 | 斩断腐朽大道 | 被新仙界用来镇压异端 | 防止新循环再次僵化 |

### 1.1 归墟炉破碎口径

归墟炉不是简单收集品，主线不写成九十九块碎片拼图。

正式口径：

> 残炉主体在手，六件主残片补炉，终局以墟眼开炉。

六件主残片分别对应炉壁残铭、镇炉足、炉口环或封火残盖、引源耳、炉心旧火、炉底归纹或旧炉座。大量小炉屑可作为补火、引路、证据和风险来源，但不构成阶段性主升级。

### 1.2 九器职责边界

一句话边界：

> 归墟炉负责处理残败本源，其他九器负责各自领域的秩序接口。归墟炉能重炼残缺，但不能独占命、魂、界、法、道的最终解释权。

| 器或机构 | 负责什么 | 归墟炉不能抢什么 |
|----------|----------|------------------|
| 归墟炉 | 拆解、去杂、逼残值、合同源、改用途、重炼残败本源 | 不能直接改命、复活、稳界、传法、斩道 |
| 司命盘 | 记录命数裂痕、危机和自愿选择 | 归墟炉不能安排命运 |
| 轮回灯 | 残魂归处、生死边界、亡魂入轮回 | 归墟炉不能随便复活死人 |
| 镇界印 / 补天尺 | 世界裂缝、界壁、灵脉、修补尺度 | 归墟炉不能单独修好整个世界 |
| 万法碑 | 道统、功法、术法传承 | 归墟炉不能凭空补完整神功 |
| 万宝殿 | 宝物孕育、器脉、完整宝物体系 | 归墟炉偏废器残本源，不是万能炼宝殿 |
| 登仙台 | 正常接引飞升者进入大循环 | 归墟炉不能替代飞升秩序 |
| 太虚舟 | 太虚航行、外来变量、世界间通路 | 归墟炉不能变成随意穿界工具 |
| 斩道剑 | 斩断腐朽大道和固化权柄 | 归墟炉不能靠“炼一下”解决制度腐烂 |

执行要求：

1. 中后期每次归墟炉触及命、魂、界、法、道，都必须引出对应九器或对应职责。
2. 归墟炉可以把残败本源处理成可用状态，但最终秩序要由对应职责承接。
3. 终局不是归墟炉包办九器，而是归墟炉开炉，九器分职，万界共担循环。

## 3. 时间本源限制

时间能力风险极高。

“时”只能用于：

- 古今残影。
- 遗迹回溯。
- 世界流速异常。
- 太虚航行定位。

前中期禁止随意逆转时间。
