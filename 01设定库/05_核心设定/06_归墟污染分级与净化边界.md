# 归墟污染分级与净化边界

> 本文档细化归墟污染的等级、传播、检测、净化、社会位置和写作边界。核心目标：让归墟污染者既能成为敌人，也能成为制度失败的证据，而不是普通魔物。

---

## 1. 基本定义

归墟污染不是归墟本身邪恶。

归墟污染的本质是：

> 残败本源长期无法进入正常重炼循环，沉积、腐烂、反噬到生灵、器物、地脉、魂魄和世界之中。

归墟是终末和回收机制。

污染是循环堵塞后的腐烂现象。

### 1.1 感官写法与恐怖感

归墟污染的恐怖点不是“黑气很毒”，而是残败本源让生灵、器物和世界忘记自身原本结构。

低层残灰表现：

- 旧铁、冷雨、熄灭炉火后的苦味。
- 指腹先干麻，再从冷变烫。
- 灵草不是枯萎，而是根系像忘了怎么吸水。
- 法器不是碎裂，而是器纹一寸寸失去承灵性。
- 绳结、皮扣、木门闩会变脆，像连结本身被磨掉。

高层污染表现：

- 人不是单纯发疯，而是记忆断裂、寿元漏失、执念被放大。
- 法宝不是普通损坏，而是灵性腐烂，器灵可能保留一部分痛苦意识。
- 地脉不是贫瘠，而是时序、节气和生机循环开始错乱。
- 残界不是荒凉，而是世界残民仍活在不断重复的失败回声里。

一句话写法：

> 法器不是碎了，而是灵性一层层腐烂；人不是疯了，而是记忆断裂、寿元漏失、执念被放大；世界不是荒了，而是地脉开始忘记春天该怎么来。

第一卷只写残灰接触层，不正式展开污染者。

## 2. 污染来源

| 来源 | 说明 | 常见地点 |
|------|------|----------|
| 归墟残灰 | 归墟炉或残败本源重炼不完全留下的灰烬 | 废丹堆、旧器残片、古战场 |
| 墟眼泄漏 | 仙界核心墟眼向外渗出的残败力量 | 中天玄阙、仙界裂域 |
| 补墟失败 | 跨源强行填补导致本源腐烂 | 补墟阵、血阵、魂阵 |
| 残界腐化 | 破碎世界长期无人修复 | 灵界残界、秘境废墟 |
| 轮回堵塞 | 亡魂不能入循环，怨魂和魂力沉积 | 古战场、渡劫失败地 |
| 旧器失衡 | 旧天庭器碎片职能残缺 | 九器遗迹、旧天庭废墟 |

污染不一定来自坏人作恶。

很多污染来自制度拖延、循环堵塞和错误补救。

## 3. 污染对象

| 对象 | 表现 |
|------|------|
| 修士 | 寿元异常流失、记忆断裂、道心执念放大 |
| 妖兽 | 真形腐烂、血脉失控、攻击性上升 |
| 法宝 | 器灵疯癫、器纹腐蚀、宝气变浊 |
| 丹药 | 药性逆转、丹毒变异、服后伤命火 |
| 灵田 | 灵草畸变、根系黑化、灵机滞涩 |
| 洞天 | 界壁渗灰、时序混乱、地脉衰败 |
| 残魂 | 重复死亡记忆、无法入轮回 |
| 世界 | 天灾增多、天劫异常、飞升通道畸变 |

## 4. 污染分级

| 等级 | 名称 | 状态 | 是否可逆 | 典型表现 |
|------|------|------|----------|----------|
| 0 | 残灰接触 | 表层沾染 | 可清理 | 法器发暗、灵草萎黄、修士疲惫 |
| 1 | 浅染 | 灵机或法力轻度受污 | 大多可逆 | 法力滞涩、梦见残影、寿元轻微流失 |
| 2 | 入脉 | 经脉、器脉或地脉被污染 | 可治但有代价 | 经脉刺痛、法宝反噬、灵田黑根 |
| 3 | 蚀魂 | 神魂和记忆被残败本源侵入 | 部分可逆 | 记忆断片、幻听、执念放大 |
| 4 | 腐源 | 命火、本源、血源或界源受损 | 很难逆转 | 寿元暴跌、血脉枯败、洞天裂解 |
| 5 | 半墟化 | 自我与污染纠缠，仍有残存理智 | 只能局部救治 | 身体或法宝半腐化，靠近残界 |
| 6 | 墟潮载体 | 成为污染扩散节点 | 基本不可逆 | 引发污染潮、吞噬本源、诱发墟眼共鸣 |

高等级污染者不等于该被简单杀掉。

但若已成为墟潮载体，处理重点从救个人变成保护更多生灵。

## 5. 污染传播

归墟污染不是普通瘟疫。

它通过“残败本源接触”传播。

| 传播方式 | 说明 |
|----------|------|
| 本源接触 | 接触高污染残器、残灰、残界土壤 |
| 法力互侵 | 斗法中污染法力侵入经脉 |
| 神识探查 | 神识强行窥探污染核心后反噬 |
| 炼化误用 | 将污染材料炼丹、炼器、布阵 |
| 血源牵连 | 百族源血被污染后牵连族契 |
| 洞天扩散 | 洞天界核受污后污染内部灵机 |
| 补墟回流 | 新仙界强行跨源补墟失败后反冲 |

低等级污染不会一碰就死。

高等级污染会通过法力、神魂、本源和界源层面扩散，防治难度远高于凡俗疫病。

## 6. 检测手段

| 手段 | 能看见什么 | 风险 |
|------|------------|------|
| 肉眼观察 | 黑灰、腐斑、灵草畸变 | 只能看表层 |
| 神识探查 | 法力滞涩、器脉腐烂 | 神魂反噬 |
| 医修辨伤 | 经脉、命火、丹毒异常 | 需近距离接触 |
| 阵法测污 | 地脉、洞天、灵田污染 | 布阵耗材 |
| 归墟炉感应 | 残败本源类型和可重炼性 | 炉火消耗，同源波纹 |
| 轮回灯残光 | 残魂是否堵塞轮回 | 需要旧天庭传承 |
| 补墟账本 | 污染来源和责任链 | 仙界篇才可接触 |

神识探查污染物必须谨慎。

低阶修士强行探查三级以上污染，可能直接形成神魂伤。

## 7. 净化方式

| 污染等级 | 可用方式 | 代价 |
|----------|----------|------|
| 0 | 清洗、焚烧、低阶净尘符、隔离 | 材料损耗 |
| 1 | 医修导引、净灵丹、阵法排污 | 法力和时间 |
| 2 | 续脉、排污阵、重炼受污材料 | 经脉或器脉会留下弱点 |
| 3 | 养魂、护魂、轮回灯残法、长期闭关 | 记忆缺损或心魔 |
| 4 | 本源修补、归墟炉中后期能力、同源本源替换 | 寿元、本源、炉火巨大消耗 |
| 5 | 局部重炼、封印、剥离污染人格或器灵 | 可能救不回完整本人 |
| 6 | 镇压、拆解污染节点、送入新循环重炼 | 通常不能保留个体原貌 |

净化不是把黑气驱走就完。

污染越深，越涉及“救回来的还是不是原来那个人”。

## 8. 归墟炉的净化边界

归墟炉能处理残败本源，所以能参与净化。

但边界必须明确：

| 阶段 | 能力 | 不可做 |
|------|------|--------|
| 前期 | 清理废丹、残器、枯草中的少量残灰 | 治好污染者 |
| 中期 | 重炼部分受污器物、轻度经脉污染 | 无代价修复命火 |
| 灵界 | 修补残界、祖地、洞天的局部污染 | 直接救回整族亡魂 |
| 仙界 | 配合轮回灯、补墟账本处理污染源头 | 私自重写生死 |
| 终局 | 把墟眼变成循环炉口 | 让污染从此永远不存在 |

归墟炉不是消毒工具。

它的正确用法是让残败本源回到循环，而不是把所有污染都压回别人身体里。

## 9. 污染者社会位置

### 9.1 下界

下界修士多把污染者当成：

- 魔物。
- 邪修。
- 怪病。
- 不祥之人。

下界缺乏检测手段，所以常常错杀、误判或隐瞒。

这给前中期提供悲剧样本。

### 9.2 灵界

灵界对污染者更有经验，但更冷酷。

常见处置：

- 隔离区。
- 污染战场。
- 残界苦役。
- 黑市研究。
- 大族把小族污染记成“自然衰败”。

灵界污染者可以保留理智，他们不必全是敌人。

### 9.3 仙界

仙界官方把污染者当成证明补墟必要性的证据。

常见说法：

> 看，归墟污染如此危险，所以下界和灵界必须继续供源。

真相是：

新仙界长期堵塞循环和强行补墟，本身就是污染扩大的原因之一。

## 10. 与魔修、邪修区别

| 类型 | 根源 | 是否主动选择 |
|------|------|--------------|
| 邪修 | 夺取他人资源、命火、本源 | 多数主动 |
| 魔修 | 极端功法和执念道路 | 可主动，可被逼 |
| 鬼修 | 肉身死后以魂体修行 | 不必邪恶 |
| 归墟污染者 | 被残败本源侵蚀 | 多数不是主动 |

污染者可以作恶，但不能因为被污染就天然有罪。

写法要区分：

- 被污染仍努力保持理智的人。
- 借污染力量主动害人的人。
- 被制度丢进污染环境的受害者。
- 已经成为墟潮载体的灾害节点。

## 11. 剧情使用

| 阶段 | 用法 |
|------|------|
| 下界 | 小范围污染样本，展示归墟不是邪恶但很危险 |
| 第五卷 | 古战场残魂和天劫异常与污染挂钩 |
| 灵界 | 残界、百族祖地、血源污染扩大 |
| 仙界 | 污染成为补墟制度的反证 |
| 终局 | 墟眼是最高等级循环堵塞，不是一个怪物巢穴 |

## 12. 写作硬规则

1. 归墟污染不是归墟本身邪恶，而是循环堵塞后的腐烂。
2. 污染等级越高，越不能轻易治好。
3. 神识探查污染物有风险，不能免费侦察。
4. 归墟炉能重炼残败本源，但不能无代价救回所有污染者。
5. 污染者要有社会位置和悲剧感，不能全写成怪物。
6. 新仙界必须利用污染制造“补墟必要性”的舆论。
7. 终局解决的是循环堵塞，不是杀光污染者。
