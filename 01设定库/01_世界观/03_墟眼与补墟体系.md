# 墟眼与补墟体系

## 1. 墟眼名称

| 叫法 | 使用者 | 含义 |
|------|--------|------|
| 墟眼 | 旧天庭遗民、知情高层 | 真实称呼，指仙界核心归墟伤口 |
| 天阙 | 新仙界官方 | 美化称呼，意为天有缺，需仙界补之 |
| 天漏 | 仙界底层、飞升者私下 | 俗称，指仙界一直在漏本源 |
| 归墟伤 | 少数旧天庭文献 | 强调这是归墟倒灌造成的伤口 |

## 2. 位置

墟眼位于仙界核心禁域：

> 中天玄阙。

中天玄阙三层：

| 层级 | 表象 | 真相 |
|------|------|------|
| 玄阙天宫 | 仙界最高中枢 | 对外展示威严 |
| 补天内府 | 官方补天机构 | 处理万界本源和飞升者资源 |
| 墟眼禁域 | 绝密禁地 | 归墟之力倒灌留下的伤口 |

## 3. 表现

墟眼不是普通地洞，而是无法愈合的道则伤口。

表现：

- 周围大道规则破碎。
- 仙灵气被缓慢吞噬。
- 靠近者寿元、命数、法宝灵性被磨损。
- 仙界高层必须不断投入界源、气运、魂力、宝气维持封禁。

## 4. 补天司与补墟司

新仙界对外机构：

> 补天司。

内部真实部门：

> 补墟司。

补天司继承旧天庭补天院名义，宣称维护万界稳定。

补墟司负责实际收割、分拣、运输、投入墟眼。

## 5. 补墟体系

| 机构 | 职责 |
|------|------|
| 补天司 | 对外管理补天、镇界、飞升事务 |
| 补墟司 | 内部负责墟眼填补和本源分配 |
| 新司命台 | 运行天命系统，筛选可培养棋子 |
| 登仙监 | 管理飞升通道、验源、登记 |
| 镇界军 | 镇压反抗下界、归墟污染者和飞升叛修 |
| 万宝库 | 收缴、拆解、炼化飞升者法宝和古宝 |
| 仙矿府 | 安置低价值飞升者，让其挖仙矿、交仙税 |
| 洗尘池 | 清洗飞升者下界因果，实为检测与剥离本源 |

## 6. 补墟资源

| 资源 | 来源 | 用途 |
|------|------|------|
| 界源 | 下界、中层世界 | 稳定墟眼边缘 |
| 气运 | 气运强者、系统宿主、宗门、王朝 | 延缓仙界道则衰败 |
| 魂力 | 渡劫失败者、亡魂、轮回残片 | 填补墟眼深层裂缝 |
| 宝气 | 古宝、灵宝、飞升者本命法宝 | 修补仙界法则器脉 |
| 血脉 | 真灵、妖族、特殊体质 | 强化封禁阵基 |
| 道果 | 高阶修士道途凝结 | 供仙界高层续命和稳道 |

## 7. 补墟周期

| 层级 | 周期 | 内容 | 下界表现 |
|------|------|------|----------|
| 日常补墟 | 常年 | 飞升者、系统棋子、仙税、废料上缴 | 废丹废器也有人回收，底层资源越来越紧 |
| 千年抽源 | 约千年 | 抽取若干下界灵脉、宗门气运、古宝残片 | 某些修仙国突然衰落，灵脉枯竭 |
| 万年大补 | 约万年 | 从灵界级世界抽取界源 | 灵界百族大战、天劫异常、仙界监察频繁 |

## 8. 破局

墟眼不能只补，必须重炼。

赵长生终局不是找到更高级材料堵洞，而是把墟眼从吞噬万界的伤口，重炼成万界循环的炉口。
