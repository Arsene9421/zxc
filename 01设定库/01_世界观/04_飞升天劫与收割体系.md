# 飞升、天劫与收割体系

## 1. 旧飞升

旧天庭时期，飞升是登仙台接引。

修士完成自身修行后，登仙台接引其进入更大的天地循环。

飞升者根据道途进入：

- 继续修行。
- 镇守一界。
- 参与补天。
- 护持轮回。
- 入太虚寻道。
- 守归墟炉火。
- 传法万界。

## 2. 新飞升

新仙界时期，正常仙门已断。

下界残留的飞升通道，是新仙界改造过的收割通道。

飞升不是奖励，而是：

- 筛选。
- 登记。
- 验源。
- 分流。
- 回收。

## 3. 天劫变化

| 旧天劫 | 新天劫 |
|--------|--------|
| 考验修士是否能承载更高道则 | 检测修士本源、气运、法宝、血脉 |
| 渡过后接引入登仙台 | 渡过后进入新仙界登记体系 |
| 失败者魂归轮回或散入归墟 | 失败者可能被仙界提前回收 |

## 4. 飞升者命运

| 类型 | 命运 |
|------|------|
| 普通飞升者 | 登记、服役、挖仙矿、交仙税 |
| 战力强者 | 编入仙兵，替仙界镇压下界或归墟污染 |
| 血脉特殊者 | 被仙族、大族、仙府带走研究或炼血 |
| 法宝特殊者 | 法宝被夺，人被监控 |
| 气运强者 | 送入司命台系统阵营 |
| 天命系统持有者 | 成熟后回收系统成果 |
| 不服管束者 | 打入矿场、炼成仙器材料或被抹去命数 |

## 5. 前中期痕迹

第一卷到第九卷逐步埋：

- 飞升传说越来越少。
- 古籍飞升记载和现实不符。
- 化神修士渡劫失败后消失得太干净。
- 天劫像是在扫描，而不只是考验。
- 归墟炉聚魂后能看到渡劫失败者被收走的残影。
