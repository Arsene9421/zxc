# 仙界底层区域地图

> 地图地理补档。本文用于承载第十卷仙界底层：飞升台外营、登仙监、洗尘池、验源廊、仙籍署、分流台、仙矿府、飞升者役区、黑籍坊市、残仙器坊、镇界军征调营、新司命台分殿和补墟司外账房。本文不展开中天玄阙、补墟黑账、旧天庭废墟和墟眼禁域。

---

## 1. 设计目的

仙界底层地图要解决七个问题：

1. 主角组从登仙台残路进入仙界后，为什么仍然是底层，而不是飞升成仙就自由。
2. 飞升者如何被登记、验源、洗尘、录籍、分流、征税和服役。
3. 赵长生如何继续保持凡人流气质：低调、伪装、躲账、捡漏、从残仙器和废仙石重新起家。
4. 纪晚凝为什么能在第十卷合理登场：她是制度执行者，负责飞升者登记、验源和分流，不是直接掌握黑账的高层。
5. 飞升者群像如何落地：周栖石在仙矿，梅见微在黑籍坊市，钟离砺在镇界军征调营，许守仓在仙籍署，陶铸在残仙器坊。
6. 沈临风系统后台如何在第十卷露出接口，但不直接进入新司命台中枢。
7. 第十一卷补墟司和司命台线如何从第十卷底层空间自然接上。

仙界底层的核心味道：

> 飞升不是抵达自由，而是从天劫筛选口走进一套更完整、更冷静、更有账本的登记体系。

## 2. 总地图

第十卷仙界底层先开放一片区域，不展开整个仙界。

```text
                         中天玄阙方向 / 不开放
                                  ↑
                         补墟司内城 / 不开放
                                  ↑
        新司命台分殿 —— 登仙监内署 —— 镇界军征调营
              ↑              ↑              ↑
        命数校验室       仙籍署       军籍分流台
              ↑              ↑              ↑
       黑籍坊市 —— 飞升台外营 —— 仙矿府外营
              ↑              ↑              ↑
        残仙器坊       洗尘池       仙矿役区
              ↑              ↑
          废仙石堆 —— 验源廊 —— 分流台
                           ↑
                    登仙台残路落点
```

底层区域关系：

| 区域 | 控制者 | 功能 | 第十卷用途 |
|------|--------|------|------------|
| 登仙台残路落点 | 登仙监、验源仙使投影残权 | 接收飞升者和异常落点 | 主角组入界、异常波纹 |
| 飞升台外营 | 登仙监、低阶仙籍官 | 临时拘束、编号、等候 | 飞升者群像、仙界底层第一印象 |
| 洗尘池 | 洗尘使、补墟司外派、登仙监 | 清洗、检测、标记 | 本源、血源、法宝、系统痕迹风险 |
| 验源廊 | 登仙监、补墟司接口 | 灵、宝、魂、血、运、法、界、时、道初筛 | 赵长生和沈临风异常 |
| 仙籍署 | 许守仓等低阶仙籍官 | 临籍、役籍、军籍、罪籍、黑籍接口 | 仙籍压迫具体化 |
| 分流台 | 登仙监、仙矿府、镇界军、仙府下线 | 分配役期和去处 | 周栖石、钟离砺等分流 |
| 仙矿府外营 | 仙矿府、矿吏 | 飞升者矿役 | 周栖石线 |
| 黑籍坊市 | 失籍飞升者、伪籍贩子、灰色商人 | 非法生存、伪籍、情报 | 梅见微线、赵长生躲账 |
| 残仙器坊 | 陶铸、黑市器修、废料商 | 残仙器、废仙石、器纹 | 赵长生仙界捡漏 |
| 镇界军征调营 | 镇界军、军籍仙官 | 征调战力强者 | 钟离砺线 |
| 新司命台分殿 | 司命台下级管理员 | 系统接口、命数校验 | 沈临风系统后台初露 |
| 补墟司外账房 | 纪晚凝、严慎前置接口 | 明账、分流额度、验源记录 | 纪晚凝制度线开端 |

## 3. 仙界底层规则

### 3.1 飞升者先是被拘束者

飞升者刚到仙界，不是“仙人同道”，而是待登记对象。

他们必须经过：

```text
落点拘束
→ 临时编号
→ 洗尘池
→ 验源廊
→ 仙籍署
→ 分流台
→ 役籍 / 军籍 / 府籍 / 监控 / 黑籍风险
```

流程表面有序，实际目标是：

1. 接管身份。
2. 接管命数接口。
3. 检测本源价值。
4. 拆分资源去向。
5. 安排仙税和役期。
6. 判断是否值得培养、监控、征调或回收。

### 3.2 仙界底层不是贫民窟，而是制度低层

仙界底层要有秩序感。

它不是脏乱无规则的凡俗棚区，而是：

1. 登记清楚。
2. 税役明确。
3. 出入有牌。
4. 资源有价。
5. 黑市存在但被默许。
6. 飞升者想活下去，必须学会在制度缝隙里动。

赵长生的凡人流气质在这里更重要：

> 他不能用下界声望压人，只能重新从一个残仙器、一块废仙石、一张临籍牌开始。

## 4. 登仙台残路落点

主角组从第九卷登仙台残路进入仙界，合理落点不应是仙宫，而是登仙监控制下的残路落点。

落点类型：

| 类型 | 状态 | 剧情用途 |
|------|------|----------|
| 正常接引点 | 按队列进入飞升台外营 | 其他飞升者群像 |
| 残路偏落点 | 阵路不稳，落在外营边缘或废阵沟 | 主角组可低调混入 |
| 黑籍暗渠 | 黑市伪籍者利用的裂隙 | 梅见微线和赵长生躲账 |
| 验源旁路 | 被洗源阵扫到但未完整录入 | 赵长生命数断页、归墟炉异常 |

落点风险：

1. 验源残光扫描。
2. 仙籍官核对飞升痕迹。
3. 登仙监巡守清点人数。
4. 新司命台分殿读取命数。
5. 沈临风系统被后台感应。

## 5. 飞升台外营

飞升台外营是第十卷第一主舞台。

组成：

1. 临时编号棚。
2. 飞升者等候区。
3. 低阶医治处。
4. 临籍牌发放处。
5. 守营仙兵岗。
6. 分流告示墙。
7. 仙税说明碑。
8. 飞升者争执区。

飞升者群像在这里出现：

| 人物 | 空间位置 | 功能 |
|------|----------|------|
| 周栖石 | 等候区或仙矿府点名队列 | 证明普通飞升者会被役籍和矿役压住 |
| 梅见微 | 外营边缘或黑籍暗渠 | 展示失籍者如何躲登记 |
| 钟离砺 | 分流告示墙或军籍队列 | 展示战力强者被征调 |
| 许守仓 | 临籍牌发放处 | 低阶仙籍官，执行制度而非终极反派 |
| 纪晚凝 | 验源异常复核或分流复核现场 | 冷静、专业、制度执行者登场 |

外营的压迫感来自：每个人刚飞升时都带着希望，但站在告示墙前发现自己先欠了一笔仙税和役期。

## 6. 洗尘池

洗尘池表面说法是洗去下界尘缘，实际是检测、剥离和标记。

组成：

1. 洗尘外池。
2. 验源水道。
3. 道基映照壁。
4. 法宝暂扣台。
5. 血源取样亭。
6. 命数校验镜。
7. 异常隔间。

检测内容：

| 项目 | 表层说法 | 实际作用 |
|------|----------|----------|
| 下界浊气 | 洗去尘垢 | 判断出身世界和界源痕迹 |
| 法宝灵性 | 防止危险器物入界 | 标记、暂扣、拆解或入库 |
| 血脉 | 记录仙体适配 | 血税、血源、仙族研究 |
| 神魂 | 稳定飞升者 | 魂账、轮回异常、司命接口 |
| 气运 | 判断仙途 | 新司命台筛选棋子 |
| 道统 | 登记功法 | 法税和道统封锁 |

赵长生风险：

1. 归墟炉波纹可能被洗尘水激发。
2. 命数断页让校验镜读数不连续。
3. 青槐旧灾和旧器残痕可能引来异常复核。

沈临风风险：

1. 系统接口可能被命数校验镜识别。
2. 新司命台分殿可能发出后台召回或定位。

纪晚凝登场可选位置：

1. 异常复核间。
2. 血源取样亭记录异常。
3. 分流复核台审核明账。

她不应一出场就看见黑账。她能看见的是明账里的异常和流程上的不合理。

## 7. 验源廊

验源廊连接洗尘池和仙籍署，是第十卷最重要的制度空间之一。

九类初筛：

| 验源口 | 检测 | 后续去向 |
|--------|------|----------|
| 灵源口 | 灵根、法力、灵脉痕 | 普通仙籍、仙矿、仙府 |
| 宝源口 | 本命法宝、器纹、古宝 | 万宝库、残仙器标记 |
| 魂源口 | 神魂、残魂、轮回痕 | 轮回异常、魂账监控 |
| 血源口 | 血脉、真血、族源 | 血税、仙族研究、补墟血账 |
| 运源口 | 气运、天命加权 | 新司命台、系统线 |
| 法源口 | 功法、道统、秘术 | 法税、万法碑封锁 |
| 界源口 | 出身世界、洞天、小世界痕 | 界税、抽源名单 |
| 时源口 | 时痕、旧遗迹接触 | 高危封存 |
| 道源口 | 道果、道痕、特殊道途 | 高层监控 |

第十卷只需展示初筛，不要把九源黑账全部解释完。

## 8. 仙籍署

仙籍署位于飞升台外营上层，是临籍、役籍、军籍、罪籍和黑籍压力的具体地点。

组成：

1. 临籍窗口。
2. 役籍登记房。
3. 军籍征调柜。
4. 罪籍封印室。
5. 欠税榜。
6. 失籍追捕牌。
7. 仙籍牌库。
8. 仙籍官值房。

许守仓适合在这里出场。

他不是大反派，而是低阶执行者：

1. 怕失职。
2. 怕登记指标出错。
3. 知道流程不公平，但更怕自己丢职。
4. 可以冷漠，也可以偶尔提醒一句规矩。

仙籍署压迫：

1. 无籍者不能合法租屋、买丹、入城。
2. 临籍者必须按时验源和分流。
3. 欠税会降籍或转役。
4. 黑籍一旦暴露，可转罪籍。
5. 仙籍越完整，资源越多，控制也越强。

## 9. 分流台

分流台连接仙籍署、仙矿府、镇界军、仙府、补墟司外账房和新司命台分殿。

分流去向：

| 去向 | 对象 | 表层理由 | 实际压力 |
|------|------|----------|----------|
| 仙矿府 | 普通低价值飞升者、欠税者 | 役期抵税 | 周栖石线 |
| 镇界军 | 战力强、斗法经验足者 | 战功换籍 | 钟离砺线 |
| 仙府下院 | 血脉、法宝、功法有价值者 | 培养 | 监控和利用 |
| 新司命台分殿 | 气运、系统、命数异常者 | 天命扶持 | 系统回收 |
| 补墟司外账房 | 本源异常、界源异常者 | 复核 | 纪晚凝线 |
| 黑籍坊市 | 逃漏、伪籍、失籍者 | 非法 | 梅见微线 |

分流台的冲突：

1. 飞升者以为分流是安排，实际是定价。
2. 赵长生要避免被界源口和宝源口标记。
3. 沈临风要避免被运源口直接送往新司命台。
4. 纪晚凝能看见分流逻辑，但未必看见分流背后的黑账。

## 10. 仙矿府外营与仙矿役区

仙矿府是第十卷底层压迫的重要空间。

组成：

1. 点名台。
2. 矿役棚。
3. 仙矿入口。
4. 废仙石堆。
5. 矿伤医棚。
6. 役期账房。
7. 逃役追捕处。

周栖石适合在这里出场。

仙矿作用：

1. 证明飞升后仍可能沦为劳役。
2. 提供废仙石、残仙材、矿渣等捡漏入口。
3. 展示役籍和仙税的长期压迫。
4. 给赵长生仙界凡人流重新起步空间。

限制：

1. 仙矿府不是让主角无限刷材料的副本。
2. 仙矿有矿吏、点名、役期、伤亡和逃役风险。
3. 废仙石也可能含污染、验源残痕或追踪印。

## 11. 黑籍坊市

黑籍坊市位于飞升台外营和仙矿府之间的灰色夹缝。

它不是公开合法坊市，但仙界底层默认它存在。

组成：

1. 伪籍摊。
2. 黑籍客栈。
3. 残仙器坊。
4. 失籍医馆。
5. 仙税掮客。
6. 逃役路引铺。
7. 情报茶楼。
8. 黑市传送阵残基。

梅见微适合在这里出场。

黑籍坊市功能：

1. 展示没有仙籍几乎寸步难行。
2. 给赵长生和沈临风提供躲避正式流程的空间。
3. 提供伪籍、废仙石、残仙器、情报和风险。
4. 连接第十一卷司命台和补墟司追查线。

禁写：

1. 不能把黑籍坊市写成自由乌托邦。
2. 不能让伪籍万能。
3. 不能让梅见微天然信任主角。

## 12. 残仙器坊

残仙器坊位于黑籍坊市边缘，是赵长生仙界捡漏起点。

陶铸可作为残仙器坊主。

组成：

1. 残仙器货架。
2. 废仙石堆。
3. 器纹修补台。
4. 熔料炉。
5. 拆器小室。
6. 黑市验货间。

可用资源：

| 资源 | 用法 | 限制 |
|------|------|------|
| 残仙器 | 归墟炉识别残缺、修复低阶仙器碎片 | 有追踪印和器毒风险 |
| 废仙石 | 仙界底层灵力来源 | 品质不稳，可能欠税追踪 |
| 断器纹 | 陆维舟后续界阵和仙器纹路对照 | 不能一眼全懂 |
| 旧天庭器痕 | 远期伏笔 | 第十卷不能解释完整九器 |

残仙器坊的意义：

> 仙界篇继续凡人流，不是主角飞升后直接抢仙宝，而是从别人不要、看不懂、修不起、怕追踪的残物里重新开路。

## 13. 镇界军征调营

镇界军征调营位于分流台另一侧，负责把战力强、斗法经验足、出身有特殊价值的飞升者编入军籍。

组成：

1. 军籍登记台。
2. 战力测试场。
3. 军印绑定室。
4. 仙兵营房。
5. 镇界任务墙。
6. 伤兵回收处。

钟离砺适合在这里出场。

镇界军作用：

1. 证明强者也不自由，强者会被征调。
2. 展示飞升者可能被用来镇压其他下界或归墟污染。
3. 为第十三卷镇界军和厉沉霄线提前埋空间。

第十卷不要展开镇界军高层，只展示征调营和军籍压力。

## 14. 新司命台分殿

新司命台分殿位于登仙监内署旁，不是中枢。

第十卷只展示接口：

1. 命数校验室。
2. 气运评级镜。
3. 系统异常记录柜。
4. 天命任务残符台。
5. 分殿管理员影像。

沈临风风险：

1. 系统后台有反应。
2. 天命值记录被校验。
3. 系统可能发布“前往分殿复核”或“远离赵长生”类任务。
4. 他拒绝后会失去部分系统保护。

第十卷不能直接让沈临风夺司命台权限。那是第十一卷以后。

## 15. 补墟司外账房

补墟司外账房不是真正补墟司核心，而是登仙监、洗尘池、验源廊和分流台之间的明账接口。

纪晚凝可在这里办公或复核。

组成：

1. 飞升者分流明账。
2. 验源异常记录。
3. 仙税额度表。
4. 役期折算簿。
5. 法宝暂扣清单。
6. 血源取样记录。
7. 纪晚凝值房。

纪晚凝第十卷边界：

1. 她能看明账。
2. 她能复核异常。
3. 她能执行仙籍、仙税、验源制度。
4. 她会压下个体痛苦以维护总账稳定。
5. 她不能一开始就看到黑账。
6. 她不能因为赵长生一句话动摇。

她的第一次裂缝应来自：

1. 赵长生命数断页录不进去。
2. 沈临风系统痕迹与司命分殿接口冲突。
3. 白倩霜血源记录与灵界血税旧账不完全吻合。
4. 仙矿役期和飞升者价值分流出现不合理损耗。

## 16. 主要人物动线

## 17. 第十卷常用路线

### 17.1 正常飞升者路线

```text
登仙台残路落点
→ 飞升台外营
→ 洗尘池
→ 验源廊
→ 仙籍署
→ 分流台
→ 仙矿府 / 镇界军 / 仙府下院
```

用途：

1. 展示飞升不是奖励。
2. 展示仙籍、仙税和役期。
3. 让飞升者群像自然出现。

### 17.2 主角组低调路线

```text
残路偏落点
→ 飞升台外营边缘
→ 洗尘池旁路
→ 黑籍坊市
→ 残仙器坊
→ 仙矿废仙石堆
```

用途：

1. 保留凡人流气质。
2. 避免赵长生和沈临风立刻被正式系统锁死。
3. 给归墟炉仙界低阶试验材料。

### 17.3 纪晚凝制度路线

```text
验源廊
→ 仙籍署
→ 分流台
→ 补墟司外账房
→ 异常复核间
```

用途：

1. 让纪晚凝以专业制度执行者身份登场。
2. 让她看见异常，但不立刻背叛制度。
3. 让她与赵长生、沈临风、白倩霜的线产生冲突。

### 17.4 系统后台接口路线

```text
命数校验镜
→ 新司命台分殿
→ 气运评级镜
→ 系统异常记录柜
```

用途：

1. 让沈临风知道系统在仙界有后台。
2. 为第十一卷司命台主线做入口。
3. 不在第十卷解决系统。

## 19. 禁写项

1. 不能让主角组第十卷直接进入中天玄阙。
2. 不能让纪晚凝一出场就知道补墟黑账。
3. 不能让赵长生刚到仙界就公开归墟炉。
4. 不能让沈临风立刻破解系统后台。
5. 不能让仙界底层写成无秩序贫民窟，压迫必须来自制度和账。
6. 不能让黑籍坊市成为万能避难所。
7. 不能让残仙器坊无限提供仙器机缘。
8. 不能让飞升者群像立刻服主角。
9. 不能把仙籍官都写成坏人，许守仓这种低阶执行者要有岗位压力。
10. 不能让飞升者一经洗尘就完全失去反抗能力，否则后续群像无从成立。
11. 不能让第十卷解释完整墟眼和补墟黑账。

## 20. 使用提示

第十卷写仙界底层时，应按三层递进：

1. 第一层：飞升者以为登仙，实际先被拘束、登记、验源。
2. 第二层：赵长生和沈临风从正式流程边缘逃进黑籍坊市和残仙器坊，继续低调求活。
3. 第三层：纪晚凝通过验源、仙籍、分流和外账房与主角组冲突，开始看见明账里的异常。

仙界底层不是终局真相，只是制度入口。

第十卷的核心味道不是“终于到仙界”，而是：

> 下界修士仰望了一生的飞升，在仙界这里只是一个窗口、一块临籍牌、一场洗尘、一笔役期和一张等待分流的账。
