# 新司命台职位体系

> 仙界机构补档。本文用于承载新司命台的职位、权限、空间、任务模板、天命值结算、宿主评级、命数黑账和沈临风反向夺权路径。本文不替代 `05_核心设定/02_天命系统与司命台.md`，也不把钟无咎扩成完整人物单档。

---

## 1. 设计目的

新司命台职位体系补档要解决七个问题：

1. 沈临风的系统后台不是抽象面板，而是新司命台职位、权限、任务模板和命数账本共同运行的结果。
2. 季玄屏作为系统管理员的权限边界在哪里。
3. 钟无咎作为司命台主簿和命数黑账守门人，在职位体系中处于什么位置。
4. 新司命台如何与补墟司、登仙监、仙籍署、天机阁污染、飞升回收发生接口。
5. 天命值、气运评级、系统奖励和宿主回收如何被账本化。
6. 沈临风如何从被任务安排，走向反追踪、夺回残火和重定司命盘规则。
7. 新司命台的罪为什么不是“推演命数”，而是把记录危机变成安排命运。

新司命台的核心味道：

> 它不说“你必须死”，它只把你的一生拆成任务、评级、奖励、惩罚和回收价值。

## 2. 基本定位

| 项目 | 内容 |
|------|------|
| 名称 | 新司命台 |
| 前身 | 旧天庭司命台主体被新仙界夺取、改造 |
| 表层职责 | 观测天命、扶持变量、降低大劫失控概率 |
| 实际功能 | 筛选、培养、安排、记录和回收高价值命数与气运变量 |
| 对外表现 | 天命系统、气运评级、机缘引导、天机污染、任务模板 |
| 活跃卷段 | 第一卷埋异常，第七到九卷反噬，第十到十三卷后台展开 |
| 核心人物 | 沈临风、季玄屏、钟无咎、谢未明 |

旧司命台职责是记录命数裂痕、扶持可能补天之人。

新司命台的问题是：它不再只是记录危机，而是主动安排命运，并把成熟变量接入飞升回收和补墟账本。

## 3. 总空间

新司命台可分为分殿和中枢两层。

```text
登仙监 / 洗尘池 / 命数校验镜
        ↓
新司命台分殿
        ↓
气运评级室 —— 任务模板库 —— 天命值结算室
        ↓
宿主档案阁 —— 奖励调拨台 —— 回收名单室
        ↓
司命玄屏廊
        ↓
命数黑账阁
        ↓
司命盘残核 / 不完全开放
```

| 空间 | 功能 | 相关人物 |
|------|------|----------|
| 新司命台分殿 | 第十卷系统后台接口，不是中枢 | 下级管理员、沈临风风险 |
| 气运评级室 | 评定气运、天命加权、变量价值 | 季玄屏可调用 |
| 任务模板库 | 生成主线、支线、紧急、惩罚任务 | 季玄屏核心权限 |
| 天命值结算室 | 奖励、削减、冻结宿主权限 | 沈临风拒绝任务后进入 |
| 宿主档案阁 | 记录宿主、失败宿主、异常宿主 | 林停云等失败样本 |
| 奖励调拨台 | 调拨实物、机缘、权限型奖励 | 接补墟司、万宝库、仙籍资源 |
| 回收名单室 | 标记成熟宿主或失控宿主 | 飞升回收线 |
| 司命玄屏廊 | 隔绝宿主反向窥探后台 | 季玄屏防线 |
| 命数黑账阁 | 保存命数安排、宿主回收、强制任务源头 | 钟无咎职位节点 |

第十卷只展示分殿和接口，第十一卷进入任务模板、结算和宿主档案，第十三卷才触及命数黑账。

## 4. 职位总表

| 职位 | 权限 | 代表或使用 |
|------|------|------------|
| 分殿录事 | 接收登仙监命数校验、记录飞升者气运异常 | 第十卷底层接口 |
| 气运评级使 | 给修士、宗门、宿主做气运和天命加权 | 系统显示气运值来源 |
| 任务模板管理员 | 维护任务模板、调整任务奖励和惩罚 | 季玄屏 |
| 天命值结算官 | 奖励、削减、冻结、恢复宿主权限 | 季玄屏权限之一 |
| 宿主档案官 | 管理系统宿主、失败宿主、异常宿主档案 | 林停云等失败样本 |
| 奖励调拨官 | 将功法、丹药、法宝、机缘和权限奖励接入系统 | 证明奖励不是凭空 |
| 回收司命使 | 标记成熟宿主、失败宿主和不可控宿主 | 飞升回收接口 |
| 司命玄屏守 | 防止宿主反向窥探后台 | 季玄屏中后期防线 |
| 司命台主簿 | 维护命数黑账、任务源头、封存记录 | 钟无咎 |
| 残火监校官 | 监测旧司命台残火异常 | 沈临风底层残火线 |

钟无咎目前只按司命台主簿和命数黑账守门人处理，不扩写个人弧线。

## 5. 权限层级

| 层级 | 可做 | 不能做 |
|------|------|--------|
| 分殿权限 | 读取命数校验、发送复核任务、标记异常 | 不能改写宿主主线 |
| 评级权限 | 调整气运显示、机缘评级、危险提示 | 不能凭空造机缘 |
| 模板权限 | 发布支线、主线、惩罚任务 | 不能完全理解归墟炉变量 |
| 结算权限 | 奖励、扣除、冻结天命值 | 对拒绝任务后的沈临风会反噬但不绝对控制 |
| 回收权限 | 标记成熟宿主、失败宿主、异常宿主 | 需要登仙监、飞升通道或高层法旨配合 |
| 主簿权限 | 查看命数黑账、封存任务源头、覆盖签押 | 不能修复司命盘旧义，只能维护新仙界秩序 |
| 司命盘残核权限 | 触及旧司命台底层规则 | 被旧残火、沈临风、谢未明和赵长生变量干扰 |

新司命台不是全能天道。它强在系统网络、评级、任务和回收链；弱点是无法完整读取命数断页、归墟炉波纹、旧司命台残火和真正的自愿选择。

## 6. 任务体系

系统任务来自新司命台任务模板库。

| 任务类型 | 表现 | 后台作用 |
|----------|------|----------|
| 新手任务 | 引导宿主熟悉系统、获得小奖励 | 建立依赖和基础命数记录 |
| 支线任务 | 接近某人、探索某地、获取某物 | 引导宿主进入既有机缘或监控区域 |
| 主线任务 | 推动宿主道途和飞升方向 | 把宿主培养成可回收变量 |
| 紧急任务 | 危险预警、逃离、保护关键目标 | 纠正变量偏移 |
| 惩罚任务 | 高奖高罚、强制选择 | 测试宿主服从度 |
| 回收任务 | 前往登仙台、司命台分殿或指定机缘 | 将成熟宿主导入回收流程 |

沈临风第一卷看到的是新手和支线任务；第七到九卷才真正遇到惩罚任务和系统反噬；第十卷以后发现任务来自后台职位体系。

## 7. 天命值结算

天命值不是客观天道认可。

它是新司命台给宿主套上的加权、保护、记录和回收接口。

| 结算行为 | 表层表现 | 实际含义 |
|----------|----------|----------|
| 增加天命值 | 宿主运气变好、奖励变多 | 新司命台加大培养投入 |
| 削减天命值 | 任务失败惩罚、机缘减少 | 取消保护和加权 |
| 冻结天命值 | 系统卡顿、奖励延迟 | 后台复核宿主偏移 |
| 清零风险 | 系统反噬、命数外壳不稳 | 宿主被列入回收或封存名单 |
| 异常溢出 | 系统乱码、任务冲突 | 旧司命台残火和新控制逻辑冲突 |

沈临风拒绝“远离赵长生”任务时，失去的不是天然气运，而是新司命台套在他身上的任务保护和奖励通道。

## 8. 宿主评级

新司命台将系统宿主分为几类。

| 类型 | 定义 | 命运 |
|------|------|------|
| 低价值宿主 | 气运较弱，只能完成基础任务 | 用完后淡出或作为小额回收 |
| 成长型宿主 | 能持续完成任务，积累资源和道途 | 重点培养，飞升后回收成果 |
| 高价值变量 | 异世魂火、特殊灵根、特殊道途 | 重点监控和奖励 |
| 偏移宿主 | 多次违背任务逻辑，但仍有回收价值 | 惩罚、修正、重定向 |
| 失控宿主 | 拒绝系统、反追踪后台、破坏任务源头 | 列入封存或回收名单 |
| 失败宿主 | 飞升失败、系统崩坏、被抽干气运 | 作为案例或残余资源处理 |

沈临风最初是高价值变量，拒绝任务后被转为偏移宿主，再到仙界篇被视为失控宿主。

## 9. 季玄屏的位置

季玄屏是新司命台管理员，主要掌握任务模板、天命值结算、宿主评级和回收标记的中层权限。

他能做：

1. 调整沈临风任务流。
2. 提高或削减任务奖励。
3. 发布“远离赵长生”一类惩罚任务。
4. 调用宿主档案和失败案例。
5. 将沈临风异常上报钟无咎。
6. 用司命玄屏阻止沈临风反向窥探。

他不能做：

1. 完全理解归墟炉。
2. 准确评级赵长生。
3. 单独决定所有命数黑账。
4. 替钟无咎封存最高权限记录。
5. 代表裴玄阙的终局保全道路。

季玄屏的压迫来自后台权限，不来自嘲讽或个人战力。

## 10. 钟无咎的位置

钟无咎目前定位为：

> 司命台主簿，命数黑账守门人。

职位功能：

1. 维护命数黑账。
2. 封存任务源头。
3. 覆盖中层管理员签押。
4. 处理高危宿主和失控变量。
5. 将不可公开的任务源头与保全派法旨连接。
6. 在第十三卷阻止沈临风夺回司命盘规则。

钟无咎和季玄屏的区别：

| 人物 | 功能 |
|------|------|
| 季玄屏 | 任务模板、天命值、宿主评级、系统操作层 |
| 钟无咎 | 命数黑账、任务源头、封存记录、主簿高权限 |

写作边界：

1. 暂不扩写钟无咎个人身世。
2. 暂不写完整人物弧线。
3. 不让他替代裴玄阙的最高理念反派功能。
4. 不让他替季玄屏承担系统中层对手功能。
5. 他是命数黑账门后的人，不是新司命台全部罪责。

## 11. 沈临风反向夺权路径

沈临风不能靠砸系统面板解决问题。

合理路径：

```text
第一卷：发现系统无法读取赵长生
第二到五卷：发现任务频繁围绕赵长生和旧物
第七到八卷：系统将百族血源视为任务价值
第八到九卷：拒绝远离赵长生任务，天命值被削
第十卷：仙界底层发现新司命台分殿接口
第十一卷：反追踪任务模板和宿主档案
第十二卷：接触旧司命台残火和司命盘旧义
第十三卷：与季玄屏、钟无咎争夺命数黑账和司命规则
第十四卷：重定司命盘，只记录危机和自愿选择
```

夺权代价：

1. 短期失去系统保护。
2. 风灵根残缺问题暴露。
3. 奖励渠道断裂。
4. 命数外壳不稳。
5. 被新司命台列入失控宿主。
6. 需要谢未明命数证据和赵长生归墟炉变量配合。

## 13. 命数黑账

命数黑账不是普通宿主档案。

内容包括：

1. 系统任务真实源头。
2. 高价值宿主回收名单。
3. 失败宿主处理记录。
4. 天命值折损账。
5. 气运加权和抽取记录。
6. 赵长生不可评级记录。
7. “远离赵长生”任务源头。
8. 保全派要求司命台配合的法旨。

命数黑账的可怕之处：

> 它证明系统不是天道奖励，而是一套把异数培养、定价、修正和回收的制度。

## 16. 可用剧情动作

写新司命台时，不要只写系统乱码。可用具体动作：

1. 任务模板被刷新。
2. 天命值被削减或冻结。
3. 宿主档案出现“偏移”标记。
4. 失败宿主案例被调出。
5. 奖励来源被追到账目。
6. 赵长生不可评级记录触发后台报警。
7. 季玄屏用司命玄屏阻断反窥探。
8. 钟无咎用主簿权限封存任务源头。
9. 谢未明追出一条命数污染路径。
10. 沈临风主动放弃系统保护，换取一次自主选择。

## 17. 写作禁忌

1. 不能把新司命台写成游戏后台公司。
2. 不能把季玄屏写成系统客服或精灵。
3. 不能让钟无咎抢季玄屏或裴玄阙的功能。
4. 不能让沈临风砸掉系统界面就解决司命台问题。
5. 不能让系统奖励凭空无限生成高阶资源。
6. 不能让新司命台全知全能，尤其不能完整读取赵长生。
7. 不能把命数记录本身写成罪，罪在于安排命运和回收变量。
8. 不能让谢未明推演直接剧透全部后台。
9. 不能让沈临风拒绝系统后毫无代价。

## 18. 使用提示

新司命台职位体系要围绕三层冲突写：

1. 系统层：沈临风以为任务、奖励、天命值是外挂。
2. 职位层：季玄屏等管理员用模板、评级和结算安排宿主。
3. 黑账层：钟无咎等主簿封存任务源头和宿主回收证据。

沈临风终局不是拿到更高级系统权限，而是重定规则：

> 司命盘可以记录危机，但不能替众生安排命运。
