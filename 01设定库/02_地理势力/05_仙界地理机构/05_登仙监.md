# 登仙监

> 仙界机构补档。本文用于承载飞升者登记、登仙台残路接收、洗尘池、验源廊、仙籍分流、验源仙使和登仙监官职。本文不展开补墟黑账和新司命台中枢。

---

## 1. 设计目的

登仙监补档要解决六个问题：

1. 第九卷登仙台残路如何自然接入第十卷仙界底层。
2. 飞升者为什么刚到仙界不是自由仙人，而是待登记、待验源、待分流对象。
3. 洗尘池、验源廊、仙籍署、分流台之间的流程如何运转。
4. 纪晚凝为什么能在第十卷以制度执行者身份接触主角组。
5. 沈临风系统后台、赵长生命数断页、白倩霜血源、飞升者群像如何在登仙监流程中被标记。
6. 登仙监如何接补墟司外账房、新司命台分殿、镇界军征调营和仙矿府。

登仙监的核心味道：

> 飞升者仰望一生的仙门，在仙界这里只是一个窗口、一池洗尘水、一条验源廊和一张分流名单。

## 2. 基础定位

| 项目 | 内容 |
|------|------|
| 机构名 | 登仙监 |
| 表层职责 | 接引飞升者、登记身份、洗去下界尘缘、分配仙籍 |
| 实际功能 | 接管飞升者命数、道途、本源、税役和分流价值 |
| 活跃卷段 | 第九卷残路伏笔，第十卷核心，第十一卷接补墟司和新司命台 |
| 上级接口 | 补天司名义、中天玄阙法旨、补墟司外账房 |
| 下游接口 | 仙籍署、仙矿府、镇界军、新司命台分殿、补墟司外账房 |

登仙监不是终局最高反派机构。它是仙界制度入口，负责把飞升者放进后续账本。

## 3. 总流程

新仙界飞升者流程：

```text
登仙台残路落点
→ 飞升台外营
→ 临时编号
→ 洗尘池
→ 验源廊
→ 仙籍署
→ 分流台
→ 仙矿府 / 镇界军 / 仙府 / 新司命台 / 补墟司复核 / 黑籍风险
```

| 流程 | 表层说法 | 实际作用 |
|------|----------|----------|
| 落点拘束 | 防止飞升者刚入界失控 | 控制人数、记录异常波纹 |
| 临时编号 | 便于排队和救治 | 建立临籍和追踪编号 |
| 洗尘池 | 洗去下界尘缘 | 检测、剥离、标记和暂扣 |
| 验源廊 | 评估修行资质 | 筛出灵、宝、魂、血、运、法、界、时、道价值 |
| 仙籍署 | 录入仙籍 | 接管身份、税役、命数接口 |
| 分流台 | 按能力安排去处 | 给仙矿、镇界军、司命台、补墟司分配对象 |

## 4. 机构空间

```text
登仙台残路落点
    ↓
飞升台外营
    ↓
临时编号棚 —— 低阶医治处
    ↓
洗尘池
    ↓
验源廊
    ↓
仙籍署
    ↓
分流台
    ↓
仙矿府 / 镇界军征调营 / 新司命台分殿 / 补墟司外账房
```

| 空间 | 功能 | 剧情用途 |
|------|------|----------|
| 残路落点 | 接收正常飞升者、偏落点、残路异常 | 主角组非荣耀入界 |
| 飞升台外营 | 等候、编号、临时拘束 | 飞升者群像第一印象 |
| 临时编号棚 | 发临籍号、记录出身世界 | 仙界第一道身份接管 |
| 低阶医治处 | 处理飞升伤势 | 证明仙界有秩序，但救治也入账 |
| 洗尘池 | 清洗、检测、标记 | 赵长生、沈临风、白倩霜异常风险 |
| 验源廊 | 九源初筛 | 接补墟司、司命台、血账、界账 |
| 仙籍署 | 录临籍、役籍、军籍、罪籍、黑籍追索 | 许守仓等低阶仙籍官 |
| 分流台 | 分配仙矿、镇界军、司命台、补墟复核 | 第十卷制度冲突核心 |

## 5. 官职体系

| 官职 | 职责 | 剧情功能 |
|------|------|----------|
| 登仙监正 | 管理飞升台、验源和分流总流程 | 不必早登场，可作为上级压力 |
| 验源仙使 | 主持九源检测、判断异常 | 第九卷可用投影，第十卷可用真人或下属 |
| 洗尘使 | 管洗尘池、标记和暂扣 | 检测赵长生和沈临风异常 |
| 仙籍录事 | 录临籍、役籍、军籍、罪籍 | 许守仓类型低阶执行者 |
| 分流判官 | 将飞升者分流到各机构 | 让飞升者价值定价具体化 |
| 暂扣库吏 | 暂扣法宝、功法、特殊血样 | 接宝账、法账、血账 |
| 巡监仙兵 | 看守外营、追捕逃漏 | 不是镇界军主力，但能制造底层压力 |
| 异常复核官 | 处理命数断页、系统痕迹、血源异常 | 纪晚凝可在此登场或接手复核 |

登仙监低层执行者不必都坏。许守仓这类人怕失职、怕指标、怕上级，不一定知道黑账。

## 6. 洗尘池

洗尘池表面是洗去下界尘缘。

实际检测：

| 项目 | 表层说法 | 后续去向 |
|------|----------|----------|
| 下界浊气 | 适应仙界灵机 | 出身世界和界源痕迹 |
| 法宝灵性 | 防危险器物入界 | 万宝库、宝账、暂扣库 |
| 血脉 | 记录体质 | 血税、补墟血账、仙族研究 |
| 神魂 | 稳定飞升者 | 魂账、轮回异常、司命接口 |
| 气运 | 判断仙途 | 新司命台、系统宿主评级 |
| 功法 | 登记道统 | 法税、万法碑封锁 |
| 洞天或界源 | 防止私带危险小世界 | 界账、补墟司复核 |

主角组风险：

- 赵长生命数断页导致命数校验不连续。
- 归墟炉波纹可能被洗尘水激发。
- 沈临风系统接口可能被新司命台分殿感应。
- 白倩霜血源链路可能触发血源口。
- 陆维舟界阵痕迹可能触发界源口。

## 7. 验源廊

验源廊是登仙监和补墟司、新司命台、镇界军连接最紧的空间。

| 验源口 | 检测内容 | 常规去向 | 风险 |
|--------|----------|----------|------|
| 灵源口 | 灵根、法力、灵脉痕 | 普通仙籍、仙矿、仙府 | 出身世界被标记 |
| 宝源口 | 本命法宝、古宝、器纹 | 暂扣库、万宝库 | 归墟炉波纹风险 |
| 魂源口 | 神魂、残魂、轮回痕 | 魂账复核 | 飞升失败者证据被压 |
| 血源口 | 血脉、真血、族源 | 血税、补墟司外账 | 白倩霜风险 |
| 运源口 | 气运、系统、天命加权 | 新司命台分殿 | 沈临风风险 |
| 法源口 | 功法、道统、秘术 | 法税、万法碑封锁 | 道统被接管 |
| 界源口 | 洞天、小世界、出身界源 | 界税、抽源名单 | 陆维舟和赵长生风险 |
| 时源口 | 时痕、遗迹接触 | 高危封存 | 前中期少用 |
| 道源口 | 道果、道痕、特殊道途 | 高层监控 | 后期使用 |

第十卷只需展示初筛，不要解释完整黑账。

## 8. 分流台

分流台决定飞升者进入哪条制度路径。

| 去向 | 对象 | 表层理由 | 实际压力 |
|------|------|----------|----------|
| 仙矿府 | 低价值、欠税、无靠山飞升者 | 役期抵税 | 周栖石线 |
| 镇界军 | 战力强、斗法经验足者 | 战功换籍 | 钟离砺线 |
| 仙府下院 | 血脉、法宝、功法有价值者 | 培养 | 监控和利用 |
| 新司命台分殿 | 气运强、系统、命数异常者 | 天命扶持 | 系统回收 |
| 补墟司外账房 | 本源异常、界源异常、验源差额 | 复核 | 纪晚凝线 |
| 黑籍风险 | 逃漏、伪籍、录不进者 | 非法 | 梅见微线和赵长生躲账 |

飞升者以为分流是安排，实际是定价。

## 9. 与核心人物关系

| 人物 | 登仙监接触点 | 作用 |
|------|--------------|------|
| 赵长生 | 命数校验、宝源口、界源口 | 命数断页和归墟炉异常 |
| 沈临风 | 运源口、新司命台分殿 | 系统后台初露 |
| 白倩霜 | 血源口、补墟司外账 | 血账前置 |
| 纪晚凝 | 异常复核、分流复核、外账房 | 制度执行者登场 |
| 周栖石 | 分流至仙矿府 | 普通飞升者被役籍压住 |
| 梅见微 | 黑籍风险或残路偏落点 | 失籍者生存线 |
| 钟离砺 | 分流至镇界军 | 强者也不自由 |
| 许守仓 | 仙籍署窗口 | 低层执行者具体化 |
| 陶铸 | 残仙器坊后续 | 飞升者被暂扣法宝和残器捡漏接口 |

## 10. 卷际使用

| 卷 | 用法 |
|----|------|
| 第九卷 | 登仙台残路出现验源投影，证明飞升是筛选前哨 |
| 第十卷 | 登仙监成为仙界底层第一主舞台，主角组进入登记、洗尘、验源、分流体系 |
| 第十一卷 | 登仙监记录被补墟司和新司命台反查，沈临风系统后台暴露 |
| 第十三卷 | 登仙监旧记录证明飞升回收不是例外，而是长期制度 |
| 第十四卷 | 新登仙台和飞升规则重定，登仙监旧职能被清算和改写 |

## 12. 写作禁忌

1. 不能把登仙监写成纯粹坏人衙门。
2. 不能让飞升者一到仙界就自由行动。
3. 不能让洗尘池一次性剥夺所有反抗能力。
4. 不能让纪晚凝一出场就知道黑账。
5. 不能让验源廊完整解释九种本源黑账。
6. 不能让赵长生公开归墟炉。
7. 不能让沈临风立刻破解新司命台后台。
8. 不能让黑籍坊市成为万能避难所。

## 13. 使用提示

写登仙监时，每次出场都要带出流程压迫：

1. 编号。
2. 排队。
3. 洗尘。
4. 验源。
5. 录籍。
6. 分流。
7. 欠税或役期。

它的存在是为了证明：

> 仙界不是飞升者的终点，而是他们被重新定价的开始。
