# 旧天庭废墟地图

> 仙界终局前置地图补档。本文用于承载第十二卷旧天庭废墟、旧职残迹、九器旧位、旧账对照、复辟派行动区和主角组拒绝复辟旧天庭的空间依据。本文不是旧天庭总史，也不替代秦照古、洛闻霜、穆天衡、温隙等人物单档。

---

## 1. 设计目的

旧天庭废墟地图要解决七个问题：

1. 第十二卷不能只写“进入一片古老遗迹”，而要让旧天庭的职责、荣耀、失败和权柄诱惑都有具体场所。
2. 秦照古为什么能提出复辟旧天庭，而且这个主张为什么有可敬的一面。
3. 洛闻霜为什么不归复辟派管，她的轮回灯线为什么必须有独立空间。
4. 穆天衡和旧军序为什么能制造武力压迫，不只是秦照古身边的打手。
5. 温隙为什么能提供旧籍、旧账和时痕证词，但不能变成全知资料老人。
6. 旧天庭九器残迹如何在空间上呈现“公共职责”和“集中权柄风险”两面。
7. 主角组为什么必须拒绝原样复辟旧天庭，而不是因为旧天庭不值得尊重。

旧天庭废墟的核心味道：

> 这里每一处残殿都证明旧天庭曾经救过万界，也每一处断阵都证明万界不能再把全部命运交给高处。

## 2. 基础定位

| 项目 | 内容 |
|------|------|
| 名称 | 旧天庭废墟 |
| 活跃卷段 | 第十二卷核心空间，第十三卷作为旧账和旧器证据回声 |
| 空间性质 | 旧天庭崩塌后残留的中枢遗址、旧职空间和九器旧位 |
| 表层意义 | 上古天庭遗迹，旧道职、旧器、旧账和旧阵残存地 |
| 深层意义 | 证明旧天庭曾承担公共职责，也证明集中权柄曾被篡改并导致循环断裂 |
| 主要人物 | 秦照古、洛闻霜、穆天衡、温隙 |
| 主线功能 | 旧天庭复辟诱惑、旧账对照、九器旧义、赵长生第三条路前置 |

旧天庭废墟不是纯粹宝藏副本。

它的每个高价值遗迹都必须同时带着两层含义：

1. 旧天庭曾经维护过飞升、轮回、镇界、补天、司命、道统和归墟循环。
2. 旧天庭将这些职责集中在少数旧职和九器体系内，一旦保全派篡改核心阵枢，万界无力及时参与纠偏。

## 3. 进入路径

旧天庭废墟不能像普通秘境一样随便进入。

合理入口有四条：

| 入口 | 开启条件 | 叙事用途 |
|------|----------|----------|
| 登仙台旧裂 | 登仙台残路、飞升验源异常、旧接引符共振 | 连接第九卷和第十卷飞升真相 |
| 旧籍缝门 | 温隙的旧籍残权、旧账片段、遗址经验 | 让温隙成为入口证人但非全知者 |
| 司命残火引路 | 沈临风系统底层旧残火异常 | 将新司命台线接回旧司命台旧义 |
| 归墟炉波纹 | 归墟炉残片靠近旧炉位时产生同源波纹 | 让赵长生接触旧职责，但不被旧天庭预设为继承人 |

进入旧墟必须有代价：

1. 旧籍会被新仙界追索。
2. 归墟炉波纹会被中天玄阙察觉。
3. 司命系统会出现任务冲突和校验错误。
4. 旧职禁制会误判新仙籍、黑籍和命数断页者。
5. 旧墟内时痕不稳定，不能当成安全藏身地。

## 4. 总体空间

旧天庭废墟按旧天庭职责从外到内分布。

```text
遗址外环：旧籍荒廊 / 天门碎阶 / 接引残桥
        ↓
旧职环：镇界残城 —— 旧司命断台 —— 轮回灯坞
        ↓
道统环：万法碑林 —— 斩道剑痕 —— 补天尺廊
        ↓
中枢环：登仙残台 —— 太虚舟坞 —— 归墟炉裂坑
        ↓
九器中庭 / 旧议庭 / 崩塌阵枢
```

| 区域 | 旧职责 | 当前状态 | 主要人物 |
|------|--------|----------|----------|
| 旧籍荒廊 | 登记旧道职、旧账、旧籍身份 | 残册破损，新仙籍无法正常通行 | 温隙 |
| 天门碎阶 | 飞升者入天庭前的接引阶 | 台阶断裂，仍残留旧接引规则 | 温隙、秦照古 |
| 镇界残城 | 巡查界壁、地脉、界源稳定 | 城墙断裂，旧镇界印残权仍能启动 | 秦照古、穆天衡 |
| 旧司命断台 | 记录命数裂痕、提示补天危机 | 司命盘旧位崩碎，残火与系统互相干扰 | 沈临风、谢未明 |
| 轮回灯坞 | 接引残魂、维持轮回牵引 | 灯焰微弱，魂账断点明显 | 洛闻霜 |
| 万法碑林 | 保存万界道统 | 大量碑面被封、裂、刮改 | 顾长庚、温隙 |
| 斩道剑痕 | 斩断腐朽大道 | 剑痕仍在，但可被旧军序误用为正统压制 | 顾长庚、穆天衡 |
| 补天尺廊 | 测量世界裂缝并制定修补尺度 | 尺痕错乱，可对照新补墟账 | 纪晚凝、陆维舟 |
| 登仙残台 | 正常接引飞升者进入更大循环 | 阵枢被改写过，能证明飞升通道变质 | 赵长生、沈临风 |
| 太虚舟坞 | 横渡太虚、接引新道和外部变量 | 航渠干涸，残留时痕旧录 | 温隙、沈临风 |
| 归墟炉裂坑 | 回收残败本源、重炼新生 | 旧炉位崩塌，赵长生归墟炉发生共鸣 | 赵长生 |
| 九器中庭 | 九器旧位和职责汇合处 | 旧中枢破碎，复辟派最想重启此地 | 秦照古、赵长生 |

## 5. 外环：旧籍荒廊与天门碎阶

旧籍荒廊是旧天庭废墟的入口层。

表面形态：

1. 倒塌的玉阶。
2. 断裂的旧籍牌。
3. 被新仙界刮改过的道职名录。
4. 只剩半边的接引门。
5. 旧账残页在风里像灰一样飘。

温隙在这里有价值：

1. 他能认出旧籍格式。
2. 他知道哪些旧籍门槛会排斥新仙籍。
3. 他能指出新仙界刮改旧职的痕迹。
4. 他也会隐瞒部分旧账，用来交换安全。

旧籍荒廊的作用不是证明旧天庭完美，而是提供对照：

| 对照项 | 旧籍荒廊显示 | 后续用途 |
|--------|--------------|----------|
| 飞升者身份 | 旧飞升者被接入职责和循环位置 | 对照登仙监的登记、验源和分流 |
| 道职 | 道职本为公共职责接口 | 对照新仙界把道职变成征收权柄 |
| 账本 | 旧账记录修补、调度和损耗 | 对照补墟司明账、内账和黑账 |
| 司命 | 旧司命记录危机 | 对照新司命台安排命运和回收变量 |

天门碎阶适合做第十二卷初入旧墟的第一场压力：

1. 新仙籍者被判为“非旧职”。
2. 黑籍者触发追索铃。
3. 赵长生命数断页让禁制无法判定。
4. 沈临风系统弹出错误任务或旧残火提示。
5. 秦照古第一次以旧籍身份打开残阶，展示旧天庭合法性诱惑。

## 6. 镇界残城

镇界残城是秦照古和穆天衡的重要行动区。

旧职责：

1. 稳定界壁。
2. 镇压界源乱流。
3. 护送补天队进入破损世界。
4. 在归墟污染外溢时建立隔离阵。
5. 保护飞升台和下层世界接引通道。

当前状态：

1. 城墙裂缝中仍有旧镇界印残光。
2. 旧军营还保存部分战录。
3. 界裂沙盘残缺，但能显示旧天庭曾修补过哪些世界。
4. 一些镇界阵眼被新仙界改写过，成为后世锁界和抽源阵的前身。

秦照古在这里展现旧职正当性：

> 旧镇界不是锁死万界，而是稳住万界。

穆天衡在这里展现复辟危险：

> 万界若要活，就必须重新服从旧军序。

镇界残城要同时给出两类证据：

| 证据 | 证明 |
|------|------|
| 旧镇界战录 | 旧天庭曾真实守过界裂和残界 |
| 旧军征调令 | 旧天庭也曾自上而下强制调度各界力量 |
| 镇界残印拓片 | 镇界职责原本不是抽源压迫 |
| 阵枢改写痕 | 保全派曾借镇界印和登仙台切断部分下界 |

这一层是赵长生拒绝复辟的重要依据：

1. 旧职责值得保留。
2. 旧军序不能复原为万界最高命令。
3. 镇界权柄必须从“高处命令”改为“万界节点协作”。

## 7. 旧司命断台

旧司命断台是沈临风和谢未明的关键空间。

旧司命台原本职责：

1. 记录命数裂痕。
2. 标记大劫前兆。
3. 提醒可能补天之人。
4. 不直接安排众生命运。
5. 不把变量培养成可回收资源。

当前状态：

1. 司命盘旧位崩碎。
2. 残火在断台下方时明时灭。
3. 新司命台系统接近这里时会出现任务冲突。
4. 命数线在这里不是金色机缘，而是大量断线、乱线和被改写的痕迹。

旧司命断台不能让沈临风直接“升级系统”。

它能提供的是：

| 内容 | 用途 |
|------|------|
| 司命残录 | 证明旧司命台旧义是记录危机 |
| 任务源头裂痕 | 证明新司命台在旧残火上覆盖任务模板 |
| 失败宿主影迹 | 证明系统不是单独服务沈临风 |
| 赵长生不可评级回声 | 证明命数断页不是天命加持，而是司命系统读不到 |

沈临风在这里的关键选择不是拿到更强外挂，而是第一次真正明白：

> 司命可以提醒危机，但不能替众生安排选择。

## 8. 轮回灯坞

轮回灯坞是洛闻霜的独立空间。

它不属于秦照古的复辟派直接控制。

旧职责：

1. 接引残魂。
2. 净化怨秽。
3. 辨别残魂、魂影、真魂。
4. 维持轮回牵引。
5. 防止亡魂被当作资源或权力工具。

当前状态：

1. 灯坞大半沉入魂雾。
2. 轮回灯残焰只剩微光。
3. 魂账旧牌上有大量断点。
4. 一部分魂灯灰被新仙界补墟魂账截走。
5. 旧灯规仍能阻止强行拘魂。

轮回灯坞要让读者看见两件事：

1. 旧天庭确实曾把亡魂归处当成公共职责。
2. 即使旧天庭复辟，也不能让任何权力私有轮回灯。

洛闻霜在这里可以阻止三种错误：

| 错误 | 她的边界 |
|------|----------|
| 赵长生想用归墟炉强聚熟人魂影 | 魂影不是本人，不能强行当作复活材料 |
| 秦照古想让轮回灯归旧天庭复辟中枢 | 轮回边界高于旧职复原 |
| 新仙界想把魂灯灰写入补墟魂账 | 亡魂不是补墟可消耗项 |

轮回灯坞提供后续魂账证据：

1. 魂灯灰签。
2. 亡魂断点。
3. 轮回牵引被截留的灯痕。
4. 洛闻霜可与纪晚凝后期对照补墟司魂账。

## 9. 万法碑林与斩道剑痕

万法碑林和斩道剑痕主要服务顾长庚、道统保存者和旧正统诱惑。

万法碑林旧职责：

1. 保存万界道统。
2. 防止功法断绝。
3. 将危险道统封存而非彻底抹杀。
4. 在大劫后保留各界重建修行路的可能。

当前问题：

1. 部分碑面被旧天庭自己封住。
2. 部分碑面被新仙界刮改。
3. 部分道统被玄阙仙族后续垄断。
4. “保存”与“封锁”的边界在这里变得模糊。

斩道剑痕旧职责：

1. 斩断腐朽大道。
2. 终止已经伤害万界循环的高位道统。
3. 防止旧规则僵化成永恒压迫。

当前危险：

1. 穆天衡会把斩道剑痕解释为旧军序的正统权柄。
2. 新仙界会把斩道名义变成镇压异端。
3. 顾长庚必须明白斩道不是谁站得高谁就有资格斩。

这一层提供的核心命题：

> 正统若不能接受清算，也会变成要被斩断的腐朽大道。

可用证据：

| 证据 | 后续用途 |
|------|----------|
| 万法碑残拓 | 证明旧天庭保存过开放道统 |
| 封碑令残片 | 证明旧天庭也曾以保护名义封存道统 |
| 斩道剑痕回声 | 推动顾长庚区分正统职责和正统压迫 |
| 道职改写记录 | 连接温隙旧籍证词和新仙界道职滥权 |

## 10. 补天尺廊

补天尺廊是旧天庭修补世界的技术空间。

旧职责：

1. 测量世界裂缝。
2. 判断界壁损伤。
3. 制定补天尺度。
4. 调度补天资源。
5. 记录修补后的世界恢复情况。

当前状态：

1. 尺痕断裂。
2. 一部分旧补天数据仍可读取。
3. 部分刻度被保全派改写，后来成为新仙界补墟折源公式的来源之一。
4. 陆维舟能从阵纹上看出“修界”和“抽界”的技术差异。
5. 纪晚凝后期能用这里的旧账对照补墟司内账。

补天尺廊要证明：

| 旧补天 | 新补墟 |
|--------|--------|
| 先判断世界伤口 | 先判断可抽额度 |
| 目标是修复裂缝 | 目标是填补墟眼 |
| 记录修补责任 | 记录征收和投入 |
| 世界仍是循环主体 | 世界被降格成资源来源 |

这里可以放置第十二卷重要转折：

1. 纪晚凝看到旧补天账格式，发现补墟司所谓“祖制”并非原样继承。
2. 陆维舟证明旧阵的方向是让万界节点恢复协作，而不是把界源单向抽往仙界核心。
3. 赵长生意识到归墟炉终局不能只是“堵墟眼”，而要重炼整个循环口。

## 11. 登仙残台与太虚舟坞

登仙残台是飞升真相的旧对照。

旧登仙台职责：

1. 接引完成阶段修行者。
2. 将飞升者纳入更大循环。
3. 帮助飞升者接触更高层道统、职责和世界协作。
4. 不以飞升者为本源回收对象。

当前状态：

1. 接引阵断裂。
2. 保全派改写痕迹清晰。
3. 一部分阵路通往新仙界登仙监体系。
4. 飞升不再是荣耀，而被改成验源、登记、分流和回收。

太虚舟坞是外部变量和时痕旧录空间。

它不能让主角组随意穿越时间。

可用功能：

1. 保存少量太虚航迹。
2. 证明旧天庭曾接引新道和外部可能。
3. 解释沈临风异世魂火不是普通夺舍。
4. 提供账本被改写前的时痕残影。
5. 为终局“万界参与新循环”提供太虚方向的概念伏笔。

温隙在太虚舟坞能提供线索，但不能掌握时本源核心：

| 能做 | 不能做 |
|------|--------|
| 认出旧航渠格式 | 倒转时间 |
| 指认某段账本改写前的残影 | 改变第一次归墟大潮 |
| 帮主角组通过一段旧籍航门 | 直接抵达中天玄阙核心 |
| 说明旧天庭也曾接纳外部变量 | 解释全部穿越原因 |

## 12. 归墟炉裂坑与九器中庭

归墟炉裂坑是赵长生最危险的旧职共鸣空间。

它必须明确三点：

1. 赵长生能靠归墟炉与旧炉位共鸣。
2. 这不证明他是旧天庭预设继承人。
3. 归墟炉旧职责要被他重新理解，而不是归还旧天庭中枢。

归墟炉裂坑可呈现：

1. 旧炉壁残纹。
2. 归墟残灰凝成的黑色砂层。
3. 旧炉心崩碎后留下的空洞。
4. 九器中庭向炉位汇聚的断裂阵线。
5. 旧天庭最后试图重炼大潮却失败的痕迹。

九器中庭是旧天庭废墟最核心的复辟诱惑。

旧九器旧位：

| 九器 | 废墟旧位表现 | 第十二卷作用 |
|------|--------------|--------------|
| 归墟炉 | 裂坑、炉灰、残纹 | 赵长生面对旧职责 |
| 司命盘 | 断台、残火、命数裂录 | 沈临风面对系统真相 |
| 登仙台 | 断裂接引阵 | 对照飞升收割 |
| 补天尺 | 尺廊、界裂刻度 | 对照补墟抽源 |
| 万法碑 | 碑林、刮改碑面 | 对照道统垄断 |
| 轮回灯 | 灯坞、魂账断点 | 对照亡魂黑账 |
| 镇界印 | 残城、旧印拓片 | 对照镇界军锁界 |
| 太虚舟 | 舟坞、时痕航渠 | 对照外部变量和新道 |
| 斩道剑 | 剑痕、断令 | 对照正统和腐朽大道 |

秦照古在九器中庭提出的主张：

> 九器归旧位，旧职复原，天庭重立，万界才能重新有秩序。

赵长生必须在这里形成反驳：

> 九器职责要留下，但旧天庭垄断九器的方式不能复原。万界循环不能再由一座天庭替众生安排。

## 13. 旧议庭与崩塌阵枢

旧议庭是旧天庭失败证据最集中的地方。

这里不适合只写成宏伟大殿。

它应当有三类残留：

| 残留 | 证明 |
|------|------|
| 循环派议令 | 旧天庭确实曾尝试救更多世界 |
| 保全派改写令 | 保全派确实改写镇界印和登仙台阵枢 |
| 中枢封闭记录 | 旧天庭权柄过度集中，普通世界无法参与纠错 |

崩塌阵枢要承担第十二卷最重要的灰度：

1. 保全派背叛不是旧天庭所有人的罪。
2. 但旧天庭把关键权柄集中在中枢，使少数人一次改写就能切断多界。
3. 旧天庭第一次大潮失败，不只是外敌太强，也有制度承载过限。
4. 复辟旧议庭意味着重新制造一个能替万界决定命运的高处。

可用场景：

1. 秦照古为循环派旧令辩护。
2. 穆天衡认为当年如果军序更强，保全派就不能得手。
3. 温隙拿出被刮掉一半的中枢封闭记录。
4. 洛闻霜指出轮回灯不应被任何议庭调度为复辟资源。
5. 赵长生最终拒绝坐上旧议庭给出的“归墟炉旧主位”。

| 卷 | 用法 |

第十二卷的结尾不应是“主角组获得旧天庭认可”。

更合适的落点：

1. 主角组带走旧账和旧义证据。
2. 秦照古承认赵长生有资格重炼循环，但仍不认同他拒绝复辟。
3. 洛闻霜明确轮回灯不会交给任何中枢私有。
4. 穆天衡或旧军残部继续保持压力。
5. 新仙界察觉旧墟被打开，中天玄阙终局压力前置。

## 17. 退场与回收

旧天庭废墟不能在第十二卷被彻底修好。

合理回收：

1. 部分旧职空间被重新点亮，但只作为证据和职责回声。
2. 旧议庭不复原。
3. 九器旧位不重新归旧天庭掌控。
4. 秦照古、穆天衡等复辟派仍可保留旧墟守望者位置。
5. 温隙等旧籍证人进入清账流程，但不能掌握新制度。
6. 洛闻霜将轮回灯残焰带入终局生死边界，而非旧天庭复辟。

终局方向：

> 旧天庭废墟被保留为旧职责和旧失败的见证地，不再成为万界最高权力中心。

## 18. 写作禁忌

1. 不能把旧天庭废墟写成普通刷宝秘境。
2. 不能让旧天庭显得完美无错。
3. 不能让秦照古的复辟方案成为正确答案。
4. 不能让洛闻霜归秦照古或穆天衡调度。
5. 不能让温隙变成全知资料库。
6. 不能让穆天衡只是秦照古打手。
7. 不能让赵长生因为归墟炉共鸣就天然继承旧天庭。
8. 不能让沈临风在旧司命断台直接获得万能系统权限。
9. 不能提前把第十三卷中天玄阙和墟眼禁域的黑账一次性揭完。
10. 不能让旧天庭所有失败都推给保全派个人背叛，旧制度承载和权柄集中也必须承担后果。

## 19. 使用提示

写旧天庭废墟时抓四层：

1. 旧职层：飞升、司命、轮回、镇界、补天、道统曾经有公共职责。
2. 旧器层：九器不是宝物收藏，而是万界循环九条规则的旧载体。
3. 旧账层：旧账证明新仙界篡改了旧职责，但也证明旧制度有集中权柄问题。
4. 复辟层：秦照古和穆天衡的可敬之处越足，赵长生拒绝旧天庭复辟才越有重量。

旧天庭废墟最终要证明：

> 赵长生不是忘恩负义地抛弃旧天庭，而是在旧天庭牺牲和失败的废墟上，拒绝让万界再次把命运交给同一座高处。
