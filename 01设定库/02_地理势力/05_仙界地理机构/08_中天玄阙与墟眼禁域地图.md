# 中天玄阙与墟眼禁域地图

> 仙界终局空间补档。本文用于承载第十三卷中天玄阙、玄阙天宫、补天内府、补墟黑账汇合、镇界封锁、新司命台命数黑账和墟眼禁域。本文不替代 `03_补墟司内部科室.md`、`04_新司命台职位体系.md` 或裴玄阙个人单档。

---

## 1. 设计目的

中天玄阙与墟眼禁域地图要解决八个问题：

1. 第十三卷终局前压力不能只靠“最高反派在宫殿里”，必须有权力中枢、账本中枢、军政封锁和真实灾厄空间。
2. 裴玄阙为什么能代表保全派最高压迫，而不是单个高境界恶人。
3. 墟眼为什么是真实危机，不是新仙界编出来的谎言。
4. 新仙界错误为什么不是“补墟本身曾经毫无必要”，而是把临时自救制度化、永久化、特权化。
5. 血账、魂账、界账、运账、旧账和司命黑账如何在终局空间汇合。
6. 镇界军、补墟司、新司命台、登仙监和玄阙仙族如何在空间上形成合围。
7. 主角组为什么不能靠一次潜入偷账或一次单挑杀掉裴玄阙解决问题。
8. 赵长生如何从“堵住墟眼”推进到“把墟眼重炼成循环炉口”。

中天玄阙的核心味道：

> 这里确实在漏天，也确实有人借补天之名，把万界一层层写进燃料账。

## 2. 基础定位

| 项目 | 内容 |
|------|------|
| 名称 | 中天玄阙 |
| 对外名义 | 仙界最高圣地、天阙所在、补天中枢 |
| 实际性质 | 新仙界权力中枢、墟眼封禁大阵核心、补墟体系总接口 |
| 核心禁域 | 墟眼禁域 |
| 活跃卷段 | 第十三卷核心，第十四卷重炼循环前最终空间 |
| 代表人物 | 裴玄阙、厉沉霄、藏烬君、钟无咎、纪晚凝、沈临风、赵长生 |
| 主线功能 | 保全派道路正面压迫、黑账汇合、真实危机显形、重炼循环前置 |

中天玄阙不能写成单纯反派老巢。

它必须同时成立两件事：

1. 没有封禁大阵，墟眼确实会继续吞噬仙界核心并外溢到万界循环。
2. 封禁大阵长期依赖收割万界、本源黑账和权贵豁免，已经不再真正修复危机，只是在延缓旧权力的崩塌。

## 3. 总体空间

中天玄阙按“公开权力到真实伤口”的方向分层。

```text
中天外阙：玄阙云门 / 仙籍追索关 / 镇界卫戍环
        ↓
玄阙天宫：议令殿 / 保全法旨库 / 玄阙仙族内苑
        ↓
补天内府：补天明殿 / 万源转输廊 / 补墟总账枢
        ↓
黑账汇合层：血账门 / 魂账门 / 界账门 / 运账门 / 旧账校验门
        ↓
封禁大阵层：九源阵环 / 镇界军令台 / 司命黑账接口
        ↓
墟眼禁域：观测台 / 归墟伤边缘 / 墟潮回声区 / 核心伤口
```

| 层级 | 表象 | 真相 | 主要人物 |
|------|------|------|----------|
| 中天外阙 | 仙界最高禁城外围 | 仙籍、军籍、黑籍追索和镇界封锁入口 | 厉沉霄、镇界军 |
| 玄阙天宫 | 仙界权力中心 | 保全派发布法旨、维护合法性 | 裴玄阙、玄阙高层 |
| 补天内府 | 对外补天机构 | 分拣、调拨、转输万界本源 | 补天司、补墟司 |
| 黑账汇合层 | 绝密账库接口 | 血、魂、界、运、旧账证据汇合 | 藏烬君、纪晚凝 |
| 封禁大阵层 | 镇压墟眼的大阵 | 九源投入、军令封锁、命数遮蔽 | 裴玄阙、钟无咎、厉沉霄 |
| 墟眼禁域 | 归墟伤口 | 仙界核心被打穿后的道则伤口 | 赵长生、裴玄阙 |

## 4. 中天外阙

中天外阙是第十三卷的高压入口。

组成：

1. 玄阙云门。
2. 仙籍追索关。
3. 黑籍照影台。
4. 镇界卫戍环。
5. 补墟物资转运关。
6. 玄阙法旨传令廊。

功能：

| 空间 | 表层用途 | 实际用途 |
|------|----------|----------|
| 玄阙云门 | 朝见中天玄阙 | 按仙籍、道职、军籍、旧籍筛人 |
| 仙籍追索关 | 防止叛籍入阙 | 捕捉黑籍、旧籍、命数异常者 |
| 黑籍照影台 | 识别伪籍 | 赵长生、沈临风等异常目标压力点 |
| 镇界卫戍环 | 守卫玄阙 | 封锁证据、拦截旧天庭残部和清账者 |
| 物资转运关 | 转运补天资源 | 血源、魂源、界源等补墟物资进入中枢 |
| 法旨传令廊 | 传达玄阙命令 | 让登仙监、补墟司、新司命台、镇界军协同 |

中天外阙要写出一种制度压迫：

> 你不是打进一座宫门，而是先被身份、账本、军令和命数一起审问。

厉沉霄适合在这里调动镇界军。

他能做：

1. 封锁外阙。
2. 以军令压过普通账官质疑。
3. 阻断旧天庭废墟证人进入玄阙。
4. 护送补墟物资进入转输关。
5. 镇压黑籍飞升者和失控宿主。

他不能做：

1. 代表完整墟眼真相。
2. 查看所有补墟黑账。
3. 替裴玄阙承担终局理念罪责。

## 5. 玄阙天宫

玄阙天宫是裴玄阙的权力中心。

组成：

1. 议令殿。
2. 保全法旨库。
3. 玄阙仙族内苑。
4. 道职授印台。
5. 万界贡籍壁。
6. 中天观墟天窗。

玄阙天宫负责制造合法性。

| 空间 | 功能 | 叙事作用 |
|------|------|----------|
| 议令殿 | 发布保全派决策 | 裴玄阙公开提出牺牲万界保仙界核心 |
| 保全法旨库 | 保存封账令、抽源令、镇界令 | 证明黑账不是执行层偶然贪墨 |
| 玄阙仙族内苑 | 高层仙族续命、豁免和资源分配 | 证明制度已被特权寄生 |
| 道职授印台 | 授予补墟、镇界、司命、登仙等权柄 | 顾长庚可斩不该存在的道职权力 |
| 万界贡籍壁 | 展示万界对仙界的供奉和补天贡献 | 实为万界本源流失的美化墙 |
| 观墟天窗 | 远观墟眼封禁状况 | 裴玄阙展示真实危机的空间 |

裴玄阙在玄阙天宫必须有真实论据：

1. 墟眼正在扩大。
2. 第二次归墟大潮逼近。
3. 旧天庭失败过。
4. 很多下界和中层世界确实没有独立抵御大潮的能力。
5. 封禁大阵一旦停摆，仙界核心会先崩，随后影响飞升、轮回、命数和界源秩序。

他的错误也必须清楚：

1. 他把临时牺牲变成永久收割。
2. 他把万界从循环参与者降成燃料。
3. 他封存黑账，拒绝公开旧账。
4. 他容忍玄阙权贵以补墟名义续命和豁免。
5. 他明知补墟效率下降，仍以更高抽源维持表面稳定。

## 6. 补天内府

补天内府是中天玄阙对外最体面的机构空间。

组成：

1. 补天明殿。
2. 万源转输廊。
3. 九源分拣池。
4. 补墟总账枢。
5. 外账回执墙。
6. 折源法阵井。

表层说法：

> 这里收纳万界资源，维持天阙不坠，保护万界免于大潮。

实际功能：

1. 接收登仙监、灵界监察链、抽源账站、仙矿府、镇界军和新司命台送来的资源与账。
2. 将血、魂、界、运、宝、法、灵、时、道九类本源分拣。
3. 生成投入封禁大阵的额度。
4. 将不适合公开的项目转入黑账汇合层。
5. 把权贵豁免和损耗差额平到下层账目。

补天内府要和补墟司内部科室形成关系：

```text
补墟司外账房 / 明账厅 / 内账廊
        ↓
封账阁 / 黑账深库
        ↓
补天内府补墟总账枢
        ↓
九源阵环
        ↓
墟眼封禁
```

纪晚凝进入补天内府时，不能靠情绪喊冤。

她必须带着：

1. 明账与内账差额。
2. 血账、魂账、界账的封存痕迹。
3. 旧天庭旧账对照。
4. 灵界抽源账站回执。
5. 镇界军护送军令。
6. 新司命台运账接口。

这样她面对裴玄阙时才不是“个人良心”，而是制度证据。

## 7. 黑账汇合层

黑账汇合层是第十三卷证据争夺的核心。

它不是一个单独书架，而是一组必须由不同证据线共同开启的门。

```text
血账门 —— 魂账门 —— 界账门
     \        |        /
      运账门 —— 旧账校验门
             ↓
        黑账汇合厅
```

| 门 | 需要的证据 | 对应人物线 |
|----|------------|------------|
| 血账门 | 猎血许可、血名册、血税名单、百族源血证言 | 白倩霜、明厌、灵界线 |
| 魂账门 | 魂灯灰签、轮回灯断点、渡劫残魂记录 | 洛闻霜、纪晚凝 |
| 界账门 | 界源刻度拓片、残界抽源图、旧补天账 | 陆维舟、叶疏彤、赵长生 |
| 运账门 | 系统任务源头、天命值折损、宿主回收名单 | 沈临风、谢未明 |
| 旧账校验门 | 旧天庭正常循环账、道职改写记录、保全派旧令 | 秦照古、温隙、纪晚凝 |

藏烬君在这里的作用：

1. 守住黑账门。
2. 合并血、魂、界、运等敏感项目。
3. 焚毁副录表层。
4. 用法旨覆盖签押链。
5. 相信公开真相会造成更大崩溃。

藏烬君不能替裴玄阙承担全部罪责。

他是守门人，不是道路源头。

黑账汇合层要证明：

> 新仙界不是没有账，而是每条账线都被拆开封存，直到所有受害者彼此看不见。

## 8. 新司命台接口

中天玄阙内不需要重复建立完整新司命台，但必须有司命黑账接口。

位置：

1. 封禁大阵层边缘。
2. 补墟总账枢旁侧。
3. 玄阙法旨库暗线下方。

接口功能：

| 内容 | 作用 |
|------|------|
| 命数黑账入口 | 接钟无咎主簿权限 |
| 任务源头封存符 | 证明系统任务受保全派法旨影响 |
| 宿主回收名单 | 证明沈临风等宿主是可回收变量 |
| 天命值折损账 | 接补墟司运账 |
| 赵长生不可评级记录 | 证明命数断页和归墟炉变量不被系统掌握 |

季玄屏和钟无咎的空间差异：

| 人物 | 活动空间 | 功能 |
|------|----------|------|
| 季玄屏 | 新司命台分殿、任务模板、天命值结算和司命玄屏廊 | 系统操作层对手 |
| 钟无咎 | 命数黑账入口、任务源头封存、主簿高权限 | 第十三卷高权限守门人 |

沈临风在这里要完成的不是“夺取系统最高权限”，而是：

1. 证明系统任务源头被保全派接管。
2. 公开宿主回收和天命值折损账。
3. 切断命数黑账对任务模板的强制覆盖。
4. 重定司命盘方向：只记录危机，不安排命运。

## 9. 镇界封锁与军令台

封禁大阵层设有镇界军令台。

作用：

1. 调动中天卫戍军。
2. 封锁外阙和补天内府。
3. 护送补墟资源进入九源阵环。
4. 将清账者、旧籍者、黑籍者标为玄阙危害。
5. 在墟眼波动时强行征调飞升者军籍修士。

厉沉霄在这里能制造第十三卷武力压力。

他可以用三种理由封锁证据：

| 理由 | 表层说法 | 实际问题 |
|------|----------|----------|
| 防归墟污染 | 证据区靠近墟眼，必须封锁 | 封存账证和证人 |
| 防旧天庭复辟 | 旧籍残部危险 | 把旧账证人和复辟派混同 |
| 防黑籍叛乱 | 飞升者、失控宿主、黑籍会扰乱封禁 | 把制度受害者打成破坏者 |

镇界封锁不能被顾长庚一剑解决。

合理破解需要：

1. 陆维舟找出镇界军令台与封禁阵的阵路差异。
2. 顾长庚斩断不该存在的道职军令。
3. 纪晚凝拿出军令与抽源护送回执。
4. 沈临风切断命数追索接口。
5. 赵长生用归墟炉处理封禁阵中残败本源，而不是吞掉整座军阵。

## 10. 九源封禁大阵

九源封禁大阵是中天玄阙维持墟眼的核心。

它不是单纯吸灵气的阵法。

九源投入：

| 本源 | 主要来源 | 问题 |
|------|----------|------|
| 灵 | 仙税、仙矿、灵脉抽取 | 底层长期贫瘠 |
| 宝 | 飞升者法宝、残仙器、万宝库拆解 | 本命法宝被拆为宝账 |
| 魂 | 渡劫残魂、亡魂截留、魂灯灰 | 破坏轮回 |
| 血 | 百族真血、血税、猎血商队 | 小族和源血链路被榨取 |
| 运 | 系统宿主、宗门气运、天命值 | 命运被安排和回收 |
| 法 | 道统、功法、万法碑残片 | 修行道路被垄断 |
| 界 | 下界、残界、祖地、洞天 | 世界根基被抽空 |
| 时 | 时痕、太虚航迹、旧录残影 | 风险极高，少量使用 |
| 道 | 高阶道果、旧道职权柄 | 高位修士和旧职被消耗 |

九源封禁大阵的现状：

1. 投入越来越多。
2. 稳定时间越来越短。
3. 权贵豁免越来越重。
4. 黑账封存越来越深。
5. 墟眼反噬开始沿阵环回流。

这个大阵要击穿裴玄阙的核心论据：

> 不是牺牲从来不需要，而是这套牺牲已经不能真正救万界，只能继续喂住旧权力。

## 11. 墟眼禁域

墟眼禁域是真实归墟伤口所在。

表现：

1. 大道规则在这里断裂成黑白交错的裂纹。
2. 仙灵气靠近后被缓慢吞噬。
3. 法宝灵性会脱落。
4. 修士寿元和命数会被磨损。
5. 魂魄会听见不属于自己的回声。
6. 归墟残灰会从伤口边缘飘出。
7. 封禁大阵的九源投入会在边缘被磨成无名灰光。

墟眼不是普通洞口。

它不能被：

1. 一块神石堵住。
2. 一个大能牺牲封住。
3. 杀掉裴玄阙后自然愈合。
4. 赵长生随手用归墟炉吞掉。
5. 旧天庭九器原样归位后直接恢复。

墟眼能被终局处理的方向：

1. 公开旧账和黑账，解除玄阙对危机解释权的垄断。
2. 切断永久收割式九源投入。
3. 重启司命盘、轮回灯、登仙台、补天尺等职责，但不复原旧天庭权力。
4. 由陆维舟建立万界节点阵，让万界重新参与循环。
5. 由赵长生用归墟炉把墟眼从吞噬伤口重炼成循环炉口。

## 12. 墟眼观测台

墟眼观测台是裴玄阙与赵长生正面辩论的理想空间。

裴玄阙可以在这里展示：

1. 墟眼扩大曲线。
2. 封禁失效时的万界回响。
3. 旧天庭第一次失败残影。
4. 某些下界确实已经无法自救。
5. 九源投入后墟眼短暂稳定的记录。

赵长生不能否认这些。

他必须反证：

1. 九源投入的稳定时间正在缩短。
2. 权贵豁免和高层消耗让补墟成本越来越重。
3. 灵界、下界、飞升者和百族被抽后，循环基础更弱。
4. 补墟体系在制造下一轮更大的归墟失衡。
5. 只有重炼循环口，才不是继续往伤口里填众生。

观测台上的冲突不是善恶互骂，而是两条路的实证对撞。

## 13. 证据汇合路径

第十三卷的证据必须多线汇合。

| 证据线 | 起点 | 中天玄阙汇合点 | 终局用途 |

## 17. 退场与回收

中天玄阙不能因裴玄阙败亡就简单炸掉。

合理回收：

1. 玄阙天宫失去解释危机和垄断补墟的权力。
2. 补天内府从收割机构被清算为公开修补接口。
3. 补墟司黑账被公开，血、魂、界、运等账线剥离。
4. 新司命台命数黑账被重定，司命盘只记录危机和自愿选择。
5. 镇界军保留真实守界职责，剥离护送抽源和封锁证据职能。
6. 墟眼禁域不再是玄阙私有禁地，而被重炼为万界循环炉口。

最终方向：

> 仙界仍可成为循环中枢，但不能再成为万界统治者和本源收割者。

## 18. 写作禁忌

1. 不能把墟眼危机写成假的。
2. 不能让杀掉裴玄阙等于解决新仙界。
3. 不能让赵长生用归墟炉直接吞掉墟眼。
4. 不能让中天玄阙只是豪华宫殿，没有账本、军令、司命和封禁大阵。
5. 不能让裴玄阙所有论据都是谎言。
6. 不能让玄阙权贵替裴玄阙承担全部理念罪责。
7. 不能让纪晚凝靠私人感情转向，必须靠账本和责任。
8. 不能让沈临风拿到更高级系统权限后就解决司命台。
9. 不能让镇界军全员知道完整黑账。
10. 不能让旧天庭复辟成为解决中天玄阙问题的答案。

## 19. 使用提示

写中天玄阙时抓五层：

1. 权力层：玄阙天宫、保全法旨、裴玄阙的真实危机论据。
2. 账本层：补天内府、黑账汇合、纪晚凝公开清账。
3. 命数层：新司命台接口、命数黑账、沈临风拒绝被安排。
4. 武力层：镇界军令台、厉沉霄、军令封锁证据。
5. 伤口层：墟眼禁域、九源大阵死循环、赵长生重炼循环炉口。

中天玄阙最终要证明：

> 裴玄阙守住的不是纯粹的天，而是一座真实漏血、却被权力垄断了修补资格的天。赵长生要重炼的也不是一座宫殿，而是万界重新参与循环的入口。
