# 镇界军

> 仙界机构补档。本文用于承载镇界军、军籍、飞升者征调、镇界兵阵、护送抽源、界裂处置、钟离砺和厉沉霄线。本文不是厉沉霄个人单档，也不替代终局保全派理念。

---

## 1. 设计目的

镇界军补档要解决六个问题：

1. 新仙界为什么不只靠账本和系统压人，也有能执行封锁、征调和镇压的武力层。
2. 强大飞升者为什么飞升后也不自由，会被军籍、军印和战功体系绑定。
3. 厉沉霄为什么不是享乐权贵，而是相信镇压能换稳定的军人。
4. 镇界军如何同时承担真实守界职责和抽源护送职责。
5. 顾长庚、陆维舟、纪晚凝、赵长生如何分别从剑道、界阵、账证、循环方案击穿镇界军逻辑。
6. 终局清账时，执行层军令责任如何被处理。

镇界军的核心味道：

> 它确实堵过界裂，也确实护送过抽源账箱。问题是它越来越分不清两者。

## 2. 基础定位

| 项目 | 内容 |
|------|------|
| 机构名 | 镇界军 |
| 表层职责 | 镇压界裂、守卫仙界边境、处理归墟污染、征调飞升强者 |
| 实际功能 | 镇压拒税世界、护送抽源物资、清理黑籍飞升者和旧天庭残部 |
| 活跃卷段 | 第十卷伏笔，第十三卷正面展开 |
| 代表人物 | 钟离砺、厉沉霄 |
| 上级接口 | 中天玄阙、保全派法旨、登仙监分流台 |
| 下游接口 | 军籍营、镇界兵阵、抽源护送队、归墟污染战场 |

镇界军内部不全是坏人。它有真守界者、军令执行者、飞升者军籍修士、借军功换命者和高层保全派。

## 3. 总空间

```text

    ↓

    ↓

    ↓

```

| 空间 | 功能 | 剧情用途 |

| 镇界军征调营 | 接收登仙监分流来的强者 | 钟离砺线，强者也不自由 |
| 军籍登记台 | 将飞升者转入军籍 | 军籍比役籍体面，但束缚更深 |
| 军印绑定室 | 绑定军令、战功、命数接口 | 飞升者命数受军令限制 |
| 战力测试场 | 判断战斗方向和军种 | 顾长庚、钟离砺对照 |
| 仙兵营房 | 军籍修士日常 | 展示镇界军不是一次性反派 |
| 伤兵回收处 | 治疗、记录、退役和再利用 | 证明军功体系也会消耗人 |
| 镇界任务墙 | 界裂、污染、追捕、护送任务 | 真实守界和抽源镇压混杂 |

| 抽源护送司 | 护送补墟物资和账箱 | 纪晚凝查账证据 |
| 黑籍追剿队 | 追捕失籍、拒役、旧籍残部 | 仙界底层压力 |

## 4. 军籍制度

镇界军通过军籍绑定飞升者。

| 项目 | 表层好处 | 实际束缚 |
|------|----------|----------|
| 军籍 | 获得资源、战功、仙界合法身份 | 随时征调，命数受军印记录 |
| 军印 | 可借军阵、战功兑换、身份通行 | 违令反噬，逃军转罪籍 |
| 战功 | 换仙石、功法、丹药、减税 | 战功任务常与镇压和抽源护送相关 |
| 军令 | 明确目标和保护范围 | 不问来源，只问执行 |
| 退役 | 理论上役满可转职籍或府籍 | 高危任务和伤残让退役很难 |

钟离砺可作为被征调飞升者样本。

他不是普通矿役，但他也不自由。强者在仙界也会被制度定价：弱者挖矿，强者入军。

## 5. 军种与职能

| 军种 | 表层职责 | 实际用途 |
|------|----------|----------|
| 镇界营 | 镇压界裂、稳定界壁 | 也负责锁界和封闭被抽源世界 |
| 归墟污染队 | 清理归墟污染者和残界污染 | 有时掩盖抽源后遗症 |
| 飞升者征调营 | 训练新飞升战力 | 将高战力飞升者绑定军籍 |
| 黑籍追剿队 | 追捕失籍、逃役、拒税者 | 清理制度漏洞和证人 |
| 抽源护送司 | 护送补墟资源 | 护送血源、魂源、界源账箱 |
| 中天卫戍军 | 守中天玄阙外围 | 终局前高压封锁 |

镇界军最复杂之处在于：同一支队伍今天可能堵住真实界裂，明天可能护送从下界抽来的界源残核。

## 6. 镇界兵阵

镇界军核心战力不是单个将领，而是军印、兵阵、仙籍和补给线。

| 兵阵 | 用法 | 可被击穿点 |
|------|------|------------|
| 锁界阵 | 封锁一片空间或界裂 | 陆维舟可找阵眼和地脉反压 |
| 镇源阵 | 稳住界源残核或抽源通道 | 证据显示它也用于抽源护送 |
| 断路阵 | 切断黑籍、旧籍、残界逃路 | 赵长生命数断页和归墟炉可能制造空隙 |
| 军印合阵 | 多名军籍修士共享军印力量 | 顾长庚可斩不该存在的军令权柄 |
| 污染隔离阵 | 封住归墟污染区 | 可被滥用为封存证据区域 |

镇界兵阵不能被一剑全破。合理破解需要陆维舟界阵、顾长庚斩道、纪晚凝账证和赵长生归墟炉共同推进。

## 7. 厉沉霄的位置

厉沉霄是镇界军高阶将领，代表保全派武力执行层。

他能做：

1. 调动镇界兵阵。
2. 封锁证据区域。
3. 镇压黑籍飞升者和旧天庭残部。
4. 护送抽源账箱。
5. 以军令要求纪晚凝、登仙监、仙矿府配合。
6. 与顾长庚形成“守界是否等于服从军令”的冲突。

他不能做：

1. 代表裴玄阙的全部理念。
2. 知道完整补墟黑账。
3. 替藏烬君承担账本罪责。
4. 被简单写成腐败权贵。

厉沉霄的错误是长期把“镇住局面”当成“解决问题”。

## 8. 钟离砺的位置

钟离砺是被征调的仙兵飞升者样本。

| 项目 | 内容 |
|------|------|
| 身份 | 下界或灵界飞升上来的强者，被分流入镇界军 |
| 表层目标 | 用战功换资源、仙籍和更高修行路 |
| 实际困境 | 军印绑定，任务越来越像镇压和护送抽源 |
| 叙事功能 | 证明飞升强者也会被制度收编 |

钟离砺可与主角组冲突后合作，也可作为终局飞升者旧账代表之一。

他不能一开始就完全看透镇界军问题。最好让他先相信战功和军籍是飞升者少数能向上走的路。

## 9. 抽源护送

镇界军和补墟司的关键接口是抽源护送。

护送对象：

1. 界源残核。
2. 血源封匣。
3. 魂灯碎焰或魂账证物。
4. 飞升者回收名单。
5. 黑账副本或封账法旨。
6. 归墟污染隔离物。

表层说法：

- 防止污染外泄。
- 防止叛修劫掠。
- 保护补天物资。
- 稳定中天玄阙封禁。

实际问题：

- 护送来源不清的本源物资。
- 护送过程封锁证人和证据。
- 镇界军以军令压过普通仙官质疑。
- 飞升者军籍修士可能不知道自己护送的是被抽来的世界根基。

纪晚凝可通过护送回执和补墟账本对上军令，发现镇界军并非只在守界。

## 13. 退场与回收

镇界军不能因厉沉霄战败就整体消失。

合理回收：

1. 第十卷先以军籍和征调压飞升者。
2. 第十三卷厉沉霄战败、失印、停令或受审，证明执行层被击穿。
3. 钟离砺等飞升者军籍代表进入终局清账。
4. 镇界职责被保留，但护送抽源和镇压拒税世界的职能被清算。
5. 新循环中，镇界军不能继续听命于中天玄阙高层私权，而要回到公共守界职责。

## 14. 写作禁忌

1. 不能把镇界军写成全员恶兵。
2. 不能把厉沉霄写成享乐权贵或无脑打手。
3. 不能让镇界军知道完整黑账。
4. 不能让顾长庚单挑解决全部军阵问题。
5. 不能让钟离砺一出场就完全支持主角组。
6. 不能把真实界裂危机写成假的。
7. 不能让镇界军替裴玄阙承担终局理念罪责。

## 15. 使用提示

写镇界军时抓三层：

1. 底层：飞升者被征调，军籍看似向上路。
2. 中层：军令、军印、战功和兵阵形成秩序。
3. 高层：守界职责被保全派用来护送抽源和封锁证据。

镇界军的存在是为了证明：

> 一个制度坏到深处时，连真正守过界的人，也会被迫守住错误的边界。
