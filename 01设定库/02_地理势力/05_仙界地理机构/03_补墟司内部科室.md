# 补墟司内部科室

> 仙界机构补档。本文用于承载补墟司内部科室、权限层级、办公空间、明账、内账、黑账、封账机制，以及纪晚凝、严慎、傅照灯、藏烬君之间的制度位置。本文不展开完整墟眼禁域，也不替代纪晚凝个人单档。

---

## 1. 设计目的

补墟司内部科室补档要解决七个问题：

1. 纪晚凝为什么一开始相信制度，且只能看到明账和部分内账。
2. 严慎作为账师为什么能看见差额，却不等于正义吹哨人。
3. 傅照灯为什么能保护一部分证据和人，却无法靠修补派解决黑账。
4. 藏烬君为什么是黑账守门人，而不是终局最高反派。
5. 血账、魂账、界账如何在机构空间中被封存、合并、改写和隐藏。
6. 补墟司如何与登仙监、新司命台、镇界军、灵界监察链、仙矿府发生接口。
7. 第十一卷到第十三卷的查账、封账、清账如何有具体路径。

补墟司的核心味道：

> 它不是没有账，而是每一层账都告诉你：再往里翻，会死更多人。

## 2. 基本定位

| 项目 | 内容 |
|------|------|
| 对外名义 | 补天司下属或内设机构，负责统筹补天资源 |
| 内部实质 | 补墟司，负责统计、折算、调拨、封存投入墟眼的本源资源 |
| 权力来源 | 中天玄阙封禁大阵、保全派法旨、仙籍道职体系 |
| 叙事卷段 | 第十卷外账伏笔，第十一卷正式展开，第十二卷旧账对照，第十三卷黑账汇合 |
| 主要人物 | 纪晚凝、严慎、傅照灯、藏烬君 |
| 核心冲突 | 清账 vs 平账 vs 修补 vs 封账 |

补墟司不能写成全员邪恶。

它确实承担过延缓墟眼危机的职责，但长期运行后，已经把临时自救变成永久收割，把万界伤口写成补墟额度。

## 3. 总空间

补墟司内部空间可以按权限从外到内展开。

```text
登仙监 / 洗尘池 / 验源廊
        ↓
补墟司外账房
        ↓
明账厅 —— 仙籍署 / 仙矿府 / 镇界军回执
        ↓
内账廊 —— 本源折算室 —— 额度复核室
        ↓
封账阁 —— 法旨库 —— 证人禁室
        ↓
黑账深库
        ↓
中天玄阙封禁阵接口 / 不开放
```

| 区域 | 权限 | 功能 | 相关人物 |
|------|------|------|----------|
| 外账房 | 低到中 | 接收登仙监、仙籍署、验源廊明账 | 纪晚凝第十卷主要空间 |
| 明账厅 | 中 | 仙税、役期、飞升者分流、普通补墟额度 | 纪晚凝、严慎 |
| 内账廊 | 中高 | 本源分类、额度折算、损耗对照 | 严慎、傅照灯 |
| 本源折算室 | 中高 | 灵、宝、魂、血、运、法、界、时、道分类折算 | 严慎可接触部分 |
| 额度复核室 | 中高 | 判断某批资源是否足以维持封禁 | 纪晚凝后期进入 |
| 封账阁 | 高 | 合并、封存、遮蔽敏感账项 | 傅照灯可触边，藏烬君掌权 |
| 法旨库 | 高 | 保存保全派法旨、豁免令、封存令 | 藏烬君、上级法旨 |
| 证人禁室 | 高 | 看押或隔离账师、飞升者、旧籍证人 | 严慎可能被送入 |
| 黑账深库 | 极高 | 血账、魂账、界账和权贵豁免原册 | 藏烬君 |

第十一卷不能直接打开黑账深库。黑账应先通过封账痕迹、差额、被抹签押和证人失踪表现。

## 4. 科室总表

| 科室 | 表层职责 | 实际功能 | 叙事用途 |
|------|----------|----------|----------|
| 外账房 | 接收飞升者分流、仙税、验源记录 | 将登仙监和仙籍体系接入补墟账 | 纪晚凝登场和初始信念 |
| 明账厅 | 公开账务核对 | 让制度看起来清楚、公正、可复核 | 第十到十一卷制度压迫 |
| 内账廊 | 本源额度和损耗复核 | 将具体伤害折成灵、宝、魂、血、界等类别 | 严慎发现差额 |
| 折源科 | 九种本源折算 | 把人、族群、亡魂、世界拆成可投入额度 | 证明制度语言遮蔽人命 |
| 平账科 | 差额修补、账目归并 | 把异常写成损耗、贪墨或流程误差 | 严慎职业本能 |
| 修补科 | 限制过度抽源、流程缓冲 | 保护一部分证据和人，但无法动根 | 傅照灯线 |
| 封账阁 | 敏感账目封存 | 把血账、魂账、界账遮到黑账之前 | 藏烬君前置压力 |
| 黑账深库 | 原始黑账和法旨 | 封存制度最难看的证据 | 第十三卷证据争夺 |
| 旧账校对室 | 对照旧天庭正常循环账 | 后期确认新仙界篡改旧职责 | 第十二卷转折 |

## 5. 四层账本

补墟司账本分四层。

| 层级 | 名称 | 内容 | 谁能看 |
|------|------|------|--------|
| 第一层 | 明账 | 仙税、役期、飞升者分流、普通资源入账 | 普通仙官、外账房、纪晚凝早期 |
| 第二层 | 内账 | 本源分类、补墟额度、损耗、折算公式 | 补墟司核心账师、严慎部分权限 |
| 第三层 | 黑账 | 强制抽源、猎血、炼魂、系统回收、权贵豁免 | 藏烬君等极少数守门人 |
| 第四层 | 旧账 | 旧天庭正常循环账和新仙界篡改痕迹 | 旧天庭遗迹、纪晚凝后期接触 |

纪晚凝的成长必须按层推进：

1. 相信明账。
2. 发现明账和内账对不上。
3. 追到黑账封存痕迹。
4. 拿旧账对照，确认补墟体系已经背叛原职责。
5. 公开清账，而不是私自掌账。

## 6. 外账房

外账房是补墟司面向登仙监、仙籍署、仙矿府和镇界军的接口。

接收内容：

1. 飞升者临籍和分流记录。
2. 洗尘池验源异常。
3. 仙税和役期折算。
4. 法宝暂扣清单。
5. 血源取样记录。
6. 仙矿、镇界军、黑籍追捕回执。

纪晚凝第十卷适合在这里登场。

她能看到：

- 流程是否合规。
- 某个飞升者是否异常。
- 某批分流是否超额。
- 某些血源、界源、运源记录是否需要复核。

她看不到：

- 血账原册。
- 魂账磨损录。
- 界账抽源图。
- 权贵豁免法旨。
- 墟眼真实损耗总账。

这样能保证她前期不是装不知道，而是真的处在制度给她看的范围内。

## 7. 明账厅

明账厅负责让补墟司看起来有序。

明账项目：

| 项目 | 表层意义 | 隐藏问题 |
|------|----------|----------|
| 仙税入账 | 飞升者应缴资源 | 税额背后可能有强制分流 |
| 役期折算 | 仙矿、镇界军服役抵税 | 役期死亡和伤残常被合并损耗 |
| 飞升者分流 | 仙矿、军籍、府籍、司命台等去向 | 部分人消失在黑账或系统回收 |
| 验源异常 | 血、魂、界、运等高价值标签 | 标签可能直接接入补墟额度 |
| 法宝暂扣 | 防止危险器物入界 | 飞升者法宝被拆解入宝账 |

明账不是假的。它记录的多数内容都真实存在。问题在于它只记录制度愿意被看见的部分。

## 8. 内账廊

内账廊负责将明账中的资源和损耗折成补墟额度。

严慎可以接触部分内账。

内账常见分类：

| 类别 | 内容 |
|------|------|
| 灵账 | 灵脉、仙石、灵田产出、仙矿 |
| 宝账 | 法宝、残仙器、器纹、旧器 |
| 魂账表层 | 渡劫失败者残魂、魂灯异常记录 |
| 血账表层 | 血样、真血登记、血税额度 |
| 运账 | 天命值、气运评级、系统奖励消耗 |
| 法账 | 功法、道统、秘术登记 |
| 界账表层 | 出身世界、洞天、小世界、祖地病灶 |

严慎的问题在这里出现：

1. 他能看见明账和内账数字对不上。
2. 他第一反应是找损耗、贪墨或流程误差。
3. 他害怕继续追查会触动封账阁和黑账。
4. 他必须在平账和作证之间选择。

## 9. 折源科

折源科负责把九种本源折成可入账额度。

| 本源 | 折算方式 | 危险 |
|------|----------|------|
| 灵 | 灵石、灵脉、洞府效率 | 容易把下界灵脉衰败写成自然损耗 |
| 宝 | 法宝、残器、器纹 | 飞升者本命法宝可能被拆成宝账 |
| 魂 | 残魂、魂灯、轮回异常 | 亡魂容易被写成补墟耗材 |
| 血 | 真血、源血链路、族契献血 | 百族被写成血税 |
| 运 | 气运、天命值、系统奖励 | 新司命台接口 |
| 法 | 功法、道统、万法碑残片 | 道统被保护名义垄断 |
| 界 | 界源、祖地、残界、洞天 | 下界和灵界祖地被抽空 |
| 时 | 时痕、遗迹残影 | 前中期少用，风险高 |
| 道 | 道果、道痕、旧职权柄 | 高阶修士和旧天庭遗迹线 |

折源科的罪在于：它把具体的人、族群、亡魂和世界，翻译成干净的额度表。

## 10. 平账科

平账科负责处理差额。

常见说法：

- 自然损耗。
- 战损归并。
- 执行层贪墨。
- 验源误差。
- 役期折损。
- 临时补录。
- 法旨合并。

严慎可以隶属于平账科或与其长期协作。

平账科不能写成坏人窝。它的很多仙官可能只是怕账乱、怕封禁失控、怕高层问责。正因为他们把“平住”看得比“查清”重要，才让制度坏账持续多年。

## 11. 修补科

修补科是制度修补派活动空间。

傅照灯适合隶属或主导此线。

修补科能做：

1. 延迟过度征收。
2. 为低层账师提供流程保护。
3. 在账册被焚前留下副录。
4. 利用法旨漏洞拖延封账。
5. 推动部分赔付、减税、减役。
6. 短暂保护纪晚凝、严慎或证人。

修补科不能做：

1. 直接打开黑账深库。
2. 取消补墟司。
3. 抹除保全派法旨。
4. 阻止所有抽源。
5. 替纪晚凝完成清账选择。

傅照灯的核心矛盾就在这里：

> 他能救下一页账，却救不了账外正在被继续抽源的世界。

## 12. 封账阁

封账阁是黑账深库之前的门。

它的表层职责是保护敏感账目，防止公开造成恐慌或被敌对势力利用。

实际职责：

1. 合并血账、魂账、界账敏感项目。
2. 抹除低层签押链。
3. 封存证人。
4. 焚毁或改写副本。
5. 让黑账只留下可控灰烬。

藏烬君可以掌握封账阁或直接掌握黑账深库。

封账阁常见动作：

- 一本账被移走。
- 一个差额项被合并。
- 一个证人被调离。
- 一条签押链被法旨覆盖。
- 一份副录被焚成灰。

## 13. 黑账深库

黑账深库保存补墟司最核心罪证。

三类核心黑账：

| 黑账 | 内容 | 对应人物线 |
|------|------|------------|
| 血账 | 百族真血、血税、族契献血、猎血许可、源血抽样 | 白倩霜、明厌、百族契约 |
| 魂账 | 渡劫失败者残魂、亡魂截留、轮回灯磨损、炼魂坏账 | 洛闻霜、轮回灯 |
| 界账 | 下界抽源、中层祖地衰败、残界封存、界源图 | 叶疏彤、陆维舟、赵长生 |

附属黑账：

- 运账：天命系统、宿主回收、天命值折损。
- 宝账：飞升者法宝拆解、旧器封存。
- 法账：道统封锁、万法碑残片。
- 权贵豁免记录：玄阙仙族和高层道统被减免补墟负担。

黑账不能在第十一卷一次性被偷出来。它应该在第十一卷以痕迹出现，第十二卷通过旧账对照逼近，第十三卷由多条证据汇合后公开。

## 16. 卷际使用

| 卷 | 使用重点 |
|----|----------|
| 第十卷 | 外账房、验源异常、仙籍分流，纪晚凝以制度执行者登场 |
| 第十一卷 | 明账和内账差额，严慎、傅照灯登场，司命台和补墟司接口暴露 |
| 第十二卷 | 旧账对照，确认差额不是近期贪墨，而是制度长期转嫁 |
| 第十三卷 | 封账阁和黑账深库正面进入，藏烬君守血账、魂账、界账 |
| 第十四卷 | 清账、公开旧账、重定仙籍和补墟职责，补墟司不再作为收割机构存在 |

## 17. 可用剧情动作

写补墟司时，不要只讲概念。每次出场可以使用具体动作：

1. 纪晚凝复核一批飞升者分流账。
2. 严慎发现同一批仙矿役籍在明账和内账人数不一致。
3. 傅照灯用流程漏洞拖延封存一页副录。
4. 封账阁调走某个低层账师。
5. 藏烬君焚毁账册表层，只留下账火残灰。
6. 白倩霜的灵界血名册编号出现在补墟司血账表层。
7. 沈临风系统奖励对应的运账消耗被纪晚凝发现异常。
8. 旧天庭旧账显示同一职责原本是救界，不是抽界。

## 18. 写作禁忌

1. 不能把补墟司写成全员坏官机构。
2. 不能让纪晚凝一出场就知道完整黑账。
3. 不能让严慎一开始就是正义吹哨人。
4. 不能让傅照灯成为无责好人内应。
5. 不能让藏烬君替裴玄阙承担全部罪责。
6. 不能让黑账只是一句“很多坏账”，必须有血账、魂账、界账实体。
7. 不能让赵长生偷到账本后私自裁决所有人。
8. 不能让公开清账变成烧掉所有账。清账是公开、追责、赔付、改规则和承接后果。
9. 不能把墟眼危机写成假的。危机真实，错误在于收割制度化和封账永久化。

## 19. 使用提示

补墟司内部科室写法要抓四种人：

1. 清账者：纪晚凝后期，承认账背后有具体死者和世界。
2. 平账者：严慎前期，先想把差额归并为损耗。
3. 修补者：傅照灯，想少伤人但不敢立刻掀桌。
4. 封账者：藏烬君，相信公开真相会死更多人。

这四种人共同证明：

> 新仙界最可怕的不是没有人知道问题，而是每个人都能找到一个继续不翻到底的理由。
