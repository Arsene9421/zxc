# 霜羽族

> 灵界势力补档。本文用于承载白倩霜的族群责任、霜羽寒原、霜羽祖巢、祖契裂痕、源血链路、族内派系和猎血压力。本文不是白倩霜个人单档，也不是第七卷章纲。

---

## 1. 设计目的

霜羽族补档要解决五个问题：

1. 白倩霜为什么不能被赵长生救走后就脱离族群。
2. 霜羽族为什么会把“交出白倩霜”视为理性选择，而不是全员降智。
3. 祖地、源血、族契、族运如何具体落在空间和制度上。
4. 猎血制度如何从外部商队、大族压力和灵界监察链共同压到一个小族身上。
5. 终局重订百族契约时，霜羽族能提供什么具体旧伤和新规则样本。

霜羽族的核心不是“女主出身的漂亮小族”，而是：

> 一个衰败小族在灭族压力下，能不能不把族中最有价值的人写成还债材料。

## 2. 基础定位

| 项目 | 内容 |
|------|------|
| 族名 | 霜羽族 |
| 类别 | 灵界羽族小族，真灵后裔残支 |
| 祖地 | 霜羽寒原深处的霜羽祖巢 |
| 血脉特征 | 风雪、寒羽、短距空行、族契感应 |
| 当前状态 | 祖地衰败、祖契裂痕加重、年轻族人血脉变薄 |
| 核心资源 | 霜羽源泉、旧翼碑林、祖契台、内巢封雪洞 |
| 核心危机 | 白倩霜返祖真血能感应源血链路，被猎血商队、大族和监察链盯上 |
| 剧情卷段 | 第六卷伏线，第七卷核心，第八卷入账，第九卷提供血源证据 |

霜羽族不是灵界大族。它有祖地和族契，但没有能力长期正面对抗玄鳞族、猎血商队和灵界监察链。

## 3. 霜羽族历史

霜羽族曾是羽族中一支小真灵后裔，祖上与寒风、雪羽、太虚空路有微弱关联。

旧时职责：

1. 守护霜羽寒原一段空路。
2. 以族契稳定寒原风雪灵机。
3. 在百族迁徙中提供短距空路和侦察。
4. 保存部分真灵寒羽血脉。

衰败原因：

1. 祖地灵机多年变薄。
2. 霜羽源泉水脉转冷转浊，源血链路断续。
3. 多次百族战争后，族中高阶修士凋零。
4. 猎血商路绕过寒原外道，频繁捕捉族中血脉较纯者。
5. 灵界监察链将霜羽族祖契裂痕登记为“族群风险”，实际把它推入血税名单。

霜羽族不能写成古老强族落魄版。它更像一个已经习惯紧缩、忍让、献出边缘族人的小族。

霜羽族主要空间见 `02_灵界百族地图与祖地.md`，本文细化族内使用。

| 空间 | 功能 | 剧情用途 |
|------|------|----------|
| 霜羽寒原外道 | 连接猎血商路、百族盟场和祖巢外缘 | 追猎、封锁、白倩霜回族前的压迫 |
| 风雪哨台 | 族中侦察点，年轻族人轮守 | 展示族群仍想自保，但能力有限 |
| 霜羽外寨 | 普通族人、幼弱、病弱和外来商队接触区 | 展示祖地衰败对日常生活的影响 |
| 祖契台 | 族中裁断、祭契、族规宣告处 | 白倩霜面对“交出个人保全族”的公开压力 |
| 霜羽源泉 | 源血链路感应点，祖地灵机核心 | 白倩霜真血共鸣，叶疏彤识别生机病灶 |
| 旧翼碑林 | 族史、迁徙、战死者名录和旧契碑 | 证明霜羽族曾守护过空路，也曾牺牲过族人 |
| 内巢封雪洞 | 真血者闭关或封存旧羽骨之处 | 白倩霜觉醒、源血链路证据、猎血最终争夺点 |

## 5. 祖地病灶

霜羽祖地衰败不是单纯灵气不足。

表层表现：

1. 风雪灵机越来越薄。
2. 源泉结寒斑，水中有灰白丝线。
3. 新生儿寒羽纹路变淡。
4. 年轻族人短距空行更容易失衡。
5. 祖契台回应变慢。
6. 外寨药圃冻伤，灵草难以续种。

深层原因：

1. 霜羽源血链路被多次猎血术污染。
2. 族契中有被迫签下的“献血豁免”旧条。
3. 灵界监察链把部分祖地损耗登记为自然衰败。
4. 大族和猎血商队将霜羽族特殊血脉标成可采资源。
5. 万年抽源前后，祖地界源和族运被拆成不同账目转走。

叶疏彤可以看出祖地病灶不是普通寒毒或灵泉衰竭；赵长生可以通过归墟炉从猎血器具和源泉残痕中还原一部分污染证据。但两人都不能无代价修好整个祖地。

## 6. 血脉与源血链路

霜羽族血脉大致分四层：

| 层级 | 名称 | 表现 | 风险 |
|------|------|------|------|
| 普通族血 | 寒羽纹 | 抗寒、短距滑翔、风雪感应 | 猎血价值低，但祖地衰败后也会变弱 |
| 灵血 | 霜翎纹 | 可短暂借风雪灵机，适合侦察 | 易被低阶血脉鉴定盯上 |
| 古血 | 旧翼纹 | 能感应旧翼碑林，觉醒片段族史 | 容易被族中保守派视为责任人 |
| 真血 | 霜羽真血 | 能感应源血链路和祖契裂痕 | 猎血核心目标，白倩霜属于此层 |

白倩霜不是源血本人。

她的定位是：

> 霜羽族返祖真血携带者，能感应本族源血位置和族契裂痕，是源血链路钥匙。

这意味着：

1. 大族要她，不是为了普通取血，而是为了定位源血链路。
2. 猎血商队追她，是因为她能被卖成高价值血源钥匙。
3. 族中保守派想交出她，是因为误以为交出钥匙可换祖地短期豁免。
4. 赵长生救她不能只救走个人，还必须面对她与族群之间的链路。

## 7. 祖契结构

霜羽族祖契不是单纯族规，而是族群、祖地、源血和族运之间的约束。

| 族契层级 | 内容 | 当前问题 |
|----------|------|----------|
| 家契 | 各支系负责外寨、哨台、药圃、源泉守护 | 部分支系认为白倩霜一支应承担献血旧债 |
| 支族契 | 族中各支共同守祖地，按血脉和贡献分配资源 | 支族之间互相推责，年轻族人不满 |
| 祖契 | 霜羽族与祖地、源泉、旧翼碑林的总契 | 裂痕加重，回应变慢，保守派主张牺牲真血者补契 |
| 监察伪契 | 灵界监察链登记的豁免、献血、赔偿条款 | 把被迫献血写成族群自愿 |

祖契原本有保护功能：

- 保护幼弱。
- 分担寒原风雪侵蚀。
- 让族中真血者能感应祖地病灶。
- 在灭族风险前发出警兆。

祖契被扭曲后变成压迫：

- 族中长老可借祖契要求某个个体为全族献身。
- 监察链可借族契裁定，把交人写成合法。
- 大族可借豁免条款，将霜羽族的短期生存变成长期血债。

白倩霜线的重点不是毁掉族契，而是重订族契：

> 族契必须承认族中个体也是目的，不是祖地、族运和长老会的耗材。

## 8. 族内派系

霜羽族内部不能写成全员迫害白倩霜，也不能写成全员无条件护她。

| 派系 | 代表 | 主张 | 问题 |
|------|------|------|------|
| 保守献契派 | 白曜 | 交出白倩霜，换监察豁免和大族暂缓追索 | 把短期平安建立在牺牲个体上 |
| 守祖忍让派 | 部分长老和外寨管事 | 不主动交人，但也不敢公开反抗监察链 | 拖延只会让猎血压力加重 |
| 年轻护族派 | 白疏影等年轻族人 | 不愿交出白倩霜，想重新守祖地 | 热血不足以解决祖地病灶和大族封锁 |
| 旧契见证者 | 老祭契人、碑林守者 | 记得祖契旧义，知道献血豁免并非自愿 | 证据残缺，难以压过现行裁定 |
| 逃散族人 | 寒原外道、散血市中的霜羽混血 | 不信祖地能保护人，只想活下去 | 证明族契若不改，会继续逼人逃离 |

白曜不是单纯恶人。

他可以真心害怕灭族，也可以真心认为自己在做痛苦但必要的决定。但他的错误在于：把“族群还活着”压过了“族人也是人”。

白疏影等年轻族人不能只负责喊口号。他们要承担代价，例如冒险送信、守哨、藏人、带主角组进旧翼碑林，也可能因经验不足导致危机扩大。

## 9. 对外关系

| 对象 | 关系 | 剧情功能 |
|------|------|----------|
| 玄鳞族 | 大族压迫者和水路封锁者 | 以猎血和豁免逻辑逼迫霜羽族交人 |
| 苍羽族 | 羽族大族和空路压力来源 | 可提供空路信息，也会居高临下看待霜羽族 |
| 魂梦族 | 魂源和祖魂祭相关族群 | 可为旧契证言、残魂证据提供支线 |
| 猎血商队 | 直接追猎者 | 将白倩霜标价，把族契争议变成货物路线 |
| 灵界监察链 | 合法化者 | 通过血名册、族契裁定和献血豁免把追猎制度化 |
| 散血市 | 灰色缓冲地 | 霜羽逃散者、混血者和假族契来源 |

霜羽族与玄鳞族的冲突不能写成小族正义、大族邪恶的单线。

玄鳞族内部也可以有人知道猎血不对，但大族自保逻辑会让他们选择牺牲小族换豁免。霜羽族内部也会有人配合猎血。这样才能支撑灵界篇的制度复杂性。

## 13. 写作禁忌

1. 不能把霜羽族写成白倩霜的陪嫁背景。
2. 不能把白倩霜写成源血本人或移动血包。
3. 不能让赵长生用归墟炉炼化霜羽血脉获取战力。
4. 不能让白倩霜一被救就放弃族群。
5. 不能把保守长老全写成蠢坏，他们要有真实灭族恐惧。
6. 不能把年轻族人写成只会热血喊话，他们要付行动代价。
7. 不能让祖契只有压迫功能，它原本也必须有保护功能。
8. 不能让霜羽族问题靠击杀一个猎血商队就结束。
9. 不能让霜羽族终局统治百族，终局是重订契约，不是换一个血脉高贵者掌权。

## 14. 使用提示

写霜羽族时，要同时抓三层压力：

1. 祖地压力：源泉衰败、祖契裂痕、族运变薄。
2. 族内压力：保守交人、年轻反抗、旧契残证、逃散族人。
3. 外部压力：猎血商队、大族豁免、监察裁定、补墟入账。

白倩霜的真正选择不是“跟不跟赵长生走”，而是：

> 她是否能在不被族群吞掉个人的前提下，重新把自己和族群连接起来。
