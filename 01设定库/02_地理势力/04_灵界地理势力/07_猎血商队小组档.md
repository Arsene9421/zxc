# 猎血商队小组档

> 灵界势力补档。本文用于承载猎血商队、阮青鳞、商路货栈、血样交易、大族代执、监察默许和私吞边界。本文不是阮青鳞个人单档，也不替代 `06_灵界监察链.md`。

---

## 1. 设计目的

猎血商队小组档要解决六个问题：

1. 猎血为什么不是一个反派追杀白倩霜，而是商路、货栈、名单、许可和大族需求共同运行。
2. 阮青鳞为什么能追猎，却不能代表整个猎血制度。
3. 商队怎样把血样、假族契、逃族路线和大族代执包装成正常贸易。
4. 监察链为什么会默许商队行动，又为什么不能完全控制商队私利。
5. 主角组击败一个小组后，能夺到哪些证据，不能立刻解决哪些问题。
6. 第六到第七卷的追逃如何从空间、利益和文书上成立。

猎血商队的核心味道：

> 他们不是在暗巷里抢人，而是在商路上用货单、通行印、血样箱和族契副本把人运成货。

## 2. 基础定位

| 项目 | 内容 |
|------|------|
| 名称 | 猎血商队 |
| 活跃区域 | 落界边城、散血市、猎血商路、霜羽寒原外道、玄鳞大泽水门、百族盟场 |
| 表层身份 | 灵材、兽骨、丹材、矿石和族地特产商队 |
| 实际功能 | 按血名册和猎血许可追索特殊血脉、血样、逃族者和源血链路钥匙 |
| 主要代表 | 阮青鳞 |
| 上游接口 | 灵界监察链、玄鳞族等大族、血名册暗点 |
| 下游接口 | 黑血货栈、百族盟场、大族封库、抽源账站 |
| 活跃卷段 | 第六卷到第七卷为主，第八卷以证据和残部回收 |

猎血商队不是灵界最高权力。

它的定位是：

> 监察链不方便直接动手、大族不愿公开脏手时，商队负责把名单变成押运路线。

## 3. 商路位置

猎血商队活动在灵界底层和百族祖地之间。

```text
落界边城
→ 散血市
→ 黑血货栈
→ 猎血商路
→ 霜羽寒原外道
→ 百族盟场
→ 玄鳞大泽水门
```

| 节点 | 商队用途 | 剧情作用 |
|------|----------|----------|
| 落界边城 | 接新货、查外来修士、盯下界残路落点 | 白倩霜登场，主角组重新底层求生 |
| 散血市 | 买血样、买假族契、买逃族路线 | 展示血脉被定价 |
| 黑血货栈 | 暗货中转、押人暂存、清洗货单 | 阮青鳞小组主要据点 |
| 猎血商路 | 押送、伏击、调包、灭口 | 第六到第七卷追逃主路 |
| 霜羽寒原外道 | 封住白倩霜回祖地路线 | 族群责任线被逼出来 |
| 百族盟场 | 把追猎包装成族契裁定 | 让猎血合法化 |
| 玄鳞大泽水门 | 大族接货、换豁免、封锁水路 | 证明商队背后有大族需求 |

## 4. 小组结构

猎血商队可有多支小组。阮青鳞小组是第六到第七卷可写的代表，不是唯一商队。

| 岗位 | 功能 | 写法 |
|------|------|------|
| 商队管事 | 接单、议价、安排路线、和监察暗点对接 | 阮青鳞承担此位 |
| 血样师 | 保存血样、鉴定真伪、处理污染 | 可卷纲阶段具名 |
| 契文师 | 制作假族契、抄录裁定副本、改货单 | 用来证明文书链存在 |
| 押车修士 | 押送目标和暗货 | 负责追逃、伏击和护车 |
| 货栈账手 | 管账、分货、销毁边角证据 | 主角组可从其账册夺证 |
| 监察耳目 | 不一定穿监察服，负责确认名单内目标 | 证明默许不是公开命令 |
| 大族联系人 | 接玄鳞族等大族需求 | 不常露面，避免大族直接沾手 |

商队小组的危险不在于单人战力最高，而在于：

1. 人熟路熟。
2. 有货栈。
3. 有通行印。
4. 有名单。
5. 有大族接货。
6. 有监察链默许。

## 5. 阮青鳞的位置

| 项目 | 内容 |
|------|------|
| 身份 | 猎血商队管事，熟悉商路、血样交易和大族代执 |
| 功能 | 第六到第七卷执行层压力代表 |
| 性格方向 | 精明、滑、会算风险，习惯把人说成货 |
| 欲望 | 通过高价值血源交易换取修行资源、商路地位和大族庇护 |
| 恐惧 | 被监察链当私猎替罪羊，被大族灭口，被主角组夺到许可证据 |
| 底线 | 不碰名单外太高风险目标，不和大族硬顶 |
| 错误 | 以为只要手续齐全，人就可以被定价和押送 |

阮青鳞不能写成色欲反派。

他对白倩霜的兴趣不是美色，而是：

1. 她是霜羽返祖真血。
2. 她可能定位源血链路。
3. 她能让玄鳞族或其他大族换到补墟豁免。
4. 她的族契争议已经被监察链写成可追索目标。

阮青鳞可以狡诈、狠、贪，但不要把他写成制度源头。他只是把制度最脏的一步做成了生意。

## 6. 货物结构

猎血商队表面货物和暗货必须分层。

| 类型 | 表层说法 | 实际用途 |
|------|----------|----------|
| 灵草、兽骨、矿石 | 普通商货 | 掩护路线和税单 |
| 血样瓶 | 医修样本、血脉鉴定材料 | 定价、追踪、比对血名册 |
| 假族契 | 族群通行文书 | 帮逃族者逃，也能把逃族者卖回去 |
| 族契副本 | 调停证据 | 包装交人、献血、赔偿合法性 |
| 封血针、锁脉环 | 医疗器具 | 短时压制真血和血脉术 |
| 货车暗格 | 贵重灵材暗仓 | 押送活人、残魂证物或血样箱 |
| 旧血牌 | 旧标记牌 | 证明目标曾被追索或已定价 |

血样不是万能追踪器。

限制：

1. 血样保存需要冷匣、封符和血样师维护。
2. 真血目标可伪装、稀释或借祖地气息干扰比对。
3. 血样越珍贵，越容易被商队私吞或调包。
4. 归墟残灰会污染血样，导致比对失真。
5. 赵长生可从废血瓶、锁脉环和封符残片中还原部分证据，但不能用归墟炉直接清掉所有追踪。

## 7. 大族代执

大族代执是猎血商队存在的关键。

流程：

```text
大族提出需求
→ 监察链或血名册暗点确认目标在册
→ 商队取得猎血许可或默认通行
→ 货栈安排路线和押车
→ 追索目标、血样或族契证物
→ 大族接货或百族盟场裁定
→ 战损、赔偿或血税进入后续账目
```

大族为什么不直接动手：

1. 直接动手会破坏百族盟场表面规则。
2. 小族反抗容易引发公开族战。
3. 监察链需要流程和文书，不需要大族明抢。
4. 商队可以被切割，一旦出事可称为私猎。
5. 大族可用货栈、假契和代执遮住自身责任。

玄鳞族是当前最重要的大族接口，但不是唯一接口。后续苍羽族、山魁族或其他大族也可有类似灰色需求。

## 8. 监察默许与私吞边界

灵界监察链对猎血商队的态度不是完全掌控，而是有限默许。

监察链默许的部分：

1. 名单内目标追索。
2. 指定商路通行。
3. 族契争议对象押送。
4. 血样采集和上交。
5. 战损和赔偿转账。

监察链不允许或表面不允许的部分：

1. 名单外私猎。
2. 私吞真血。
3. 私自逼死关键血源目标。
4. 伪造监察法印。
5. 把可入账目标卖给黑市或邪修。
6. 造成大规模无法平账的族群冲突。

这给剧情提供灰度：

- 明厌可以利用阮青鳞。
- 阮青鳞也可能私吞或调包。
- 玄鳞族可借商队追索。
- 商队出事后，各方都可把责任推给商队。

## 9. 常用行动

写猎血商队时，尽量用具体动作推进。

| 行动 | 表面动作 | 实际含义 |
|------|----------|----------|
| 查货单 | 检查灵材和兽骨 | 对血样箱和暗货核码 |
| 换车牌 | 商队调度 | 更换追猎路线，躲过散血市耳目 |
| 留血签 | 医修鉴定 | 给目标下追踪锚点 |
| 递通行印 | 商路通关 | 证明监察默许 |
| 买假契 | 帮人逃族 | 建立二次勒索或回卖 |
| 封车 | 保护贵货 | 押送活人或特殊血样 |
| 灭散账 | 清理账册边角 | 消除私吞证据 |

阮青鳞小组的追猎不应靠瞬移围捕。

合理追猎方式：

1. 通过散血市消息锁定白倩霜大致路线。
2. 用货栈账手查她经过的医馆、客栈或假契摊。
3. 在猎血商路设伏。
4. 借百族盟场裁定逼迫霜羽族交人。
5. 联系玄鳞族水门封锁退路。
6. 被主角组破坏后，转为夺证和灭口。

## 11. 可夺取证据

| 证据 | 来源 | 后续用途 |
|------|------|----------|
| 猎血许可副本 | 阮青鳞或货栈暗柜 | 证明商队不是私猎，而是名单内代执 |
| 血样编号册 | 血样师冷匣 | 接灵界血名册和补墟血账 |
| 黑血货栈账册 | 货栈账手 | 证明血样交易长期存在 |
| 假族契模板 | 契文师 | 证明逃族路线和回卖路线是一体两面 |
| 玄鳞水门回执 | 大族联系人 | 证明大族接货和豁免交易 |
| 监察通行印痕 | 商队货单或车牌 | 接明厌和监察链 |
| 锁脉环残片 | 押车修士或战场 | 赵长生可还原猎血器具残痕 |

证据不能一次性解释完整补墟黑账。

它们只能证明：

1. 猎血不是私事。
2. 商队有许可。
3. 大族接货。
4. 监察链默许。
5. 后续还有抽源账站和仙界账口。

## 12. 卷际使用

| 卷 | 用法 |
|----|------|
| 第六卷 | 白倩霜被追猎，阮青鳞小组通过散血市、货栈和商路制造压力 |
| 第七卷 | 商队追猎与霜羽祖地、百族盟场、玄鳞大泽水门接上 |
| 第八卷 | 商队账册、猎血许可和血样编号进入抽源账站证据链 |
| 第九卷 | 登仙台残路验源时，血样编号和白倩霜血源证据回收 |
| 第十卷以后 | 纪晚凝在补墟司血账中看到灵界猎血商队回执 |

## 13. 退场与回收

阮青鳞小组可以被击败，但猎血商路不能因此消失。

可选回收方式：

1. 阮青鳞被杀或重伤，主角组夺到猎血许可副本。
2. 阮青鳞出卖上游，暴露明厌或玄鳞族水门回执。
3. 商队被监察链灭口，证明执行层可被切割。
4. 黑血货栈被毁，但血名册暗点转移。
5. 第八卷抽源账站接手商队账册，说明商队只是外围手。

不要让猎血商队死光。更好的结构是：一个小组败了，路线暴露了，制度换手继续。

## 14. 写作禁忌

1. 不能把阮青鳞写成色欲追人反派。
2. 不能让猎血商队成为制度源头。
3. 不能让击败一个商队就解决白倩霜族群问题。
4. 不能让血样追踪万能准确。
5. 不能让商队没有账、没有货、没有路线，只靠追杀出场。
6. 不能让玄鳞族或监察链完全不留痕迹。
7. 不能让赵长生用归墟炉直接清除所有血脉追踪和族契压力。
8. 不能把散血市写成纯恶黑市，它也是夹缝活人的生存地。

## 15. 使用提示

写猎血商队时抓三层：

1. 商路层：货栈、车队、货单、通行印、暗格。
2. 血源层：血样、假族契、锁脉环、源血钥匙。
3. 制度层：猎血许可、大族代执、监察默许、抽源入账。

阮青鳞的失败要让读者明白：

> 白倩霜不是被坏人看上，而是被一条商路、一套名单和一群愿意把族人写成货物的人共同追索。
