# 灵界监察链

> 灵界势力补档。本文用于承载明厌、俞简、血名册、族契裁定、猎血合法化、战损入账、抽源账站和登仙台验源口之间的制度链。本文不是明厌个人单档，也不是补墟司核心黑账档。

---

## 1. 设计目的

灵界监察链补档要解决六个问题：

1. 灵界猎血为什么不是单个商队或邪修行为，而是有名单、印信、裁定和账目的制度。
2. 明厌的权力从哪里来，又为什么不能让他知道完整新仙界黑账真相。
3. 霜羽族、玄鳞族、魂梦族、山魁地脉、残界带和登仙台残路如何被同一套监察流程串起。
4. 血源、界源、族运和战损如何从灵界事件转入仙界补墟外围账口。
5. 谢未明、沈临风、白倩霜和纪晚凝后期如何从灵界证据接到司命台和补墟司。
6. 主角组击败一名监察使后，为什么制度不会立刻结束。

监察链的核心味道：

> 灵界不是没人管，而是每一次伤害都能被写成“有据可查”。

## 2. 基本定位

| 项目 | 内容 |
|------|------|
| 名称 | 灵界监察链 |
| 表层职责 | 记录百族冲突、调停族契争议、登记特殊血脉、防止归墟污染扩散 |
| 实际功能 | 把血脉、祖地、界源、族运、战损和飞升者价值写入补墟外围账目 |
| 活跃卷段 | 第六卷到第九卷 |
| 前中段代表 | 明厌 |
| 抽源账代表 | 俞简 |
| 外部协作 | 猎血商队、大族联系人、百族盟场、登仙台验源投影 |
| 后续接入 | 仙界登仙监、补墟司外账房、补墟司血账和界账 |

灵界监察链不公开说自己替仙界抽血。它的公开话术是“防止百族失控、归墟污染扩散和族契崩坏”。

## 3. 总链路

灵界监察链不是一个地点，而是一串空间和权限节点。

```text
落界边城监察驿
    ↓
散血市血名册暗点
    ↓
猎血商路许可栈
    ↓
百族盟场族契裁断台
    ↓
残界带战损登记处
    ↓
抽源账站
    ↓
登仙台残路验源口
    ↓
仙界补墟司外账房 / 登仙监 / 新司命台接口
```

| 节点 | 表层功能 | 实际功能 |
|------|----------|----------|
| 落界边城监察驿 | 登记外来修士、百族冲突、血脉异常 | 标记新入灵界变量和高价值血脉 |
| 散血市血名册暗点 | 记录混血、逃族者和血脉伤 | 收集血样、假族契、逃族路线 |
| 猎血商路许可栈 | 管控商队和危险货物 | 给名单内追猎提供合法通行 |
| 百族盟场族契裁断台 | 裁断族契争议 | 把强迫献血写成族群自愿或赔偿 |
| 残界带战损登记处 | 记录战争和残界损耗 | 将死亡、祖地衰败、界源缺口拆账 |
| 抽源账站 | 汇总战损、血税、界源额度 | 把灵界损耗转成补墟外围额度 |
| 登仙台残路验源口 | 检测渡劫者和飞升者 | 将血、运、界、魂等价值接入仙界流程 |

## 4. 监察名义与实际功能

| 表层名义 | 对外说法 | 实际作用 |
|----------|----------|----------|
| 血脉登记 | 防止血脉暴走和猎血私斗 | 标记真血、古血、源血钥匙 |
| 族契调停 | 防止百族战争扩大 | 让弱族献血和交人合法化 |
| 战损记录 | 防止虚报战功和损耗 | 将血源、魂源、界源折算成补墟额度 |
| 归墟污染防控 | 防止残界污染扩散 | 把被污染地区列入抽源或封存名单 |
| 飞升资格复核 | 避免异常修士冲击登仙台 | 提前筛选高价值飞升对象 |

监察链真正可怕的地方不是秘密杀人，而是让受害者在账面上变成合理损耗。

## 5. 主要职位

| 职位 | 代表 | 权限 | 剧情功能 |
|------|------|------|----------|
| 边城监察使 | 明厌前期可兼 | 登记异常、调动驿吏、封存局部证据 | 第六卷压迫入口 |
| 血名册执笔 | 监察链低阶文书 | 记录血脉、逃族者、血样编号 | 让猎血有名单来源 |
| 商路许可官 | 猎血商路暗点人员 | 发放通行、扣押货物、转交许可 | 合法化追猎和押送 |
| 族契裁定使 | 明厌核心权限之一 | 判定献血、赔偿、交人是否合法 | 第七卷白倩霜族契压力 |
| 战损账官 | 俞简前置或下线 | 汇总战争、残界、祖地损耗 | 第八卷抽源账站入口 |
| 抽源账官 | 俞简 | 折算血、界、魂、运等补墟额度 | 证明万年抽源是流程 |
| 验源接引使 | 登仙台残路投影或残权 | 接收渡劫者血源、界源、命数数据 | 第九卷飞升回收前哨 |

明厌负责“裁定与合法性”，俞简负责“入账与折算”。两者不能写成同一个功能。

## 6. 关键文书

| 文书 | 内容 | 用法 | 可被击穿点 |
|------|------|------|------------|
| 血名册 | 特殊血脉、真血感应、族契裂痕、逃族者路线 | 猎血商队和监察使按名单追索 | 名册只记录制度需要的分类，不记录胁迫 |
| 族契裁定副本 | 某族是否自愿献血、交人、赔偿 | 将猎血合法化 | 可被旧祖契、证人、源血反噬推翻 |
| 猎血许可 | 商队在指定区域追索指定目标的凭证 | 给阮青鳞等人通行和抓捕理由 | 可证明私猎背后有监察默许 |
| 战损入账草表 | 战争死亡、祖地损耗、界源裂缝、血脉折损 | 接入抽源账站 | 草表与真实现场不符 |
| 抽源额度简表 | 血、界、魂、运等可转补墟额度 | 接入仙界外围账口 | 投入越多，灵界越衰败 |
| 验源前置记录 | 渡劫者和飞升者的本源标签 | 接入登仙台残路和仙界底层 | 与沈临风系统、谢未明命数证据冲突 |

## 7. 猎血合法化流程

标准流程：

```text
血脉异常被发现
→ 血名册暗点记录
→ 监察使标记为族契风险
→ 大族或商队提出追索
→ 百族盟场或监察驿出具裁定
→ 猎血商队代执
→ 交人、献血或战损入账
→ 抽源账站折算额度
```

以白倩霜为例：

1. 她的返祖真血被标为源血链路钥匙。
2. 霜羽族祖契裂痕被写成族群风险。
3. 保守长老的交人倾向被监察链包装成“族内自愿”。
4. 猎血商队获得追索许可。
5. 玄鳞族等大族可用代执名义接手。
6. 若成功，白倩霜不会被写成被卖，而会被写成“族契争议核心已移交”。

主角组要击穿的是这整条流程，而不是只杀一个抓人的商队头目。

## 8. 战损与抽源流程

第八卷万年抽源前后，监察链进入更深层功能。

```text
百族战争
→ 战损登记
→ 祖地衰败归因
→ 血源、界源、魂源拆账
→ 抽源账站折算
→ 仙界外围账口接收
→ 登仙台残路验源复核
```

常见伪账：

| 伪账名义 | 真相 |
|----------|------|
| 自然祖地衰败 | 长期抽源和猎血术污染 |
| 百族内乱战损 | 仙界监察推动大族和小族互耗 |
| 族契自愿赔偿 | 弱族被迫献血、交人或献祖地资源 |
| 残界污染封存 | 抽界源后留下的病灶被归咎归墟污染 |
| 飞升资格异常 | 高价值本源被提前标记 |

俞简的作用在这里：他不必亲手杀人，只需要把所有损耗折成可上报的额度。

## 9. 权限与限制

| 权限 | 用法 | 限制 |
|------|------|------|
| 监察法印 | 标记族契异常、封存证据、调动驿吏 | 离开监察链节点后威力下降 |
| 血名册调用 | 查询血脉、逃族者、族契风险 | 名册可能被伪造、删改或反向追踪 |
| 族契裁定 | 将交人、赔偿、献血写成合法 | 旧祖契、源血反噬和公开证据可推翻 |
| 商路封锁 | 要求货栈、驿站、商队配合追索 | 大族和黑市会绕开或互相利用 |
| 战损入账 | 将死亡和祖地损耗归类 | 入账越多，越暴露制度链 |
| 验源上报 | 将高价值变量传往登仙台或仙界 | 赵长生、沈临风、谢未明会造成异常 |

监察链不是全知全能。

它的弱点：

1. 依赖文书、印信和节点传递。
2. 低层监察使未必知道上层黑账。
3. 名册只记录分类，不记录完整真相。
4. 猎血商队和大族会私吞、伪报、灭口。
5. 赵长生可从残器、血契、猎血器具中还原部分证据。
6. 谢未明可追出命数污染痕迹。
7. 沈临风系统任务可能与监察名单发生后台冲突。

|------|--------------|----------|

| 沈临风 | 系统任务可能读取或避开监察名单 | 发现系统奖励和血名册有后台接口 |
| 白倩霜 | 被列为族契争议核心和高价值真血 | 击穿“有账即合法”的核心人物 |

## 11. 卷际使用

| 卷 | 使用方式 |
|----|----------|
| 第六卷 | 落界边城监察驿和血名册暗点出现，白倩霜被标记，明厌投下制度阴影 |
| 第七卷 | 百族盟场族契裁定，霜羽族交人压力公开化，明厌压迫白倩霜 |
| 第八卷 | 残界带战损登记和抽源账站展开，俞简将血账、界账、魂账拆分入账 |
| 第九卷 | 登仙台残路验源口接收灵界证据，飞升回收真相与血源筛选合流 |
| 第十卷以后 | 纪晚凝从补墟司外账房和血账记录中看到灵界监察链回执 |

| 证据 | 作用 | 后续用途 |
|------|------|----------|
| 血名册残页 | 证明白倩霜、霜羽族、散血民被分类定价 | 白倩霜血账线 |
| 猎血许可 | 证明猎血商队不是私猎，而是名单内代执 | 击穿阮青鳞和大族说辞 |
| 族契裁定副本 | 证明所谓自愿交人有监察和大族压力 | 百族契约重订 |
| 战损入账草表 | 证明死亡、祖地衰败、残界崩解被拆账 | 第八卷万年抽源 |
| 抽源额度简表 | 证明灵界损耗接入仙界补墟额度 | 纪晚凝补墟账本线 |
| 验源前置记录 | 证明登仙台残路早已筛选血、界、运等价值 | 第九卷飞升回收 |

## 13. 退场与回收

监察链不能因明厌失败而消失。

可用回收方式：

1. 明厌失去血名册和监察法印，暴露前中段链路。
2. 俞简接手抽源账站，说明更深层入账还在继续。
3. 猎血商队被击败后，大族改用族契裁定和验源通道继续追索。
4. 第九卷登仙台残路把灵界监察记录接入仙界底层。
5. 第十一卷以后，纪晚凝在补墟司外账和内账中看到灵界回执。

监察链的阶段性失败应该带来证据和权限裂缝，而不是让灵界制度立刻崩掉。

## 14. 写作禁忌

1. 不能把灵界监察链写成公开邪恶组织。
2. 不能让明厌知道完整墟眼和补墟黑账真相。
3. 不能让猎血问题都归咎于一个商队或一个大族。
4. 不能让血名册变成万能地图，名册也有遗漏、伪造和权限遮蔽。
5. 不能让主角组拿到一页账就说服百族。
6. 不能让监察链全员高智全能，低层执行者要有岗位局限。
7. 不能把族契裁定写成纯法律文书，它必须和祖地、源血、族运相连。
8. 不能让赵长生用归墟炉直接抹除所有猎血污染。
9. 不能让沈临风系统准确解释监察链，它只能提供局部异常和任务冲突。

## 15. 使用提示

写灵界监察链时，每次出场都要带一个具体动作：

1. 一个名字被写进血名册。
2. 一个族契被裁定。
3. 一个商队获得追索许可。
4. 一场战争被拆成战损。
5. 一处祖地衰败被写成自然损耗。
6. 一个飞升者在登仙台前被验源标记。

这条线的核心证明是：

> 灵界猎血不是暗处恶事，而是从名单、印信、商路、裁定、战损到验源的一整条账本制度。
