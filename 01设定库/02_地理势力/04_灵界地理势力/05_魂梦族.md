# 魂梦族

> 灵界势力补档。本文用于承载魂梦族、魂梦雾泽、祖魂祭、残魂证言、魂源污染和轮回灯伏笔。本文不替代洛闻霜和轮回灯设定，也不能让魂梦族承担复活功能。

---

## 1. 设计目的

魂梦族补档要解决五个问题：

1. 灵界篇如何提前埋魂源和轮回灯伏笔，但不抢仙界篇洛闻霜功能。
2. 飞升失败者残魂、百族战争亡魂和猎血受害者证言如何有可信保存方式。
3. 祖魂祭为什么既能保护族群记忆，也可能被监察链和补墟账本利用。
4. 第七卷到第九卷如何让“魂账”从气氛伏笔变成证据链。
5. 赵长生聚魂能力如何被限制，避免变成随意复活。

魂梦族的核心不是“梦境神秘族”，而是：

> 一个守着残梦和祖魂的族群，能让死者留下证言，却不能替死者回到人间。

## 2. 基础定位

| 项目 | 内容 |
|------|------|
| 族名 | 魂梦族 |
| 类别 | 灵界魂族分支，祖魂和梦境亲和 |
| 祖地 | 魂梦雾泽 |
| 血脉特征 | 梦雾感知、残魂安置、祖魂祭、记忆碎片保存 |
| 当前状态 | 保持中立，避开百族战争，但已被魂源账和轮回异常波及 |
| 核心资源 | 祖魂祭坛、魂灯渡口、残梦石林、梦雾外泽 |
| 主要人物 | 梦婆罗 |
| 剧情卷段 | 第七卷伏笔，第八卷魂源入账，第九卷残魂证言 |

魂梦族不是鬼修族，也不是亡灵法师族。它们的职责更接近守证、送魂、保留族群梦记。

## 3. 魂梦雾泽

魂梦雾泽位于灵界百族地图中部偏幽静的水泽区域。

```text
梦雾外泽
    ↓
残梦石林
    ↓
魂灯渡口
    ↓
祖魂祭坛
    ↓
梦婆罗居所
    ↓
残魂寄留地
```

| 空间 | 功能 | 剧情用途 |
|------|------|----------|
| 梦雾外泽 | 迷雾、梦影、魂息混杂 | 主角组第一次接触魂梦族，误判真假 |
| 残梦石林 | 保存百族战争残梦和旧证言 | 猎血、抽源、飞升失败伏笔 |
| 魂灯渡口 | 安置短暂停留的残魂 | 证明残魂不是复活对象 |
| 祖魂祭坛 | 族群记忆和祖魂祭核心 | 祖魂祭、魂源稳定、族运记录 |
| 梦婆罗居所 | 魂梦族祭师居处 | 第七到九卷梦婆罗出场点 |
| 残魂寄留地 | 飞升失败者或战死者残魂短留处 | 第九卷残魂证言前置 |

魂梦雾泽要写成边界空间：生者能听见死者余音，但不能把死者拉回生者世界。

## 4. 魂梦族能力

| 能力 | 用法 | 限制 |
|------|------|------|
| 梦雾感知 | 感应残魂、梦影、魂伤和记忆碎片 | 只能感知片段，不等于全知 |
| 祖魂祭 | 让族中重要记忆短暂回响 | 需要祭坛、族人、代价，不能随意召魂 |
| 残梦保存 | 保存战争、猎血、飞升失败残影 | 残梦会破碎、失真、被污染 |
| 魂灯引路 | 让迷失残魂离开梦雾 | 不是轮回灯，只是低阶引路 |
| 魂源辨污 | 分辨魂源被截留、磨损、污染的痕迹 | 不能净化完整魂账 |

魂梦族不能复活死人。

它们能做的是：

1. 留住短暂证言。
2. 保护残魂不被立刻吞噬。
3. 让生者知道亡魂去了哪里。
4. 提醒赵长生：聚魂不是复活。

## 5. 祖魂祭

祖魂祭是魂梦族与祖地、族史和残魂边界相连的仪式。

正常作用：

1. 安置战死族人残念。
2. 保存族群关键记忆。
3. 在重大危机前让祖魂发出警兆。
4. 让年轻族人知道过去战争和迁徙代价。
5. 防止亡魂被邪修或补墟魂账轻易截走。

风险：

1. 祖魂祭被过度使用，会损耗祭师神魂。
2. 残梦可能被监察链或魂修篡改。
3. 魂源污染会让祖魂祭变成噩梦回响。
4. 新仙界若发现魂梦族能保存证言，会将其列为魂账风险。

祖魂祭不是万能档案馆。证言必须残缺、片段化、有代价。

## 6. 梦婆罗

梦婆罗是魂梦族祭师，灵界篇魂源伏笔人物。

| 项目 | 内容 |
|------|------|
| 身份 | 魂梦族祭师，守祖魂和残梦 |
| 首次重点 | 第七卷魂梦雾泽支线或百族与真灵阶段 |
| 表层动机 | 避免魂梦族卷入百族战争 |
| 深层职责 | 守住残魂边界，不让亡魂被猎血、抽源和补墟账吞掉 |
| 功能 | 为第九卷飞升失败者残魂证言提供前置 |
| 边界 | 不能直接解释轮回灯，不能成为洛闻霜替代品 |

梦婆罗可以帮助主角组，但她必须谨慎。

她担心：

1. 主角组把残魂证言当武器使用。
2. 魂梦族暴露后被仙界盯上。
3. 赵长生误以为归墟炉聚魂能复活死人。
4. 白倩霜血账线会把魂梦族也拖入百族战火。

## 7. 魂源与补墟账

魂梦族最重要的后期功能是让魂账提前有迹可循。

魂源异常表现：

1. 战死者残魂消失得太干净。
2. 祖魂祭听不到某些已死族人的梦回。
3. 飞升失败者没有正常入梦，也没有散归归墟。
4. 梦雾中出现被截断的魂火痕。
5. 魂灯渡口有空灯，灯芯却像被抽走。

这些现象后期接入：

- 第九卷飞升失败者残魂证言。
- 第十二卷洛闻霜和轮回灯。
- 第十三卷藏烬君的魂账磨损录。
- 第十四卷重定亡魂归处。

## 8. 与灵界监察链关系

魂梦族表面中立，但无法完全避开监察链。

监察链关注魂梦族的原因：

1. 魂梦族能保存战损真实证言。
2. 祖魂祭能证明某些死亡并非自然战损。
3. 残魂寄留地可能保存飞升回收证据。
4. 魂源异常会暴露补墟魂账。

监察链可能采取：

- 要求登记祖魂祭。
- 以归墟污染风险封存残梦石林。
- 派账官核查战损魂数。
- 将魂梦族列为魂源异常族群。

魂梦族不宜直接正面对抗监察链。它们更适合隐藏证言、拖延登记、保护少量残魂。

| 白倩霜 | 血账证据与魂账证据互补 | 百族战争不是只有血脉伤，也有亡魂旧账 |
| 叶疏彤 | 生机线与亡魂边界对照 | 让生机修复不能越过生死边界 |

| 洛闻霜 | 后期轮回灯正主 | 魂梦族只做伏笔和证人，不替代洛闻霜 |

| 卷 | 用法 |

## 11. 可提供的关键证据

| 证据 | 来源 | 后续用途 |
|------|------|----------|
| 残梦石林影像 | 百族战争、猎血、飞升失败残影 | 证明战损账被改写 |
| 魂灯空灯 | 魂灯渡口 | 证明亡魂被截走而非正常散去 |
| 祖魂祭断声 | 祖魂祭坛 | 证明某些死亡无法归祖 |
| 飞升失败者残魂片段 | 残魂寄留地 | 第九卷飞升回收证据 |
| 梦婆罗证言 | 祭师记忆和仪式记录 | 第十二卷轮回灯线前置 |
| 魂源污染痕 | 梦雾、残魂、祭坛 | 第十三卷魂账磨损录 |

## 12. 退场与回收

魂梦族不需要被彻底解决。

合理回收：

1. 梦婆罗提供一次关键残魂证言后退回魂梦雾泽。
2. 魂梦族不能跟随主角组长期行动，以免抢洛闻霜功能。
3. 第九卷后，魂梦族提供证据但不解释完整轮回灯。
4. 第十三卷魂账公开时，魂梦族证言成为一条佐证。
5. 终局后，魂梦族恢复守证和安魂职责，而不是掌控轮回。

## 13. 写作禁忌

1. 不能让魂梦族复活死人。
2. 不能让梦婆罗直接解释轮回灯全貌。
3. 不能让魂梦族成为随队神秘外挂。
4. 不能让残魂证言完整、清晰、无代价。
5. 不能让赵长生通过归墟炉随便聚魂问话。
6. 不能让魂梦族替洛闻霜完成生死边界功能。
7. 不能把梦境写成万能幻术战斗体系。

## 14. 使用提示

写魂梦族时，始终抓住三句话：

1. 残魂能作证，但不能复活。
2. 梦能保存片段，但不能代替真相。
3. 亡魂有归处，不应被写成补墟账里的材料。

魂梦族的存在是为了给灵界篇提前埋下这个问题：

> 如果飞升失败者连尸骨和残魂都没有留下，他们到底去了哪里？
