# 玄鳞族

> 灵界势力补档。本文用于承载玄鳞族的大族地位、水域封锁、猎血制度、豁免逻辑、内部派系和玄鳞大泽空间。本文不是玄烬个人单档，也不是第七卷战斗章纲。

---

## 1. 设计目的

玄鳞族补档要解决五个问题：

1. 灵界大族为什么会参与猎血，而不是单纯邪恶。
2. 白倩霜和霜羽族面对的大族压力如何具体落地。
3. 百族战争中“大族换豁免、小族被牺牲”的逻辑如何成立。
4. 玄鳞族内部为什么不能写成铁板一块。
5. 第七卷玄鳞大泽如何作为水域、血脉封库、商路和监察暗点的综合舞台。

玄鳞族的核心不是“坏大族”，而是：

> 一个有能力自保的大族，在仙界抽源压力下，把弱族血脉写成换取豁免的筹码。

## 2. 基础定位

| 项目 | 内容 |
|------|------|
| 族名 | 玄鳞族 |
| 类别 | 灵界鳞族大族，控制水域和商路 |
| 祖地 | 玄鳞大泽 |
| 血脉特征 | 水脉、鳞甲、防御、血脉封存、战船水阵 |
| 当前地位 | 灵界大族之一，掌控大泽、水门、猎血审讯水牢和水路商税 |
| 核心资源 | 水脉祭坛、血脉封库、鳞甲战船坞、水牢、监察水路暗点 |
| 主要矛盾 | 以牺牲弱族血脉换取本族抽源豁免，内部也有人反对 |
| 剧情卷段 | 第六卷伏线，第七卷核心压力，第八卷豁免逻辑回收 |

玄鳞族不是灵界最高统治者，但它在水域和商路上足够强，使小族绕不开它。

## 3. 玄鳞大泽

玄鳞大泽是玄鳞族祖地和权力中心。

```text
猎血商路水门
    ↓
水门关
    ↓
鳞甲战船坞 —— 监察水路暗点
    ↓
外泽交易泊
    ↓
血脉封库 —— 猎血审讯水牢
    ↓
水脉祭坛
    ↓
玄鳞议事水殿
```

| 空间 | 功能 | 剧情用途 |
|------|------|----------|
| 猎血商路水门 | 控制商路、水路和追猎通行 | 白倩霜、霜羽族外道被封锁 |
| 水门关 | 税关、验货、扣押逃族者 | 展示大族合法盘查 |
| 鳞甲战船坞 | 玄鳞族战船和水阵集结地 | 第七卷水域追击和军力压迫 |
| 外泽交易泊 | 合法货物和灰色血样交易 | 猎血经济不是地下小摊 |
| 监察水路暗点 | 灵界监察链在玄鳞大泽的接口 | 明厌、血名册、商路许可接入 |
| 血脉封库 | 保存血样、族契副本、血脉器具 | 血账证据和玄鳞族罪证 |
| 猎血审讯水牢 | 审讯逃族者和真血目标 | 玄烬等猎血者施压空间 |
| 水脉祭坛 | 玄鳞族祖地和水脉核心 | 证明玄鳞族自身也怕抽源 |
| 玄鳞议事水殿 | 族中派系议事处 | 展示强硬派、自保派、反猎血少数派冲突 |

玄鳞大泽不是反派巢穴。它应有族人生活、商路秩序、祭祀、修行、守卫和水脉病灶。

## 4. 血脉与水脉

玄鳞族血脉以水脉和鳞甲为核心。

| 血脉层级 | 表现 | 用途 | 风险 |
|----------|------|------|------|
| 普通鳞血 | 水中呼吸、鳞甲防护、耐水压 | 水域劳作、低阶战修 | 抽源后鳞甲变脆 |
| 灵鳞血 | 可引水脉灵机，操控小型水阵 | 战船、巡逻、商路控制 | 易被水脉祭坛绑定 |
| 古鳞血 | 祖鳞纹、血脉封存术 | 血脉封库和族契水印 | 容易执迷大族纯血 |
| 玄鳞真血 | 大泽水脉共鸣、鳞甲法相 | 族中高阶战力和祭坛守护 | 是仙界监察豁免谈判筹码 |

玄鳞族自身也有源血和祖地压力。

这点很重要：玄鳞族不是因为无聊才猎血，而是知道如果本族被列入抽源异常，整个大泽水脉也会受损。它的错误在于把这种压力转嫁给更弱的小族。

## 5. 豁免逻辑

玄鳞族参与猎血的核心不是个人贪婪，而是“用弱族资源换本族短期豁免”。

标准逻辑：

1. 灵界监察链给大族设定抽源额度。
2. 大族若能提供特殊血源、界源或战损额度，可换取本族祖地缓征。
3. 小族真血和源血链路钥匙被标为高价值目标。
4. 玄鳞族通过猎血商队、水门封锁和族契裁定获取目标。
5. 监察链将其写成百族争端、族契赔偿或合法代执。
6. 玄鳞族获得短期豁免，弱族祖地继续衰败。

这条逻辑必须让读者觉得现实、冷酷、可怕，而不是蠢坏。

## 6. 内部派系

玄鳞族内部至少有三派。

| 派系 | 代表 | 主张 | 问题 |
|------|------|------|------|
| 猎血强硬派 | 玄烬 | 主动猎取小族真血，换取本族豁免和水脉稳定 | 将弱族当耗材，最直接压迫白倩霜 |
| 现实自保派 | 族中多数长老 | 不喜猎血，但认为不献弱族就轮到本族 | 用现实恐惧维护坏制度 |
| 反猎血少数派 | 可设大族叛逃者或年轻水阵师 | 认为今日献小族，明日本族也会被账本吞掉 | 权限弱，容易被视为叛族 |

玄烬可作为第七卷主战斗反派，但他不能代表全族全部复杂性。

现实自保派更危险：他们未必亲自下水牢，却会在议事水殿里签字。

## 7. 玄烬

玄烬是玄鳞族猎血强硬派代表。

| 项目 | 内容 |
|------|------|
| 身份 | 玄鳞族猎血者、水门封锁和追索执行人 |
| 动机 | 猎取白倩霜等真血目标，换取本族水脉豁免 |
| 功能 | 让白倩霜线面对大族武力、商路封锁和血脉审讯 |
| 边界 | 可作为阶段战斗反派，不宜承载全部制度罪责 |

玄烬不能写成色欲反派。

他可怕在于：他真心认为小族的痛苦比不上玄鳞大泽千万族人的水脉安全。

## 8. 与霜羽族关系

玄鳞族对白倩霜和霜羽族的策略：

1. 先通过血名册确认白倩霜是真血目标。
2. 再通过猎血商队和水门关掌握她的逃亡路线。
3. 然后推动霜羽族内部保守派承认“交出她可换短期平安”。
4. 最后以代执、审问或保护名义将她带入玄鳞大泽。

玄鳞族不一定直接屠灭霜羽族。

更符合本书气质的压迫是：

- 封锁外道。
- 断掉药材和水路交易。
- 扣押霜羽族逃散者。
- 借监察链催促族契裁定。
- 宣称只要交出白倩霜，霜羽族可暂缓被列入抽源异常。

这比单纯派兵灭族更制度化，也更能逼白倩霜做选择。

## 9. 与灵界监察链关系

玄鳞族不是监察链下属，但双方互相利用。

| 玄鳞族提供 | 监察链提供 |
|------------|------------|
| 水路封锁、战船、审讯水牢、血样收集 | 血名册、族契裁定、猎血许可、豁免记录 |
| 大族武力和商路控制 | 合法性和入账渠道 |
| 小族和逃族者信息 | 补墟额度和短期缓征 |

双方冲突：

1. 玄鳞族想私吞部分血脉收益。
2. 监察链要把血源、界源和战损写进账。
3. 明厌不允许猎血商队越过名单私吞真血。
4. 俞简会将玄鳞族的豁免也写入更深账册。

玄鳞族以为自己在利用制度，其实也在被制度记录。

## 10. 与核心人物关系

| 人物 | 关系 | 作用 |
|------|------|------|
| 白倩霜 | 高价值真血目标 | 逼她证明自己不是族群耗材 |
| 赵长生 | 破坏猎血器具和还原血账证据者 | 让玄鳞族意识到废料路线能反查罪证 |
| 沈临风 | 系统可能将玄鳞任务标成高价值机缘 | 冲击他对任务奖励的认知 |
| 叶疏彤 | 能看出水脉和祖地病灶 | 证明大族自身也在被抽源恐惧逼迫 |
| 陆维舟 | 可识破水阵和水门封锁 | 给主角组破局提供技术路径 |
| 谢未明 | 能追踪豁免账和命数污染 | 证明大族豁免不是天机自然结果 |
| 明厌 | 监察链合作又互相防备 | 制度合法性和大族执行力结合 |

| 卷 | 用法 |
|----|------|
| 第六卷 | 通过猎血商队、血样交易和水门传闻埋伏玄鳞族 |
| 第七卷 | 玄鳞大泽、水门封锁、血脉封库、玄烬登场，形成白倩霜线主要外压 |

| 证据 | 来源 | 后续用途 |
|------|------|----------|
| 水门追索令 | 水门关、监察暗点 | 证明白倩霜被追不是私怨 |
| 血脉封库记录 | 血脉封库 | 证明多族血样长期被收集 |
| 猎血审讯水牢残印 | 水牢 | 证明逃族者和真血目标被审讯 |
| 豁免协议副本 | 议事水殿或监察暗点 | 证明大族用弱族血脉换缓征 |
| 水脉祭坛病灶 | 水脉祭坛 | 证明玄鳞族自身也怕抽源，不是单纯贪婪 |
| 玄烬私令 | 猎血强硬派 | 证明强硬派越界私猎，但不能替制度背全部罪 |

## 13. 退场与回收

玄烬可以败，但玄鳞族不能因此完全解决。

合理回收：

1. 玄烬败亡或被族内抛弃，留下血脉封库证据。
2. 现实自保派承认猎血链存在，但仍不愿公开豁免协议。
3. 反猎血少数派提供一段水路证据，但不足以推翻全族。
4. 第八卷抽源账站证明玄鳞族豁免也是补墟账本的一部分。
5. 终局重订百族契约时，玄鳞族必须面对自己曾转嫁给弱族的旧债。

## 14. 写作禁忌

1. 不能把玄鳞族写成全族邪恶。
2. 不能把玄烬写成色欲反派。
3. 不能让大族豁免逻辑变成口号，必须落到水门、封库、祭坛和账本。
4. 不能让白倩霜被抓后只等赵长生救，她必须通过族契和血源证据主动反击。
5. 不能让玄鳞族被一个战斗反派带崩，制度问题要延续到第八卷。
6. 不能让玄鳞族成为终局统治者或简单投诚对象。

## 15. 使用提示

写玄鳞族时抓三层：

1. 武力层：水门、战船、水牢、玄烬。
2. 制度层：血名册、猎血许可、豁免协议。
3. 恐惧层：玄鳞族自身也怕大泽水脉被抽源。

它的存在是为了证明：

> 大族不是没有恐惧，而是它们更有能力把恐惧转嫁给弱族。
