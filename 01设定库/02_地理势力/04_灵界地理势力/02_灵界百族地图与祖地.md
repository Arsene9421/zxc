# 灵界百族地图与祖地

> 地图地理补档。本文用于承载第六卷到第九卷的灵界底层、百族分布、霜羽族祖地、猎血商路、灵界监察链、残界带、万年抽源节点和登仙台残路。本文不是百族势力名录，也不是白倩霜单人档。

---

## 1. 设计目的

灵界地图要解决七个问题：

1. 主角组从下界进入灵界后，为什么仍要从底层重新求生，而不是立刻成为高等世界贵客。
2. 灵界百族不是简单“妖族皮肤”，而是有祖地、族契、族运、商路、战区和监察链的社会结构。
3. 白倩霜为什么不能只被救走，她的困境必须落在霜羽族祖地、源血链路和族契裂痕上。
4. 猎血制度为什么是空间化产业链：标记、追猎、交易、合法化、入账各有地点。
5. 万年抽源为什么不是一句设定，而会表现为祖地衰败、残界崩解、账站入册和战争引导。
6. 灵界监察使、账官、猎血商队、大族、小族、混血群体各自在地图上有行动范围。
7. 第九卷登仙台残路必须从灵界空间自然长出来，而不是临时出现一个飞升副本。

灵界不是资源更富的下界，也不是单纯换怪升级。它的压迫来自：

> 资源更多，垄断更深；族群更多，账本更细；天空更高，飞升的门却更像筛选口。

## 2. 总地图

灵界可先按“边城、商路、百族祖地、大族控制区、残界带、监察链、登仙台残路”七层展开。

```text
                              登仙台残路 / 洗源投影
                                         ↑
                              北天残阶 / 渡劫荒台
                                         ↑
                     残界带 —— 界源战场 —— 抽源账站
                        ↑             ↑             ↑
          魂梦雾泽 —— 百族盟场 —— 山魁地脉 —— 监察链中枢
                        ↑             ↑
        霜羽寒原 —— 猎血商路 —— 玄鳞大泽
             ↑             ↑             ↑
        霜羽祖巢       散血市        苍羽天岭
             ↑             ↑
                    落界边城
                         ↑
                  下界残破登仙路落点
```

核心分区：

| 区域 | 首次重点卷 | 控制者 | 作用 |
|------|------------|--------|------|
| 落界边城 | 第六卷 | 人族商盟、散血民、监察驿、猎血商队 | 换地图后底层生存、白倩霜登场 |
| 散血市 | 第六卷 | 混血群体、黑市商队、灰色医修 | 证明血脉等级制度荒谬 |
| 猎血商路 | 第六到第七卷 | 阮青鳞、商队、玄鳞族下线、监察耳目 | 追捕、交易、消息泄露 |
| 霜羽寒原/霜羽祖巢 | 第七卷 | 霜羽族、白曜、白疏影 | 白倩霜族群责任和源血链路 |
| 玄鳞大泽 | 第七卷 | 玄鳞族、玄烬等猎血者 | 大族压迫和水域封锁 |
| 苍羽天岭 | 第七卷 | 苍羽族、绛羽等侦察者 | 空路、百族横向盟友 |
| 魂梦雾泽 | 第七到第九卷 | 魂梦族、梦婆罗 | 祖魂祭、残魂证言、轮回灯伏笔 |
| 山魁地脉 | 第八卷 | 山魁族、石灵匠修、界源矿修 | 界源、地脉、抽源伤害 |
| 残界带 | 第八卷 | 残界守界人、流民、监察账官 | 修界线、界源战场、灵界据点 |
| 抽源账站 | 第八卷 | 俞简、灵界监察链、仙界下线 | 万年抽源入账 |
| 登仙台残路 | 第九卷 | 衡阙使、验源投影、残台旧役 | 飞升筛选、回收证据、进入仙界底层 |

## 3. 灵界空间规则

### 3.1 灵界不等于资源天堂

灵界资源确实比下界丰富。

但资源分布遵循三条规则：

1. 核心灵脉和祖地被大族、古道统、监察链把持。
2. 小族和散修能接触到的是边缘资源、污染资源、废弃残界和高风险商路。
3. 飞升、渡劫、祖地、源血和界源相关资源都被记录、标记、入账。

所以主角组进入灵界后仍要从底层开始。不是因为他们变弱，而是他们面对的是更完整的垄断体系。

### 3.2 祖地不是风景

祖地是族群、族契、源血和族运的空间节点。

祖地至少包含：

1. 族群居住区。
2. 祭坛或祖契台。
3. 源血链路节点。
4. 族史碑。
5. 族中幼弱和病弱者安置地。
6. 族契裁断处。
7. 通往外界商路或逃亡路线。

祖地被抽源后的表现：

1. 灵泉变浊。
2. 新生儿血脉变薄。
3. 族契回应变慢。
4. 祖地防护阵出现裂纹。
5. 族中保守派更倾向牺牲个体换短期平安。

### 3.3 灵界监察链

灵界监察链不是一个地点，而是一串节点。

```text
落界边城监察驿
→ 猎血商路血名册暗点
→ 百族盟场族契裁断台
→ 残界带战损登记处
→ 抽源账站
→ 登仙台残路验源口
```

作用：

1. 标记特殊血脉。
2. 记录族契异常。
3. 将战争、追猎、抽源写成合法损耗。
4. 把灵界血账、界账和飞升回收接到仙界补墟账本。

明厌适合作为监察链的前中段代表；俞简适合作为抽源账站和战损入账代表。

## 4. 第六卷空间：落界边城

落界边城是主角组从下界残破登仙路进入灵界后的初始区域。

它不是繁华仙城，而是靠近下界残路、商路、散血市和猎血路线的边缘城。

组成：

1. 下界残路落点。
2. 散修棚区。
3. 百族杂货街。
4. 散血市入口。
5. 猎血商队货栈。
6. 人族客栈和功法馆。
7. 灵界监察驿。
8. 低阶医馆。
9. 临时斗法台。

作用：

1. 展示灵界人族不是天然统治者。
2. 展示百族混居、等级、交易和歧视。
3. 让白倩霜以被追猎者身份出现。
4. 让赵长生适应灵界灵机、界压和资源价格。
5. 让沈临风系统第一次对灵界血脉目标显示异常价值。

禁写：

1. 不要让主角组刚入灵界就住进大族祖地。
2. 不要让边城全员妖族，必须有人族、散血民、商队、医修、监察驿。
3. 不要让白倩霜出场就是求救，她应先戒备、试探、误导。

## 5. 散血市

散血市位于落界边城背街，是混血群体、低阶医修、血脉鉴定师、黑市商队和逃族者聚集的灰色市场。

这里不是纯邪恶黑市。

它的存在说明：

1. 血脉等级制度之外有大量活人。
2. 混血者不被大族承认，却最懂各族夹缝。
3. 猎血商队、医修、逃族者和监察耳目都在这里找消息。

可写地点：

| 地点 | 功能 | 剧情用途 |
|------|------|----------|
| 血脉鉴定棚 | 鉴定血脉、真伪、污染 | 白倩霜隐藏真血，赵长生看见血脉市场 |
| 低阶医馆 | 治血脉伤、猎血术后遗症 | 桑禾或木灵医者线接入 |
| 混血客栈 | 散血民聚集 | 纪流火做向导和掮客 |
| 黑血货栈 | 违法血样、假族契、逃族路线 | 阮青鳞线索 |
| 旧族契摊 | 残破族契、祖地地图碎片 | 白倩霜祖地线索 |

散血市的规则：

1. 每个人都可能卖消息，但不是每个人都愿意卖人。
2. 血脉高低在这里仍有价格，但不等于尊严。
3. 赵长生不能通过归墟炉随便清洗所有血脉伤。

## 6. 猎血商路

猎血商路连接落界边城、玄鳞大泽、散血市、霜羽寒原和百族盟场。

表面货物：

1. 灵草。
2. 兽骨。
3. 丹材。
4. 矿石。
5. 族地特产。

暗中货物：

1. 血样。
2. 假族契。
3. 逃族者情报。
4. 真血者路线。
5. 被合法化的“族契罪人”。

商路节点：

```text
落界边城
→ 散血市
→ 猎血货栈
→ 霜羽寒原外道
→ 百族盟场
→ 玄鳞大泽水门
```

阮青鳞的功能：

1. 不是色欲反派。
2. 是商队利润、血税名单、监察默许和大族需求共同组成的执行人。
3. 可被击败，但他背后的商路不能因他一死就消失。

## 7. 霜羽寒原与霜羽祖巢

霜羽族是灵界小族，祖地位于霜羽寒原深处。

霜羽寒原特点：

1. 风雪强。
2. 灵气清冷但日渐稀薄。
3. 空路不稳，地面有冰裂谷。
4. 祖地防护阵有裂痕。
5. 猎血商路可从外道绕入。

霜羽祖巢结构：

```text
寒原外道
   ↓
霜羽外寨 —— 白疏影等年轻族人活动区
   ↓
祖契台 —— 族契裁断处
   ↓
霜羽源泉 —— 源血链路感应点
   ↓
旧翼碑林 —— 族史、真灵残影、失败迁徙记录
   ↓
内巢封雪洞 —— 白倩霜真血共鸣处
```

白倩霜回祖地时必须面对三种压力：

1. 猎血商队和大族追索。
2. 白曜等保守长老想交出她换短期平安。
3. 白疏影等年轻族人想救她，但怕全族因此灭亡。

霜羽祖地的核心问题不是“谁当族长”，而是：

> 一个族群能不能在不牺牲某一个人的前提下，重新修补祖契和源血链路。

## 8. 玄鳞大泽

玄鳞族是灵界大族之一，控制玄鳞大泽及周边水域商路。

玄鳞大泽组成：

1. 水门关。
2. 鳞甲战船坞。
3. 血脉封库。
4. 猎血审讯水牢。
5. 水脉祭坛。
6. 玄鳞族议事水殿。
7. 监察链水路暗点。

玄鳞族不能写成全族邪恶。

内部可有三类人：

1. 猎血强硬派：玄烬等人，认为猎取小族真血是换取豁免的必要代价。
2. 现实自保派：知道仙界监察压迫，但选择牺牲弱族保本族。
3. 反猎血少数派：认为大族这样做迟早也会被账本吞掉。

玄鳞大泽的用途：

1. 展示大族垄断商路和水域。
2. 展示猎血如何被包装成族契审判。
3. 给白倩霜线提供大族压力。
4. 给后续万年抽源提供大族投靠仙界的逻辑。

## 9. 苍羽天岭

苍羽天岭是羽族大族和空路侦察者活动区域。

特点：

1. 山岭高，空路复杂。
2. 风雷灵机强。
3. 适合侦察、传讯、空战。
4. 对霜羽族这种小羽族有天然上位压力。

绛羽可从这里进入。

苍羽天岭的功能：

1. 给主角组提供空路情报，但不是免费帮助。
2. 展示羽族内部也有大族与小族差异。
3. 让百族战争不是地面两军对冲，而有空路、商路、水路和祖地多线。
4. 为后续太虚舟、空路和万界航路埋一点远期气质，但不提前解释旧天庭。

## 10. 魂梦雾泽

魂梦雾泽是魂梦族祖地附近的梦雾水泽。

组成：

1. 梦雾外泽。
2. 祖魂祭坛。
3. 残梦石林。
4. 魂灯渡口。
5. 梦婆罗居所。
6. 残魂寄留地。

用途：

1. 第七卷埋魂源和轮回灯伏笔。
2. 第九卷引导飞升失败者残魂作证。
3. 让赵长生明白残魂影像不等于复活。
4. 给洛闻霜后期轮回灯线留空间回声。

禁写：

1. 不能让魂梦族随便复活死人。
2. 不能让梦婆罗直接解释轮回灯。
3. 不能用魂梦雾泽抢洛闻霜的后期功能。

## 11. 山魁地脉与石灵脉区

山魁地脉位于灵界中部偏北，是山灵族、石灵族和界源矿修活动区域。

组成：

1. 地脉城。
2. 石灵工坊。
3. 界源矿裂。
4. 镇脉碑群。
5. 山魁族祖坛。
6. 灵界监察采源点。

功能：

1. 让界源不是抽象概念，而是地脉、矿裂、祖坛和残界共同组成的根。
2. 石砚山等人物可在这里为陆维舟界阵线提供对照。
3. 万年抽源前，地脉会先出现裂纹、矿脉哑火、山灵沉睡等征兆。
4. 仙界监察会把这些写成“自然衰败”或“战损”，而不是抽源后果。

## 12. 残界带

残界带是灵界第八卷核心空间之一。

它不是完整世界，而是许多小世界、洞天碎片、界核残片、战后断界和旧阵封存区组成的危险地带。

地貌：

1. 浮空残岛。
2. 断裂界壁。
3. 残界流民营。
4. 界核碎光。
5. 失重裂谷。
6. 残阵桥。
7. 守界人小庙。

余照适合在这里出场。

残界带用途：

1. 让赵长生从“修物”真正进入“修界”。
2. 让陆维舟从洞府阵升级到界阵。
3. 让叶疏彤处理残界生机和移栽问题。
4. 让白倩霜看见源血、族运和界源都可能被抽成账。
5. 让沈临风系统出现“远离赵长生”任务前的压力递增。

残界带禁写：

1. 不能把残界当随身洞天奖励。
2. 不能一修就完整恢复。
3. 不能没有流民、界压、界核和维护成本。

## 13. 抽源账站

抽源账站是第八卷制度压迫的空间核心。

它可以设在残界带和监察链中枢之间，表面是战损登记和族契裁断机构，实际负责把血源、界源、族运和战争损耗入账。

组成：

1. 战损登记厅。
2. 血税名单室。
3. 族契裁断台。
4. 界源刻度井。
5. 监察使值房。
6. 账官暗室。
7. 传往仙界的账符台。

俞简适合在这里登场。

抽源账站作用：

1. 证明万年抽源不是天灾，而是制度化计算。
2. 让谢未明发现天机阁推演源头被污染。
3. 让沈临风系统任务与抽源名单发生冲突。
4. 让纪晚凝后期补墟账本有前置证据。

## 14. 百族盟场

百族盟场位于几条大商路交汇处，是百族临时议事、裁断族契、交换俘虏和发布停战令的地方。

它不是真正公平议会。

空间设置：

1. 外族等候台。
2. 大族席。
3. 小族席。
4. 族契裁断石。
5. 监察旁听席。
6. 血誓池。
7. 交易货场。

用途：

1. 白倩霜在这里面对“交出一个人保全族”的公开压力。
2. 大族可以用盟场规则包装猎血。
3. 小族之间也会互害，不是天然团结。
4. 赵长生不能靠一句话说服百族，需要证据、资源、退路和代价。

## 15. 第九卷空间：登仙台残路

登仙台残路位于灵界高处的北天残阶一带。

它不是完整飞升之门，而是一条被新仙界改造成筛选、验源、登记前哨的残破接引路线。

组成：

1. 渡劫荒台。
2. 登仙旧阶。
3. 洗源阵影。
4. 验源门。
5. 飞升失败者遗物区。
6. 旧役残影廊。
7. 仙界投影口。
8. 潜入仙界底层的裂隙。

第九卷主要功能：

1. 证明渡劫不是奖励，而是筛选和验源。
2. 让白倩霜提供血源证据。
3. 让谢未明提供命数和天机污染证据。
4. 让陆维舟修补残路节点。
5. 让沈临风拒绝系统引导。
6. 让赵长生确认飞升回收不是传闻。

进入仙界方式：

| 方式 | 用法 | 限制 |
|------|------|------|
| 伪装飞升者 | 混入被筛选队列 | 会被验源门扫描 |
| 修补残路 | 陆维舟临时稳住断阶 | 材料和法力消耗巨大 |
| 归墟炉遮残缺 | 只遮一瞬命数和源痕 | 会留下异常波纹 |
| 谢未明扰乱命册 | 用错算记录反制一段司命污染 | 会伤神魂 |
| 白倩霜血源证据 | 揭穿验源对百族血源的筛选 | 可能暴露自身真血 |

卷尾不能写荣耀飞升。主角组进入仙界应是伪装、借道、破局或被迫潜入，落点必须是仙界底层。

## 19. 禁写项

1. 不能把灵界写成妖族皮肤版宗门地图。
2. 不能让人族天然统治灵界。
3. 不能让大族全员邪恶、小族全员高尚。
4. 不能让白倩霜离开霜羽族后就不管族群。
5. 不能让猎血只靠单个变态反派推进。
6. 不能让祖地只是风景，必须承载族契、源血、族运和病灶。
7. 不能让万年抽源只是一句背景设定。
8. 不能让残界带变成主角随身洞天奖励。
9. 不能让登仙台残路成为稳定飞升通道。
10. 不能让第九卷直接进入仙宫，必须落到仙界底层。
11. 不能让谢未明靠推演剧透一切。
12. 不能让赵长生用归墟炉吞血脉或吸族运升级。

## 20. 使用提示

灵界篇写地图时，要保持四段递进：

1. 第六卷：先让主角组在落界边城和散血市重新成为底层。
2. 第七卷：再让霜羽祖地、百族盟场和玄鳞大泽证明猎血是族群制度问题。
3. 第八卷：用残界带和抽源账站证明血源、界源、族运都被写进账。
4. 第九卷：用登仙台残路证明飞升本身已经被改造成筛选和回收。

灵界篇的核心味道不是“终于到了高级地图”，而是：

> 高级世界不是更自由，而是连血脉、祖地、界源和飞升都被登记得更细。赵长生要做的不是从这里拿更多资源，而是看清资源背后的账。
