# 抽源账站专题档

> 灵界制度补档。本文用于承载抽源账站、俞简、战损登记、界源刻度井、抽源额度、补墟外围账口和纪晚凝后期查账接口。本文不是补墟司黑账总档，也不替代 `06_灵界监察链.md`。

---

## 1. 设计目的

抽源账站专题档要解决七个问题：

1. 万年抽源为什么不是一句背景设定，而是有场地、账官、刻度、文书和传账流程。
2. 俞简为什么不必亲手杀人，却能把灵界损耗变成补墟额度。
3. 战损、血税、魂源、界源和族运如何被拆开登记。
4. 界源刻度井如何让“世界根基被抽走”变成可见空间。
5. 灵界监察链怎样通过抽源账站接入仙界补墟外围账口。
6. 主角组能在第八卷拿到哪些证据，不能提前揭开哪些仙界黑账。
7. 纪晚凝后期为什么能从补墟司账本里回查到灵界旧账。

抽源账站的核心味道：

> 它不说谁被害了，只说这一年灵界可补额度又多了三成。

## 2. 基础定位

| 项目 | 内容 |
|------|------|
| 名称 | 抽源账站 |
| 位置 | 残界带、山魁地脉和监察链中枢之间 |
| 表层名义 | 战损登记、族契复核、界源灾情统计、归墟污染防控 |
| 实际功能 | 将血源、魂源、界源、族运和战争损耗折算为补墟外围额度 |
| 代表人物 | 俞简 |
| 上游接口 | 灵界监察链、猎血商队、百族盟场、残界带、山魁地脉 |
| 下游接口 | 登仙台残路验源口、仙界补墟司外账房、补墟司血账和界账 |
| 活跃卷段 | 第八卷核心，第九卷回收，第十卷以后由纪晚凝账线接收 |

抽源账站不是仙界补墟司本部。

它是补墟体系下沉到灵界的外围站点，负责把灵界发生的伤害先翻译成仙界能接收的账目语言。

## 3. 空间位置

抽源账站设在几条高压线路交汇处。

```text
山魁地脉
    ↓
残界带战损登记处
    ↓
抽源账站
    ↓
登仙台残路验源口
    ↓
仙界补墟司外账房
```

它靠近残界带，是因为残界带容易被写成“自然崩解”。

它靠近山魁地脉，是因为界源矿裂和地脉衰败能直接提供界账数据。

它接百族盟场，是因为族契裁定和血税赔偿能转成血账额度。

它接登仙台残路，是因为被筛选的渡劫者、飞升者和证据会在这里进入仙界流程。

## 4. 总布局

```text
外站门牌：灵界战损与灾情复核站
        ↓
战损登记厅 —— 血税名单室 —— 族契复核台
        ↓
魂灯灰柜 —— 残魂证物室
        ↓
界源刻度井 —— 地脉量尺廊
        ↓
折源算室 —— 抽源额度台
        ↓
证人隔房 —— 封档夹道
        ↓
仙界账符台 / 补墟外围账口
```

| 空间 | 表层功能 | 实际功能 |
|------|----------|----------|
| 战损登记厅 | 统计战争死亡、毁坏、迁徙 | 将百族冲突拆成可折算损耗 |
| 血税名单室 | 记录献血、赔偿、血脉伤 | 将猎血和族契交人接入血账 |
| 族契复核台 | 复核族契裁定是否合规 | 把被迫献血写成族内自愿 |
| 魂灯灰柜 | 保存残魂灯灰和魂证 | 将亡魂异常先做魂账表层 |
| 界源刻度井 | 检测地脉、残界和祖地界源变化 | 将世界病灶量化成抽源额度 |
| 地脉量尺廊 | 对接山魁地脉和石灵脉区 | 证明抽源伤害有技术记录 |
| 折源算室 | 把血、魂、界、运等拆算 | 俞简核心办公区 |
| 抽源额度台 | 生成上报额度简表 | 接仙界外围账口 |
| 证人隔房 | 暂押账师、族人、残界守界人 | 防止证据外泄 |
| 仙界账符台 | 传送账符、回执和法旨 | 接补墟司外账房 |

## 5. 俞简的位置

| 项目 | 内容 |
|------|------|
| 身份 | 抽源账官，灵界监察链入账代表 |
| 功能 | 将明厌等前段监察获得的名单、战损、裁定和证据折成补墟外围额度 |
| 性格方向 | 清瘦、谨慎、语气平，习惯把伤害称为差额 |
| 欲望 | 把所有失控事件写成可上报、可解释、可平衡的账 |
| 恐惧 | 灵界损耗失控，账目对不上，导致仙界上层直接扩大抽源 |
| 错误 | 以为折算准确就是减少混乱，忽视折算本身正在吞掉世界 |
| 边界 | 不掌握完整墟眼黑账，不代表补墟司最高层 |

俞简不是比明厌更会打的人。

明厌负责裁定合法性，俞简负责把合法性变成额度。

俞简的压迫来自：

1. 他能把死亡写成战损。
2. 他能把族契交人写成赔偿。
3. 他能把祖地衰败写成自然灾变。
4. 他能把残界崩解写成归墟污染扩散。
5. 他能把这些全部合并成“本期可补额度”。

## 6. 入账流程

标准流程：

```text
前端事件发生
→ 监察链提交文书
→ 抽源账站复核
→ 分类拆源
→ 界源刻度井校验
→ 生成抽源额度简表
→ 仙界账符台上传
→ 补墟司外账房接收回执
```

前端事件包括：

1. 猎血商队交回血样。
2. 百族盟场裁定某族赔偿。
3. 霜羽源泉衰败。
4. 玄鳞族以豁免换交小族信息。
5. 魂梦族残魂证言被封存。
6. 山魁地脉出现界源裂缝。
7. 残界带崩解或流民迁徙。
8. 登仙台残路验源前置记录回流。

## 7. 战损登记

战损登记厅表面负责避免百族互相虚报。

常见登记项目：

| 项目 | 表层说法 | 实际风险 |
|------|----------|----------|
| 死亡人数 | 战争死亡 | 可被转入魂账 |
| 血脉折损 | 伤残和血脉退化 | 可被转入血账 |
| 祖地损耗 | 灵泉浑浊、地脉变薄 | 可被转入界账 |
| 族运波动 | 迁徙、灭支、幼弱减少 | 可被转入运账 |
| 残界崩解 | 归墟污染或自然衰败 | 可被转入界源抽取 |
| 飞升失败 | 渡劫者陨落或失踪 | 可接登仙台验源和魂账 |

伪账手法：

1. 把猎血导致的血脉伤写成战后伤病。
2. 把抽源后的祖地衰败写成自然灾变。
3. 把大族驱逐小族写成自愿迁徙。
4. 把飞升失败者残魂写成归墟污染清理。
5. 把残界守界人失踪写成界压意外。

主角组要夺的不是一张“坏人名单”，而是战损前后对不上、现场和账册对不上的证据。

## 8. 界源刻度井

界源刻度井是抽源账站最重要的空间。

它不是普通水井，而是一座插入残界带和地脉量尺之间的测源阵井。

表现：

1. 井壁刻有九源刻度。
2. 界源波动会在井水中显出裂纹影。
3. 山魁地脉送来的石尺会在井边校准。
4. 残界核心碎光被投入井中，会显出“可抽”“封存”“污染”“归零”等判定。
5. 井底有通往仙界账符台的细小阵线。

界源刻度井能做：

1. 判断某片残界是否还有可抽界源。
2. 判断祖地衰败是否达到可上报标准。
3. 把界源伤害量化成刻度。
4. 给仙界外围账口提供看似客观的数据。

它不能做：

1. 真正修复残界。
2. 判断抽源是否正当。
3. 显示受害者名字。
4. 阻止俞简篡改分类。

赵长生、陆维舟、叶疏彤在这里可以共同看见：所谓界源额度背后，是地脉、灵植、流民和祖地一起变薄。

## 9. 抽源额度

抽源额度不是单一数字。

| 额度 | 来源 | 后续去向 |
|------|------|----------|
| 血源额度 | 真血、血样、族契赔偿、猎血许可 | 补墟司血账 |
| 魂源额度 | 飞升失败者、残魂灯灰、战死者魂证 | 补墟司魂账表层 |
| 界源额度 | 残界、祖地、山魁地脉、洞天碎片 | 补墟司界账 |
| 运源额度 | 族运折损、百族迁徙、天命变量 | 新司命台和补墟司运账 |
| 宝源额度 | 猎血器具、残界旧器、飞升失败遗物 | 万宝库和补墟司宝账 |
| 法源额度 | 族中传承、旧契、残碑和道统碎片 | 万法碑残线和法账 |

抽源额度简表的可怕之处：

> 读起来没有一个人名，实际上每一项都压着一个族群、一处祖地或一片残界。

## 10. 补墟外围账口

抽源账站不直接进入黑账深库。

它先接补墟外围账口。

```text
抽源账站
→ 灵界账符
→ 登仙台残路验源口
→ 仙界补墟司外账房
→ 明账厅 / 内账廊
→ 封账阁 / 黑账深库
```

外围账口能接收：

1. 抽源额度简表。
2. 猎血许可回执。
3. 战损入账草表。
4. 界源刻度井校验符。
5. 飞升验源前置记录。
6. 证人隔房封存名单。

外围账口不会写：

1. 谁被逼迫。
2. 谁伪造族契。
3. 哪个大族私吞。
4. 哪个仙界权贵得到豁免。
5. 墟眼真实投入效率。

这能保证第八卷只揭开外围账，不提前解释完整仙界黑账。

## 11. 与核心人物关系

| 人物 | 关系 | 叙事作用 |
|------|------|----------|
| 赵长生 | 可从残器、账符灰、井水刻痕还原部分真相 | 从修物进入修界和识别账本残痕 |
| 沈临风 | 系统可能把抽源额度任务化 | 逼他看见系统奖励也接补墟消耗 |
| 白倩霜 | 血源额度证人 | 证明百族真血和族契被写进账 |
| 叶疏彤 | 祖地和残界生机证人 | 证明界源伤害不是自然衰败 |
| 陆维舟 | 界源刻度井和地脉量尺对抗者 | 技术上证明刻度井抽的是世界根基 |
| 谢未明 | 命数污染追踪者 | 证明抽源账和天机错算有接口 |
| 明厌 | 前段裁定和名单来源 | 他的文书进入俞简账站 |
| 俞简 | 抽源账官 | 让灵界伤害转成补墟额度 |
| 纪晚凝 | 后期接收账口的人 | 第十卷以后能回查灵界账符和外账房回执 |

## 12. 可夺取证据

| 证据 | 来源 | 后续用途 |
|------|------|----------|
| 战损入账草表 | 战损登记厅 | 证明百族死亡和祖地衰败被拆账 |
| 血税名单副册 | 血税名单室 | 接白倩霜血账线 |
| 族契复核签押 | 族契复核台 | 证明所谓自愿赔偿有监察压力 |
| 魂灯灰签 | 魂灯灰柜 | 接洛闻霜轮回灯和魂账线 |
| 界源刻度拓片 | 界源刻度井井壁 | 证明残界和祖地被量化抽取 |
| 抽源额度简表 | 抽源额度台 | 接纪晚凝补墟司内账 |
| 仙界账符残灰 | 仙界账符台 | 证明灵界账站与仙界外账房相连 |
| 证人隔房名册 | 封档夹道 | 证明账站控制证人 |

这些证据需要分散夺取，不能一次性全拿。

更好的写法是：

1. 第六卷拿到猎血许可。
2. 第七卷拿到族契裁定副本。
3. 第八卷拿到抽源额度简表和界源拓片。
4. 第九卷用登仙台验源记录确认仙界接收。
5. 第十到十一卷由纪晚凝在补墟司外账中对上回执。

## 13. 卷际使用

| 卷 | 使用方式 |
|----|----------|
| 第六卷 | 只以传闻、账符残片、猎血商队口风出现，不正式展开 |
| 第七卷 | 霜羽祖地和百族盟场证据显示商队追猎最终会进入账站 |
| 第八卷 | 抽源账站正式成为核心空间，俞简登场，界源刻度井和战损登记展开 |
| 第九卷 | 登仙台残路回收抽源账站证据，验源口接收灵界外围账 |
| 第十到十一卷 | 纪晚凝在补墟司外账房和内账廊看到灵界账符回执 |
| 第十三卷 | 灵界血账、魂账、界账作为黑账深库的一部分公开 |

## 14. 退场与回收

抽源账站可以被破坏，但不能让万年抽源就此结束。

可选回收：

1. 主角组毁掉界源刻度井的一层量尺，阻止一次扩大抽源。
2. 俞简转移关键账册，只留下草表和账符残灰。
3. 账站被上级封存，证人隔房被清空。
4. 俞简被调往更高账口或被灭口，证明灵界账站只是外围。
5. 抽源账站证据进入纪晚凝线，由仙界补墟司继续回查。

第八卷结尾应是“证据到手，制度更深”，不是“抽源结束”。

## 15. 写作禁忌

1. 不能把抽源写成抽灵气或挖灵矿这么简单。
2. 不能让俞简成为亲手杀人的普通反派。
3. 不能让抽源账站提前解释完整墟眼黑账。
4. 不能让界源刻度井成为主角刷界源的资源点。
5. 不能让一张抽源额度表说服所有百族。
6. 不能让纪晚凝提前在灵界登场查账，除非后续明确改卷纲。
7. 不能把归墟污染当成抽源的真实原因；很多污染标签是遮掩抽源后果。
8. 不能让赵长生用归墟炉直接修好整片残界带。

## 16. 使用提示

写抽源账站时，抓四个动作：

1. 登记：把死者、族群、祖地和残界写进表。
2. 拆源：把具体伤害拆成血、魂、界、运等类别。
3. 折算：把类别换成补墟额度。
4. 传账：把额度送往仙界外围账口。

抽源账站的核心证明是：

> 万年抽源不是天灾，也不是一次阴谋，而是一套能把世界根基写成整齐数字的制度。
