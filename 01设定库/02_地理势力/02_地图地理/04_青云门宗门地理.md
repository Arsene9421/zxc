# 青云门宗门地理

> 地图地理补档。本文不是青云门势力制度档，而是第二卷到第五卷可反复使用的宗门空间、部门位置、日常动线、管控节点和剧情落点规则。青云门制度详见 `01设定库/02_地理势力/03_下界势力/01_青云门.md`。

---

## 1. 设计目的

青云门宗门地理要解决五个问题：

1. 第二卷进入宗门后，读者知道赵长生、叶疏彤、陆维舟、顾长庚每天在哪里活动。
2. 外门任务、贡献点、灵田、丹房、阵堂、剑堂、藏经阁和庶务库不只是名词，而是有距离、有门槛、有路线、有看守的具体空间。
3. 宗门压迫不靠掌门长老天天出场，而是靠山门、执事堂、账房、库房、通行牌和内外门边界自然发生。
4. 赵长生的捡漏线从青石坊市废料场升级到宗门废料谷、丹房炉灰间、阵堂废盘房和藏经阁残卷室。
5. 第二卷到第五卷的下界舞台有连续空间，不是每卷换一张互不相关的地图。

青云门不能写成宏大仙宫。它是下界偏僻区域的小中型宗门，强在“规矩完整、账目完整、通道完整”，不是强在高阶修士满天飞。

## 2. 区域定位

青云门位于小青云山一带，距离青石坊市不远。

大致关系：

```text
青石坊市
  ↑
  |  半日到一日山路，低阶修士需看脚程和伤势
  |
枯石岭外缘
  ↑
  |
小青云山南麓
  ↑
  |
青云门外山门
```

区域气质：

1. 山不高，但层层设卡。
2. 灵气比青石坊市强，但远不算富裕。
3. 前山看起来规整，后山和边角处多是修补痕迹。
4. 外门拥挤，内门清净，灵田和作坊永远忙。
5. 地形天然制造阶层感：越往上走，灵气越足，门槛越高。

## 3. 总布局

青云门可按“山脚、外门、技艺区、内门、后山”五层写。

```text
                         后山禁坡 / 旧矿洞 / 废料谷
                                      ↑
                         藏经阁偏院 —— 后山小径
                                      ↑
            剑堂峰  ——  内门界阶  ——  掌门议事峰
               ↑                      ↑
          演武坪                  内门静修区
               ↑
  灵田坡 —— 外门广场 —— 丹房谷 —— 阵堂坞
     ↑          ↑          ↑          ↑
  叶氏田区   执事堂     炉灰间     修盘房
     ↑          ↑          ↑          ↑
  外门杂院 —— 任务碑 —— 庶务库 —— 杂务货道
               ↑
            外山门
               ↑
          山脚外驿 / 青石路
```

布局原则：

| 层级 | 地点 | 功能 | 剧情意义 |
|------|------|------|----------|
| 山脚 | 外驿、杂役棚、货车场 | 接人、验牌、运货 | 第二卷入门、追索第一卷余波 |
| 前山 | 外山门、外门广场、执事堂、任务碑 | 规则入口 | 贡献点、身份牌、任务和纠纷 |
| 外门两翼 | 灵田坡、丹房谷、阵堂坞、剑堂峰下坪 | 日常劳动和技艺 | 核心同伴登场与专业线 |
| 内门边界 | 内门界阶、内门静修区、剑堂上院 | 资格分层 | 顾长庚正统压力、赵长生无资格感 |
| 后山 | 废料谷、旧矿洞、藏经阁偏院、试炼林 | 残缺资源和风险 | 赵长生捡漏、小秘境和断仙门伏笔 |

## 4. 入口与门禁

### 4.1 山脚外驿

山脚外驿位于青石路尽头，是青石坊市、附属家族、散修任务者进入青云门前的临时停留点。

功能：

1. 检查通行牌。
2. 接收外门任务材料。
3. 暂存灵米、药材、废料车和阵材。
4. 安排杂役、临时工和送货人等候。

可写压力：

1. 赵长生第一卷逃亡污名尚未完全消失，山脚外驿会查他的来历。
2. 沈临风的系统任务可能指向山门，但身份解释不能过于轻松。
3. 叶氏、陆氏、顾氏这种附属组织的人在这里有不同待遇。
4. 送货车和废料车的账册，是青石坊市废料链接入青云门的第一处空间证据。

### 4.2 外山门

外山门不是恢宏天门，而是一座青石牌楼加低阶护山阵。

管控：

1. 外门弟子凭身份牌进出。
2. 附属家族送货要登记。
3. 散修任务者需由执事堂担保。
4. 夜间只开侧门，货道另行登记。

限制：

1. 护山阵能挡低阶修士硬闯，但不是无敌仙阵。
2. 门禁看守主要管秩序和账，不会替底层修士主持正义。
3. 第一卷后续追索可以在这里形成压力，但不能让青云门高层立刻盯上归墟炉。

### 4.3 杂务货道

杂务货道绕过外门广场，通向丹房谷、庶务库、废料谷和外门仓房。

用途：

1. 运送灵草、灵米、丹渣、炉灰、坏阵盘、残器。
2. 底层杂役、车夫、学徒常走。
3. 管事偷换、压价、藏账最容易发生。

赵长生进入宗门后，最先熟悉的不应是内门风景，而是这条灰尘重、药味重、账目重的货道。

## 5. 外门核心区

### 5.1 外门广场

外门广场是第二卷最常出现的公共空间。

组成：

1. 任务碑。
2. 贡献点告示墙。
3. 外门执事堂前阶。
4. 临时摊位和等候区。
5. 低阶比试台。

功能：

1. 发布任务。
2. 公布考核。
3. 处理外门纠纷。
4. 登记任务失败和贡献扣罚。
5. 让不同阶层的外门弟子互相看见。

可写戏：

1. 赵长生第一次看见任务和贡献点规则。
2. 叶疏彤因灵田产量被点名。
3. 陆维舟因坏阵赔付被推出来背锅。
4. 顾长庚从剑堂峰下经过，天然带来正统对照。
5. 沈临风的系统发布“接近关键人物”或“获取外门资格”任务，与现实贡献规则发生冲突。

### 5.2 外门执事堂

外门执事堂位于外门广场上首，管任务、贡献、身份牌和纠纷。

空间设置：

| 区域 | 用途 | 压力 |
|------|------|------|
| 前厅 | 接任务、交任务、登记身份 | 人多、排队、被管事晾着 |
| 侧账房 | 贡献点、罚扣、赔偿 | 账面合理，底层难辩 |
| 小审室 | 外门纠纷、盗物、斗殴处理 | 莫守廷可在此出场 |
| 牌籍柜 | 身份牌、担保、附属家族名册 | 赵长生出身和第一卷污名被查 |
| 后廊 | 管事私下谈事 | 灰色交易和回扣发生处 |

执事堂不是反派总部。它的压迫来自“你要讲规矩，就必须先有身份、担保、证据和时间”。

### 5.3 庶务库

庶务库靠近执事堂和杂务货道，是资源兑换、材料发放和废料入账处。

分区：

1. 灵石小库。
2. 材料柜。
3. 丹药兑换间。
4. 任务押金柜。
5. 废料登记房。
6. 旧账封存柜。

剧情用途：

1. 贡献点兑换不公平可以在这里落地。
2. 赵长生能看见废料从丹房、阵堂和青石坊市入库，再流向后山废料谷。
3. 庶务库账房可以证明宗门制度不是没有账，而是账的口径偏向上层。
4. 叶氏债契、陆氏坏阵赔付、青石坊市废料账都可在庶务库交叉。

禁写：

1. 不能让赵长生随便偷库。
2. 不能让庶务库全员无脑贪污。
3. 不能让一张账册立刻解释仙界补墟体系。

## 6. 灵田坡

灵田坡位于外门西侧，地势比外门广场低一些，靠近一条浑浊小灵泉。

组成：

1. 外门公田。
2. 叶氏承包田区。
3. 赤纹灵禾试种田。
4. 灵泉井。
5. 灵田院小屋。
6. 看田棚和晒药架。
7. 小型聚灵阵和水渠。

空间气质：

1. 风比外门广场湿。
2. 灵气有起伏，越靠旧灵泉越浑。
3. 地里常有残枯禾根和补土痕迹。
4. 叶疏彤在这里不是“漂亮女修登场”，而是卷着袖子处理快死的灵禾。

叶疏彤日常动线：

```text
叶氏田区
→ 灵泉井
→ 灵田院小屋
→ 丹房谷送丹材
→ 庶务库核产量
→ 外门广场看任务
```

可写冲突：

1. 灵田产量差一成，贡献点被扣。
2. 丹材药性不均，被丹房压价。
3. 阵堂修的聚灵阵太急，伤了土性。
4. 叶氏旧债在庶务库和灵田院之间来回滚。
5. 赵长生能用归墟炉处理残根、废土或枯草，但不能一夜救活整片田。

## 7. 丹房谷

丹房谷位于外门东南侧，靠山壁开炉，便于排烟和控火。

组成：

1. 前炉院：低阶丹药炼制。
2. 药材库：灵草、丹材和保存柜。
3. 炉灰间：废炉灰和丹渣。
4. 试药房：低阶药性测试。
5. 丹房学徒住处。
6. 废丹封缸区。
7. 通往庶务库和废料谷的后道。

赵长生第二卷捡漏重点在这里升级：

| 地点 | 可用资源 | 限制 |
|------|----------|------|
| 炉灰间 | 炉灰、丹渣、残火石 | 有称重，有账，有学徒看守 |
| 废丹封缸区 | 废丹、坏丹、半成丹 | 丹毒风险高，不能直接吃 |
| 药材库边角 | 枯草、烂根、药性不均材料 | 需要叶疏彤辨识配合 |
| 后道 | 废料运往庶务库和废料谷 | 容易被邹连城或管事盯上 |

丹房谷的意义：

1. 把第一卷回春丹铺的废丹线升级到宗门层级。
2. 让赵长生知道废料不是没人要，而是被制度性入账。
3. 给叶疏彤的药性辨识、赵长生的归墟炉和沈临风的系统奖励制造对照。

## 8. 阵堂坞

阵堂坞位于外门东侧偏后，靠近山壁和一片旧石台，适合试阵。

组成：

1. 修盘房。
2. 阵材棚。
3. 阵纹石台。
4. 小试阵场。
5. 废阵盘堆。
6. 勘脉小队库房。
7. 通往灵田坡、外山路和后山小径的岔道。

陆维舟日常动线：

```text
修盘房
→ 阵材棚
→ 小试阵场
→ 灵田坡修护小阵
→ 庶务库核赔付
→ 外门广场交任务
```

可写冲突：

1. 阵材不足，主脉拿好料，旁支修坏账。
2. 灵田阵聚灵效率和叶疏彤的土性判断冲突。
3. 外门执事要求“先能用”，陆维舟要求“不能害人”。
4. 赵长生在废阵盘中发现不合常理的残缺。
5. 阵堂后方小径可接后山废料谷，为后续小秘境和旧阵痕埋路。

阵堂不能写成陆维舟一个人的实验室。它是职业责任、材料成本和坏阵后果的空间承载。

## 9. 剑堂峰

剑堂峰位于外门北侧偏高处，靠近内门界阶。

分层：

1. 峰下演武坪：外门和旁听弟子可见。
2. 剑廊：挂低阶剑器、旧剑谱碑。
3. 剑堂前院：正式剑堂弟子修炼处。
4. 剑堂上院：内门剑修和重点弟子区域。
5. 顾氏支脉小院：靠近剑堂，但不在最上层。

顾长庚日常动线：

```text
顾氏支脉小院
→ 剑堂晨课
→ 演武坪
→ 藏经阁剑诀区
→ 外门广场任务牌
→ 剑堂回课
```

空间作用：

1. 顾长庚的正统压迫来自高处，但他也受高处规矩压迫。
2. 剑堂峰能让赵长生直观看见“资格”有空间高度。
3. 剑堂比试可发生在峰下演武坪，不必写成大型擂台赛。
4. 顾长庚不应在普通外门场景里像恶少一样乱晃，他的行动应带有任务、晨课、比试和宗门名声压力。

禁写：

1. 剑堂不能全是傲慢弟子。
2. 顾长庚不能靠家世随意欺压赵长生。
3. 赵长生不能轻易偷学完整正统剑诀。

## 10. 藏经阁

藏经阁位于内门界阶旁，正阁靠内，偏院靠外。

结构：

| 区域 | 可进入者 | 内容 | 剧情用途 |
|------|----------|------|----------|
| 外阁 | 外门弟子、贡献点兑换者 | 基础功法、低阶法术、任务册 | 赵长生补基础 |
| 技艺架 | 丹、器、符、阵、灵植低阶手札 | 需贡献点或部门担保 | 主角团专业线互证 |
| 残卷室 | 需廖苍或执事许可 | 破损玉简、旧道统残页 | 归墟炉补字、断仙门伏笔 |
| 内阁界门 | 内门弟子和长老许可 | 完整进阶功法 | 资格压迫 |
| 偏院旧库 | 半废旧书、错抄本、失败记录 | 低价值但有真线索 | 赵长生捡漏和顾长庚正统反思 |

藏经阁的关键不是“主角拿神功”，而是让残缺知识变得可写：

1. 残卷缺字。
2. 玉简有裂纹。
3. 功法前后矛盾。
4. 旧批注被涂改。
5. 权限不足，只能看一部分。

廖苍作为藏经阁老人，可以提供残卷入口，但不能一出场看穿归墟炉，也不能直接解释断仙门真相。

## 11. 后山废料谷

后山废料谷位于丹房谷和阵堂坞后方，经杂务货道和小径可达。

组成：

1. 丹灰坑。
2. 废阵盘堆。
3. 残器棚。
4. 坏玉简封箱。
5. 废灵石粉槽。
6. 旧矿洞入口。
7. 宗门禁制边线。

作用：

1. 承接青石坊市废料场的升级版。
2. 让赵长生继续有凡人流低成本试错空间。
3. 给归墟炉同源碎片、小秘境残痕、旧天庭残器碎屑留下合理藏身处。
4. 让废料链从“底层垃圾”逐渐变成“制度化回收链”的证据。

限制：

1. 废料谷有账有禁制，不能随便搬空。
2. 废料中的污染、丹毒、残阵反噬都真实存在。
3. 赵长生捡漏越多，越容易引起丹房、阵堂、庶务库和灰账派注意。

## 12. 试炼林与小秘境入口

试炼林位于后山和外门边界，是第二卷卷尾小秘境的合理承载地。

组成：

1. 低阶妖兽圈养区。
2. 采药任务区。
3. 旧阵界碑。
4. 半塌矿洞。
5. 宗门旧禁制断口。
6. 小秘境残门。

使用规则：

1. 第二卷前半只当普通外门任务区。
2. 中段可出现旧禁制异常、阵堂修补失败、采药弟子失踪。
3. 后段再揭示小秘境残门与归墟炉同源碎片有关。
4. 宗门长老可以争夺同源碎片，但不能知道归墟炉完整来历。
5. 小秘境不是随身空间奖励，而是青云门旧地理中的危险裂口。

## 14. 第二卷常用路线

### 14.1 赵长生低调生存路线

```text
外门杂院
→ 外门广场任务碑
→ 丹房谷炉灰间
→ 庶务库废料登记房
→ 后山废料谷
→ 藏经阁偏院
→ 外门杂院
```

用途：

1. 继续凡人流底层路线。
2. 让归墟炉试验有材料来源和代价。
3. 让赵长生逐步接触宗门灰账。

### 14.2 叶疏彤灵田路线

```text
叶氏田区
→ 旧灵泉井
→ 灵田院小屋
→ 丹房谷药材库
→ 庶务库核产量
→ 外门广场
```

用途：

1. 展示灵植不是口头专业。
2. 让叶氏旧债、丹房压价和灵田病灶互相扣住。
3. 给赵长生和叶疏彤合作提供现实任务，而不是一见钟情。

### 14.3 陆维舟修阵路线

```text
阵堂坞修盘房
→ 阵材棚
→ 灵田坡小阵
→ 庶务库赔付账
→ 后山小径试阵
```

用途：

1. 展示阵法成本和坏阵责任。
2. 给洞府经营线提前埋职业经验。
3. 让陆维舟的谨慎有空间来源。

### 14.4 顾长庚正统路线

```text
顾氏支脉小院
→ 剑堂晨课
→ 演武坪
→ 藏经阁剑诀区
→ 外门广场任务牌
→ 内门界阶前
```

用途：

1. 展示正统资源真实存在。
2. 展示顾长庚也被家族和剑堂路线约束。
3. 和赵长生的废料路线形成视觉和制度对照。

### 14.5 宗门废料链路线

```text
青石坊市废料车
→ 山脚外驿
→ 杂务货道
→ 庶务库废料登记房
→ 丹房炉灰间 / 阵堂废盘房
→ 后山废料谷
```

用途：

1. 回收第一卷废料线。
2. 解释赵长生为什么还能接触残缺资源。
3. 埋下更高层废料回收链的制度影子。

## 16. 卷际变化

| 卷 | 青云门空间用法 | 变化 |
|----|----------------|------|
| 第一卷 | 只在青石坊市以“上宗、办事处、外门管事、废料链”出现 | 不展开宗门全貌 |
| 第二卷 | 外门广场、灵田坡、丹房谷、阵堂坞、剑堂峰、藏经阁、废料谷全面启用 | 主角团成形，宗门制度压迫落地 |
| 第三卷 | 庶务库、外门任务、筑基资源、剑堂推荐和家族路线加重 | 宗门与修仙家族资源垄断交叉 |
| 第四卷 | 洞府线外移，青云门通过产权、阵法合同、灵脉和长老代理继续施压 | 宗门从舞台变成资源权力来源 |
| 第五卷 | 青云门在修仙国大战和断仙门前暴露正统局限 | 宗门不能回答飞升异常，只能提供局部证据 |

## 18. 禁写项

1. 不能把青云门写成大宗仙宫。
2. 不能把青云门写成纯反派巢穴。
3. 不能让赵长生刚进门就直接接触掌门和核心长老。
4. 不能让外门贡献点像游戏面板一样绝对公平。
5. 不能让赵长生随便偷库、偷功法、偷废料而没有后果。
6. 不能让叶疏彤的灵田线只变成陪主角种田。
7. 不能让陆维舟的阵法没有成本、没有责任、没有失败记录。
8. 不能让顾长庚在空间上和普通恶少一样无目的乱逛。
9. 不能让藏经阁直接给出旧天庭和仙界真相。
10. 不能让小秘境成为安全奖励空间。
11. 不能让宗门所有管事都蠢坏，压力必须来自岗位、账目和资源口径。

## 19. 使用提示

第二卷写青云门时，应优先写路线，而不是解释制度。

推荐顺序：

1. 先让赵长生从山脚外驿和外山门感受到身份压力。
2. 再让外门广场和执事堂展示贡献点规则。
3. 接着用丹房谷、灵田坡、阵堂坞、剑堂峰分别引出赵长生、叶疏彤、陆维舟、顾长庚的专业线。
4. 中段让庶务库和后山废料谷把废料链升级。
5. 后段让试炼林和藏经阁残卷室把小秘境和断仙门伏笔露出一角。

青云门的核心味道不是“主角进入宗门终于稳定”，而是：

> 他从坊市的黑暗小巷，走进了看似有规矩的山门，却发现规矩也会吃人，只是吃得更慢、更有账本。
