# 第一卷逃亡路线

> 地图地理补档。本文不是章纲，而是第一卷卷尾逃亡的空间动线、追兵路径、风险节点和落点规则。后续拆章时可以引用本档，但不能把本档直接当作固定章节表。

---

## 1. 设计目的

第一卷卷尾逃亡要解决四个问题：

1. 赵长生为什么必须离开青石坊市。
2. 他为什么能逃出去，而不是被坊市、丹铺和上宗管事立刻按死。
3. 第一次杀人为什么成立，不靠境界碾压，也不靠临时爆种。
4. 逃亡之后为什么自然进入第二阶段，而不是立刻安全、立刻拜入宗门、立刻解释大世界观。

逃亡本质不是“跑出城门”，而是从熟悉的底层空间逃进更危险的荒野空间。爽点来自：赵长生用底层经验、废料地形、残器工具和沈临风的信息提示，在比自己更强、更有资源的追兵手里撕开一条窄路。

## 2. 逃亡前提

逃亡发生在第一卷后段，时间建议为深夜到破晓前。

触发条件：

1. 赵长生出售或使用异常资源后，被周槐安、裴九、小帮派和坊市散修多方盯上。
2. 沈临风的系统任务要求他接近赵长生，导致他也卷入追查。
3. 上宗外门管事蒋怀礼开始介入废料和账目，坊市层面的普通纠纷升级为“有人盗取上宗废料资源”。
4. 赵长生意识到继续留在青石坊市，只会被扣上偷盗、私炼、勾结黑市或残害同道的罪名。

逃亡前不能让赵长生准备得过于完美。他可以有几样低阶物资和临时判断，但不能像提前规划数月的成熟逃亡。

| 废灵脉支沟 | 废弃灵脉余脉沟壑 | 卷尾临时躲藏点，不是长期根据地 |

## 4. 路线分段

### 4.1 回春丹铺后巷

后巷连接丹铺后院、药渣房、废炉灰堆和杂役住处。

赵长生熟悉这里：

1. 哪块青砖松动。
2. 哪个药渣桶味道最重。
3. 哪段墙根可以藏人。
4. 哪个时辰丹铺小伙计最容易偷懒。

限制：

1. 后巷不是安全区，周槐安最容易在这里派人堵他。
2. 后巷离主街不远，动静太大会引来巡市修士。
3. 不能在这里长期缠斗，赵长生必须尽快转入黑市巷或废料场。

### 4.2 黑市巷

黑市巷不是大型黑市，而是青石坊市外圈的灰色交易缝隙。

这里适合：

1. 混入夜间交易人流。
2. 利用残器摊、假符摊、二手储物袋摊制造视线遮挡。
3. 通过裴九或其他牙人放出假方向。
4. 用小额灵石、残器或消息换取半真半假的误导。

风险：

1. 裴九会卖消息，不会讲义气。
2. 黑市巷的人都记得陌生面孔和异常交易。
3. 这里能拖慢追兵，但不能真正甩掉追兵。

### 4.3 废料场

废料场是第一卷低阶战斗最重要的地形。

组成：

1. 丹渣坑。
2. 炉灰堆。
3. 残器铁片区。
4. 破木车棚。
5. 旧灰井。
6. 堆放待上缴废料的棚屋。

赵长生优势：

1. 知道哪些灰坑表面结壳、下面空陷。
2. 知道哪些残器锋利但灵性已失，能当普通凶器。
3. 知道炉灰能呛人、遮眼、遮血腥。
4. 知道废料车什么时候进出、车辙通向哪里。

这里可以承载第一次杀人。杀人不能写成潇洒收割，而要写成慌乱、疼痛、误判、反应慢半拍和不得不继续逃。

### 4.4 废料车道

废料车道连接废料场和北小门。

特征：

1. 路窄，常有车辙。
2. 灰尘重，气味杂。
3. 路边有破棚、旧木桩和遮雨麻布。
4. 夜间看守通常偷懒，但一旦接到蒋怀礼消息会立刻变危险。

逃亡用法：

1. 借车辙遮脚印。
2. 把带血布条、废器残片混入废料。
3. 借废料车短暂遮身。
4. 用灰布、药渣味掩盖身上血腥和丹气。

### 4.5 北小门

北小门不是正式大门，而是废料、采药人和低阶散修常走的小门。

平时状态：

1. 看守松散。
2. 有人收小钱放行。
3. 通行记录粗糙。
4. 夜间会放一两辆废料车出去。

报警后状态：

1. 蒋怀礼可以要求核查废料车和出门人。
2. 周槐安可以指认赵长生盗取丹铺废料。
3. 小帮派可以在门外埋人。
4. 巡市修士未必立刻抓人，但会倾向保护交钱更多的一方。

北小门的意义是“封锁来得慢”，不是“没人管”。

### 4.6 枯石岭外坡

枯石岭外坡是逃出坊市后的第一段荒野。

地形：

1. 碎石坡。
2. 枯草和低矮灌木。
3. 废弃采药点。
4. 零散兽洞。
5. 夜间风声大，视线差。

风险：

1. 法力消耗后爬坡困难。
2. 受伤会加重失血。
3. 野外灵气稀薄，恢复慢。
4. 低阶妖兽或毒虫不强，但足以拖住疲惫修士。

### 4.7 乱石药径

乱石药径是旧采药人踩出的窄路，不在坊市正式地图上。

使用条件：

1. 赵长生听过杂役和老散修提起，但不能对每一段都熟。
2. 沈临风可以通过系统危险提示辨别一处错误岔路，但不能直接给完整地图。
3. 夜间走药径必须付出擦伤、摔倒、迷路和法力消耗的代价。

这里适合安排一次“差点走错”的危机，让沈临风的信息价值成立，但不能让系统替他们逃命。

### 4.8 断溪谷外围

断溪谷外围地气混乱，溪水常年断流，溪床里残留浅薄灵机和药瘴。

优点：

1. 气味和灵机混乱，追踪难度上升。
2. 正式巡市修士不愿深追。
3. 坊市散修单独追来会犹豫。

危险：

1. 药瘴可能刺激伤口和经脉。
2. 地面有暗沟和碎裂溪床。
3. 灵气不能稳定恢复法力。
4. 赵长生和沈临风都不具备成熟野外生存能力。

### 4.9 废灵脉支沟

废灵脉支沟是卷尾落点。

这里可以暂时藏身，但不是安全根据地。

特征：

1. 支沟狭窄，入口被碎石和枯藤遮住。
2. 沟底残留微弱灵气，能勉强压住伤势。
3. 深处有旧采药棚或半塌石龛。
4. 石壁可能露出一截古旧残纹，为“仙门已断”或断仙门线留下低调伏笔。

卷尾只允许出现“看不懂的残痕”和“无法解释的异常”，不能解释旧天庭、新仙界、补墟司或司命台。

## 5. 关键节点

| 节点 | 控制者 | 风险 | 可用爽点 |
|------|--------|------|----------|
| 回春丹铺后巷 | 周槐安和丹铺伙计 | 被堵、被栽赃 | 赵长生利用熟悉后勤动线脱身 |
| 黑市巷 | 裴九、小摊主、散修 | 消息外泄、被卖路线 | 用假交易或假方向拖慢追兵 |
| 废料场 | 小帮派、废料看守 | 正面冲突、血迹暴露 | 地形反杀、残器实战、灰尘遮掩 |
| 废料车道 | 废料车夫、门房 | 车账核查、脚印追踪 | 借车辙和炉灰抹掉痕迹 |
| 北小门 | 门房、巡市修士 | 封门、查通行牌 | 抢在封锁前钻出坊市 |
| 枯石岭外坡 | 无固定控制者 | 失血、摔伤、迷路 | 由坊市压迫转入荒野压迫 |
| 乱石药径 | 旧采药人残路 | 错路、埋伏、毒虫 | 沈临风提示一次危险岔路 |
| 断溪谷外围 | 无固定控制者 | 药瘴、暗沟、追兵余波 | 追兵不敢深追，主角代价显现 |
| 废灵脉支沟 | 无固定控制者 | 临时躲藏，不能久留 | 卷尾残纹、炉壁残字、系统报错 |

## 6. 追兵与封锁路径

### 6.1 周槐安

周槐安不能亲自追太远。

他更合理的动作：

1. 派丹铺伙计守后巷。
2. 指控赵长生盗取废丹、炉灰或丹铺财物。
3. 向巡市修士和蒋怀礼递话，把私人压榨包装成坊市治安事件。
4. 用赵长生杂役契制造合法压迫。

限制：

1. 他怕丹铺私吞废料的账被翻出来。
2. 他不愿把事情闹到上宗彻查。
3. 他能制造第一波堵截，但不应成为野外追杀主力。

### 6.2 裴九

裴九负责消息流动。

他可能：

1. 把赵长生走黑市巷的消息卖给小帮派。
2. 把沈临风出现的消息卖给坊市散修。
3. 在被威胁或被小利诱导时，故意给出半假路线。
4. 卷后继续留在青石坊市，作为后续黑市消息线的可回收角色。

裴九不能变成讲义气帮主角逃命的人。他的价值是“可交易、可背叛、可误导”。

### 6.3 小帮派头目

小帮派头目最适合成为第一次杀人的直接压力。

合理行动：

1. 派人守棚区和废料场。
2. 想抢赵长生身上的异常资源，而不是替周槐安主持公道。
3. 低估赵长生，认为一个丹铺杂役不敢下死手。
4. 在废料场或废灰坑边缘堵人。

限制：

1. 修为可以高于赵长生，但不能高到无解。
2. 不应带太多训练有素手下，否则赵长生无法合理逃脱。
3. 被反杀后要引发更大恐惧和追捕，而不是直接解决问题。

### 6.4 坊市散修

坊市散修追的是利益。

他们的特点：

1. 嗅到异常灵草、丹药、残器或储物袋就会跟。
2. 没有统一指挥，容易互相猜忌。
3. 比巡市修士更灵活，但也更贪生怕死。
4. 一旦确认主角不是软柿子，就可能转为远远跟踪或出售消息。

散修追兵适合制造压迫和尾随感，不适合写成整齐军阵。

### 6.5 蒋怀礼

蒋怀礼是正式压力，不是瞬移追杀者。

他能做：

1. 以外门管事身份要求坊市协查。
2. 查废料车、丹渣、残器和账册。
3. 给北小门传令。
4. 要求巡市修士封锁部分出入口。
5. 把赵长生逃亡定性为“偷盗上宗废料后畏罪潜逃”。

他不能做：

1. 第一时间准确知道赵长生所有路线。
2. 亲自出现在每个节点。
3. 立刻识破归墟炉。
4. 调动高阶修士围捕一个底层杂役。

### 6.6 巡市修士

巡市修士维护的是秩序、税和通行规则，不是正义。

反应逻辑：

1. 夜间先看谁给钱、谁有名册、谁能证明身份。
2. 普通斗殴不急，涉及上宗废料才会升级。
3. 封门需要流程和人手，不能一念之间全城铁桶。
4. 如果追到荒野边缘，除非有足够好处，否则不会冒险深入断溪谷。

## 7. 三条备选路线

### 7.1 路线 A：北小门废料路线

这是最终采用路线。

```text
后巷 → 黑市巷 → 废料场 → 废料车道 → 北小门 → 枯石岭 → 断溪谷
```

优势：

1. 赵长生熟悉废料场和废料车道。
2. 废料气味、炉灰和车辙能遮掩痕迹。
3. 北小门管控弱，封锁慢于东门。
4. 逃出后能接枯石岭和断溪谷，合理转入野外。

劣势：

1. 路线和赵长生日常劳动有关，一旦被周槐安或小帮派判断出来，容易被堵。
2. 废料场适合反杀，也适合埋伏。
3. 北小门一旦收到蒋怀礼传令，就会成为窄口死地。

采用理由：

赵长生不是有权有钱的人，只能走底层缝隙。北小门废料路线最符合他的社会位置。

### 7.2 路线 B：西门货路

```text
后巷 → 旧货街 → 西门货车道 → 商队岔路
```

优势：

1. 车多人杂，容易混入货流。
2. 路面宽，短时间内逃跑速度较快。
3. 可伪装成临时搬货杂役。

劣势：

1. 西门货账更完整，蒋怀礼更容易通过账册查人。
2. 货路上帮派、车夫和门房彼此熟悉，陌生人反而显眼。
3. 逃出后进入商道，容易被正式追兵沿路查问。

结论：

可作为误导路线，不适合作为真实逃亡主线。

### 7.3 路线 C：南棚区暗沟路线

```text
后巷 → 外圈棚区 → 南侧排水暗沟 → 荒草坡
```

优势：

1. 人口密集，痕迹混乱。
2. 棚区暗沟不进正式地图。
3. 适合短时间甩开巡市修士。

劣势：

1. 南棚区是小帮派地盘，容易被围在窄巷。
2. 暗沟低矮潮湿，受伤后行动困难。
3. 视野差，沈临风和赵长生不熟悉会互相拖累。
4. 出口不稳定，容易写成运气逃生。

结论：

适合被追兵误判为赵长生可能选择的路，也可以作为临时绕行，不适合作为卷尾主线。

### 7.4 东门不可用

东门是正式入口，有青云门通行牌、税牌和主街巡查。

报警后东门必然最先被查：

1. 赵长生无正式身份优势。
2. 沈临风身份更容易暴露。
3. 蒋怀礼的外门身份在东门最有用。

东门只能用于沈临风入场，不能用于卷尾逃亡。

## 8. 战斗与第一次杀人地点

首选地点：

```text
废料场边缘 / 废灰坑 / 通往北小门的破木车棚附近
```

选择理由：

1. 赵长生熟悉地面，知道灰坑、残器、破木车和旧灰井的位置。
2. 地形能解释低阶越级：不是正面打赢，而是利用坑陷、遮眼、残器和对手轻敌。
3. 血迹、碎器和脚印可以暂时被炉灰遮掩，但不能完全消失。
4. 沈临风可以提供一次系统危险提示或敌人动作信息，但不能替赵长生杀人。
5. 杀人后能顺势接入废料车道和北小门，不需要额外绕路。

第一次杀人的执行边界：

1. 敌人修为可以是炼气三层到炼气四层，必须被环境削弱或误导。
2. 赵长生要受伤，至少出现外伤、法力耗损或经脉震痛。
3. 不能写成主角冷酷熟练收尸。第一次杀人要有手抖、恶心、短暂失神或呼吸错乱。
4. 物证处理必须粗糙：带血灰布、敌人储物袋、残器、脚印、破损衣角都可能留下后患。
5. 杀人后最大的选择不是庆祝，而是“现在不走就死”。

可用战斗元素：

| 元素 | 用法 | 限制 |
|------|------|------|
| 炉灰 | 遮眼、呛咳、遮血 | 不能直接毒死修士 |
| 残器铁片 | 临时凶器或陷阱 | 不等于完整法器 |
| 废灰坑 | 让敌人失衡或踩空 | 需要赵长生提前知道位置 |
| 低阶符箓 | 短暂扰乱或挡一下 | 数量必须少，成本要显痛 |
| 沈临风提示 | 提醒危险方向或敌人动作 | 不能给完整解法 |
| 归墟炉 | 可影响残器或炉灰气息 | 不能公开暴露真正能力 |

## 9. 逃亡资源消耗

赵长生可携带资源：

| 资源 | 数量尺度 | 用途 |
|------|----------|------|
| 碎灵石或散碎银钱 | 很少 | 买通、误导、换一条小消息 |
| 发霉灵米或干粮 | 一两日量 | 野外短暂维持体力 |
| 低阶疗伤粉 | 一小包 | 止血，不治内伤 |
| 修复过的残器或锋利工具 | 一件 | 防身和反杀 |
| 灰布、破麻袋 | 一两件 | 遮脸、裹伤、抹除痕迹 |
| 低阶符箓 | 最多一两张 | 保命或扰乱，不能连续使用 |

沈临风可贡献：

1. 系统危险提示。
2. 对某个追兵意图的即时判断。
3. 一次短距离速度、遮掩或护身类奖励。
4. 对错误路线的排除。

沈临风不能贡献：

1. 传送。
2. 完整地图。
3. 高阶护身法宝。
4. 无限任务奖励。
5. 替赵长生承担所有后果。

消耗规则：

1. 法力消耗必须影响后半段逃亡。
2. 伤势不能靠一粒药瞬间痊愈。
3. 储物袋、敌人遗物和血衣会带来追踪风险。
4. 逃亡越顺利，后续追捕越不成立；因此卷尾必须保留损失和后患。

## 10. 时间与封锁节奏

建议时间线：

| 时间 | 事件状态 | 空间压力 |
|------|----------|----------|
| 入夜后 | 丹铺或黑市冲突爆发 | 周槐安、小帮派先动 |
| 子夜前后 | 赵长生和沈临风转入废料场 | 黑市巷开始传消息 |
| 后半夜 | 废料场反杀或重伤追兵 | 蒋怀礼收到消息但未完全封锁 |
| 黎明前 | 冲过北小门或绕过门外看守 | 北小门开始收紧 |
| 天将亮 | 进入枯石岭和乱石药径 | 坊市追兵不愿深追 |
| 破晓后 | 抵达断溪谷外围或废灵脉支沟 | 正式通缉或追查开始形成 |

封锁不能太快，否则主角无法合理逃出；封锁也不能太慢，否则坊市制度没有压迫感。合理状态是：赵长生抢在正式封门前出去，但他出去之后，青石坊市已经形成对他的污名和追查。

## 11. 卷尾落点

卷尾落点定为：

```text
断溪谷外围 / 废灵脉支沟 / 半塌旧采药棚
```

卷尾状态：

1. 赵长生受伤，法力不足，短时间不能回坊市。
2. 沈临风系统完成“逃离青石坊市”支线，但出现异常报错。
3. 两人从互相试探变成临时生死合作者，但不等于完全信任。
4. 归墟炉吸收某件残片或炉灰后，出现“万物归墟，仙门已断”的低调提示。
5. 废灵脉支沟或旧采药棚中出现一处看不懂的旧残纹，为后续断仙门或青槐旧灾方向埋线。

卷尾不能去青槐村旧址。

原因：

1. 青槐旧灾是赵长生深层过去，不能在第一卷逃亡后立刻回收。
2. 第一卷卷尾的功能是“离开坊市”，不是“解释身世”。
3. 去青槐村会让情绪过早转向身世揭秘，削弱底层逃亡后的现实压力。

第二阶段承接：

1. 他们不是直接作为贵客进入青云门。
2. 他们可以通过荒野、旧采药路、临时悬赏、散修队伍或青云门外围杂务间接接近第二卷舞台。
3. 青石坊市留下的污名、账册和追兵，要成为后续进入宗门前的压力，而不是第一卷结束就清零。

## 12. 禁写项

第一卷逃亡阶段禁止：

1. 飞行逃亡。
2. 高阶修士亲自围捕。
3. 系统传送、系统隐身、系统抹除因果。
4. 归墟炉公开暴露真正能力。
5. 周槐安、蒋怀礼或裴九知道归墟炉本质。
6. 赵长生完美清理所有目击者和物证。
7. 杀完人后冷静潇洒地搜刮全场。
8. 逃亡后立刻安全闭关。
9. 第一卷解释旧天庭、新仙界、补墟司、司命台真相。
10. 第一卷卷尾直接返回青槐村旧址。
11. 荒野生存写得像郊游。
12. 把巡市修士写成无脑正义官差。

## 13. 使用提示

后续拆第一卷章纲时，本档只提供空间规则。

写作时要抓住三层递进：

1. 坊市内：人情、账册、身份和资源压迫。
2. 出门口：通行、封锁、追兵和时间差。
3. 荒野中：伤势、法力、方向、夜路和生存压力。

第一卷逃亡的核心味道不是“主角终于自由”，而是“他从一个吃人的地方逃进另一个更大的未知之地，但手里终于有了一点能活下去的东西”。
