# 青石坊市地图与日常

> 本文档是第一卷场景使用表。
> 目标：写章节时能快速选场景、安排冲突和埋线。

## 1. 简易平面关系

```text
                  北小门
                    |
                 废料车道
                    |
      枯石岭 ← 外圈棚区 ← 黑市巷
                    |       |
                废料场 — 旧货街
                    |       |
              回春丹铺后院  灵米铺
                    |       |
                 青石主街 — 青云办事处
                    |
                  东门
```

南侧多散修棚屋和临时摊，西侧是货车和废料出入口，北侧通枯石岭。

## 3. 日常规则

青石坊市有规矩，但规矩只保护有钱人。

表面规则：

- 入城要通行牌。
- 摆摊要交摊费。
- 店铺要交税。
- 夜间禁止斗法。
- 废料必须入账。

实际情况：

- 巡市修士收钱办事。
- 外圈棚区死个散修没人查。
- 黑市巷只要不闹到主街，没人管。
- 掌柜和外门管事可以串通账册。
- 废料上缴前常被掉包。

## 4. 可反复使用的日常细节

- 雨后青石板发黑发亮。
- 丹铺后院有苦涩药渣味。
- 废料场有热灰和霉草混合味。
- 棚区夜里有人咳血、骂街、磨刀。
- 灵井旁老散修抱怨井水越来越淡。
- 旧货街摊主总说“古修遗物，概不退换”。
- 青云办事处门口贴着欠税名单。
- 黑市巷入口有人蹲着磨竹签，实际是望风。

## 6. 第一卷主要动线

赵长生日常：

> 破屋 → 灵井 → 回春丹铺后院 → 废料场 → 破屋

赵长生试探交易：

> 破屋 → 旧货街边缘 → 灵米铺 → 绕路回棚区

沈临风找赵长生：

> 东门 → 青石主街 → 系统提示 → 回春丹铺 → 后院

被盯梢：

> 旧货街 → 黑市巷 → 棚区 → 废料场

卷尾逃亡：

> 回春丹铺后巷 → 黑市巷 → 废料场 → 北小门 → 枯石岭
