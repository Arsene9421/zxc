# 下界修仙国与古战场地图

> 地图地理补档。本文用于承载第三卷到第五卷的修仙国坊市、筑基资源竞争、洞府废山、修仙国大战、天机阁取证、古战场残魂、断仙门残路和进入灵界前的空间逻辑。本文不是卷纲，也不是战役细纲。

---

## 1. 设计目的

下界后段地图要解决六个问题：

1. 第三卷为什么从青云门外门自然扩展到修仙国坊市、拍卖会、黑市和修仙家族。
2. 筑基资源为什么会被家族、宗门、暗庄和丹师掮客垄断，而不是随便买到。
3. 第四卷洞府为什么不是安全种田，而是卡在宗门、散修盟、废灵脉和产权契约之间。
4. 第五卷修仙国大战为什么不是热血国战，而是灵矿、国运、古战场残痕和飞升异常共同引爆。
5. 天机阁、古战场、断仙门残痕为什么能在空间上汇合。
6. 下界进入灵界为什么不能写成正常飞升，而要通过残破登仙台、界裂或天裂台残路冒险离开。

这张地图的核心不是“世界变大”，而是：赵长生从青石坊市的一粒废丹，逐步看见整个下界都处在被抽干、被误导、被回收的边缘。

## 2. 区域总览

下界后段主舞台暂定为：

> 南岐修仙国。

青石坊市和青云门位于南岐修仙国东南边缘。南岐不是凡俗王朝，也不是大型仙朝，而是由修仙宗门、修仙家族、灵矿城、军修府和凡俗城池混合支撑的小型修仙国。

总关系：

```text
                         北澜修仙国方向
                                ↑
                         玄河战线 / 赤岭关
                                ↑
                         断星原古战场
                                ↑
                         天裂台残路
                                ↑
       天机阁南岐分阁 —— 南岐王城 —— 玄河主坊市
                                ↑
              罗氏赤霞药谷 —— 筑基拍卖场
                                ↑
                         灰鹤废山洞府
                                ↑
                  青云门 —— 小青云山 —— 青石坊市
```

地图分卷使用：

| 卷 | 主要空间 | 功能 |
|----|----------|------|
| 第三卷 | 南岐腹地、玄河主坊市、筑基拍卖场、罗氏赤霞药谷 | 筑基资源、拍卖暗庄、修仙家族垄断 |
| 第四卷 | 灰鹤废山、洞府灵脉、散修盟路税、灵契坊 | 洞府经营、护山阵、灵田闭环、产权冲突 |
| 第五卷 | 玄河战线、赤岭关、断星原古战场、天机阁南岐分阁、天裂台残路 | 修仙国大战、天机污染、古战场证据、断仙门 |

## 4. 第三卷空间：玄河主坊市

玄河主坊市是南岐修仙国腹地最大的低中阶交易中心。

它不比青石坊市“干净”，只是压迫方式更成熟。

组成：

1. 主街商铺。
2. 筑基拍卖场。
3. 丹师掮客楼。
4. 修仙家族会馆。
5. 黑市暗巷。
6. 散修客栈。
7. 护送行会。
8. 天机阁符简铺。
9. 城门验牌处。

和青石坊市的区别：

| 青石坊市 | 玄河主坊市 |
|----------|------------|
| 低阶废料、散修小摊为主 | 筑基主药、法器、丹方和家族交易为主 |
| 追杀靠帮派和散修 | 压迫靠账、拍卖、担保和暗庄 |
| 废料链显露底层压迫 | 筑基资源显露阶层垄断 |
| 赵长生靠熟悉地形求生 | 赵长生要学会在更复杂规则中布局 |

### 4.1 筑基拍卖场

筑基拍卖场不是公平市场。

空间设置：

| 区域 | 用途 | 可写冲突 |
|------|------|----------|
| 前厅验资处 | 验灵石、担保、身份 | 低阶散修连入场资格都难拿 |
| 散席区 | 普通修士竞价 | 容易被抬价和盯梢 |
| 家族包厢 | 罗氏、宗门、商盟包厢 | 资源未开拍已被内定一半 |
| 暗账房 | 押金、欠款、暗庄记录 | 苏明河可在此出场 |
| 后货仓 | 拍品临时存放 | 抢夺、换货、泄露买主信息 |
| 离场暗道 | 贵客离场通道 | 赵长生可利用但不能随便掌控 |

筑基资源必须有具体门槛：

1. 入场验资。
2. 担保人。
3. 押金。
4. 家族预定。
5. 暗庄抬价。
6. 事后盯梢。

### 4.2 黑市暗巷

玄河主坊市黑市比青石黑市更成体系。

交易：

1. 筑基主药消息。
2. 假丹方。
3. 残缺古丹炉。
4. 洞府契。
5. 无主储物袋。
6. 旧战场残器。
7. 天机阁未公开的卦辞抄本。

风险：

1. 秦璧这类丹师掮客倒卖假丹方。
2. 苏明河可泄露买主信息。
3. 假古物中可能混入真旧器残片，但也可能有污染。
4. 沈临风系统会把部分机缘标成高价值，但不提示后续账。

### 4.3 护送行会

护送行会连接第三卷和第五卷。

第三卷用途：

1. 关沉舟等护送修士出场。
2. 赵长生雇人或加入队伍进入赤霞药谷、灰鹤废山方向。
3. 展示散修不是全员劫修，也有专业路线。

第五卷回收：

1. 护送修士被征调为战线军修。
2. 旧护送路线变成战备路线。
3. 曾经的商路成为逃难路。

## 5. 罗氏赤霞药谷

罗氏赤霞药谷位于玄河主坊市以西，是第三卷筑基资源垄断的核心地点。

资源：

1. 筑基丹主药之一。
2. 赤霞芝、凝露草、养脉花等低中阶丹材。
3. 药谷灵泉。
4. 家族丹房。
5. 旁支药田。
6. 药奴契和外包灵农。

空间结构：

```text
罗氏会馆
   ↑
赤霞药谷外门
   ↑
旁支药田 —— 药奴棚
   ↑
主脉灵泉 —— 丹材封库
   ↑
内谷禁田
```

可写冲突：

1. 罗氏主脉控制主药，旁支和外姓灵农承担种植风险。
2. 罗景衡争筑基资源，不只是个人挑衅，而是家族名额压力。
3. 林照这类旁支女修可展示家族内部也有人被牺牲。
4. 叶疏彤能看出药谷土性和丹材保存问题，但不能靠一眼解决罗氏垄断。
5. 赵长生可通过废丹、残器、旧炉痕找到破局边角，不能正面抢空药谷。

罗氏的错误不是“有资源”，而是把筑基机会变成家族封闭循环。

## 6. 第四卷空间：灰鹤废山洞府

灰鹤废山位于青云门和玄河腹地之间的边缘废灵脉带。

它表面荒废，实际位置尴尬：

1. 离青云门不远，宗门能伸手。
2. 离玄河商路不远，散修盟能收路税。
3. 地下有半死灵脉，适合洞府经营。
4. 附近有旧器痕、废矿洞和小水脉，适合埋归墟炉后续线索。

总布局：

```text
青云门方向
   ↓
灰鹤山口 —— 散修盟路卡
   ↓
废矿坡 —— 旧器痕石壁
   ↓
洞府主院 —— 灵圃台 —— 药圃水渠
   ↓              ↓
护山阵基 —— 退路暗沟 —— 后山废洞
   ↓
玄河商路方向
```

### 6.1 洞府主院

洞府主院不能写成主角空间。

它至少需要：

1. 主修静室。
2. 丹房小炉间。
3. 器物修补间。
4. 灵圃台。
5. 仓房。
6. 护山阵眼。
7. 退路暗沟。
8. 临时安置棚。

维护成本：

1. 灵石。
2. 阵旗。
3. 阵钉。
4. 水渠修护。
5. 药圃人手。
6. 巡山人。
7. 契约税。
8. 与散修盟、青云门、灵契坊的关系成本。

### 6.2 灵圃台

灵圃台承接叶疏彤线。

作用：

1. 把叶氏灵田经验升级为洞府灵圃经营。
2. 展示残土复苏、丹材保存、赤纹灵禾改良。
3. 给第五卷灾民和伤员安置埋伏笔。
4. 后续进入灵界时，洞天灵圃可以成为界压缓冲的一部分，但不能无代价带所有人飞升。

限制：

1. 灵圃产出有限。
2. 土性修复慢。
3. 灵泉不稳。
4. 战争来临时，灵圃会成为救人资源，也会成为被抢资源。

### 6.3 护山阵和退路

护山阵承接陆维舟线。

组成：

1. 外层迷阵。
2. 内层护田阵。
3. 主院防护阵。
4. 退路暗沟阵。
5. 后山废洞封禁。
6. 临时撤离阵点。

陆维舟在第四卷的成长不是布出强阵，而是第一次把阵法当成一群人的退路。

限制：

1. 阵法要维护。
2. 阵材会坏。
3. 地脉不稳会反噬。
4. 大战时阵法不能保住所有方向。

### 6.4 散修盟路卡

方啸林控制灰鹤山口和玄河商路之间的路卡。

作用：

1. 收路税。
2. 控制护送线。
3. 出售消息。
4. 有时保护散修，有时勒索散修。

方啸林可以被击败后转灰色合作，但不能突然忠心。他代表的是边缘地带的现实秩序：没有宗门直接管，就由能控路的人定规矩。

### 6.5 灵契坊

灵契坊不一定是单独城池，可设在玄河主坊市和灰鹤废山之间的契约据点。

孟槐可在此出场。

功能：

1. 洞府契。
2. 灵田契。
3. 债契。
4. 雇佣契。
5. 避战迁居契。

灵契坊的意义是：让“洞府归谁、灵田归谁、战后赔谁”都有文字和代价，不让洞府经营变成口头占山。

## 7. 第五卷空间：南岐王城与玄河战线

南岐王城是第五卷大战调度入口。

王城不是主要爽点舞台，只承担大势、军令和天机阁公开结果。

组成：

1. 国主府。
2. 军修府。
3. 天机阁南岐分阁。
4. 商盟总会。
5. 战备丹库。
6. 征调榜。
7. 难民外城。

可写冲突：

1. 萧问甲操盘战争，把灵矿、国运和宗门利益绑在一起。
2. 天机阁战前推演部分正确，关键处错位。
3. 青云门、罗氏、散修盟都被卷入战争，却各自想保自己资源。
4. 叶疏彤看到战争最先伤到灵田、凡城和低阶修士。
5. 顾长庚看到宗门正统在大战中未必站在弱者一边。

### 7.1 玄河战线

玄河战线位于南岐和北澜修仙国交界。

组成：

1. 赤岭关。
2. 玄河渡口。
3. 灵矿带。
4. 战阵台。
5. 伤兵营。
6. 军修粮药仓。
7. 凡城避难道。

战争不是简单正邪：

| 表层理由 | 深层利益 |
|----------|----------|
| 北澜犯境 | 灵矿带归属 |
| 南岐守国 | 国运和宗门资源重分 |
| 天机阁劫兆 | 推演被污染和人为利用 |
| 古战场异动 | 残阵、残魂和断仙门证据即将暴露 |

卫从戎适合在赤岭关或伤兵营出场。他要让读者看见：大战里也有人认真守凡城和低阶修士，不是所有军修都只为上层利益。

### 7.2 战备路线

战备路线把第三、第四卷旧空间全部卷进第五卷。

```text
玄河主坊市
→ 南岐王城
→ 赤岭关
→ 断星原外缘
```

支线：

```text
青云门
→ 灰鹤废山洞府
→ 散修盟路卡
→ 玄河战线伤兵营
```

用途：

1. 让洞府不再是安全后方。
2. 让叶疏彤的灵圃变成救伤资源。
3. 让陆维舟的护山阵、撤离阵和临时战阵进入实战。
4. 让赵长生面对“救自己团队”和“救更多低阶修士”的资源分配压力。

## 8. 断星原古战场

断星原古战场位于玄河战线更北，表面是旧修仙国大战遗址，深层与旧登仙台断裂、飞升异常和天劫验源有关。

地形：

1. 黑色碎石原。
2. 半埋战旗。
3. 残阵沟壑。
4. 焦雷坑。
5. 枯魂林。
6. 旧碑群。
7. 断裂高台阴影。

可用区域：

| 区域 | 内容 | 剧情用途 |
|------|------|----------|
| 外缘拾荒区 | 残器、破甲、旧符片 | 展示古战场经济和危险 |
| 残阵沟壑 | 错乱阵纹、断裂阵眼 | 陆维舟看见界阵雏形 |
| 焦雷坑 | 天劫残痕、验源痕迹 | 赵长生确认天劫不只是雷劈 |
| 枯魂林 | 残魂、魂火、旧怨 | 洛闻霜线远期伏笔，楚钧可出场 |
| 旧碑群 | 飞升、守界、登仙旧字 | 谢未明取证，顾长庚正统动摇 |
| 断裂高台阴影 | 天裂台残路外显 | 卷尾进入灵界前置 |

古战场风险：

1. 残阵误触。
2. 神魂污染。
3. 旧器残片引发归墟炉异常。
4. 天机推演错位。
5. 战争双方都想封锁证据。
6. 古战场拾荒者可能既是向导，也是卖消息者。

## 9. 天机阁南岐分阁

天机阁南岐分阁位于南岐王城外侧高台，不直接隶属国主府。

组成：

1. 观星台。
2. 命册房。
3. 卦房。
4. 古卷库。
5. 外勤锚物库。
6. 公告碑。
7. 封存错算记录的小库。

空间作用：

1. 谢未明正式登场。
2. 闻人策维护天机阁权威。
3. 战前推演发布给南岐各宗门和家族。
4. 古战场锚物从外勤带回，在这里与命册副录互证。
5. 赵长生命数断页在命册房表现为空页、断线或错页。

天机阁不能解决所有问题。

它最多能证明：

1. 推演被污染。
2. 赵长生命数无法连续记录。
3. 飞升、天劫、断仙门存在异常。
4. 古战场证据与旧卷相互印证。

它不能解释完整新仙界补墟体系。

## 10. 天裂台残路

天裂台是断星原深处的旧登仙台残址之一。

它不是完整登仙台，也不是安全传送阵。

组成：

1. 断裂台基。
2. 残接引阵。
3. 界裂缝。
4. 旧天劫痕。
5. 半毁碑文。
6. 失效守台禁制。
7. 飞升失败者残影。

空间功能：

1. 卷尾证明正常仙门已断。
2. 让赵长生确认飞升不是奖励，而是被筛选和回收。
3. 让沈临风系统第一次严重阻止追查。
4. 让谢未明的天机证据和现场证据互证。
5. 让陆维舟的阵法线第一次接触登仙台节点。
6. 让叶疏彤的洞天灵圃成为抵抗界压和保护同伴的一部分。
7. 让顾长庚看见正统飞升叙事崩塌。

进入灵界方式：

| 方案 | 用法 | 限制 |
|------|------|------|
| 残破登仙台借道 | 借旧接引阵残留方向进入灵界 | 风险极高，落点不可控 |
| 界裂强行穿越 | 通过界裂缝离开下界 | 伤肉身、神魂和洞府灵圃 |
| 归墟炉稳残路 | 归墟炉只稳住残缺节点，不负责传送 | 可能暴露同源波纹，引来后续追查 |
| 陆维舟临时补阵 | 临时修补一个可承载数人的阵口 | 材料不足，无法大规模迁移 |
| 叶疏彤灵圃护生 | 以生机缓冲界压 | 消耗灵圃根基，不是免费携带 |

卷尾禁用正常飞升。主角组应是借道、伪装、强行破局或被界裂卷入灵界，不是荣耀登仙。

## 11. 主角团卷际动线

### 11.1 赵长生

```text
青云门废料谷
→ 玄河主坊市黑市暗巷
→ 筑基拍卖场
→ 灰鹤废山洞府
→ 断星原焦雷坑
→ 天裂台残路
```

赵长生的路线从资源捡漏，升级到洞府经营，再升级到飞升异常证据。

### 11.2 沈临风

```text
青云门任务碑
→ 玄河主坊市机缘任务
→ 筑基拍卖场系统奖励异常
→ 南岐王城战前任务
→ 天裂台系统强制阻止
```

沈临风的路线要让系统越来越像“带账的导航”，不是单纯辅助。

### 11.3 叶疏彤

```text
青云门灵田坡
→ 罗氏赤霞药谷
→ 灰鹤废山灵圃台
→ 玄河战线伤兵营
→ 天裂台界压缓冲
```

叶疏彤的路线要从灵田产量，走到洞府生机，再走到战争救伤和界压护生。

### 11.4 陆维舟

```text
青云门阵堂坞
→ 玄河护送路线
→ 灰鹤废山护山阵
→ 赤岭关临时战阵
→ 断星原残阵沟壑
→ 天裂台残接引阵
```

陆维舟的路线要从修盘房责任，升级到洞府退路，再升级到界阵和登仙台节点。

### 11.5 顾长庚

```text
青云门剑堂峰
→ 筑基资源竞争
→ 南岐王城征调榜
→ 赤岭关战线
→ 旧碑群
→ 天裂台残碑
```

顾长庚的路线要让他从宗门正统，走到战争现实，再看到飞升正统叙事破裂。

### 11.6 谢未明

```text
天机阁南岐分阁
→ 命册房
→ 古卷库
→ 断星原取证
→ 天裂台残路
```

谢未明的路线不是“神算登场”，而是从错算记录、命册断页和现场证据中确认天机污染。

## 12. 关键路线

### 12.1 第三卷筑基资源路线

```text
青云门
→ 玄河主坊市
→ 筑基拍卖场
→ 黑市暗巷
→ 罗氏赤霞药谷
→ 灰鹤废山方向
```

用途：

1. 证明筑基资源被家族和市场共同垄断。
2. 让赵长生不靠正统宗门资源筑基。
3. 让叶疏彤、顾长庚、沈临风都面对资源规则。

### 12.2 第四卷洞府经营路线

```text
青云门
→ 灰鹤废山
→ 灵契坊
→ 散修盟路卡
→ 玄河主坊市补给线
→ 洞府主院
```

用途：

1. 证明洞府经营不是安全种田。
2. 让宗门、散修盟、灵契坊和商路都能施压。
3. 让团队分工真正落地。

### 12.3 第五卷断仙门路线

```text
灰鹤废山洞府
→ 南岐王城
→ 天机阁南岐分阁
→ 赤岭关
→ 断星原古战场
→ 天裂台残路
→ 灵界未知落点
```

用途：

1. 让战争和天机阁自然接上。
2. 让古战场证据不是孤立秘境。
3. 让断仙门和进入灵界成为第五卷空间终点。

## 14. 战争机制边界

第五卷战争不能写成普通两国争霸。

战争至少有四层：

1. 表层：南岐和北澜争灵矿、关隘和旧战场边界。
2. 中层：宗门、家族、商盟借战争重分资源。
3. 深层：天机阁推演被污染，战争被引向断星原。
4. 暗层：古战场和天裂台残路暴露飞升异常。

战争禁写：

1. 不写大军无脑冲锋。
2. 不让赵长生一个人决定国战胜负。
3. 不让叶疏彤、陆维舟、顾长庚只围观。
4. 不让战争操盘者知道完整仙界真相。
5. 不让天机阁一句卦辞直接解释断仙门。
6. 不把低阶军修和凡人城池当背景数字。

## 15. 禁写项

1. 不能把南岐修仙国写成大型仙朝。
2. 不能让拍卖会变成公平商店。
3. 不能让罗氏只因看主角不顺眼而争资源。
4. 不能让灰鹤废山变成无成本主角基地。
5. 不能让洞府灵圃无限产出资源。
6. 不能让修仙国大战只为主角升级服务。
7. 不能让天机阁完整剧透仙界补墟体系。
8. 不能让古战场残魂解释全部真相。
9. 不能让天裂台成为稳定传送门。
10. 不能让正常飞升在第五卷成立。
11. 不能让主角组全员无代价进入灵界。
12. 不能让下界人物在第五卷后像用完即弃一样消失。

## 16. 使用提示

第三到第五卷写地图时，要保持空间递进：

1. 第三卷：从宗门资源规则，进入修仙国市场和家族垄断。
2. 第四卷：从买资源，进入经营资源节点和承担维护成本。
3. 第五卷：从经营一处洞府，进入战争、天机、古战场和飞升异常。

下界后段的核心味道不是“地图越开越大”，而是：

> 赵长生走得越远，越发现当初青石坊市那点废料、灵米和炉灰，不是底层个案，而是整个下界被抽空之前最细小的回声。
