# 青云门

> 主角团背后组织补档之一。青云门是下界第二到第五卷的主要宗门压力来源，不是纯反派宗门，也不是主角升级福利点。它的作用是把灵田、丹房、阵法、剑堂、藏经阁、贡献点和筑基资源组织成一套可写的宗门秩序。
>
> 宗门空间、外门动线、山门门禁、灵田坡、丹房谷、阵堂坞、剑堂峰、藏经阁、后山废料谷和试炼林地理见：`01设定库/02_地理势力/02_地图地理/04_青云门宗门地理.md`。

---

## 1. 基础定位

| 项目 | 内容 |
|------|------|
| 名称 | 青云门 |
| 名字状态 | 定稿可用 |
| 所属区域 | 下界，青石坊市上宗体系，第二卷正式进入 |
| 主要使用卷 | 第二卷到第五卷 |
| 势力层级 | 下界小中型宗门，能压住坊市和周边家族，但不是一域霸主 |
| 核心资源 | 外门资格、贡献点、灵田、丹房、阵法材料、筑基名额、藏经残卷 |
| 叙事功能 | 宗门资源秩序、正统资格、外门生存、主角团初次成形 |
| 代表人物 | 叶疏彤、陆维舟、顾长庚、莫守廷、孟玉娘、邹连城、廖苍 |

青云门不能写成“全员坏人宗门”。

它真正的压迫来自：规矩看似公平，但进入规矩的资格、获取贡献点的路径、承受失败的成本，并不公平。

## 2. 宗门结构

| 层级 | 组成 | 关注点 | 剧情用途 |
|------|------|--------|----------|
| 掌门与长老议事层 | 掌门、执法长老、庶务长老、丹房长老、剑堂长老、阵堂长老 | 宗门存续、资源分配、外部压力 | 不必频繁登场，但决定制度方向 |
| 内门 | 筑基以上弟子、核心技艺弟子、重点剑修 | 筑基资源、功法传承、门内名额 | 顾长庚正统压力来源 |
| 外门 | 炼气中后期弟子、灵田、阵法、丹房杂务人员 | 贡献点、任务、晋升资格 | 叶疏彤、陆维舟、赵长生进入宗门秩序的主要舞台 |
| 杂役与附庸 | 灵农家族、阵坊旁支、坊市供货者、低阶杂役 | 活路、债务、契约、被克扣 | 证明宗门秩序向下传导压力 |
| 外围坊市 | 青石坊市、回春丹铺、旧货摊、灵米铺 | 废料、丹药、低阶市场 | 第一卷和第二卷之间的资源链接口 |

青云门的核心不是“强”，而是“完整”：它有外门、有岗位、有考核、有账，有足够的制度力量压住底层修士。

## 4. 贡献点与任务规则

青云门外门用贡献点维持表面公平。

常见来源：

1. 灵田产量。
2. 丹房杂务。
3. 护送任务。
4. 修阵修器。
5. 采药、巡山、守矿。
6. 上缴材料和废料。

常见消耗：

1. 功法借阅。
2. 丹药兑换。
3. 灵室修炼。
4. 法器租用。
5. 筑基辅材名额。
6. 外门晋升考核。

问题在于：

| 表面规则 | 实际问题 |
|----------|----------|
| 多劳多得 | 高收益任务需要资格、担保和背景 |
| 贡献可查 | 任务失败扣分，底层承受不起 |
| 兑换公开 | 稀缺资源常被内门和家族预定 |
| 外门可晋升 | 外门弟子要先活过长期低收益任务 |
| 废料统一处理 | 废丹、残器、炉灰会被管事和丹房二次盘剥 |

赵长生的第一层爽点不是“打脸宗门”，而是在这套账里找到别人看不上的残缺价值。

## 5. 宗门内部派系

| 派系 | 代表位置 | 主张 | 可写冲突 |
|------|----------|------|----------|
| 守成派 | 掌门、庶务长老、部分执事 | 先保宗门资源和外部安全 | 会把底层压力合理化 |
| 技艺派 | 丹房、阵堂、灵田院部分修士 | 技艺能养活宗门，但需要材料和失败空间 | 与贡献点短期考核冲突 |
| 正统派 | 剑堂、传功线 | 宗门名声和正道资格必须维护 | 顾长庚前期受此影响 |
| 灰账派 | 部分管事、丹房中层、废料链联系人 | 废料、旧货、暗账都是实际利益 | 连接青石坊市和赵长生捡漏线 |
| 求活派 | 外门弟子、杂役、附庸家族 | 只想拿到足够资源继续修行 | 能展示宗门底层群像 |

派系之间不必频繁开会，重点是让不同岗位的人做出不同选择。

## 8. 卷际使用

| 卷 | 用法 | 重点 |
|----|------|------|
| 第一卷 | 只以“上宗”“外门管事”“废料链”形式存在 | 不提前展开宗门全貌 |
| 第二卷 | 正式进入青云外门，叶疏彤、陆维舟、顾长庚登场 | 外门任务、灵田、阵堂、剑堂 |
| 第三卷 | 筑基资源竞争扩大，宗门和修仙家族利益交错 | 贡献点不等于公平 |
| 第四卷 | 洞府经营和宗门权力冲突 | 洞府产权、灵脉争夺、阵法合同 |
| 第五卷 | 修仙国大战和断仙门前，青云门暴露正统局限 | 宗门无法回答飞升异常 |

## 10. 禁写项

1. 不能写成纯反派宗门。
2. 不能写成免费修炼学院。
3. 不能让掌门长老过早知道归墟炉或仙界补墟真相。
4. 不能让顾长庚因为宗门问题变成恶少。
5. 不能让叶疏彤、陆维舟的专业性被青云门背景吞掉。
6. 不能用一句“宗门腐败”解释所有压迫，必须落实到任务、贡献点、灵田、丹房、阵堂和剑堂。

## 11. 使用提示

写青云门时，每个冲突最好落在一项具体制度上：

1. 贡献点不够。
2. 灵田产量未达标。
3. 废丹去向被查。
4. 阵盘坏账无人承担。
5. 剑堂名额被预定。
6. 藏经阁残卷权限不足。

青云门的存在是为了证明：

> 下界宗门不需要人人作恶，也能通过规矩、资格和账目，把底层修士压得喘不过气。
