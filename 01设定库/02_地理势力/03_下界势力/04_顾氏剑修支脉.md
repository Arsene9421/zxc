# 顾氏剑修支脉

> 主角团背后组织补档之一。顾氏剑修支脉是顾长庚的正统压力来源，不是恶少家族。它的作用是把剑堂资格、宗门名声、家族期待、正统功法和顾长庚后期的斩道选择绑定起来。

---

## 1. 基础定位

| 项目 | 内容 |
|------|------|
| 名称 | 顾氏剑修支脉 |
| 名字状态 | 定稿可用；对外可称青崖顾氏支脉 |
| 所属区域 | 青云门剑堂体系内的修仙家族支脉 |
| 主要使用卷 | 第二卷、第三卷、第五卷，第十二卷以后回照 |
| 势力层级 | 小型剑修支脉，依附青云门剑堂，不是顶级世家 |
| 核心资源 | 剑堂推荐、正统剑诀、剑器养护、同代名额、家族名声 |
| 核心压力 | 必须守住正统门面，不能输给无资格者，不能让家族剑名蒙尘 |
| 代表人物 | 顾长庚，族中长辈和剑堂师长可卷纲阶段具名 |

顾氏剑修支脉不能写成“靠家世欺负主角”的家族。

它真正的压迫来自：它把“正统”当成责任，也当成门槛。顾长庚前期最大的问题，是还分不清这两者。

## 2. 支脉来历

顾氏原本不是大族，而是青云门剑堂长期培养出来的剑修支脉。

支脉能维持地位，靠三件事：

1. 每代至少有一名能进剑堂核心序列的弟子。
2. 家族剑诀必须保持正统，不走邪路、偏门和过度借外物。
3. 在宗门需要武力门面时，顾氏弟子必须站出来。

这让顾长庚不是享受资源的纨绔，而是被推到正统前台的人。

## 3. 内部结构

| 位置 | 职责 | 可写冲突 |
|------|------|----------|
| 支脉长辈 | 维护家族和剑堂关系，争取推荐名额 | 希望顾长庚稳，不希望他质疑宗门 |
| 剑诀保管人 | 管理家族剑诀、剑谱和历代心得 | 不轻易给旁人看残缺和失败记录 |
| 同代子弟 | 与顾长庚同族竞争或依附 | 有人敬佩他，有人嫉妒他，有人怕他失败拖累全族 |
| 剑器养护人 | 养护低阶飞剑、剑胚、剑匣 | 剑修也需要材料和维护成本 |
| 外务联系人 | 对接青云门剑堂、任务和比试 | 容易把宗门名声压到顾长庚身上 |

顾氏内部可以有压力和短视，但不要写成全员恶毒。

## 4. 正统资源

| 资源 | 作用 | 限制 |
|------|------|------|
| 剑堂推荐 | 让顾长庚进入更好的剑修培养序列 | 推荐不是实力本身，也会形成资格傲慢 |
| 正统剑诀 | 法力运转、剑气、道心训练 | 稳定但路径窄，难容残缺路 |
| 剑器养护 | 飞剑、剑胚、剑匣、磨剑材料 | 剑修不是只凭心气，维护也耗资源 |
| 历代剑谱 | 成功心得和失败记录 | 家族常展示成功，隐藏失败 |
| 正道名声 | 外出任务、比试、宗门信任 | 名声会变成枷锁 |

顾长庚的压迫感来自这些资源真实存在，而不是他嘴上自称天才。

## 5. 与青云门剑堂关系

| 青云门剑堂 | 顾氏支脉 |
|------------|----------|
| 提供正统培养、剑诀进阶、比试平台 | 提供稳定剑修苗子和宗门门面 |
| 需要顾长庚这种同代榜样 | 需要剑堂资格维持家族地位 |
| 看重宗门名声和战斗力 | 看重家族剑名和推荐名额 |
| 不希望核心弟子质疑宗门规则 | 不希望顾长庚走出剑堂安排 |

顾长庚的成长线必须从这里开始：

> 他不是一开始就知道规矩有问题，他曾真心相信规矩能筛出值得托付正道的人。

## 6. 与顾长庚关系

| 阶段 | 支脉影响 | 顾长庚变化 |
|------|----------|------------|
| 第二卷 | 家族和剑堂给他资源、身份、正统资格 | 他自然形成对赵长生残缺路的俯视 |
| 第三卷 | 筑基资源竞争中，家族希望他稳稳拿到名额 | 他第一次看见无资格者也有不得不争的理由 |
| 第五卷 | 修仙国大战和宗门选择冲击正统认知 | 他开始怀疑“宗门规矩”等不等于“正道” |
| 第十二卷 | 旧天庭斩道剑残痕和穆天衡旧军序诱惑出现 | 他必须分清正统职责和旧权力复辟 |
| 第十四卷 | 可成为斩道剑候选执剑者之一 | 不以顾氏名声凌驾众生，只斩不该存在的道职权力 |

顾长庚不能后期完全抛弃家族。更好的写法是：他承认家族给了自己正统底子，也承认家族正统看不见规矩外的人。

## 7. 内部分歧

| 派别 | 主张 | 用法 |
|------|------|------|
| 守名派 | 顾氏必须守住剑堂名额和正统名声 | 给顾长庚现实压力 |
| 清正派 | 剑修正统不能欺弱，名声必须配得上行为 | 可作为顾长庚底线来源 |
| 求功派 | 只要赢、只要拿名额，灰色手段可接受 | 可制造早期家族内部反面样本 |
| 避祸派 | 不要卷入宗门大局和断仙门异常 | 与顾长庚后期选择冲突 |

这些派别不必都具名。卷纲需要时选择一两人落地即可。

## 8. 与其他主角团关系

| 对象 | 顾氏视角 | 冲突或变化 |
|------|----------|------------|
| 赵长生 | 无资格者、残缺路、不可预测 | 顾长庚从俯视到承认其道路有效 |
| 沈临风 | 路数轻浮，机缘来得太便利 | 顾长庚后期承认拒绝天命也是道心 |
| 叶疏彤 | 灵田线同门，早期不被剑修系统重视 | 顾长庚后期承认生机也是正道 |
| 陆维舟 | 技术路线，不够锋利但稳 | 第四卷以后理解“守住退路”也是守道 |
| 秦照古、穆天衡 | 旧天庭正统诱惑 | 顾长庚必须拒绝复辟式正统 |

## 9. 可提供的剧情资源

| 资源 | 用法 | 限制 |
|------|------|------|
| 剑堂比试 | 顾长庚压迫感、赵长生残缺路对照 | 不能写成普通打脸擂台 |
| 正统剑诀 | 顾长庚战斗和道心基础 | 不能让赵长生轻易偷学替代 |
| 历代剑谱 | 显示成功和失败的正统记录 | 失败记录要有价值 |
| 家族名额压力 | 筑基资源和内门资格冲突 | 不等于恶意欺压 |
| 剑器养护成本 | 证明剑修也有资源消耗 | 不能让剑修只靠一腔热血 |

## 10. 卷际使用

| 卷 | 用法 | 状态 |
|----|------|------|
| 第二卷 | 顾长庚登场，剑堂和顾氏支脉提供正统压迫感 | 他不是反派，但天然站在规矩内 |
| 第三卷 | 筑基资源竞争中，顾氏希望他稳拿资源 | 资源公平问题第一次刺到他 |
| 第五卷 | 宗门正统面对战争和断仙门异常失灵 | 顾长庚开始修正道心 |
| 第十二卷 | 旧天庭复辟派和斩道剑残痕诱惑 | 顾氏正统与旧正统对照 |
| 第十三卷以后 | 顾长庚斩断不该存在的道职权力 | 正统从资格转为责任 |

## 11. 禁写项

1. 不能把顾氏写成恶少家族。
2. 不能让顾长庚只靠家世压人。
3. 不能让顾氏正统全错，否则顾长庚没有重量。
4. 不能让顾氏正统全对，否则后期斩道线无从成立。
5. 不能让赵长生轻易夺走顾长庚的剑道专业性。
6. 不能让顾长庚后期变成赵长生附庸，他应认可但不同路。

## 12. 使用提示

写顾氏剑修支脉时，压力最好落在正统细节上：

1. 一个剑堂推荐名额。
2. 一场不能输的比试。
3. 一份家族剑谱只记录胜者。
4. 一柄飞剑需要维护材料。
5. 一句“顾氏子弟不能失了剑堂脸面”。

顾氏支脉的存在是为了证明：

> 正统不是天然坏，但正统若只看资格，就会看不见规矩外的人也在求道。
