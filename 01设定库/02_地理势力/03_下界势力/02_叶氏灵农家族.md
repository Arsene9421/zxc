# 叶氏灵农家族

> 主角团背后组织补档之一。叶氏灵农家族是叶疏彤的原始压力来源，不是单纯贫穷背景。它的作用是把灵田产权、宗门附庸、丹材供给、旧债、生机病灶和叶疏彤的道途绑定起来。

---

## 1. 基础定位

| 项目 | 内容 |
|------|------|
| 名称 | 叶氏灵农家族 |
| 名字状态 | 定稿可用 |
| 所属区域 | 青云门附属灵田区，靠近青云外门 |
| 主要使用卷 | 第二卷到第四卷，后续作为生机旧债回声 |
| 势力层级 | 没落灵农小族，依附青云门生存 |
| 核心资源 | 几亩残灵田、赤纹灵禾种源、低阶丹材保存法、旧灵泉井 |
| 核心压力 | 债契、产量、灵田产权、族中病弱者、宗门考核 |
| 代表人物 | 叶疏彤，族中长辈和幸存亲族可在卷纲阶段具名 |

叶氏不是“祖上很强，现在落魄”的普通模板。

它的价值在于：一个小灵农家族如何被宗门任务、债契和灵脉衰退一点点挤到只能拿族人未来还账。

## 2. 家族来历

叶氏曾是青云门附属灵农家族。

早年职责：

1. 替青云门种植低阶灵米。
2. 试种丹房需要的基础灵草。
3. 看护一口旧灵泉井。
4. 维护灵田土性和季节轮作。
5. 向丹房提供可保存的低阶丹材。

叶氏最有价值的不是战力，而是经验：

1. 知道不同丹材的采收时辰。
2. 知道灵泉浑浊时怎么缓土。
3. 知道废田还能不能救。
4. 知道种子坏到什么程度还值得留。

这些经验构成叶疏彤的专业底子。

## 3. 旧债来源

叶氏旧债不能只写成“欠钱”。

| 债务来源 | 表层原因 | 深层问题 | 剧情用途 |
|----------|----------|----------|----------|
| 灵田修复债 | 旧灵泉井浑浊，需借灵石修井和换土 | 灵脉衰退不是叶氏能力不足 | 叶疏彤生机线开端 |
| 种源债 | 赤纹灵禾试种失败，种子损耗过大 | 宗门将试种风险压给附属灵农 | 证明生机也会被任务化 |
| 丹材赔付债 | 丹材保存失败，被丹房压价和罚扣 | 丹房考核只看成品，不看灵田病灶 | 连接邹连城、丹房线 |
| 筑基失败债 | 族中长辈借资源冲击筑基失败 | 小族一次失败会拖垮下一代 | 让修行风险落到家庭 |
| 债契转卖 | 外部灵契师或宗门账房将旧债拆分转卖 | 欠债人不知道自己真正欠谁 | 第四卷洞府契约线伏笔 |

叶疏彤的压力不是“我家穷”，而是：

> 灵田少一成产量，债就滚一层；债滚一层，族里就少一个人能继续修。

## 4. 家族结构

| 位置 | 功能 | 可写冲突 |
|------|------|----------|
| 名义家主 | 对接青云门灵田院和债契 | 不一定坏，但长期选择守成 |
| 灵田老手 | 记得旧灵泉、土性、种源变化 | 可提供经验，也可能不信叶疏彤新法 |
| 年轻族人 | 想离开灵田去外门争资格 | 觉得种田没有前途 |
| 病弱族人 | 被灵田病气、丹材毒性或突破失败拖累 | 让叶疏彤不能轻易抛下家族 |
| 债契联系人 | 与青云门庶务、丹房、灵契师对接 | 可能隐瞒真实债额或被迫签坏约 |

卷纲阶段可视需要给族中长辈具名。当前先锁功能，不急着堆亲属名单。

## 5. 灵田和种源

### 5.1 残灵田

叶氏灵田的问题：

1. 灵气不稳。
2. 土性偏枯。
3. 灵泉偶尔泛浊。
4. 赤纹灵禾成活率下降。
5. 丹材药性不均，容易被丹房压价。

表层看，是叶氏经营不善。

深层可在后续揭开：下界灵脉和界源本就处在衰退趋势中，叶氏只是最早感受到的人之一。

### 5.2 赤纹灵禾

赤纹灵禾是叶疏彤名字意象的落点。

| 项目 | 内容 |
|------|------|
| 品阶 | 低阶灵禾，不是神物 |
| 特点 | 枯叶见光后会泛出极淡彤色 |
| 用途 | 可做低阶灵米，也可入温养类丹材 |
| 难点 | 对土性、灵泉和采收时辰敏感 |
| 叙事作用 | 证明叶疏彤的生机道不是凭空催生，而是疏通残败生机 |

赤纹灵禾不能写成超级灵植。它应当是能让读者看懂的低阶资源爽点。

## 6. 与青云门关系

| 青云门部门 | 叶氏关系 | 冲突点 |
|------------|----------|--------|
| 灵田院 | 叶氏直属考核部门 | 产量、轮作、种源、灵泉修复 |
| 丹房 | 收购叶氏低阶丹材 | 压价、保存失败、药性不均 |
| 庶务库 | 登记债契和贡献抵扣 | 债额计算、利滚利、抵押灵田 |
| 外门执事堂 | 叶氏族人进入外门的入口 | 外门名额有限，家族不能都上岸 |
| 阵堂 | 修护灵田小阵和灵泉阵 | 阵材不足，修阵费用继续加债 |

青云门不是刻意毁叶氏。它只是把宗门成本自然往下压，叶氏这种附属家族最先被压垮。

## 7. 与叶疏彤关系

| 阶段 | 关系 | 叶疏彤的选择 |
|------|------|--------------|
| 初期 | 她是叶氏最能救田的人，也是最不能离开的人 | 习惯一个人扛债和产量 |
| 第二卷 | 通过青云外门灵田任务与赵长生接触 | 学会把问题拆成可验证的药性、土性、灵泉 |
| 第三卷 | 筑基资源争夺让叶氏旧债和个人道途冲突 | 不能为了家族债把自己卖进宗门规则 |
| 第四卷 | 洞府灵圃建立后，叶氏经验转化为团队资源闭环 | 从替叶氏续命，转向建立更大的生机循环 |
| 后期 | 灵界祖地、残界、生机旧债都能回照叶氏灵田 | 明白一块小田和一个世界的病灶有相通之处 |

叶疏彤不能因为爱上赵长生就“摆脱家族包袱”。她是把家族旧债看清、整理、偿还或切断不合理部分。

## 8. 可提供的剧情资源

| 资源 | 用法 | 限制 |
|------|------|------|
| 低阶灵田 | 种田经营、丹材供给、灵气病灶展示 | 产量有限，不能变成无限资源点 |
| 赤纹灵禾 | 叶疏彤专业爽点和名字意象 | 低阶资源，不可神化 |
| 丹材保存法 | 连接丹房、赵长生废丹线 | 保存失败也要有成本 |
| 旧灵泉井 | 灵脉病灶和界源衰退伏笔 | 不提前解释仙界抽源 |
| 债契 | 现实压力、灵契师、洞府产权线 | 债务必须具体到谁持有、怎么滚 |

## 9. 卷际使用

| 卷 | 用法 | 状态 |
|----|------|------|
| 第二卷 | 叶疏彤登场，叶氏旧债和灵田产量压力明确 | 家族是负担，也是专业来源 |
| 第三卷 | 筑基资源和旧债冲突扩大 | 叶疏彤不能被家族债卖掉道途 |
| 第四卷 | 洞府灵圃建立，叶氏经验被转化 | 家族线可阶段缓解，但不能完全消失 |
| 第五卷 | 修仙国大战中灵田、灾民和生机责任扩大 | 叶疏彤从家族责任走向下界生机责任 |
| 第六卷以后 | 灵界祖地和残界修复回照叶氏小田 | 小田经验升级为补界生机 |

## 10. 禁写项

1. 不能把叶氏写成拖累女主的极品亲族集合。
2. 不能让叶疏彤靠嫁给赵长生解决家族债。
3. 不能把赤纹灵禾写成逆天神物。
4. 不能让叶氏旧债只是一句欠钱，必须有灵田、种源、丹材、筑基失败和债契转卖。
5. 不能让叶疏彤为了家族无限牺牲自己的道途。
6. 不能让叶氏过早知道界源、补墟司和仙界真相。

## 11. 使用提示

写叶氏时，压力要落到小事：

1. 一袋种子坏了三成。
2. 一口灵泉浑了半夜。
3. 一亩灵田少收一成。
4. 一批丹材被丹房压价。
5. 一张债契转到陌生人手里。

叶氏的存在是为了证明：

> 生机不是口号，是有人在一亩快死的灵田前，知道每一粒种子为什么活不下来。
