# 天机阁下界分支

> 主角团背后组织补档之一。天机阁下界分支是谢未明的原始组织压力来源，不是万能情报机构。它的作用是把天机推演、命数污染、保守派权威、断仙门证据和新司命台污染的早期痕迹组织起来。
>
> 南岐修仙国、天机阁南岐分阁、玄河战线、断星原古战场和天裂台残路的空间位置见：`01设定库/02_地理势力/02_地图地理/05_下界修仙国与古战场地图.md`。

---

## 1. 基础定位

| 项目 | 内容 |
|------|------|
| 名称 | 天机阁下界分支 |
| 名字状态 | 定稿可用 |
| 所属区域 | 下界修仙国和宗门之间的半独立推演组织 |
| 主要使用卷 | 第五卷正式登场，前四卷可用传闻、符简、卦辞轻铺 |
| 势力层级 | 下界高阶情报和推演组织分支，不是全知天道机构 |
| 核心资源 | 星盘、命册副录、因果锚物、推演术、古战场残痕记录 |
| 核心压力 | 天机权威失效、推演结果被污染、保守派不愿承认错算 |
| 代表人物 | 谢未明、闻人策、年轻天机弟子 |

天机阁下界分支的核心矛盾不是“算得准不准”，而是：

> 当推演被污染以后，天机阁是承认自己看见的东西可能是假天机，还是继续维护权威？

## 2. 组织来历

天机阁不是青云门这种地方宗门。

它更接近跨区域的推演和情报组织，在下界各修仙国、宗门和古战场附近设有分支。

下界分支的公开职责：

1. 观测灾兆。
2. 推演修仙国大势。
3. 记录宗门、家族、王朝气运变化。
4. 为大战、兽潮、秘境、古战场异常提供警示。
5. 保存飞升、天劫、断仙门相关传闻碎片。

实际问题：

1. 它只能看见片段，不知道仙界完整真相。
2. 它的部分天机通道已经被新司命台污染。
3. 它越相信旧有推演权威，越容易把污染结果当成真相。
4. 它能发现异常，但内部未必愿意公开异常。

## 3. 组织结构

| 部门/位置 | 职责 | 可写冲突 |
|-----------|------|----------|
| 观星台 | 记录星象、天劫异动、灾兆方位 | 星象异常越来越多，但解释体系还停在旧经验 |
| 命册房 | 保存命牌副录、宗门名册、气运变动记录 | 赵长生断页导致命册空白或错位 |
| 卦房 | 负责具体推演、象辞、吉凶判断 | 推演结果可能被污染，还要承担反噬 |
| 外勤行走 | 进入战场、秘境、宗门收集锚物 | 证据最真实，但最容易被上层压下 |
| 古卷库 | 保存飞升异常、古战场残魂、断仙门传闻 | 旧记录残缺，容易被保守派解释成野史 |
| 阁务长老 | 管理分支声誉、收费、对外发布结果 | 不愿承认天机阁错算，维护组织权威 |

天机阁下界分支的压迫感来自“权威”。

它不需要派打手追杀主角，只要一句“此人命数有异，恐为劫源”，就能让宗门和修仙国开始防备。

## 4. 内部派系

| 派系 | 代表人物 | 主张 | 错误或风险 |
|------|----------|------|------------|
| 保守权威派 | 闻人策 | 天机阁推演体系仍然可靠，异常可以被解释为个案 | 不愿承认污染，容易把错算转嫁给被推演者 |
| 校错派 | 谢未明 | 推演失败本身也是证据，必须追查污染源 | 容易过度追求算清一切，把人当命数样本 |
| 阁务实用派 | 阁务长老、收费卦师 | 天机服务要维持声誉和收益 | 倾向发布对各方都能接受的模糊结论 |
| 外勤证据派 | 年轻行走、古战场记录者 | 现场痕迹比卦辞可靠 | 地位低，容易被上层压证据 |
| 旧卷派 | 古卷库老人 | 飞升异常和断仙门旧闻不能轻易否定 | 资料残缺，缺少现世证据支撑 |

闻人策不是单纯反派。

他的问题是：他维护的是一套曾经有用的推演权威，但当这套权威被污染后，他仍然要求别人服从结果。

## 5. 推演资源和流程

一次天机阁下界推演通常需要五类材料：

1. 命牌、血样、法器残痕或旧物作为锚点。
2. 星象、地脉、战场残气作为环境参照。
3. 法力、神魂、寿元或灵材作为推演代价。
4. 命册副录和旧卷作为历史对照。
5. 推演者道心稳定，避免被反噬和污染。

常规流程：

| 步骤 | 内容 | 剧情用途 |
|------|------|----------|
| 定问 | 明确要问人、事、地还是大势 | 问得越具体，代价越高 |
| 取锚 | 取命牌、旧物、血痕、法器碎片 | 赵长生命数断页会让锚点失效 |
| 起盘 | 星盘、卦盘、命册副录共同推演 | 可出现象辞、残影、错位 |
| 校验 | 与现场证据、旧卷记录互证 | 谢未明价值所在 |
| 发布 | 对外给出吉凶、趋势或警示 | 保守派可能修改措辞维护权威 |

## 6. 命数污染表现

天机阁下界分支能最早接触到新司命台污染的“下游症状”。

| 表现 | 表层解释 | 深层真相 |
|------|----------|----------|
| 推演总指向飞升 | 飞升是修士大势所趋 | 新司命台把高价值变量引向回收通道 |
| 气运高者劫数变密 | 天命之子多磨难 | 系统和司命台加权后引来更多筛选 |
| 命册出现空页 | 被推演者遮蔽天机 | 赵长生命数断页无法连续记录 |
| 卦辞前后矛盾 | 推演者功力不足 | 天机通道被污染或接口混入司命台结果 |
| 战争大势被误判 | 修仙国气运波动 | 国运、宗门气运被牵引进补墟体系 |
| 飞升异常被淡化 | 古籍传闻不可信 | 断仙门和飞升回收被长期遮掩 |

谢未明的突破点不是“我算得更准”，而是：

> 如果所有推演都在同一个方向上错，那错的可能不是某一次推演，而是天机通道本身。

## 7. 与谢未明关系

| 阶段 | 天机阁对谢未明的影响 | 谢未明的变化 |
|------|----------------------|--------------|
| 初期 | 给他推演术、命册知识和天机权威感 | 相信自己能算清命数污染源 |
| 第五卷 | 推演赵长生失败，命册出现断页 | 第一次承认天机阁体系看不清关键变量 |
| 第七卷到第九卷 | 系统任务和天机推演互相矛盾 | 开始与沈临风互证司命污染 |
| 第十一卷以后 | 新司命台后台证据出现 | 确认天机阁下界分支只是污染下游 |
| 终局 | 参与新司命盘规则 | 从“推演命运”改为“记录危机” |

谢未明不能完全背叛天机阁，也不能盲目维护天机阁。

更稳的路线是：他保留推演术的价值，否定天机阁垄断解释权。

## 8. 与闻人策关系

| 项目 | 内容 |
|------|------|
| 闻人策身份 | 天机阁下界分支保守派或资深推演师 |
| 表层动机 | 维护天机阁权威，避免修仙国大战中各方不再相信推演 |
| 核心恐惧 | 如果承认天机被污染，天机阁多年发布的警示和裁断都要被重查 |
| 与谢未明冲突 | 谢未明认为错算是证据，闻人策认为谢未明动摇根基 |
| 可被击穿点 | 现场证据、古战场残痕、赵长生断页、沈临风系统异常共同证明推演源头不干净 |

闻人策不宜写成一心害谢未明的人。

他最有用的台词方向是：

> 天机阁若今日承认天机可污，明日天下修士还信谁的劫兆？

谢未明必须回答：

> 不承认污染，才是把天下修士继续引向假天机。

| 闻人策 | 保守派对手 | 维护权威与承认污染的冲突 |

## 10. 第五卷使用方式

天机阁下界分支正式使用应集中在第五卷。

| 剧情节点 | 用法 | 结果 |
|----------|------|------|
| 战前推演 | 天机阁判断修仙国大战走势 | 结果部分准确，关键处错位 |
| 推演赵长生 | 命册空页、卦盘断线、锚物反噬 | 谢未明确认命数断页 |
| 闻人策压证 | 保守派要求把异常写成个案 | 谢未明与天机阁权威冲突 |
| 古战场取证 | 外勤证据和残魂记录证明飞升异常 | 天机阁旧卷开始与现实互证 |
| 断仙门残痕 | 推演指向飞升通道异常，但不解释仙界全貌 | 第五卷结尾留下更大疑问 |

第五卷不能让天机阁完整解释仙界补墟体系。

它只能证明：下界看到的天机已经不干净，飞升路和命数记录有问题。

## 11. 可提供的剧情资源

| 资源 | 用法 | 限制 |
|------|------|------|
| 命册副录 | 记录修士命数片段、宗门名册、劫兆 | 对赵长生不完整 |
| 星盘和卦盘 | 推演大势、灾兆、战场吉凶 | 可能被污染或反噬 |
| 古卷残录 | 飞升异常、断仙门、古战场传闻 | 不能直接解释新仙界 |
| 外勤锚物 | 战场残器、命牌碎片、旧天劫痕 | 需要现场验证 |
| 错算记录 | 证明天机阁多次在同一方向上误判 | 谢未明追查污染的关键 |

## 12. 回收方式

| 人物/线索 | 最迟回收点 | 回收方式 |
|-----------|------------|----------|
| 闻人策 | 第五卷或第八卷前后 | 被证据击败，不必死亡，可保留为天机阁权威崩塌回声 |
| 年轻天机弟子 | 第五卷 | 展示推演者也会被污染误导，可作为下界回声 |
| 命册副录 | 第九卷前 | 与登仙台残路、飞升回收互证 |
| 错算记录 | 第十一卷到第十三卷 | 连接新司命台任务模板和命数黑账 |
| 古卷残录 | 第十二卷 | 与旧司命台旧义对照 |

## 13. 禁写项

1. 不能把天机阁写成万能情报局。
2. 不能让谢未明靠推演剧透剧情。
3. 不能让闻人策变成无脑反派。
4. 不能让天机阁下界分支知道完整新仙界补墟真相。
5. 不能让赵长生命数断页等于免疫追踪，天机推演失败后仍可用物证、人证、法力痕迹追查。
6. 不能让天机污染一出现就被彻底解决，它只是新司命台污染的下游症状。

## 14. 使用提示

写天机阁下界分支时，每次出场最好带出一个“推演不可靠但有价值”的细节：

1. 卦辞说对一半，错在关键处。
2. 命册能记别人，记不住赵长生。
3. 星盘指向飞升，却解释不了为何飞升者失踪。
4. 闻人策要求维护结论，谢未明要求保留错算记录。
5. 现场锚物比卦辞更接近真相。

天机阁下界分支的存在是为了证明：

> 天机不是天道本身。当天机被污染，最先崩塌的不是推演术，而是人们对权威解释的依赖。
