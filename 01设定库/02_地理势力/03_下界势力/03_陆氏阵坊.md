# 陆氏阵坊

> 主角团背后组织补档之一。陆氏阵坊是陆维舟的原始压力来源，不是给主角团队免费修阵的后台。它的作用是把低阶阵法、阵材成本、修阵责任、坏阵风险和陆维舟的技术路线绑定起来。

---

## 1. 基础定位

| 项目 | 内容 |
|------|------|
| 名称 | 陆氏阵坊 |
| 名字状态 | 定稿可用 |
| 所属区域 | 青云门外门附属坊区，兼接青石坊市和青云门阵堂订单 |
| 主要使用卷 | 第二卷到第四卷，后续作为陆维舟技术源头回声 |
| 势力层级 | 小型阵法家族和职业作坊，不是大修仙世家 |
| 核心资源 | 低阶阵盘、阵旗、阵材边角料、修阵经验、洞府小阵图 |
| 核心压力 | 宗门订单、坏阵赔付、阵材不足、主脉旁支分配不公 |
| 代表人物 | 陆维舟，阵坊主脉管事和修盘房学徒可卷纲阶段具名 |

陆氏阵坊的重点不是“阵法世家天才”，而是职业作坊的责任感：

> 一个阵盘省错一枚灵钉，死的可能是守夜的低阶修士。

## 2. 阵坊结构

| 部门 | 职责 | 可写冲突 |
|------|------|----------|
| 主坊 | 接宗门正式订单、护山小阵、灵田阵维护 | 吃稳定利润，优先保主脉 |
| 修盘房 | 修旧阵盘、补阵旗、拆废阵材 | 陆维舟常在此线，接触残阵和坏账 |
| 器纹室 | 刻基础阵纹、处理阵基器件 | 容易和炼器、材料成本冲突 |
| 勘脉小队 | 看灵田、洞府、山道的地势灵机 | 外勤风险高，贡献不稳定 |
| 账房 | 计算阵材、工钱、赔付和宗门扣款 | 坏阵责任常被压给旁支和学徒 |

陆氏阵坊内部不是大家族内斗戏，而是职业分工造成的现实压力。

## 3. 主脉与旁支

陆维舟可定为陆氏阵坊旁支。

| 分支 | 资源 | 压力 | 人物用途 |
|------|------|------|----------|
| 主脉 | 接宗门正式订单、掌握完整阵图、拥有更好阵材 | 必须保住阵坊名声和利润 | 可给陆维舟压力，但不必写成反派 |
| 旁支 | 修残阵、跑外勤、做边角料改造 | 责任重，利润薄，出错先背锅 | 陆维舟技术谨慎的来源 |
| 学徒 | 画基础阵线、磨阵钉、整理阵材 | 工钱低，学不到完整阵图 | 展示阵法职业不是凭天赋一夜成型 |

陆维舟早期怕大因果，是因为他知道阵法失败不是“没打赢”，而是别人信了你布的阵，然后死在里面。

## 4. 阵材与成本

低阶阵法不能写成随手画符。

常见成本：

1. 阵盘木料或石料。
2. 阵旗布料和灵线。
3. 阵钉、阵眼石、低阶灵石。
4. 勘测地脉的时间。
5. 试阵失败损耗。
6. 后期维护费用。

常见坏账：

| 坏账类型 | 说明 | 剧情用途 |
|----------|------|----------|
| 省料坏阵 | 为压成本少用阵材，短期能用，长期出事 | 陆维舟拒绝做坏阵的底线 |
| 旧阵误修 | 残阵来历不明，补错一个节点会反噬 | 赵长生归墟炉残缺识别线 |
| 灵田阵失衡 | 只追求聚灵，伤土性和灵植根系 | 与叶疏彤生机线冲突 |
| 洞府阵欠维护 | 阵法初建有效，几年后灵石和阵旗跟不上 | 洞府经营成本 |
| 宗门压价 | 宗门以长期订单压低阵坊利润 | 主脉把成本转给旁支 |

## 5. 与青云门关系

| 青云门环节 | 陆氏关系 | 冲突点 |
|------------|----------|--------|
| 阵堂 | 陆氏阵坊半附属、半外包 | 阵堂拿名声，阵坊承担修补细活 |
| 灵田院 | 陆氏维护部分灵田阵和灵泉阵 | 聚灵效率和土性生机可能冲突 |
| 外门执事堂 | 外勤勘脉、护送阵盘、任务结算 | 任务失败扣贡献，旁支先被问责 |
| 洞府契约 | 第四卷洞府阵、护山阵、退路阵 | 阵法必须和产权、维护绑定 |
| 丹房和器房 | 采购阵材、器纹件、废料再利用 | 残器和废阵盘给赵长生捡漏入口 |

陆氏阵坊是青云门制度压力的一部分，但也被青云门压榨。

## 6. 与陆维舟关系

| 阶段 | 关系 | 陆维舟的变化 |
|------|------|--------------|
| 初期 | 旁支阵修，修盘房出身，接触大量残阵和坏账 | 技术谨慎，只想修好自己能负责的阵 |
| 第二卷 | 通过废阵盘和赵长生接触 | 发现残缺阵法不一定只能丢弃 |
| 第三卷 | 筑基资源争夺中负责退路和低阶阵法 | 开始承担团队生死责任 |
| 第四卷 | 洞府护山阵和灵圃阵成型 | 从修小阵转向设计资源循环节点 |
| 第五卷以后 | 战争、残界、登仙台让他理解阵法可以承界 | 超出陆氏阵坊层级，但不能忘记成本意识 |

陆维舟的技术爽点来自“稳”，不是随手开挂。

## 7. 可提供的剧情资源

| 资源 | 用法 | 限制 |
|------|------|------|
| 废阵盘 | 第二卷捡漏、残阵修复、赵长生识别残缺 | 不能每个废阵盘都有大机缘 |
| 低阶护田阵 | 叶疏彤灵田线和阵法生机冲突 | 聚灵不等于养田 |
| 洞府小阵图 | 第四卷洞府经营核心 | 要有维护成本 |
| 阵材边角料 | 让赵长生、陆维舟合作试错 | 数量有限，品质不稳 |
| 坏阵赔付记录 | 展示阵法失败后果 | 可推动陆维舟拒绝坏阵 |

## 8. 内部冲突

| 冲突 | 具体表现 | 用法 |
|------|----------|------|
| 主脉利润 vs 旁支责任 | 主脉接订单，旁支修坏账 | 陆维舟被边缘化但获得残阵经验 |
| 阵法效率 vs 生机平衡 | 聚灵阵可能伤灵田土性 | 与叶疏彤形成专业互证 |
| 稳定安全 vs 临场冒险 | 沈临风敢试系统阵盘，陆维舟要求验证 | 双主角和陆维舟互动 |
| 宗门考核 vs 职业底线 | 省料阵更容易交差 | 陆维舟底线落地 |
| 小阵修补 vs 大界阵 | 陆维舟从作坊技术走向万界节点 | 长线成长方向 |

## 9. 卷际使用

| 卷 | 用法 | 状态 |
|----|------|------|
| 第二卷 | 陆维舟登场，废阵盘、修盘房、阵材成本明确 | 阵坊压力初显 |
| 第三卷 | 低阶阵法用于筑基资源争夺和撤离路线 | 陆维舟开始承担团队安全 |
| 第四卷 | 洞府护山阵、灵圃阵、退路阵成型 | 阵坊职业经验升级为洞府经营 |
| 第五卷 | 战争阵线和宗门阵法局限暴露 | 陆维舟认识阵法不能只保几个人 |
| 第八卷以后 | 残界、登仙台、万界节点回照早期阵坊经验 | 小阵逻辑升级为界阵逻辑 |

## 10. 禁写项

1. 不能把陆氏阵坊写成大家族爽文后台。
2. 不能让陆维舟拿到无限阵材。
3. 不能让阵法没有维护成本。
4. 不能让陆维舟每次靠灵感一夜布阵。
5. 不能让主脉全是坏人，旁支全是好人。
6. 不能让阵法压过叶疏彤的生机专业、谢未明的命数专业和赵长生的归墟重炼主线。

## 11. 使用提示

写陆氏阵坊时，冲突最好落到技术细节：

1. 阵材缺一味。
2. 阵眼偏半寸。
3. 灵田阵聚灵太急伤根。
4. 旧阵盘有一处不合常理的残缺。
5. 阵坊账房把坏阵赔付压给旁支。

陆氏阵坊的存在是为了证明：

> 阵法不是画在纸上的聪明，而是别人敢把命交给你的结构。
