# 严慎

> 第一批跨卷二级人物。严慎是补墟司账师，不是天然正义的吹哨人。他的作用是让纪晚凝从明账和内账的细小误差开始，看见补墟体系里的结构性坏账。

---

## 1. 基础信息

| 项目 | 内容 |
|------|------|
| 姓名 | 严慎 |
| 名字状态 | 暂定，可保留 |
| 首次登场 | 第十一卷：司命台与补墟司 |
| 主要活跃 | 第十一卷到第十二卷，可在第十三卷以账证回收 |
| 人物层级 | C 级证据人物，纪晚凝线关键入口 |
| 身份 | 补墟司账师，负责明账和部分内账核对 |
| 所属势力 | 补墟司执行线，偏平账派 |
| 修为位置 | 仙界中低层仙官，建议真仙到玄仙之间 |
| 专长 | 账目校验、损耗分类、额度折算、签押追溯 |
| 立场 | 相信平账能维持墟眼封禁，前期不认为自己在作恶 |

严慎这个名字要落实到人物行为上：他谨慎、怕乱、怕失职，也怕因为一笔账揭开整个补墟司无法承受的真相。

## 2. 人物核心

| 项目 | 内容 |
|------|------|
| 核心欲望 | 把账平住，让补墟司能继续运转，让墟眼封禁不因账面混乱崩盘 |
| 核心恐惧 | 账目公开后引发仙界秩序震荡，补墟司底层仙官先被清算 |
| 底线 | 不主动伪造黑账，不亲手把无辜者写成可用耗材 |
| 缺点 | 把长期误差称为“损耗”，用技术语言回避人的死亡和世界衰败 |
| 爽点功能 | 让纪晚凝查账从一笔小错开始，逐步击穿制度谎言 |
| 世界观功能 | 连接明账、内账、飞升者分流、补墟额度和损耗转嫁 |
| 冲突功能 | 逼纪晚凝面对“制度需要平账”与“账背后有人命”的冲突 |

严慎前期不是英雄。他看到小额差额时，第一反应不是揭发，而是找一个合规理由把账平掉。

这正是他的可写处。

## 3. 所属制度：补墟司账师链

严慎所在的位置不接触最高黑账，但能看见明账和内账之间越来越不自然的差额。

表层职责：

1. 核对仙税入账。
2. 计算补墟额度。
3. 汇总飞升者分流损耗。
4. 整理仙矿、镇界军、登仙监回执。
5. 将异常损耗转交上级封存。

实际处境：

1. 他知道账不完全干净。
2. 他以为问题来自执行层贪墨。
3. 他不愿相信差额是制度结构造成。
4. 他害怕纪晚凝继续追查会触动黑账守门人。
5. 他最终必须在平账和作证之间选择。

## 4. 能力与权限

| 权限/能力 | 用法 | 限制 |
|-----------|------|------|
| 明账核对 | 查仙税、役期、资源入账和飞升者分流 | 只能看到表层，很多项目被上级合并 |
| 内账抽样 | 对补墟额度、损耗和本源分类做小范围复核 | 权限不稳定，超范围会触发封账 |
| 签押追溯 | 追查某笔账经过哪些仙官签押 | 高层法旨可覆盖签押链 |
| 损耗折算 | 将缺口折成灵、宝、魂、血、界等类别 | 折算本身可能掩盖具体受害者 |
| 账印留痕 | 在账册上留下细小校验痕迹 | 可被藏烬君等黑账人物抹除 |

严慎不是战斗型人物。保护他、追杀他、争夺他留下的账证，才是剧情价值。

## 5. 与核心人物关系

| 对象 | 初始关系 | 核心冲突 | 变化方向 |
|------|----------|----------|----------|
| 纪晚凝 | 同司账务接点 | 纪晚凝要查差额，严慎要把差额归为损耗 | 从阻拦，到提示，到可能交出关键账证 |
| 赵长生 | 仙籍异常对象 | 赵长生的命数断页和归墟炉痕迹让账目无法归类 | 严慎意识到并非所有异常都该被封账 |
| 沈临风 | 运账异常对象 | 系统奖励和天命值折损在账面上有痕迹 | 严慎可提供司命台和补墟司账目接口 |
| 傅照灯 | 制度修补派同僚 | 傅照灯愿意修制度，严慎只想先平账 | 二人可互相保护，也可因是否公开账证分歧 |
| 藏烬君 | 黑账守门人 | 严慎的账证可能触及黑账入口 | 藏烬君可派人灭口或封存严慎账印 |
| 裴玄阙 | 遥远高层压力 | 严慎不直接接触裴玄阙，但保全派法旨决定哪些账不能查 | 严慎逐渐发现自己平的账服务的是更高层道路 |

## 6. 与纪晚凝的冲突

严慎对纪晚凝的劝阻不是坏人威胁，而是同僚式现实判断：

> 账差可以补，封禁不能乱。你把这一页翻到底，先死的未必是高层。

他的三层回避：

1. 把差额归为自然损耗。
2. 把重复异常归为执行层贪墨。
3. 把结构性错误归为非常时期的必要平账。

纪晚凝必须击穿的是：

1. 平账不等于真相。
2. 损耗不是没有姓名的数字。
3. 墟眼危机真实，也不能让所有坏账合法化。
4. 补墟司若只会封账，就不再是补墟，而是在替墟眼喂账。

## 7. 卷际弧线

| 卷 | 作用 | 状态变化 |
|----|------|----------|
| 第十一卷 | 作为纪晚凝账务同僚登场，发现明账和内账差额 | 试图劝纪晚凝停止追查，把问题归为损耗 |
| 第十二卷 | 在旧账对照中发现差额不是近期贪墨，而是新仙界制度长期转嫁 | 开始动摇，但仍害怕公开账证 |
| 第十三卷 | 账证被藏烬君或补墟司高层盯上 | 可被灭口、转入保护、留下账印证据或成为清账证人 |

## 8. 可选退场方式

| 方式 | 用法 |
|------|------|
| 死亡留账 | 被灭口前留下明账、内账差额和签押链，推动纪晚凝彻底转向 |
| 证人留存 | 作为低阶账师出面作证，证明底层仙官也知道部分异常 |
| 后台保护 | 被傅照灯或纪晚凝藏起，后续用于第十三卷清账 |
| 失败退场 | 因恐惧选择继续平账，最后被黑账反噬或失去账师资格 |

严慎可以死，但死亡不能只是煽情，必须留下可用账证。

## 9. 可提供的关键证据

| 证据 | 说明 | 后续接入 |
|------|------|----------|
| 明账与内账差额 | 同一批飞升者、仙税或补墟额度在两套账里数字不同 | 纪晚凝查账线 |
| 补墟额度递增表 | 证明投入越来越多，封禁效果越来越差 | 裴玄阙道路失败 |
| 损耗转嫁签押 | 证明下界、百族、飞升者损耗被写成自然损耗 | 血账、界账、魂账 |
| 飞升者分流异常 | 证明部分飞升者没有进入正常仙籍流程 | 仙矿、镇界军、司命台回收 |
| 账印残痕 | 证明黑账守门人曾篡改或封存账册 | 藏烬君线 |

## 10. 禁写项

1. 不能一登场就是正义吹哨人。
2. 不能让他知道完整黑账和墟眼真相。
3. 不能让他替纪晚凝完成清账选择。
4. 不能把他写成只会胆小怕死的工具人。
5. 不能让他的死亡没有账证价值。
6. 不能让平账派全是坏人，严慎要体现制度中层的真实恐惧。

## 11. 使用提示

写严慎时，要让他的每一次出场都带着账务细节：

1. 一笔差额。
2. 一个被合并的损耗项。
3. 一个飞升者分流名单。
4. 一个不该出现的签押。
5. 一句“先平账”的职业本能。

他的存在是为了证明：

> 新仙界的罪不只在黑账深处，也在无数人把人的命和世界的伤口写成“损耗”的那一刻。
