# 终局代表人物映射表

> 人物侧收束补档。本文用于锁定第十四卷终局中“谁代表什么旧账”，防止终局临时拉新人物发言。本文不是终局大会名单，也不是新增核心角色入口。

---

## 1. 设计目的

终局代表人物映射表要解决五个问题：

1. 下界、百族、飞升者、仙矿、旧账、制度残余和亡魂旧账必须由前文人物自然承接。
2. 第十四卷不能新增一批高魅力代表人物抢走长线人物功能。
3. 终局清算不能只有赵长生、沈临风和三女主发言，也不能变成无源头的群体口号。
4. 每个代表人物必须带着前文旧账、证据或责任进入终局。
5. 制度重炼要通过具体人和具体账体现，而不是一句“天下重定”解决。

终局代表的核心原则：

> 谁在前文承受过制度，谁才有资格在终局被看见；谁只是终局才出现，谁就只能做背景群众。

## 2. 使用边界

| 项目 | 规则 |
|------|------|
| 代表不是席位 | 不写成各族各界开会分座次 |
| 代表不是官职 | 终局不分封新仙帝、新天官、新皇族 |
| 代表不是演讲工具 | 每个人必须带证据、伤口、选择或责任 |
| 代表不是新主角 | 不新增高权重人物压过核心八人和后期理念人物 |
| 代表不等于完美受害者 | 灰色人物也可作证，但必须承担自身责任 |

终局重点是旧账公开、新规则重订和循环重炼，不是所有人物轮流发表立场。

## 4. 主代表与辅助代表

终局不能让所有代表都拥有同等戏份。建议按三层处理：

| 层级 | 人物 | 用法 |
|------|------|------|
| 核心执行层 | 赵长生、沈临风、叶疏彤、白倩霜、纪晚凝、陆维舟、顾长庚、谢未明、洛闻霜、秦照古 | 承担终局行动、证据互证和路线选择 |
| 关键见证层 | 周栖石、钟离砺、梅见微、温隙、傅照灯、严慎、厉沉霄 | 以旧账、证据、受审或选择进入终局 |
| 背景回声层 | 宁晚秋、卫从戎、白疏影、绛羽、散血民、许守仓、林停云 | 按卷纲需要以人物、遗账、证词或档案出现 |

宁晚秋若第一卷已死，不强行复活。她的功能可以由灵米账、棚区旧账、遗留证词或赵长生记忆承担。

## 5. 下界代表线

下界代表不是“凡人代表”这么空泛，而是三类旧债：

1. 坊市底层旧债：杂役契、棚区租、发霉灵米、废丹工钱。
2. 修仙国旧债：战争、古战场、断仙门、下界修士被困死。
3. 生机旧债：灵田变薄、灵井水淡、灵脉被上层长期抽走。

建议承接：

| 承接方式 | 用法 |
|----------|------|
| 宁晚秋活着 | 作为底层女性样本进入后期回声，不升级核心人物 |
| 宁晚秋已死 | 以灵米账、棚区旧账、旧衣物、证词或赵长生记忆回收 |
| 卫从戎或下界军修 | 承接修仙国大战和下界战损 |
| 叶疏彤 | 承接生机和灵田旧债，不替代全部下界底层发声 |

终局不要让下界代表临时喊“我们也受苦”。前文必须留有账、物、人或记忆。

## 6. 百族代表线

百族代表不能只靠白倩霜一人说完。

白倩霜承接的是：

1. 返祖真血。
2. 霜羽族祖契。
3. 源血链路钥匙。
4. 百族契约重订。
5. 真血者不是族群还款材料。

辅助声音需要补足：

| 类型 | 建议人物 | 功能 |
|------|----------|------|
| 小族青年 | 绛羽 | 说明小族不是大族和霜羽族附庸 |
| 族内责任 | 白疏影 | 说明霜羽族不是只有交人派和护人派 |
| 散血民 | 散血市证人或混血商旅 | 说明血统等级如何伤害无族之人 |
| 魂源证人 | 梦婆罗或祁归年 | 提供魂源旧账前置，但不替代洛闻霜 |

百族线终局要完成的是重订契约，不是让白倩霜成为百族女王。

## 7. 飞升者代表线

飞升者代表由 `10_飞升者代表小组.md` 承接，本文只锁终局功能。

| 人物 | 代表面 | 是否建议进入终局 |
|------|--------|------------------|
| 周栖石 | 仙矿役籍、底层飞升者、矿账 | 是，首选飞升者旧账见证 |
| 梅见微 | 失籍、黑籍、伪籍、灰网 | 可进入终局，也可作为第十到十一卷证据通道 |
| 钟离砺 | 军籍、镇界军征调、强者被收编 | 是，承接军籍飞升者和镇界军清算 |

飞升者线要证明：

1. 飞升不是自由成仙。
2. 弱者被役籍和矿账吞掉。
3. 失籍者被逼入灰网。
4. 强者也会被军籍和军令绑定。
5. 新循环必须改飞升通道、仙籍和分流规则。

## 8. 旧账见证线

旧账见证必须有双重结构：

1. 旧天庭旧账：旧天庭曾承担公共职责，也曾失败。
2. 新仙界黑账：新仙界以真实危机为名长期收割万界。

| 人物 | 功能 | 边界 |
|------|------|------|
| 温隙 | 旧籍、旧账、时痕缝隙证人 | 不代表旧天庭全对 |
| 纪晚凝 | 补墟账本清账者 | 不因爱情倒戈，必须由账本和责任推动 |
| 严慎 | 明账与内账差额入口 | 不替纪晚凝承担最终清账 |
| 傅照灯 | 修补派责任与作证 | 不能被洗成无责好人 |
| 藏烬君 | 黑账守门证据来源 | 不把罪责全压给一个守门人 |
| 秦照古 | 旧制价值和失败的承认者 | 不能带领复辟成功 |

## 9. 制度残余线

终局清算不能以“裴玄阙败了”结束。

制度残余至少包括：

1. 补墟司仍有账要清。
2. 新司命台仍有命数黑账要公开。
3. 镇界军仍要区分守界职责和抽源镇压。
4. 登仙监、仙籍署、仙矿府和黑籍坊市仍有善后。
5. 低层执行者不能全部被一句“奉命”免罪，也不能全部杀光。

| 人物 | 使用方式 |
|------|----------|
| 傅照灯 | 承认修补派拖延代价，参与清账善后 |
| 严慎 | 以账师证据证明明账、内账和黑账差额 |
| 许守仓 | 可作为低层仙籍官样本，说明制度执行压力 |
| 钟无咎 | 作为命数黑账守门权限，不扩成完整主反派 |
| 厉沉霄 | 作为镇界军执行层受审、失印或被迫停令 |

## 10. 终局使用方式

建议终局采用“证据分线汇合”，而不是集中开会。

| 分线 | 主要人物 | 场景建议 |
|------|----------|----------|
| 补墟账公开 | 纪晚凝、严慎、傅照灯、藏烬君证据 | 补墟黑账深库或中天玄阙账阵 |
| 命数黑账击穿 | 沈临风、谢未明、季玄屏、钟无咎权限 | 新司命台命数黑账阁或司命盘残核 |
| 血契重订 | 白倩霜、白疏影、绛羽、散血民 | 百族契约投影或源血链路回响 |
| 飞升通道重定 | 周栖石、梅见微、钟离砺、陆维舟 | 登仙台新节点、仙籍重录处 |
| 镇界军清算 | 顾长庚、钟离砺、厉沉霄 | 镇界军令台或军印解除场 |
| 亡魂归处 | 洛闻霜、魂灯灰签、飞升失败者残魂 | 轮回灯残职与魂账交界 |
| 循环重炼 | 赵长生、九器、核心同伴 | 墟眼禁域，不写成单人登基 |

## 11. 禁写项

1. 不能在第十四卷临时新增“万界代表”“百族盟主”“飞升者领袖”抢戏。
2. 不能让终局变成所有人排队发表理念。
3. 不能让白倩霜一人替全部百族说完。
4. 不能让周栖石、梅见微、钟离砺全变成主角团长期核心。
5. 不能把钟无咎扩写成裴玄阙同级终局主反派。
6. 不能让傅照灯、严慎等制度人物用一次作证洗掉过去责任。
7. 不能让旧天庭人物证明“复辟正确”。
8. 不能让赵长生私自掌握所有账本和命数。
9. 不能写成新仙界高层死光后自然天下太平。
10. 不能让归墟炉包办血账、魂账、界账和命数账的全部解决。

## 12. 使用提示

后续拆第十四卷时，每个代表人物出场前先检查三个问题：

1. 他在前文哪一卷留下旧账或证据。
2. 他在终局要改变哪条制度规则。
3. 他是否会抢走核心人物不可替代的职责。

若答不上来，就不要把这个人物升级成终局代表。

终局的真正收束不是“大家都来了”，而是：

> 前文每一笔被写成数字的伤害，都在重炼前重新变回具体的人、族群、亡魂和世界。
