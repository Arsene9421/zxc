# 穆天衡

> 第一批跨卷二级人物。穆天衡是旧天庭复辟派将领，不是秦照古的跟班。他的作用是让复辟派拥有武力、军序和可敬但危险的秩序诱惑，并与顾长庚的正统剑道形成冲突。

---

## 1. 基础信息

| 项目 | 内容 |
|------|------|
| 姓名 | 穆天衡 |
| 名字状态 | 暂定，可保留 |
| 首次登场 | 第十二卷：旧天庭废墟 |
| 主要活跃 | 第十二卷，可在第十三卷回声或残部延续 |
| 人物层级 | C 级理念武力人物，复辟派军序代表 |
| 身份 | 旧天庭复辟派将领，旧军序残部统领 |
| 所属势力 | 旧天庭复辟派，偏军序强硬派 |
| 修为位置 | 旧籍高阶战修，建议金仙层级或残缺太乙战力，具体境界卷纲再锁 |
| 专长 | 军阵、旧镇界残印、战场调度、军令压制 |
| 立场 | 认为旧天庭必须重立，众生需要由高处重新统御 |

穆天衡要写出“衡”的一面：他不是只想打仗，而是相信天地失衡后，必须用军序重新压回秩序。

问题在于，他把“秩序”理解为自上而下的命令。

## 2. 人物核心

| 项目 | 内容 |
|------|------|
| 核心欲望 | 夺回旧道职和旧军序，重建能统一调度万界的天庭武力 |
| 核心恐惧 | 没有中央军序，万界在第二次归墟大潮中各自求活、互相撕裂 |
| 底线 | 不愿把众生直接当燃料，也厌恶新仙界权贵借补墟续命 |
| 缺点 | 把公共职责和上位命令混同，相信牺牲可以由高处替众生决定 |
| 爽点功能 | 给顾长庚和赵长生一个“可敬但不能跟随”的旧制诱惑 |
| 世界观功能 | 证明旧天庭复辟派不是秦照古一人，而有旧军、旧印、旧战绩 |
| 冲突功能 | 让主角组拒绝新仙界的同时，也不能简单恢复旧天庭 |

穆天衡不能是漫画式战争狂。

他可以救人，可以守旧阵，可以对新仙界怒不可遏。但他一旦掌权，仍会把“服从旧军令”放在“众生自决”之前。

## 3. 所属制度：旧天庭军序残部

旧天庭军序原本有公共职责：

1. 镇守界裂。
2. 护送补天队。
3. 清理归墟污染外溢。
4. 调度各界兵力支援大潮。
5. 在天劫和飞升通道崩坏时保护撤离。

穆天衡继承的是残缺军序。

现状问题：

1. 旧军令没有完整旧天庭授权。
2. 旧军残部长期被新仙界追杀，心态更强硬。
3. 旧镇界印残缺，容易将守界变成控界。
4. 复辟派把旧职责和旧权力绑定，认为九器归位就该恢复旧职。

## 4. 能力与权限

| 权限/能力 | 用法 | 限制 |
|-----------|------|------|
| 旧军令 | 调动旧军残部和遗址军阵 | 授权残缺，无法代表完整旧天庭 |
| 镇界残印 | 临时稳住界裂、压制归墟污染 | 可救人，也可变成锁界工具 |
| 天衡军阵 | 以军序合力压制强敌 | 依赖服从和阵位，难适应自主联盟 |
| 旧战录 | 保存旧天庭抗大潮战事记录 | 记录可证明旧牺牲，也会暴露旧军强制征调 |
| 斩令威压 | 对旧道职、旧籍、旧军有压制力 | 对赵长生命数断页和新循环理念不稳定 |

穆天衡的战力可以强，但他最关键的不是“打得过谁”，而是用军序压人时让读者理解诱惑和危险。

## 5. 与核心人物关系

| 对象 | 初始关系 | 核心冲突 | 变化方向 |
|------|----------|----------|----------|
| 秦照古 | 同属复辟派但路线不同 | 秦照古更重旧义和旧职，穆天衡更重军令和执行 | 体现复辟派内部并非铁板一块 |
| 赵长生 | 新循环候选者 | 赵长生尊重旧牺牲但拒绝重建垄断天庭 | 穆天衡试图证明没有中央军序就无法抗大潮 |
| 顾长庚 | 正统剑修对照 | 穆天衡诱使顾长庚接受旧军正统和斩道权柄 | 顾长庚必须分清正统职责和正统压迫 |
| 陆维舟 | 界阵技术线 | 穆天衡相信军阵和旧镇界印，陆维舟追求万界节点协作 | 二人可围绕界阵指挥权冲突 |
| 温隙 | 旧籍苟活者 | 穆天衡看不起温隙的妥协 | 温隙的证词可反证旧军也曾强制征调 |
| 裴玄阙 | 新仙界保全派敌人 | 双方都承认危机真实，但解决方式都是自上而下 | 可形成“旧军序”和“新保全”的镜像 |

## 6. 与顾长庚的冲突

穆天衡对顾长庚的诱惑：

> 剑若只为自证清白，不如入旧军序，替万界斩出一条可服从的正道。

他的三层诱惑：

1. 给顾长庚旧天庭正统战录。
2. 让顾长庚看见旧军确实守过万界。
3. 提供斩道剑残痕或旧军斩令，让顾长庚拥有更大“正统权力”。

顾长庚必须拒绝的是：

1. 正统不等于服从旧权力。
2. 斩道不能由上位者垄断。
3. 救世不能变成替众生决定一切。
4. 旧天庭有牺牲，也有失败和压迫。

## 7. 卷际弧线

| 卷 | 作用 | 状态变化 |
|----|------|----------|
| 第十二卷 | 作为复辟派军序代表登场，展示旧天庭仍有残军和武力 | 可敬、强硬、看不上主角组的松散协作 |
| 第十二卷后段 | 与顾长庚、秦照古围绕旧职复原发生冲突 | 暴露复辟派内部的军序强制倾向 |
| 第十三卷 | 可作为旧军残部回声，短暂协助抗保全派，但不接受新循环主导 | 最终败退、留守旧墟，或承认旧军序不能复辟 |

## 8. 可选退场方式

| 方式 | 用法 |
|------|------|
| 战败退场 | 被顾长庚或主角组击败，证明旧军序不能压过新循环 |
| 留守旧墟 | 不加入主角组，率残部守旧天庭废墟，承认不再有资格号令万界 |
| 战死回收 | 为挡新仙界保全派战死，但死亡前仍不完全认同主角路线 |
| 理念败退 | 看见旧战录中的强制征调证据后沉默，不洗白也不彻底转阵营 |

不建议让穆天衡被写成秦照古一句话就能命令的下属。他要有自己的军序逻辑。

## 9. 可提供的关键证据

| 证据 | 说明 | 后续接入 |
|------|------|----------|
| 旧军战录 | 证明旧天庭曾为守万界付出真实牺牲 | 秦照古旧牺牲线 |
| 旧镇界残印 | 证明镇界职责原本不是抽源压迫 | 厉沉霄、新镇界军对照 |
| 旧军征调令 | 证明旧天庭也曾强制调度各界力量 | 拒绝复辟旧权力 |
| 斩令残痕 | 诱发顾长庚正统和斩道线选择 | 顾长庚第十二卷弧线 |
| 复辟派军序名单 | 证明复辟派有组织，而非秦照古个人执念 | 旧天庭派系线 |

## 10. 禁写项

1. 不能写成无脑武夫或战争狂。
2. 不能让他只是秦照古手下。
3. 不能让旧天庭复辟派完全正确。
4. 不能让顾长庚轻易被旧正统收编。
5. 不能让穆天衡掌握完整终局答案。
6. 不能让旧军序问题被一句“他们也是为了天下”抹平。

## 11. 使用提示

写穆天衡时，每次出场都要带出旧军序：

1. 一道旧军令。
2. 一枚镇界残印。
3. 一段可敬的旧战录。
4. 一条不容商量的征调命令。
5. 一次对顾长庚的正统诱惑。

他的存在是为了证明：

> 旧天庭值得被记住，但不值得被原样复辟。
