# 傅照灯

> 第一批跨卷二级人物。傅照灯是补墟司制度修补派，不是纯好人。他的作用是给纪晚凝提供一个制度内部的复杂同僚：他承认补墟必要，也厌恶权贵滥用，却长期参与过有伤害的执行。

---

## 1. 基础信息

| 项目 | 内容 |
|------|------|
| 姓名 | 傅照灯 |
| 名字状态 | 暂定，可保留 |
| 首次登场 | 第十一卷：司命台与补墟司 |
| 主要活跃 | 第十一卷到第十三卷 |
| 人物层级 | C 级制度同僚，可作为纪晚凝线的内部复杂人物 |
| 身份 | 补墟司制度修补派仙官，纪晚凝同僚或前辈 |
| 所属势力 | 补墟司执行线，偏制度修补派 |
| 修为位置 | 仙界中层仙官，建议玄仙层级，具体境界卷纲再锁 |
| 专长 | 内部流程、账册保全、封存例外、法旨漏洞、人员保护 |
| 立场 | 相信补墟仍有必要，但反对黑账、过度抽源和权贵豁免 |

傅照灯名字里的“照灯”不是照亮全世界，而是在破制度里保住一盏小灯。

问题在于，小灯照不穿黑账时，他曾经选择先维持制度。

## 2. 人物核心

| 项目 | 内容 |
|------|------|
| 核心欲望 | 在不引发仙界崩盘的前提下修补补墟司，让补墟回到必要而有限的职责 |
| 核心恐惧 | 账本全面公开导致补墟司瘫痪，墟眼封禁立刻失控 |
| 底线 | 不愿主动替权贵遮掩纯粹私利，但会为了保封禁签下有伤害的命令 |
| 缺点 | 过度相信制度内部修补，迟迟不愿承认体系已经从根上坏账 |
| 爽点功能 | 让纪晚凝转向不是孤立觉醒，而是在同僚分歧、保护和拉扯中完成 |
| 世界观功能 | 体现补墟司内部不是全员反派，有清账派、平账派、封账派和修补派 |
| 冲突功能 | 逼纪晚凝回答：修补是否还够，何时必须公开清账 |

傅照灯可以保护人，也可以签过错误命令。

他的复杂性就在于：他不是坏人，但如果没有赵长生的新循环方案，他最多只能让坏制度慢一点伤人。

## 3. 所属制度：补墟司制度修补派

制度修补派的主张：

1. 墟眼真实存在，补墟不能立刻停止。
2. 仙界需要账本和道职，不可能一夜取消。
3. 黑账、权贵豁免、过度抽源必须被限制。
4. 飞升者、百族、下界的损耗应被纳入赔付和减征。
5. 改革必须尽量避免仙界核心立刻崩盘。

他们的问题：

1. 权限有限。
2. 容易被封账派利用为缓冲。
3. 不敢公开完整黑账。
4. 没有替代循环方案时只能小修小补。
5. 曾经为了维持补墟司签下有害执行令。

傅照灯是这条线的具体人物。

## 4. 能力与权限

| 权限/能力 | 用法 | 限制 |
|-----------|------|------|
| 流程缓冲 | 延迟某些征收、审查、封账或追捕流程 | 只能拖延，不能彻底取消高层法旨 |
| 账册保全 | 在账本被焚毁前留下副录、暗号或校验痕迹 | 被藏烬君发现后会失效 |
| 法旨漏洞 | 利用补墟司内部条款保护严慎、纪晚凝或证据 | 高层可用保全法旨压过 |
| 内部人脉 | 调动少数修补派、低层账师、巡账小吏 | 人脉害怕牵连，不稳定 |
| 道职灯印 | 暂时照出被合并或遮蔽的账目痕迹 | 不能照穿完整黑账，且消耗道职信用 |

傅照灯的能力要像制度内部的人：能拖、能藏、能保一段，不能替主角组解决全部问题。

## 5. 与核心人物关系

| 对象 | 初始关系 | 核心冲突 | 变化方向 |
|------|----------|----------|----------|
| 纪晚凝 | 同僚或前辈 | 傅照灯要修补，纪晚凝后期要公开清账 | 从劝阻，到协助，到第十三卷必须选边 |
| 严慎 | 可保护的账师 | 严慎想平账，傅照灯知道平账不够但也怕公开 | 可保护严慎或帮他留下账证 |
| 藏烬君 | 封账派对手 | 藏烬君要封黑账，傅照灯想限制黑账扩张 | 第十三卷形成补墟司内部冲突 |
| 赵长生 | 制度外替代方案 | 傅照灯怀疑赵长生只是破坏者，后来发现他有重炼循环可能 | 是否支持赵长生取决于新方案是否能承接墟眼压力 |
| 叶疏彤 | 下界生机旧债代表 | 傅照灯过去可能签过下界减损不充分的账 | 叶疏彤让他面对“修补过慢”的具体后果 |
| 白倩霜 | 血税受害链代表 | 傅照灯反对过度血税，但未能阻止 | 白倩霜线逼他承认内部修补没有保护百族 |
| 裴玄阙 | 保全派高层压力 | 傅照灯不认同权贵滥用，却承认裴玄阙危机论部分真实 | 最终必须拒绝“继续封账保核心”的路线 |

## 6. 与纪晚凝的冲突

傅照灯给纪晚凝的不是反派压迫，而是难题：

> 你若今天公开全部账，明天墟眼裂口扩大，死的人你也要负责。

他的三层立场：

1. 先查清，不要急着公开。
2. 先修补，不要动摇封禁。
3. 先保核心，不要让底层仙官替高层受死。

纪晚凝必须回应的是：

1. 拖延本身会继续死人。
2. 黑账不公开就无法追责。
3. 补墟司不能永远用真实危机抵消制度罪责。
4. 清账不是烧账，而是公开、赔付、改规则和承接后果。

## 7. 卷际弧线

| 卷 | 作用 | 状态变化 |
|----|------|----------|
| 第十一卷 | 作为纪晚凝同僚登场，提醒她账目风险，保护或试探严慎 | 坚持制度内部修补，不支持公开对抗 |
| 第十二卷 | 旧账对照后发现问题比想象更深 | 开始为纪晚凝留副录，但仍不敢彻底站队 |
| 第十三卷 | 藏烬君封账、黑账曝光、裴玄阙路线压下 | 必须选择封账保稳定，还是公开清账承受震荡 |

## 8. 可选退场方式

| 方式 | 用法 |
|------|------|
| 公开作证 | 承认自己签过错误命令，同时提供内部流程证据 |
| 留在制度内善后 | 终局后不进主角团，负责补墟司清算和赔付执行 |
| 牺牲保账 | 为保严慎账证或纪晚凝清账通道牺牲，但不可洗掉过往责任 |
| 选择封账 | 若走反面弧线，他在第十三卷选择封账，被纪晚凝击败或越过 |

傅照灯可以成为同盟，但不是无责好人。

## 9. 可提供的关键证据

| 证据 | 说明 | 后续接入 |
|------|------|----------|
| 内部备忘录 | 记录修补派曾多次提示黑账风险 | 证明补墟司内部知道问题 |
| 副录账册 | 被藏烬君焚毁前留下的校验副本 | 纪晚凝清账线 |
| 权贵豁免疑点 | 证明部分玄阙仙族被减免补墟负担 | 权贵耗材派清算 |
| 保护签押 | 证明严慎、低层账师或证人被人暂时保下 | 补墟司内部复杂性 |
| 有害命令签押 | 傅照灯自己签过的命令，证明修补派也有责任 | 终局制度责任 |

## 10. 禁写项

1. 不能写成纯好人内应。
2. 不能让他一出场就完全支持纪晚凝。
3. 不能让修补派替主角组解决黑账问题。
4. 不能把他过去签过的有害命令轻轻放过。
5. 不能让他和严慎功能重复：严慎是账师入口，傅照灯是制度修补派。
6. 不能让他替纪晚凝承担清账主责。

## 11. 使用提示

写傅照灯时，每次出场都要带出制度内部的两难：

1. 他救下一个人，却救不了一批人。
2. 他保住一页账，却暂时不能公开。
3. 他利用流程拖延，却让损害继续发生。
4. 他劝纪晚凝谨慎，却被纪晚凝逼问谨慎的代价。
5. 他最终必须选择站在封账还是清账一边。

他的存在是为了证明：

> 坏制度里也有人想修灯，但灯若只照案头，不照账外死去的人，修补就会变成拖延。
