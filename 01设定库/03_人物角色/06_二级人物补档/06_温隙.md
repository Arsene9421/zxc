# 温隙

> 第一批跨卷二级人物。温隙是旧籍仙官，不是高大全旧天庭遗民。他的作用是提供旧籍、仙籍、道职和旧账之间的缝隙证词，同时证明旧天庭残余不全是英雄、复辟者或牺牲者。

---

## 1. 基础信息

| 项目 | 内容 |
|------|------|
| 姓名 | 温隙 |
| 名字状态 | 暂定，可保留 |
| 首次登场 | 第十二卷：旧天庭废墟 |
| 主要活跃 | 第十二卷到第十四卷，可作为旧账证人回收 |
| 人物层级 | C 级证据人物，旧籍和旧账接口 |
| 身份 | 旧籍仙官，旧天庭崩塌后在新仙界缝隙中苟活 |
| 所属势力 | 旧籍苟活者，不完全归属复辟派 |
| 修为位置 | 境界可不高，建议真仙到玄仙之间，依靠旧籍权限和遗址经验自保 |
| 专长 | 旧籍辨认、旧账片段、道职差异、遗址通行、时痕旧录 |
| 立场 | 先求活，再决定说多少真话 |

温隙的“隙”要写实：他活在旧天庭和新仙界之间的缝里，也活在真话和自保之间的缝里。

## 2. 人物核心

| 项目 | 内容 |
|------|------|
| 核心欲望 | 保住旧籍身份和性命，同时不让自己掌握的旧账彻底失传 |
| 核心恐惧 | 被复辟派当叛徒，被新仙界当旧患，被主角组当无用的脏证人 |
| 底线 | 不主动把旧账全部卖给玄阙高层；但他曾经隐瞒、交换、妥协 |
| 缺点 | 过度求活，关键时刻容易先藏证据、后谈道义 |
| 爽点功能 | 让旧账不是神秘遗物，而是从一个不干净的证人手里艰难拿到 |
| 世界观功能 | 解释旧籍、新仙籍、道职变形、旧账残片和时痕记录 |
| 冲突功能 | 逼赵长生和纪晚凝判断：证人有瑕疵，证词是否仍可用 |

温隙不能写成万能资料库。

他知道的是缝隙里的东西：旧籍如何被追索、旧账如何被改写、道职如何变形，绝不是旧天庭全貌。

## 3. 所属制度：旧籍苟活者

旧籍是旧天庭时期留下的身份、职责和权限接口。

新仙界建立后，旧籍仙官有三种命运：

1. 被清洗。
2. 投靠新仙界换取新仙籍。
3. 躲入旧墟、黑籍网络和遗址缝隙。

温隙属于第三类，但他曾与第二类有交易。

他能解释：

1. 旧籍和新仙籍的区别。
2. 旧道职与新道职的差异。
3. 旧账中哪些职责被新仙界篡改。
4. 哪些旧天庭残部是真复辟，哪些只是求活。
5. 时痕旧录如何记录某些被改写前的航迹和账目。

## 4. 能力与权限

| 权限/能力 | 用法 | 限制 |
|-----------|------|------|
| 旧籍辨认 | 分辨旧天庭遗址权限、旧道职残印、旧账格式 | 只能识别他接触过的旧职系统 |
| 遗址通行 | 借旧籍残权通过部分废墟禁制 | 权限破碎，失败会触发旧阵反噬 |
| 旧账片段 | 提供旧天庭正常循环账和新仙界改写对照 | 片段残缺，需要纪晚凝整理 |
| 时痕旧录 | 记录少量太虚航迹、旧账形成时刻或账本改写前影像 | 不可当时间倒流，只是证据残影 |
| 伪籍经验 | 知道黑籍、旧籍和新仙籍互相遮掩的方法 | 容易被新仙界追踪，也可能牵连赵长生 |

温隙可以与时本源有微弱关联，但不能成为时本源核心战力。

## 5. 与核心人物关系

| 对象 | 初始关系 | 核心冲突 | 变化方向 |
|------|----------|----------|----------|
| 赵长生 | 旧账证人和可疑求活者 | 赵长生需要证据，但不能无条件信任温隙 | 赵长生判断瑕疵证人仍可用于清账，但不能让他掌账 |
| 纪晚凝 | 旧账入口 | 纪晚凝需要旧账对照明账、内账、黑账 | 温隙提供片段，纪晚凝负责整理成可公开证据 |
| 秦照古 | 被轻视的旧籍苟活者 | 秦照古认为温隙出卖过旧天庭尊严 | 二人冲突体现旧遗民内部裂痕 |
| 穆天衡 | 旧军强硬派厌恶对象 | 穆天衡看不起温隙的妥协，温隙害怕旧军复辟后清算自己 | 温隙可反证旧军序也有压迫性 |
| 洛闻霜 | 旧灯和旧账接点 | 温隙可能知道轮回灯旧职记录的一小段 | 他不懂亡魂边界，不能替代洛闻霜 |
| 谢未明 | 命数和旧账互证 | 谢未明可检查温隙证词中的命数污染 | 双方能验证旧账是否被司命台改写 |

## 6. 旧籍与新仙籍对照

温隙最重要的戏不是讲历史，而是拿出对照。

| 项目 | 旧籍 | 新仙籍 |
|------|------|--------|
| 身份逻辑 | 职责和循环位置 | 登记、税役、命数接管 |
| 道职逻辑 | 履行公共职责 | 获得征收和压制权限 |
| 账本逻辑 | 记录修补、调度和损耗 | 记录征收、分流和补墟耗材 |
| 飞升逻辑 | 接入更大循环 | 进入验源、分流和仙税体系 |
| 命数逻辑 | 记录危机 | 安排变量和回收棋子 |

这张对照可用于纪晚凝转向：她不是因为旧天庭完美而动摇，而是因为新仙界把旧职责改成了收割接口。

## 7. 卷际弧线

| 卷 | 作用 | 状态变化 |
|----|------|----------|
| 第十二卷 | 在旧天庭废墟中登场，提供旧籍和旧账片段 | 先藏一半真相，只愿交换安全 |
| 第十三卷 | 旧账与补墟黑账对照，证明黑账被系统改写 | 被复辟派、新仙界和主角组同时争夺 |
| 第十四卷 | 可作为旧账见证者之一进入清账场景 | 不是英雄代表，而是“有瑕疵但必须作证”的旧制度幸存者 |

## 8. 可选退场方式

| 方式 | 用法 |
|------|------|
| 证人留存 | 在终局公开旧账时作证，承担自己曾经隐瞒和交易的责任 |
| 被旧军清算 | 穆天衡或复辟派残部想杀他，暴露旧复辟的强硬面 |
| 被新仙界灭口 | 因旧账片段触及黑账改写，被保全派追杀 |
| 留守旧墟 | 终局后负责整理旧籍残档，不进入新权力中心 |

不建议让温隙牺牲成完美英雄。他的灰度要保留到最后。

## 9. 可提供的关键证据

| 证据 | 说明 | 后续接入 |
|------|------|----------|
| 旧籍残册 | 证明旧籍不是新仙籍，也不接管命数税役 | 仙籍制度清算 |
| 旧账片段 | 记录旧天庭正常补天和镇界账目 | 纪晚凝旧账对照 |
| 道职改写记录 | 证明新仙界把公共职责改成征收权力 | 顾长庚、纪晚凝、赵长生 |
| 时痕旧录 | 少量账本被改写前的残影证据 | 时本源承接，太虚舟伏笔 |
| 旧遗民交易名单 | 证明旧天庭残余内部也有妥协和投靠 | 拒绝旧天庭美化 |

## 10. 禁写项

1. 不能写成全知资料老人。
2. 不能写成纯洁旧天庭忠臣。
3. 不能让他掌握时本源核心能力。
4. 不能让他的证词无需验证。
5. 不能让他替秦照古或洛闻霜说明全部旧天庭历史。
6. 不能把旧籍苟活者简单写成叛徒。

## 11. 使用提示

写温隙时，每次出场都要带出“缝隙”：

1. 一份不完整旧账。
2. 一个他不愿说出的交换条件。
3. 一处旧籍能过、新仙籍过不了的遗址禁制。
4. 一段证明新仙界改写旧职的残痕。
5. 一次他先自保、后补救的选择。

他的存在是为了证明：

> 真相不总由干净的人交出来，但清账不能只接受干净证人。
