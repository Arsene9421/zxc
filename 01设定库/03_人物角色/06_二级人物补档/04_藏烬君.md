# 藏烬君

> 第一批跨卷二级人物。藏烬君是补墟黑账守门人，不是终局最高反派。他的作用是守住血账、魂账、界账三类黑账，让白倩霜、洛闻霜、叶疏彤和纪晚凝的证据线在第十三卷汇合。

---

## 1. 基础信息

| 项目 | 内容 |
|------|------|
| 姓名 | 藏烬君 |
| 名字状态 | 暂定，可作为道号或职名 |
| 首次登场 | 第十三卷：墟眼与归墟大潮，也可在第十一卷以封账痕迹伏笔出现 |
| 主要活跃 | 第十三卷 |
| 人物层级 | B 级制度守门人，终局黑账压力人物 |
| 身份 | 补墟黑账守门人，掌管血账、魂账、界账的封存与改写 |
| 所属势力 | 补墟司封账派，背后连接玄阙仙族和保全派核心 |
| 修为位置 | 高阶仙官，建议金仙到太乙边缘，具体境界卷纲再锁 |
| 专长 | 黑账封存、账册改写、证据焚毁、墟眼耗材分类 |
| 立场 | 相信黑账一旦公开，仙界核心和墟眼封禁都会立刻崩盘 |

“藏烬”不是藏火，而是藏住世界被烧过之后留下的灰烬。

他知道一部分真相，也知道真相很难看，但他选择把灰烬封进账册深处。

## 2. 人物核心

| 项目 | 内容 |
|------|------|
| 核心欲望 | 保住补墟黑账不被公开，维持中天玄阙对墟眼封禁的最后秩序 |
| 核心恐惧 | 血账、魂账、界账同时曝光，引发仙界制度崩盘和万界反噬 |
| 底线 | 不让低层执行者随意窥黑账；他认为这是保护秩序，不是保护自己 |
| 缺点 | 把“公开真相会造成震荡”当成永久封账理由 |
| 爽点功能 | 让第十三卷的证据争夺具象化，黑账不是概念，而是可夺、可破、可清算的账册 |
| 世界观功能 | 汇合血源、魂源、界源和补墟制度死循环 |
| 冲突功能 | 逼纪晚凝从查账者变成清账者，逼主角组证明有替代循环方案 |

藏烬君不能只写成贪腐权贵。

他可怕在于：他可能真的相信公开黑账会让更多世界立刻死去，因此宁可继续喂墟眼，也不允许任何人翻账。

## 3. 所属制度：补墟黑账守门链

补墟账本分明账、内账、黑账、旧账。藏烬君守的是第三层黑账。

他控制的三类核心账：

1. 血账：百族真血、血税、族契献血、源血抽样。
2. 魂账：亡魂截留、渡劫失败者残魂、轮回灯磨损、炼魂坏账。
3. 界账：下界和中层世界界源抽取、祖地衰败、残界封存。

他的职责不是记账，而是决定：

1. 哪些账可以被看见。
2. 哪些账必须合并成损耗。
3. 哪些签押必须消失。
4. 哪些证人必须被封口。
5. 哪些世界的灰烬不允许留下名字。

## 4. 能力与权限

| 权限/能力 | 用法 | 限制 |
|-----------|------|------|
| 黑账封印 | 封存血账、魂账、界账原册 | 需要补墟司道职和中天玄阙阵权支持 |
| 焚账法旨 | 焚毁账册表层，使证据只剩灰烬和残印 | 旧账、归墟炉还原、轮回灯照魂可留下反证 |
| 耗材分类 | 将人、族群、亡魂、世界拆成可投入墟眼的类别 | 分类越细，越暴露制度罪证 |
| 权贵豁免 | 将玄阙仙族、道统高层从抽源名单中摘出 | 可被纪晚凝和严慎追出签押链 |
| 账火反噬 | 以黑账污点反噬窥账者 | 对清账道心、归墟炉重炼和轮回灯有抵抗风险 |

藏烬君可以有强大战斗力，但不能只靠战斗解决。他必须被账、证人、旧账和替代方案共同击穿。

## 5. 与核心人物关系

| 对象 | 初始关系 | 核心冲突 | 变化方向 |
|------|----------|----------|----------|
| 纪晚凝 | 清账者和封账者 | 纪晚凝要公开黑账，藏烬君要证明公开会造成更大灾难 | 纪晚凝必须用证据和新规则击穿封账理由 |
| 赵长生 | 归墟炉持有者 | 赵长生能还原残账和重炼灰烬，但不能私自掌账 | 藏烬君试图把赵长生列为最高风险耗材 |
| 白倩霜 | 血账受害链代表 | 白倩霜证明血税和猎血不是局部恶行 | 血账曝光后，白倩霜线进入百族契约重订 |
| 叶疏彤 | 生机和界源旧债代表 | 她能看出界源抽取后的生机病灶 | 界账让下界和灵界祖地衰败有账可查 |
| 洛闻霜 | 魂账边界代表 | 轮回灯能照出亡魂被截留和磨损 | 藏烬君与守灯残职形成直接冲突 |
| 严慎 | 低层账师 | 严慎的差额可能触及黑账入口 | 藏烬君可封存、灭口或利用严慎背锅 |
| 傅照灯 | 制度修补派 | 傅照灯想修账，藏烬君认为修补不如封存 | 二人体现补墟司内部的封账与修补冲突 |
| 裴玄阙 | 高层道路代表 | 藏烬君替裴玄阙路线守住最难看的证据 | 不能替裴玄阙承担终局理念罪责 |

## 6. 三账汇合

第十三卷使用藏烬君时，必须让三账具体出现。

| 账目 | 被谁击穿 | 证明什么 |
|------|----------|----------|
| 血账 | 白倩霜、明厌旧证、百族证人 | 猎血和血税是制度，不是私猎 |
| 魂账 | 洛闻霜、轮回灯、守灯旧规 | 亡魂被截留、磨损、投入补墟不是自然轮回 |
| 界账 | 叶疏彤、陆维舟、残界证据 | 下界和灵界祖地衰败来自长期抽界源 |

纪晚凝的职责不是一个人破所有账，而是把三条证据收成可公开、可追责、可改规则的清账路径。

## 7. 卷际弧线

| 卷 | 作用 | 状态变化 |
|----|------|----------|
| 第十一卷 | 以封账痕迹、被抹签押、黑账禁令形式出现 | 只让读者知道有人在账后遮掩 |
| 第十二卷 | 旧账对照暴露黑账改写痕迹，藏烬君压力上升 | 开始主动处理严慎、傅照灯或纪晚凝留下的痕迹 |
| 第十三卷 | 正面登场，守住血账、魂账、界账汇合口 | 被证据、旧账和新循环方案共同击穿，可败亡或被账火反噬 |

## 8. 可选退场方式

| 方式 | 用法 |
|------|------|
| 账火反噬 | 他试图焚毁黑账，结果被血账、魂账、界账共同反噬 |
| 封印失败 | 黑账被公开，他保住肉身但失去道职和账权 |
| 被高层抛弃 | 玄阙仙族或保全派把他推为黑账替罪者，暴露制度自保本能 |
| 证据败退 | 他不悔改，但被迫承认黑账确实存在并经过高层法旨改写 |

不建议把藏烬君写成最终大 Boss。他是黑账之门，不是墟眼和保全道路本身。

## 9. 可提供的关键证据

| 证据 | 说明 | 后续接入 |
|------|------|----------|
| 血账封卷 | 记录百族真血、血税、族契献血和猎血许可 | 白倩霜、百族契约 |
| 魂账磨损录 | 记录亡魂截留、轮回灯失位、渡劫失败者残魂去向 | 洛闻霜、轮回灯 |
| 界账抽源图 | 标注哪些下界、中层世界、祖地被抽源 | 叶疏彤、陆维舟、新登仙台 |
| 权贵豁免记录 | 证明玄阙仙族和高层道统逃避补墟消耗 | 裴玄阙路线和权贵耗材派区分 |
| 黑账改写法旨 | 证明封账不是个人行为，而有保全派法旨支持 | 终局清账 |

## 10. 禁写项

1. 不能把他写成纯粹贪官。
2. 不能让他替裴玄阙承担全部罪责。
3. 不能让黑账只是一句“很多坏账”，必须有血账、魂账、界账实体。
4. 不能让纪晚凝靠偷一本账就完成转向。
5. 不能让赵长生私自掌控黑账后直接裁决所有人。
6. 不能让藏烬君知道旧天庭全部真相，他只掌握新仙界黑账核心部分。

## 11. 使用提示

写藏烬君时，每次出场都要围绕“封账”：

1. 一本账被烧。
2. 一个证人被封口。
3. 一笔损耗被合并。
4. 一条血、魂、界证据被改写。
5. 一句“公开真相会死更多人”的封账理由。

他的存在是为了证明：

> 新仙界不是没有账，而是把万界烧成灰以后，还要有人专门把灰藏起来。
