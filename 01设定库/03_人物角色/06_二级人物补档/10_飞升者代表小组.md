# 飞升者代表小组

> 人物侧收束补档。本文细化周栖石、梅见微、钟离砺三名飞升者代表，让仙界底层旧账有具体见证者。本文不是新增主角团，也不把飞升者群像扩成长期核心队伍。

---

## 1. 设计目的

飞升者代表小组要解决六个问题：

1. 飞升者在仙界不是抽象受害群体，而是役籍、黑籍、军籍三类具体困境。
2. 周栖石、梅见微、钟离砺不再停留在阶段角色池的功能位。
3. 第十卷仙界底层必须有人能让读者看见“飞升之后仍是底层”。
4. 第十四卷至少有一名飞升者能作为旧账见证进入终局。
5. 飞升者代表不能全部变成主角团长期伙伴，避免仙界篇人物膨胀。
6. 仙籍、分流、仙矿、黑籍、军籍和镇界军要通过人物命运互相印证。

飞升者小组的核心定位：

> 他们不是新主角团，而是三种飞升后被制度定价的人。

## 2. 小组总表

| 人物 | 身份 | 代表面 | 表层目标 | 终局建议 |
|------|------|--------|----------|----------|
| 周栖石 | 仙矿飞升者 | 役籍、矿账、底层劳役 | 熬过役期，赎回自由仙籍 | 首选飞升者旧账见证 |
| 梅见微 | 失籍飞升者 | 黑籍、伪籍、灰网生存 | 躲避追捕，弄到可用身份 | 可作黑籍证人和情报通道 |
| 钟离砺 | 被征调仙兵飞升者 | 军籍、战功、镇界军 | 以战功换资源和身份 | 建议进入终局军籍清算 |

三人共同证明：

1. 飞升后弱者被役籍吞掉。
2. 失籍者被逼入灰网。
3. 强者被军籍绑定。
4. 仙界底层不是修士梦想，而是登记、分流和回收体系。

## 3. 小组边界

| 边界 | 说明 |
|------|------|
| 不做核心团 | 三人不能长期跟随赵长生每卷行动 |
| 不做恋爱线 | 三人都不进入赵长生或沈临风感情线 |
| 不做纯受害者 | 每个人都有自保、误判或灰色选择 |
| 不做同一种苦难 | 役籍、黑籍、军籍必须区分清楚 |
| 不替纪晚凝清账 | 他们提供证据和见证，清账主责仍在纪晚凝 |
| 不替沈临风司命线 | 他们可被命数记录影响，但不承担系统主线 |

## 4. 周栖石

### 4.1 基础信息

| 项目 | 内容 |
|------|------|
| 姓名 | 周栖石 |
| 名字状态 | 暂定，可保留 |
| 首次登场 | 第十卷：仙界底层 |
| 主要活跃 | 第十卷，可在第十三到十四卷回收 |
| 人物层级 | D 到 C 级飞升者代表 |
| 身份 | 仙矿飞升者，役籍矿修 |
| 所属空间 | 登仙监分流台、仙矿府、矿役营、仙籍署外账 |
| 代表旧账 | 仙矿役籍、矿契、矿账、飞升者劳役 |

周栖石的名字可以保留“栖石”意象：飞升后没有云台可栖，只能栖在矿石之间。

### 4.2 人物核心

| 项目 | 内容 |
|------|------|
| 核心欲望 | 熬满役期，拿到真正可通行的自由仙籍 |
| 核心恐惧 | 役期被矿账不断追加，永远留在仙矿府 |
| 底线 | 不愿出卖同矿飞升者去换减役，但可能隐瞒证据自保 |
| 缺点 | 习惯忍耐，早期不相信反抗有用 |
| 爽点功能 | 让赵长生在仙界重新看到底层账和残器捡漏空间 |
| 世界观功能 | 证明飞升不是终点，而是进入役籍和矿账 |
| 终局功能 | 首选飞升者旧账见证，证明仙矿旧债需要清算 |

周栖石不是热血反抗领袖。他越想安稳熬过去，越能说明制度可怕：连“我只想干完活离开”都做不到。

### 4.3 可提供证据

| 证据 | 来源 | 用途 |
|------|------|------|
| 役籍矿契 | 仙矿府入役时签押 | 证明飞升者被登记为矿役 |
| 矿账追加册 | 役期、工具、治疗、食宿费用 | 证明役期会被账目不断延长 |
| 废仙石残料 | 仙矿低价值残料 | 给赵长生仙界捡漏入口 |
| 伤残飞升者名册 | 矿役营或伤兵回收处 | 证明底层飞升者被消耗后无人负责 |
| 仙矿护送回执 | 仙矿府与补墟司、镇界军接口 | 接纪晚凝清账线 |

### 4.4 卷际弧线

| 卷 | 用法 |
|----|------|
| 第十卷 | 作为仙矿飞升者登场，带主角组看见役籍、矿账和仙界底层残料 |
| 第十一卷 | 可通过矿账回执接补墟司外账，或被追查证据牵连 |
| 第十三卷 | 仙矿账进入补墟黑账汇合，周栖石可作为旧账证人回归 |
| 第十四卷 | 首选进入终局飞升者见证，推动飞升通道和役籍规则重定 |

### 4.5 退场建议

首选不死。

原因：周栖石若只是死于仙矿压迫，功能会退化成催泪受害者。更稳的写法是让他带着矿账活到终局，证明底层飞升者不只作为尸体被看见。

可选退场：

1. 继续活着作证。
2. 被迫逃成黑籍，与梅见微线短暂交叉。
3. 为保矿账证据付出伤残代价，但不建议无意义死亡。

## 5. 梅见微

### 5.1 基础信息

| 项目 | 内容 |
|------|------|
| 姓名 | 梅见微 |
| 名字状态 | 暂定，可保留 |
| 首次登场 | 第十卷：仙界底层或黑籍坊市 |
| 主要活跃 | 第十卷到第十一卷，可在终局回声 |
| 人物层级 | D 到 C 级灰网人物 |
| 身份 | 失籍飞升者，黑籍灰网中介或逃亡者 |
| 所属空间 | 黑籍坊市、伪籍铺、登仙监追捕链、仙籍署边缘 |
| 代表旧账 | 失籍、伪籍、灰网互害、无仙籍生存困境 |

梅见微的功能是让仙界底层不只有矿役苦难，也有身份消失后的灰色生存。

### 5.2 人物核心

| 项目 | 内容 |
|------|------|
| 核心欲望 | 获得一份能长期使用的伪籍或合法仙籍 |
| 核心恐惧 | 被登仙监、仙籍署或镇界军抓回，转罪籍或矿役 |
| 底线 | 不直接把新飞升者卖给抽源链，但会卖情报、坑交易对象 |
| 缺点 | 不轻信任何人，习惯先留退路再谈道义 |
| 爽点功能 | 提供黑籍坊市、伪籍、追捕和灰网情报 |
| 世界观功能 | 证明没有仙籍几乎无法在仙界合法生活 |
| 终局功能 | 黑籍与灰网证人，可进入终局也可作为证据来源 |

梅见微不能写成单纯可怜人。她为了活下去做过不干净的事，这能让黑籍灰网更真实。

### 5.3 可提供证据

| 证据 | 来源 | 用途 |
|------|------|------|
| 伪籍样本 | 黑籍坊市伪籍铺 | 证明仙籍制度催生灰网 |
| 追捕回执 | 登仙监或镇界军黑籍追剿队 | 证明失籍者被制度追捕 |
| 黑市路线 | 黑籍坊市和残仙器坊 | 给赵长生仙界底层活动空间 |
| 失籍名单 | 被洗尘池或验源廊漏出的人 | 接登仙监和仙籍署线 |
| 灰网交易账 | 黑籍者互害、买卖身份、换庇护 | 说明底层不是天然互助 |

### 5.4 卷际弧线

| 卷 | 用法 |
|----|------|
| 第十卷 | 作为黑籍情报点登场，与赵长生互相试探，不轻易站队 |
| 第十一卷 | 提供伪籍、黑籍追捕或司命台异常宿主线索 |
| 第十三卷 | 若回归，可提供黑籍坊市如何被封锁、清理或灭口的证据 |
| 第十四卷 | 可作为黑籍证人出现，也可只以证据和名单回收 |

### 5.5 退场建议

梅见微不必一定进终局正面场。

可选方向：

1. 作为情报节点活到后期，但不进入核心行动。
2. 被迫交出灰网账，换取一批黑籍者逃生。
3. 第十三卷被封锁线牵连，留下伪籍账和追捕回执。
4. 终局只以黑籍证词出现，避免人物过多。

## 6. 钟离砺

### 6.1 基础信息

| 项目 | 内容 |
|------|------|
| 姓名 | 钟离砺 |
| 名字状态 | 暂定，可保留 |
| 首次登场 | 第十卷：登仙监分流台或镇界军征调营 |
| 主要活跃 | 第十卷、第十三卷、第十四卷 |
| 人物层级 | C 级军籍飞升者代表 |
| 身份 | 被征调的仙兵飞升者 |
| 所属空间 | 镇界军征调营、军籍登记台、镇界任务墙、抽源护送司 |
| 代表旧账 | 军籍、战功、军印绑定、强者被收编 |

钟离砺证明：强者飞升后也不一定自由。弱者挖矿，强者入军。

### 6.2 人物核心

| 项目 | 内容 |
|------|------|
| 核心欲望 | 用战功换取资源、身份和真正向上走的机会 |
| 核心恐惧 | 失去军籍后沦为黑籍、罪籍或被回收 |
| 底线 | 初期服从军令，但不愿亲手屠杀无辜世界 |
| 缺点 | 早期过度相信军功和秩序，轻视黑籍与矿役困境 |
| 爽点功能 | 让主角组和镇界军逻辑发生正面冲突 |
| 世界观功能 | 证明军籍是高战力飞升者的制度枷锁 |
| 终局功能 | 进入军籍清算，和厉沉霄、镇界军形成执行层对照 |

钟离砺不能一登场就支持赵长生。他最初应该认为主角组是在破坏自己好不容易得到的向上路径。

### 6.3 可提供证据

| 证据 | 来源 | 用途 |
|------|------|------|
| 镇界军征调册 | 镇界军征调营 | 证明强者飞升后被军籍分流 |
| 军印绑定记录 | 军印绑定室 | 证明军籍接管行动和命数接口 |
| 战功兑换册 | 镇界任务墙 | 证明资源与高危军令绑定 |
| 抽源护送军令 | 抽源护送司 | 证明镇界军参与补墟物资流转 |
| 飞升者战损表 | 伤兵回收处 | 证明军籍飞升者也被消耗 |

### 6.4 卷际弧线

| 卷 | 用法 |
|----|------|
| 第十卷 | 与主角组因军籍和黑籍问题冲突，展示战功诱惑 |
| 第十一卷 | 可因追捕失控宿主、护送账证或黑籍任务再次交叉 |
| 第十三卷 | 看见抽源护送和封锁证据后动摇，与厉沉霄形成上下级冲突 |
| 第十四卷 | 建议作为军籍飞升者代表进入终局清算，推动军印和军籍规则重定 |

### 6.5 退场建议

钟离砺适合活到终局。

可选方向：

1. 失去军印后作证。
2. 在第十三卷违令保护证据，成为军籍叛逆。
3. 战败后被主角组救下，但仍不完全加入主角团。
4. 终局参与解除军籍绑定，保留公共守界职责。

## 7. 三人关系

三人不必长期同行，但可以在第十卷形成短暂交叉。

| 关系 | 初始状态 | 冲突 |
|------|----------|------|
| 周栖石与梅见微 | 矿役和黑籍互相不信 | 周栖石怕被梅见微骗走身份，梅见微看不起矿役忍耐 |
| 周栖石与钟离砺 | 役籍和军籍的对照 | 钟离砺觉得军籍至少有路，周栖石看见军籍也在消耗人 |
| 梅见微与钟离砺 | 黑籍与追剿体系天然对立 | 钟离砺初期可能参与追捕黑籍，后期发现追捕链封证据 |

三人终局不需要成为亲密伙伴。更好的效果是：他们从互不理解到共同证明飞升制度的三种伤口。

## 8. 与核心人物关系

| 核心人物 | 关系 | 用法 |
|----------|------|------|
| 赵长生 | 底层互证和残料入口 | 通过周栖石、梅见微看见仙界废料、黑市和矿账 |
| 沈临风 | 系统宿主和飞升者分流对照 | 系统奖励与仙籍、军籍、回收链发生冲突 |
| 纪晚凝 | 清账者 | 三人提供飞升者旧账，推动纪晚凝面对制度后果 |
| 陆维舟 | 登仙台和分流节点 | 证明飞升通道不是自然道路，而是可改结构 |
| 顾长庚 | 军令和正统剑道冲突 | 钟离砺线让顾长庚面对“秩序是否正当” |
| 谢未明 | 命数记录 | 追踪飞升者分流、军印和系统回收的命数接口 |

## 9. 终局分配

建议终局分配：

| 人物 | 终局权重 | 说明 |
|------|----------|------|
| 周栖石 | 必选或强烈建议 | 飞升者旧账首选见证，承接仙矿役籍 |
| 梅见微 | 可选 | 黑籍灰网证人，若终局拥挤可用证据替代真人 |
| 钟离砺 | 建议进入 | 军籍飞升者代表，承接镇界军清算 |

不要为了“整齐”让三人都在终局发长篇陈词。周栖石和钟离砺足以撑住飞升者两条主线，梅见微可作为灰网证据补足第三面。

## 10. 禁写项

1. 不能把三人写成新核心三人组。
2. 不能让他们一遇见赵长生就完全信任。
3. 不能把周栖石写成只会受苦的苦情角色。
4. 不能把梅见微写成单纯善良的可怜黑籍。
5. 不能把钟离砺写成一开始就看透镇界军。
6. 不能让三人替纪晚凝、陆维舟、顾长庚解决仙籍、登仙台和军令主线。
7. 不能用死亡替代证据，尤其不要让周栖石只为情绪牺牲。
8. 不能把飞升者困境写成“努力不够”，必须落在登记、分流、仙籍、役期和军籍。

## 11. 使用提示

写飞升者代表小组时，每个人都要带一个具体制度动作：

1. 周栖石：签役籍、还矿账、挖废仙石、看役期被追加。
2. 梅见微：换伪籍、躲追捕、卖情报、在灰网里保命。
3. 钟离砺：受军印、接军令、换战功、发现护送对象不干净。

他们共同证明：

> 飞升不是从苦海上岸，而是被仙界重新登记、定价和分流。
