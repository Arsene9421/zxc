# 灵界账官俞简

> 人物侧收束补档。俞简是抽源账官，灵界监察链的入账代表。他不靠亲手杀人制造压迫，而是把名单、战损、族契、残魂和界源刻度折成补墟外围额度。

---

## 1. 基础信息

| 项目 | 内容 |
|------|------|
| 姓名 | 俞简 |
| 名字状态 | 暂定，可保留 |
| 首次登场 | 第八卷：万年抽源 |
| 主要活跃 | 第八卷到第九卷，后续由账符和回执接入第十到十三卷 |
| 人物层级 | C 级制度人物，小单档 |
| 身份 | 抽源账官，灵界监察链入账代表 |
| 所属势力 | 灵界监察链、抽源账站、仙界补墟司外围账口 |
| 修为位置 | 灵界账官型中层执行者，具体境界卷纲再锁 |
| 专长 | 战损拆账、血税名单复核、界源刻度折算、仙界账符上传 |
| 立场 | 相信准确入账能降低失控，避免仙界扩大抽源 |

俞简的压迫来自“平静”和“精确”。

他不需要高声威胁别人。只要把一处祖地衰败写成可补额度，那处祖地的痛苦就已经被制度拿走。

## 2. 人物核心

| 项目 | 内容 |
|------|------|
| 核心欲望 | 把灵界所有失控事件写成可上报、可解释、可平衡的账 |
| 核心恐惧 | 账目对不上，导致仙界上层判定灵界失控并扩大抽源 |
| 底线 | 不喜欢私吞、虚报和无账猎杀，认为那会破坏秩序 |
| 缺点 | 把折算准确误认为伤害可控，忽视入账本身正在吞掉世界 |
| 爽点功能 | 让主角组看见“制度杀人”不是比武，而是入账 |
| 世界观功能 | 连接明厌、抽源账站、登仙台残路和纪晚凝后期查账 |
| 冲突功能 | 逼白倩霜、叶疏彤、陆维舟、谢未明分别从血、生机、界阵、命数击穿账目 |

俞简不把自己看成恶人。

他会认为：若没有账站把损耗算清，灵界只会被更粗暴地判为高危抽源区。

## 3. 所属制度

俞简处在明厌之后、纪晚凝之前。

```text
血名册、族契裁定、猎血许可
明厌等监察使
    ↓
战损登记、血税名单、族契复核、界源刻度
俞简 / 抽源账站
    ↓
抽源额度简表、仙界账符、外账回执
补墟司外围账口
    ↓
纪晚凝后期清账线
```

明厌负责让猎血、交人和族契裁定变得“合法”。

俞简负责让合法性变成额度。

两人不能写成同一个功能。

## 4. 外貌与说话方式

建议方向：

1. 清瘦。
2. 衣袍干净，袖口有淡旧墨痕。
3. 语气平，不急不怒。
4. 习惯把伤害称为差额、波动、折损、额度。
5. 遇到尸体、残魂、祖地枯竭时，第一反应是核对分类。

可用表达习惯：

| 他说 | 真相 |
|------|------|
| 本期差额过大 | 死的人和损耗太多 |
| 血源波动需复核 | 有人被猎血或真血被标记 |
| 界源刻度偏薄 | 祖地和残界被抽空 |
| 证人需隔房 | 证人被控制 |
| 上账后再议 | 伤害先进入制度，再谈是否正当 |

## 5. 权限与能力

| 权限/能力 | 用法 | 限制 |
|-----------|------|------|
| 战损拆账 | 将战争、迁徙、死亡、祖地衰败拆成血、魂、界、运等类别 | 依赖前端文书和现场刻度，可能被证据击穿 |
| 血税名单复核 | 接收血名册、猎血许可和族契裁定 | 不能凭空制造血脉，只能改分类和折算 |
| 族契入账 | 把交人、献血、赔偿写成族内自愿或战后赔付 | 旧祖契、源血反噬和证人可推翻 |
| 界源刻度 | 调用界源刻度井、地脉量尺和残界碎光 | 能量化伤害，不能证明抽源正当 |
| 仙界账符 | 上传抽源额度简表，接收外账房回执 | 只接外围账口，不掌握完整黑账 |
| 证人隔房 | 暂押账师、族人、残界守界人 | 容易留下名册和押记证据 |

俞简不是战斗型人物。若有战斗，应通过账站阵禁、监察下线、封档夹道和证据封存形成压力。

## 6. 与核心人物关系

| 对象 | 初始关系 | 核心冲突 | 变化方向 |
|------|----------|----------|----------|
| 赵长生 | 账站异常变量 | 赵长生能从残器、账符灰和井水刻痕还原被删掉的具体伤害 | 俞简发现“残缺之物”会反咬整齐账目 |
| 沈临风 | 系统标记变量 | 系统可能把抽源额度任务化，俞简账目与系统奖励来源互证 | 让沈临风看见任务奖励也接消耗链 |
| 白倩霜 | 血源额度证人 | 她要证明真血不是族契赔偿材料 | 击穿血税名单和族契入账 |
| 叶疏彤 | 生机旧伤证人 | 她能看出祖地衰败不是自然病灶 | 击穿自然灾变伪账 |
| 陆维舟 | 界阵证据对手 | 他能证明刻度井抽的是世界根基 | 技术上破坏或校正界源刻度井 |
| 谢未明 | 命数污染追踪者 | 他能追踪抽源账和天机错算的接口 | 把俞简账符接到新司命台污染 |
| 明厌 | 前段监察使 | 明厌标记合法性，俞简折算额度 | 二人共同构成灵界制度链 |
| 纪晚凝 | 后期清账者 | 她在补墟司外账和内账中对上俞简账符回执 | 俞简成为她追查灵界旧账的入口 |

## 7. 与明厌的分工

| 项目 | 明厌 | 俞简 |
|------|------|------|
| 主要卷段 | 第六到第九卷 | 第八到第九卷 |
| 核心动作 | 标记、裁定、许可、封存 | 复核、拆账、折算、传账 |
| 压迫来源 | 监察法印、血名册、族契裁定 | 抽源额度、界源刻度井、仙界账符 |
| 代表问题 | 有印信是否等于合法 | 有账目是否等于可控 |
| 失败后果 | 暴露监察链前段 | 暴露补墟外围账口 |

明厌败退后，俞简可以接住更深一层压力：主角组以为拿到名单已经足够，结果发现名单只是入账原料。

## 8. 关键场景

| 场景 | 用法 |
|------|------|
| 战损登记厅 | 俞简把百族死亡、迁徙、祖地损耗拆成分类 |
| 血税名单室 | 白倩霜发现真血和散血民被写成额度 |
| 族契复核台 | 族内胁迫被写成自愿赔偿 |
| 魂灯灰柜 | 飞升失败和战死残魂被暂记魂账 |
| 界源刻度井 | 陆维舟、叶疏彤证明界源损耗不是自然波动 |
| 折源算室 | 俞简核心办公区，所有伤害在这里变成整齐数字 |
| 证人隔房 | 主角组发现账站控制证人而不是只统计灾情 |
| 仙界账符台 | 证明灵界账站与补墟司外账房相连 |

## 9. 可提供的关键证据

| 证据 | 来源 | 后续接入 |
|------|------|----------|
| 战损入账草表 | 战损登记厅 | 证明死亡、祖地衰败和残界崩解被拆账 |
| 血税名单副册 | 血税名单室 | 接白倩霜血账线 |
| 族契复核签押 | 族契复核台 | 证明所谓自愿赔偿有监察压力 |
| 魂灯灰签 | 魂灯灰柜 | 接洛闻霜轮回灯和魂账线 |
| 界源刻度拓片 | 界源刻度井 | 接陆维舟界阵线和叶疏彤生机线 |
| 抽源额度简表 | 抽源额度台 | 接纪晚凝补墟司内账 |
| 仙界账符残灰 | 仙界账符台 | 证明灵界账站与仙界外账房相连 |
| 证人隔房名册 | 封档夹道 | 证明账站控制证人 |

这些证据不能一次性全拿。第八卷应拿到足以证明账站存在和外围账口相连的证据，但不要提前揭开完整补墟黑账。

## 10. 卷际弧线

| 卷 | 作用 | 状态变化 |
|----|------|----------|
| 第六卷 | 只以账符残片、商队口风或监察链传闻出现 | 不正式登场，保持压迫在幕后 |
| 第七卷 | 霜羽祖地和百族盟场证据显示猎血最终会进入账站 | 俞简名字可出现在复核签押或账符上 |
| 第八卷 | 正式登场，主持抽源账站、界源刻度井和折源算室 | 作为制度线主轴对手，压迫主角组 |
| 第九卷 | 登仙台残路接收灵界账符，主角组确认仙界外围账口存在 | 俞简可败退、转移账册、被灭口或被迫留下证言 |
| 第十到十一卷 | 纪晚凝在补墟司外账房看到灵界账符回执 | 俞简不必真人出现，但账符成为清账入口 |
| 第十三卷 | 灵界血账、魂账、界账进入黑账汇合 | 俞简账目作为黑账证据的一部分公开 |

## 11. 可选退场方式

| 方式 | 用法 |
|------|------|
| 败退转移 | 俞简带走核心账册，只留下草表、拓片和账符残灰 |
| 被上级灭口 | 证明抽源账站只是外围，他知道得不够多但仍危险 |
| 被迫作证 | 不洗白，只承认分类和实际损害不一致 |
| 留作追查对象 | 第九卷后暂时消失，纪晚凝后期通过回执追到他的签押 |
| 账本站毁人未死 | 主角组毁掉界源刻度井部分量尺，俞简本人转入仙界外账线 |

不建议让赵长生杀掉俞简就解决抽源账站问题。杀人只能中断一个账官，不能摧毁入账制度。

## 12. 责任边界

俞简需要承担的责任：

1. 明知部分族契有胁迫，仍按自愿赔偿入账。
2. 明知祖地衰败与抽源有关，仍可写成自然灾变。
3. 明知证人隔房会压制证言，仍用封档名义执行。
4. 明知额度扩大意味着灵界继续变薄，仍以“避免更大抽源”为理由继续折算。

俞简不承担的责任：

1. 不知道完整墟眼黑账。
2. 不是补墟司最高层。
3. 不亲自设计新仙界收割制度。
4. 不代表裴玄阙的保全派终局理念。
5. 不替明厌、猎血商队、大族和补墟司承担全部罪责。

## 13. 与纪晚凝的后期接口

纪晚凝不应提前在灵界登场查账。

更稳的连接方式：

1. 第八卷主角组拿到抽源额度简表和仙界账符残灰。
2. 第九卷登仙台残路确认这些记录曾被验源口接收。
3. 第十卷纪晚凝在补墟司外账房见到同一类灵界账符。
4. 第十一卷她发现明账中的额度与灵界真实损害不一致。
5. 第十三卷俞简签押、严慎账证、傅照灯副录和藏烬君黑账汇合。

这样纪晚凝的转向仍由账本证据和制度责任推动，不需要靠情感立场提前倒戈。

## 14. 禁写项

1. 不能把俞简写成战斗型强敌。
2. 不能把他写成普通账房小反派，一杀就完。
3. 不能让他知道完整墟眼和补墟司黑账真相。
4. 不能让他比明厌更早承担猎血裁定功能。
5. 不能让他替补墟司、登仙监、新司命台承担全部罪责。
6. 不能让他被几句质问就洗白。
7. 不能让赵长生夺走一张表就说服灵界所有族群。
8. 不能让纪晚凝提前到灵界亲自查账，除非整体卷纲调整。
9. 不能把抽源账站写成只抽灵气或挖灵矿。
10. 不能让归墟炉直接修好所有被入账的世界伤害。

## 15. 使用提示

写俞简时，每次出场都要让他做一个“把伤害变成账”的动作：

1. 将死者人数改写成战损差额。
2. 将真血者移交写成族契赔偿。
3. 将祖地衰败写成自然灾变。
4. 将残魂灯灰写成魂源待复核。
5. 将界源裂痕写成可补额度。
6. 将证人隔离写成封档保护。
7. 将仙界回执写成流程完成。

俞简的存在是为了证明：

> 最可怕的账官不是篡改每一个数字，而是让每一个真实伤口只有变成数字后才被制度承认。
