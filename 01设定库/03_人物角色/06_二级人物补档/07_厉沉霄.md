# 厉沉霄

> 第一批跨卷二级人物。厉沉霄是镇界军高阶将领，代表新仙界保全派的武力层。他不是享乐权贵，而是把镇压当成稳定、把守界和护送抽源混为一谈的军人。

---

## 1. 基础信息

| 项目 | 内容 |
|------|------|
| 姓名 | 厉沉霄 |
| 名字状态 | 暂定，可保留 |
| 首次登场 | 第十三卷：墟眼与归墟大潮，也可在第十卷通过镇界军传闻伏笔 |
| 主要活跃 | 第十三卷 |
| 人物层级 | C 级到 B 级武力压力人物，保全派执行层代表 |
| 身份 | 镇界军高阶将领，保全派武力层 |
| 所属势力 | 镇界军，中天玄阙保全派执行层 |
| 修为位置 | 高阶战修，建议金仙或太乙边缘，具体境界卷纲再锁 |
| 专长 | 镇界兵阵、军印、飞升者征调、界裂镇压、战场封锁 |
| 立场 | 忠于秩序，认为叛乱会引发界裂和大规模死亡 |

厉沉霄的错误不是享乐腐败，而是长期把“镇住局面”当成“解决问题”。

他越认真，越能体现保全派武力层的危险。

## 2. 人物核心

| 项目 | 内容 |
|------|------|
| 核心欲望 | 镇住界裂、镇住叛乱、保住中天玄阙不在大潮前崩溃 |
| 核心恐惧 | 仙界军序失控，拒税世界、黑籍飞升者和旧天庭残部同时引爆界裂 |
| 底线 | 不纵容纯粹私欲屠杀，不认同低阶仙族把军队当家兵 |
| 缺点 | 只看战场稳定，不问稳定背后的抽源、征调和黑账 |
| 爽点功能 | 让顾长庚和主角组面对“不是坏人但必须击败”的高压军阵 |
| 世界观功能 | 连接镇界军、飞升者征调、护送抽源、保全派法旨 |
| 冲突功能 | 证明新仙界不只靠账本压人，也有能执行封锁和镇压的武力系统 |

厉沉霄可以尊重强者，也可以敬佩真正守界的人。

但只要他的军印还替抽源路线护航，他就站在主角组对面。

## 3. 所属制度：镇界军

镇界军表层职责：

1. 镇压界裂。
2. 守卫仙界边境。
3. 处理归墟污染外溢。
4. 护送补墟资源。
5. 征调飞升者入军籍。

实际功能：

1. 镇压拒绝仙税和抽源的世界。
2. 护送界源、血源、魂源等补墟耗材。
3. 清理黑籍飞升者和旧天庭残部。
4. 以“防止界裂”为名封锁证据区域。
5. 将飞升者强者编入军籍，绑定命数和战功。

厉沉霄不一定知道所有黑账，但他知道镇界军护送过“不能问来源”的补墟物资。

## 4. 能力与权限

| 权限/能力 | 用法 | 限制 |
|-----------|------|------|
| 镇界军印 | 调动军阵、锁定区域、压制黑籍飞升者 | 依赖中天玄阙授权和军籍系统 |
| 镇界兵阵 | 大范围封锁、围杀、抗归墟污染 | 阵眼和补给线可被陆维舟识破 |
| 飞升者征调 | 将飞升者编入军籍，以战功换资源 | 会留下强制征调和军籍绑定证据 |
| 护送抽源 | 保护补墟物资、界源残核、血魂账箱 | 证据一旦曝光，会证明镇界军参与抽源 |
| 军令压制 | 对低阶仙官、军籍修士和役籍飞升者形成强制命令 | 对无籍、旧籍、命数断页者效果不稳定 |

厉沉霄应当需要策略和证据共同击败。单纯硬打会削弱镇界军制度感。

## 5. 与核心人物关系

| 对象 | 初始关系 | 核心冲突 | 变化方向 |
|------|----------|----------|----------|
| 顾长庚 | 正统剑修对照 | 顾长庚斩不该存在的道职权力，厉沉霄维护军令和镇界秩序 | 二人围绕“守界是否等于服从军令”冲突 |
| 赵长生 | 高危黑籍或仙籍异常者 | 赵长生破坏抽源路线，厉沉霄视其为引发界裂的危险源 | 赵长生必须证明有替代循环，不只是破坏 |
| 陆维舟 | 界阵对手 | 陆维舟寻找新登仙台和万界节点，厉沉霄用镇界兵阵封锁旧路径 | 技术和军阵对抗 |
| 纪晚凝 | 补墟司清账者 | 纪晚凝查到账中护送记录，厉沉霄要求她服从保全大局 | 清账和军令发生冲突 |
| 钟离砺 | 被征调飞升者 | 钟离砺可体现镇界军内部飞升者困境 | 厉沉霄可能欣赏钟离砺，却仍把他当军籍资源 |
| 裴玄阙 | 保全派高层 | 厉沉霄执行保全派法旨 | 他不代表最高理念，但能证明理念如何落地为镇压 |

## 6. 与顾长庚的冲突

厉沉霄对顾长庚的质问：

> 你斩一道军令，若界裂因此扩大，死的人算谁的？

他的三层压力：

1. 用真实界裂证明危机存在。
2. 用军令效率证明松散联盟来不及反应。
3. 用飞升者军籍和战功诱惑顾长庚承认镇界军的必要性。

顾长庚必须击穿的是：

1. 守界不等于守抽源路线。
2. 军令不能替代道义判断。
3. 镇压短期有效，长期可能加速世界枯竭。
4. 斩道不是斩秩序，而是斩掉把秩序变成收割的权力。

## 7. 卷际弧线

| 卷 | 作用 | 状态变化 |
|----|------|----------|
| 第十卷 | 通过钟离砺、仙兵征调和军籍制度埋伏镇界军压力 | 厉沉霄不必正面登场 |
| 第十三卷前段 | 作为镇界军将领登场，封锁证据区域和抽源路线 | 认为主角组破坏封禁，会造成更大死亡 |
| 第十三卷后段 | 被顾长庚、陆维舟、纪晚凝分别从剑道、界阵、账证击穿 | 可战败、失印、被迫面对军令护送的是抽源而非守界 |

## 8. 可选退场方式

| 方式 | 用法 |
|------|------|
| 战败失印 | 被顾长庚斩断不该存在的军印权限，镇界军阵失效 |
| 证据动摇 | 看到护送记录和抽源账证后短暂停令，但不立刻加入主角组 |
| 军令殉身 | 执行最后军令时战死，保留复杂性但不能美化其镇压责任 |
| 后台审判 | 终局清账时作为执行层代表受审，证明制度责任不止高层 |

不建议让厉沉霄突然热血倒戈。他最多能在证据面前停令或拒绝一条明显权贵私令。

## 9. 可提供的关键证据

| 证据 | 说明 | 后续接入 |
|------|------|----------|
| 镇界军征调册 | 记录飞升者被强制编入军籍 | 飞升者代表和仙籍清算 |
| 护送抽源军令 | 证明镇界军护送过界源、血源、魂源物资 | 补墟黑账和裴玄阙道路 |
| 军印授权链 | 证明封锁和镇压来自保全派法旨 | 中天玄阙责任 |
| 界裂处置记录 | 证明危机真实存在，不是保全派凭空捏造 | 赵长生替代循环方案 |
| 飞升者战损表 | 证明军籍飞升者被当消耗品使用 | 终局飞升者旧账 |

## 10. 禁写项

1. 不能写成享乐权贵或无脑打手。
2. 不能让他知道完整黑账。
3. 不能让他靠战力替代裴玄阙理念压迫。
4. 不能让顾长庚一剑秒掉所有镇界军问题。
5. 不能把镇界军全写成坏人，内部要有飞升者和真守界者。
6. 不能让厉沉霄倒戈过快，削弱军序压力。

## 11. 使用提示

写厉沉霄时，每次出场都要带出军队逻辑：

1. 一道封锁令。
2. 一枚军印。
3. 一批被征调的飞升者。
4. 一次真实界裂危机。
5. 一条护送抽源的军令。

他的存在是为了证明：

> 新仙界的保全道路不是靠口号维持，而是靠军印、军籍、兵阵和一批相信镇压就是稳定的人执行。
