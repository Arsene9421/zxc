# 明厌

> 第一批跨卷二级人物。明厌是灵界监察使，不是单卷打手。他的作用是把灵界猎血、百族族契、血脉名单和仙界补墟账本连接起来。

---

## 1. 基础信息

| 项目 | 内容 |
|------|------|
| 姓名 | 明厌 |
| 名字状态 | 暂定，可保留 |
| 首次登场 | 第六卷：灵界底层 |
| 主要活跃 | 第六卷到第九卷 |
| 人物层级 | C 级阶段主轴人物，可短期升级为制度线对手 |
| 身份 | 灵界监察使，仙界补墟体系下沉到灵界的执行者 |
| 所属势力 | 灵界监察链，背后接仙界补墟司外围账口 |
| 修为位置 | 灵界中上层执行者，建议炼虚或合体前后，具体境界卷纲再锁 |
| 专长 | 血脉登记、族契异常识别、战损入账、监察法印 |
| 立场 | 维护监察秩序，不认为自己是猎血恶人 |

明厌这个名字可以理解为“明知可厌而仍要执行”。他不是不知道制度难看，而是相信有些难看的事必须有人做。

## 2. 人物核心

| 项目 | 内容 |
|------|------|
| 核心欲望 | 维持灵界百族冲突在可登记、可征收、可上报的范围内 |
| 核心恐惧 | 百族失控导致灵界被仙界判为高危抽源区，所有族群一起被扩大征收 |
| 底线 | 不亲手做无账猎杀，不允许猎血商队越过监察名单私吞真血 |
| 缺点 | 把“有账可查”误认为“伤害可控”，长期忽视账背后的具体人命 |
| 爽点功能 | 让读者看见主角组不是打一个猎血商队就解决问题，而是在撬制度 |
| 世界观功能 | 连接百族、猎血、血税、族契、补墟账本 |
| 冲突功能 | 逼白倩霜证明自己不是族群耗材，也逼赵长生看见灵界制度链 |

他不是色欲反派，也不是大族打手。

他可怕的地方在于：他可以平静地告诉小族，交出一个真血者，比全族被列为抽源异常更划算。

## 3. 所属制度：灵界监察链

明厌所处的监察链不是公开说“我们替仙界抽血”。

表层职责：

1. 记录百族冲突。
2. 调停族契争议。
3. 登记特殊血脉。
4. 统计战争损耗。
5. 防止归墟污染借族战扩散。

实际功能：

1. 把特殊血脉标为补墟可用资源。
2. 把弱族献血写成族契自愿。
3. 把百族战争写成自然战损。
4. 把祖地衰败、族运折损、界源病灶拆成不同账目。
5. 把血源、界源和魂源线索逐步送往仙界外围账口。

明厌通常不直接押送血脉目标。他负责让押送变得“合法”。

## 4. 能力与权限

| 权限/能力 | 用法 | 限制 |
|-----------|------|------|
| 监察法印 | 标记族契异常、封存战场证据、调用低阶监察下线 | 只能在灵界监察体系认可的区域内稳定生效 |
| 血名册 | 记录特殊血脉、真血感应、族契裂痕 | 名册未必记录真相，只记录制度需要的分类 |
| 族契裁定 | 以监察身份判定某族献血、赔偿、交人是否合法 | 可被旧族契、源血反噬或白倩霜线击穿 |
| 战损入账 | 把战争死亡、祖地损耗、血脉折损转成账目 | 需要俞简等账官接手才能进入补墟账 |
| 监察网 | 调动边城、商队、族中文书和大族联系人 | 信息链复杂，容易被主角组反向追踪 |

明厌的战斗能力不应压过白倩霜、赵长生和沈临风的主线。他的压迫主要来自权限、名单和合法性。

## 5. 与核心人物关系

| 对象 | 初始关系 | 核心冲突 | 变化方向 |
|------|----------|----------|----------|
| 赵长生 | 监察对象外来修士 | 赵长生不断破坏名单、账册和族契裁定 | 从追查异常修士，到意识到赵长生能还原账本罪证 |
| 沈临风 | 系统异常变量 | 系统任务和监察名单偶尔出现冲突 | 让沈临风发现系统奖励与血脉名单有后台接口 |
| 白倩霜 | 重点血脉目标 | 明厌要把她写成族契争议对象，白倩霜要证明自己不是资源 | 白倩霜线击穿他“有账即合法”的判断 |
| 叶疏彤 | 生机修复者 | 她能看出祖地衰败不是自然病灶 | 叶疏彤提供生机旧伤证据，反证监察账 |
| 谢未明 | 命数证据盟友的潜在对手 | 谢未明能追出名单背后的命数污染 | 明厌成为灵界监察账通往司命污染的中转点 |
| 俞简 | 灵界监察账官 | 明厌标记，俞简入账 | 二人共同把灵界损耗写入补墟额度 |
| 阮青鳞 | 猎血商队管事 | 明厌允许名单内追猎，不许私猎私吞 | 可互相利用，也可互相灭口 |

## 6. 与白倩霜的冲突

明厌对白倩霜的判断不是“我要抢一个美人”，而是：

> 霜羽族祖契裂得太深，白倩霜的返祖真血已经不是个人问题，而是灵界监察必须登记的族群风险。

他的三层操作：

1. 把白倩霜列为“族契争议核心”。
2. 迫使霜羽族内部承认“交出她可换短期豁免”。
3. 把追猎写成族群自愿、监察见证、大族代执。

白倩霜必须击穿的是：

1. 族群不能以保全名义吞掉个人。
2. 监察见证不等于自愿。
3. 真血者不是族契欠账的还款材料。
4. 族契可以重订，不只能献人。

## 7. 卷际弧线

| 卷 | 作用 | 状态变化 |
|----|------|----------|
| 第六卷 | 作为灵界监察使登场，记录白倩霜和霜羽族异常 | 站在制度高处，认为主角组只是破坏秩序的外来变量 |
| 第七卷 | 参与族契裁定，让白倩霜面对“交出个人保全族”的压力 | 发现赵长生能破坏猎血器具并还原部分罪证 |
| 第八卷 | 与俞简的监察账衔接，证明万年抽源不是传闻，而是入账流程 | 开始遮掩账目漏洞，防止上级追责 |
| 第九卷 | 登仙台残路前后被谢未明、沈临风反追踪，暴露灵界监察和仙界验源的接口 | 可败退、被灭口、转为证人，具体走向卷纲再定 |

## 8. 可选退场方式

| 方式 | 用法 |
|------|------|
| 失败退场 | 被主角组夺走血名册和监察法印，制度功能转交俞简或仙界验源投影 |
| 证据退场 | 被上级灭口，留下灵界监察和补墟司外围账口相连的证据 |
| 转化退场 | 不洗白，但承认部分账目被篡改，提供一段关键证言 |
| 跨卷败退 | 第六至第八卷持续压迫，第九卷与登仙台线一起回收 |

不建议让明厌成为长期大反派。他最好作为灵界制度链的入口，被击败后暴露更高层的俞简、验源仙使投影和补墟司账本。

## 9. 可提供的关键证据

| 证据 | 说明 | 后续接入 |
|------|------|----------|
| 血名册残页 | 记录霜羽族、玄鳞族、散血民等血脉分类 | 白倩霜血账线 |
| 族契裁定副本 | 证明所谓自愿献血实际由监察和大族共同压迫 | 百族盟约重订 |
| 监察法印裂痕 | 可追踪到仙界外围账口 | 纪晚凝补墟账本线 |
| 战损入账草表 | 把百族战争损耗拆成血、界、魂等账目 | 第八卷万年抽源 |
| 猎血商队许可 | 证明阮青鳞不是私猎，而是名单内代执 | 猎血制度链 |

## 10. 禁写项

1. 不能写成色欲反派。
2. 不能写成单纯大族走狗。
3. 不能让他知道完整墟眼和新仙界黑账真相。
4. 不能让他靠个人战力压过制度功能。
5. 不能让他一失败就洗白成主角阵营。
6. 不能把灵界猎血问题都推给他一个人。

## 11. 使用提示

写明厌时，要让他的每一次出场都带来一种制度压力：

1. 某个名字被写进名单。
2. 某个族契被裁定。
3. 某个战损被改写成自愿。
4. 某个商队获得合法追猎理由。
5. 某个小族被迫在交人和灭族之间选择。

他的存在是为了证明：

> 灵界猎血不是暗巷里的恶事，而是有名单、有印信、有账目的制度。
