| 项目 | 内容 |
|------|------|
| 首次登场 | 第十卷，仙界底层 |
| 人物层级 | A，核心道侣与长线人物 |

## 2. 人物核心

| 项目 | 内容 |
|------|------|
| 核心欲望 | 让制度对结果负责，而不是只维持漂亮总账 |
| 核心恐惧 | 自己多年执行的制度其实一直在合法化屠戮 |
| 底线 | 愿意看真实账本，承认制度必须为具体死账负责 |
| 缺点 | 长期只看总账，前期会压下个体痛苦以维护整体稳定 |
| 爽点功能 | 从制度内部查账、翻账、清账 |
| 世界观功能 | 仙籍、仙税、补墟账本、飞升者回收 |
| 情感功能 | 逼赵长生回答：推翻旧秩序后，账由谁来还 |

## 3. 她为什么相信补墟体系

纪晚凝不能一开始就知道自己错了。

她相信补墟体系，是因为她确实见过：

- 墟眼漏本源。
- 仙界封禁阵松动。
- 归墟污染者增多。
- 飞升者和下界本源能短期延缓危机。
- 旧天庭失败后的混乱残卷。

她的错误不是“没看见危机”，而是太久只看总账。

她看见的是“本源损耗下降一成”，看不见那一成背后是多少下界灵脉、百族源血、飞升者魂火。

## 6. 能力与限制

纪晚凝的强处不是战斗碾压，而是制度权限和账本理解。

可写能力：

- 看懂仙籍异常。
- 调阅明账和部分内账。
- 识别验源流程里的伪账。
- 用道职印信短时间压制无籍飞升者。
- 在补墟司内部寻找漏洞。

限制：

- 她不能一开始就看黑账。
- 她不能一场嘴炮就背叛制度。
- 她转向后也要背负曾经执行制度的旧债。
- 她不能成为赵长生的情报工具人，必须有自己的清账职责。

## 7. 禁写项

1. 不能写成冰山女官被主角魅力融化。
2. 不能因为爱情倒戈。
3. 不能一查账就立刻推翻几千年信念。
4. 不能把她曾经执行制度的责任轻轻抹掉。
5. 不能让她终局成为仙后或新权贵。
