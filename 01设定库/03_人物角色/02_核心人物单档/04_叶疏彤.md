# 叶疏彤

## 1. 基础信息

| 项目 | 内容 |
|------|------|
| 首次登场 | 第二卷，青云外门灵田任务 |
| 人物层级 | A，核心道侣与长线人物 |
| 身份 | 没落灵农家族后人，外门灵田线人物 |
| 灵根 | 木水双灵根，偏木，不是天灵根 |

| 主线 | 生机线、洞府灵圃、补界生机 |
| 终局 | 新循环生机线核心，不做仙后和权力附属 |

## 2. 名字意象

“疏”是疏理、疏通、疏枝去病。

“彤”不直接是草木字，需要用人物意象补足：叶疏彤早期擅长救活一种濒死的赤纹灵禾，枯叶见光后会泛出极淡彤色。这个意象对应她的道：不是凭空造出生机，而是把被堵住、被压住、快死的生机重新疏出来。

她的生机不是温室里的花，而是废田里仍能返青的一线红光。

## 3. 人物核心

| 项目 | 内容 |
|------|------|
| 核心欲望 | 保住家族最后几亩灵田，后来让残败之地重新能养人 |
| 核心恐惧 | 自己再努力也只能看着灵田、亲人、同伴一点点枯死 |
| 底线 | 不把灵田、人命和生机当一次性耗材 |
| 缺点 | 习惯忍债和守成，不轻易求助，早期容易把责任压在自己身上 |
| 爽点功能 | 种田经营、丹材闭环、废土返生 |
| 世界观功能 | 补足归墟炉“重炼残缺”之后的“如何新生” |
| 情感功能 | 逼赵长生学会共同承担，而不是把风险隔开 |

## 4. 出身与现实压力

叶家曾是青云门附属灵农家族，靠几亩灵田和低阶丹材供给活着。后来灵脉衰退、债契转卖、族中长辈突破失败，家族只剩下贫瘠灵田和旧债。

叶疏彤不是宗门公主，也不是天才灵植师。

她的专业来自现实压力：

- 灵田产量少一成，就还不起债。
- 丹材保存失败，就会被丹房压价。
- 灵泉浑浊，就要用更长时间调土。
- 她没有资格浪费种子，所以每一次试种都要记账。

她和赵长生能互相理解，是因为两人都知道资源不够时，人会被迫变得很细。

## 7. 禁写项

1. 不能一见钟情。
2. 不能变成洞府管家或正宫管理者。
3. 不能靠献出生机本源帮赵长生升级。
4. 不能确认关系后失去专业性。
5. 不能用温柔替代判断，她可以安静，但必须有自己的选择。
