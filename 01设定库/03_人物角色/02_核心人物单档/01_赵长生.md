# 赵长生

## 1. 定位

主主角。

五行杂灵根，青石坊市丹铺杂役，命数断页之人，归墟炉持有者。

他不是天命之子，不是高贵血脉，不是旧天庭提前安排的救世主。

他是一个本该死在青槐旧灾里，却因一念之善活下来的底层小修。

“长生”不是天命暗示，而是凡俗父母给孩子取的愿名。乱世、药瘴、修士斗法和山灾太多，普通人给孩子取这个名字，只是盼他活久一点、别早死。

所以这个名字的核心是反讽：他叫长生，却在七岁时已经被命簿判死过一次；他后来追求的也不是个人永远不死，而是让坏死的万界循环重新活下去。

## 2. 性格

- 谨慎，不轻信机缘。
- 能忍，但不是懦弱。
- 不主动害无辜。
- 对确认的敌人不留后患。
- 不出风头，先留退路。
- 不做空口善事，但有自己的底线。

一句话：

> 外冷内有一线火。

更完整的性格、缺点、互动规则见：`../03_关系情感/03_双主角性格与关系.md`。

## 3. 成长线

| 阶段 | 心态 |
|------|------|
| 前期 | 我只想活下去 |
| 中期 | 我要长生，我要掌握自己的命 |
| 后期 | 我不接受仙界把下界众生当资源 |
| 终局 | 仙路已断，我来重炼 |

## 4. 青槐旧灾

赵长生七岁时，青槐村一夜覆灭。

坊市档案：

> 散修斗法，引动山洪与药瘴，青槐村死绝。

真相：

旧天庭残器碎屑扰乱青槐村附近地脉和命数，司命台残簿将全村命火判为熄灭。

赵长生在命簿上已经死过一次。

无名老修用一块沾有归墟炉灰的残铜护住赵长生命火。炉灰残铜后来融入赵长生心脉，只留下胸口极淡黑色烙痕。

## 5. 命数残缺

结果：

- 命簿上赵长生是已死之人。
- 天命系统无法完整读取他的未来。
- 天机推演只能看到断页。
- 仙界监察者初期会忽略他。
- 归墟炉能感应到他身上的残缺气息。

核心句：

> 完整者入不得归墟。有缺者，方能重炼。

## 6. 第一卷表现

第一卷只露痕迹：

- 杂役契上写“青槐灾民赵长生”。
- 掌柜骂他“死人堆里捡回来的小命”。
- 他遇到黑雨、药瘴、苦井水时短暂失神。
- 胸口旧烙痕在归墟炉第一次真正启动时发热。
- 沈临风系统无法显示他的气运，只显示“关键人物”。

不能第一卷揭露命簿已死真相。

## 7. 人物核心细化

| 项目 | 内容 |
|------|------|
| 核心欲望 | 先活下去，再掌握自己的命，最终让坏死的万界循环重新活下去 |
| 核心恐惧 | 自己和身边人再次被写进别人的账里，像废丹、残器、下界一样被榨干 |
| 底线 | 不主动害无辜，不把人当材料，不用牺牲弱者换自己长生 |
| 最大缺点 | 过度谨慎，习惯独自背风险，容易把亲近的人排除在危险之外 |
| 爽点功能 | 废物翻身、低成本试错、残缺重炼、以小搏大但不乱越级 |
| 世界观功能 | 归墟炉、命数断页、残缺可炼、终局重炼循环 |
| 情感功能 | 从不信任任何人，到承认道侣和同伴不是负担，而是各自有道 |

赵长生不能写成天生冷酷。

他的冷来自长期贫穷、缺靠山、缺退路。底层修士没有资格热血，因为一次冲动就会死。他做事先算灵石、伤势、逃路、痕迹和后账，不是因为无情，而是因为他太清楚“没有后路”的代价。

## 8. 资源线

赵长生的成长必须始终和资源账绑定。

| 阶段 | 他缺什么 | 他如何补 |
|------|----------|----------|
| 坊市杂役 | 灵米、废丹、住处、低阶法术 | 清炉灰、捡废料、归墟炉小试验 |
| 外门阶段 | 完整功法、稳定灵气、身份资格 | 丹房废料、残卷互证、低调完成任务 |
| 筑基阶段 | 筑基主药、护法环境、可靠丹方 | 黑市、拍卖暗庄、残丹方重炼 |
| 洞府阶段 | 灵脉、护山阵、丹药生意、可信人手 | 废山经营、叶疏彤灵田、陆维舟阵法 |
| 灵界阶段 | 适应界压、残界资源、本源认知 | 洞天碎片、百族证据、界源修补 |
| 仙界阶段 | 仙籍、仙石、残仙器、避开验源 | 黑籍伪装、废仙石、残仙器坊 |

他每一次变强，都要回答三件事：

1. 资源从哪里来。
2. 代价由谁承担。
3. 会不会暴露归墟炉或命数断页。

## 9. 关系线

| 对象 | 初始状态 | 核心冲突 | 成熟状态 |
|------|----------|----------|----------|
| 沈临风 | 危险的系统持有者 | 系统是否可信，沈临风是否会被系统利用 | 生死搭档，分别重炼循环和司命规则 |
| 叶疏彤 | 灵田和丹材合作者 | 赵长生习惯把风险挡在自己身外 | 生机道侣，共同承担补界责任 |
| 白倩霜 | 百族盟友，初期互相试探 | 救一个人还是尊重一族的命 | 血源道侣，重订百族契约 |
| 纪晚凝 | 补墟司制度对手 | 总账能否抹去具体死账 | 清账道侣，公开旧账 |
| 陆维舟 | 技术合作者 | 陆维舟怕大因果，赵长生需要界阵 | 新界阵维护者 |
| 顾长庚 | 正统竞争者 | 正统道路和残缺道路互相质疑 | 认可但不同路 |
| 谢未明 | 命数观察者 | 赵长生抗拒被算，谢未明想算清断页 | 命数证据盟友 |

赵长生最大的情感成长不是“学会爱”，而是学会承认：

> 把所有风险都挡在自己身上，不是保护同伴，而是剥夺他们承担自己道路的资格。

## 10. 分卷弧线

| 卷 | 人物推进 |
|----|----------|
| 1 | 从只想活下去，到第一次杀人后明白自己已经无法回到普通杂役生活 |
| 2 | 进入宗门体系，发现规矩能保护人，也能筛掉无资格者 |
| 3 | 筑基不靠正统渠道，第一次真正把自己的路和宗门路分开 |
| 4 | 经营洞府，开始从独自求活转向小团队资源循环 |
| 5 | 看见断仙门和天劫异常，个人长生目标第一次被世界真相压住 |
| 6 | 灵界底层重开局，确认更高世界不是答案，只是更高层压迫 |
| 7 | 面对白倩霜和百族线，学会不能把人从族群责任里单独救走 |
| 8 | 从修物进入修界，理解残缺不是器物特例，而是世界病灶 |
| 9 | 登仙台残路让他确认飞升是回收，不是奖赏 |
| 10 | 仙界底层重新求活，从躲账开始理解账本制度 |
| 11 | 与沈临风、纪晚凝互证司命台和补墟账本 |
| 12 | 拒绝旧天庭复辟，承认旧制度曾有功但不能照旧恢复 |
| 13 | 面对裴玄阙，明确危机真实不等于收割正当 |
| 14 | 重炼墟眼和仙界循环，不坐仙帝位 |

## 11. 禁写项

1. 不能因为名字叫长生，就写成天生命定长生。
2. 不能把谨慎写成面瘫和装逼。
3. 不能让归墟炉无代价解决道德选择。
4. 不能把三条感情线写成赵长生的奖励。
5. 不能让他全职业登顶，必须保留叶疏彤、陆维舟、谢未明等人的专业性。
