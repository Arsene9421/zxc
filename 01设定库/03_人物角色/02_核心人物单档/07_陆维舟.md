## 2. 人物核心

| 项目 | 内容 |
|------|------|
| 核心欲望 | 做出真正能保命、承界、护住退路的阵 |
| 核心恐惧 | 自己修的阵只能护住几个人，却挡不住更大的世界崩坏 |
| 底线 | 不做明知会害死底层修士的坏阵，不把阵法当割人命的网 |
| 缺点 | 怕卷入大因果，技术上谨慎保守，早期只想修好自己的阵盘 |
| 爽点功能 | 废阵修复、洞府经营、界阵落地 |
| 世界观功能 | 洞府、小世界、登仙台、万界节点 |
| 队伍功能 | 证明赵长生不能一个人包办所有技术 |

## 3. 人物气质

陆维舟不是热血天才，也不是工具人。

他像一个真正的技术修士：

- 说话不多，但一说就是结构、材料、阵眼和风险。
- 不喜欢临场拍脑袋。
- 讨厌沈临风把系统奖励的阵盘直接拿来试。
- 对赵长生的残器重炼很感兴趣，但更关心稳定性。
- 早期怕被拖入大因果，因为阵法一旦出错，死的不是一个人。

## 6. 能力边界

陆维舟可以解决技术问题，但不能替赵长生解决核心命题。

可写：

- 修废阵盘。
- 改洞府护山阵。
- 稳定界裂。
- 找阵眼漏洞。
- 修登仙台节点。

不可写：

- 一人一夜布出跨界大阵。
- 不消耗资源修好所有阵。
- 取代赵长生完成归墟重炼。
- 取代叶疏彤处理生机，取代谢未明处理命数。

## 7. 禁写项

1. 不能变成随叫随到的阵法工具箱。
2. 不能全职业兼修登顶。
3. 不能每次都临场灵感爆发解决危机。
4. 不能没有个人恐惧和代价。
