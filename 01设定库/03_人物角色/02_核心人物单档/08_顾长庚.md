## 2. 人物核心

| 项目 | 内容 |
|------|------|
| 核心欲望 | 守住真正值得守的道，而不是只赢同代修士 |
| 核心恐惧 | 自己一生相信的宗门正统只是漂亮外壳 |
| 底线 | 不欺弱，不用正道名义做邪事，不为胜负毁掉剑心 |
| 缺点 | 过度相信宗门规矩和正统资格，早期看不见规则外的人 |
| 爽点功能 | 正统天才压迫感、剑修战斗、理念对照 |
| 世界观功能 | 正统功法、万法碑、斩道剑、旧天庭道路 |
| 队伍功能 | 让赵长生的残缺路始终面对正统质疑 |

## 3. 他不是恶少

顾长庚不能写成嫉妒赵长生的反派富家子。

他前期压迫感来自“资格”：

- 他有宗门培养资格。
- 他能进更好的剑堂。
- 他能争正规的筑基资源。
- 他相信贡献点、规矩、传承序列本该如此。

他的问题不是坏，而是他早期以为：

> 只要规矩足够正，走不上去的人就是不够强。

赵长生的存在会逼他看见：有些人连进入规矩的资格都没有。

## 6. 战斗与道心

顾长庚的战斗应当干净、正、锋利。

适合：

- 正面剑压。
- 同境强攻。
- 破邪阵。
- 守阵眼。
- 关键时刻斩断道职权力。

不适合：

- 阴毒暗算。
- 靠家世压人。
- 频繁输给赵长生后无能狂怒。

他的道心破裂点不是输给赵长生，而是发现：

> 自己守的规矩，可能正把无资格者排除在生路之外。

## 7. 禁写项

1. 不能写成恶少反派。
2. 不能只负责被赵长生打脸。
3. 不能让剑道只剩战斗爽点，没有道心问题。
4. 不能后期突然完全投靠赵长生，他应认可但保留自己的路。
