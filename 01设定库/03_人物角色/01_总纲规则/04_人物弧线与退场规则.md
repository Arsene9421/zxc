# 人物弧线与退场规则

> 本文档解决两个问题：一是长线人物如何成长，二是阶段人物如何退出。长篇最容易崩的不是人物少，而是人物登场后没有去向，或为了情绪随意死亡。

---

## 1. 人物弧线总原则

1. 核心人物要有“欲望、缺点、选择、代价、结果”五段。
2. 长线人物每个大阶段必须至少推进一次认知或关系。
3. 阶段人物功能结束后必须退场、转后台、死亡、成为证据或升级为长线。
4. 人物死亡不能只为刺激主角，必须改变资源、证据、责任或世界理解。
5. 人物背叛必须有前置利益和恐惧，不靠突然降智。
6. 人物被说服前必须先被事实、代价或失败逼到墙角。

## 2. 弧线类型

| 类型 | 定义 | 适用人物 |
|------|------|----------|
| 生存弧 | 从求活到掌握命运 | 赵长生、下界底层人物、飞升者群像 |
| 破命弧 | 从相信天命到拒绝安排 | 沈临风、谢未明、系统前宿主 |
| 生机弧 | 从保一亩田到补一界生机 | 叶疏彤、木灵族医者、灵植人物 |
| 族群弧 | 从保个人到重订族群责任 | 白倩霜、百族人物 |
| 清账弧 | 从执行制度到公开旧账 | 纪晚凝、补墟司人物 |
| 正统弧 | 从相信规矩到选择底线 | 顾长庚、宗门人物 |
| 复辟弧 | 从怀念旧制到承认旧制已败 | 秦照古、旧天庭人物 |
| 边界弧 | 从守旧规则到重定循环边界 | 洛闻霜、轮回灯人物 |
| 保全弧 | 从真实危机出发走向制度腐烂 | 裴玄阙、新仙界保全派 |
| 证据弧 | 从碎片线索到关键证据 | 老散修、残魂、账师、飞升失败者 |

## 3. 核心人物弧线表

| 人物 | 初始欲望 | 核心缺点 | 关键选择 | 终局状态 |
|------|----------|----------|----------|----------|
| 赵长生 | 活下去，掌握自己的命 | 习惯独自背风险，过度谨慎 | 不接受仙界把众生当资源，选择重炼循环 | 归墟道祖，不坐仙帝位 |
| 沈临风 | 靠系统成为主角 | 前期信系统，怕死，容易膨胀 | 拒绝系统“远离赵长生”任务 | 重定司命盘规则 |
| 叶疏彤 | 保住灵田和家族残债 | 过度习惯自己扛，不轻易求助 | 从经营灵田走向补界生机 | 新循环生机线核心 |
| 白倩霜 | 逃过猎血，保住自己 | 害怕族群责任吞掉个人 | 不只逃离，而是重订百族契约 | 血源循环和百族盟约代表 |
| 纪晚凝 | 执行补墟职责，维持总账 | 长期只看总账，不看具体人命 | 公开黑账，承认制度罪责 | 新循环清账和监察职责 |
| 陆维舟 | 做可靠阵修，保住退路 | 害怕卷入太大因果 | 从修阵盘到修界阵 | 新登仙台和界阵维护者 |
| 顾长庚 | 走正统剑道，守宗门秩序 | 过度相信正统资格 | 斩断不该存在的道职权力 | 新斩道剑候选执剑者之一 |
| 谢未明 | 算清命数 | 想用推演掌控不确定 | 承认命不可垄断 | 新司命盘记录者之一 |
| 秦照古 | 恢复旧天庭 | 不愿承认旧天庭失败 | 承认复辟不是答案 | 守残墟或转入新循环 |
| 洛闻霜 | 守轮回灯旧规 | 害怕归墟炉越过生死边界 | 接受新循环，但坚持亡魂边界 | 轮回灯重启者之一 |
| 裴玄阙 | 保住仙界核心 | 把万界降格成燃料 | 拒绝承认制度已无法解决危机 | 道路被证据击穿 |

## 4. 分阶段推进规则

### 4.1 下界篇

下界篇人物弧线重点：

| 人物 | 推进要求 |
|------|----------|
| 赵长生 | 从被压迫杂役变成能建立小团队和洞府的人 |
| 沈临风 | 从把系统当外挂，到发现奖励有代价痕迹 |
| 叶疏彤 | 从灵田求活，到成为洞府生机核心 |
| 陆维舟 | 从修废阵盘，到承担洞府安全 |
| 顾长庚 | 从正统天才，到看见宗门秩序护不住所有人 |
| 谢未明 | 第五卷登场，用推演失败证明命数污染 |

下界篇不能让人物过早完成终局认知。

### 4.2 灵界篇

灵界篇人物弧线重点：

| 人物 | 推进要求 |
|------|----------|
| 赵长生 | 从修物、修器，进入修界、修残界的认知 |
| 沈临风 | 从怀疑系统，到敢拒绝系统关键任务 |
| 叶疏彤 | 生机线从灵田扩展到祖地和界源恢复 |
| 白倩霜 | 从被追猎者变成百族契约重订的主体 |
| 陆维舟 | 从洞府阵修升级到界阵维护者 |
| 谢未明 | 从天机推演者变成司命台证据追踪者 |

灵界篇不能让白倩霜只做被救者，也不能让百族问题在一卷内解决。

### 4.3 仙界篇

仙界篇人物弧线重点：

| 人物 | 推进要求 |
|------|----------|
| 赵长生 | 从躲仙籍到清补墟账，再到重炼循环 |
| 沈临风 | 从系统反抗者到司命规则重启者 |
| 纪晚凝 | 从制度执行者到清账责任人 |
| 秦照古 | 从复辟诱惑到旧制失败证明 |
| 洛闻霜 | 从守旧轮回边界到接受新循环边界 |
| 裴玄阙 | 从合理危机提出者到道路失败者 |

仙界篇不能把全部问题简化成杀掉裴玄阙。

## 5. 退场类型

| 退场类型 | 适用对象 | 要求 |
|----------|----------|------|
| 死亡退场 | 阶段反派、证据人物、战争群像 | 死亡必须改变局势或留下证据 |
| 失败退场 | 阶段反派、理念对手 | 失败后其道路局限要更清楚 |
| 后台退场 | 灰色商人、宗门小人物、灵界盟友 | 保留回声，不占主线篇幅 |
| 留守退场 | 洞府人物、灵界据点人物、下界幸存者 | 负责维护某个基地或制度后果 |
| 证据退场 | 残魂、账师、失败飞升者 | 留下账本、残影、物证、誓言 |
| 转化退场 | 小反派、制度执行者 | 不必洗白，但可转为证人或灰色合作 |
| 升级长线 | 原阶段人物承担跨卷功能 | 必须补入索引和人物弧线 |

退场不是消失。退场后的影响要能被后文看见。

## 6. 死亡规则

### 6.1 可以死的人物

| 类型 | 可死亡条件 |
|------|------------|
| 底层样本人物 | 死亡展示规则杀人，而不是只刺激主角 |
| 阶段反派 | 死亡结束当卷资源冲突，并留下上游线索 |
| 战争群像 | 死亡体现战争代价，不能为了凑惨 |
| 证据人物 | 死前必须留下有效证据或改变某人认知 |
| 理念执行者 | 死亡不能替代理念失败，还要有证据击穿 |

### 6.2 不宜轻易死亡的人物

| 人物 | 原因 |
|------|------|
| 叶疏彤 | 生机线和下界情感线核心，死亡会削断世界观功能 |
| 白倩霜 | 百族和血源线核心，死亡容易变成血包悲剧 |
| 纪晚凝 | 补墟账本和仙界制度线核心，必须承担清账 |
| 陆维舟 | 界阵和登仙台节点需要长期技术承担者 |
| 谢未明 | 命数和司命盘记录线需要长期互证 |
| 洛闻霜 | 轮回灯边界功能不可被赵长生替代 |

这些人物可以重伤、失势、分线、被迫留守，但不应为了制造刺激随便杀掉。

## 7. 背叛与转向规则

背叛可以写，但必须有三层准备：

1. 利益准备：人物为什么会得到好处或避免损失。
2. 恐惧准备：人物怕失去什么。
3. 价值准备：人物说服自己这样做仍然合理。

可写背叛：

- 小族长老牺牲白倩霜换取族群短期安全。
- 灵界大族投靠仙界换取万年抽源豁免。
- 补墟司同僚封账保护仙界核心。
- 修仙家族旁支出卖主家换筑基机会。

禁写背叛：

- 没铺垫突然黑化。
- 为了制造误会强行隐瞒。
- 女主或同伴因吃醋、争宠、降智背叛。
- 反派临死突然洗白且没有代价。

## 8. 人物回收检查

每写完一卷卷纲前，检查以下问题：

1. 本卷新增的有名人物是否都有去向。
2. 本卷死去的人物是否改变了局势或证据。
3. 本卷留下的人物是否会在后续卷有功能。
4. 本卷反派失败后，其上游制度是否更清楚。
5. 本卷主角组成员是否至少有一人经历认知变化。
6. 本卷是否有普通人或底层修士承担世界真实感。
7. 本卷是否有人物被主角替代了专业功能。

## 9. 复活与残魂边界

人物死亡后不能随便复活。

规则：

1. 残魂影像不等于本人。
2. 轮回灯负责亡魂归处，归墟炉不能取代轮回灯。
3. 归墟炉可以从残物、残痕、残魂中还原证据，但不能无代价复活完整人格。
4. 高阶人物若以残魂形式出现，必须有寿元、神魂、本源或轮回代价。
5. 死亡角色回声可以存在，但不能变成随身导师。

该规则与 `../../04_修行体系/30_神魂夺舍与轮回细则.md` 保持一致。

## 10. 防人物膨胀规则

1. 同一功能的角色超过两个，要合并或明确分工。
2. 阶段人物跨两卷仍无新功能，应退场。
3. 新增长线人物必须回答：他承担哪条现有人物不能承担的线。
4. 后期新增人物不能抢走已铺人物的终局职责。
5. 女主、同伴、反派、群像都要服务世界运转，而不是增加热闹感。

最终目标：

> 人物多，但每个人有位置；人物退，但后果还在。
