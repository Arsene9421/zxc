# 人物体系总纲与角色层级

> 本文档解决人物设定的系统化问题：十四卷长篇不能只有几个核心人物和零散反派，必须先建立角色层级、功能标准、投放密度、退场规则和审计口径。

---

## 1. 当前问题判断

现有人物文档已经定下了双主角、三条女主线、若干长线同伴和理念反派，但还存在四个硬缺口：

1. 只有“谁重要”，还没有“每类人物最低要承担什么功能”。
2. 只有长线人物较清楚，阶段人物、制度人物、群像人物不足。
3. 只有关系网雏形，缺少每卷角色配置和人物进出机制。
4. 只有人物定位，缺少统一模板和后续审计清单。

所以全设定审计应当放在人物系统补齐之后。

## 2. 人物设计总原则

1. 每个重要人物必须有独立欲望，不是主角功能插件。
2. 每个长期人物必须绑定一条世界观线、修行线、职业线、制度线或情感线。
3. 每个阶段人物必须服务一个明确冲突，功能结束后要死亡、退场、转入后台或升级成长线。
4. 群像人物可以简，但不能假。底层人物要有具体生计，高层人物要有具体职责。
5. 反派不能只有恶意，还要有资源需求、制度位置或理念理由。
6. 女性角色不能默认进入恋爱线，恋爱线也不能吞掉人物道途。
7. 人物死亡、背叛、转向、牺牲都必须改变局势或认知，不能只做情绪消费。
8. 后期人物不能只负责解释设定，必须带来新压力、新证据或新选择。

## 3. 角色层级

| 层级 | 名称 | 代表 | 最低要求 | 退场规则 |
|------|------|------|----------|----------|
| S | 双主角 | 赵长生、沈临风 | 独立成长线、终局职责、互相纠偏 | 不退场，只能阶段性分线 |
| A | 核心道侣与长线同伴 | 叶疏彤、白倩霜、纪晚凝、陆维舟、谢未明、顾长庚 | 独立目标、专业能力、卷际成长、终局职责 | 可暂退，不应被剧情遗忘 |
| B | 长线理念对手 | 秦照古、洛闻霜、裴玄阙 | 能提出主角必须回答的问题 | 失败方式要证明其道路局限 |
| C | 阶段主轴人物 | 每卷主要盟友、阶段反派、制度执行者 | 承担当卷核心矛盾 | 功能结束后必须有明确去向 |
| D | 制度与职业人物 | 仙籍官、账师、阵师、丹师、族老、商队管事 | 让世界运转可信 | 可反复出现，但不抢主线 |
| E | 群像与样本人物 | 飞升者、矿修、棚区修士、小族族人 | 展示某类人的真实处境 | 可死亡、留守、转为证据 |
| F | 伏笔人物 | 无名老修、旧天庭残影、系统前宿主 | 留下问题、物证或未解因果 | 回收时必须改变理解 |

层级不是战力排行，而是叙事责任排行。

## 4. 各层级最低信息量

| 层级 | 必填信息 |
|------|----------|
| S | 出身、灵根或异常、性格缺点、核心欲望、恐惧、底线、修行路线、终局职责、与另一主角的互补和冲突 |
| A | 初登场卷、专业能力、独立目标、资源需求、道途选择、与赵长生关系、与沈临风关系、阶段成长、终局功能 |
| B | 理念、合理性、错误点、证据链、与赵长生冲突点、失败方式、失败后世界如何变化 |
| C | 当卷功能、资源动机、冲突对象、能制造什么压力、是否可杀、是否可转化、卷尾去向 |
| D | 职位、职责、利益来源、受谁管、能提供什么信息、会遮蔽什么真相 |
| E | 生计、处境、代表问题、与主角一次具体交集、是否作为后续回声 |
| F | 伏笔物、伏笔话、首次出现方式、回收卷、回收后解释什么 |

如果一个人物达不到对应层级的最低信息量，应降级为工具角色或暂不命名。

## 5. 人物功能分类

每个有名人物至少承担下列一种功能，核心人物至少承担两种。

| 功能 | 说明 | 示例 |
|------|------|------|
| 资源功能 | 推动资源争夺、价格、债务、交易 | 丹铺掌柜、拍卖暗庄、仙矿吏 |
| 职业功能 | 展示丹、器、符、阵、灵植、天机等技艺 | 叶疏彤、陆维舟、谢未明 |
| 世界观功能 | 让归墟、司命台、百族、补墟账本落地 | 白倩霜、纪晚凝、秦照古 |
| 情感功能 | 形成信任、道侣、同伴、背叛或告别 | 叶疏彤、白倩霜、纪晚凝 |
| 理念功能 | 提出主角必须回答的问题 | 顾长庚、秦照古、裴玄阙 |
| 压迫功能 | 制造具体生存压力或制度压力 | 胡三奎、灵界监察使、仙籍官 |
| 证据功能 | 留下账本、残魂、符诏、旧器痕迹 | 谢未明、洛闻霜、飞升失败者 |
| 后果功能 | 让主角的选择产生代价和回声 | 宁晚秋、下界灾民、仙矿飞升者 |

禁止出现只负责“夸主角、送资源、解释设定、制造误会”的重要人物。

## 6. 人物投放密度

十四卷长篇需要稳定的人物密度，但不能人物爆炸。

| 阶段 | 每卷建议新命名人物 | 可保留进入后续的人物 | 说明 |
|------|--------------------|------------------------|------|
| 下界篇 1-5 | 4-8 个 | 每卷 1-3 个 | 建立凡人流具体生活和宗门、家族、战争结构 |
| 灵界篇 6-9 | 5-9 个 | 每卷 2-4 个 | 百族、猎血、监察、残界、登仙台都需要代表人物 |
| 仙界篇 10-14 | 6-10 个 | 每卷 2-5 个 | 仙籍、补墟司、司命台、旧天庭和权贵体系要有制度人物 |

命名原则：

1. 只要人物会跨三章以上或承担证据、牺牲、反转、关系功能，就应命名。
2. 一次性工具人物可以只用身份称呼。
3. 有名人物必须有去向，不能用完消失。
4. 同一功能不要堆三个同质人物。

## 7. 名字锁定等级

后续开发中，人物姓名分为三档。

| 等级 | 含义 | 操作 |
|------|------|------|
| 定稿名 | 已进入核心设定，原则上不改 | 赵长生、沈临风、叶疏彤、白倩霜、纪晚凝等 |
| 暂名 | 功能已定，姓名可在卷纲阶段微调 | 多数阶段反派、制度人物、群像人物 |
| 角色位 | 只定功能，不急着命名 | 每卷小角色、一次性交易者、某类群像样本 |

人物体系文档可以先大量使用暂名和角色位。审计时优先审功能，不优先审名字好不好听。

## 8. 人物关系硬规则

1. 长线人物不能只和赵长生发生关系。
2. 沈临风必须有自己的关系网，尤其是谢未明、纪晚凝、新司命台人物。
3. 三条女主线不能互相替代，也不能互相争宠。
4. 同伴之间必须能在专业上互相需要，而不是等主角安排。
5. 反派之间要有上下游关系，不能每卷随机刷一个坏人。
6. 高层理念反派必须能解释为什么底层反派会存在。

## 9. 人物审计口径

后续做全设定审计时，人物部分至少要审以下问题：

| 问题 | 不合格表现 |
|------|------------|
| 人物是否太少 | 后九卷只剩主角升级和揭秘 |
| 人物是否同质 | 每个反派都是贪婪，每个女修都善良，每个仙官都坏 |
| 人物是否断线 | 重要人物登场后长期无功能 |
| 人物是否抢线 | 阶段人物解决了主角该解决的问题 |
| 人物是否降智 | 为推动剧情突然犯蠢、误会、嘴硬 |
| 感情线是否烂俗 | 救一次就爱、争宠、双修升级、女主资源化 |
| 反派是否空泛 | 只说宏大理念，没有具体账本、职位、利益链 |
| 退场是否有效 | 死亡或离队没有改变世界、证据或人物选择 |

## 10. 当前人物系统结论

人物设定的下一步不是继续增加主角光环，而是补齐三类人物：

1. 分卷角色配置：每卷需要什么角色，谁登场，谁退场，谁延续。
2. 三阶段角色池：下界、灵界、仙界分别有哪些阶段人物和制度人物。
3. 弧线退场规则：长线人物怎么成长，阶段人物怎么回收，避免人物堆积。

这些补完后，再进入全设定一致性审计会更稳。
