# 人物设定模板与检查清单

> 本文档用于后续新增、审计和修订人物。所有跨卷人物、重要阶段人物、理念反派、制度人物和感情线人物，都应按本模板补齐核心字段。

---

## 1. 使用范围

必须使用完整模板的人物：

1. 双主角和三条女主线。
2. 长线同伴、长线对手、理念反派。
3. 每卷阶段主反派。
4. 跨卷制度人物。
5. 会死亡但承担关键证据的人物。

可以使用简化模板的人物：

1. 一次性交易者。
2. 不跨卷的普通群像。
3. 只出现一两场的低阶执行者。

不建议命名的人物：

1. 只负责传一句话。
2. 没有独立行动。
3. 没有后续回收价值。
4. 可以用身份称呼替代。

## 4. 新增人物前检查

新增有名人物前，先回答：

1. 这个人物的功能能否由已有角色承担？
2. 如果不能，他承担的是资源、职业、世界观、情感、理念、压迫、证据、后果中的哪一种？
3. 他是否有自己的欲望，而不是只等主角来触发？
4. 他是否有资源需求或职位责任？
5. 他是否会和主角之外的人发生有效关系？
6. 他是否有明确退场或长期职责？
7. 他的存在是否会挤掉已有长线人物的专业性？

如果有三项答不上来，暂不命名。

## 5. 反派检查清单

反派不能只靠恶意成立。

| 检查项 | 合格标准 |
|--------|----------|
| 动机 | 资源、职位、恐惧、理念至少有一项具体 |
| 能力 | 他的境界、职位、工具能解释他为什么有压迫感 |
| 上游 | 他背后是否有制度、市场、宗门、家族或账本 |
| 错误 | 他错在哪里，不只是他坏在哪里 |
| 失败 | 他失败后，主角获得的不只是战利品，还应获得证据、空间或认知 |
| 回声 | 他被击败后，其代表的问题是否仍在 |

禁写反派：

- 随机挑衅主角的恶少。
- 无理由知道主角秘密的追杀者。
- 为了衬托主角而强行降智的高手。
- 只会喊宏大理念没有具体职位和账本的人。

## 6. 同伴检查清单

| 检查项 | 合格标准 |
|--------|----------|
| 专业性 | 有主角不能替代的专业能力 |
| 独立目标 | 离开主角也知道自己要什么 |
| 成长代价 | 变强需要资源、时间、失败和选择 |
| 分歧 | 能在合理处和主角不同意 |
| 去向 | 不随队时也有职责 |
| 边界 | 不替主角解决核心难题 |

同伴不是跟班，也不是工具箱。

## 7. 女主与女性角色检查清单

女性角色先按人物判断，再判断是否进入感情线。

| 检查项 | 合格标准 |
|--------|----------|
| 独立目标 | 她最想要的东西不是赵长生 |
| 道途 | 她有自己的修行方向和终局职责 |
| 社会位置 | 宗门、族群、仙界或底层处境具体 |
| 关系代价 | 感情会带来现实风险，不是免费奖励 |
| 专业性 | 确认关系后能力不下降 |
| 互证 | 她能质疑或补足赵长生道路 |

禁写：

- 一救倾心。
- 争宠雌竞。
- 失身负责。
- 炉鼎升级。
- 女主献本源给主角。
- 女主确认关系后失去道途。

## 8. 制度人物检查清单

仙籍官、账师、监察使、宗门执事、家族管事这类人物要有职位逻辑。

| 检查项 | 合格标准 |
|--------|----------|
| 职责 | 他每天具体处理什么事 |
| 考核 | 上级如何评价他 |
| 利益 | 他靠什么吃饭、升职、避祸 |
| 权限 | 他能调用什么资源或惩罚 |
| 盲区 | 他不知道什么，或故意不看什么 |
| 责任 | 他做错事是否有制度后果 |

制度人物越具体，世界越可信。

## 9. 群像人物检查清单

群像人物不需要完整传记，但要真实。

至少补齐：

1. 他靠什么活。
2. 他当前最怕什么。
3. 他为什么无法轻易离开。
4. 他和主角的一次具体交集。
5. 他代表哪类人的处境。
6. 他是否会留下回声。

例如：

| 群像 | 合格写法 |
|------|----------|
| 仙矿飞升者 | 有役期、债务、伤势、同乡和赎籍目标 |
| 棚区女修 | 有工钱、欠债、保护费和选择困境 |
| 小族族人 | 有祖地、族契、献血压力和族内分歧 |
| 宗门杂役 | 有任务、贡献点、克扣和升外门幻想 |

## 10. 卷纲前人物核对表

每卷正式写卷纲前，核对：

1. 本卷核心人物是谁。
2. 本卷新增人物哪些是定稿名、暂名、角色位。
3. 本卷至少一个普通人或底层群像是谁。
4. 本卷反派代表什么资源链或制度链。
5. 本卷是否推进至少一名长线人物的弧线。
6. 本卷是否有不该提前知道真相的人知道了太多。
7. 本卷结束后，新增有名人物都去了哪里。
8. 本卷人物变化是否需要回写索引或总表。

## 11. 审计前人物核对表

全设定一致性审计前，人物系统必须满足：

1. `02_人物体系总纲与角色层级.md` 已明确人物层级。
2. `03_分卷角色配置表.md` 已覆盖十四卷。
3. `../04_阶段角色池/03_阶段角色池_下界篇.md` 已覆盖第一到第五卷。
4. `../04_阶段角色池/04_阶段角色池_灵界篇.md` 已覆盖第六到第九卷。
5. `../04_阶段角色池/05_阶段角色池_仙界篇.md` 已覆盖第十到第十四卷。
6. `04_人物弧线与退场规则.md` 已规定死亡、退场、背叛和回收。
7. 核心人物与阶段人物没有明显功能冲突。
8. 女主、同伴、反派和制度人物都不只是主角附属。

满足以上条件后，才适合进入全设定一致性审计。
