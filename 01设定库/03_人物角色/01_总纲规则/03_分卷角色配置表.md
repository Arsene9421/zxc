# 分卷角色配置表

> 本表不是章纲，也不是剧情执行表。它只锁定十四卷的人物容量、角色功能、登场退场和关系推进，防止后续写卷纲时人物线断裂或后期空心。

## 2. 下界篇配置

### 第一卷：青石坊市

功能重点：底层生存、废料市场、第一次合作、第一次杀人。

必须有的人物类型：

| 类型 | 功能 |
|------|------|
| 丹铺压迫者 | 让赵长生的日常困境具体化，克扣工钱、废丹去向、杂役契 |
| 黑市中间人 | 打开残器、废丹、破功法交易链 |
| 棚区暴力者 | 逼出第一次反杀，不需要知道仙界真相 |
| 底层女修样本 | 展示女修在资源缝隙中的生存，不进入主恋爱线 |
| 老散修或残影人物 | 提供飞升异常、仙门已断的民间碎片 |

不能做：

1. 不让叶疏彤提前登场抢走第一卷生存线。
2. 不提前解释旧天庭、新仙界、墟眼。
3. 不把第一卷反派写成终局阴谋代理人。

### 第二卷：青云外门

功能重点：宗门制度、生机线、阵法线、正统对照线。

新增核心人物：

| 人物 | 功能 |
|------|------|
| 叶疏彤 | 灵植、生机、低处经营，后续第一条道侣线 |
| 陆维舟 | 阵法、洞府、界阵，解决主角不能全职业包办的问题 |
| 顾长庚 | 正统宗门道路和剑道对照 |

必要阶段人物：

- 外门执事：控制任务、贡献、资源入口。
- 灵田管事：制造灵田、灵植、债务和生机线压力。
- 丹房师兄：让废丹和丹药品质规则进入宗门层面。
- 藏经阁老人：提供残卷和道统残缺感，但不能剧透。

### 第三卷：筑基风波

功能重点：筑基资源、修仙家族、拍卖会、黑市暗庄。

必要人物：

| 类型 | 功能 |
|------|------|
| 修仙家族主敌 | 控制筑基主药、联姻、家族资源垄断 |
| 拍卖暗庄人物 | 证明公开市场也有内幕和账目 |
| 筑基失败者 | 展示突破失败和底层代价 |
| 丹师掮客 | 连接丹药、丹毒、假丹方和资源骗局 |

人物线要求：

1. 叶疏彤和赵长生建立信任，但不急着确认感情。
2. 顾长庚与赵长生竞争筑基资源，但不能变成恶少反派。
3. 沈临风第一次发现系统奖励不干净。

### 第四卷：洞府金丹

功能重点：洞府经营、灵田、阵法、残器库、金丹雏形。

必要人物：

| 类型 | 功能 |
|------|------|
| 散修盟人物 | 证明废山也有地盘利益和灰色秩序 |
| 宗门长老或内门代理 | 觊觎洞府和灵脉，不让种田变安全区 |
| 灵契师或账房 | 让洞府产权、契约、债务具体化 |
| 药圃人手 | 展示叶疏彤不是一个人养完整洞府 |

人物线要求：

1. 陆维舟必须成为洞府安全的关键，不能只是阵盘工具人。
2. 叶疏彤负责灵田和丹材闭环，体现生机线价值。
3. 赵长生开始承担团队资源分配压力。

### 第五卷：断仙门

功能重点：修仙国大战、古战场、天机阁、断仙门真相。

新增核心人物：

| 人物 | 功能 |
|------|------|
| 谢未明 | 命数污染、天机阁、司命台证据链 |

必要阶段人物：

- 天机阁保守派：相信推演权威，拒绝承认天机污染。
- 战争操盘者：把修仙国大战写成资源、国运、宗门利益合谋。
- 古战场残魂：提供旧天庭或飞升异常证据，但不完整解释。
- 底层军修或灾民：让大战不是地图事件，而是具体伤亡。

卷尾要求：

下界篇结束时，读者应知道“飞升有问题”，但还不该完全知道补墟账本真相。

## 3. 灵界篇配置

### 第六卷：灵界底层

功能重点：换地图后仍从底层开始，百族格局初显，猎血制度进入。

新增核心人物：

| 人物 | 功能 |
|------|------|
| 白倩霜 | 真灵血脉、百族责任、血源本源，第二条道侣线 |

必要人物：

- 灵界监察使：不是最终反派，而是仙界制度的第一层代理。
- 猎血商队管事：让白倩霜危机具体化。
- 小族幸存者：证明白倩霜不是单独被追杀，而是族群被压迫。
- 灵界底层人族：证明人族在灵界也不是天然统治者。

### 第七卷：百族与真灵

功能重点：百族战争、真灵血脉、族契和猎血链。

必要人物：

| 类型 | 功能 |
|------|------|
| 大族猎血者 | 展示大族如何把小族当材料 |
| 小族长老 | 提出牺牲白倩霜保族的残酷选择 |
| 羽族侦察者 | 打开百族横向盟友，不只写白倩霜一族 |
| 魂族祭师 | 为后期轮回灯和魂源线埋伏 |
| 百族混血者 | 证明血脉等级制度本身荒谬 |

人物线要求：

1. 白倩霜不能只被赵长生救走，必须面对族群责任。
2. 叶疏彤和白倩霜的冲突应来自生机和族群优先级，不来自争宠。
3. 沈临风必须被百族现实打破系统视角。

### 第八卷：万年抽源

功能重点：抽源制度、残界、灵界据点、仙界监察账目。

必要人物：

- 投靠仙界的大族人物：代表“牺牲弱族换本族豁免”的道路。
- 残界守界人：让界源不是抽象概念。
- 灵界监察账官：让抽源进入账本，而不是单纯战争。
- 弱族代表：证明万年抽源最先压垮小族。

人物线要求：

1. 谢未明应在此卷发现天机阁推演源头被污染。
2. 陆维舟必须通过残界和界阵升级专业功能。
3. 叶疏彤的生机线要和界源修复产生联系。

### 第九卷：登仙台残路

功能重点：渡劫、验源、登仙台、飞升回收。

必要人物：

| 类型 | 功能 |
|------|------|
| 登仙台看守者 | 执行飞升筛选，不一定知道全部真相 |
| 验源仙使投影 | 让仙界制度第一次正式压到灵界 |
| 飞升失败者残魂 | 提供飞升回收证据 |
| 灵界反抗者 | 证明不是只有主角发现问题 |

卷尾要求：

主角组进入仙界不能是荣耀飞升，而是伪装、借道或破局进入底层。

## 4. 仙界篇配置

### 第十卷：仙界底层

功能重点：飞升者不是得道逍遥，而是被登记、验源、服役和征税。

新增核心人物：

| 人物 | 功能 |
|------|------|
| 纪晚凝 | 补墟司仙官，制度执行者和后期清账道侣 |

必要人物：

- 仙籍官：负责登记、临籍、役籍、黑籍压力。
- 仙矿飞升者：展示飞升之后仍是底层。
- 失籍飞升者：展示没有仙籍的生存困境。
- 残仙器坊主：给赵长生仙界捡漏入口。

### 第十一卷：司命台与补墟司

功能重点：系统后台、司命台、补墟司内账、沈临风反向夺权。

必要人物：

| 类型 | 功能 |
|------|------|
| 司命台管理员 | 证明系统任务不是天意，而是后台规则 |
| 补墟账师 | 帮纪晚凝看到明账、内账差异 |
| 系统前宿主或失败棋子 | 让沈临风看到自己的未来可能性 |
| 追捕仙官 | 代表制度自我保护 |

人物线要求：

1. 沈临风不能只砸系统，要理解系统的账目和权限。
2. 纪晚凝不能直接倒戈，只能开始怀疑。
3. 赵长生从躲避仙籍转向寻找账本证据。

### 第十二卷：旧天庭废墟

功能重点：旧天庭不是完美答案，轮回边界限制归墟炉。

新增核心人物：

| 人物 | 功能 |
|------|------|
| 秦照古 | 旧天庭复辟派，尊重但错误的道路 |
| 洛闻霜 | 轮回灯、残魂归处、生死边界 |

必要人物：

- 旧天庭复辟派将领：证明旧制仍有组织和诱惑。
- 守碑残灵：提供旧道统证据，但有残缺视角。
- 旧籍仙官：展示旧道职和新仙籍的冲突。
- 旧天庭失败见证者：承认旧天庭曾救人，也曾失败。

### 第十三卷：墟眼与归墟大潮

功能重点：终局理念反派正面登场，三条道路论战。

新增核心人物：

| 人物 | 功能 |
|------|------|
| 裴玄阙 | 新仙界保全派最高理念反派 |

必要人物：

- 新仙界保全派将领：执行裴玄阙路线。
- 司命台主簿：握有命数黑账。
- 补墟黑账守门人：掌握血账、魂账、界账。
- 仙界权贵代表：证明补墟制度已固化为利益链。

人物线要求：

1. 裴玄阙必须讲得出真实危机，不是疯子。
2. 秦照古和纪晚凝都要被迫面对自己道路的缺口。
3. 白倩霜、叶疏彤、洛闻霜的血源、生机、魂源证据必须参与击穿补墟道路。

### 第十四卷：归墟重炼

功能重点：旧秩序清账，新循环分工，人物职责回收。

必要人物：

| 类型 | 功能 |
|------|------|
| 下界代表 | 证明下界不是抽象牺牲品 |
| 百族代表 | 参与重订百族契约 |
| 飞升者代表 | 参与修改仙籍和飞升规则 |
| 旧账见证者 | 公开补墟旧账 |
| 制度残余 | 证明重炼不是一场战斗就自然完成 |

终局要求：

1. 赵长生不坐仙帝位。
2. 沈临风重定司命盘规则。
3. 叶疏彤、白倩霜、纪晚凝分别承担生机、血源、清账责任。
4. 陆维舟、谢未明、洛闻霜分别承担界阵、命数记录、轮回灯责任。
5. 顾长庚、秦照古、裴玄阙的道路必须有明确结论。

## 5. 使用规则

1. 本表只定人物配置，不替代卷纲。
2. 新增人物时先判断其层级和功能，再决定是否命名。
3. 卷纲阶段如果新增核心人物，必须回写本表和人物索引。
4. 如果某卷没有新的制度人物或群像人物，说明舞台可能不够具体。
5. 如果某卷只有反派没有普通人，说明世界会显得空。
