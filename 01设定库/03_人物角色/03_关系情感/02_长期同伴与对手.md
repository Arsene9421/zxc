# 长期同伴与对手

## 1. 设计原则

赵长生可以有同伴，但同伴不能变成无条件跟班。

长期同伴要满足三点：

1. 有专业能力，能解决主角不擅长的问题。
2. 有独立目标，必要时可以和赵长生发生分歧。
3. 有成长代价，不能靠赵长生给资源就一路升级。

长期对手要满足三点：

1. 不是单纯嫉妒主角。
2. 能代表某条正统道路或制度逻辑。
3. 后期即使失败，也能让主角的选择更清楚。

## 2. 陆维舟

### 2.1 定位

| 项目 | 设定 |
|------|------|
| 首次登场 | 第二卷，青云外门 |
| 身份 | 外门阵修，兼修低阶炼器 |
| 灵根 | 土金火三灵根，资质普通但手稳 |
| 主线 | 阵法、洞府、界阵、登仙台节点 |
| 与赵长生 | 技术合作者，长期同伴 |
| 与沈临风 | 经常被沈临风拉去试系统奖励，负责泼冷水 |

### 2.2 人物功能

陆维舟解决的是“赵长生不能一个人包办所有技术”的问题。

他承担：

- 护山阵。
- 洞府改造。
- 逃亡阵盘。
- 界裂稳定。
- 登仙台残路修补。
- 终局新界阵节点维护。

### 2.3 成长线

| 阶段 | 内容 |
|------|------|
| 第二卷 | 修补宗门废阵盘，和赵长生因废料认识 |
| 第四卷 | 成为洞府护山阵核心设计者 |
| 第五卷 | 在修仙国大战中第一次用阵法救下大量低阶修士 |
| 第八卷 | 接触灵界空间阵，理解界源和地脉不是单纯材料 |
| 第九卷 | 参与登仙台残路修补，正式接入主线大局 |
| 第十四卷 | 参与新登仙台和万界节点阵建设 |

## 3. 顾长庚

### 3.1 定位

| 项目 | 设定 |
|------|------|
| 首次登场 | 第二卷，青云外门 |
| 身份 | 青云门重点培养的剑修天才 |
| 灵根 | 金火双灵根 |
| 主线 | 正统宗门道路、剑道、道心对照 |
| 与赵长生 | 前期竞争和冲突，中期认可，后期并肩但不完全同路 |
| 与沈临风 | 嫌他油滑，后期承认他敢逆天命 |

### 3.2 人物功能

顾长庚不是反派富二代。

他代表一条看似正确的路：天赋不错，入宗门，守规矩，争资源，修正统剑道。

他的存在能反衬赵长生：

| 顾长庚 | 赵长生 |
|--------|------|
| 被宗门看见 | 被系统和宗门忽略 |
| 走正统功法 | 从残卷里补路 |
| 资源不宽裕但有资格竞争 | 连竞争资格都要自己偷出来 |
| 相信宗门规矩 | 先相信活下去 |

### 3.3 成长线

| 阶段 | 内容 |
|------|------|
| 第二卷 | 作为宗门天才登场，压迫感强，但不主动害赵长生 |
| 第三卷 | 因筑基资源和赵长生发生竞争，见识赵长生的狠和稳 |
| 第五卷 | 修仙国大战中发现宗门并不总是正义 |
| 第八卷 | 灵界篇重新登场或通过残界线汇合，剑道从守规矩变成守底线 |
| 第十二卷 | 面对旧天庭斩道剑残痕，开始追问“什么道值得斩” |
| 第十四卷 | 可成为新斩道剑候选执剑者之一，但不能凌驾众生 |

## 4. 谢未明

### 4.1 定位

| 项目 | 设定 |
|------|------|
| 首次登场 | 第五卷，断仙门 |
| 身份 | 天机阁推演修士 |
| 主线 | 命数污染、天机阁、司命台证据链 |
| 与赵长生 | 看不清赵长生命数，因此产生兴趣和戒备 |
| 与沈临风 | 司命台线关键搭档 |

### 4.2 人物功能

谢未明不是负责剧透的算命先生。

他的核心作用是证明：

- 天机推演不是天然真理。
- 下界天机阁看到的“天命”已经被司命台污染。
- 赵长生的命数断页不是气运太强，而是记录层面出了缺口。
- 沈临风的系统任务可以被反向追踪。

### 4.3 成长线

| 阶段 | 内容 |
|------|------|
| 第五卷 | 推演赵长生失败，确认断仙门不只是传说 |
| 第七卷 | 发现系统和司命盘残录之间的冲突 |
| 第八卷 | 在万年抽源前后发现天机阁推演源头被污染 |
| 第九卷 | 为登仙台残路提供证据，证明飞升是筛选回收 |
| 第十一卷 | 与沈临风反向追踪新司命台后台 |
| 第十四卷 | 参与新司命盘规则：只记录危机，不安排命运 |

## 5. 秦照古

### 5.1 定位

| 项目 | 设定 |
|------|------|
| 首次登场 | 第十二卷，旧天庭废墟 |
| 身份 | 旧天庭遗民，复辟派代表 |
| 主线 | 旧天庭荣耀、旧制失败、复辟诱惑 |
| 与赵长生 | 尊重赵长生，但要求他恢复旧天庭秩序 |
| 与沈临风 | 把沈临风视为司命台残火选中的变量 |

### 5.2 人物功能

秦照古用来防止故事把答案写得太简单。

他不是坏人。

他可以拿出旧天庭真实做过的好事：

- 接引飞升者，不回收飞升者。
- 维护轮回。
- 修补下界裂缝。
- 开放部分道统。
- 在第一次归墟大潮中确实牺牲过大量仙官。

但他的问题是：他只记得旧天庭最好的一面，不愿承认旧天庭也曾僵化、超载并失败。

### 5.3 与赵长生的冲突

| 秦照古 | 赵长生 |
|--------|------|
| 认为旧天庭只要恢复就能解决问题 | 认为照旧恢复会重复失败 |
| 要求九器归位旧职 | 要求九器重组新循环 |
| 相信旧制度还能修 | 相信残缺制度必须重炼 |

## 6. 仙界底层飞升者群像

仙界篇不能只写仙官和权贵，也要写被飞升体系吞掉的人。

建议先准备三类群像，具体姓名后续仙界篇再定：

| 类型 | 作用 |
|------|------|
| 矿场飞升者 | 证明飞升不是奖励，而是服役和纳税 |
| 仙兵飞升者 | 证明强者会被编入镇压体系 |
| 失籍飞升者 | 证明没有仙籍的飞升者在仙界几乎无法生存 |

其中至少一人应与赵长生形成短期同盟，但不要过早扩成长线队伍，避免仙界篇人物失控。
