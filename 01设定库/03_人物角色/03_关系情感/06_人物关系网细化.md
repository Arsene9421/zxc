# 人物关系网细化

> 本文档补“横向关系”。单人档回答一个人是谁，本文档回答这些人之间如何互相牵动、互相限制、互相推动。

---

## 1. 关系类型

本书人物关系不能只分朋友、敌人、道侣。至少有七类：

| 关系类型 | 含义 | 主要人物 |
|----------|------|----------|
| 生死同谋 | 共同承担杀人、逃亡、系统异常等不可公开的风险 | 赵长生、沈临风 |
| 道侣互证 | 情感、道途、资源和风险部分绑定 | 赵长生与叶疏彤、白倩霜、纪晚凝 |
| 专业互补 | 每个人保留不可替代的专业线 | 叶疏彤、陆维舟、谢未明、纪晚凝 |
| 正统对照 | 不一定敌对，但长期质疑主角道路 | 顾长庚、秦照古 |
| 证据盟友 | 以账本、残魂、命数、族契等证据连接 | 谢未明、纪晚凝、洛闻霜、白倩霜 |
| 制度对手 | 不是私人仇恨，而是位置和职责冲突 | 纪晚凝、仙籍官、补墟司、司命台人物 |
| 理念敌人 | 能提出一条看似合理但必须被击穿的道路 | 秦照古、裴玄阙 |

关系设计原则：

1. 亲近关系也要有冲突。
2. 敌对关系也要有合理性。
3. 道侣关系不能吞掉人物专业性。
4. 同伴之间必须有横向关系，不能都只和赵长生连接。
5. 后期证据必须通过多人互证，不能只靠赵长生发现真相。

## 3. 三条女主关系网

三位女主不是互相排位，而是三条世界线的代表。

| 关系 | 冲突议题 | 不能靠什么解决 | 成熟合作 |
|------|----------|----------------|----------|
| 叶疏彤、白倩霜 | 生机修复和血源自主的优先级 | 不能靠赵长生调停争宠 | 叶疏彤修复祖地生机，白倩霜保留族契自主 |
| 叶疏彤、纪晚凝 | 下界生机旧债和补墟账本 | 不能靠纪晚凝“洗白” | 纪晚凝公开旧账，叶疏彤提供生机损伤证据 |
| 白倩霜、纪晚凝 | 血税和猎血制度 | 不能靠私人道歉 | 纪晚凝承认制度罪责，白倩霜参与重订百族契约 |
| 三人共同 | 新循环里生机、血源、账本如何并行 | 不能设正宫，不做后宫排位 | 分可独行，合可破局 |

三女主共同在场时，核心话题不能是赵长生爱谁更多，而是：

1. 生机如何落地。
2. 血源如何归族。
3. 旧账如何公开。
4. 新规则如何防止再次把人变成资源。

## 4. 关系阶段

| 阶段 | 关系重点 | 标志 |
|------|----------|------|
| 第一阶段：互相有用 | 低信任合作 | 共同得利，但各留后手 |
| 第二阶段：共同担险 | 生死人情 | 逃亡、护法、伤势、秘密开始绑定 |
| 第三阶段：价值冲突 | 关系开始变重 | 对方的选择会影响自己的道途 |
| 第四阶段：道途互证 | 不再只是同行 | 能当面质疑，也能共同承担代价 |
| 第五阶段：终局分工 | 各有职责 | 不围着赵长生转，各自承担新循环职责 |

重要关系必须至少经过前三阶段，不能直接跳到生死相托。

## 5. 关系破裂与修复规则

可写破裂：

- 赵长生隐瞒归墟炉风险，导致同伴被卷入。
- 沈临风因系统任务做出错误判断。
- 白倩霜因族群责任拒绝随队行动。
- 纪晚凝前期执行制度，造成主角阵营损失。
- 顾长庚坚持宗门规矩，与赵长生残缺路冲突。

修复必须满足：

1. 当事人承认具体错误。
2. 付出实际代价。
3. 关系规则发生改变。
4. 后续行动证明不是口头和解。

禁用修复方式：

- 一句道歉就翻篇。
- 靠赵长生个人魅力压过分歧。
- 靠误会解除解决结构性冲突。
- 靠救命强行抵消所有旧账。
