# 主要人物出场与在场调度表

> 本文档解决“谁什么时候出场、谁在哪些卷真正参与主线、谁后台留守”。它不是章纲，只是人物调度表。

---

## 1. 调度标记

| 标记 | 含义 |
|------|------|
| 主轴 | 本卷核心推动人物 |
| 在场 | 参与主要行动，但不是本卷第一推动 |
| 分线 | 不与主角同队，但有独立线 |
| 后台 | 通过消息、据点、账本、旧事存在 |
| 回归 | 久别后重新进入主线 |
| 未登场 | 本卷不使用 |

调度原则：

1. 每卷主轴人物不宜超过 3 到 5 人。
2. 不是所有核心人物都要一直随队。
3. 分线人物必须有自己的目标，不能只是离队等回归。
4. 后台人物要有回声，否则就是断线。

## 5. 使用规则

1. 卷纲阶段新增人物时，先看本表是否容量过满。
2. 若某核心人物连续三卷只有后台状态，必须给出合理任务或回归节点。
3. 道侣不等于必须随队，分线更能保留人物独立性。
4. 后期全员汇合时，要按证据线分工，不要写成所有人围观赵长生。
