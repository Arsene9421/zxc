# 家族宗门派系与人物归属

> 本文档补人物背后的社会归属。人物不是凭空出现的，他们来自家族、宗门、族群、机构、派系和账本系统。归属决定资源、压力、立场和可被利用点。

---

## 1. 归属设计原则

1. 主角团可以反制度，但不能没有社会来处。
2. 家族、宗门、派系不是背景板，而是资源和压力来源。
3. 小人物也要有生计归属：丹铺、灵田、阵坊、天机阁、补墟司、百族祖地。
4. 人物叛离旧归属后，旧债不会自动消失。
5. 派系内部不能铁板一块，必须有保守派、利益派、改革派、求活派。

## 3. 下界归属系统

### 3.1 青石坊市

| 势力/场所 | 相关人物 | 功能 |
|-----------|----------|------|
| 回春丹铺 | 赵长生、周槐安、吴小满 | 废丹、炉灰、杂役契、底层克扣 |
| 上宗废料链 | 蒋怀礼 | 把坊市废料和宗门资源回收连接起来 |
| 黑市巷 | 裴九、罗三缺 | 残器、破玉简、伪古物、消息泄露 |
| 棚区帮派 | 胡三奎、方阿照 | 底层暴力、保护费、第一次杀人压力 |
| 灵米铺 | 宁晚秋 | 灵米价格、底层女修生存、欠债 |

青石坊市的核心不是“大势力”，而是小账：

- 工钱。
- 灵米。
- 废丹。
- 废料账。
- 保护费。
- 杂役契。

青云门内部必须有层次：

1. 掌门和长老看宗门存续。
2. 执事看任务和贡献。
3. 外门看资格。
4. 杂役看活路。
5. 灵田和丹房看产量。

不同位置的人都可以不坏，但制度会自然制造压迫。

## 6. 派系冲突

| 冲突 | 双方 | 人物作用 |
|------|------|----------|
| 底层资源 vs 宗门秩序 | 赵长生、青云门执事线 | 证明规矩不天然公平 |
| 灵农家族 vs 宗门产量 | 叶疏彤、灵田管事 | 生机也被任务化 |
| 正统剑堂 vs 残缺道路 | 顾长庚、赵长生 | 两条修行路互证 |
| 天机阁权威 vs 命数污染 | 谢未明、闻人策 | 推演不等于真理 |
| 百族自保 vs 个体自主 | 白倩霜、霜羽族长老 | 族群不能吞掉个人 |
| 灵界大族 vs 小族 | 玄鳞族、霜羽族 | 大族借仙界监察压小族 |
| 补墟司总账 vs 具体死账 | 纪晚凝、飞升者群像 | 制度必须为结果负责 |
| 新司命台 vs 自主命数 | 沈临风、季玄屏 | 天命系统不是天道 |
| 旧天庭复辟 vs 新循环 | 秦照古、赵长生 | 尊重旧牺牲不等于复辟 |
| 新仙界保全 vs 万界参与 | 裴玄阙、全阵营 | 危机真实不等于收割正当 |
