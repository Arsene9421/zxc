# 阶段角色池：仙界篇

> 本文档补齐第十卷到第十四卷的仙界人物。仙界篇不能只写更高级的宗门和更强的敌人，而要通过仙籍、仙税、仙矿、补墟司、新司命台、旧天庭废墟和中天玄阙，展示制度化压迫如何运行。

---

## 1. 仙界篇人物功能

仙界篇人物必须承担五类功能：

1. 飞升者群像：证明飞升不是奖励，而是进入登记和税役体系。
2. 制度执行者：让仙籍、道职、补墟账本有具体执行人。
3. 证据人物：提供明账、内账、黑账、旧账。
4. 旧天庭人物：证明旧天庭曾救人，也曾失败，不能简单复辟。
5. 终局理念人物：让新仙界保全派的道路有真实压迫感。

纪晚凝的转变必须来自账本证据、制度责任和现实后果，不能来自恋爱冲动。

### 2.3 第十卷人物边界

1. 飞升者不能都立刻服主角，要各有生计和恐惧。
2. 许守仓不是大反派，他只是仙籍制度最低层执行者。
3. 纪晚凝登场时要冷静、专业、可信，不要写成冰山女官模板。
4. 赵长生进入仙界必须继续低调求活，不直接挑战中天玄阙。

3. 严慎如果死亡，必须留下账目证据，不要只做催泪。
4. 纪晚凝在此卷只能动摇，不能完成转向。

3. 旧天庭废墟要同时展示伟大和失败，不能让读者误以为复辟就是答案。
4. 顾长庚在此卷应面对斩道剑或万法碑残痕，推进正统道路选择。

4. 藏烬君必须让血账、魂账、界账三类证据具体出现。

第十四卷不适合大量新增高魅力人物。终局重点是回收长线人物和制度责任。

### 6.1 终局代表人物位

| 百族代表 | 参与重订百族契约，承认小族也有发声权，映射见 `../06_二级人物补档/09_终局代表人物映射表.md` |

2. 不让赵长生和道侣们成为新仙界皇族。
3. 不让人物职责被一句“天下太平”抹掉。
4. 制度修改要通过代表人物和公开账本体现。

## 8. 仙界篇审计要点

后续审计仙界篇人物时，重点检查：

1. 飞升者群像是否足够具体，还是只写主角继续捡漏。
2. 纪晚凝是否因账本和责任转变，而不是恋爱转变。
3. 司命台人物是否让沈临风主线独立成立。
4. 旧天庭人物是否同时展示旧制价值和旧制失败。
5. 裴玄阙是否有真实危机支撑，而不是空喊牺牲。
6. 终局人物职责是否并行，不形成新后宫、新天庭、新仙帝。
