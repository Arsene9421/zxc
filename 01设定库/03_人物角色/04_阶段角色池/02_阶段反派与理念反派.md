# 阶段反派与理念反派

## 1. 反派设计原则

反派分三层：

1. 底层资源反派：为了灵石、丹药、功法、地盘杀人。
2. 中层制度反派：不是每个人都知道真相，但他们靠制度吃饭。
3. 高层理念反派：知道代价，也认为代价值得。

前期反派越具体越爽，后期反派越合理越有压迫感。

## 4. 中层反派的逻辑

中层反派通常不知道完整真相。

他们的共同逻辑是：

- 上面要求收废料，他们就收。
- 上面要求筛选飞升者，他们就筛选。
- 上面给他们资源和职位，他们就维护制度。
- 他们不一定认为自己邪恶，只认为底层本来就该付代价。

这类反派适合制造持续压迫，但不能都写成傻子。

## 5. 纪晚凝的灰色位置

纪晚凝属于制度执行者，但不是阶段反派。

她前期站在赵长生对面，因为她维护补墟秩序。

她后期有转向空间，因为她有三个底线：

1. 她愿意看真实账本。
2. 她承认制度必须为结果负责。
3. 她不愿意为了权贵私利继续伪造补墟必要性。

她不能因为私人感情倒戈，必须被证据和现实推翻。

## 6. 秦照古的理念冲突

秦照古不是终局反派，而是旧天庭道路的代表。

他给赵长生的诱惑是：

> 你拿着归墟炉，沈临风拿着司命残火，我们可以恢复旧天庭。只要旧职归位，万界还有救。

赵长生必须拒绝。

拒绝原因：

1. 旧天庭当年已经失败。
2. 九器旧职能不足以应对第二次归墟大潮。
3. 复辟会重新制造高高在上的天庭。
4. 新循环必须允许万界自我生长，而不是继续由仙界或天庭代替众生安排命运。

## 7. 裴玄阙

### 7.1 定位

| 项目 | 设定 |
|------|------|
| 首次正式登场 | 第十三卷 |
| 身份 | 新仙界保全派最高代表之一，中天玄阙掌权者 |
| 主线 | 墟眼、补墟体系、牺牲万界保仙界 |
| 与赵长生 | 终局理念反派 |
| 与沈临风 | 天命系统和新司命台的最高受益者之一 |

### 7.2 他为什么有压迫感

裴玄阙不靠疯癫和恶趣味制造压迫。

他真正可怕的地方是：他能拿出一部分真实理由。

- 墟眼确实存在。
- 第二次归墟大潮确实逼近。
- 仙界一旦塌陷，万界循环可能失序。
- 旧天庭当年确实失败过。
- 低阶世界单独看很惨，但在他账上只是延缓大崩溃的成本。

他的错误不在于“看不见危机”，而在于：

1. 把临时牺牲变成永久制度。
2. 把万界众生从参与者降成燃料。
3. 隐瞒账本，让权贵借补墟名义续命。
4. 明知制度腐烂，仍然认为只要仙界核心还在，一切都值得。

### 7.3 与赵长生的终局对照

| 裴玄阙 | 赵长生 |
|--------|------|
| 保住仙界核心，再谈万界 | 不重炼仙界，万界没有新路 |
| 牺牲低层世界延缓墟眼 | 把墟眼重炼成循环炉口 |
| 天命可以被安排 | 命数只能记录危机，不能垄断众生 |
| 秩序比个体重要 | 秩序若靠吞噬个体续命，就该被重炼 |

### 7.4 失败方式

裴玄阙不能被简单打死了事。

更好的失败方式是三重证据击穿他的道路：

1. 司命台账：证明系统培养棋子不是救万界，而是回收变量。
2. 补墟账：证明抽源效率越来越低，权贵消耗越来越高。
3. 轮回账：证明大量亡魂没有进入循环，而是被制度磨成资源。

最后赵长生不是证明“危机不存在”，而是证明“裴玄阙的办法已经无法解决危机，只是在延缓旧秩序死亡”。
