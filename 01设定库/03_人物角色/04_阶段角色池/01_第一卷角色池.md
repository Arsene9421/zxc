# 第一卷角色池

## 1. 角色设计原则

第一卷反派不需要知道仙界真相。

他们只需要被具体资源利益驱动。

敌人越具体，赵长生第一次反杀越爽。

## 2. 反派池

| 角色 | 表层动机 | 背景作用 |
|------|----------|----------|
| 丹铺掌柜 | 克扣工钱，压榨杂役，私吞废丹 | 表现底层资源紧张 |
| 坊市散修 | 盯上赵长生卖出的异常灵草或丹药 | 表现修士为资源可以杀人 |
| 上宗外门管事 | 负责收废料、查账、拿回扣 | 埋废料上缴体系 |
| 黑市牙人 | 倒卖残器、灾民、低阶功法 | 展示残缺资源市场 |
| 小帮派头目 | 控制贫民区和废料堆 | 给第一次杀人提供压力 |

## 3. 配角池

| 角色 | 用途 |
|------|------|
| 老散修 | 提供飞升异常传闻 |
| 灵米铺老板 | 展示底层灵食涨价 |
| 罗三缺（旧货摊主） | 出售残器、破玉简、伪古物；记住赵长生挑残物和可能有回灵残器 |
| 阿七（棚区少年） | 棚区跑腿少年，鼻子灵，第 2 章闻到米香；只作底层视线和消息风险，不发展成温情支线 |
| 丹铺小伙计 | 展示赵长生在杂役中的位置 |

## 4. 后台角色

| 角色 | 用途 |
|------|------|
| 无名老修 | 青槐旧灾救下赵长生，前期只留残影 |
| 上宗账房 | 废料上缴制度的中间层 |
| 系统后台 | 通过沈临风系统异常表现 |

## 5. 地图联动

第一卷卷尾逃亡与第一次杀人地点，按 `01设定库/02_地理势力/02_地图地理/03_第一卷逃亡路线.md` 执行。

小帮派头目最适合在废料场边缘、废灰坑或通往北小门的破木车棚附近形成直接压力。这里能让赵长生依靠熟悉地形、炉灰、残器和沈临风一次信息提示完成反杀，同时保留受伤、物证、追捕和心理后果。
