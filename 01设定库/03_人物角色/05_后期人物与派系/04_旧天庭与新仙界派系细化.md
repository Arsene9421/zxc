# 旧天庭与新仙界派系细化

> 本文档补后期人物背后的派系系统。目标是让秦照古、洛闻霜、裴玄阙、纪晚凝等人不是孤立角色，而是各自带着制度、资源、旧债和组织压力进入剧情。

---

## 1. 设计原则

1. 旧天庭不能等于完美正义，新仙界也不能只等于坏人聚集。
2. 派系必须有资源、权柄、恐惧和现实利益，不能只有口号。
3. 同一派系内部要有分歧，不能铁板一块。
4. 后期冲突不能只靠高境界打斗，要靠账本、道职、仙籍、轮回、司命、界阵等制度证据推进。
5. 终局不是旧天庭复辟，也不是新仙界换皇帝，而是重炼循环。

## 2. 旧天庭残余派系

旧天庭已经崩塌，后期出现的是残余派系、旧职残影、旧籍修士和遗址权柄。

### 2.1 循环派残脉

| 项目 | 内容 |
|------|------|
| 核心主张 | 万界都在循环内，不能为了仙界核心永久牺牲下层世界 |
| 历史位置 | 第一次归墟大潮时主张全力修补万界 |
| 现状 | 大量战死，只剩残魂、旧账、少量旧籍和残职传人 |
| 代表人物 | 洛闻霜部分继承此线，守碑残灵、部分旧账见证者 |
| 资源 | 旧账、轮回灯残焰、万法碑残片、补天旧记录 |
| 问题 | 牺牲巨大但失败过，无法证明原方案仍可应对第二次大潮 |

循环派提供“旧天庭曾经救人”的证据，但不能提供完整终局答案。

### 2.2 复辟派

| 项目 | 内容 |
|------|------|
| 核心主张 | 九器归旧位，旧职复原，重立天庭秩序 |
| 历史情绪 | 认为新仙界是保全派背叛后的伪秩序 |
| 代表人物 | 秦照古、穆天衡（见 `../06_二级人物补档/05_穆天衡.md`） |
| 资源 | 旧籍、镇界残印、旧天庭军序、旧道职名分 |
| 优点 | 仍记得公共职责，不把众生当燃料 |
| 问题 | 把旧职责和旧权力混同，容易重新制造高位垄断 |

复辟派必须有可敬之处，否则秦照古没有重量；也必须有结构性问题，否则终局会滑向恢复旧天庭。

### 2.3 守灯残职

| 项目 | 内容 |
|------|------|
| 核心主张 | 轮回归处不可私有，亡魂不能被任何制度当材料 |
| 代表人物 | 洛闻霜、灯奴无名 |
| 资源 | 轮回灯残焰、魂账旧规、守灯誓约 |
| 冲突对象 | 新仙界补墟魂账、邪魂修、复辟派激进者 |
| 问题 | 旧灯残缺，只守边界不足以主动清算黑账 |

守灯残职不等同复辟派。洛闻霜可以尊重秦照古，但不会为了复辟旧天庭而交出轮回灯。

### 2.4 道统保存者

| 项目 | 内容 |
|------|------|
| 核心主张 | 保存万法碑、斩道剑、旧道统和各界修行道路 |
| 代表人物 | 守碑残灵、顾长庚旧天庭线关联人物 |
| 资源 | 万法碑残片、斩道剑残痕、旧功法和道统记录 |
| 优点 | 证明旧天庭曾保存并开放部分道统 |
| 问题 | 也曾以“保护”为名封存危险道统，留下权柄垄断隐患 |

这条线主要服务顾长庚：正统不是永远正确，斩道也不是谁强谁斩。

### 2.5 旧籍苟活者

| 项目 | 内容 |
|------|------|
| 核心主张 | 不一定有主张，先活过新仙界追索 |
| 代表人物 | 温隙（见 `../06_二级人物补档/06_温隙.md`）等旧籍仙官 |
| 资源 | 旧籍身份、旧账片段、遗址出入经验 |
| 作用 | 证明旧天庭崩塌后，遗民不全是英雄或复辟者 |
| 问题 | 为求活可能出卖旧账、投靠新仙界或隐瞒真相 |

旧籍苟活者能提供灰度，避免旧天庭线只有英雄和叛徒。

### 2.6 旧保全派残影

| 项目 | 内容 |
|------|------|
| 核心主张 | 当年切断部分世界，是为了保住仙界火种 |
| 历史后果 | 改写镇界印和登仙台阵枢，导致循环断裂和墟眼形成 |
| 现状 | 主体演变为新仙界保全派，也可能留有旧时期记录 |
| 作用 | 让裴玄阙路线有历史根源，不是凭空出现 |
| 问题 | 初始也许有自救理由，但后来变成永久收割制度 |

## 3. 新仙界主要派系

### 3.1 保全派

| 项目 | 内容 |
|------|------|
| 核心主张 | 先保仙界核心，再谈万界 |
| 最高代表 | 裴玄阙 |
| 权力中心 | 中天玄阙 |
| 资源 | 墟眼封禁、补墟大阵、仙界核心道职、保全派法旨 |
| 真实理由 | 墟眼和第二次归墟大潮确实存在 |
| 根本错误 | 把临时牺牲制度化，把万界降格为燃料 |

保全派内部可分三层：

| 层级 | 人物类型 | 逻辑 |
|------|----------|------|
| 核心决策层 | 裴玄阙、玄阙高层 | 明知全局账本，也坚持封账保核心 |
| 执行层 | 厉沉霄（见 `../06_二级人物补档/07_厉沉霄.md`）、登仙监高官、补墟司高层 | 相信命令和秩序，未必掌握全部黑账 |
| 受益层 | 玄阙仙族、资源权贵 | 以补墟名义延寿、续族、垄断道统 |

### 3.2 补墟司执行线

| 项目 | 内容 |
|------|------|
| 表层职责 | 统计资源、分配补墟额度、维持墟眼封禁 |
| 相关人物 | 纪晚凝、严慎（见 `../06_二级人物补档/03_严慎.md`）、傅照灯（见 `../06_二级人物补档/08_傅照灯.md`）、藏烬君（见 `../06_二级人物补档/04_藏烬君.md`） |
| 资源 | 明账、内账、黑账、旧账接口、道职印信 |
| 内部分歧 | 清账派、平账派、封账派、黑账守门人 |
| 叙事作用 | 让纪晚凝转变有制度过程 |

补墟司不能只写成坏官机构。它确实维持过一部分仙界稳定，但稳定方式已经被黑账和权贵消耗拖入死循环。

### 3.3 新司命台

| 项目 | 内容 |
|------|------|
| 表层职责 | 观测天命、扶持变量、管理系统任务 |
| 实际功能 | 筛选、培养、回收有价值命数和气运变量 |
| 相关人物 | 沈临风、季玄屏（见 `../06_二级人物补档/02_季玄屏.md`）、钟无咎、谢未明 |
| 资源 | 系统后台、命数黑账、任务模板、气运评级 |
| 叙事作用 | 支撑沈临风从系统受益者变成规则反叛者 |

新司命台的核心罪不是“推演命数”，而是把推演变成安排命运和回收变量。

### 3.4 登仙监与仙籍体系

| 项目 | 内容 |
|------|------|
| 表层职责 | 接引飞升者、验源、录入仙籍、分流 |
| 实际功能 | 把飞升者纳入税役、道途、本源和命数接口 |
| 相关人物 | 许守仓、验源仙使投影、洗尘使 |
| 资源 | 仙籍、洗尘池、验源阵、罪籍和黑籍追索 |
| 叙事作用 | 证明飞升不是奖励，而是进入登记体系 |

这一派系适合制造仙界底层压迫，不宜承担终局最高理念。

### 3.5 镇界军

| 项目 | 内容 |
|------|------|
| 表层职责 | 镇压界裂、守卫仙界边境、处理归墟污染 |
| 实际功能 | 镇压拒税世界、护送抽源、清理叛籍飞升者 |
| 相关人物 | 钟离砺、厉沉霄（见 `../06_二级人物补档/07_厉沉霄.md`） |
| 资源 | 军籍、军印、镇界兵阵、飞升者征调 |
| 内部分歧 | 有人真信守界，有人只服军令，有人借战功换命 |

镇界军让“保全派不是只有账本，也有武力”具体化。

### 3.6 玄阙仙族与权贵耗材派

| 项目 | 内容 |
|------|------|
| 核心主张 | 补墟可以继续，前提是自身族群和道统优先保全 |
| 相关人物 | 玄阙仙族代表、藏烬君（见 `../06_二级人物补档/04_藏烬君.md`）背后的利益链 |
| 资源 | 高阶洞府、仙脉份额、道统封锁、补墟豁免权 |
| 叙事作用 | 证明制度已经被利益寄生 |
| 问题 | 他们不一定关心墟眼，只关心补墟体系带来的特权 |

这一派比裴玄阙更庸俗，但不能取代裴玄阙的理念压迫。裴玄阙代表道路错误，权贵耗材派代表制度腐烂。

### 3.7 制度修补派

| 项目 | 内容 |
|------|------|
| 核心主张 | 承认补墟必要，但反对黑账、过度抽源和权贵滥用 |
| 相关人物 | 傅照灯（见 `../06_二级人物补档/08_傅照灯.md`）、部分补墟司中层、部分镇界军 |
| 资源 | 内部证据、执行经验、有限账权 |
| 叙事作用 | 让新仙界内部不是全员反派 |
| 问题 | 如果没有赵长生的新循环方案，他们只能小修小补 |

这条线可以给纪晚凝转向提供同僚压力和现实阻力。

### 3.8 仙界底层与黑籍网络

| 项目 | 内容 |
|------|------|
| 核心诉求 | 活过役期、摆脱追索、保住修行路 |
| 相关人物 | 周栖石、梅见微、陶铸、失籍飞升者群像 |
| 资源 | 黑市伪籍、残仙器、矿区消息、底层互助 |
| 叙事作用 | 让仙界篇继续保留凡人流底层质感 |
| 问题 | 底层不天然高尚，也可能为了活命出卖他人 |

## 4. 关键派系冲突

| 冲突 | 双方 | 代表人物 | 剧情用途 |
|------|------|----------|----------|
| 旧复辟 vs 新循环 | 旧天庭复辟派 vs 赵长生路线 | 秦照古、赵长生 | 拒绝恢复旧天庭 |
| 旧守灯 vs 新魂账 | 守灯残职 vs 补墟黑账 | 洛闻霜、藏烬君、纪晚凝 | 证明亡魂不可入账为材料 |
| 新保全 vs 万界参与 | 保全派 vs 主角组和各界代表 | 裴玄阙、赵长生 | 终局理念冲突 |
| 清账 vs 封账 | 补墟司内部 | 纪晚凝、严慎、傅照灯、藏烬君 | 推动纪晚凝转向 |
| 命数记录 vs 命运安排 | 新司命台 vs 沈临风、谢未明 | 季玄屏、钟无咎 | 系统线收束 |
| 正统道职 vs 道职滥权 | 道统保存者、顾长庚 vs 新仙界道职权力 | 顾长庚、守碑残灵 | 剑道和斩道剑线 |
| 真实危机 vs 特权续命 | 裴玄阙理念层 vs 玄阙权贵层 | 裴玄阙、玄阙仙族 | 区分道路错误和利益腐烂 |

## 5. 人物归属与压力

| 人物 | 派系归属 | 压力来源 | 可被击穿点 |
|------|----------|----------|------------|
| 秦照古 | 旧天庭复辟派 | 旧天庭牺牲、复辟派期待、旧职崩散 | 旧制无法防止权柄再次垄断 |
| 洛闻霜 | 守灯残职 | 轮回灯残缺、亡魂黑账、新仙界追索 | 单靠守边界不能清算黑账 |
| 裴玄阙 | 新仙界保全派 | 墟眼扩大、第二次大潮、玄阙权力 | 补墟体系已无法真正解决危机 |
| 纪晚凝 | 补墟司执行线 | 道职职责、账本误差、上级法旨 | 明账、内账、黑账、旧账逐层对不上 |
| 沈临风 | 新司命台变量 | 系统任务、命数外壳、回收风险 | 系统奖励对应命数黑账 |
| 谢未明 | 天机阁转司命证据线 | 推演失效、命数污染 | 天机不是天道，只是被污染的信息层 |
| 顾长庚 | 正统剑道和旧道统线 | 宗门正统、斩道剑诱惑 | 正统若不清算权力，也会变成压迫 |

## 6. 证据链如何汇合

后期不能靠一个秘密解释全部真相。证据要分线汇合。

| 证据线 | 起点 | 中段推进 | 终局用途 |
|--------|------|----------|----------|
| 补墟账 | 纪晚凝发现明账和内账误差 | 查黑账、对照旧账 | 证明抽源制度已经死循环 |
| 司命账 | 沈临风系统任务异常 | 谢未明追踪命数污染 | 重启司命盘，只记录危机 |
| 魂账 | 洛闻霜照出亡魂断点 | 查被截留亡魂和炼魂坏账 | 重启轮回灯，防止亡魂入补墟 |
| 血账 | 白倩霜和百族源血线 | 揭开族契、猎血、血税 | 重订百族契约 |
| 界账 | 陆维舟和界阵线 | 证明界源被抽导致世界病灶 | 建新登仙台和万界节点阵 |
| 旧账 | 秦照古、温隙、旧天庭遗址 | 证明新仙界篡改旧职责 | 尊重旧牺牲但不复辟旧权力 |

只有这些证据汇合，裴玄阙的“保全必要性”才会被真正击穿。

## 7. 卷际使用

| 卷 | 派系使用重点 |
|----|--------------|
| 第十卷 | 登仙监、仙籍体系、仙矿府、黑籍网络，展示仙界底层压迫 |
| 第十一卷 | 补墟司执行线和新司命台，推动纪晚凝与沈临风各自制度线 |
| 第十二卷 | 旧天庭复辟派、守灯残职、道统保存者，展示旧制价值和失败 |
| 第十三卷 | 保全派、镇界军、补墟黑账守门人、玄阙权贵，形成终局压迫 |
| 第十四卷 | 各派系代表进入清账、重炼、公开旧账和新规则确立 |

## 8. 写作硬规则

1. 秦照古代表旧复辟诱惑，但不能代表全体旧天庭。
2. 洛闻霜不归秦照古管，她的轮回边界高于复辟需要。
3. 裴玄阙必须有真实危机论据，但不能因此无罪。
4. 纪晚凝不能因为私人感情转向，必须被账本和责任推动。
5. 新仙界底层执行者不必都知道黑账，制度压迫可以由普通人执行。
6. 权贵耗材派可以恶，但不能让他们替裴玄阙承担全部罪责。
7. 终局不能让赵长生继承旧天庭或中天玄阙的垄断权柄。
