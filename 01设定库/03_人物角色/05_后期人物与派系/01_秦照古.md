# 秦照古

> 后期长线理念对手。秦照古代表旧天庭最值得尊重、也最容易诱人复辟的一面。他不是反派恶人，但他的答案不能成为终局答案。

---

## 1. 基本定位

| 项目 | 设定 |
|------|------|
| 姓名 | 秦照古 |
| 首次登场 | 第十二卷：旧天庭废墟 |
| 身份 | 旧天庭遗民，复辟派代表，旧镇界道职残存者 |
| 所属派系 | 旧天庭复辟派 |
| 长线功能 | 旧天庭荣耀、旧制失败、复辟诱惑 |
| 与赵长生 | 尊重他持有归墟炉，但要求他恢复旧天庭秩序 |
| 与沈临风 | 把他视为旧司命台残火带回来的变量 |
| 叙事属性 | 可敬的理念对手，不是终局反派 |

“照古”不是沉迷过去，而是照见古制。他看得见旧天庭曾经做过的正确之事，也因为看得太清楚，反而难以承认旧制已经无法原样回到万界之上。

## 2. 核心命题

秦照古要回答的问题：

> 尊重旧天庭的牺牲，是否等于应该复辟旧天庭的权力？

他的答案：

> 旧天庭曾经维持轮回、接引飞升、修补破界。新仙界把万界当燃料，是因为旧职崩散、九器离位。只要旧天庭重立，万界循环仍有救。

赵长生必须否定的不是旧天庭的牺牲，而是“旧权力天然有资格重新安排万界”的逻辑。

## 3. 人物内核

| 维度 | 内容 |
|------|------|
| 核心欲望 | 让旧天庭旧职归位，恢复万界循环秩序 |
| 核心恐惧 | 承认旧天庭不是被外敌打败，而是旧制度本身也已承载过限 |
| 最大优点 | 记得旧职责，不把众生视作账面耗材 |
| 最大缺点 | 把“旧职责的正当性”和“旧权力的正当性”混在一起 |
| 底线 | 不接受新仙界把下界、百族、飞升者当燃料 |
| 盲点 | 以为九器归位、旧官复职就足以应对第二次归墟大潮 |

他不会享乐，不贪生，不惧死。正因为他有牺牲和责任，复辟道路才有诱惑力。

## 4. 旧天庭履历

秦照古曾属于旧天庭镇界体系，主要职责不是统治某一界，而是巡查中下层世界的界壁、地脉、飞升台和轮回牵引是否稳定。

他亲历过三件事：

1. 旧天庭正常运转时，飞升者确实会被接入更大循环，而不是登记、分流、压榨。
2. 第一次归墟大潮前，旧天庭确实全力修补过大量下界和中层世界。
3. 保全派改写镇界印和登仙台阵枢时，旧循环瞬间断裂，许多旧职修士死在各自岗位上。

这让他对新仙界保全派有真实仇恨，也让他对旧天庭产生近乎信仰的执念。

## 5. 力量与限制

秦照古不能写成一出场就能横扫全书的老怪。

| 项目 | 规则 |
|------|------|
| 境界状态 | 旧籍残身，真实修为已被旧灾和沉眠磨损 |
| 最强状态 | 在旧天庭废墟、旧道职遗址、镇界残阵内可短暂借旧职权柄 |
| 离开废墟后 | 实力明显下滑，需要燃烧旧籍或旧职印痕维持 |
| 擅长 | 镇界、封锁、旧阵调度、旧账识别、九器旧职认定 |
| 不擅长 | 处理新仙界黑账、系统后台、百族血源新债和归墟炉新用法 |
| 代价 | 每次借旧职权柄都会加速旧籍崩散，最终失去旧天庭身份凭证 |

他的压迫感来自旧制度的合法性和经验，不只来自境界压制。

## 6. 资源与权柄

| 资源 | 用法 | 限制 |
|------|------|------|
| 旧籍 | 证明旧天庭身份，可打开部分旧天庭遗址 | 被新仙界视为高危，且本身在崩散 |
| 镇界残印 | 调动旧镇界阵的残余权限 | 对完整新仙界制度无效，只能撬开缝隙 |
| 旧账碎卷 | 证明旧天庭曾经不是收割体系 | 只能证明旧制旧功，不能证明复辟可行 |
| 复辟派残众 | 穆天衡等旧天庭遗民、残职仙官 | 内部也有激进者，秦照古未必能完全约束 |
| 九器旧职知识 | 知道归墟炉、司命盘、轮回灯等旧器原本职责 | 会误判赵长生的新循环道路 |

## 7. 与核心人物关系

| 人物 | 关系 | 冲突或互证 |
|------|------|------------|
| 赵长生 | 尊重、拉拢、理念冲突 | 秦照古要归墟炉归旧职；赵长生要重炼循环 |
| 沈临风 | 怀疑和期待并存 | 他认为沈临风可重燃司命台，但沈临风拒绝再做命数安排者 |
| 顾长庚 | 正统诱惑 | 秦照古可用斩道剑旧痕诱导顾长庚相信旧正统 |
| 谢未明 | 旧天机与新司命的证据互证 | 谢未明能证明司命台污染，却不能证明旧司命台可直接复原 |
| 纪晚凝 | 旧账与补墟账对照 | 秦照古厌恶她的补墟司身份，但她能拿出现制度坏账 |
| 洛闻霜 | 同属旧天庭残线，但路线不同 | 洛闻霜守轮回边界，不愿让旧天庭借亡魂复辟 |
| 裴玄阙 | 旧敌 | 秦照古认为裴玄阙路线是保全派背叛的最终形态 |

## 8. 与赵长生的核心冲突

| 秦照古 | 赵长生 |
|--------|--------|
| 旧天庭曾经救过万界，所以旧职应当恢复 | 旧天庭值得尊重，但旧职不能自动恢复权力 |
| 九器各归其位，万界循环可重启 | 九器必须重组，不再由天庭垄断职责 |
| 仙界必须重新成为万界中枢 | 中枢可以存在，但不能再凌驾万界 |
| 众生需要被更高秩序保护 | 众生必须能参与自身命数和世界循环 |

这条冲突不能写成赵长生单纯骂醒老人。赵长生必须拿出事实：

1. 第一次归墟大潮不是只因叛徒，而是旧循环承载过限。
2. 旧天庭的集中权柄让保全派一次篡改就能切断多界。
3. 第二次归墟大潮需要万界参与新循环，不能再靠天庭单向维护。
4. 旧职复辟会重新制造“少数高位者替众生安排命运”的结构。

## 9. 人物弧线

| 阶段 | 作用 | 内心变化 |
|------|------|----------|
| 第十二卷 | 在旧天庭废墟登场，提出恢复旧天庭路线 | 相信赵长生和沈临风是旧职归位的机会 |
| 第十二卷后段 | 展示旧天庭牺牲、旧账和旧道职残留 | 让主角组理解旧制确有可敬之处 |
| 第十三卷 | 面对裴玄阙和新仙界保全派 | 强化“新仙界是背叛者后裔”的判断 |
| 第十三卷后段 | 被补墟账、轮回账、司命账共同冲击 | 第一次承认旧天庭也没能防止权柄被篡改 |
| 第十四卷 | 在复辟和新循环之间做选择 | 放弃旧权力复辟，保留旧职责见证 |

## 10. 关键情节功能

秦照古至少要承担四个后期功能：

1. 让旧天庭不是背景板，而是有人记得、有人想恢复的真实制度。
2. 让赵长生不能简单站在“反新仙界”一边，而要同时拒绝旧复辟。
3. 推动顾长庚的正统剑道面对“旧正统是否仍正”的问题。
4. 提供旧账对照，让纪晚凝确认新补墟账本不是旧循环的延续，而是篡改后的收割。

## 11. 退场或终局安排

秦照古不适合被赵长生杀死作为爽点。

可选终局方向：

| 方向 | 说明 |
|------|------|
| 守旧天庭残墟 | 承认旧制不可复辟，但守住旧牺牲的记忆和旧账证据 |
| 交出旧籍 | 把旧天庭身份凭证交给新循环公开保管，不再作为复辟合法性 |
| 入新循环有限道职 | 只保留镇界职责，不保留统治权力 |

不建议让他完全加入主角团当普通同伴。他最好保持一种清醒但有距离的见证者位置。

## 12. 禁写项

1. 不能写成愚蠢怀旧老人。
2. 不能写成最终幕后黑手。
3. 不能让“恢复旧天庭”解决终局问题。
4. 不能让他一番话就被赵长生说服。
5. 不能让旧天庭所有问题都推给新仙界保全派，旧制度本身也要承担失败责任。
6. 不能让他抢走顾长庚、谢未明、纪晚凝各自的证据线。
