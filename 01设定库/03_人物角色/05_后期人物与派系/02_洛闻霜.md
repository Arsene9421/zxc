# 洛闻霜

> 后期女性长线角色。洛闻霜负责轮回灯、亡魂归处和复活边界。她存在的核心意义，是让归墟炉不能变成万能复活工具。

---

## 1. 基本定位

| 项目 | 设定 |
|------|------|
| 姓名 | 洛闻霜 |
| 首次登场 | 第十二卷：旧天庭废墟 |
| 身份 | 旧轮回灯守灯人后裔，轮回残职继承者 |
| 所属派系 | 旧天庭守灯残职，不完全属于复辟派 |
| 长线功能 | 轮回灯、生死边界、魂账证据 |
| 感情定位 | 暂不进入赵长生恋爱线 |
| 与赵长生 | 合作者与边界警醒者 |
| 与纪晚凝 | 轮回账和补墟魂账的对照者 |

“闻霜”强调她对死亡冷意的敏感。她不是冰冷无情，而是比任何人都清楚：亡者的残影、执念、真魂和轮回归处不能混为一谈。

## 2. 核心命题

洛闻霜要回答的问题：

> 有能力触碰亡魂的人，是否有资格替亡魂决定归处？

她的答案：

> 轮回不是仓库，不是复活池，也不是强者的私产。亡魂归处必须有边界，否则救人会变成另一种掠夺。

这条线直接限制三类危险写法：

1. 归墟炉聚魂等于复活。
2. 主角想救谁就能救谁。
3. 旧天庭或新仙界都可以用“秩序”名义调度亡魂。

## 3. 人物内核

| 维度 | 内容 |
|------|------|
| 核心欲望 | 让被截留、炼化、污染的亡魂重新进入应有循环 |
| 核心恐惧 | 轮回灯被任何一方拿去当复活、献祭或统治工具 |
| 最大优点 | 边界感极强，不因同情放弃原则 |
| 最大缺点 | 习惯把痛苦压成职责，难以承认自己也会愤怒和疲惫 |
| 底线 | 不允许以救人名义强行拘魂、留魂、炼魂 |
| 盲点 | 初期过度相信“守规矩”能保护亡魂，低估新仙界黑账规模 |

她不是冷淡旁观者。她的情绪很深，只是长期被守灯职责压住。

## 4. 轮回灯传承

旧天庭轮回灯的职责：

1. 接引残魂。
2. 净化怨秽。
3. 断开前世无法承载的因果。
4. 让魂魄重新进入生死循环。
5. 防止亡魂滞留后腐化成归墟污染。

洛闻霜继承的不是完整轮回灯，而是轮回灯的一缕残焰和守灯人规矩。

| 内容 | 设定 |
|------|------|
| 残焰 | 可辨别残魂、魂影、真魂的差异 |
| 灯芯 | 可短暂引魂、安魂、照出魂账断点 |
| 灯油 | 需要命火、功德旧债、魂灯材料或特定地脉支持 |
| 灯规 | 不拘无主真魂，不以亡魂换生者利益，不替权力伪造轮回 |

## 5. 能力与边界

| 能力 | 可做 | 不可做 |
|------|------|--------|
| 闻魂 | 感知残魂、魂影、真魂的大致状态 | 不能凭气息断定完整生平 |
| 引魂 | 把散乱残魂引回灯下，避免被炼化或污染 | 不能把已归档亡魂强行拉回 |
| 安魂 | 平息怨秽，让亡魂进入轮回牵引 | 不能让亡魂带完整记忆转世 |
| 照账 | 发现魂账被截留、磨损、伪造的痕迹 | 不能单独证明补墟全部黑账 |
| 护魂 | 短时护住濒死者真魂不散 | 不能替代肉身、命火、本源修复 |

硬规则：

> 洛闻霜能证明“还有没有可救的真魂”，不能保证“一定能救回来”。

## 6. 与归墟炉的关系

归墟炉负责重炼残败本源，轮回灯负责亡魂归处。

二者可以配合：

1. 归墟炉清理残败本源和归墟污染。
2. 轮回灯确认亡魂状态和轮回牵引方向。
3. 赵长生提供重炼能力，洛闻霜提供生死边界。
4. 二者都不能越过命火、命数、本源和轮回规则随意复活。

冲突点：

| 赵长生可能想做 | 洛闻霜的反应 |
|----------------|--------------|
| 看见熟人魂影后想强行聚回 | 指出魂影不是本人，只是残留记录 |
| 想救一批战死者 | 要求先区分真魂、残魂、怨秽和已归档亡魂 |
| 想用归墟炉净化魂账 | 允许配合，但禁止把亡魂当材料 |
| 想为了终局牺牲部分亡魂换循环启动 | 明确反对，除非是亡魂自愿且不被伪造 |

## 7. 与核心人物关系

| 人物 | 关系 | 冲突或互证 |
|------|------|------------|
| 赵长生 | 合作者、边界监督者 | 她让赵长生明白重炼不是任意挽回 |
| 沈临风 | 命数外壳与异世魂火的识别者 | 她能看出他不是普通夺舍，也不是无代价穿越 |
| 谢未明 | 命数和魂账互证 | 谢未明看命数断点，她看魂账断点 |
| 纪晚凝 | 补墟魂账对照者 | 纪晚凝查账，洛闻霜证明账背后是真魂损耗 |
| 白倩霜 | 族魂和血源边界互证 | 百族祖魂不能被当作血源材料 |
| 秦照古 | 同属旧天庭残线，但不同路 | 秦照古想复旧职，她只守轮回边界 |
| 裴玄阙 | 理念敌人 | 裴玄阙把魂账视作补墟成本，她视作不可越线的公共秩序 |

## 8. 情感线

洛闻霜暂不进入赵长生恋爱线，原因不是她“不够重要”，而是她承担的生死边界不能被恋爱需求稀释。

她的情感推进重点是：

1. 从只信守灯规矩，到承认旧规矩已无法应对新仙界黑账。
2. 从独自守灯，到允许赵长生、纪晚凝、谢未明共同补上证据链。
3. 从保护亡魂不被打扰，到主动为被截留亡魂讨回循环位置。

她可以与主角组建立深度信任，但这种信任首先是职责互证，不是暧昧调味。

## 9. 人物弧线

| 阶段 | 作用 | 内心变化 |
|------|------|----------|
| 第十二卷 | 携轮回灯残焰登场，阻止主角组误用残魂 | 认为所有人都可能越过生死边界 |
| 第十二卷后段 | 展示旧天庭轮回职责和旧灯残缺 | 承认赵长生并非想把亡魂当资源 |
| 第十三卷 | 查出魂账被新仙界补墟体系截留、磨损、伪造 | 从守边界转为追旧账 |
| 第十四卷 | 协助重启轮回灯作为新循环边界组件 | 不复辟旧轮回权柄，只重建亡魂归处规则 |

## 10. 关键情节功能

洛闻霜至少要承担五个功能：

1. 限制归墟炉万能复活。
2. 区分残魂、魂影、真魂，让神魂设定可执行。
3. 证明新仙界补墟体系不仅抽灵、抽血、抽界源，也动过魂账。
4. 让沈临风穿越身份在魂魄层面有可验证边界。
5. 在终局为新循环加入“生死边界”，防止赵长生成为新的生死垄断者。

## 11. 退场或终局安排

洛闻霜适合进入终局职责，不适合牺牲成单纯情绪爆点。

终局方向：

| 方向 | 说明 |
|------|------|
| 重启轮回灯 | 参与恢复亡魂归处，但不独占轮回权柄 |
| 公开魂账 | 与纪晚凝共同把魂账从补墟黑账中剥离出来 |
| 守灯而不封灯 | 建立规则：轮回可被监督，不可被私有 |

## 12. 禁写项

1. 不能写成复活工具人。
2. 不能用她给主角开“死者召回”外挂。
3. 不能把她写成冷冰冰的说明书角色。
4. 不能让她因为同情或爱情随意破坏轮回规矩。
5. 不能让轮回灯替新仙界处理恶意炼魂后的烂账。
6. 不能让她的非恋爱定位变成边缘化，终局必须有职责。
